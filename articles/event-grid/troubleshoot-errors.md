# <a name="troubleshoot-azure-event-grid-errors"></a>Ошибки, ошибки Azure Event Grid ошибки
Это руководство по устранению неполадок предоставляет вам список кодов ошибок Azure Event Grid, сообщения об ошибках, их описания и рекомендуемые действия, которые следует предпринять при получении этих ошибок. 

## <a name="error-code-400"></a>Код ошибки: 400."
| Код ошибки | Сообщение об ошибке | Описание | Рекомендация |
| ---------- | ------------- | ----------- | -------------- | 
| HttpstatusCode.BadRequest<br/>400 | Название темы должно быть от 3 до 50 символов в длину. | Длина пользовательского названия темы должна соться от 3 до 50 символов в длину. В названии темы допускаются только буквы, цифры и символ '-'. Кроме того, название не должно начинаться со следующих зарезервированных слов: <ul><li>Microsoft</li><li>Сетка событий</li><li>Система</li></ul> | Выберите другое название темы, которое соответствует требованиям к названию темы. |
| HttpstatusCode.BadRequest<br/>400 | Доменное имя должно быть от 3 до 50 символов в длину. | Длина доменного имени должна соться от 3 до 50 символов в длину. В названии темы допускаются только буквы, цифры и символ '-'. Кроме того, название не должно начинаться со следующих зарезервированных слов:<ul><li>Microsoft</li><li>Сетка событий</li><li>Система</li> | Выберите другое доменное имя, которое соответствует требованиям доменного имени. |
| HttpstatusCode.BadRequest<br/>400 | Недействительный срок годности. | Время истечения срока действия подписки события определяет, когда подписка на событие будет отклонена. Это значение должно быть действительным значением DateTime в будущем.| Убедитесь, что срок действия подписки на события в действительном формате DateTime будет установлен в будущем. |

## <a name="error-code-409"></a>Код ошибки: 409
| Код ошибки | Сообщение об ошибке | Описание | Рекомендуемое действие |
| ---------- | ------------- | ----------- | -------------- | 
| HttpstatusCode.Conflict <br/>409 | Тема с указанным именем уже существует. Выберите другое название темы.   | Специальное имя темы должно быть уникальным в одном регионе Azure для обеспечения правильной работы публикации. Одно и то же имя можно использовать в разных регионах Azure. | Выберите другое название для темы. |
| HttpstatusCode.Conflict <br/> 409 | Домен с указанным уже существует. Выберите другое доменное имя. | Доменное имя должно быть уникальным в одном регионе Azure, чтобы обеспечить правильную работу публикации. Одно и то же имя можно использовать в разных регионах Azure. | Выберите другое имя для домена. |
| HttpstatusCode.Conflict<br/>409 | Достигнута квота. Для получения дополнительной информации [Azure Event Grid limits](../azure-resource-manager/management/azure-subscription-service-limits.md#event-grid-limits)об этих ограничениях см.  | Каждая подписка Azure имеет ограничение на количество ресурсов Azure Event Grid, которые она может использовать. Некоторые или все эти квоты были превышены, и больше ресурсов не может быть создано. |    Проверьте текущее использование ресурсов и удалите все, что не требуется. Если вам все еще нужно увеличить квоту, отправьте электронное письмо [aeg@microsoft.com](mailto:aeg@microsoft.com) с точным количеством необходимых ресурсов. |
