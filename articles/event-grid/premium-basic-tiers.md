# <a name="azure-event-grid-premium-and-basic-tiers"></a>Уровни "Премиум" и "Базовый" (Сетка событий Azure)
Сетка событий Azure использует две ценовые категории: **Премиум** и **Базовый**. В ценовой категории "Базовый" применяется оплата по мере использования. В этой категории предоставляются все базовые средства публикации и подписки, необходимые для использования Сетки событий в моделях программирования на основе событий. В ценовой категории "Премиум" предоставляется больше возможностей, включая функции безопасности корпоративного уровня. Эта ценовая категория доступна в режиме **ранней предварительной версии**, и многие функции еще разрабатываются.

## <a name="overview"></a>Обзор
Все пользовательские разделы и домены в Сетке событий относятся к одной из двух ценовых категорий: "Базовый" или "Премиум". Начиная с API версии `2020-04-01-preview`, вы можете выбрать нужную ценовую категорию при создании раздела или домена. По умолчанию используется ценовая категория "Базовый". Разделы и домены, созданные с помощью старых версий API, по умолчанию получают ценовую категорию "Базовый". Сведения о том, как изменить ценовую категорию разделов и доменов, см. в [этой статье](update-tier.md).

## <a name="capabilities-and-features"></a>Функции и возможности

В таблице ниже описаны различия между этими ценовыми категориями.

|       &nbsp;                                           | Basic           | Premium        |
| ------------------------------------------------------ | --------------- | -------------- |
| Правила брандмауэра для входящего трафика IP                          | Preview (Предварительный просмотр)  | Preview (Предварительный просмотр) |
| Теги службы для исходящего трафика                                | Preview (Предварительный просмотр)  | Preview (Предварительный просмотр) |
| Интеграция частной конечной точки с виртуальной сетью для входящего трафика          | Недоступно   | Preview (Предварительный просмотр) |

## <a name="availability"></a>Доступность
На этапе ранней предварительной версии разделы и домены ценовой категории "Премиум" с интеграцией частных конечных точек доступны в следующих регионах:

- Восточная часть США
- западная часть США 2
- Центрально-южная часть США

## <a name="pricing-and-quotas"></a>Цены и квоты
Сведения о ценах на использование ценовой категории "Базовый" см. на странице с [ценами на Сетку событий](https://azure.microsoft.com/pricing/details/event-grid/). Цены для ценовой категории "Премиум" пока не объявлены, и до их появления эта категория предоставляется бесплатно.

К ресурсам в ценовой категории "Премиум" и "Базовый" одинаково применяются существующие квоты на число разделов и доменов, а также пропускную способность, пока не будут объявлены цены на ценовую категорию "Премиум".
