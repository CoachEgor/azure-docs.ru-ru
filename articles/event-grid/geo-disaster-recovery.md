# <a name="server-side-geo-disaster-recovery-in-azure-event-grid"></a>Гео-аварийное восстановление на стороне сервера в Azure Event Grid
Event Grid теперь имеет автоматическое геоаварийное восстановление (GeoDR) мета-данных не только для новых, но и для всех существующих доменов, тем и подписки на события. При сбывании всего региона Azure event Grid уже будет синхронизирована со всеми метаданными инфраструктуры, связанными с событиями, с парной областью. Ваши новые события начнут течь снова без вмешательства вами. 

Аварийное восстановление измеряется двумя показателями:

- [Цель точки восстановления (RPO)](https://en.wikipedia.org/wiki/Disaster_recovery#Recovery_Point_Objective): минуты или часы данных, которые могут быть потеряны.
- [Цель времени восстановления (RTO)](https://en.wikipedia.org/wiki/Disaster_recovery#Recovery_time_objective): минуты часов служба может быть вниз.

Автоматический сбой Event Grid имеет различные RpOs и RTOs для ваших метаданных (подписка на события и т.д.) и данных (событий). Если вам нужна другая спецификация от следующих, вы все еще можете реализовать свой [собственный клиент-стороны неудачу над использованием темы здоровья apis](custom-disaster-recovery.md).

## <a name="recovery-point-objective-rpo"></a>Целевая точка восстановления (RPO)
- **Метаданные RPO**: ноль минут. Всякий раз, когда ресурс создается в Event Grid, он мгновенно реплицируется в разных регионах. При сбою происходит, метаданные не теряются.
- **Данные RPO**: Если ваша система здорова и догнал а также существующий трафик во время регионального сбоя, RPO для событий составляет около 5 минут.

## <a name="recovery-time-objective-rto"></a>Целевое время восстановления (RTO)
- **Метаданные RTO**: Хотя обычно это происходит гораздо быстрее, в течение 60 минут, Event Grid начнет принимать создавать / обновлять / удалять звонки по темам и подписки.
- **Данные RTO**: Как и метаданные, как правило, происходит гораздо быстрее, однако в течение 60 минут, Event Grid начнет принимать новый трафик после регионального сбоя.

> [!NOTE]
> Стоимость метаданных GeoDR на Event Grid составляет: $0.
