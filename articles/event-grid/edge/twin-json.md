# <a name="module-twin-json-schema"></a>Модуль двойной схемы JSON

Event Grid on IoT Edge интегрируется с экосистемой IoT Edge и поддерживает создание тем и подписок через Модуль Twin. Он также сообщает о текущем состоянии всех тем и подписки на события в зарегистрированных свойствах на Модуле Twin.

> [!WARNING]
> Из-за ограничений в экосистеме IoT Edge все элементы массива в следующем примере json были закодированы как строки json. Смотрите `EventSubscription.Filter.EventTypes` `EventSubscription.Filter.AdvancedFilters` и ключи в следующем примере.

## <a name="desired-properties-json"></a>Нежелательные свойства JSON

* Значение каждой пары ключей в разделе тем имеет точно такую же схему `Topic.Properties` JSON, которая используется для API при создании тем.
* Значение каждой пары ключей в разделе **EventSubscriptions** имеет точно такую же схему `EventSubscription.Properties` json, которая используется для aPI при создании тем.
* Чтобы удалить тему, установите `null` ее значение в желаемых свойствах.
* Удаляние подписки на события через желаемые свойства не поддерживается.

```json
{
    "topics": {
        "twinTopic1": {},
        "twinTopic2": {
            "inputSchema": "customEventSchema"
        }
    },
    "eventSubscriptions": {
        "twinTopic1WebhookSub": {
            "topic": "twinTopic1",
            "retryPolicy": {
                "eventExpiryInMinutes": 120,
                "maxDeliveryAttempts": 30
            },
            "destination": {
                "endpointType": "WebHook",
                "properties": {
                    "endpointUrl": "https://localhost:4438"
                }
            },
            "filter": {
                "subjectBeginsWith": "^",
                "subjectEndsWith": "$",
                "isSubjectCaseSensitive": false,
                "includedEventTypes": "[\"et1\",\"et2\"]",
                "advancedFilters": "[{\"value\":true,\"operatorType\":\"BoolEquals\",\"key\":\"data.b\"},{\"values\":[\"\\\"\",\"c\"],    \"operatorType\":\"StringContains\",\"key\":\"data.s\"}]"
            }
        },
        "twinTopic2EdgeHubSub": {
            "topic": "twinTopic2",
            "deliveryPolicy": {
                "approxBatchSizeInBytes": 200000,
                "maxEventsPerBatch": 25
            },
            "destination": {
                "endpointType": "EdgeHub",
                "properties": {
                    "outputName": "twinTopic2EdgeHubSub"
                }
            },
            "filter": {
                "advancedFilters": "[{\"value\":true,\"operatorType\":\"BoolEquals\",\"key\":\"dAt\\\"A.a\"},{\"values\":[\"\\\"\",    \"c\"],\"operatorType\":\"StringContains\",\"key\":\"dAt\\\"A.a\"}]"
            }
        }
    }
}
```

## <a name="reported-properties-json"></a>Сообщенные свойства JSON

Сообщаемый раздел свойств двойника модуля включает в себя следующую информацию:

* Набор тем и подписок, которые существуют в магазине модуля
* Любые ошибки, возникающие при создании желаемых тем/подписок на события
* Любые ошибки загрузки (например, желаемые свойства JSON разбора не удалось)
