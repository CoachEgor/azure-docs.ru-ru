# <a name="release-notes-azure-event-grid-on-iot-edge"></a>Примечания к выпуску: Azure Event Grid на Edge IoT

## <a name="100-preview1"></a>1.0.0-предварительный просмотр1

Первоначальный выпуск Azure Event Grid на IoT Edge. Включены функции:

* Создание темы
* Создание подписки на события
* расширенные фильтры
* Пакетная обработка выходных данных
* Политики повтора
* Модуль для публикации модуля
* Публикация в WebHook в качестве пункта назначения
* Публикация в Концентратор IoT Edge в качестве пункта назначения
* Публикация облачного сервиса Azure Event Grid в качестве пункта назначения
* Устойчивое состояние для метаданных
* Интеграция модуля памяти Blob

Теги:`1.0.0-preview1`

## <a name="100-preview2"></a>1.0.0-preview2

Предварительный просмотр 2 Azure Event Grid на IoT Edge добавил:

* Настраиваемые сохраняющиеся события на диск
* Метрики темы
* Метрики подписки на события
* Публикация концентратов событий в качестве пункта назначения
* Публикация в очередях на автобусы в качестве пункта назначения
* Публикация в сервисных направлениях автобусов в качестве пункта назначения
* Публикация в очередях хранения в качестве пункта назначения

Теги: `1.0.0-preview2` `1.0`, ,`latest`
