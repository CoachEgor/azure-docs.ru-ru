# <a name="partner-topics-in-azure-event-grid-preview"></a>Партнерские разделы в Сетке событий Azure (предварительная версия)
Партнерские разделы позволяют подключать сторонние источники событий непосредственно к Сетке событий. Благодаря этому виду интеграции вы можете подписаться на события от партнеров так же, как вы подписываетесь на события, поступающие из служб Azure. 

## <a name="available-partners"></a>Доступные партнеры
Первый партнер, доступный через партнерские разделы Сетки событий, — это Auth0. [Партнерский раздел Auth0](auth0-overview.md) позволяет соединить учетные записи Auth0 и Azure. Эта интеграция позволяет реагировать на события, вести журналы событий и отслеживать события Auth0 в режиме реального времени.

[Испытайте ее](auth0-how-to.md) прямо сейчас. Для этого войдите в свою учетную запись Auth0 и создайте интеграцию с Сеткой событий. После нажатия кнопки "Создать" в Auth0 вы увидите ожидающий ваших действий раздел Auth0 в своей учетной записи Azure. Щелкните "Активировать". Теперь вы можете создавать подписки на события, маршрутизировать, фильтровать и доставлять события так же, как вы это делаете для любого другого источника событий.

## <a name="pricing"></a>Цены
Партнерские разделы тарифицируются так же, как и системные разделы.

## <a name="limits"></a>Ограничения
Разделы партнеров находятся на этапе общедоступной предварительной версии. В общедоступной предварительной версии на партнерские разделы распространяются [те же ограничения](https://docs.microsoft.com/azure/azure-resource-manager/management/azure-subscription-service-limits#event-grid-limits), что и на системные и пользовательские разделы.

## <a name="how-do-i-become-an-event-grid-partner"></a>Как стать партнером Сетки событий?
Инфраструктура, созданная для запуска этой службы, позволяет новым партнерам упростить и ускорить интеграцию своих возможностей по работе с событиями в службу "Сетка событий". Дополнительные сведения см. в документации по [подключению партнеров](partner-onboarding-overview.md).
