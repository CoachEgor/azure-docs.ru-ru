# <a name="visualize-your-graph-data-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Визуализация данных в Azure Cosmos DB Gremlin API graph с помощью решений в области визуализации данных

Рекомендуется использовать следующие решения по [сообщества Apache Tinkerpop](http://tinkerpop.apache.org/#poweredby) для визуализации графа Gremlin.

## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Логотип Linkurious](./media/graph-visualization/linkurious-logo.jpg)

Linkurious Enterprise используется graph технологии и визуализации данных для преобразования сложные наборы данных в интерактивные visual сетей. Платформа подключается к источникам данных и позволяет предупреждают без труда переходить между миллиарды сущностей и связей. Результатом является новый возможность обнаруживать подозрительные отношения без работа с запросами или таблицами.

Интерактивная визуализация интерфейс Linkurious Enterprise предлагает простой способ анализа сложных данных. Поиск определенных сущностей, подключения, чтобы выявить скрытые отношения и применить макеты по своему усмотрению, чтобы клубка сложная топология сетей. Linkurious теперь совместима с Azure Cosmos DB Gremlin API. Он подходит для сценариев end-to-end graph визуализации и поддерживает возможности чтения и записи с помощью пользовательского интерфейса. 

![Демонстрация Linkurious](./media/graph-visualization/linkurious-demo.gif)

**Дальнейшие действия:** Вы можете запросить [демонстрационную версию Linkurious с Azure Cosmos DB](https://linkurio.us/contact/).
