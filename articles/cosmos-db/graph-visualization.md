# <a name="visualize-graph-data-stored-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Визуализации данных graph в Azure Cosmos DB Gremlin API с помощью решений в области визуализации данных

Вы можете визуализировать данные, хранящиеся в Azure Cosmos DB Gremlin API с помощью различных решений в области визуализации данных. Рекомендуется использовать следующие решения по [сообщества Apache Tinkerpop](http://tinkerpop.apache.org/#poweredby) для визуализации данных графа.

## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Логотип Linkurious](./media/graph-visualization/linkurious-logo.jpg)

[Linkurious Enterprise](https://linkurio.us/product/) использует graph технологии и визуализации данных Чтобы превратить сложные наборы данных в интерактивные visual сетей. Платформа подключается к источникам данных и позволяет предупреждают без труда переходить между миллиарды сущностей и связей. Результатом является новый возможность обнаруживать подозрительные отношения без работа с запросами или таблицами.

Интерактивный интерфейс Linkurious Enterprise предлагает простой способ анализа сложных данных. Поиск определенных сущностей, подключения, чтобы выявить скрытые отношения и применять макеты по своему усмотрению, чтобы клубка сложная топология сетей. Linkurious Enterprise теперь совместима с Azure Cosmos DB Gremlin API. Он подходит для сценариев end-to-end graph визуализации и поддерживает возможности чтения и записи с помощью пользовательского интерфейса. Вы можете запросить [демонстрационную версию Linkurious с Azure Cosmos DB](https://linkurio.us/contact/)

## <a name="cambridge-intelligence"></a>Кембридж аналитики

![Логотип Кембридж аналитики](./media/graph-visualization/ci-logo.png)

[Кембридж аналитики](https://cambridge-intelligence.com/products/) наборов средств визуализации диаграммы теперь поддерживают Azure Cosmos DB. Azure Cosmos DB поддерживает следующие два наборов средств визуализации.

- [Контуры для разработчиков JavaScript](https://cambridge-intelligence.com/keylines/)

- [Повторно графа для разработчиков React](https://cambridge-intelligence.com/regraph/)

![Демонстрация Кембридж аналитики](./media/graph-visualization/ci-demo-2.gif)

Эти наборы средств позволяют создавать визуализации graph высокой производительности и приложения для анализа для варианта использования. Они возможности мощные средства отображения графики Library(WebGL) Web и тщательно созданный код, чтобы предоставить пользователям возможность быстрого и информативные визуализации. Эти инструменты совместимы с помощью любого браузера, устройства, сервера или базы данных и предоставляются пошаговые учебники, полностью документированных API и интерактивные демонстрации.
