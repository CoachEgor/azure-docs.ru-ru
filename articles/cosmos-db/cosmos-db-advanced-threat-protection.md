# <a name="advanced-threat-protection-for-azure-cosmos-db-preview"></a>Расширенная защита от угроз для Azure Cosmos DB (Предварительный просмотр)

Расширенная защита от угроз для Azure Cosmos DB предоставляет дополнительный уровень интеллекта безопасности, который обнаруживает необычные и потенциально опасные попытки получить доступ к учетным записям Azure Cosmos DB или использовать их. Этот уровень защиты позволяет устракиваться в угрозах, даже не будучи экспертом по безопасности, и интегрировать их с центральными системами мониторинга безопасности.

Оповещения безопасности срабатывают при возникновении аномалий в активности. Эти оповещения о безопасности интегрированы с [Azure Security Center,](https://azure.microsoft.com/services/security-center/)а также отправляются по электронной почте администраторам подписки с подробной информацией о подозрительной деятельности и рекомендациями о том, как исследовать и исправления угроз.

> [!NOTE]
>
> * Расширенная защита от угроз для Azure Cosmos DB в настоящее время доступна только для API S'L.
> * Расширенная защита от угроз для Azure Cosmos DB в настоящее время недоступна в правительственных и суверенных облачных регионах Azure.

Для полного изучения оповещений о безопасности мы рекомендовали включить [диагностическую запись в Azure Cosmos DB,](https://docs.microsoft.com/azure/cosmos-db/logging)которая регистрирует операции самой базы данных, включая операции CRUD по всем документам, контейнерам и базам данных.

## <a name="threat-types"></a>Типы угроз

Расширенная защита от угроз для Azure Cosmos DB обнаруживает аномальные действия, указывающие на необычные и потенциально опасные попытки получить доступ к базам данных или эксплуатировать их. В настоящее время он может вызвать следующие предупреждения:

- **Доступ из необычных местоположений:** Это оповещение запускается при изменении шаблона доступа к учетной записи Azure Cosmos, где кто-то подключился к конечной точке Azure Cosmos DB из необычного географического положения. В некоторых случаях оповещение обнаруживает законное действие, то есть новое приложение или эксплуатацию разработчика. В других случаях оповещение обнаруживает вредоносное действие от бывшего сотрудника, внешнего злоумышленника и т.д.

- **Необычная извлечение данных**: Это оповещение срабатывает, когда клиент извлекает необычный объем данных из учетной записи Azure Cosmos DB. Это может быть симптомом некоторых эксфильтрации данных, выполняемой для передачи всех данных, хранящихся в учетной записи, во внешний хранилище данных.

## <a name="set-up-advanced-threat-protection"></a>Настройка расширенной защиты от угроз

### <a name="set-up-atp-using-the-portal"></a>Настройка ATP с помощью портала

1. Запуск портала Azure на [https://portal.azure.com](https://portal.azure.com/).

2. Из учетной записи Azure Cosmos DB из меню **«Настройки»** выберите **расширенную безопасность.**

    ![Настройка ATP](./media/cosmos-db-advanced-threat-protection/cosmos-db-atp.png)

3. В лезвии конфигурации **расширенной безопасности:**

    * Нажмите на опцию **Расширенная защита угроз,** чтобы установить его **на ON.**
    * Щелкните **Сохранить**, чтобы сохранить новую или обновленную политику Расширенной защиты от угроз.   

### <a name="set-up-atp-using-rest-api"></a>Настройка ATP с помощью REST API

Используйте команды Rest API для создания, обновления или получения настройки расширенной защиты угроз для определенной учетной записи Azure Cosmos DB.

* [Расширенная защита от угроз - Создание](https://go.microsoft.com/fwlink/?linkid=2099745)
* [Расширенная защита от угроз - Получить](https://go.microsoft.com/fwlink/?linkid=2099643)

### <a name="set-up-atp-using-azure-powershell"></a>Настройка ATP с помощью Azure PowerShell

Используйте следующие смдлеты PowerShell:

* [Включение службы "Расширенная защита от угроз"](https://go.microsoft.com/fwlink/?linkid=2099607&clcid=0x409)
* [Получите расширенную защиту от угроз](https://go.microsoft.com/fwlink/?linkid=2099608&clcid=0x409)
* [Отключить расширенную защиту от угроз](https://go.microsoft.com/fwlink/?linkid=2099709&clcid=0x409)

### <a name="using-azure-resource-manager-templates"></a>Использование шаблонов диспетчера ресурсов Azure

Используйте шаблон менеджера ресурсов Azure для настройки Программы Космос dB с включенной расширенной защитой от угроз.
Для получения дополнительной информации [см.](https://azure.microsoft.com/resources/templates/201-cosmosdb-advanced-threat-protection-create-account/)

### <a name="using-azure-policy"></a>Использование политики Azure

Используйте политику Azure для обеспечения расширенной защиты от угроз для Cosmos DB.

1. Запустите страницу Azure **Policy - Definitions** и ищите **расширенную защиту от угроз для политики Cosmos DB.**

    ![Политика поиска](./media/cosmos-db-advanced-threat-protection/cosmos-db.png) 

1. Нажмите на **расширенную защиту от угроз развертывания для политики CosmosDB,** а затем нажмите **Назначайте Assign.**

    ![Выберите подписку или группу](./media/cosmos-db-advanced-threat-protection/cosmos-db-atp-policy.png)


1. Из поля **Scope** щелкните три точки, выберите подписку Azure или группу ресурсов, а затем нажмите **Выберите**.

    ![Страница определения политики](./media/cosmos-db-advanced-threat-protection/cosmos-db-atp-details.png)


1. Введите другие параметры и нажмите **Назначай**.
