# <a name="aggregate-functions-in-azure-cosmos-db"></a>Агрегатные функции в Azure Cosmos DB
[!INCLUDE[appliesto-sql-api](includes/appliesto-sql-api.md)]

Агрегатные функции выполняют вычисление для набора значений в `SELECT` предложении и возвращают одно значение. Например, следующий запрос возвращает количество элементов в `Families` контейнере:

## <a name="examples"></a>Примеры

```sql
    SELECT COUNT(1)
    FROM Families f
```

Вы получите такие результаты:

```json
    [{
        "$1": 2
    }]
```

Можно также вернуть только скалярное значение статистической функции с помощью ключевого слова VALUE. Например следующий запрос возвращает число значений как одно число:

```sql
    SELECT VALUE COUNT(1)
    FROM Families f
```

Вы получите такие результаты:

```json
    [ 2 ]
```

Кроме того, можно объединять агрегаты с фильтрами. Например, следующий запрос возвращает количество элементов с состоянием адреса `WA` .

```sql
    SELECT VALUE COUNT(1)
    FROM Families f
    WHERE f.address.state = "WA"
```

Вы получите такие результаты:

```json
    [ 1 ]
```

## <a name="types-of-aggregate-functions"></a>Типы агрегатных функций

API SQL поддерживает следующие агрегатные функции. `SUM` и `AVG` используют числовые значения, и `COUNT` , и `MIN` `MAX` работают с числами, строками, логическими значениями и значениями NULL.

| Функция | Описание |
|-------|-------------|
| COUNT | Возвращает число элементов в выражении. |
| SUM   | Возвращает сумму всех элементов в выражении. |
| MIN   | Возвращает минимальное значение выражения. |
| MAX   | Возвращает максимальное значение выражения. |
| AVG;   | Возвращает среднее арифметическое значений в выражении. |

Можно также выполнить статистическую обработку результатов итерации массива.

> [!NOTE]
> В обозреватель данных портал Azure статистические запросы могут объединять частичные результаты только на одной странице запроса. Пакет SDK создает одно совокупное значение для всех страниц. Для выполнения статистических запросов с помощью кода требуется пакет SDK для .NET 1.12.0, пакет SDK для .NET Core 1.1.0 или Java SDK 1.9.5 или более поздней версии.
