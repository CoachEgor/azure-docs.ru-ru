# <a name="regional-presence-with-azure-cosmos-db"></a>Географическое присутствие службы Azure Cosmos DB

Azure Cosmos DB — это базовая служба в Azure и, по умолчанию всегда доступны во всех регионах, где предоставляется Azure. Сейчас Azure доступна в [54 регионах](https://azure.microsoft.com/global-infrastructure/regions/) по всему миру. 

[![Регионы, в которых доступна служба Azure Cosmos DB](./media/regional-presence/regional-presence.png)](./media/regional-presence/regional-presence.png#lightbox)

Cosmos DB доступна для клиентов в пяти различных облачных средах Azure:

* **Общедоступный пиринг Azure** — облако, которое доступно по всему миру.

* **Azure предоставляемая 21Vianet в Китае** доступны благодаря партнерским отношениям между корпорацией Майкрософт и компанией 21Vianet, одним из страны крупнейшим Интернет-провайдером в Китае.

* **Azure для Германии** — предоставляет службы по модели доверенного лица по вопросам данных. Это означает, что данные клиента хранятся в Германии в T-Systems International GmbH, дочернем подразделении компании Deutsche Telecom, выступающем в роли доверенного лица по вопросам данных в Германии.

* **Azure для государственных организаций** — доступна в четырех регионах в Соединенных Штатах для государственных организаций США и их партнеров. 

* **Azure для государственных организаций для министерства обороны (DoD)** доступен в двух регионах США для министерства обороны США.

## <a name="regional-presence-with-global-distribution"></a>Географическое присутствие с глобальным распределением

Все API-интерфейсы, предоставляемые Azure Cosmos DB (включая SQL, MongoDB, Cassandra, Gremlin и таблицы) доступны во всех регионах Azure по умолчанию. Например у вас есть MongoDB и Cassandra API-интерфейсы предоставляемые Azure Cosmos DB не только во всех глобальных регионах Azure, но также в независимых облаках для Китая, Германии, для государственных организаций и министерства обороны (DoD) регионах.

Azure Cosmos DB — [глобально распределенная](distribute-data-globally.md) обслуживание базы данных. Вы можете связать любое количество регионов Azure с учетной записью Azure Cosmos, и ваши данные будут автоматически и прозрачно реплицированы. Добавить или удалить регион в учетной записи Azure Cosmos можно в любое время. Благодаря комплексной возможности глобального распределения и протоколу репликации с несколькими источниками Azure Cosmos DB обеспечивает задержку операций чтения и записи менее 10 мс на уровне 99-го процентиля, возможность чтения и записи на уровне 99,999, а также возможность гибко масштабировать подготовленную пропускную способность для операций чтения и записи во всех регионах, связанных с учетной записью Azure Cosmos. Azure Cosmos DB также предлагает пять четко определенных моделей согласованности, и вы можете применить к своим данным конкретную модель. Наконец, Azure Cosmos DB — единственная служба базы данных в отрасли, предоставляет полную [уровень соглашения об уровне обслуживания](https://azure.microsoft.com/support/legal/sla/cosmos-db/v1_2/) охватывающий подготовленной пропускной способности и задержки на 99-го процентиля, высокий уровень доступности и согласованность. Выше возможности доступны во всех облаках Azure.
