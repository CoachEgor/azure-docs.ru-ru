# <a name="regional-presence-with-azure-cosmos-db"></a>Географическое присутствие службы Azure Cosmos DB

Azure Cosmos DB — это базовая служба в Azure. по умолчанию она всегда доступна во всех регионах, где доступна служба Azure. Сейчас Azure доступна в [54 регионах](https://azure.microsoft.com/global-infrastructure/regions/) по всему миру. 

[![Регионы, в которых доступна Azure Cosmos DB](./media/regional-presence/regional-presence.png)](./media/regional-presence/regional-presence.png#lightbox)

Cosmos DB доступна для клиентов в пяти различных облачных средах Azure:

* **Общедоступный пиринг Azure** — облако, которое доступно по всему миру.

* Версия **Azure для Китая (21vianet** ) доступна с помощью уникальной связи между корпорацией Майкрософт и 21vianet, одним из крупнейших поставщиков услуг Интернета в Китае.

* **Azure для Германии** — предоставляет службы по модели доверенного лица по вопросам данных. Это означает, что данные клиента хранятся в Германии в T-Systems International GmbH, дочернем подразделении компании Deutsche Telecom, выступающем в роли доверенного лица по вопросам данных в Германии.

* **Azure для государственных организаций** — доступна в четырех регионах в Соединенных Штатах для государственных организаций США и их партнеров. 

* Службы **Azure для государственных организаций в области защиты (DoD)** доступны в двух регионах США в Отдел обороны США.

## <a name="regional-presence-with-global-distribution"></a>Географическое присутствие с глобальным распределением

Все API-интерфейсы, предоставляемые Azure Cosmos DB (включая SQL, MongoDB, Cassandra, Gremlin и Table), доступны во всех регионах Azure по умолчанию. Например, вы можете использовать API MongoDB и Cassandra, предоставляемые Azure Cosmos DB не только во всех глобальных регионах Azure, но также и в облаках независимых, таких как Китай, Германия, правительственный отдел и регионы с защитой по требованию.

Azure Cosmos DB является [глобально распределенной](distribute-data-globally.md) службой базы данных. Вы можете связать любое количество регионов Azure с учетной записью Azure Cosmos, и ваши данные будут автоматически и прозрачно реплицированы. Добавить или удалить регион в учетной записи Azure Cosmos можно в любое время. Благодаря комплексной возможности глобального распределения и протоколу репликации с несколькими источниками Azure Cosmos DB обеспечивает задержку операций чтения и записи менее 10 мс на уровне 99-го процентиля, возможность чтения и записи на уровне 99,999, а также возможность гибко масштабировать подготовленную пропускную способность для операций чтения и записи во всех регионах, связанных с учетной записью Azure Cosmos. Azure Cosmos DB также предлагает пять четко определенных моделей согласованности, и вы можете применить к своим данным конкретную модель. Наконец, Azure Cosmos DB является единственной службой базы данных в отрасли, предоставляющей комплексную [соглашение об уровне обслуживания (SLA)](https://azure.microsoft.com/support/legal/sla/cosmos-db/v1_2/) , включающую подготовленную пропускную способность, задержку на 99-м процентиль, высокий уровень доступности и согласованность. Перечисленные выше возможности доступны во всех облаках Azure.
