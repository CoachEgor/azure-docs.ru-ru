# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

В этой статье описаны причины и решения для **503003 РазделNotFound** ошибок.

## <a name="symptoms"></a>Симптомы

Запросы в IoT Концентратор не сдается с ошибкой **503003 РазделNotFound**.

## <a name="cause"></a>Причина

Эта ошибка является внутренней для Концентратора IoT и, вероятно, преходящая. Посмотреть [причину внутренней ошибки сервера IoT](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause)Hub.

## <a name="solution"></a>Решение
