# <a name="use-one-click-ingestion-to-ingest-data-into-azure-data-explorer"></a>Используйте одним нажатием кнопки для приема данных в Azure Data Explorer

В этой статье показано, как использовать один клик для быстрого приема новой таблицы в форматах JSON или CSV. Данные могут понижаться из хранилища или локального файла в существующую таблицу или новую таблицу. Используйте интуитивно понятный мастер одним щелчком мыши, и ваши данные глотают в течение нескольких минут. Затем можно отсеять таблицу и запускать запросы с помощью веб-мина Azure Data Explorer.

## <a name="prerequisites"></a>Предварительные требования

* Если у вас еще нет подписки Azure, создайте [бесплатную учетную запись](https://azure.microsoft.com/free/) Azure, прежде чем начинать работу.
* Войти в [приложение](https://dataexplorer.azure.com/).
* Создайте [кластер и базу данных Azure Data Explorer.](create-cluster-database-portal.md)
* Воссоединитесь с [веб-системой и](https://dataexplorer.azure.com/) [добавьте подключение к кластеру.](/azure/data-explorer/web-query-data#add-clusters)

## <a name="ingest-new-data"></a>Глотать новые данные

1. Нажмите правой *кнопкой мыши базы данных* или *строки таблицы* в левом меню web-мото-миноносного файла и выберите **новые данные Ingest (Preview).**

    ![Выберите один клик в веб-единоборщее](media/ingest-data-one-click/one-click-ingestion-in-webui.png)   
 
1. В окне **ingest new data (Preview)** выберите вкладку **Исходитель** и завершите **детали проекта:**

    * Для **таблицы**выберите существующее название таблицы из выпадающего меню или выберите **Создать новую,** чтобы сделать новую таблицу.
    * Для **типа ingestion**выберите либо **из хранилища,** либо из **файла.**
      * Если вы выбрали **из хранилища,** выберите **ссылку на хранилище,** чтобы добавить URL. Используйте [URL Blob SAS](/azure/vs-azure-tools-storage-explorer-blobs#get-the-sas-for-a-blob-container) для личных учетных записей. 
      * Если вы выбрали **файл,** выберите **Просмотр** и перетащите файл в поле.
    * Выберите **схему Edit** для просмотра и отсвачения конфигурации столбца таблицы.
 
    ![Информация источника источника приема одного клика](media/ingest-data-one-click/one-click-ingestion-source.png) 

    > [!TIP]
    > При выборе **новых данных Ingest (Preview)** в строке *таблицы* выбранное имя таблицы будет отображаться в **деталях проекта.**

1. При выборе существующей таблицы окно **столбцов карты** открывается для отображения столбцов исходных данных в целевые столбцы таблиц. 
    * Используйте **столбец Omit** для удаления целевого столбца из таблицы.
    * Используйте **новый столбец,** чтобы добавить новый столбец в таблицу.

    ![Окно столбцов карты](media/ingest-data-one-click/one-click-map-columns-window.png)

1. Во вкладке **Schema:**

    * Выберите **тип сжатия** из выпадающего меню, а затем выберите либо **несжатый,** либо **ГИП.**
    * Выберите **формат данных** из выпадающего меню, а затем выберите **JSON,** **CSV,** **TSV,** **SCSV**, **SOHSV,** **TSVE**, или **PSV.** 
        * При выборе формата **JSON** необходимо также выбрать **уровни JSON**от 1 до 10. Уровни влияют на изображение данных столбца таблицы. 
        * Если вы выберете формат, не являющийся JSON, необходимо выбрать флажок **Включите имена столбцов,** чтобы проигнорировать строку заголовка файла.
    * **Имя отображения** устанавливается автоматически, но может быть отредактировано.
    * При выборе существующей таблицы можно выбрать **столбцы карты,** чтобы открыть окно **столбцов Карты.**

    ![Схема формата CSV одним щелчком мыши](media/ingest-data-one-click/one-click-csv-format.png)

1. Над панелью **редактора** выберите кнопку **v,** чтобы открыть редактор. В редакторе можно просматривать и копировать автоматические запросы, генерируемые из входных данных. 

1. В таблице: 
    * Нажмите правой щелкнуть новые заголовки столбцов для **изменения типа данных,** **переименовать столбец**, **Удалить столбец**, **Сортировать восходящий**, или **сортировать нисходящий**. На существующих столбцах доступна только сортировка данных. 
    * Дважды щелкните новое имя столбца для отослаить.

1. Выберите **Начало приема для** создания таблицы и отображения и для начала приема данных.

    ![Схема формата JSON одним щелчком мыши](media/ingest-data-one-click/one-click-json-format.png) 
 
## <a name="query-data"></a>Данные запросов

1. В **заполненном** окне приема данных все три шага будут отмечены зелеными контрольными отметками, если проглатывание данных завершится успешно.
 
    ![Полное проглатывание данных одним щелчком мыши](media/ingest-data-one-click/one-click-data-ingestion-complete.png)

1. Выберите кнопку **v,** чтобы открыть запрос. Копирование веб-мина для отсеивателя запроса.

1. Меню справа содержит **быстрые запросы** и **варианты инструментов.** 

    * **Быстрые запросы** включают ссылки на веб-узел с примерами запросов.
    * **Инструменты** включают в себя ссылку на **команды Drop** в веб-мине, `.drop` которые позволяют устранить проблемы, запустив соответствующие команды.

    > [!TIP]
    > Вы можете потерять данные при использовании `.drop` команд. Используйте их осторожно.
