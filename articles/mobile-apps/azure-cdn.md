# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Сеть доставки содержимого в Azure с помощью сети доставки содержимого Azure
[Сеть доставки содержимого Azure](https://azure.microsoft.com/services/cdn/) предлагает разработчикам глобальное решение для быстрой доставки содержимого с высокой пропускной способностью, например видео, изображений или статических файлов, путем кэширования содержимого на сотни точек присутствия по всему миру. Сеть доставки содержимого также может ускорить динамическое содержимое, которое не может быть кэшировано, чтобы сократить время загрузки, сэкономить пропускную способность и повысить скорость реагирования. Эти преимущества вы получаете при разработке или управлении веб-сайтами или мобильными приложениями, а также для кодирования и распространения потоковых мультимедиа, игровых программных продуктов, обновлений встроенного по или конечных точек IoT.

## <a name="azure-content-delivery-network-features"></a>Функции сети доставки содержимого Azure
  - Легко работать со службами Azure, такими как веб-приложения, службы мультимедиа, хранилище и облачные службы.
  - Легко обрабатывайте пики трафика и интенсивные нагрузки.
  - Интегрируйтесь с Akamai и Verizon для создания и адаптации стратегии доставки содержимого.
  - Использовать динамическое ускорение сайтов.
  - Управление поведением кэширования сети доставки содержимого с помощью правил кэширования.
  - Получите поддержку пользовательского домена HTTPS.
  - Просмотрите журналы диагностики Azure, чтобы просмотреть основную аналитику и экспортировать их в хранилище BLOB-объектов Azure, в концентраторы событий Azure или Azure Monitor.
  - Повышение производительности за счет сжатия файлов.
  - Используйте геофильтрацию для ограничения содержимого сети доставки содержимого по странам или регионам.
