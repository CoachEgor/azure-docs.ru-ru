# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Сеть доставки контента в Azure с сетью доставки контента Azure
[Azure Content Delivery Network](https://azure.microsoft.com/services/cdn/) предлагает разработчикам глобальное решение для быстрого предоставления пользователям контента с высокой пропускной способностью, такого как видео, изображения или статических файлов, путем кэширования их содержимого до сотен точек присутствия по всему миру. Сеть доставки контента также может ускорить динамический контент, который не может быть кэширован, чтобы сократить время загрузки, сохранить пропускную способность и повысить отзывчивость. Вы получаете эти преимущества независимо от того, разрабатываете ли вы веб-сайты или мобильные приложения или кодируете и распространяете потоковые носители, игровое программное обеспечение, обновления прошивки или конечные точки IoT.

## <a name="azure-content-delivery-network-features"></a>Функции сети доставки содержимого Azure
  - Бесперебойно работайте с службами Azure, такими как веб-приложения, медиа-службы, хранилища и облачные службы.
  - Легко обрабатывать движения шипы и тяжелые нагрузки.
  - Интегрируйтесь с Akamai и Verizon, чтобы построить и адаптировать стратегию доставки контента.
  - Используйте динамическое ускорение сайта.
  - Управление поведением кэширования сети доставки содержимого с помощью правил кэширования.
  - Получите пользовательскую поддержку домена HTTPS.
  - Просматривайте журналы диагностики Azure, чтобы просмотреть основные аналитики и экспортировать их в хранилища Azure Blob, концентраторы событий Azure или Azure Monitor.
  - Улучшение производительности путем сжатия файлов.
  - Используйте геофильтрацию для ограничения содержимого сети доставки по странам или регионам.
