# <a name="content-delivery-network-on-azure-with-azure-cdn-service"></a>Сеть доставки содержимого в Azure со службой Azure CDN
[Azure CDN](https://azure.microsoft.com/services/cdn/) предлагает разработчикам глобальное решение для быстрой доставки содержимого с высокой пропускной способностью, например видео, изображений или статических файлов, путем кэширования содержимого в сотни точек присутствия (POP) по всему миру. Azure CDN также может ускорить динамическое содержимое, которое не может быть кэшировано, и позволяет сократить время загрузки, сэкономить пропускную способность и повысить скорость реагирования — будь то разработка или управление веб-сайтами или мобильными приложениями, а также кодирование и распространение потоковой передачи. мультимедиа, Игровая программа, обновления встроенного по или конечные точки IoT.

## <a name="azure-cdn-features"></a>Возможности Azure CDN
  - **Эффективно** работает со службами Azure, включая веб-приложения, службы мультимедиа, хранилище и облачные службы.
  - Легко **обрабатывайте пики трафика и интенсивные нагрузки**.
  - **Интеграция с Akamai и Verizon** для создания и адаптации стратегии доставки содержимого.
  - Динамическое ускорение сайтов.
  - Управление поведением Azure CDN кэширования с помощью **правил кэширования**.
  - **Поддержка пользовательского домена**HTTPS.
  - **Журналы диагностики Azure** для просмотра основных аналитиков и их экспорта в хранилище BLOB-объектов, концентраторы событий или Azure Monitor.
  - Повышение производительности за счет **сжатия файлов**.
  - **Географическая фильтрация** для ограничения Azure CDN содержимого по странам или регионам.
