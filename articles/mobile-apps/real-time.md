# <a name="build-real-time-applications-with-azure-signalr"></a>Создавайте приложения в режиме реального времени с помощью Azure SignalR

[Azure SignalR](https://azure.microsoft.com/services/signalr-service/) позволяет легко добавлять в приложения функции, работающие в режиме реального времени. Эта функция в режиме реального времени позволяет службе отправлять обновления содержимого в подключенные мобильные приложения. В результате приложения обновляются без необходимости опроса сервера или отправки новых HTTP-запросов на обновления.

## <a name="azure-signalr-features"></a>Функции Azure SignalR
- **Широкая поддержка клиентов** — она работает с широким спектром клиентов, таких как веб-и мобильные браузеры, классические приложения, серверный процесс, устройства IOT и игровые консоли.
- Служба SignalR позволяет **нескольким экземплярам работать вместе** для масштабирования до миллионов подключений мобильных приложений. Он также поддерживает несколько глобальных регионов для сегментирования, высокой доступности или аварийного восстановления.
- **Предлагают многофункциональные API-интерфейсы для различных шаблонов обмена** сообщениями, чтобы отправить сообщения определенному соединению, всем соединениям или подмножеству подключений, принадлежащих определенному пользователю, или которые были помещены в произвольную группу.
