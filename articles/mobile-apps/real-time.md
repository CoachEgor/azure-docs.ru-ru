# <a name="build-real-time-applications-with-azure-signalr-service"></a>Создание приложений в режиме реального времени с помощью службы Azure SignalR

[Служба Azure SignalR](https://azure.microsoft.com/services/signalr-service/)позволяет легко добавлять в приложения функции, работающие в режиме реального времени. Эта функция в режиме реального времени позволяет службе отправлять обновления содержимого в подключенные мобильные приложения. В результате приложения обновляются без необходимости опроса сервера или отправки новых HTTP-запросов на обновления.

## <a name="azure-signalr-service-features"></a>Функции службы SignalR Azure
- В предусмотрена широкая поддержка клиентов для широкого спектра клиентов, таких как браузеры веб-и мобильных устройств, классические приложения, серверный процесс, устройства IoT и игровые консоли.
- Позволяет нескольким экземплярам работать вместе для масштабирования до миллионов подключений мобильных приложений. Он также поддерживает несколько глобальных регионов для сегментирования, высокой доступности или аварийного восстановления.
- Предлагает расширенные API для различных шаблонов обмена сообщениями для отправки сообщений определенному подключению, всем соединениям или подмножеству подключений, принадлежащих определенному пользователю или помещенных в произвольную группу.
