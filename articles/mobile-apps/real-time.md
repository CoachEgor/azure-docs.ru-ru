# <a name="build-real-time-applications-with-azure-signalr-service"></a>Создавайте приложения в режиме реального времени с помощью службы Azure SignalR

С [помощью службы Azure SignalR](https://azure.microsoft.com/services/signalr-service/)вы можете легко добавлять функциональность в режиме реального времени в приложения. Эта функциональность в режиме реального времени позволяет службе продвигать обновления содержимого в подключенные мобильные приложения. В результате приложения обновляются без необходимости опроса сервера или отправки новых запросов HTTP на обновления.

## <a name="azure-signalr-service-features"></a>Функции службы Azure SignalR
- Имеет широкую поддержку клиентов для широкого круга клиентов, таких как веб- и мобильные браузеры, настольные приложения, процесс сервера, IoT-устройства и игровые консоли.
- Позволяет нескольким экземплярам работать вместе, чтобы масштабировать сядок швахи до миллионов подключений к мобильным приложениям. Он также поддерживает несколько глобальных регионов для целей осколки, высокой доступности или аварийного восстановления.
- Предлагает богатые AI-интерфейсы для различных шаблонов обмена сообщениями для отправки сообщений в определенное соединение, все соединения или подмножество соединений, которые принадлежат конкретному пользователю или были помещены в произвольной группе.
