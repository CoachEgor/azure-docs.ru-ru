# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Геопространственные API для добавления карт, пространственных аналитик и мобильных решений с Azure Maps

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) предоставляет платформу расположения в облаке и предлагает портфель геопространственных служб, использующих новые данные сопоставления, чтобы обеспечить точный географический контекст для веб-и мобильных приложений. Разработчики, предприятия и независимые поставщики программного обеспечения могут использовать Azure Maps для создания приложений с поддержкой местоположения и решений Интернета вещей, мобильности, логистики и отслеживания активов.

## <a name="azure-maps-features"></a>Azure Maps функции
- **Поиск**: предоставляет сведения о ближайших интересующих точках, отображает расположения на карте и использует геокод (или обратный геокод) адреса для получения соответствующих координат широты и долготы.
- **Трафик**: повышает время поездки и помогает избежать пробок.
- **Maps**: предоставляет четкие, простые для чтения карты для интеграции в решение.
- **Маршрутизация**: представляет кратчайшие или самые быстрые маршруты, доступные для нескольких назначений в определенное время. Маршрутизация также предоставляет специализированные маршруты и факты для пошаговых средств, велосипедистов и коммерческих транспортных средств.
- Часовые **пояса**. позволяет легко узнать, какое время находится в любом месте мира.
- **Пространственные операции**: Расширьте логику аналитики расположения.
- **Географическое расположение**: находит страну IP-адреса.
- **Служба данных**: отправляет и хранит геопространственные данные для использования с пространственными операциями или композицией изображений.
- **Мобильность**: обеспечивает аналитику расположения в реальном времени для ближайших общедоступных транзитных служб. Сведения о прекращении, маршруте и оценке времени поездки включены.
