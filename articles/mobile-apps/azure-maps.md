# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Геопространственные API для добавления карт, пространственной аналитики и решений для мобильности с помощью Azure Maps

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) предоставляет платформу определения местоположения в облаке и предлагает портфолио геопространственных служб, которые используют свежие картографические данные для обеспечения точного географического контекста для веб- и мобильных приложений. Разработчики, предприятия и ISV могут использовать Azure Maps для создания приложений для отслеживания местоположения и решений IoT, мобильности, логистики и отслеживания активов.

## <a name="azure-maps-features"></a>Особенности Карты Лазурного имеет
- **Поиск**: Предоставляет подробную информацию о близлежащих точках интереса, отображает местоположения на карте, и использует геокод (или обратный геокод) адреса, чтобы получить соответствующие координаты широты и долготы.
- **Трафик**: Улучшает ваше время в пути и помогает вам избежать тупика.
- **Карты**: Обеспечивает четкие, легко читаемые карты для интеграции в ваше решение.
- **Маршрут:** Представляет самые короткие или быстрые маршруты, доступные для нескольких направлений в определенное время. Маршрут также предоставляет специализированные маршруты и факты для пешеходов, велосипедистов и коммерческих транспортных средств.
- **Часовые пояса**: Позволяет легко увидеть, какое время это в любой точке мира.
- **Пространственные операции**: Улучшение вашего сведения о местоположении.
- **Геолокация**: Находит страну IP-адреса.
- **Служба данных**: Загружает и хранит геопространственные данные для использования с пространственными операциями или композицией изображений.
- **Мобильность**: Обеспечивает в режиме реального времени информацию о местоположении на близлежащих услуг общественного транспорта. Включены остановки, информация о маршруте и оценка времени в пути.
