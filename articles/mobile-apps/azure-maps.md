# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Геопространственные API для добавления карт, пространственных аналитик и мобильных решений с Azure Maps

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) предоставляет платформу расположения в облаке Azure, предлагает портфель геопространственных служб, использующих новые данные сопоставления, чтобы обеспечить точный географический контекст для веб-и мобильных приложений. Она позволяет разработчикам, предприятиям и поставщикам программного обеспечения создавать приложения с поддержкой местоположения, а также решения Интернета вещей, мобильности, логистики и отслеживания ресурсов.

## <a name="azure-maps-features"></a>Azure Maps функции
- **Поиск** содержит сведения о ближайших интересующих вас точках (достопримечательности), отрисовывает местоположения на карте и геокодированные (или обратные геокоды) адреса, чтобы получить соответствующие координаты широты и долготы.
- Улучшите время поездки и не пробок с помощью **трафика**.
- Интегрируйте четкие, удобные для чтения **карты** в свое решение.
- **Маршрутизация** для предоставления доступа к самым коротким или самым быстрым маршрутам по нескольким назначениям за один раз — или предоставление специализированных маршрутов и фактов для пошаговых средств, велосипедистов и коммерческих транспортных средств.
- Часовые **пояса** позволяют пользователям легко узнать, какое время находится в любом месте мира.
- **Пространственные операции** для улучшения логики аналитики расположения.
- **Географическое положение** для поиска страны или региона IP-адреса.
- **Служба данных** для передачи и хранения геопространственных данных для использования с пространственными операциями или композицией изображений.
- **Мобильность** обеспечивает аналитику расположения в реальном времени для ближайших общедоступных транзитных служб, включая остановки, сведения о маршрутах и оценки времени поездок.
