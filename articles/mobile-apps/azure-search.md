# <a name="ai-powered-cloud-service-with-azure-search"></a>Облачная служба на основе искусственного интеллекта с поиском Azure
[Поиск Azure](https://azure.microsoft.com/services/search/) — это облачное решение типа "Поиск как услуга", которое предоставляет разработчикам API-интерфейсы и средства для добавления расширенных возможностей поиска по закрытому, разнородному содержимому в веб-, мобильных и корпоративных приложениях. Пользовательский код вызывает прием данных (индексирование) для создания и загрузки индекса. С другой стороны, код приложения выдает запросы и обрабатывает ответы. Возможности поиска определяются в клиенте на основе функций Поиска Azure. Эти функции выполняют запросы по сохраненному индексу, который вы можете создавать, администрировать и хранить в своей службе.

## <a name="azure-search-features"></a>Функции поиска Azure
- **Текстовый поиск в свободной форме** с полнотекстовым поиском, синтаксисом простых запросов и синтаксисом запросов Lucene.
- **Простые профили оценки** для моделирования релевантности как функции значений в самих документах.
- **Географический поиск** позволяет пользователям просматривать данные на основе сходства результатов поиска с физическим расположением.
- **Пользовательские функции** с автозаполнением, предложениями поиска, сортировкой и разбиением на страницы.
- Для извлечения текстовых данных из необработанного содержимого можно применить в конвейере индексирования **Поиск** по тексту и анализ текста.
- **Хранилище знаний** для сохранения обогащений, созданных во время индексирования.
- Подключение к базе данных SQL Azure, Azure Cosmos DB или хранилищу BLOB-объектов Azure в качестве **источников данных** для извлечения содержимого с возможностью поиска в первичных хранилищах данных.
- С помощью **мастера импорта данных** можно настроить индексаторы, **конструктор индексов** для создания индекса, а также **Обозреватель поиска** для проверки запросов и уточнения профилей оценки.

Найдите подробное описание [функций](/azure/search/search-what-is-azure-search#feature-descriptions)поиска Azure.
