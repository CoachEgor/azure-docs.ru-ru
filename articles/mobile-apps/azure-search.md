# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>Облачный сервис с помощью ИИ с Azure Cognitive Search
[Azure Cognitive Search](https://azure.microsoft.com/services/search/) — это облачное решение для поиска как службы, которое предоставляет разработчикам ABI и инструменты для добавления богатого поиска по сравнению с частным, неоднородным контентом в веб-, мобильных и корпоративных приложениях. Пользовательский код вызывает прием данных (индексирование) для создания и загрузки индекса. С другой стороны, код приложения выдает запросы и обрабатывает ответы. Опыт поиска определяется в клиенте с помощью функциональности из Azure Cognitive Search, с выполнением запроса по упорному индексу, который вы создаете, владеете и храните в службе.

## <a name="azure-cognitive-search-features"></a>Функции когнитивного поиска Azure
- **Поиск текста Freeform**: Обеспечивает полный поиск текста, простой синтаксис запроса и синтаксис запроса Lucene.
- **Простые профили скоринга**: Актуальность модели как функция значений в самих документах.
- **Geosearch**: Исследует данные, основанные на близости результата поиска к физическому местоположению.
- **Функции взаимодействия пользователей:** Предоставление автозаполнения, предложения по поиску, сортировка и прогулы.
- **Когнитивный поиск**: может быть применен к конвейеру индексирования для извлечения текстовой информации из исходного содержимого для анализа изображений и текста.
- **Хранилище знаний**: Сохраняет обогащения, созданные во время индексации.
- **Источники данных:** Извлекайте содержимое поиска в первичных хранилищах данных при подключении к базе данных Azure S'L, DB Azure Cosmos и хранению Azure Blob.
- **Мастер данных импорта**: Настраивает индексы. 
- **Дизайнер индекса**: Выдерживает индекс. 
- **Поиск исследователь:** Тесты запросов и уточняет скоринг профилей.

Найдите подробное описание [функций](/azure/search/search-what-is-azure-search#feature-descriptions)когнитивного поиска Azure.
