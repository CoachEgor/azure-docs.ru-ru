# <a name="add-authentication-and-manage-user-identities-in-your-mobile-apps"></a>Добавление аутентификации и управление идентификаторами пользователей в мобильных приложениях

Представление о пользователе и его поведении в приложении позволяет разработчикам лучше привлекать пользователей, создавая для них индивидуальные интерфейсы. Независимо от того, являетесь ли вы разработчиком приложений, создающим приложение для совместной работы для пользователей внутри организации, или создаете следующую платформу социальной сети, вам нужен способ проверки подлинности пользователей и управления идентификаторами пользователей. Служба управления идентификационным удостоверением является одной из наиболее важных особенностей мобильного бэк-энда службы.

Используйте следующие службы для обеспечения аутентификации пользователей в мобильных приложениях.

## <a name="visual-studio-app-center"></a>Центр приложений Visual Studio
[App Center Auth](/appcenter/auth/) — это облачный сервис управления идентификацией, который разработчики могут использовать для проверки подлинности пользователей и управления идентификаторами пользователей. App Center Auth также интегрируется с другими частями Visual Studio App Center. Разработчики могут использовать идентификацию пользователя для [просмотра данных пользователей](/appcenter/data/index) в других службах и даже [отправлять push-уведомления пользователям вместо отдельных устройств.](/appcenter/push/push-to-user#setting-user-identity) 

**Основные функции**
- Работает на Azure Active Directory B2C (Azure AD B2C). 
    - Корпоративный класс.
    - Высокая доступность.
    - Безопасный и глобальный сервис.
- Принесите свою собственную личность и возможность использовать другие популярные поставщики идентификации и управления доступом, такие как Auth0 и Firebase.
- Поддержка активного каталога Azure.
    - Подключите существующих арендаторов Azure AD. 
    - Включить аутентификации в отношении корпоративного домена.
    - Управление доступом к конфиденциальным данным.
- Простой пользовательский опыт и магический опыт SDK, обернув microsoft Аутентификация библиотека с Visual Studio App Center SDK.
- Поддержка платформдлядля iOS, Android, Xamarin и React Native.

## <a name="azure-active-directory-b2c"></a>Azure Active Directory B2C
[Azure AD B2C](https://azure.microsoft.com/services/active-directory-b2c/) — это служба управления идентификацией для потребителя (B2C), которую разработчики могут использовать для проверки подлинности своих клиентов. Эта услуга с белыми этикетками позволяет разработчикам настраивать и контролировать, как пользователи безопасно взаимодействуют со своими веб-, настольными, мобильными или одностраничными приложениями. С помощью Azure AD B2C пользователи могут регистрировать, изменять свои профили, а также входить в них и сбрасывать пароли. Azure AD B2C реализует формы протоколов OpenID Connect и OAuth 2.0. 

**Основные функции**
- Безопасная аутентификативная проверка клиентов с помощью предпочтительного поставщика идентификационных данных.
- Управление идентификацией и доступом клиентов.
- Получите поддержку социальных сетей, таких как Facebook, GitHub, Google, LinkedIn, Twitter, WeChat и Weibo.
- Подключайтесь к учетным записям пользователей с помощью отраслевых стандартных протоколов, таких как OpenID Connect или SAML, чтобы сделать управление идентификацией возможным на различных платформах.
- Обеспечить фирменную регистрацию и вписаться в систему.
- Легко интегрироваться с базами данных CRM, инструментами маркетинговой аналитики и системами проверки счетов.
- Захват данных о всхожести, предпочтениях и конверсии для клиентов.

## <a name="azure-active-directory"></a>Azure Active Directory
[Azure Active Directory](https://azure.microsoft.com/services/active-directory/) — это служба управления идентификаторами и доступом Microsoft, которая помогает вашим сотрудникам войти в систему и получить доступ к:
- Внешние ресурсы, такие как Microsoft Office 365, портал Azure и тысячи других программных приложений в качестве сервисных (SaaS).
- во внутренних ресурсах, таких как приложения в корпоративной сети и интрасети, а также в любых облачных приложениях, разработанных вашей организацией.

**Основные функции**
- Бесшовный, очень безопасный доступ, подключив пользователей к приложениям, которые им нужны.
- Всеобъемлющая защита идентификационных данных и повышенная безопасность идентификационных данных и доступа на основе контекста пользователя, местоположения, устройства, данных и приложений.
- Тысячи предварительно интегрированных приложений как для коммерческих, так и для пользовательских приложений, таких как Office 365, Salesforce.com и Box.
- Возможность управления доступом в масштабе.
