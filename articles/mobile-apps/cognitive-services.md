# <a name="cognitive-services-helps-to-solve-business-problems-by-using-ai"></a>Cognitive Services помогает решать бизнес-задачи с помощью искусственного интеллекта
[Azure Cognitive Services](https://azure.microsoft.com/services/cognitive-services/) предоставляет мощные службы для дешифрования приложений с помощью интеллектуальных алгоритмов, позволяющих просматривать, слышать, говорить, понимать и интерпретировать потребности пользователей для преобразования бизнеса с помощью искусственного интеллекта (ии). Вы можете сделать свои мобильные приложения интеллектуальными, привлекательными и контекстно важными для пользователей. Разработчики могут использовать API, пакеты SDK и службы, чтобы добавить AI в свои предложения и помочь создавать интеллектуальные приложения без прямого искусственного интеллекта или навыков обработки и анализа данных.

## <a name="cognitive-services-capabilities"></a>Cognitive Services возможности
- **зрение**;
    - Добавление тегов к изображениям, эскизы
    - Распознавание текста (распознавание рукописного ввода)
    - Обнаружение лиц
    - Распознавание эмоций
    - Аналитические сведения о видеоматериалах
    - Контроль изображений и видео
    - Настроенное распознавание изображений

- **речь**;
    - Преобразование речи в текст (транскрипция речи)
    - Персонализированная транскрипция речи (сложные слова, среда с помехами)
    - Преобразование текста в речь
    - ИДЕНТИФИКАТОР и проверка подлинности докладчика
    - Перевод речи в режиме реального времени

- **Язык**
    - Общие сведения об контекстном языке, настраиваемый анализ намерений
    - Анализ тональностиности, определение ключевых фраз
    - Преобразование текста до 60 языков
    - Модерация текста
    - Проверка орфографии

- **Решение** 
    - Модератор содержимого
    - Персонализированная Доставка содержимого
    - Детектор аномалий

- **Поиск**
    - Автоматические варианты поиска 
    - Приращение сведений о сущности
    - Исчерпывающие Новости, изображения, видео, Интернет и поиск в локальной бизнес-сущности
    - Специализированные и настроенные возможности поиска

- Поддержка контейнеров для Cognitive Services
