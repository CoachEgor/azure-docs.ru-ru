# <a name="cognitive-services-helps-to-solve-business-problems-by-using-ai"></a>Cognitive Services помогает решать бизнес-проблемы с помощью ИИ
[Azure Cognitive Services](https://azure.microsoft.com/services/cognitive-services/) предоставляет мощные сервисы для наполнить ваши приложения интеллектуальными алгоритмами для просмотра, просмотра, ознакомитель, понимания и интерпретации потребностей пользователя в преобразовании вашего бизнеса с помощью искусственного интеллекта (ИИ). Вы можете сделать ваши мобильные приложения интеллектуальными, интересными и контекстуально актуальными для пользователей. Разработчики могут использовать ABI, SDK и сервисы, чтобы добавлять ИИ в свои предложения и помогать создавать интеллектуальные приложения, не имея прямых навыков ИИ или науки о данных.

## <a name="cognitive-services-capabilities"></a>Возможности когнитивных служб
- **Зрение**
    - Пометка изображений, эскизы
    - OCR (распознавание почерка)
    - Определение лиц
    - Распознавание эмоций
    - Аналитические сведения о видеоматериалах
    - Модерация изображений и видео
    - Индивидуальное распознавание изображений

- **Речь**
    - Речь к тексту (расшифровка речи)
    - Индивидуальная транскрипция речи (сложные слова, шумная обстановка)
    - Преобразование текста в речь
    - Идентификатор динамиков и аутентификация
    - Перевод речи в реальном времени

- **Язык**
    - Контекстное понимание языка, индивидуальный анализ намерений
    - Анализ настроений, обнаружение ключевых фраз
    - Текстовый перевод до 60 языков
    - Модерация текста
    - Проверка орфографии

- **Решение** 
    - Content Moderator
    - Персонализированная доставка контента
    - Детектор аномалий

- **Поиск**
    - Предложения по автоматическому поиску 
    - Расширение информации о сущности
    - Всеобъемлющие новости, изображения, видео, веб-сайты и местные бизнес-объекты поиска
    - Индивидуальный и индивидуальный поиск

- Поддержка контейнеров для когнитивных услуг
