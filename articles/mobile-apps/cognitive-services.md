# <a name="azure-cognitive-services-to-solve-business-problems-using-artificial-intelligence"></a>Cognitive Services Azure для решения бизнес-задач с помощью искусственного интеллекта
[Azure Cognitive Services](https://azure.microsoft.com/services/cognitive-services/) предоставляет мощные службы для дешифрования приложений с помощью интеллектуальных алгоритмов, позволяющих просматривать, слышать, говорить, понимать и интерпретировать потребности пользователей для преобразования бизнеса с помощью искусственного интеллекта. Вы можете сделать свои мобильные приложения интеллектуальными, привлекательными и контекстно важными для пользователей. Интерфейсы API, пакеты SDK и службы позволяют разработчикам добавлять AI в свои предложения и помогать создавать интеллектуальные приложения без прямого искусственного интеллекта или навыков обработки и анализа данных.

## <a name="cognitive-services-capabilities"></a>Cognitive Services возможности
- **зрение**;
    - Добавление тегов к изображениям, эскизы
    - Распознавание текста (распознавание рукописного ввода)
    - Определение лиц
    - Обнаружение распознавания эмоций
    - Аналитические сведения о видеоматериалах
    - Контроль изображений и видео
    - Настроенное распознавание изображений

- **речь**;
    - Преобразование речи в текст (транскрипция речи)
    - Персонализированная транскрипция речи (сложные слова, среда с помехами)
    - Преобразование текста в речь
    - ИДЕНТИФИКАТОР и проверка подлинности докладчика
    - Перевод речи в режиме реального времени

- **Язык**
    - Контекстный анализ настраиваемых намерений в зависимости от языка
    - Анализ тональностиности, определение ключевых фраз
    - Преобразование текста до 60 языков
    - Модерация текста
    - Проверка орфографии

- **Принятие** 
    - Модератор содержимого
    - Персонализированная Доставка содержимого
    - Детектор аномалий

- **Поиск**
    - Автоматические варианты поиска 
    - Приращение сведений о сущности
    - Исчерпывающие Новости, изображения, видео, Интернет и поиск в локальной бизнес-сущности
    - Специализированные и настроенные возможности поиска

- Поддержка **контейнеров** для Cognitive Services
