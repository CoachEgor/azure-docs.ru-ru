# <a name="perform-automated-testing-on-thousands-of-mobile-devices-simultaneously"></a>Выполните автоматизированное тестирование на тысячах мобильных устройств одновременно
Тестирование может быть сложным и все более сложным из-за фрагментации устройства, сложности приложений, быстрых циклов выпуска, коротких сеансов и высоких ожиданий пользователей. Вам необходимо регулярно следить за тем, чтобы предоставлять тщательно проверенные приложения, обеспечивающие большой опыт работы с клиентами. Тестирование не является легкой задачей, и это не очень хорошее использование времени разработчика для ручного тестирования каждого релиза.

Проблемы тестирования могут быть решены эффективно. Вы можете тестировать мобильные приложения на широком диапазоне устройств с помощью автоматизированного тестирования. Можно использовать облачные сервисы для ускорения выпуска мобильных приложений и преодоления проблем тестирования, которые добавляют узкие места в разработку и развертывание приложений.

## <a name="importance-of-automated-testing"></a>Важность автоматизированного тестирования
- Увеличьте охват теста, ориентируясь на различные форм-факторы, типы подключения и версии ОС.
- Экономьте время, деньги и ресурсы, тестируя на разных устройствах и охват оС.
- Улучшайте точность, избегая повторяющихся ручных тестов, и у нас будет больше времени для создания новых автоматизированных программных тестов и работы со сложными функциями.
- Непрерывно проверяйте производительность приложения, обнаруживайте и исправляйте проблемы до того, как пользователи наткнутся на них и быстро отпустите их.
- Доставка высококачественных приложений, гарантируя, что ваши клиенты на широком спектре устройств получить лучший опыт.

## <a name="visual-studio-app-center"></a>Центр приложений Visual Studio
[App Center Test](/appcenter/test-cloud/) — это облачный сервис, который можно использовать для проведения автоматизированных тестов интерфейса на тысячах реальных устройств в облаке android и iOS для нативных и гибридных мобильных приложений. С более чем 2000 устройств, вы можете написать тестовые скрипты, которые могут быть выполнены автоматически, чтобы сделать тестирование легко.

**Основные функции**
   - Тесты, которые работают на реальных устройствах, которые не коренятся или не работают в тюрьме, чтобы обеспечить наилучшее приближение сред пользователей.
   - Скриншоты автоматически фиксируются и прикрепляются к каждому тестовому этапу. Они могут быть просмотрены бок о бок для всех целевых устройств в тестовом запуске.
   - Системы устройств для всех целевых устройств в тестовом запуске и стекследы для неудачных этапов тестирования.
   - Параллельно тест выполняется на нескольких устройствах одновременно, чтобы ускорить время к результатам.
   - Популярная поддержка тестовых рамок, таких как Appium (Java with JUnit) для Android и iOS, Espresso (Java) для Android, Calabash для Android и iOS, Xamarin.UITest для Android и iOS, и XCUITest для iOS.
   - Поддержка платформдлядля iOS, Android Xamarin и React Native.
