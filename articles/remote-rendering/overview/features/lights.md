# <a name="lights"></a>Свет

По умолчанию удаленно отрисованные объекты горят с помощью [неба.](sky.md) Для большинства приложений этого уже достаточно, но вы можете добавить дополнительные источники света на сцену.

> [!IMPORTANT]
> Только [материалы PBR](pbr-materials.md) подвержены воздействию источников света. [Цветовые материалы](color-materials.md) всегда кажутся полностью яркими.

> [!NOTE]
> Тени отлива в настоящее время не поддерживаются. Удаленное рендерирование Azure оптимизировано для визуализации огромного количества геометрии, при необходимости используя несколько графических процессоров. Традиционные подходы к теневому отливу в таких сценариях не очень хорошо работают.

## <a name="common-light-component-properties"></a>Общие свойства световых компонентов

Все типы света происходят `LightComponent` из абстрактного базового класса и разделяют эти свойства:

* **Цвет:** Цвет света в [гамма-пространстве.](https://en.wikipedia.org/wiki/SRGB) Альфа игнорируется.

* **Интенсивность:** Яркость света. Для точечных и топочных огней интенсивность также определяет, как далеко светит свет.

## <a name="point-light"></a>Точечный свет

В Azure Remote `PointLightComponent` Rendering можно излучать свет не только из одной точки, но и из небольшой сферы или небольшой трубки, чтобы имитировать более легкие источники света.

### <a name="pointlightcomponent-properties"></a>Свойства PointLightComponent

* **Радиус:** Радиус по умолчанию равен нулю, и в этом случае свет действует как точечный свет. Если радиус больше нуля, он действует как источник сферического света, который изменяет внешний вид зеркальных бликов.

* **Длина:** Если `Length` оба `Radius` они не являются нулевыми, свет действует как трубчатый свет. Это может быть использовано для имитации неоновых труб.

* **ОслаблениеCutoff:** Если оставить (0,0) затухать свет зависит `Intensity`только от его . Тем не менее, вы можете предоставить пользовательские мин / макс расстояния, на которых интенсивность света масштабируется линейно до 0. Эта функция может быть использована для обеспечения меньшего диапазона влияния определенного света.

* **ProjectedCubemap:** Если установить на допустимую [кубическую карту,](../../concepts/textures.md)текстура проецируется на окружающую геометрию света. Цвет кубики модулируется с цветом света.

## <a name="spot-light"></a>Точечный свет

Он `SpotLightComponent` похож на `PointLightComponent` свет, ограниченный формой конуса. Ориентация конуса определяется *отрицательной z-оси сущности владельца.*

### <a name="spotlightcomponent-properties"></a>Свойства SpotLightComponent

* **Радиус:** То же, `PointLightComponent`что и для .

* **SpotAngleDeg:** Этот интервал определяет внутренний и внешний угол конуса, измеренный по степени. Все внутри внутреннего угла освещается полной яркостью. Падение применяется к внешнему углу, который генерирует полутени, как эффект.

* **FalloffЭкспонент:** Определяет, насколько резко переходы от падения между внутренним и внешним углом конуса. Более высокое значение приводит к более резкому переходу. По умолчанию 1.0 приводит к линейному переходу.

* **ОслаблениеCutoff:** То же, `PointLightComponent`что и для .

* **Projected2dТекстура:** Если установить на действительную [2D текстуру,](../../concepts/textures.md)изображение проецируется на геометрию, что свет светит на. Цвет текстуры модулируется с цветом света.

## <a name="directional-light"></a>Направленный свет

Имитирует `DirectionalLightComponent` источник света, который находится бесконечно далеко. Свет светит в направлении *отрицательной z-оси сущности владельца.* Позиция организации игнорируется.

Дополнительных свойств нет.

## <a name="performance-considerations"></a>Вопросы производительности

Источники света оказывают значительное влияние на производительность рендеринга. Используйте их осторожно и только в том случае, если этого требует приложение. Любое статическое глобальное состояние освещения, включая статический направленный компонент, может быть достигнуто с [помощью пользовательской текстуры неба](sky.md)без дополнительной стоимости рендеринга.
