# <a name="cut-planes"></a>Вырезать самолеты

*Вырезать плоскости* является визуальной особенностью, которая клипы пикселей на одной стороне виртуальной [плоскости,](../../concepts/meshes.md)выявление внутри сетки .
На рисунке ниже показан эффект. Слева показана оригинальная сетка, справа можно заглянуть внутрь сетки:

![Вырезать плоскости](./media/cutplane-1.png)

## <a name="limitations"></a>Ограничения

* В настоящее время Azure Remote Rendering поддерживает **максимум восемь активных плоскостей разреза.** Вы можете создать больше компонентов плоскости отрезока, но если вы попытаетесь включить более одновременно, он проигнорирует активацию. Сначала отключите другие плоскости, если вы хотите переключиться, какой компонент должен повлиять на сцену.
* Каждая плоскость разреза влияет на все удаленно отрисованные объекты. В настоящее время нет способа исключить конкретные объекты или части сетки.
* Вырезать плоскости являются чисто визуальной особенностью, они не влияют на результат [пространственных запросов.](spatial-queries.md) Если вы хотите, чтобы луч бросили в вырезанной сетке, вы можете настроить отправную точку луча, чтобы быть на разрезе плоскости. Таким образом, луч может поразить только видимые части.

## <a name="performance-considerations"></a>Вопросы производительности

Каждая активная плоскость разреза несет небольшую стоимость во время рендеринга. Отключите или удалите вырезать плоскости, когда они не нужны.

## <a name="cutplanecomponent"></a>CutPlaneКомпонент

Вы добавляете плоскость разреза к месту, создавая *CutPlaneComponent.* Местоположение и ориентация самолета определяется [сущностью](../../concepts/entities.md)владельца компонента.

```cs
void CreateCutPlane(AzureSession session, Entity ownerEntity)
{
    CutPlaneComponent cutPlane = (CutPlaneComponent)session.Actions.CreateComponent(ObjectType.CutPlaneComponent, ownerEntity);
    cutPlane.Normal = Axis.X; // normal points along the positive x-axis of the owner object's orientation
    cutPlane.FadeColor = new Color4Ub(255, 0, 0, 128); // fade to 50% red
    cutPlane.FadeLength = 0.05f; // gradient width: 5cm
}
```

### <a name="cutplanecomponent-properties"></a>Свойства CutPlaneComponent

Следующие свойства выставлены на компоненте плоскости разреза:

* **Включено:** Можно временно отключить отрезанные плоскости, отключив компонент. Инвалиды сократить самолеты не несут рендеринга накладных расходов, а также не учитываются в отношении глобального предела вырезать плоскости.

* **Нормальный:** Определяет, какое направление (X,-X,-Y,-Y,-Y,--) используется в качестве обычного направления плоскости. Это направление относительно ориентации сущности владельца. Перемещение и поворот сущности владельца для точного размещения.

* **FadeColor** и **FadeLength:**

  Если альфа-значение *FadeColor* не является нулевым, пиксели, близкие к плоскости разреза, исчезнут в сторону части RGB FadeColor. Сила альфа-канала определяет, будет ли он полностью исчезать к цвету исчезают или только частично. *FadeLength* определяет, на каком расстоянии это увядание будет иметь место.
