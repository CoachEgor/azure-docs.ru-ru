# <a name="cut-planes"></a>Вырезание плоскостей

*Вырезанная плоскость* — это визуальная функция, которая вырезает Пиксели на одной стороне виртуальной плоскости и раскрывает внутреннюю часть [сеток](../../concepts/meshes.md).
Этот результат показан на рисунке ниже. В левой части показана исходная сетка, которая в правой части может выглядеть внутри сетки:

![Вырезать плоскость](./media/cutplane-1.png)

## <a name="limitations"></a>Ограничения

* В течение этого времени Удаленная визуализация Azure поддерживает не более **восьми активных вырезанных плоскостей**. Вы можете создать более вырезанные компоненты плоскости, но если вы попытаетесь включить большее количество одновременно, активация будет пропущена. Сначала отключите другие плоскости, если хотите переключить, какой компонент должен повлиять на сцену.
* Каждая вырезанная плоскость влияет на все удаленно визуализированные объекты. В настоящее время нет способа исключить определенные объекты или части сетки.
* Вырезание плоскостей — это визуальная функция, которая не влияет на результат [пространственных запросов](spatial-queries.md). Если вы хотите выполнить приведение лучей к вырезанной сетке, можно настроить начальную точку луча, чтобы она находилась на плоскости вырезания. Таким образом, луч может попасть только на видимые части.

## <a name="performance-considerations"></a>Вопросы производительности

При подготовке каждой активной плоскости вырезания возникает небольшая стоимость. Отключите или удалите вырезанные плоскости, если они не нужны.

## <a name="cutplanecomponent"></a>кутпланекомпонент

Чтобы добавить в сцену вырезанную плоскость, создайте *кутпланекомпонент*. Расположение и ориентация плоскости определяются [сущностью](../../concepts/entities.md)"владелец" компонента.

```cs
void CreateCutPlane(AzureSession session, Entity ownerEntity)
{
    CutPlaneComponent cutPlane = (CutPlaneComponent)session.Actions.CreateComponent(ObjectType.CutPlaneComponent, ownerEntity);
    cutPlane.Normal = Axis.X; // normal points along the positive x-axis of the owner object's orientation
    cutPlane.FadeColor = new Color4Ub(255, 0, 0, 128); // fade to 50% red
    cutPlane.FadeLength = 0.05f; // gradient width: 5cm
}
```

### <a name="cutplanecomponent-properties"></a>Свойства Кутпланекомпонент

Следующие свойства представлены в компоненте "вырезать плоскость":

* **Включено:** Можно временно отключить вырезание плоскостей, отключив компонент. Отключенные вырезание плоскостей не требуют затрат на подготовку к просмотру, а также не учитываются в глобальном ограничении плоскости.

* **Обычная ситуация:** Указывает, какое направление (+ X,-X, + Y,-Y, + Z,-Z) используется в качестве нормали к плоскости. Это направление относительно ориентации сущности-владельца. Переместите и поверните сущность «владелец» для точного размещения.

* **Фадеколор** и **фаделенгс:**

  Если альфа-значение *фадеколор* не равно нулю, пикселы, близкие к вырезанной плоскости, будут покрываться с частью RGB в фадеколор. Сила альфа-канала определяет, будет ли она полностью перенаправлены к цвету исчезания или только частично. *Фаделенгс* определяет, на какое расстояние будет происходить это выцветание.
