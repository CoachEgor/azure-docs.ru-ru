# <a name="single-sided-rendering"></a>Односторонняя отрисовка

Большинство модулей подготовки отчетов используют применение [обратной передачи](https://en.wikipedia.org/wiki/Back-face_culling) для повышения производительности. Однако, если сетки вырезаются с помощью [вырезания плоскостей](cut-planes.md), пользователи часто просматривает задную сторону треугольников. Если эти треугольники исключаются, результат не выглядит убедительным.

Способ надежного предотвращения этой проблемы заключается в том, чтобы визуализировать треугольники на *двух*сторонах. Как и отсутствие использования использующих обратных сторон, влияет на производительность. по умолчанию Удаленная визуализация Azure переключается только на двустороннюю отрисовку для сеток, пересекающихся с плоскостью вырезания.

Параметр *односторонняя отрисовки* позволяет настроить это поведение.

> [!CAUTION]
> Параметр односторонняя отрисовки является экспериментальной функцией. В будущем он может снова удалиться. Не изменяйте значение по умолчанию, если оно не разрешает критическую проблему в приложении.

## <a name="prerequisites"></a>Предварительные условия

Параметр односторонняя отрисовки влияет только на сетки, [преобразованные](../../how-tos/conversion/configure-model-conversion.md) с `opaqueMaterialDefaultSidedness` параметром в `SingleSided`значение. По умолчанию этот параметр имеет значение `DoubleSided`.

## <a name="single-sided-rendering-setting"></a>Односторонняя Настройка отрисовки

Существует три разных режима:

**Обычная ситуация:** В этом режиме сетки всегда подготавливаются к просмотру по мере их преобразования. Это означает, что сетки, `opaqueMaterialDefaultSidedness` преобразованные `SingleSided` с помощью параметра со значением, всегда будут отображаться с включенным отбором обратных лиц, даже если они пересекаются в виде вырезанной плоскости.

**Динамикдаублесидинг:** В этом режиме, когда вырезанная плоскость пересекает сетку, она автоматически переключается на двухстороннюю отрисовку. Этот режим используется по умолчанию.

**Алвайсдаублесидед:** Заставляет отрисовку односторонних геометрической визуализации выполняться в любое время. Этот режим обычно предоставляется, поэтому можно легко сравнить влияние на производительность между односторонней и двусторонней отрисовкой.

Изменение параметров односторонней отрисовки можно выполнить следующим образом.

``` cs
void ChangeSingleSidedRendering(AzureSession session)
{
    SingleSidedSettings settings = session.Actions.SingleSidedSettings;

    // Single-sided geometry is rendered as is
    settings.Mode = SingleSidedMode.Normal;

    // Single-sided geometry is always rendered double-sided
    settings.Mode = SingleSidedMode.AlwaysDoubleSided;
}
```
