# <a name="color-materials"></a>Цветные материалы

*Цветовые материалы* являются одним из [поддерживаемых типов материалов](../../concepts/materials.md) в удаленном рендеринге Azure. Они используются для [сетки,](../../concepts/meshes.md) которые не должны получать какой-либо освещения, а быть полной яркости во все времена. Это может быть в случае "светящихся" материалов, таких как приборные панели автомобиля, лампочки, или для данных, которые уже включает статическое освещение, такие как модели, полученные с помощью [фотограмметрии.](https://en.wikipedia.org/wiki/Photogrammetry)

Цветовые материалы являются более эффективными для визуализации, чем [материалы PBR](pbr-materials.md) из-за их более простой модели затенения. Они также поддерживают различные режимы прозрачности.

## <a name="common-material-properties"></a>Общие свойства материала

Эти свойства являются общими для всех материалов:

* **albedoColor:** Этот цвет умножается на другие цвета, такие как *albedoMap* или *цвета вершины.* Если *прозрачность* включена на материале, альфа-канал используется для `1` регулировки непрозрачности, с смыслом полностью непрозрачным и `0` смысл полностью прозрачным. По умолчанию белый.

  > [!NOTE]
  > Поскольку цветные материалы не отражают окружающую среду, полностью прозрачный цветовой материал становится невидимым. Это отличается для [материалов PBR](pbr-materials.md).

* **albedoMap:** [2D текстура](../../concepts/textures.md) для значений альбедо на пиксель.

* **alphaClipEnabled** и **alphaClipThreshold:** Если *alphaClipEnabled* истинен, все пиксели, где альфа-значение альбедо ниже, чем *alphaClipThreshold,* не будут нарисованы. Альфа-отсечение можно использовать даже без обеспечения прозрачности и гораздо быстрее в визуализации. Альфа обрезанные материалы по-прежнему медленнее, чтобы сделать, чем полностью непрозрачные материалы, однако. По умолчанию альфа-отсечение отключено.

* **textureCoordinateScale** и **textureCoordinateOffset:** шкала умножается на координаты УФ текстуры, к ней добавляется смещение. Может быть использован для растяжения и сдвиг текстуры. Шкала по умолчанию (1, 1) и смещение (0, 0).

* **использованиеVertexColor:** Если сетка содержит цвета вершины и эта опция включена, цвета вершины сетки умножаются на *albedoColor* и *albedoMap.* По умолчанию цвета вершины отключены.

* **являетсяDoubleSided:** Если двусторонность установлена на истину, треугольники с этим материалом отображаются, даже если камера смотрит на их задние грани. По умолчанию эта опция отключена. Смотрите также [одностороннюю визуализацию.](single-sided-rendering.md)

## <a name="color-material-properties"></a>Свойства цветового материала

Следующие свойства специфичны для цветных материалов:

* **vertexMix:** Это значение `0` `1` между и определяет, насколько сильно цвет вершины в [сетке](../../concepts/meshes.md) способствует окончательному цвету. При значении по умолчанию 1 цвет вершины умножается на цвет альбедо полностью. При значении 0 цвета вершины полностью игнорируются.

* **transparencyMode:** В отличие от [материалов PBR,](pbr-materials.md)цветные материалы различают различные режимы прозрачности:

  1. **Непрозрачный:** Режим по умолчанию отражает прозрачность. Альфа-отсечение по-прежнему возможно, хотя, и должно быть предпочтительным, если достаточно.
  
  1. **AlphaBlended:** Этот режим аналогичен режиму прозрачности для материалов PBR. Он должен быть использован для просвечиваемых материалов, таких как стекло.

  1. **Добавка:** Этот режим является самым простым и эффективным режимом прозрачности. Вклад материала добавляется к отрисованной изображению. Этот режим можно использовать для имитации светящихся (но все еще прозрачных) объектов, таких как маркеры, используемые для выделения важных объектов.
