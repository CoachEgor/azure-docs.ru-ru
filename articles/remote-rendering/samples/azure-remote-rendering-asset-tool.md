# <a name="azure-remote-rendering-asset-tool-arrt"></a>Средство для работы с объектами службы "Удаленная отрисовка Azure" (ARRT)

Средство для работы с объектами службы "Удаленная отрисовка Azure" (АРРТ) — это классическое приложение с открытым исходным кодом, разработанное в C++ /Qt.

![ARRT](./media/azure-remote-rendering-asset-tool.png "Снимок экрана ARRT")

## <a name="features"></a>Компоненты

Приложение можно использовать для следующих действий.

* Загрузка трехмерной модели
* Управление преобразованием модели
* Создание сеанса удаленной подготовки к просмотру и управление им
* Загрузка трехмерной модели
* Предварительный просмотр трехмерной модели
* Изменение материалов модели

Вы также можете использовать его в качестве примера с открытым исходным кодом, чтобы узнать, как реализовать внешний интерфейс для пакета SDK для удаленной подготовки к просмотру Azure C++, используя клиентскую библиотеку службы хранилища Azure для управления преобразованием трехмерной модели.
