# <a name="unity-showcase-application"></a>Приложение для демонстрации Unity

Приложение для демонстрации Unity — это пример приложения на основе Unity, в котором демонстрируются функции службы "Удаленная отрисовка Azure". Приложение может выполняться как в редакторе Unity, так и на устройстве Hololens 2.

![Приложение для демонстрации](./media/showcase-app.png "Приложение для демонстрации")

## <a name="source-repository"></a>Исходный репозиторий

Исходный код для этого приложения вместе с инструкциями по сборке и развертыванию можно найти в репозитории [с примерами для службы "Удаленная отрисовка Azure"](https://github.com/Azure/azure-remote-rendering/tree/master/Unity/Showcase).
