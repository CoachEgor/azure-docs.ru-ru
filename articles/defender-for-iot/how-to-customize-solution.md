# <a name="customize-your-defender-for-iot-solution"></a>Настройка защитника для решения IoT

В этом руководство вы узнаете, как настроить различные параметры в защитнике для Интернета вещей.

> [!div class="checklist"]
> * Настройка рекомендаций по решению
> * Изменить параметры

## <a name="configure-solution-recommendations"></a>Настройка рекомендаций по решению

Чтобы настроить в защитнике рекомендации по решению IoT, выполните следующие действия.

1. Откройте **Центр Интернета вещей** на портале Azure.
1. Выберите и откройте **Параметры** в разделе **Безопасность** в меню слева.
1. В разделе **Параметры**выберите **Рекомендуемые конфигурации**.
1. Отключите или включите рекомендации по решению, относящиеся к вашей организации и рабочим процессам.

## <a name="change-settings"></a>Изменить параметры

Управление параметром защитника для IoT:

Чтобы настроить параметры защитника для Интернета вещей, выполните следующие действия.

1. Откройте **Центр Интернета вещей** на портале Azure.
1. В меню слева в разделе **Безопасность**выберите и откройте **Параметры**.
1. В разделе * * Сбор данных * * * выберите параметр решения, который вы хотите изменить.
1. Не забудьте всегда щелкать **сохранить** в верхней части любого экрана настройки, чтобы сохранить изменения параметров.
