# <a name="deploy-the-management-console"></a>Развертывание консоли управления
В этой статье описывается развертывание консоли управления Azure Defender для локальной сети IoT.

## <a name="the-on-premises-management-console"></a>Локальная консоль управления

Локальная консоль управления предоставляет объединенное представление всех сетевых ресурсов и предоставляет представление в реальном времени ключевых показателей и оповещений о рисках для всех ваших средств. Тесная интеграция с рабочими процессами SOC и запуск книг позволяет легко определять приоритеты действий по устранению рисков и взаимной корреляции угроз.

- Целостность — сокращение сложности с помощью единой унифицированной платформы для управления активами, управления рисками и уязвимостью, а также мониторинга угроз с реагированием на инциденты.

- Агрегирование и корреляция — отображение, агрегирование и анализ данных и оповещений, собранных со всех сайтов.

- Управление всеми датчиками — Настройка и мониторинг всех датчиков из одного места.

   ![Просмотр Azure Defender для управления сайтом IoT](media/updates/image2.png)

## <a name="deploy-the-on-premises-management-console-appliance"></a>Развертывание локального консольного устройства управления

Этот процесс требует приобретения собственного оборудования и установки программного обеспечения. Решение выполняется на сертифицированных устройствах. Сведения о приобретении сертифицированного устройства см. в [руководстве по спецификациям для оборудования Azure для Интернета вещей](https://aka.ms/AzureDefenderforIoTBareMetalAppliance) .

Дополнительные сведения о скачивании ISO-образа и установке программного обеспечения датчика см. в [руководстве по установке защитника Azure для IOT](https://aka.ms/AzureDefenderforIoTInstallSensorISO) .

**Чтобы развернуть локальную консоль управления, выполните следующие действия.**

1. Перейдите к Microsoft Azure Защитнику для Интернета вещей.

2. Выберите **локальную консоль управления**.

   ![Представление локальной консоли управления в защитнике Azure для Интернета вещей](media/updates/image15.png)

3. Выберите версию 3,1 в меню **Выбор версии** .

4. Выберите **скачать** и сохраните файл.

5. После установки программного обеспечения выполните задачи настройки сети. Дополнительные сведения см. в [руководстве по настройке сети "защитник Azure для Интернета вещей"](https://aka.ms/AzureDefenderForIoTNetworkSetup) .
