# <a name="quickstart-prepare-your-network-for-azure-defender-for-iot-deployment"></a>Краткое руководство. Подготовка сети к развертыванию Azure Defender для Интернета вещей

## <a name="choose-your-hardware"></a>Выбор оборудования
Azure Defender для Интернета вещей поддерживает различные сертифицированные устройства, в том числе:
- Сертифицированные предварительно настроенные устройства, на которых уже установлено программное обеспечение
- Ненастроенные сертифицированные устройства, на которых можно загрузить и установить программное обеспечение датчиков
- Спецификации локальных устройств консоли управления. Локальная консоль управления недоступна в качестве предварительно настроенного устройства. 

Дополнительные сведения см. в [руководстве по спецификациям оборудования](https://aka.ms/AzureDefenderforIoTBareMetalAppliance).


## <a name="install-components"></a>Установка компонентов
После получения компонентов оборудования и загрузки пакета программного обеспечения с портала Azure Defender для Интернета вещей можно установить программное обеспечение компонентов. Дополнительные сведения см. в [руководстве по установке](https://aka.ms/AzureDefenderforIoTInstallSensorISO).

## <a name="onboard-sensors-to-the-azure-defender-for-iot-portal"></a>Подключение датчиков на портале Azure Defender для Интернета вещей 
 В этой статье содержатся общие сведения о процессах развертывания, адаптации и обновления датчиков Azure Defender для Интернета вещей и локальной консоли управления. Этот процесс выполняется с помощью следующих инструкций на странице Azure Defender для Интернета вещей и последующих страницах.

![Снимок экрана: страница приветствия Azure](media/updates/image4.png)

Чтобы открыть страницу приветствия, сделайте следующее:

1. Войдите в учетную запись Azure и выполните поиск по запросу "Azure Defender для Интернета вещей".

1. **Для развертывания датчика необходимо следующее.** Приобрести предварительно настроенное оборудование в Arrow или купить собственное устройство и установить на нем программное обеспечение датчика.
1. **Для развертывания локальной консоли управления** следует купить собственное устройство и установить на нем программное обеспечение локальной консоли управления. Дополнительные сведения см. в записи блога




## <a name="network-setup"></a>Настройка сети
После установки системных компонентов Defender для Интернета вещей можно настроить сетевое оборудование для оптимального мониторинга сети. Дополнительные сведения см. в [руководстве по настройке сети](https://aka.ms/AzureDefenderForIoTNetworkSetup).
