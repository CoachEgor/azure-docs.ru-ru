# <a name="introducing-azure-defender-for-iot-preview"></a>Введение в предварительную версию защитника Azure для IoT

> [!IMPORTANT]
> Сейчас служба "защитник Azure для Интернета вещей" находится в общедоступной предварительной версии.
> Эта предварительная версия предоставляется без соглашения об уровне обслуживания и не рекомендована для использования рабочей среде. Некоторые функции могут не поддерживаться или их возможности могут быть ограничены. Дополнительные сведения см. в статье [Дополнительные условия использования предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

Поскольку предприятия реализуют цифровое преобразование для повышения эффективности и производительности, доски и группы управления все еще интересуются ответственностью и финансовым рискам, полученным при развертывании огромного количества неуправляемых "Интернет вещей" (IoT) и устройств с операционными технологиями.
Субъекты угроз, предназначенные для этой развернутой уязвимой зоны, могут привести к существенному влиянию на предприятия, включая безопасность и инциденты окружающей среды, дорогостоящие простои в работе и кражу

Устаревшие устройства IoT и a не поддерживают агенты и часто не поддаются исправлению, неправильно настроены и невидимы для ИТ-групп, что делает их программными целями для субъектов угроз, которые ищут более детальные сведения в корпоративных сетях. 

Традиционные средства мониторинга сетевой безопасности, разработанные для корпоративных ИТ-сетей, не могут решить эти среды из-за отсутствия глубокого понимания специализированных протоколов, устройств и поведений "компьютер-компьютер" (M2M), обнаруженных в средах IoT и в среде.

Защитник Azure для IoT — это комплексное решение, которое постоянно обнаруживает, отслеживает и контролирует угрозы, риски и уязвимости на всех устройствах Интернета вещей и устройств, а также является ли они более новыми устройствами, управляемыми через центр Интернета вещей Azure или устаревшими неуправляемыми устройствами. Защитник Azure для Интернета вещей помогает ускорить реагирование на инциденты, предоставляет сведения о рабочих проблемах и упрощает защиту от гибридных рабочих нагрузок за счет предоставления унифицированного центра Интернета вещей и контроля и управления.

Это целостное решение предназначено для ключевых вариантов использования, включающих:
- Автоматическое обнаружение активов для всех устройств IoT и в
- Непрерывное управление рисками и уязвимостью IoT
- IoT, Поиск угроз и реагирование на инциденты
- Операционная эффективность 

Защитник Azure для Интернета вещей — это единое решение для обеспечения безопасности Интернета вещей на всех устройствах. Она предоставляет средства для обеспечения безопасности в новых центрах Интернета вещей и устройствах, управляемых через центр Интернета вещей Azure, а также возможности мониторинга сетевого уровня без агента, необходимые для защиты неуправляемых активов Интернета вещей и ресурсов.

## <a name="unified-visibility-and-control"></a>Унифицированное представление и управление

Получите единое представление о безопасности для всех локальных и облачных рабочих нагрузок, включая решение Azure IoT. Подключение новых устройств и применение политик безопасности в рабочих нагрузках (конечные устройства, устройства Microsoft Edge, Центр Интернета вещей) для обеспечения соответствия стандартам безопасности и повышенной безопасности.

## <a name="adaptive-threat-prevention"></a>Адаптивное предотвращение угроз

Используйте защитник для Интернета вещей, чтобы постоянно отслеживать безопасность компьютеров, сетей и служб Azure. Выберите один из сотен встроенных средств оценки безопасности или создайте свой собственный в центральном защитнике для панели мониторинга центра Интернета вещей. Оптимизируйте параметры безопасности и улучшите оценку безопасности с помощью рекомендаций по виртуальным машинам, сетям, приложениям и данным. Благодаря недавно добавленным возможностям IoT вы можете уменьшить уязвимую область для решения Интернета вещей Azure и устранить проблемы, прежде чем они смогут использовать их.

## <a name="intelligent-threat-detection-and-response"></a>Интеллектуальная система обнаружения угроз и ответные меры

Используйте расширенную аналитику и Microsoft Intelligent Security Graph, чтобы получить ребро над развивающимися кибератак-атаками. Встроенная аналитика поведения и машинное обучение выявляет атаки и эксплойты с нулевыми днями. Контролируйте решение Интернета вещей на предмет входящих атак и действий после нарушения. Оптимизируйте исследование и исправление устройств с помощью интерактивных средств и контекстной аналитики угроз.
