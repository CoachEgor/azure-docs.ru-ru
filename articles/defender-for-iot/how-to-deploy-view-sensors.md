# <a name="view-and-manage-onboarded-sensors"></a>Просмотр и управление встроенными датчиками

В этой статье описывается, как просматривать и удалять встроенные датчики, а также загружать новые файлы активации для встроенных датчиков.

## <a name="update-sensor-management-mode"></a>Обновление режима управления датчика

Может потребоваться обновить режим, управляемый датчиком. После этого необходимо удалить текущий датчик со страницы Управление датчиками и передать новый файл активации.

- **Работа в облачном управляемом режиме вместо локального управляемого режима**: обновление файла активации для локально управляемого датчика с помощью файла активации для управляемого облачного датчика. После повторной активации обнаружение датчиков отображается на портале "датчик" и "защитник Azure для IoT". После успешной отправки файла повторной активации в Azure отправляются только что обнаруженные сведения об оповещении.

- **Работа в локально управляемом режиме вместо облачного управляемого режима**: обновление файла активации для управляемого облачного датчика с помощью файла активации для локально управляемого датчика. После повторной активации сведения об обнаружении датчика отображаются только на датчике.

- **Свяжите датчик с новым центром Интернета вещей**. Вы можете связать датчик с новым центром Интернета вещей. Для этого повторно зарегистрируйте датчик и отправьте новый файл активации.

**Чтобы повторно активировать датчик, выполните следующие действия.**

1. Перейдите на страницу "защитник Azure для IoT", " **Управление датчиками** ".

2. Выберите датчик, для которого требуется отправить новый файл активации.

3. Удалите его.

4. Подключите датчик снова **со страницы подключения** в новом режиме или с новым центром Интернета вещей.

5. Скачайте файл активации со страницы **скачать файл активации** .

6. Войдите в консоль датчика защитника Azure для IoT.

7. В консоли датчика выберите **Параметры системы** , а затем щелкните повторная **Активация**.

   ![Снимок экрана представления повторной активации](media/updates/image14.png)

8. Выберите **Отправить** и выберите сохраненный файл.

9. Выберите **Активировать**.
 
## <a name="delete-sensors"></a>Удаление датчиков

При удалении управляемого облачного датчика информация не будет отправляться в центр Интернета вещей.

Удалить локально Управляемые датчики, если вы больше не работаете с ними.

**Удаление датчиков:**

1. Перейдите на страницу **управления датчиками** Azure Defender для IOT.

2. Выберите датчики, которые требуется удалить.

3. Выберите **удалить датчик**.
