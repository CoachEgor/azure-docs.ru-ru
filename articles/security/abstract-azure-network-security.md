# <a name="azure-network-security"></a>Сетевая безопасность Azure
## <a name="abstract"></a>Аннотация
Сетевые службы Azure по своему определению повышают гибкость, доступность, устойчивость, безопасность и целостность. В этом техническом документе приводятся сведения о сетевых функциях Azure. В нем также содержится информация о том, как клиенты могут использовать встроенные функции безопасности в Azure для защиты своих информационных ресурсов.
Этот технический документ предназначен для следующих категорий читателей:
- технические руководители, сетевые администраторы и разработчики, которым нужны доступные и поддерживаемые в Azure решения по обеспечению безопасности;
- эксперты или руководители бизнес-процессов, которые хотят получить общий обзор технологий и служб Azure, относящихся к сетевой безопасности в общедоступном облаке Azure.
