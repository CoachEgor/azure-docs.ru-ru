# <a name="threat-modeling-tool-update-release-71607021---722019"></a>Угроза Моделирование Инструмент обновления релиз 7.1.60702.1 - 7/2/2019

Версия 7.1.60702.1 инструмента моделирования угроз Microsoft (TMT) была выпущена 2 июля 2019 года и содержит следующие изменения:

- Улучшение специальных возможностей
- Исправления ошибок
- Включение сообщества с открытым исходным кодом при условии набора трафаретов

## <a name="feature-changes"></a>Изменения функций

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>Доступен новый набор трафаретных устройств, предоставляемый сообществом с открытым исходным кодом

Сообщество с открытым исходным кодом предоставило план-баттл для моделирования медицинских устройств. После обновления новый набор трафаретов появится в меню выбора шаблона. Для получения информации о вкладе трафаретов или содержимого в шаблоны, просмотрите информацию на [странице проекта GitHub](https://github.com/Microsoft/threat-modeling-templates).
