# <a name="threat-modeling-tool-update-release-71607021---722019"></a>Threat Modeling Tool обновления выпуска 7.1.60702.1 — 7/2/2019

Версия 7.1.60702.1 Microsoft Threat Modeling Tool (ТМТ) была выпущена 2 2019 июля и содержит следующие изменения:

- Улучшения специальных возможностей
- Исправления ошибок
- Включение набора трафаретов, предоставленных сообществом с открытым исходным кодом

## <a name="feature-changes"></a>Изменения функций

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>Доступен новый набор элементов "медицинские устройства", предоставленный сообществом с открытым кодом.

Набор трафаретов для моделирования медицинских устройств был создан сообществом с открытым кодом. После обновления новый набор трафаретов появится в раскрывающемся меню выбора шаблона. Сведения о том, какие трафареты или содержимое добавляются в шаблоны, см. на [странице GitHub](https://github.com/Microsoft/threat-modeling-templates)проекта.

## <a name="system-requirements"></a>Требования к системе

- Поддерживаемые операционные системы
  - [Юбилейное обновление Microsoft Windows 10](https://blogs.windows.com/windowsexperience/2016/08/02/how-to-get-the-windows-10-anniversary-update/#HTkoK5Zdv0g2F2Zq.97) или более поздней версии
- Требуемая версия .NET
  - [.NET 4.7.1](http://go.microsoft.com/fwlink/?LinkId=863262) или более поздней версии
- Дополнительные требования
  - Для получения обновлений средства и шаблонов требуется подключение к Интернету.
