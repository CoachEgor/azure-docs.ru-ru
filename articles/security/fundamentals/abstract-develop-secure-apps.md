# <a name="develop-secure-cloud-applications-on-azure"></a>Разработка безопасных облачных приложений в Azure
## <a name="abstract"></a>Аннотация
Этот документ представляет собой общее руководство по вопросам контроля и обеспечения безопасности, которые следует учитывать на каждом этапе жизненного цикла разработки приложений для облака. Реализация этих концепций до выпуска продукта поможет вам создать более защищенное программное обеспечение. Представленные в этом документе рекомендации основаны на нашем опыте обеспечения безопасности в Azure, а также опыте наших клиентов.

Этот документ предназначен для специалистов по разработке систем, разработчиков и тест-инженеров всех уровней, которые создают и развертывают в Azure защищенные решения.
