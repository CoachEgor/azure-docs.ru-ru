# <a name="azure-security-mvp-program"></a>Программа специалистов MVP по вопросам обеспечения безопасности в Azure

Microsoft Most Valuable Professionals (MVP) — это самые активные участники технических сообществ, продемонстрировавшие исключительную готовность помогать другим. Специалисты MVP позволяют другим получить максимальную отдачу от их опыта с помощью технологий Майкрософт. Они делятся своей увлеченностью, знаниями и опытом с техническим сообществом и командой Майкрософт.

Теперь представители технического сообщества могут получить специальный статус в сфере безопасности Microsoft Azure. Самые ценные специалисты Microsoft MVP получают статус MVP по Microsoft Azure за свой вклад в усовершенствование решений Azure.

![Microsoft MVP](./media/mvp/azure-security-mvp-fig1.png)

Нет никаких критериев для того, чтобы стать MVP. Это отчасти потому, что это зависит от технологии и его жизненного цикла. Некоторые из критериев включают в себя:

- влияние вклада кандидата в онлайн-форумы, например Microsoft Answers, TechNet и MSDN;
- вики-страницы и интернет-содержимое;
- конференции и группы пользователей;
- подкасты, веб-сайты, блоги и социальные сети;
- статьи и книги.
