# <a name="azure-services-by-fedramp-and-dod-cc-srg-audit-scope"></a>Службы Azure, FedRAMP и министерства обороны США CC Характерных аудита область

Облачные службы корпорации Майкрософт для государственных организаций соответствует требованиям ресурсоемких нам Федеральная & программа управления авторизации (FedRAMP) и министерства обороны США, сведения влияние уровней 2 – 5. Развертывание защищенных служб, включая Azure для государственных организаций, Office 365 США Государственные и Dynamics 365 для государственных организаций, федеральных и Оборонные агентства могут использовать богатый набор совместимых служб.

В этой статье предоставляет подробный список в области облачных служб в Azure и Azure для государственных организаций для предложений для соответствия требованиям FedRAMP и министерства обороны США Характерных "Копия".

#### <a name="terminologysymbols-used"></a>Терминология и символы, используемые

* Министерства обороны США Характерных CC = министерства обороны облачных вычислений по обеспечению требований безопасности
* IIL = уровень влияния сведения
* FedRAMP = Федеральная программа управления и авторизацией  
* : heavy_check_mark: = указывает служба получила этой области аудита.
