# <a name="azure-security-documentation"></a>Документация по системе безопасности Azure

Средства безопасности интегрированы в каждое решение Azure. Azure предлагает уникальную систему защиты на основе глобальной службы безопасности, сложных элементов управления для клиентов и безопасной защищенной инфраструктуры. Это мощное сочетание помогает защитить приложения и данные, а также обеспечивает соответствие требованиям и экономичную защиту для организаций любых размеров.
