# <a name="get-help-with-your-system"></a>Получение помощи по работе с системой

Для получения помощи в системе Avere vFXT для Azure приведены способы получения поддержки:

* **Проблема Avere vFXT** — используйте портал Azure, чтобы открыть запрос в службу поддержки для кластера Avere vFXT, как описано [ниже](#open-a-support-ticket-for-your-avere-vfxt).
* **Квота** — при наличии проблем, связанных с квотой, [запросите увеличение квоты](#request-a-quota-increase).
* **Документация и примеры** — если вы обнаружили проблемы с этой документацией или примерами, прокрутите страницу вниз и используйте раздел **Обратная связь** для поиска имеющихся проблем и при необходимости сообщите о новой проблеме.

## <a name="open-a-support-ticket-for-your-avere-vfxt"></a>Открытие запроса в службу поддержки для Avere vFXT

Если возникают проблемы при развертывании или использовании Avere vFXT, отправьте запрос на поддержку с помощью портала Azure.

Выполните следующие действия, чтобы в запросе в службу поддержки был указан нужный ресурс кластера. Прикрепление ресурса к запросу помогает нам перенаправить его подходящему сотруднику службы поддержки.

1. Из [https://portal.azure.com](https://portal.azure.com), выберите **группы ресурсов**. Просмотрите группу ресурсов, содержащую кластер vFXT, где возникла проблема, и нажмите на одну из виртуальных машин кластера Avere.

    ![снимок экрана области обзора группы ресурсов на портале Azure с выделенной виртуальной машиной](media/avere-vfxt-ticket-vm.png)

1. На странице виртуальной машины прокрутите до нижней части левой области, а затем выберите **Новый запрос на получение поддержки**.

    ![Снимок экрана портала Azure: страница виртуальной машины (той же, которая была показана на предыдущем снимке экрана). Левое меню прокручивается внизу, а 'Новый запрос поддержки' обведен.](media/avere-vfxt-ticket-request.png)

1. На первой странице запроса поддержки выберите тип проблемы и убедитесь, что выбрана правильная подписка.

   Под **обслуживанием**, щелкните **Все обслуживания** и посмотрите под **хранением** для того чтобы выбрать **Avere vFXT**.

   Добавьте краткое резюме и выберите тип проблемы.

    ![скриншот нового экрана запроса поддержки на портале Azure. Выбрана вкладка «Основы». Элементы экрана включают тип проблемы, подписку, обслуживание, резюме и тип проблемы.](media/ticket-basics.png)

   Для продолжения нажмите кнопку **Далее**.

1. Вторая страница формы поддержки содержит предложения по устранению проблемы на основе краткого описания. Нажмите кнопку **«Следующая»** внизу, если вам все же нужно создать билет поддержки.

   ![скриншот нового экрана запроса поддержки с выбранной вкладкой Solutions. Текстовое поле в середине имеет название "Рекомендуемое решение" и объясняет возможные средства правовой защиты.](media/ticket-solutions.png)

1. На третьей странице предоставьте подробную информацию - это включает информацию о вашем кластере, времени возникновения проблемы, серьезности и способе связаться с вами. Заполните информацию и нажмите **кнопку «Следующая»** внизу.

   ![скриншот нового экрана запроса поддержки с выбранной вкладкой «Подробности». Информационные поля организованы по категориям «Проблемные детали», «Метод поддержки» и «Контактная информация».](media/ticket-details.png)

1. Просмотрите информацию на последней странице и нажмите **Создать**. На ваш адрес электронной почты будут отправлены подтверждение и номер запроса, после чего с вами свяжется сотрудник службы поддержки.

## <a name="request-a-quota-increase"></a>Запрос на увеличение квоты

Ознакомьтесь с разделом о [квоте для кластера vFXT](avere-vfxt-prereqs.md#quota-for-the-vfxt-cluster), чтобы узнать, какие компоненты требуются для развертывания кластера Avere vFXT для Azure. Вы можете [запросить увеличение квоты](https://docs.microsoft.com/azure/azure-portal/supportability/resource-manager-core-quotas-request) на портале Azure.
