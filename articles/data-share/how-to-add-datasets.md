# <a name="how-to-add-datasets-to-an-existing-share-in-azure-data-share-preview"></a>Добавление наборов данных в существующую общую папку в предварительной версии общего ресурса Azure

В этой статье объясняется, как добавить наборы данных в уже существующий общий ресурс с помощью предварительной версии Azure Data Share. Это позволяет совместно использовать больше данных с теми же получателями без необходимости создавать новую общую папку.

Сведения о том, как добавлять наборы данных при создании общего ресурса, см. в руководстве по [совместному использованию данных](share-your-data.md) .

## <a name="navigate-to-a-sent-data-share"></a>Переход к отправленной общей папке данных

В предварительной версии общего ресурса данных Azure перейдите к отправленному общему ресурсу и выберите вкладку **наборы данных** . Нажмите кнопку **+ Add DataSets (+ добавить наборы данных** ), чтобы добавить дополнительные наборы данных.

![Добавление наборов данных](./media/how-to/how-to-add-datasets/add-datasets.png)

На панели справа выберите тип набора данных, который необходимо добавить, и нажмите кнопку **Далее**. Выберите подписку и группу ресурсов данных, которые вы хотите добавить. С помощью стрелок раскрывающегося списка найдите и установите флажки рядом с данными, которые нужно добавить.

![Добавление наборов данных](./media/how-to/how-to-add-datasets/add-datasets-side.png)

После нажатия кнопки " **добавить наборы данных**" наборы данных будут добавлены в общую папку. Примечание. Ваши потребители должны активировать моментальный снимок, чтобы они могли видеть новые наборы данных. Если параметры моментальных снимков настроены, потребители увидят новые наборы данных после завершения следующего запланированного снимка. Без настройки параметров моментальных снимков потребитель должен вручную активировать полную или добавочную копию данных для получения обновлений. Дополнительные сведения о моментальных снимках см. в разделе [моментальные снимки](terminology.md).
