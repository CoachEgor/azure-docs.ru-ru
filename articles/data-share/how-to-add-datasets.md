# <a name="how-to-add-datasets-to-an-existing-share-in-azure-data-share"></a>Как добавить наборы данных к существующей доле в акции данных Azure

В этой статье объясняется, как добавить наборы данных к уже существующему обмену данными с помощью Azure Data Share. Это позволяет обмениваться большим объемом данных с теми же получателями без создания новой доли.

Для получения информации о том, как добавлять [Share data](share-your-data.md) наборы данных при создании общего использования, см.

## <a name="navigate-to-a-sent-data-share"></a>Перейдите к отправленной доле данных

В Azure Data Share перейдите к отправленной акции и выберите вкладку **Datasets.** **Нажмите кнопку «Добавить наборы данных»,** чтобы добавить больше наборов данных.

![Добавление наборов данных](./media/how-to/how-to-add-datasets/add-datasets.png)

В панели справа выберите тип набора данных, который вы хотели бы добавить, а затем нажмите **Кнопка Next.** Выберите группу подписки и ресурсов данных, которые вы хотели бы добавить. Используя стрелки выпадения, найдите, а затем проверьте поле рядом с данными для добавления.

![Добавление наборов данных](./media/how-to/how-to-add-datasets/add-datasets-side.png)

Как только вы нажмете **Добавить наборы данных,** наборы данных будут добавлены к вашей акции. Примечание: Снимок должен быть запущен вашими потребителями для того, чтобы они видели новые наборы данных. При настройке моментальных параметров потребители будут видеть новые наборы данных после завершения следующего запланированного снимка. Без настройки моментального снимка потребитель должен вручную вызвать полную или постепенную копию данных для получения обновлений. Для получения дополнительной информации о снимках, [см.](terminology.md)
