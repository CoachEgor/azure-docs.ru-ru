# <a name="how-to-add-a-recipient-to-an-existing-data-share-in-azure-data-share"></a>Как добавить получателя к существующей доле данных в акции данных Azure

В этой статье будет показан способ добавления получателей к уже существующему обмену данными.

Для получения информации о том, как добавлять [Share data](share-your-data.md) получателей при создании общего использования, см.

## <a name="navigate-to-a-sent-data-share"></a>Перейдите к отправленной доле данных

В разделе «Общий доступ к данным Azure» перейдите к отправленной акции и выберите вкладку **«Приглашения».** Чтобы добавить новый, нажмите **Добавить получателя**.

![Добавление получателя](./media/how-to/how-to-add-recipients/add-recipient.png)

Панель будет выскочить на правой стороне страницы. Нажмите **Добавить получателя,** а затем заполнить электронную почту вашего нового получателя на пустой линии. Добавьте столько получателей, сколько вы хотели бы.

![Добавление получателя](./media/how-to/how-to-add-recipients/add-recipient-side.png)

Нажмите **Добавить и отправить приглашение**. Новому получателю будут отправлены пригласительные письма в этот общий доступ к данным.
