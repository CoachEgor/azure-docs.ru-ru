# <a name="how-to-configure-a-dataset-mapping-for-a-received-share-in-azure-data-share"></a>Как настроить отображение набора данных для полученной доли в Azure Data Share

В этой статье объясняется, как настроить отображение набора данных для полученного обмена с помощью Azure Data Share. Вы хотите сделать это, если вы приняли приглашение на обмен данными, но решили "Принять и настроить позже", или если данные совместно на месте. Возможно, вы захотите настроить отображение набора данных, если вам необходимо изменить пункт назначения для данных, которыми вы делились, или если вы хотите получать данные в сервер S'L. 

## <a name="navigate-to-a-received-data-share"></a>Перейдите к полученному обмену данными

В службе совместного использования данных Azure перейдите к полученной акции и выберите вкладку **«Подробности».** 

![Отображение набора данных](./media/dataset-mapping.png "Отображение набора данных") 

Проверьте поле рядом с набором данных, который вы хотите назначить пункт назначения. Выберите **Unmap,** чтобы отображение существующего отображения. Выберите **карту для таргетинга,** чтобы выбрать новый магазин назначения. 

![Карта к цели](./media/dataset-map-target.png "Карта к цели") 

## <a name="select-a-new-target-store"></a>Выберите новый целевой магазин

Выберите тип целевых данных, который вы хотите, чтобы данные приземлились. Для совместного использования моментальных снимков любые данные, уже существующие в любых ранее отображеных учетных записях хранилища, не будут автоматически перемещены в новый целевой магазин. Для совместного использования в месте выберите хранилище данных в указанном местоположении. Местоположение — это центр обработки данных Azure, в котором находится хранилище исходных данных поставщика данных.

![Целевая учетная запись хранения](./media/dataset-map-target-sql.png "Целевое хранилище") 

## <a name="select-a-file-format-sql-sources-only"></a>Выберите формат файла (только для источников S'L)

Если исходные данные получены из исходного источника, вы можете выбрать формат, в котором они получены. 

![Выберите формат](./media/sql-file-formats.png "Форматы файлов S'L")
