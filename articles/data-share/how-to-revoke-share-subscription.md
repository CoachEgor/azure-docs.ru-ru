# <a name="how-to-revoke-a-consumers-share-subscription-in-azure-data-share-preview"></a>Как отозвать общую подписку потребителя в предварительной версии Azure Data Share

В этой статье объясняется, как отозвать подписку на общий ресурс у одного или нескольких потребителей с помощью предварительной версии Azure Data Share. Это не даст потребителю активировать дополнительные моментальные снимки. Если потребитель еще не активировал моментальный снимок, он никогда не будет принимать данные после отмены подписки на общий ресурс. Если они ранее активировали моментальный снимок, последние данные будут оставаться в своей учетной записи.

## <a name="navigate-to-a-sent-data-share"></a>Переход к отправленной общей папке данных

В предварительной версии общего ресурса данных Azure перейдите к отправленному общему ресурсу и выберите вкладку **поделиться подписками** .

![Отозвать общую подписку](./media/how-to/how-to-revoke-share-subscription/revoke-share-subscription.png) 

Установите флажки рядом с получателями, чьи подписки нужно удалить, а затем нажмите кнопку **отозвать**. Потребитель больше не будет получать обновления своих данных.
