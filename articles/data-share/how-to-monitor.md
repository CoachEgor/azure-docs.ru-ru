# <a name="monitor-azure-data-share-preview"></a>Монитор предварительной версии общей папки данных Azure 

В этой статье объясняется, как можно отслеживать в общие папки с данными, с помощью предварительной версии Azure данных общей папки. В качестве поставщика данных вы сможете отслеживать различные аспекты данных общего доступа к связи. Сведения, такие как ли потребителей данных приняли приглашение к общему ресурсу данных, а также ли они создали подписки общей папки и начали использовать ваши данные доступны для мониторинга. 

В качестве получателя данных вы можете отслеживать моментальных снимков, быть активировано в свою подписку Azure. 

## <a name="monitor-invitation-status"></a>Состояние монитора приглашения

Просмотр состояния приглашений общего ресурса данных, перейдя к общим папкам Sent "->" приглашения. 

![Состояние приглашения](./media/invitation-status.png "состояние приглашения") 

Существует три состояния, которые могут быть ваше приглашение в:

* Ожидающие - получатель общей папки данных еще не принял приглашение.
* Принято — получатель данных общего ресурса уже принял приглашение.
* Отклонен - получатель общей папки данных отклонил приглашение.

> [!IMPORTANT]
> При удалении приглашение, после его уже принято, не эквивалентен отмене доступа. Если вы хотите остановить будущих моментальные снимки, копируются в учетную запись хранения потребителей данных, необходимо отменить доступ через *подписки на общую папку* вкладки. 

## <a name="monitor-share-subscriptions"></a>Монитор подписок на общую папку

Просмотр состояния подписки общей папки, перейдя по адресу отправлено общие папки "->" совместное использование подписок. Это позволит получить сведения об активных подписках, созданных потребителей данных после принятия приглашения. Вы можете остановить будущие обновления для данных пользователя, выбрав подписки общей папки и выбрав *отозвать*. 

## <a name="snapshot-history"></a>Журнал моментальных снимков 

На вкладке "Журнал" имеется возможность просмотреть список моментальных снимков, которые были скопированы в потребителя данных клиента. Вы сможете отслеживать частота и длительность каждого интервала моментальных снимков. 

![Моментальный снимок журнала](./media/sent-shares.png "моментальных снимков журнала") 

Можно просмотреть дополнительные сведения о каждом моментальном снимке, запустите, щелкнув на дату начала выполнения.
