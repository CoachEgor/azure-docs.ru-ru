# <a name="storsimple-virtual-array-update-12-release-notes"></a>StorSimple Виртуальный Array Обновление 1.2 примечания к выпуску

В следующих заметках о выпуске перечислены важные нерешенные и устраненные проблемы, связанные с обновлениями для виртуального массива Microsoft Azure StorSimple.

Примечания к выпуску постоянно обновляются. По мере обнаружения новых критических проблем добавляется информация о способах их устранения. Перед развертыванием виртуального массива StorSimple внимательно изучите информацию, содержащуюся в заметках о выпуске.

Обновление 1.2 соответствует версии программного обеспечения **10.0.10311.0**.

> [!IMPORTANT]
> - Обновления нарушают работу системы и приводят к перезагрузке устройства. Если выполняются какие-либо операции ввода-вывода, то устройство находится в состоянии простоя. Для получения подробных инструкций по пакетам, используемым для применения этого обновления, перейдите в [Загружайте обновление 1.2](#download-update-12).
> - Обновление 1.2 доступно вам через портал Azure только в том случае, если устройство работает с обновлением 1.0 или 1.1.

## <a name="whats-new-in-update-12"></a>Новые возможности в обновлении 1.2

Это обновление содержит следующие исправления ошибок:

- Улучшенная устойчивость при обработке удаленных файлов.
- Улучшение обработки исключений в пути запуска кода приводит к уменьшению сбоев в резервном копировании, восстановлении, облачных чтениях и автоматизированной рекультивации пространства.

## <a name="download-update-12"></a>Скачать обновление 1.2

Чтобы загрузить это обновление, перейдите на сервер [каталога обновлений Майкрософт](https://www.catalog.update.microsoft.com/Home.aspx) и загрузите пакет KB4502035. Этот пакет содержит следующие пакеты:

 - **KB4493446,** который содержит кумулятивные обновления Windows за 2012 R2 до апреля 2019 года. Для получения дополнительной информации о том, что включено в этот скат, перейдите на [апрель ежемесячный свертывание безопасности](https://support.microsoft.com/help/4493446/windows-8-1-update-kb4493446).
 - **KB3011067,** который представляет собой файл Microsoft Update Standalone Package WindowsTH-KB3011067-x64. Этот файл используется для обновления программного обеспечения на устройстве.

Скачать KB4502035 и следовать этим инструкциям, чтобы [применить обновление через локальный веб-uI](storsimple-virtual-array-install-update-11.md#use-the-local-web-ui).

## <a name="issues-fixed-in-update-12"></a>Проблемы, устраненные в обновлении 1.2

В таблице ниже содержится список проблем, устраненных в этом выпуске.

| Нет. | Компонент | Проблема |
| --- | --- | --- |
| 1 |Удаление| В предыдущих версиях программного обеспечения возникала проблема, когда использование устройства не менялось даже при удалении файлов. Эта проблема исправлена в этой версии. Путь кода уровня был сделан более устойчивым при обработке удаленных файлов.|
| 2 |Обработка исключений| В предыдущих версиях программного обеспечения возникла проблема после перезагрузки системы, которая потенциально может привести к сбоям в резервном копировании, восстановлению, чтению из облака и автоматической мелиорации пространства. Этот релиз содержит изменения в том, как обрабатываются исключения в пути запуска.|

## <a name="known-issues-in-update-12"></a>Известные проблемы в обновлении 1.2

В обновлении 1.2 не было отмечено никаких новых выпусков. Все выпуски переносятся из предыдущих релизов. Чтобы увидеть резюме известных проблем, включенных из предыдущих релизов, перейдите к [известным вопросам в обновлении 1.1](storsimple-virtual-array-update-11-release-notes.md#known-issues-in-update-11).
