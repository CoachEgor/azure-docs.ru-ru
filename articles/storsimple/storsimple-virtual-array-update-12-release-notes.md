# <a name="storsimple-virtual-array-update-12-release-notes"></a>Заметки о выпуске обновления 1.2 виртуального массива StorSimple

В следующих заметках о выпуске перечислены важные нерешенные и устраненные проблемы, связанные с обновлениями для виртуального массива Microsoft Azure StorSimple.

Заметки о выпуске постоянно обновляются. При обнаружении критические проблемы, требующие временного решения, они добавляются. Перед развертыванием виртуального массива StorSimple внимательно изучите информацию, содержащуюся в заметках о выпуске.

Обновление 1.2 соответствует версии программного обеспечения **10.0.10311.0**.

> [!IMPORTANT]
> - Обновления нарушают работу системы и приводят к перезагрузке устройства. Если выполняются какие-либо операции ввода-вывода, то устройство находится в состоянии простоя. Подробные инструкции по пакетов, используемых для применения этого обновления, см. в статье [загрузить обновление 1.2](#download-update-12).
> - Обновление 1.2 доступно вам на портале Azure только в том случае, если на устройстве установлено обновление 1.0 или 1.1.

## <a name="whats-new-in-update-12"></a>Новые возможности в обновлении 1.2

Это обновление содержит следующие исправления ошибок:

- Повышенная устойчивость при обработке удаленных файлов.
- Улучшенная обработка исключений в стартовом пути кода, что приводит к ограниченной сбоев в резервные копии, восстановления, операций чтения в облаке и автоматической реорганизации пространства.

## <a name="download-update-12"></a>Скачайте обновление 1.2

Чтобы загрузить это обновление, перейдите к [каталога Центра обновления Майкрософт](https://www.catalog.update.microsoft.com/Home.aspx) сервера и KB4502035 загружаемого пакета. Этот пакет содержит следующие пакеты:

 - **KB4493446** , содержащий накопительные пакеты обновления Windows 2012 R2 до апреля 2019 г. Дополнительные сведения о что входит в этот накопительный пакет, см. в статье [апреля ежемесячных накопительных безопасности](https://support.microsoft.com/help/4493446/windows-8-1-update-kb4493446).
 - **KB3011067** — WindowsTH — KB3011067-x64 файл изолированного пакета Центра обновления Майкрософт. Этот файл используется для обновления программного обеспечения на устройстве.

Загрузите KB4502035 и следуйте этим инструкциям, чтобы [примените обновления, с помощью локального веб-интерфейса](storsimple-virtual-array-install-update-11.md#use-the-local-web-ui).

## <a name="issues-fixed-in-update-12"></a>Проблемы, устраненные в обновлении 1.2

В таблице ниже содержится список проблем, устраненных в этом выпуске.

| № | Компонент | Проблема |
| --- | --- | --- |
| 1 |Удаление| В предыдущих версиях программного обеспечения возникла проблема, когда использование устройство не изменили даже в том случае, если файлы были удалены. Эта проблема исправлена в этой версии. Распределения по уровням путь кода стал более устойчивым при обработке удаленных файлов.|
| 2 |Обработка исключений| В предыдущих версиях программного обеспечения произошла ошибка после перезагрузки системы, который потенциально может привести к сбоям в резервные копии, восстановления, чтения из облака и автоматической реорганизации пространства. Этот выпуск содержит изменения для обработки исключений в стартовом пути.|

## <a name="known-issues-in-update-12"></a>Известные проблемы в обновлении 1.2

Нет новых проблем были выпуска, добавленные в обновлении 1.2. Все проблемы, добавленные выпуска, переносятся из предыдущих выпусков. Чтобы просмотреть список известных проблем, которые включены в предыдущих выпусках, перейдите к [известные проблемы в обновлении 1.1](storsimple-virtual-array-update-11-release-notes.md#known-issues-in-update-11).
