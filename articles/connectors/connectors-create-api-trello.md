# <a name="monitor-and-manage-trello-with-azure-logic-apps"></a>Мониторинг приложения Trello и управление им с помощью Azure Logic Apps

С помощью Azure Logic Apps и соединителя Trello можно создавать автоматизированные задачи и рабочие процессы, которые отслеживают списки, карты, доски, участников команды Trello и т. д. и управляют ими, позволяя:

* отслеживать добавление новых карточек на доски и в списки; 
* создавать и получать доски, карты и списки, а также управлять ими;
* добавлять комментарии и элементы на карты;
* перечислять доски, метки досок, карты на досках, комментарии к картам, элементы карт, участников команды и команды, где вы являетесь участником; 
* получать команды.

Вы можете использовать триггеры, которые получают ответы из учетной записи Trello и делают выходные данные доступными для других действий. Можно использовать действия, которые выполняют задачи с вашей учетной записью Trello. Кроме того, выходные данные от действий Trello могут использоваться другими действиями. Например, при добавлении новой карты на доску или в список можно отправлять сообщения с помощью соединителя Slack. Если вы не знакомы с приложениями логики, ознакомьтесь со статьей [Что такое Azure Logic Apps](../logic-apps/logic-apps-overview.md).

## <a name="prerequisites"></a>Технические условия

* Подписка Azure. Если у вас еще нет подписки Azure, [зарегистрируйтесь для получения бесплатной учетной записи Azure](https://azure.microsoft.com/free/). 

* Учетная запись Trello и учетные данные пользователя.

  Ваши учетные данные авторизуют приложение логики, чтобы оно могло создать подключение и получить доступ к вашей учетной записи Trello.

* Базовые знания [создания приложений логики](../logic-apps/quickstart-create-first-logic-app-workflow.md).

* Приложение логики, из которого необходимо получить доступ к учетной записи Trello. Чтобы начать работу с триггером Trello, [создайте пустое приложение логики](../logic-apps/quickstart-create-first-logic-app-workflow.md). Чтобы использовать действие Trello, запустите приложение логики с триггером, например **Периодичность**.

## <a name="connect-to-trello"></a>Подключение к Trello

[!INCLUDE [Create connection general intro](../../includes/connectors-create-connection-general-intro.md)]

1. Войдите на [портал Azure](https://portal.azure.com) и откройте свое приложение логики в конструкторе приложений логики, если оно еще не открыто.

1. Для пустого приложения логики в поле поиска в качестве фильтра введите trello. В списке триггеров выберите нужный триггер. 

   -или-

   В имеющихся приложениях логики на последнем шаге, в котором необходимо добавить действие, выберите **Новый шаг**. 
   В поле поиска введите trello в качестве фильтра. 
   В списке действий выберите любое необходимое действие.

   Чтобы добавить действие между шагами, переместите указатель на стрелку между шагами. 
   Выберите появившийся знак "плюс" ( **+** ), а затем щелкните **Добавить действие**.

1. Если вам будет предложено войти в Trello, предоставьте доступ к своим приложениям логики и войдите.

1. Укажите необходимые сведения для выбранного триггера или действия и продолжайте создание рабочего процесса приложения логики.
