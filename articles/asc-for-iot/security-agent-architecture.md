# <a name="security-agent-reference-architecture"></a>Архитектура ссылки агента безопасности

Центр безопасности Azure Security Для IoT предоставляет справочную архитектуру для агентов безопасности, которые регистрируют, обрабатывают, агрегируют и отправляют данные безопасности через IoT Hub.

Агенты безопасности предназначены для работы в ограниченной среде IoT и очень настраиваемы с точки зрения значений, которые они предоставляют по сравнению с потребляемыми ресурсами.

Агенты безопасности поддерживают следующие функции:

- Сбор необработанных событий безопасности из базовой операционной системы (Linux, Windows). Чтобы узнать больше о доступных сборщиках данных безопасности, [см.](how-to-agent-configuration.md)

- Агрегировать необработанные события безопасности в сообщения, отправляемые через IoT Hub.

- Authenticate с существующей идентификацией устройства или выделенной идентификацией модуля. Чтобы узнать больше, можно узнать больше о [методах проверки подлинности агента безопасности.](concept-security-agent-authentication-methods.md)

- Настройка удаленно с помощью двойного модуля **azureiotsecurity.** Чтобы узнать больше, [см.](how-to-agent-configuration.md)

Центр безопасности Azure для агентов IoT Security разработан как проекты с открытым исходным кодом и доступны на GitHub:

- [Центр безопасности Azure для агента на основе IoT C](https://github.com/Azure/Azure-IoT-Security-Agent-C)
- [Центр безопасности Azure для агента на основе IoT C](https://github.com/Azure/Azure-IoT-Security-Agent-CS)

## <a name="agent-supported-platforms"></a>Поддерживаемые агентом платформы

Центр безопасности Azure для IoT предлагает различные установщики для 32-битной и 64битовой Windows, и то же самое для 32bit и 64bit Linux. Убедитесь, что у вас есть правильный установщик агента для каждого из ваших устройств в соответствии со следующей таблицей:

| Architecture | Linux | Windows |    Сведения|
|----------|----------------------------------------------|-------------|-------------------------------------------|
| 32-разрядный  | C  | C#  ||
| 64-разрядная версия  | Силий или C           | C#      | Мы рекомендуем использовать c-агент для устройств с более ограниченными или минимальными ресурсами устройства.|
|
