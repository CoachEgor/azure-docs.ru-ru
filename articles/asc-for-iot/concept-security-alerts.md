# <a name="azure-security-center-for-iot-security-alerts"></a>Центр безопасности Azure для оповещений системы безопасности IoT

Центр безопасности Azure для Интернета вещей постоянно анализирует решение Интернета вещей с помощью расширенной аналитики и анализа угроз, чтобы предупредить о вредоносных действиях.
Кроме того, можно создавать пользовательские оповещения на основе знаний о ожидаемом поведении устройства.
Оповещение выступает в качестве индикатора потенциальной компрометации, и его следует исследовать и устранять.

В этой статье вы найдете список встроенных предупреждений, которые можно активировать в центре Интернета вещей или на устройствах IoT.
В дополнение к встроенным оповещениям центр безопасности Azure для Интернета вещей позволяет определять пользовательские оповещения на основе ожидаемого центра Интернета вещей и (или) поведения устройства.
Дополнительные сведения см. в разделе [настраиваемые оповещения](concept-customizable-security-alerts.md).
