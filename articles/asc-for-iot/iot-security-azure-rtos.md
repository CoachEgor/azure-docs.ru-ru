# <a name="azure-security-center-for-iot-security-solution-for-azure-rtos"></a>Центр безопасности Azure для решения безопасности IoT для Azure RTO 

Центр безопасности Azure для модуля безопасности IoT предоставляет комплексное решение по обеспечению безопасности для устройств Azure RTO. Azure RTO поставляется со встроенным модулем безопасности, который охватывает распространенные угрозы на устройствах операционной системы в режиме реального времени. 

![Центр безопасности Azure для Azure IoT RTO](./media/architecture/azure-rtos-security-monitoring.png)


Центр безопасности Azure для модуля безопасности IoT с поддержкой Azure RTO предоставляет следующие возможности: 
- Обнаружение вредоносных сетевых операций
- Пользовательское оповещение, задания базовых показателей поведение устройства
- Улучшение безопасности устройств санации

### <a name="detection-of-malicious-network-activities"></a>Обнаружение вредоносных сетевых действий

Отслеживается входящая и исходящая сетевая активность каждого устройства (Поддерживаемые протоколы: TCP, UDP, ICMP для IPv4 и IPv6). Центр безопасности Azure для Интернета вещей проверяет каждое из этих сетевых действий с помощью веб-канала аналитики угроз Майкрософт. Веб-канал обновляется в режиме реального времени с миллионами уникальных индикаторов угроз, собранных по всему миру. 

### <a name="device-behavior-baselining-based-on-custom-alerts"></a>Поведение устройства задания базовых показателей на основе пользовательских оповещений

Задания базовых показателей позволяет выполнять кластеризацию устройств в группах безопасности и определять ожидаемое поведение каждой группы. Так как устройства IoT обычно предназначены для работы в четко определенных и ограниченных сценариях, можно легко создать базовый уровень, определяющий ожидаемое поведение с помощью набора параметров. Любое отклонение от базовых показателей активирует оповещение. 

### <a name="improve-your-device-security-hygiene"></a>Улучшение безопасности устройств санации

Используя рекомендованную инфраструктуру центра безопасности Azure для Интернета вещей, вы получите знания о проблемах в вашей среде, которые влияют на безопасность ваших устройств. Низкая безопасность устройств IoT может обеспечить успешную атаку, если оставить ее без изменений, так как безопасность всегда измеряется самой слабой связью в любой организации. 

## <a name="get-started-protecting-azure-rtos-devices"></a>Приступая к защите устройств Azure RTO

- Центр безопасности Azure для модуля безопасности IoT для Azure RTO предоставляется бесплатно для загрузки устройств. Центр безопасности Azure для облачной службы IoT доступен в рамках 30-дневной пробной версии для каждой подписки Azure. Скачайте [Центр безопасности Azure для модуля безопасности IOT для Azure RTO](https://github.com/azure-rtos/iot-security-module-preview) , чтобы приступить к работе.
