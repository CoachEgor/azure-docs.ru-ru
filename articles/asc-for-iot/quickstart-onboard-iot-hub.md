# <a name="quickstart-onboard-azure-security-center-for-iot-service-in-iot-hub"></a>Краткое руководство. Включение службы "Центр безопасности Azure для Интернета вещей" в Центре Интернета вещей

В этой статье описано, как включить службу "Центр безопасности Azure для Интернета вещей" в существующем Центре Интернета вещей. Если у вас сейчас нет Центра Интернета вещей, ознакомьтесь со статьей [Создание Центра Интернета вещей с помощью портала Azure](https://docs.microsoft.com/azure/iot-hub/iot-hub-create-through-portal), чтобы приступить к работе. 

> [!NOTE]
> Центр безопасности Azure для Интернета вещей сейчас доступен только для Центра Интернета вещей ценовой категории "Стандартный".
> Центр безопасности Azure для Интернета вещей — это решение с одним концентратором. Если вам требуется несколько центров, используйте несколько решений Центра безопасности Azure для Интернета вещей. 

## <a name="prerequisites-for-enabling-the-service"></a>Предварительные требования для включения службы

- Рабочая область Log Analytics
  - По умолчанию Центр безопасности Azure для Интернета вещей хранит два типа сведений в рабочей области Log Analytics: **оповещения безопасности** и **рекомендации по безопасности**. 
  - Вы можете добавить хранение дополнительного типа сведений — **необработанных событий**. Обратите внимание, что хранение **необработанных событий** в Log Analytics приведет к дополнительными затратами. 
- Центр Интернета вещей (ценовая категория "Стандартный")
- Соответствие всем [требованиям службы](service-prerequisites.md) 

## <a name="enable-azure-security-center-for-iot-on-your-iot-hub"></a>Включение Центра безопасности Azure для Интернета вещей в Центре Интернета вещей 

Чтобы включить систему безопасности в Центре Интернета вещей, сделайте следующее: 

1. Откройте **Центр Интернета вещей** на портале Azure. 
1. В меню **Безопасность** щелкните **Secure your IoT solution** (Защита решения "Интернет вещей").
1. Оставьте параметр **Включить** выбранным (то есть не меняйте значение по умолчанию). 
1. Выберите рабочую область Log Analytics.
1. Укажите сведения о рабочей области Log Analytics. 
   - Выберите включение **двойника коллекции**, оставив переключатель **двойника коллекции** в положении **Вкл**.
   - Выберите хранение **необработанных событий** в дополнение к хранению типов сведений по умолчанию, выбрав в Log Analytics параметр **Store raw device security events** (Хранить необработанные события безопасности устройства). Оставьте переключатель **необработанных событий** в положении **Вкл**. 
    
1. Выберите команду **Сохранить**. 

Поздравляем! Вы включили Центр безопасности Azure для Интернета вещей в Центре Интернета вещей.
