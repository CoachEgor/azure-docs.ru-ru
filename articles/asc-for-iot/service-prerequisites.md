# <a name="azure-security-center-for-iot-prerequisites"></a>Центр безопасности Azure для предпосылок Для Обработки Вещей

В этой статье приводится объяснение различных строительных блоков службы безопасности Azure Security Center для IoT, с чего необходимо начать, и объясняется основные понятия, помогающие понять службу. 

## <a name="minimum-requirements"></a>Минимальные требования

- Стандартный уровень IoT концентратора
    - Привилегии уровня **уровня владельца** ролей RBAC 
- [Рабочая область Log Analytics](https://docs.microsoft.com/azure/azure-monitor/learn/quick-create-workspace) 
- Центр безопасности Azure (рекомендуется)
    - Использование Центра безопасности Azure является рекомендацией, а не требованием. Без Центра безопасности Azure вы не сможете просматривать другие ресурсы Azure в концентраторе IoT. 
 
## <a name="working-with-azure-security-center-for-iot-service"></a>Работа с Центром безопасности Azure для службы IoT

Центр безопасности ИОт Azure для получения информации и отчетности по IoT доступен с помощью центра безопасности Azure IoT и Центра безопасности Azure. Для включения Центра безопасности Azure для IoT в концентраторе Azure IoT требуется учетная запись с привилегиями **уровня владельца.** После включения ASC для IoT в концентраторе IoT центр безопасности Azure для получения информации о IoT отображается как функция **безопасности** в концентраторе Azure IoT и **IoT** в Центре безопасности Azure. 

## <a name="supported-service-regions"></a>Поддерживаемые регионы для службы 

Центр безопасности Azure для IoT в настоящее время поддерживается для концентраторов IoT в следующих регионах Azure:
  - Центральная часть США  
  - Восточная часть США 
  - восточная часть США 2
  - центрально-западная часть США
  - западная часть США
  - Западная часть США 2
  - Центрально-южная часть США
  - Центрально-северная часть США
  - Центральная Канада
  - Восточная Канада 
  - Северная Европа    
  - Южная Бразилия
  - Центральная Франция  
  - западная часть Соединенного Королевства 
  - южная часть Соединенного Королевства
  - Западная Европа 
  - Северная Европа 
  - Западная Япония  
  - Восточная Япония  
  - Юго-Восточная часть Австралии
  - Восточная Австралия
  - Восточная Азия   
  - Юго-Восточная Азия
  - Республика Корея, центральный регион
  - Республика Корея, южный регион 
  - Центральная Индия
  - Южная Индия

Центр безопасности Azure для IoT направляет весь трафик из всех европейских регионов в региональный центр обработки данных Западной Европы и все остальные регионы в Центральный центр обработки данных США.  
  
## <a name="wheres-my-iot-hub"></a>Где мой концентратор IoT?

Проверьте местоположение Концентратора IoT, чтобы проверить доступность сервиса перед началом работы. 

1. Откройте Центр Интернета вещей. 
2. Щелкните **Обзор**. 
3. Проверка соответствия местоположения одному из [регионов службы поддержки.](#supported-service-regions)
