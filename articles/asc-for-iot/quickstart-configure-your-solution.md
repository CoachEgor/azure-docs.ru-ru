# <a name="quickstart-configure-your-iot-solution"></a>Краткое руководство. Настройка решения Интернета вещей

В этой статье объясняется, как выполнить начальную настройку решения для обеспечения безопасности Интернета вещей с помощью Центра безопасности Azure для Интернета вещей. 

## <a name="azure-security-center-for-iot"></a>Центр безопасности Azure для Интернета вещей

Центр безопасности Azure для Интернета вещей обеспечивает эффективную комплексную безопасность для решений "Интернет вещей" на основе Azure.

Центр безопасности Azure для Интернета вещей позволяет отслеживать решение "Интернет вещей" на одной панели мониторинга, где отображаются все ваши устройства Интернета вещей, платформы Интернета вещей и серверные ресурсы в Azure.

После включения в Центре Интернета вещей Центр безопасности Azure для Интернета вещей автоматически идентифицирует другие службы Azure, также подключенные к вашему Центру Интернета вещей и связанные с вашим решением Интернета вещей.

В дополнение к автоматическому обнаружению связей можно также выбрать другие группы ресурсов Azure, которые следует пометить как часть решения "Интернет вещей". 

Выбранные параметры позволяют добавлять целые подписки, группы ресурсов или отдельные ресурсы. 

После определения всех связей ресурсов Центр безопасности Azure для Интернета вещей использует Центр безопасности Azure, чтобы предоставлять рекомендации по безопасности и оповещения для этих ресурсов.

## <a name="add-azure-resources-to-your-iot-solution"></a>Добавление ресурсов Azure в решение Интернета вещей

Чтобы добавить новый ресурс в решение Интернета вещей, сделайте следующее. 

1. Откройте **Центр Интернета вещей** на портале Azure. 
1. Откройте раздел **Ресурсы** в разделе **Безопасность** в меню слева. 
1. Щелкните **Изменить** и выберите группы ресурсов, которые относятся к решению "Интернет вещей".
1. Щелкните **Добавить**. 

Поздравляем! Вы добавили новую группу ресурсов в решение "Интернет вещей".

Центр безопасности Azure для Интернета вещей, будучи частью вашего решения "Интернет вещей", теперь отслеживает недавно добавленные группы ресурсов, а также отображает соответствующие рекомендации по безопасности и оповещения.
