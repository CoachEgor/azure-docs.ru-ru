# <a name="customize-your-azure-security-center-for-iot-solution"></a>Настройка центра безопасности Azure для решения IoT 

В этом руководство вы узнаете, как настроить различные параметры в центре безопасности Azure для Интернета вещей.  

> [!div class="checklist"]
> * Настройка рекомендаций по решению 
> * Изменить параметры 

## <a name="change-settings"></a>Изменить параметры

Управление параметром центра безопасности Azure для IoT:

В центре Интернета вещей перейдите к колонке "Обзор безопасности" в левом верхнем углу экрана, а затем в разделе "Параметры" настройте центр безопасности Azure для параметров Интернета вещей и выполните следующие действия.

1. Откройте **Центр Интернета вещей** на портале Azure. 
1. В меню слева в разделе **Безопасность**выберите и откройте **Обзор**.
1. В разделе Параметры выберите параметр решения, который вы хотите изменить.   
1. Не забудьте всегда щелкать **сохранить** в верхней части любого экрана настройки, чтобы сохранить изменения параметров. 

## <a name="configure-solution-recommendations"></a>Настройка рекомендаций по решению

Чтобы настроить центр безопасности Azure для получения рекомендаций по решениям IoT, выполните следующие действия.

1. Откройте **Центр Интернета вещей** на портале Azure. 
1. Выберите и откройте **Обзор** в разделе **Безопасность** в меню слева.
1. В разделе Параметры выберите пункт **Рекомендуемая конфигурация** в меню слева. 
1. Отключите или включите рекомендации по решению, относящиеся к вашей организации и рабочим процессам. 
1. Нажмите кнопку **сохранить** в верхней части экрана, чтобы сохранить выбранные параметры.
