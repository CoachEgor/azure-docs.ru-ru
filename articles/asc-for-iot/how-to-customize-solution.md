# <a name="customize-your-azure-security-center-for-iot-solution"></a>Настройте свой Центр безопасности Azure для решения IoT 

В этом руководстве узнайте, как настроить различные настройки в Центре безопасности Azure для IoT.  

> [!div class="checklist"]
> * Настройка рекомендаций решения 
> * Настройки изменения 

## <a name="change-settings"></a>Настройки изменения

Управление настройками Azure Security Center для настройки IoT:

На вашем концентраторе IoT перейдите на лезвие обзора безопасности в левом верхнем углу, см. "Настройки", чтобы настроить центр безопасности Azure для настроек IoT, сделайте следующее:

1. Откройте **Центр Интернета вещей** на портале Azure. 
1. Из левого меню под **безопасностью,** выберите и откройте **Обзор**.
1. В настройках выберите настройку решения, которая хочется изменить.   
1. Помните, чтобы всегда щелкните **Сохранить** в верхней части любого экрана настройки, чтобы сохранить изменения настройки. 

## <a name="configure-solution-recommendations"></a>Настройка рекомендаций решения

Чтобы настроить в Центре безопасности Azure для рекомендаций решений IoT, сделайте следующее:

1. Откройте **Центр Интернета вещей** на портале Azure. 
1. Выберите и откройте **Обзор** под **безопасностью** в левом меню.
1. В настройках выберите **Рекомендуемую конфигурацию** из левого меню. 
1. Отключить/включить рекомендации решения, имеющие отношение к вашей организации и рабочим процессам. 
1. Выберите **Сохранить** в верхней части экрана, чтобы сохранить ваши выборы.
