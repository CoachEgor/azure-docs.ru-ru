# <a name="pricing-and-associated-costs"></a>Цены и связанные затраты

> [!IMPORTANT]
> Центр безопасности Azure для Интернета вещей сейчас предоставляется в общедоступной предварительной версии.
> Эта предварительная версия предоставляется без соглашения об уровне обслуживания и не рекомендована для использования рабочей среде. Некоторые функции могут не поддерживаться или их возможности могут быть ограничены. Дополнительные сведения см. в статье [Дополнительные условия использования предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

В этой статье объясняются центр безопасности Azure (ASC) для Интернета вещей модель ценообразования, перечислены все сопутствующие затраты и объясняется, как управлять ими.

## <a name="pricing"></a>Цены

ASC для Интернета вещей, модель ценообразования состоит из двух частей и оплачивается после центра Интернета вещей [включена](quickstart-onboard-iot-hub.md) в ASC для Интернета вещей:

- Затраты по устройства — встроенных средств безопасности на основе анализа журналов центра Интернета вещей.

- Затраты по сообщение — расширенные возможности обеспечения безопасности на основе безопасности сообщений с Edge Интернета вещей или конечных устройств.

  >[!Note]
  > Сообщения о безопасности также будет взиматься за использование квоты на центр Интернета вещей.

Дополнительные сведения см. в разделе [цены на Центр безопасности](https://azure.microsoft.com/pricing/details/security-center/).

## <a name="associated-costs"></a>Сопутствующие затраты

ASC для Интернета вещей есть два типа связанных затрат, которые не являются частью прямого ценообразования.

- Потребление квоты центра Интернета вещей

- Затраты на хранение аналитики журналов

Можно сократить сопутствующие затраты, отказавшись от некоторых функций, изменив параметры.

Чтобы изменить параметры:

1. Откройте центр Интернета вещей.

2. В разделе **безопасности**, нажмите кнопку **Обзор**.

3. Щелкните **Параметры**.

Следующая таблица предоставляет сводку сопутствующие затраты и влияние каждого из них.

|     | Использование | Комментарий |
| --- | --- | --- |
| **Потребление квоты центра Интернета вещей** |  |
| [Экспорт устройств](https://docs.microsoft.com/azure/iot-hub/iot-hub-bulk-identity-mgmt#export-devices) задания (Экспорт двойника) | один раз в день | Отключить _двойников коллекции метаданных_ |
| **Журнал аналитики хранилища** |  |
| Оповещения и рекомендации для устройства| Рекомендации по безопасности и оповещения, созданные службой | Не является обязательным |
| Необработанной безопасности данных| Необработанной безопасности данных с устройств Интернета вещей, собираются агентами безопасности | Отключить _хранения неформатированного событий безопасности_ |

>[!Important]
> Отказаться имеет серьезные последствия для возможности системы безопасности.
  
| Отказаться | Последствия |
| --- | --- |
| _Коллекция метаданных двойника_ | Отключить [настраиваемые оповещения](quickstart-create-custom-alerts.md) |
| | Отключить рекомендации манифеста IoT Edge |
| | Отключить оповещения и рекомендации на основе удостоверения устройства |
| _События безопасности неформатированный Store_ | Сведения о рекомендациях базовой операционной системы устройства недоступны. |
| | Сведения о [предупреждение](concept-security-alerts.md) и [рекомендация](concept-recommendations.md) исследования недоступны |
