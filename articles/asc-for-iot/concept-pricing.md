# <a name="pricing-and-associated-costs"></a>Цены и связанные затраты

В этой статье объясняется модель ценообразования Azure Security Center для оценительных моделей IoT, обобщается все связанные с этим расходы и объясняется, как ими управлять.

## <a name="pricing"></a>Цены

Модель ценообразования Azure Security Center для IoT состоит из двух частей и выставляется счетом после [включения](quickstart-onboard-iot-hub.md) концентратора IoT в Центре безопасности Azure для IoT:

- Стоимость за устройством - встроенные возможности безопасности на основе анализа журналов IoT Hub.

- Стоимость за сообщением - расширенные возможности безопасности на основе сообщений безопасности от IoT Edge или лист устройств.

Для получения дополнительной информации [см.](https://azure.microsoft.com/pricing/details/security-center/)

## <a name="associated-costs"></a>Связанные с этим расходы

Центр безопасности Azure для IoT имеет связанные затраты, которые не являются частью прямого ценообразования:

- Расходы на хранение журналов Analytics

Можно сократить связанные с этим затраты, отказавшись от определенных функций решения. Откажитесь, изменив настройки.

Чтобы изменить настройки:

1. Открыть ioT концентратор.

1. Под **безопасностью**, нажмите **Обзор**.

1. Щелкните **Настройки**.

В следующей таблице приводится краткое изложение связанных с этим расходов и последствий каждого варианта.

|     | Использование | Комментарий |
| --- | --- | --- |
| **Хранилище аналитики журналов** |  |
| Рекомендации и оповещения устройств| Рекомендации по безопасности и оповещения, генерируемые службой | Не является факультативным |
| Необработанные данные безопасности| Необработанные данные безопасности с устройств IoT, собранные агентами безопасности | Отключить _необработанные события безопасности устройства_ |
|

>[!Important]
> Отказ от участия в выборах имеет серьезные последствия для доступности функций безопасности функций Безопасности IoT для Центра безопасности ИОт.

| Отказаться | Последствия |
| --- | --- |
| _Сбор метаданных близнецов_ | Отключить [пользовательские оповещения](quickstart-create-custom-alerts.md) |
| | Отбить рекомендации IoT Edge |
| | Отврачив рекомендации и оповещения на основе идентификации устройства |
| _Хранение необработанных событий безопасности устройств_ | Подробная информация о базовых рекомендациях оС устройства недоступна |
| | Подробная информация о [предупреждении](concept-security-alerts.md) и [рекомендации](concept-recommendations.md) расследования не доступны |
|
