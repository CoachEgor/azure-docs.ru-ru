# <a name="security-recommendations"></a>Рекомендации по обеспечению безопасности

Центр безопасности Azure для IoT сканирует ресурсы Azure и устройства IoT и предоставляет рекомендации по безопасности для уменьшения поверхности атаки.
Рекомендации по безопасности являются действенными и направлены на оказание помощи клиентам в соблюдении рекомендаций по обеспечению безопасности.

В этой статье вы найдете список рекомендаций, которые могут быть срабатываны на устройствах IoT hub и/или IoT.

## <a name="recommendations-for-iot-devices"></a>Рекомендации для устройств IoT

Рекомендации по устройству содержат аналитические данные и предложения по улучшению безопасности устройств.

| Серьезность | Имя                                                      | Источник данных | Описание                                                                                                                                                                                           |
|----------|-----------------------------------------------------------|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Средний   | Открытые порты на устройстве                                      | Агент       | На устройстве была найдена конечная точка прослушивания.                                                                                                                                                        |
| Средний   | В одной из цепей обнаружена разрешительность политики брандмауэра. | Агент       | Найдена политика брандмауэра (INPUT/OUTPUT). Политика брандмауэра должна отказать всему трафику по умолчанию и определить правила, позволяющие осуществлять необходимую связь с/из устройства.                               |
| Средний   | Найдено правило разрешительного брандмауэра в цепочке ввода     | Агент       | Установлено правило в брандмауэре, содержащее разрешительный шаблон для широкого круга IP-адресов или портов.                                                                                    |
| Средний   | Найдено правило разрешительного брандмауэра в цепочке вывода    | Агент       | Установлено правило в брандмауэре, содержащее разрешительный шаблон для широкого круга IP-адресов или портов.                                                                                   |
| Средний   | Базовая проверка системы операции не удалась           | Агент       | Устройство не соответствует [тестам CIS Linux.](https://www.cisecurity.org/cis-benchmarks/)                                                                                                        |

### <a name="operational-recommendations-for-iot-devices"></a>Оперативные рекомендации для устройств IoT

Оперативные рекомендации содержат аналитические данные и предложения по улучшению конфигурации агентов безопасности.

| Серьезность | Имя                                    | Источник данных | Описание                                                                       |
|----------|-----------------------------------------|-------------|-----------------------------------------------------------------------------------|
| Низкий      | Агент отправляет неиспользованные сообщения          | Агент       | 10% или более сообщений безопасности были меньше, чем 4 КБ в течение последних 24 часов.  |
| Низкий      | Конфигурация безопасности двойная не оптимальная | Агент       | Конфигурация безопасности с двумя вариантами не является оптимальной.                                        |
| Низкий      | Конфликт конфигурации безопасности с двумя конфигурацией    | Агент       | Конфликты были выявлены в двойной конфигурации безопасности. |                          |
|

## <a name="recommendations-for-iot-hub"></a>Рекомендации для концентратора IoT

Рекомендации оповещения обеспечивают понимание и предложения для действий по улучшению безопасности позы вашей среды.

| Серьезность | Имя                                                     | Источник данных | Описание                                                                                                                                                                                                             |
|----------|----------------------------------------------------------|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Высокий     | Идентичные учетные данные аутентификации, используемые несколькими устройствами | Центр Интернета вещей     | Учетные данные проверки подлинности Концентратора IoT используются несколькими устройствами. Это может указывать на незаконное устройство, выдающее себя за законное устройство. Дублирование использования учетных данных увеличивает риск олицетворения устройства вредоносным субъектом. |
| Средний   | Политика IP-фильтра по умолчанию должна быть отрицана                  | Центр Интернета вещей     | Конфигурация IP-фильтра должна иметь правила, определенные для разрешенного трафика, и должна по умолчанию отказать всему другому трафику по умолчанию.                                                                                                     |
| Средний   | Правило IP-фильтра включает в себя большой диапазон IP                   | Центр Интернета вещей     | Диапазон источников IP-фильтра IP-фильтра слишком велик. Чрезмерно разрешительные правила могут подвергать ваш концентратор IoT злоумышленникам.                                                                                       |
| Низкий      | Включить журналы диагностики в Концентраторе IoT                       | Центр Интернета вещей     | Включите журналы и сохраняйте до года. Сохранение журналов позволяет воссоздать следы активности для целей расследования, когда происходит инцидент безопасности или ваша сеть скомпрометирована.                                       |
|
