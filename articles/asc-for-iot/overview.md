# <a name="introducing-azure-security-center-for-iot"></a>Общие сведения о центре безопасности Azure для Интернета вещей

Унификация управления безопасностью и обеспечение комплексного обнаружения и анализа угроз для гибридных облачных рабочих нагрузок и решения Azure IoT.

## <a name="secure-your-entire-iot-solution-from-iot-devices-to-azure-cloud"></a>Защита всего решения Интернета вещей с устройств Интернета вещей в облаке Azure

Выберите беспроблемное решение без агента или воспользуйтесь комплексной безопасностью на основе агентов. Центр безопасности Azure для Интернета вещей предоставляет средства предотвращения и анализа угроз для каждого устройства, IoT Edge и центра Интернета вещей в ваших ресурсах IoT.

Так как миллиарды новых устройств подключены к Интернету и интегрированы в наши ежедневные и наши компании, группы деятельности по обеспечению безопасности должны обеспечить достаточно быстрое развитие стратегии безопасности, чтобы охватить каждую новую контактную зону. Как и любая другая система, для обеспечения надежной защиты решения Интернета вещей требуется защита на каждом этапе реализации.

Центр безопасности Azure для Интернета вещей упрощает гибридную защиту рабочей нагрузки за счет унифицированной видимости и контроля, адаптивного предотвращения угроз и интеллектуального обнаружения угроз и реагирования на рабочие нагрузки, работающие на пограничных, локальных, в Azure и в других облаках.

![Архитектура Центра безопасности Azure для Интернета вещей](./media/architecture/azure-iot-security-architecture.png)

### <a name="unified-visibility-and-control"></a>Унифицированное представление и управление

Получите единое представление о безопасности для всех локальных и облачных рабочих нагрузок, включая решение Azure IoT. Подключение новых устройств и применение политик безопасности в рабочих нагрузках (конечные устройства, устройства Microsoft пограничных устройств, центр Интернета вещей) для обеспечения соответствия стандартам безопасности и повышенной безопасности.

### <a name="adaptive-threat-prevention"></a>Адаптивное предотвращение угроз

Используйте центр безопасности Azure для Интернета вещей, чтобы постоянно отслеживать безопасность компьютеров, сетей и служб Azure. Выберите из сотен встроенных средств оценки безопасности или создайте собственную на центральном центре безопасности Azure для панели мониторинга центра Интернета вещей. Оптимизируйте параметры безопасности и улучшите оценку безопасности с помощью рекомендаций по виртуальным машинам, сетям, приложениям и данным. Благодаря недавно добавленным возможностям IoT вы можете уменьшить уязвимую область для решения Интернета вещей Azure и устранить проблемы, прежде чем они смогут использовать их.

### <a name="intelligent-threat-detection-and-response"></a>Интеллектуальная система обнаружения угроз и ответные меры

Используйте расширенную аналитику и Microsoft Intelligent Security Graph, чтобы получить ребро над развивающимися кибератак-атаками. Встроенная аналитика поведения и машинное обучение выявляет атаки и эксплойты с нулевыми днями. Контролируйте решение Интернета вещей на предмет входящих атак и действий после нарушения. Оптимизируйте исследование и исправление устройств с помощью интерактивных средств и контекстной аналитики угроз.
