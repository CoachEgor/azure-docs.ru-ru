# <a name="introducing-azure-security-center-for-iot"></a>Общие сведения о центре безопасности Azure для Интернета вещей

Унифицируйте управление безопасностью и включите сквозное обнаружение и анализ угроз в гибридных облачных рабочих нагрузках и в вашем решении Azure IoT. 

## <a name="secure-your-entire-iot-solution-from-iot-devices-to-azure-cloud"></a>Защитите все ioT-решение от устройств IoT до облака Azure

Выберите из нашего бесшовного решения без агента или воспользуйтесь преимуществами комплексной безопасности на основе агента, Azure Security Center for IoT обеспечивает предотвращение угроз и анализ для каждого устройства, IoT Edge и IoT Hub, в ваших ресурсах IoT.

Поскольку миллиарды новых устройств подключены к Интернету и интегрированы в нашу повседневную жизнь и наш бизнес, ваши группы по обеспечению безопасности должны обеспечить, чтобы их стратегии безопасности развивались достаточно быстро, чтобы покрыть каждую новую поверхность атаки. Как и любая другая система, чтобы всесторонне обезопасить свое IoT-решение, она требует защиты на каждом этапе реализации. 

Центр безопасности Azure для IoT упрощает защиту гибридных рабочих нагрузок, обеспечивая единую видимость и контроль, адаптивную предотвращение угроз и интеллектуальное обнаружение угроз и реагирование на рабочие нагрузки, работающие по краям, на территории, в Azure и в других Облака. 

![Архитектура Центра безопасности Azure для Интернета вещей](./media/architecture/azure-iot-security-architecture.png)

### <a name="unified-visibility-and-control"></a>Унифицированное представление и управление

Получите единое представление о безопасности во всех ваших рабочих нагрузках, включая решение Azure IoT. На борту новых устройств и применяйте политики безопасности на рабочих нагрузках (устройства Leaf, устройства Microsoft Edge, IoT Hub) для обеспечения соответствия стандартам безопасности и улучшения положения в области безопасности. 

### <a name="adaptive-threat-prevention"></a>Адаптивное предотвращение угроз

Используйте Центр безопасности Azure для IoT для постоянного мониторинга безопасности машин, сетей и служб Azure. Выберите из сотен встроенных оценок безопасности или создайте свой собственный центр безопасности Azure для панели мониторинга IoT Hub. Оптимизируйте настройки безопасности и улучшите оценку безопасности с помощью практических рекомендаций в виртуальных машинах, сетях, приложениях и данных. С помощью недавно добавленных возможностей IoT можно уменьшить поверхность атаки для решения Azure IoT и устранить проблемы, прежде чем они смогут быть использованы.

### <a name="intelligent-threat-detection-and-response"></a>Интеллектуальная система обнаружения угроз и ответные меры

Используйте расширенную аналитику и график интеллектуальной безопасности Microsoft, чтобы получить преимущество над эволюционирующими кибер-атаками. Встроенная поведенческая аналитика и машинное обучение выявляют атаки и эксплойты нулевого дня. Мониторинг решения IoT для входящих атак и активности после нарушения. Оптимизация исследования и исправления устройств с помощью интерактивных инструментов и контекстной информации об угрозах.
