# <a name="get-started-with-built-in-iot-hub-integration"></a>Начало работы с встроенной интеграцией Концентратора IoT

Эта опция позволяет использовать службу без использования Центра безопасности Azure для агентов безопасности IoT.

## <a name="enable-built-in-iot-hub-integration"></a>Включить встроенную интеграцию Концентратора IoT

Для обеспечения мониторинга управления идентификацией устройства, облаки устройства и облака в шаблонах связи устройства сделайте следующее, чтобы запустить службу:

1. Откройте свой **концентратор IoT**.
1. Выберите меню **обзора безопасности.**
1. Нажмите **Кнопка Безопасный IoT решение** и заполнить бортовую форму.

Поздравляем! Вы завершили включение Центра безопасности Azure для обслуживания IoT в вашем концентраторе IoT.
