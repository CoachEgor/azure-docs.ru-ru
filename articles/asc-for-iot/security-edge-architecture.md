# <a name="azure-iot-edge-security-module"></a>Модуль безопасности Azure IoT Edge

> [!IMPORTANT]
> Центр безопасности Azure для Интернета вещей сейчас предоставляется в общедоступной предварительной версии.
> Эта предварительная версия предоставляется без соглашения об уровне обслуживания и не рекомендуется для производственных worklo§1ads. Некоторые функции могут не поддерживаться или их возможности могут быть ограничены. Дополнительные сведения см. в статье [Дополнительные условия использования предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

[Azure IoT Edge](https://docs.microsoft.com/azure/iot-edge/) предоставляет широкие возможности для управления и выполнения бизнес-процессов на границе.
Часть ключа, IoT Edge играет в средах Интернета вещей сделать ее особенно привлекательной для злоумышленников.

Azure центр безопасности (ASC) для модуля безопасности Интернета вещей предоставляет решение полнофункциональных средств безопасности ваше устройство IoT Edge устройств.
ASC для модуля IoT собирает, собирает и анализирует данные о необработанной безопасности операционной системы и системы контейнеров в практические рекомендации по безопасности и оповещения.

Как и ASC для агентов безопасности Интернета вещей для устройств Интернета вещей, ASC для модуля IoT Edge предоставляет широкие возможности настройки через свой двойник модуля.
См. в разделе [настройки агента](how-to-agent-configuration.md) для получения дополнительных сведений.

ASC для модуля безопасности Интернета вещей для IoT Edge предоставляет следующие возможности:

- Собирает события необработанной безопасности из операционной системы (Linux) и систем контейнера Edge Интернета вещей.
  
  См. в разделе [ASC для настройки агента IoT](how-to-agent-configuration.md) для получения дополнительных сведений о сборщиках данных для системы безопасности.

- Анализ манифесты развертывания IoT Edge.

- Объединяет события необработанной безопасности в сообщения, проходящие через [концентратор Edge Интернета вещей](https://docs.microsoft.com/azure/iot-edge/iot-edge-runtime#iot-edge-hub).

- Удалите конфигурацию с помощью двойника модуля безопасности.

  См. в разделе [Настройка ASC для Интернета вещей агент](how-to-agent-configuration.md) для получения дополнительных сведений.

ASC для модуля безопасности Интернета вещей для IoT Edge работает в привилегированном режиме в IoT Edge.
Привилегированный режим должен разрешить модуль для мониторинга операционной системы и другими модулями IoT Edge.

## <a name="agent-supported-platforms"></a>Агент, поддерживаемый платформ

ASC для модуля безопасности Интернета вещей для IoT Edge в данный момент доступна только для Linux.
