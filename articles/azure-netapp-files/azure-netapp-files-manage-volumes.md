# <a name="manage-volumes-by-using-azure-netapp-files"></a>Управление томами с помощью Azure NetApp Files
Вы можете управлять томом, выполняя для него операции просмотра, обновления или удаления. 

Если нужно удалить том, сначала удалите все его вложенные объекты. 

1.  В колонке для управления пулами емкости щелкните колонку **Тома**. 
2.  В представлении **Обзор томов** можно просматривать, обновлять или удалять существующие тома.
