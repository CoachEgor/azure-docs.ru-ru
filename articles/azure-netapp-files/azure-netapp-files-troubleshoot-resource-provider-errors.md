# <a name="troubleshoot-azure-netapp-files-resource-provider-errors"></a>Устранение ошибок с поставщиком ресурсов Azure NetApp Files 

В этой статье описаны общие ошибки поставщика ресурсов Azure NetApp Files, их причины, решения и обходные пути (если таковая имеется).

## <a name="common-azure-netapp-files-resource-provider-errors"></a>Общие ошибки поставщика ресурсов файлов Azure NetApp

***BareMetalTenantId не может быть изменен.***  

Эта ошибка возникает при попытке обновить `BaremetalTenantId` или исправления объема и свойства имеет измененное значение.

* Причина.   
Вы пытаетесь обновить громкость, и `BaremetalTenantId` свойство имеет иное значение от значения, хранящегося в Azure.
* Решение.   
Не включайте `BaremetalTenantId` в патч и обновление (пут) запрос. Кроме того, `BaremetalTenantId` убедитесь, что это то же самое в запросе.

***ServiceLevel не может быть изменен.***  

Эта ошибка возникает, когда вы пытаетесь обновить или исправления пула емкости с другим уровнем обслуживания, когда пул емкости уже имеет объемы в нем.

* Причина.   
Вы пытаетесь обновить уровень обслуживания пула емкости, когда пул содержит тома.
* Решение.   
Удалите все тома из пула емкости, а затем измените уровень обслуживания.
* Обходное решение.   
Создайте другой пул емкости, а затем создайте объемы снова в новом пуле емкости.

***PoolId не может быть изменен***  

Эта ошибка возникает при попытке обновить или `PoolId` исправления пула емкости с измененным свойством.

* Причина.   
Вы пытаетесь обновить свойство пула `PoolId` емкости. Свойство `PoolId` является свойством только для чтения и не может быть изменено.
* Решение.   
Не включайте `PoolId` в патч и обновление (пут) запрос.  Кроме того, `PoolId` убедитесь, что это то же самое в запросе.

***CreationToken не может быть изменен.***

Эта ошибка возникает при попытке`CreationToken`изменить траекторию файла () после создания тома. Путь файла ()`CreationToken`должен быть установлен при создании тома, и он не может быть изменен позже.

* Причина.   
Вы пытаетесь изменить путь`CreationToken`файла () после создания тома, который не является поддерживаемой операцией. 
* Решение.   
Если изменение пути файла не требуется, затем рассмотрите возможность удаления параметра из запроса для отклонения сообщения об ошибке.
* Обходное решение.   
Если необходимо изменить траекторию`CreationToken`файла (), можно создать новый том с новым путем файла, а затем перенести данные в новый том.

***CreationToken должен быть не менее 16 символов в длину.***

Эта ошибка возникает,`CreationToken`когда путь файла () не соответствует требованию длины. Длина пути файла должна быть не менее одного символа в длину.

* Причина.   
Путь файла пуст.  При создании тома с помощью API требуется токен создания. Если вы используете портал Azure, путь файла генерируется автоматически.
* Решение.   
Введите по крайней мере`CreationToken`один символ в качестве пути файла ().

***Имя домена не может быть изменено.***

Эта ошибка возникает при попытке изменить доменное имя в Active Directory.

* Причина.   
Вы пытаетесь обновить свойство доменного имени.
* Решение.    
Нет. Вы не можете изменить доменное имя.
* Обходное решение.   
Удалите все тома с помощью конфигурации Active Directory. Затем удалите конфигурацию Active Directory и воссоздайте тома.

***Двойная ошибка значения для объекта ExportPolicy.Rules (ПравилоИндекс).***

Эта ошибка возникает, когда экспортная политика не определяется уникальным индексом. При определении экспортной политики все правила экспортной политики должны иметь уникальный индекс между 1 и 5.

* Причина.   
Определенная экспортная политика не соответствует требованиям правил экспортной политики. Вы должны иметь одно правило экспортной политики как минимум и пять правил экспортной политики на максимуме.
* Решение.   
Убедитесь, что индекс еще не используется и что он находится в диапазоне от 1 до 5.
* Обходное решение.   
Используйте другой индекс для правила, которое вы пытаетесь установить.

Эта ошибка отображается, когда другая обработка ошибок не смогла справиться с ошибкой при выполнении действия на ресурсе.   Она включает в себя текст 'Ошибка'. Может `{action}` быть любой`getting` `creating`из `updating`(, , или `deleting`).  Это `{resourceTypeName}` `resourceTypeName` (например, `netAppAccount`, `capacityPool` `volume`, и так далее).

* Причина.   
Эта ошибка является необработанным исключением, когда причина не известна.
* Решение.   
Свяжитесь с Центром поддержки Azure, чтобы сообщить подробную причину в журналах.
* Обходное решение.

***Имя пути файла может содержать только буквы, цифры и дефисы ("-"-") .***

Эта ошибка возникает, когда путь файла содержит неподдерживаемые символы, например, период (".), запятую (",), подчеркивать (""), или знак доллара ("$").

* Причина.   
Путь файла содержит неподдерживаемые символы, например, период (".), запятая (",), подчеркивание ("я"), или знак доллара ("$").
* Решение.   
Удалите символы, которые не являются алфавитные буквы, цифры или дефисы ("-") из файла пути вы ввели.
* Обходное решение.   
Вы можете заменить подчеркивание с дефисом или использовать капитализацию вместо пробелов, чтобы указать начало новых слов.  Например, использовать "NewVolume" вместо "нового объема".

***FileSystemId не может быть изменен.***

Эта ошибка возникает при `FileSystemId`попытке изменить.  Изменение `FileSystemdId` не является поддерживаемой операцией. 

* Причина.   
Идентификатор файловой системы устанавливается при создании громкости. `FileSystemId`не могут быть изменены впоследствии.
* Решение.   
Не включайте `FileSystemId` в патч и обновление (пут) запрос.  Кроме того, `FileSystemId` убедитесь, что это то же самое в запросе.

***ActiveDirectory с id: 'строка' не существует.***

Часть `{string}` — это значение, `ActiveDirectoryId` внесенное в свойство для подключения Active Directory.

* Причина.   
При создании учетной записи с конфигурацией Active Directory значение, которое `ActiveDirectoryId` должно быть пустым, должно быть пустым.
* Решение.   
Не включайте `ActiveDirectoryId` в запрос создания (пут).

***Недействительная апи-версия.***

Версия API либо не представлена, либо содержит недействительное значение.

* Причина.   
Значение параметра `api-version` запроса содержит недействительное значение.
* Решение.   
Используйте правильное значение версии API.  Поставщик ресурсов поддерживает множество версий API. Значение находится в формате yyyy-mm-dd.

***Недействительное значение «значение» было {1}получено для .***

Это сообщение указывает на ошибку `UnixReadOnly`в `UnixReadWrite` `Nfsv3`полях `Nfsv4`для `RuleIndex`, `AllowedClients`, , и .

* Причина.   
Запрос на ввод не удалось по крайней мере `RuleIndex` `AllowedClients`для `UnixReadOnly` `UnixReadWrite`одного из следующих полей: , , , `Nfsv`3, и `Nfsv4`.
* Решение.   
Убедитесь в том, чтобы установить все необходимые и неконфликтные параметры на командной строке. Например, нельзя `UnixReadOnly` устанавливать `UnixReadWrite` одновременно и параметры.
* Обходное решение.   
Смотрите решение выше.

Эта ошибка возникает из-за того, что внутренние записи используемых диапазонов IP имеют конфликт с недавно назначенным IP-адресом.

* Причина.   
IP-адрес, назначенный для создания объема, уже зарегистрирован.
Причиной могло быть ранее неудачное создание тома.

Свяжитесь с Центром поддержки Azure.

***Отсутствует значение для «имущества».***

Эта ошибка указывает на отсутствие требуемого свойства в запросе. Строка «свойство» содержит имя отсутствующего свойства.

* Причина.   
Запрос на ввод ную мгновения не прошел по крайней мере для одного из свойств.
* Решение.   
Убедитесь в том, чтобы установить все необходимые и неконфликтные свойства в запросе, особенно, свойство из сообщения об ошибке.

***MountTargets не могут быть изменены.***

Эта ошибка возникает, когда пользователь пытается обновить или исправления объема свойства MountTargets.

* Причина.   
Вы пытаетесь создать ресурс с именем, которое используется для существующего ресурса.
* Решение.   
Используйте уникальное имя при создании ресурса.

***Путь файла уже используется.***

Эта ошибка указывает на то, что путь файла для объема уже используется.

* Причина.   
Вы пытаетесь создать объем с траекторией файла, который является таким же, как существующий объем.
* Решение.   
Используйте уникальный путь файла при создании громкости.

***Имя слишком долго.***

Эта ошибка указывает на то, что имя ресурса не соответствует требованиям максимальной длины.

* Причина.   
Имя ресурса слишком длинное.
* Решение.   
Используйте более короткое имя для ресурса.

***Путь файла слишком длинный.***

Эта ошибка указывает на то, что путь файла для объема не соответствует требованиям максимальной длины.

* Причина.   
Путь файла объема слишком длинный.
* Решение.   
Используйте более короткий путь файла.

***Имя слишком короткое.***

Эта ошибка указывает на то, что имя ресурса не соответствует требованиям минимальной длины.

* Причина.   
Имя ресурса слишком короткое.
* Решение.   
Используйте более длинное имя для ресурса.

***Путь файла слишком короткий.***

Эта ошибка указывает на то, что путь файла объема не соответствует требованиям минимальной длины.

Путь файла громкости слишком короткий.

Увеличьте длину пути файла объема.

***API файлов Azure NetApp недосягаем.***

API Azure опирается на API файлов Нетповых Приложений Azure для управления объемами. Эта ошибка указывает на проблему с подключением API.

* Причина.   
Базовый API не отвечает, что приводит к внутренней ошибке. Эта ошибка, скорее всего, будет временной.
* Решение.   
Этот вопрос, скорее всего, будет временным. Запрос должен быть успешным через некоторое время.
* Обходное решение.   
Нет. Базовый API необходим для управления объемами.

***Нет идентификатор{0}результата операции найдено для ' '.***

Эта ошибка указывает на то, что внутренняя ошибка препятствует завершению операции.

* Причина.   
Произошла внутренняя ошибка, которая помешала завершению операции.
* Решение.   
Эта ошибка, скорее всего, будет временной. Подождите несколько минут и повторите попытку. Если проблема сохраняется, создайте билет для технической поддержки, чтобы изучить проблему.
* Обходное решение.   
Подождите несколько минут и проверьте, сохраняется ли проблема.

***Не разрешается смешивать типы протоколов CIFS и NFS***

Эта ошибка возникает при попытке создания тома и в свойствах объема есть типы протоколов CIFS (SMB) и NFS.

* Причина.   
В свойствах объема используются как типы протоколов CIFS (SMB) и NFS.
* Решение.   
Удалите один из типов протоколов.
* Обходное решение.   
Оставьте свойство типа протокола пустым или недействительным.

***Количество элементов: «Значение» для объекта: ExportPolicy.Rules (ПравилоИндекс) находится за пределами диапазона мин-макс.***

Эта ошибка возникает, когда правила экспортной политики не отвечают минимальным или максимальным требованиям диапазона. Если вы определяете экспортную политику, то она должна иметь как минимум одно правило экспортной политики и максимум пять правил экспортной политики.

* Причина.   
Определенная вами экспортная политика не соответствует требуемому диапазону.
* Решение.   
Убедитесь, что индекс еще не используется и находится в диапазоне от 1 до 5.
* Обходное решение.   
Использование экспортной политики на объемах не является обязательным. Вы можете полностью пропустить экспортную политику, если вам не нужно использовать правила экспортной политики.

***Допускается только один активный каталог***

Эта ошибка возникает при попытке создать конфигурацию Active Directory, и она уже существует для подписки в регионе. Ошибка также может произойти при попытке создать несколько конфигураций Active Directory.

* Причина.   
Вы пытаетесь создать (не обновлять) активный каталог, но один уже существует.
* Решение.   
Если конфигурация Active Directory не используется, можно сначала удалить существующую конфигурацию, а затем повторить операцию создания.
* Обходное решение.   
Нет. Допускается только один Активный каталог.

***Операция «Операция» не поддерживается.***

Эта ошибка указывает на то, что операция недоступна для активной подписки или ресурса.

* Причина.   
Операция недоступна для подписки или ресурса.
* Решение.   
Убедитесь, что операция введена правильно и что она доступна для ресурса и подписки, которые вы используете.

***OwnerId не может быть изменен***

This error occurs when you try to change the OwnerId property of the volume. Изменение OwnerId не является поддерживаемой операцией. 

* Причина.   
Свойство `OwnerId` устанавливается при создании тома. Свойство не может быть изменено впоследствии.
* Решение.   
Не включайте `OwnerId` в патч и обновление (пут) запрос. Кроме того, убедитесь, что `OwnerId` это то же самое в запросе.

***Родительский пул не найден***

Эта ошибка возникает при попытке создать объем, и пул емкости, в котором вы создаете, не найден.

* Причина.   
Бассейн емкости, в котором создается объем, не найден.
* Решение.   
Скорее всего, пул не был полностью создан или уже удален на момент создания тома.

***Операция патча не поддерживается для этого типа ресурса.***

Эта ошибка возникает при попытке изменить цель или моментальный снимок крепления.

* Причина.   
Цель крепления определяется, когда она создается, и она не может быть изменена впоследствии.
Снимки не содержат свойств, которые могут быть изменены.
* Решение.   
Нет. Эти ресурсы не имеют свойств, которые могут быть изменены.

***Размер пула слишком мал для общего размера объема.***

Эта ошибка возникает при обновлении размера пула емкости, `usedBytes` и размер меньше, чем общая стоимость всех томов в этом пуле емкости.  Эта ошибка также может возникнуть при создании нового объема или изменения размера существующего объема, а новый размер громкости превышает свободное пространство в пуле емкости.

* Причина.   
Вы пытаетесь обновить пул емкости до меньшего размера, чем usedBytes во всех объемах в пуле емкости.  Или вы пытаетесь создать объем, который больше свободного пространства в пуле вместимости.  Кроме того, вы пытаетесь изменить размер громкости и новый размер превышает свободное пространство в пуле емкости.
* Решение.   
Установите размер пула емкости на более большое значение или создайте меньший объем для объема.
* Обходное решение.   
Удалите достаточное количество, чтобы размер пула емкости был обновлен до этого размера.

***Свойство: Местоположение для снимка должно быть таким же, как том***

Эта ошибка возникает при создании снимка с расположением, не в томом, который принадлежит моментальнику.

* Причина.   
Недействительное значение в свойстве Место для снимка.
* Решение.   
Установите действительную строку в свойстве Расположение.

***Имя «resourceType» должно быть таким же, как иимя идентификатора ресурса.***

Эта ошибка возникает при создании ресурса, и вы заполняете свойство `resourceId`имени с другим значением, чем свойство имени .

* Причина.   
Недействительное значение в свойстве имени при создании ресурса.
* Решение.
