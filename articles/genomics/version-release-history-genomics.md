# <a name="version-release-history"></a>Журнал выпуска версий
Команда Microsoft Genomics регулярно выпускает обновления для клиента Python Microsoft Genomics, связанные с исправлениями и новыми функциями. 

## <a name="latest-release"></a>Последний выпуск
Текущий клиент Python — это версия 0.9.0. Он был выпущен 6 февраля 2019 года и поддерживает рабочие процессы с GATK 3.5 и GATK4. Он поддерживает выход gVCF и может принять дополнительный аргумент для сжатия вывода.


## <a name="release-history"></a>История выпусков 
Новые версии клиента Microsoft Genomics Python выпускаются примерно раз в год. По мере выпуска новых версий клиента Python Microsoft Genomics применяются обновления, связанные с исправлениями и новыми функциями. После выпуска новых версий использование предыдущих поддерживается как минимум 90 дней. Если предыдущие версии больше не поддерживаются, на этой странице появится уведомление. 

### <a name="version-090"></a>Версия 0.9.0
Версия 0.9.0 включает поддержку эхапрессии. Это эквивалентно `-bgzip` запуску, за `-tabix` которым следует выход vcf или gvcf. Для получения дополнительной информации, см [Часто задаваемые вопросы](frequently-asked-questions-genomics.md). 

### <a name="version-081"></a>Версия 0.8.1
Версия 0.8.1 с исправлениями незначительных ошибок.  

### <a name="version-080"></a>Версия 0.8.0
Версии 0.8.0 включает поддержку GATK4 и вывода данных gVCFs.  

### <a name="version-074"></a>Версия 0.7.4
Версия 0.7.4 включает поддержку приема маркеров SAS вместо ключей учетной записи во входных данных `config.txt`. Дополнительные сведения см. в руководстве по [использованию входных маркеров SAS](quickstart-input-sas.md). 

### <a name="version-073"></a>Версия 0.7.3
Версия 0.7.3 с исправлениями незначительных ошибок.

### <a name="version-072"></a>Версия 0.7.2
Версия 0.7.2 является исходной. Выпущена 1 ноября 2017 г.
