# <a name="change-to-full-screen-view-for-a-vm-session-azure-bastion-preview"></a>Изменить представление сеанс виртуальной машины во весь экран: Бастиона Azure (Предварительная версия)

Эта статья поможет вам изменить представление виртуальной машины в полноэкранный режим и обратно в браузере. Перед началом работы с виртуальной Машины, убедитесь, что выполнены шаги, чтобы [создать узел-Бастион](bastion-create-host-portal.md). Затем подключитесь к виртуальной Машине, которая требуется для работы с помощью [RDP](bastion-connect-vm-rdp.md) или [SSH](bastion-connect-vm-ssh.md).

> [!IMPORTANT]
> Эта общедоступная предварительная версия предоставляется без соглашения об уровне обслуживания и не должна использоваться для производственных рабочих нагрузок. Некоторые функции могут не поддерживаться, иметь ограничения и быть доступными не во всех расположениях Azure. См. [дополнительные условия использования для предварительных версий Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).
>

## <a name="launch-the-clipboard-tool"></a>Запустить средство буфер обмена

Во время удаленного сеанса запустите палитры средство доступа Бастиона буфер обмена, выбрав две стрелки, которые находятся в центре левой сеанса.

![средства](./media/bastion-vm-manage/left.png)

## <a name="select-full-screen"></a>Выберите во весь экран

Выберите **полноэкранном режиме** кнопку для переключения на возможности полноэкранной работы сеанса. После переключения, сеанс будет повторная инициализация на весь экран.

![во весь экран](./media/bastion-vm-manage/full-screen.png)
