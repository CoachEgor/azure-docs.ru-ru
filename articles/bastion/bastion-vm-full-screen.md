# <a name="change-to-full-screen-view-for-a-vm-session-azure-bastion"></a>Изменение полноэкранного представления для сеанса vm: Azure Bastion

Эта статья поможет вам изменить представление виртуальной машины на полный экран и обратно в вашем браузере. Прежде чем работать с VM, убедитесь, что вы следовали шагам по [созданию узла Bastion.](bastion-create-host-portal.md) Затем подключитесь к VM, с помощью которого вы хотите работать с помощью [RDP](bastion-connect-vm-rdp.md) или [SSH.](bastion-connect-vm-ssh.md)

## <a name="launch-the-clipboard-tool"></a>Запуск инструмента буфера обмена

Во время удаленной сессии запустите палитру инструмента доступа к инструменту буфера обмена Bastion, выбрав две стрелки, расположенные в левом центре сеанса.

![средства](./media/bastion-vm-manage/left.png)

## <a name="select-full-screen"></a>Выберите полный экран

Выберите кнопку **Fullscreen,** чтобы переключить сеанс на полный экран. Как только вы переключитесь, сеанс будет переизменяться на полный экран.

![полноэкранный режим](./media/bastion-vm-manage/full-screen.png)
