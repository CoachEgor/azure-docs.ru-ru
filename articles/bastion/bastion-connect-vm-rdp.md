# <a name="connect-to-a-windows-virtual-machine-using-azure-bastion"></a>Подключение к виртуальной машине Windows с помощью Azure бастиона

В этой статье показано, как безопасно и эффективно использовать RDP для виртуальных машин Windows в виртуальной сети Azure с помощью Azure бастиона. Подключиться к виртуальной машине можно непосредственно на портале Azure. При использовании службы "Бастион Azure" на виртуальной машине не нужно устанавливать клиенты, агенты или дополнительное программное обеспечение. Дополнительные сведения об Azure бастиона см. в [обзоре](bastion-overview.md).

## <a name="before-you-begin"></a>Перед началом работы

Убедитесь, что вы настроили узел бастиона Azure для виртуальной сети, в которой находится виртуальная машина. Дополнительные сведения см. в статье [Создание узла бастиона Azure](bastion-create-host-portal.md). После подготовки и развертывания службы бастиона в виртуальной сети ее можно использовать для подключения к любой виртуальной машине в этой виртуальной сети.

Бастиона предполагает, что вы используете RDP для подключения к виртуальной машине Windows и SSH для подключения к виртуальным машинам Linux. Сведения о подключении к виртуальной машине Linux см. в статье [Подключение к виртуальной машине](bastion-connect-vm-ssh.md)Linux.

### <a name="required-roles"></a>Обязательные роли
Для подключения нужны такие роли:

* роль читателя на виртуальной машине;
* роль читателя на сетевом адаптере с частным IP-адресом для виртуальной машины;
* роль читателя в ресурсе Бастиона Azure.

### <a name="ports"></a>порты;

Чтобы подключиться к виртуальной машине Windows по протоколу RDP, на виртуальной машине Windows должны быть открыты следующие порты:

* Входящие порты: RDP (3389)

## <a name="rdp"></a>Подключение по протоколу RDP

1. Откройте [портал Azure](https://portal.azure.com). Перейдите к виртуальной машине, к которой необходимо подключиться, и нажмите кнопку **подключить**. При использовании RDP-подключения виртуальная машина должна быть виртуальной машиной Windows.

   ![Подключение ВМ](./media/bastion-connect-vm-rdp/connect.png)
1. После нажатия кнопки подключить появится боковая панель с тремя вкладками: RDP, SSH и бастиона. Если бастиона был подготовлен для виртуальной сети, вкладка бастиона активна по умолчанию. Если вы не подготовили бастиона для виртуальной сети, можно щелкнуть ссылку, чтобы настроить бастиона. Инструкции по настройке см. в разделе [Configure бастиона](bastion-create-host-portal.md).

   ![Подключение ВМ](./media/bastion-connect-vm-rdp/bastion.png)
1. На вкладке бастиона (имя пользователя и пароль) для виртуальной машины и нажмите кнопку **подключить**. Подключение RDP к этой виртуальной машине через бастиона будет открываться непосредственно в портал Azure (через HTML5) с помощью порта 443 и службы бастиона.

   ![Подключение ВМ](./media/bastion-connect-vm-rdp/443rdp.png)
