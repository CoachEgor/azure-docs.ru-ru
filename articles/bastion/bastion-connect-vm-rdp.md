# <a name="connect-to-a-windows-virtual-machine-using-azure-bastion"></a>Подключение к виртуальной машине Windows с помощью Azure Bastion

Используя Azure Bastion, вы можете безопасно и беспрепятственно подключаться к своим виртуальным машинам через SSL непосредственно на портале Azure. При использовании Azure Bastion ваши вс-минене не требуют клиента, агента или дополнительного программного обеспечения. В этой статье показано, как подключиться к вашим windows VMs. Для получения информации о подключении к Linux VM, [см. Подключение к VM с помощью Azure Bastion - Linux](bastion-connect-vm-ssh.md).

Azure Bastion обеспечивает безопасное подключение ко всем виртуальным ввиртуальной сети, в которой он подготовлен. Бастион Azure позволяет защитить порты RDP и SSH ваших виртуальных машин от внешних проникновений, обеспечивая при этом безопасный доступ с использованием этих протоколов. Для получения дополнительной информации смотрите [обзор](bastion-overview.md).

## <a name="before-you-begin"></a>Перед началом

Убедитесь, что вы создали хост Azure Bastion для виртуальной сети, в которой находится VM. После того, как служба Bastion будет подготовлена и развернута в виртуальной сети, вы можете использовать ее для подключения к любому VM в виртуальной сети. Для настройки узла Azure Bastion [см.](bastion-create-host-portal.md)

### <a name="required-roles"></a>Обязательные роли

Для подключения требуются следующие роли:

* роль читателя на виртуальной машине;
* роль читателя на сетевом адаптере с частным IP-адресом для виртуальной машины;
* роль читателя в ресурсе Бастиона Azure.

### <a name="ports"></a>порты;

Чтобы подключиться к Windows VM, необходимо открыть следующие порты на Windows VM:

* Входящие порты: RDP (3389)

## <a name="connect"></a><a name="rdp"></a>Подключения

1. Откройте [портал Azure](https://portal.azure.com). Перейдите к виртуальной машине, к которую вы хотите подключиться, затем нажмите **Connect** и выберите **Бастион** из выпадения.

   ![Подключение VM](./media/bastion-connect-vm-rdp/connect.png)
1. После нажатия На Бастион появляется боковая панель, которая имеет три вкладки - RDP, SSH и Bastion. Если Бастион был подготовлен для виртуальной сети, вкладка Бастион активна по умолчанию. Если вы не предусмотрели Бастион для виртуальной сети, вы можете нажать на ссылку, чтобы настроить Бастион. Для инструкций по настройке [см.](bastion-create-host-portal.md)

   ![бастион вкладка](./media/bastion-connect-vm-rdp/bastion.png)
1. На вкладке Бастион ввейте имя пользователя и пароль для виртуальной машины, а затем нажмите **Connect.** Подключение RDP к этой виртуальной машине через Бастион откроется непосредственно на портале Azure (над HTML5) с использованием порта 443 и службы Бастиона.

   ![Подключение RDP](./media/bastion-connect-vm-rdp/443rdp.png)
