# <a name="session-monitoring-and-management-for-azure-bastion"></a>Мониторинг и управление сеансами для Azure бастиона

После подготовки и развертывания службы бастиона в виртуальной сети ее можно использовать для беспроблемного подключения к любой виртуальной машине в этой виртуальной сети. Когда пользователи подключаются к рабочим нагрузкам, Azure бастиона можно использовать для отслеживания удаленных сеансов и выполнения действий по быстрому управлению. Мониторинг сеансов Azure бастиона позволяет просматривать, к каким пользователям подключены виртуальные машины. В нем отображается IP-адрес, с которого пользователь подключился, как долго они были подключены и когда они подключены. Процесс управления сеансами позволяет выбрать текущий сеанс и принудительно отключить или удалить сеанс, чтобы отключить пользователя от текущего сеанса.

## <a name="monitor-remote-sessions"></a><a name="monitor"></a>Мониторинг удаленных сеансов

1. В [портал Azure](https://portal.azure.com)перейдите к ресурсу Azure бастиона и выберите **сеансы** на странице бастиона Azure.

   ![сеансы](./media/session-monitoring/sessions.png)
2. На странице **сеансы** можно просмотреть текущие удаленные сеансы с правой стороны.

   ![Просмотреть сеанс](./media/session-monitoring/view-session.png)
3. Выберите **Обновить** , чтобы просмотреть обновленный список удаленных сеансов. При выборе обновления Azure бастиона выберет последние данные мониторинга и обновит их на портале.

   ![refresh](./media/session-monitoring/refresh.png)

>[!IMPORTANT]
> Включите порт 4443 для входящего трафика от диспетчера шлюзов, чтобы мониторинг сеансов работал.
>

## <a name="delete-or-force-disconnect-an-ongoing-remote-session"></a><a name="view"></a>Удаление или принудительное отключение текущего удаленного сеанса

Вы можете выбрать набор сеансов и принудительно отключить их. Ниже описано, как удалить удаленные сеансы.

1. Перейдите к ресурсу Azure бастиона и выберите **сеансы** на странице бастиона Azure.

   ![navigate](./media/session-monitoring/navigate.png)
2. После выбора сеансов отобразится список удаленных сеансов.

   ![Список сеансов](./media/session-monitoring/list.png)
3. Выберите конкретный удаленный сеанс, затем на правой стороне строки сеанса выберите три эллипса, а затем щелкните **Удалить**.

   !["Удалить"](./media/session-monitoring/delete.png)
4. При выборе удалить удаленный сеанс будет отключен, и пользователю будет отображаться сообщение "вы отключили" в удаленном сеансе.

   ![отключение](./media/session-monitoring/disconnect.png)
