# <a name="session-monitoring-and-management-for-azure-bastion"></a>Мониторинг и управление сеансами для Azure Bastion

После того, как служба Bastion будет подготовлена и развернута в вашей виртуальной сети, вы можете использовать ее для беспрепятственного подключения к любому VM в этой виртуальной сети. По мере подключения пользователей к рабочим нагрузкам Azure Bastion может использоваться для мониторинга удаленных сеансов и быстрых действий управления. Мониторинг сеансов Azure Bastion позволяет просматривать, к каким пользователям подключены vMs. Он показывает IP, с которого подключен пользователь, как долго он был подключен и когда он подключался. Опыт управления сеансом позволяет выбрать текущую сессию и отключить или удалить сеанс, чтобы отключить пользователя от текущей сессии.

## <a name="monitor-remote-sessions"></a><a name="monitor"></a>Мониторинг удаленных сеансов

1. На [портале Azure](https://portal.azure.com)перейдите на ресурс Azure Bastion и выберите сеансы со **страницы** Azure Bastion.

   ![сеансы](./media/session-monitoring/sessions.png)
2. На странице **Сессии** можно увидеть текущие удаленные сеансы с правой стороны.

   ![сеанс просмотра](./media/session-monitoring/view-session.png)
3. Выберите **Обновление,** чтобы просмотреть обновленный список удаленных сеансов. При выборе обновления Azure Bastion будет получать последнюю информацию о мониторинге и обновлять ее на портале.

   ![refresh](./media/session-monitoring/refresh.png)

>[!IMPORTANT]
> Включите порт 4443 для входящего трафика от Gateway Manager для мониторинга сеансов для работы.
>

## <a name="delete-or-force-disconnect-an-ongoing-remote-session"></a><a name="view"></a>Удалить или отключить текущую удаленную сессию

Можно выбрать набор сеансов (ы) и отключить их. Следующие шаги показывают, как удалить удаленные сеансы:

1. Перейдите на ресурс Azure Bastion и выберите сеансы со **страницы** Azure Bastion.

   ![navigate](./media/session-monitoring/navigate.png)
2. После выбора сеансов вы увидите список удаленных сеансов.

   ![список сессий](./media/session-monitoring/list.png)
3. Выберите определенный удаленный сеанс, затем выберите три эллипса в правом конце строки сеанса, а затем выберите **Удалить**.

   ![удалить](./media/session-monitoring/delete.png)
4. При выборе удаления удаленная сессия будет отключена, и пользователю будет показано сообщение "Вы были отключены" в удаленном сеансе.

   ![отключение](./media/session-monitoring/disconnect.png)
