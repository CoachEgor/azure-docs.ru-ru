# <a name="copy-and-paste-to-a-virtual-machine-azure-bastion"></a>Копировать и вставить в виртуальную машину: Azure Bastion

Эта статья поможет вам скопировать и вставить текст в и из виртуальных машин при использовании Azure Bastion. Прежде чем работать с VM, убедитесь, что вы следовали шагам по [созданию узла Bastion.](bastion-create-host-portal.md) Затем подключитесь к VM, с помощью которого вы хотите работать с помощью [RDP](bastion-connect-vm-rdp.md) или [SSH.](bastion-connect-vm-ssh.md)

Для браузеров, поддерживающих расширенный доступ к API Clipboard, можно скопировать и вставить текст между локальным устройством и удаленной сессией так же, как копировать и вставить между приложениями на локальном устройстве. Для других браузеров можно использовать палитру инструментов доступа к инструменту доступа к буферу обмена Bastion.

   ![Разрешить буфер обмена](./media/bastion-vm-manage/allow.png)

Поддерживается только текстовая копия/вставить. Для прямой копирования и вставки, ваш браузер может побудить вас для буфера обмена доступ, когда сеанс Бастион в настоящее время инициализации. **Разрешить** веб-страницу для доступа к буферу обмена.

## <a name="copy-to-a-remote-session"></a><a name="to"></a>Копирование удаленного сеанса

После подключения к виртуальной машине с помощью [портала Azure ](https://portal.azure.com)выполните следующие шаги:

1. Копирование текста/содержимого с локального устройства в локальный буфер обмена.
1. Во время удаленной сессии запустите палитру инструментов доступа к инструменту доступа буфера обмена Bastion, выбрав две стрелки. Стрелки расположены в левом центре сеанса.

   ![инструмент палитра](./media/bastion-vm-manage/left.png)

   ![буфер обмена](./media/bastion-vm-manage/clipboard.png)
1. Как правило, скопированный текст автоматически отображается на палитре пасты копии Bastion. Если ваш текст не существует, а затем вставьте текст в текстовой области на палитре.
1. После того, как текст находится в текстовой области, вы можете вставить его в удаленном сеансе.

   ![вставка](./media/bastion-vm-manage/local.png)

## <a name="copy-from-a-remote-session"></a><a name="from"></a>Копия из удаленного сеанса

После подключения к виртуальной машине с помощью [портала Azure ](https://portal.azure.com)выполните следующие шаги:

1. Копирование текста/содержимого из удаленного сеанса в удаленный буфер обмена (с помощью Ctrl-C).

   ![инструмент палитра](./media/bastion-vm-manage/remote.png)
1. Во время удаленной сессии запустите палитру инструментов доступа к инструменту доступа буфера обмена Bastion, выбрав две стрелки. Стрелки расположены в левом центре сеанса.

   ![буфер обмена](./media/bastion-vm-manage/clipboard2.png)
1. Как правило, скопированный текст автоматически отображается на палитре пасты копии Bastion. Если ваш текст не существует, а затем вставьте текст в текстовой области на палитре.
1. После того, как текст находится в текстовой области, вы можете вставить его на локальное устройство.

   ![вставка](./media/bastion-vm-manage/local2.png)
