# <a name="copy-and-paste-to-a-virtual-machine-azure-bastion-preview"></a>Скопируйте и вставьте в виртуальную машину: Бастиона Azure (Предварительная версия)

Эта статья поможет вам скопировать и вставить текст из виртуальных машин, а при использовании Azure Бастиона. Перед началом работы с виртуальной Машины, убедитесь, что выполнены шаги, чтобы [создать узел-Бастион](bastion-create-host-portal.md). Затем подключитесь к виртуальной Машине, которая требуется для работы с помощью [RDP](bastion-connect-vm-rdp.md) или [SSH](bastion-connect-vm-ssh.md).

> [!IMPORTANT]
> Эта общедоступная предварительная версия предоставляется без соглашения об уровне обслуживания и не должна использоваться для производственных рабочих нагрузок. Некоторые функции могут не поддерживаться, иметь ограничения и быть доступными не во всех расположениях Azure. См. [дополнительные условия использования для предварительных версий Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).
>

Для обозревателей с поддержкой расширенного доступа API буфера обмена можно скопировать и вставить текст между устройством локальный и удаленный сеанс, так же, копирования и вставки между приложениями на локальном устройстве. Для других обозревателей можно использовать палитры средство доступа Бастиона буфер обмена.

  ![Разрешить буфер обмена](./media/bastion-vm-manage/allow.png)

Поддерживается только текст копирования и вставки. Для прямого копирования и вставки браузер может запросить доступ к буферу обмена во время инициализации сеанса Бастиона. **Разрешить** веб-страницы для доступа к в буфер обмена.

## <a name="to"></a>Скопируйте удаленный сеанс

После подключения к виртуальной машине, используя [портала Azure](https://aka.ms/BastionHost) Бастиона предварительной версии:

1. Скопируйте текст/на локальном устройстве в локальный буфер обмена.
1. Во время удаленного сеанса запустите палитры средство доступа Бастиона буфер обмена, выбрав две стрелки. Стрелки, находятся в центре левой сеанса.

    ![палитра инструментов](./media/bastion-vm-manage/left.png)

    ![Буфер обмена](./media/bastion-vm-manage/clipboard.png)

1. Как правило скопированный текст автоматически отображает палитру вставить копию Бастиона. Если текст не установлен, затем вставьте текст в текстовой области в палитре.
1. Когда текст в текстовой области, можно было вставить его в удаленный сеанс.

    ![Вставить](./media/bastion-vm-manage/local.png)

## <a name="from"></a>Скопируйте из удаленного сеанса

После подключения к виртуальной машине, используя [портала Azure](https://aka.ms/BastionHost) Бастиона предварительной версии:

1. Скопируйте текст из удаленного сеанса в удаленном буфер обмена (используя Ctrl-C).

    ![палитра инструментов](./media/bastion-vm-manage/remote.png)

1. Во время удаленного сеанса запустите палитры средство доступа Бастиона буфер обмена, выбрав две стрелки. Стрелки, находятся в центре левой сеанса.

    ![Буфер обмена](./media/bastion-vm-manage/clipboard2.png)

1. Как правило скопированный текст автоматически отображает палитру вставить копию Бастиона. Если текст не установлен, затем вставьте текст в текстовой области в палитре.
1. Когда текст в текстовой области, можно было вставить его на локальном устройстве.

    ![Вставить](./media/bastion-vm-manage/local2.png)
