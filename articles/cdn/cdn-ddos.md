# <a name="azure-cdn-ddos-protection"></a>Лазурная защита DDoS CDN

Сеть доставки контента обеспечивает защиту DDoS по дизайну. В дополнение к глобальной способности поглощать объемные атаки, Azure CDN имеет дополнительную DDoS-защиту, описанную ниже, без каких-либо дополнительных затрат.

## <a name="azure-cdn-from-microsoft"></a>Azure CDN от Майкрософт

Azure CDN от Microsoft защищен [аимурной DDoS Azure.](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview) Он интегрирован в Azure CDN с платформы Microsoft по умолчанию и без дополнительных затрат. Полный масштаб и емкость Azure CDN из глобально развернутой сети Майкрософт обеспечивает защиту от атак общего сетевого уровня посредством постоянного мониторинга трафика и смягчения в режиме реального времени. Базовая Защита От DDoS также защищает от наиболее распространенных, часто встречающихся наводнений ВНес-Кивс и объемных атак 3 и 4-го слоя, нацеленных на конечные точки CDN. Эта служба также имеет доказанный послужной список в защите корпоративных и потребительских служб корпорации Майкрософт от крупномасштабных атак.

## <a name="azure-cdn-from-verizon"></a>Azure CDN от Verizon

Azure CDN от Verizon защищен ареной собственной платформы для снижения DDoS компании Verzion. Он интегрирован в Azure CDN от Verizon по умолчанию и без каких-либо дополнительных затрат. Он обеспечивает базовую защиту от наиболее распространенных, часто встречающихся слоев 7 DNS-наводнений и объемных атак 3 и 4-го уровня, нацеленных на конечные точки CDN.

## <a name="azure-cdn-from-akamai"></a>Azure CDN от Akamai

Azure CDN от Akamai защищен анойдной платформой для снижения DDoS akamai. Он интегрирован в Azure CDN от Akamai по умолчанию и без дополнительных затрат. Он обеспечивает базовую защиту от наиболее распространенных, часто встречающихся слоев 7 DNS-наводнений и объемных атак 3 и 4-го уровня, нацеленных на конечные точки CDN.
