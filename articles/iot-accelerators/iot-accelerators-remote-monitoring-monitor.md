# <a name="tutorial-monitor-your-iot-devices"></a>Руководство. Мониторинг устройств Интернета вещей

В этом руководстве показано, как отслеживать подключенные устройства Интернета вещей с помощью акселератора решения для удаленного мониторинга. Вы можете просматривать данные телеметрии, сведения об устройстве, оповещения и ключевые показатели эффективности на панели мониторинга решения.

В этом руководстве используется два имитированных устройства для грузовых автомобилей, которые отправляют местоположение, скорость и температуру телеметрии. Грузовиками управляет организация под названием Contoso, и они подключены к акселератору решения для удаленного мониторинга. Оператору Contoso необходимо отслеживать расположение и работу одного из грузовиков (грузовика-02) на местах.

Изучив это руководство, вы:

>[!div class="checklist"]
> * фильтрация устройств на панели мониторинга;
> * просмотр телеметрии в реальном времени;
> * просмотр сведений об устройстве;
> * Просмотр оповещений устройств
> * Просмотр ключевых показателей эффективности системы

Если у вас еще нет подписки Azure, [создайте бесплатную учетную запись Azure](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), прежде чем начинать работу.

## <a name="choose-the-devices-to-display"></a>Выбор устройств для отображения

Чтобы выбрать, какие из подключенных устройств будут отображаться на странице **Панель мониторинга**, используйте фильтры. Для отображения только устройств **Грузовик**, выберите встроенный фильтр **Грузовик** в раскрывающемся списке фильтров:

[![Фильтр для грузовиков на панели мониторинга](./media/iot-accelerators-remote-monitoring-monitor/dashboardtruckfilter-inline.png)](./media/iot-accelerators-remote-monitoring-monitor/dashboardtruckfilter-expanded.png#lightbox)

После применения фильтра на карте и на панели телеметрии отображаются только те устройства, которые соответствуют условиям фильтрации. Можно увидеть, что к акселератору решений подключены два грузовика, включая грузовик-02.

[![На карте отображаются только грузовики, соответствующие фильтру](./media/iot-accelerators-remote-monitoring-monitor/dashboardtruckmap-inline.png)](./media/iot-accelerators-remote-monitoring-monitor/dashboardtruckmap-expanded.png#lightbox)

Для создания, изменения и удаления фильтров щелкните раздел **Управление группами устройств**.

## <a name="view-real-time-telemetry"></a>просмотр телеметрии в реальном времени;

На диаграмме на странице **Панель мониторинга** в акселераторе решений представлены данные телеметрии в режиме реального времени. В верхней части диаграммы данных телеметрии показаны доступные типы данных телеметрии для устройств, включая грузовик-02, выбранные с помощью текущего фильтра. По умолчанию диаграммы отображают широты грузовиков, а грузовик-02 является стационарным.

[![Типы телеметрии грузовика](./media/iot-accelerators-remote-monitoring-monitor/dashboardtelemetryview-inline.png)](./media/iot-accelerators-remote-monitoring-monitor/dashboardtelemetryview-expanded.png#lightbox)

Чтобы просмотреть данные телеметрии температуры грузовиков, щелкните **Температура**. Можно увидеть, как значение температуры для грузовика-02 изменялось за последний час.

[![Данные телеметрии температуры грузовика](./media/iot-accelerators-remote-monitoring-monitor/dashboardselecttelemetry-inline.png)](./media/iot-accelerators-remote-monitoring-monitor/dashboardselecttelemetry-expanded.png#lightbox)

## <a name="view-the-map"></a>Просмотр карты

На карте отображаются сведения о моделируемых грузовиках, выбранных с помощью текущего фильтра. Для настройки детализации расположений можно сдвигать и масштабировать карту. Цвет значка устройства на карте указывает на наличие активных **оповещений** (темно-синий) или **предупреждений** (красный) для устройства. Сводка о количестве **оповещений** и **предупреждений** отображается слева от карты.

Можно сдвигать и масштабировать карту, чтобы найти грузовик-02, а затем выбрать его на карте, чтобы просмотреть сведения о нем. Затем щелкните метку устройства, чтобы открыть панель **Сведения об устройстве**. Сведения об устройстве включают:

* Последние значения телеметрии.
* Методы, которые поддерживает устройство.
* Свойства устройства

[![Просмотр сведений об устройстве на панели мониторинга](./media/iot-accelerators-remote-monitoring-monitor/dashboarddevicedetail-inline.png)](./media/iot-accelerators-remote-monitoring-monitor/dashboarddevicedetail-expanded.png#lightbox)

## <a name="view-alerts"></a>Просмотр оповещений

В области **Оповещения** представлены подробные сведения о последних оповещениях устройства. Оповещения от грузовика-02 показывают более высокую температуру, чем нормальная температура груза.

[![Просмотр оповещений устройства на панели мониторинга](./media/iot-accelerators-remote-monitoring-monitor/dashboardsystemalarms-inline.png)](./media/iot-accelerators-remote-monitoring-monitor/dashboardsystemalarms-expanded.png#lightbox)

Вы можете изменить интервал времени оповещений с помощью фильтра. На панели по умолчанию отображаются оповещения за последний час.

## <a name="view-the-system-kpis"></a>Просмотр ключевых показателей эффективности системы

На панели **Аналитика** на странице **Панель мониторинга** отображаются ключевые показатели эффективности системы, вычисленные акселератором решений:

[![Ключевые показатели эффективности на панели мониторинга](./media/iot-accelerators-remote-monitoring-monitor/dashboardkpis-inline.png)](./media/iot-accelerators-remote-monitoring-monitor/dashboardkpis-expanded.png#lightbox)

На панели мониторинга отображаются три ключевых показателя эффективности для оповещений, определяемых фильтрами текущего устройства и временного интервала:

* Количество активных оповещений для правил, которые вызвали наибольшее число предупреждений.
* Доля оповещений по типу устройства.
* Процент оповещений, которые являются критическими.

Все оповещения для грузовика-02 являются предупреждениями о превышении нормальной температуры груза.

На агрегирование ключевых показателей эффективности влияют те же фильтры, которые устанавливают временной интервал для оповещений и определяют отображаемые устройства. По умолчанию на панели отображаются КПЭ, собранные за последний час.
