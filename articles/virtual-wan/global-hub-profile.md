# <a name="download-a-global-or-hub-based-profile-for-user-vpn-clients"></a>Скачать глобальный или концентраторный профиль для клиентов VPN пользователя

Azure Virtual WAN предлагает два типа подключения для удаленных пользователей: Глобальный и Концентратор. Используйте следующие разделы, чтобы узнать и загрузить профиль.

## <a name="global-profile"></a>Глобальный профиль

Профиль указывает на балансировщик нагрузки, который включает в себя все активные концентраторы VPN пользователя. Пользователь направляется в концентратор, который находится ближе всего к географическому местоположению пользователя. Этот тип подключения полезен, когда пользователи часто путешествуют по разным местам. Чтобы загрузить **глобальный** профиль:

1. Перейдите к виртуальной глобальной сети.
2. Нажмите **пользователь VPN конфигурации**.
3. Выделите конфигурацию, для которой вы хотите скачать профиль.
4. Нажмите **Скачать виртуальный профиль VPN пользователя WAN**.

   ![Глобальный профиль](./media/global-hub-profile/global1.png)

## <a name="hub-based-profile"></a>Профиль на основе концентратора

Профиль указывает на один концентратор. Пользователь может подключиться только к определенному концентратору, используя этот профиль. Чтобы загрузить профиль **на базе концентратора:**

1. Перейдите к виртуальной глобальной сети.
2. Нажмите **концентратор** на странице Обзор.

    ![Профиль концентратора 1](./media/global-hub-profile/hub1.png)
3. Нажмите **пользователь VPN (точка на сайт)**.
4. Нажмите **Скачать виртуальный профиль пользователя концентраторvpn**.

   ![Профиль концентратора 2](./media/global-hub-profile/hub2.png)
5. Проверьте **EAPTLS**.
6. Нажмите **Создать и скачать профиль**.

   ![Профиль концентратора 3](./media/global-hub-profile/download.png)
