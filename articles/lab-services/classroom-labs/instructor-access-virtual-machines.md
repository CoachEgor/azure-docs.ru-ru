# <a name="access-virtual-machines-as-a-student-from-the-instructor-view"></a>Доступ к виртуальным машинам в качестве учащегося из представления инструкторов
В этой статье показано, как преподаватели могут получить доступ к своим виртуальным машинам для классов, которые они применяют как студенты. 

Ниже приведен сценарий, в котором эта функция поможет. Помощник по преподаванию — это инструктор для одного класса, но учащийся в других классах. И обучающий инструктор хочет просматривать виртуальные машины учащихся и получать доступ к ним из представления инструкторов, в котором отображаются лабораторные работы, которыми они владеют. 

## <a name="access-vms-from-instructor-view"></a>Доступ к виртуальным машинам из представления преподавателя

1. Войдите на [веб-сайт служб лаборатории Azure](https://labs.azure.com). Вы видите свои собственные лаборатории. Это могут быть лабораторные работы, которые вы создали самостоятельно, или лаборатории, которые Администратор назначил Вам в качестве владельца. Дополнительные сведения см. [в статье Добавление дополнительных владельцев в существующую лабораторию](how-to-add-user-lab-owner.md) .
2. Чтобы получить доступ к виртуальным машинам для классов, которые вы выбрали в качестве учащегося, щелкните значок компьютера в правом верхнем углу. Убедитесь, что вы видите виртуальные машины, к которым можно получить доступ как к студенту. В следующем примере пользователь является помощником по преподаванию для лаборатории Python, но студентом лаборатории Java. Таким образом, пользователь видит виртуальную машину из лаборатории Java в раскрывающемся списке. Пользователь может запустить виртуальную машину и подключиться к ней.
