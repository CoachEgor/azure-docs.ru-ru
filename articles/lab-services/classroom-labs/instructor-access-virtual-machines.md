# <a name="access-virtual-machines-as-a-student-from-the-instructor-view"></a>Доступ к виртуальным машинам в качестве студента из представления инструктора
В этой статье показано, как преподаватели могут получить доступ к своим вддля для занятий, которые они посещают в качестве студентов. 

Вот сценарий, где эта функция поможет. Помощник преподавателя является инструктором для одного класса, но студент в других классах. И, преподаватель хочет просматривать и получить доступ к студенту VMs из инструктора зрения, которая показывает лаборатории они владеют. 

## <a name="access-vms-from-instructor-view"></a>Доступ к vMs-чаевых из представления инструктора

1. Вопием на [веб-сайте Лабораторных служб Azure.](https://labs.azure.com) Вы видите лаборатории, которыми вы владеете. Эти лаборатории могут быть лабораториями, которые вы создали сами, или лабораториями, которые админ назначен вам в качестве владельца. Для получения дополнительной информации [см. Как добавить дополнительных владельцев в существующую лабораторию](how-to-add-user-lab-owner.md)
2. Чтобы получить доступ к вдыхать для занятий, которые вы посещаете в качестве студента, выберите значок компьютера в правом верхнем углу. Подтвердите, что вы видите VMs, к которым вы можете получить доступ в качестве студента. В следующем примере пользователь является ассистентом преподавателя в лаборатории Python, но студентом Java-лаборатории. Таким образом, пользователь видит VM из лаборатории Java в списке выпадающих. Пользователь может запустить VM и подключиться к нему.
