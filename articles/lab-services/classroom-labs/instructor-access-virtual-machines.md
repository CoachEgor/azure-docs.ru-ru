# <a name="access-virtual-machines-as-a-student-from-the-educator-view"></a>Доступ к виртуальным машинам в качестве учащегося из представления преподавателя
В этой статье показано, как преподаватели могут получить доступ к своим виртуальным машинам для классов, которые они посещают в качестве учащихся. 

Ниже приведен сценарий, в котором эта функция поможет. Ассистент преподавателя — это преподаватель для одного класса, но учащийся для других. И ассистент преподавателя хочет получить доступ к виртуальным машинам учащихся из представления преподавателя, в котором будут показаны его лаборатории. 

## <a name="access-vms-from-educator-view"></a>Доступ к виртуальным машинам из представления преподавателя

1. Войдите на [веб-сайт Служб лабораторий Azure](https://labs.azure.com). Вы видите собственные лаборатории. Это могут быть лаборатории, которые вы создали самостоятельно, или лаборатории, которые администратор назначил вам в качестве владельца. Дополнительные сведения см. в статье [Добавление дополнительных владельцев в существующую лабораторию](how-to-add-user-lab-owner.md)
2. Чтобы получить доступ к виртуальным машинам для классов, которые вы выбрали в качестве учащегося, щелкните значок компьютера в правом верхнем углу. Убедитесь, что вы видите виртуальные машины, к которым можно получить доступ в качестве учащегося. В следующем примере пользователь является ассистентом преподавателя для лаборатории Python, но учащимся для лаборатории Java. Таким образом, пользователь видит виртуальную машину из лаборатории Java в раскрывающемся списке. Пользователь может запустить виртуальную машину и подключиться к ней. 
    
    ![Доступ к виртуальным машинам учащихся](../media/instructors-access-virtual-machines/access-student-virtual-machines.png)
