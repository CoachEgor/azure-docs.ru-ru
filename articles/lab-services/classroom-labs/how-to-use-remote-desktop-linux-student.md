# <a name="use-remote-desktop-for-linux-virtual-machines-in-a-classroom-lab-of-azure-lab-services"></a>Использование удаленного рабочего стола для виртуальных машин Linux в учебной лаборатории служб лаборатории Azure
В этой статье показано, как студенты могут подключаться к виртуальной машине Linux в лаборатории по протоколу RDP или SSH. 

Преподавателю необходимо включить функцию подключения к удаленному рабочему столу, прежде чем студенты смогут подключиться к виртуальной машине в лаборатории. Инструкции по включению функции подключения к удаленному рабочему столу преподавателем см. в статье [Включение удаленного рабочего стола для виртуальных машин Linux](how-to-enable-remote-desktop-linux.md).

> [!IMPORTANT] 
> При включении **подключения к удаленному рабочему столу** открывается только порт **RDP** на компьютерах Linux. Инструктор может подключиться к компьютеру Linux с помощью SSH в первый раз и установить пакеты RDP и GUI, чтобы вы могли подключиться к компьютеру Linux по протоколу RDP позднее. 

## <a name="connect-to-the-student-vm"></a>Подключение к виртуальной машине учащихся
Студенты могут подключиться к виртуальным машинам Linux по протоколу RDP после того, как владелец лаборатории (преподаватель/профессор) **ОПУБЛИКУЕТ** виртуальную машину шаблона с пакетами RDP и GUI, установленными на компьютере. Для этого выполните следующие действия: 

1. Когда учащийся входит на портал Labs напрямую (`https://labs.azure.com`) или с помощью регистрационной ссылки (`https://labs.azure.com/register/<registrationCode>`), отображается плитка для каждой лабораторной работы, к которой у учащегося есть доступ. 
2. На плитке выберите **запустить** , если виртуальная машина остановлена. 
3. Нажмите кнопку **Подключиться**. Вы увидите два варианта подключения к виртуальной машине: **SSH** и **Удаленный рабочий стол**.

    ![Виртуальная машина для учащихся — параметры подключения](../media/how-to-enable-remote-desktop-linux/student-vm-connect-options.png)

## <a name="connect-using-ssh-or-rdp"></a>Подключение по протоколу SSH или RDP
Если выбран параметр **SSH** , появится следующее диалоговое окно **Подключение к виртуальной машине** :  

![Строка подключения SSH](../media/how-to-enable-remote-desktop-linux/ssh-connection-string.png)

Нажмите кнопку **копирования** рядом с текстовым полем, чтобы скопировать его в буфер обмена. Сохраните строку подключения SSH. Используйте эту строку подключения из терминала SSH (например [PuTTY](https://www.putty.org/)) для подключения к виртуальной машине.

Если выбран параметр **RDP** , на компьютер загружается RDP-файл. Сохраните его и откройте для подключения к компьютеру.
