# <a name="use-remote-desktop-for-linux-virtual-machines-in-a-classroom-lab-of-azure-lab-services"></a>Используйте удаленный рабочий стол для виртуальных машин Linux в классной лаборатории Лабораторных служб Azure
В этой статье показано, как студенты могут подключиться к виртуальной машине Linux (VM) в лаборатории с помощью RDP/SSH. 

Инструктор должен включить функцию удаленного соединения рабочего стола, прежде чем студенты смогут подключиться к VM классной лаборатории. Для получения инструкций о том, как инструктор может включить функцию удаленного соединения рабочего стола, [см.](how-to-enable-remote-desktop-linux.md)

> [!IMPORTANT] 
> Включение **удаленного соединения рабочего стола** открывает порт **RDP** только на машинах Linux. Инструктор может подключиться к машине Linux с помощью SSH в первый раз, и установить RDP и GUI пакеты, так что вы можете подключиться к машине Linux с помощью RDP позже. 

## <a name="connect-to-the-student-vm"></a>Подключение к студенту VM
Студенты могут RDP в их Linux VMs после того, как владелец лаборатории (преподаватель / профессор) **публикует** шаблон VM с RDP и GUI пакеты, установленные на машине. Ниже приводятся шаги: 

1. Когда студент вписывается на портал`https://labs.azure.com`Labs непосредственно (`https://labs.azure.com/register/<registrationCode>`) или с помощью регистрационной ссылки ( ), плитка для каждой лаборатории студент имеет доступ к отображается. 
2. На плитке переключите кнопку, чтобы запустить VM, если он в остановленном состоянии. 
3. Выберите **Подключиться**. Вы видите два варианта подключения к VM: **SSH** и **удаленный рабочий стол.**

    ![Студенческий VM - варианты подключения](../media/how-to-enable-remote-desktop-linux/student-vm-connect-options.png)

## <a name="connect-using-ssh-or-rdp"></a>Подключение с помощью SSH или RDP
Если вы выберете опцию **SSH,** вы увидите следующее **подключение к виртуальному** диалоговому ящику машины:  

![Строка соединения SSH](../media/how-to-enable-remote-desktop-linux/ssh-connection-string.png)

Выберите кнопку **Copy** рядом с текстовым полем, чтобы скопировать ее на буфер обмена. Сохранить строку соединения SSH. Используйте эту строку подключения из терминала SSH (например [PuTTY](https://www.putty.org/)) для подключения к виртуальной машине.

При выборе опции **RDP** файл RDP загружается на вашу машину. Сохраните его и откройте для подключения к машине.
