# <a name="dashboard-for-classroom-labs"></a>Панель мониторинга для классных лабораторий
В этой статье описывается представление панели мониторинга классной лаборатории в Лабораторной службе Azure. 

![Панель мониторинга](../media/use-dashboard/dashboard.png)

## <a name="costs-and-billing-tile"></a>Затраты и биллинговая плитка
Эта плитка содержит следующие сведения о смете затрат:

| Параметр | Значение | 
| ------- | ----- | 
| Часы квоты | Максимальное количество часов, которые пользователь может использовать VM вне запланированных часов. |
| Запланированные часы | Часы, которые будут понесены в зависимости от графика, установленного в лаборатории. Это значение доступно только в том случае, если на всех событиях расписания указывается от/на сегодняшний день. |
| Часы/пользователи | Сумма квотных часов и запланированных часов. |
| Максимальное количество пользователей | Максимальное количество пользователей в лаборатории на основе всех виртуальных машин, которые будут востребованы. |
| Часы x пользователей | Часы/пользователи умножаются на количество пользователей. |
| Скорректированная квота | Сумма квотных часов, добавленная конкретным пользователям. |
| Общее количество часов и $/час | Стоимость в час на основе выбранного размера VM. Это основано на регулярной заработной платы, как вы идете цена. |
| Общая сметная стоимость | Это максимальная цена для этой лаборатории на основе текущих настроек. |

## <a name="template-tile"></a>Шаблон плитки
Вы видите следующую информацию на этой плитке:

- Дата создания шаблона 
- Дата последнего опубликования шаблона 

Он также имеет ссылку для навигации по странице **шаблона,** где вы можете [управлять шаблоном VM](how-to-create-manage-template.md) для класса. 

## <a name="virtual-machine-pool-tile"></a>Плитка виртуального пула машины

Вы видите следующую информацию на этой плитке:

- Количество виртуальных машин, назначенных студентам (пользователям)
- Количество виртуальных машин, которые еще не были назначены студентам

Он также имеет ссылку для навигации на страницу **виртуального пула машины,** где вы можете [управлять пулом виртуальных машин](how-to-set-virtual-machine-passwords.md) в лаборатории. 

## <a name="users-tile"></a>Плитка пользователей

Вы видите следующую информацию на этой плитке:

- Количество пользователей, зарегистрированных в классе
- Количество пользователей, которые добавляются в лабораторию, но не зарегистрированы в классе 

Он также имеет ссылку на навигацию по странице **пользователей,** где вы можете [управлять пользователями](how-to-configure-student-usage.md) для лаборатории. 

## <a name="schedules-tile"></a>Плитка расписания
Вы видите текущие запланированные события для лаборатории на плитке. Он также имеет ссылку на переход на страницу **Расписание,** где вы можете [создавать и управлять графиками.](how-to-create-schedules.md) Плитка показывает детали только для двух запланированных событий и количество оставшихся запланированных событий для лаборатории. 

![Запланированные события](../media/use-dashboard/scheduled-events.png)
