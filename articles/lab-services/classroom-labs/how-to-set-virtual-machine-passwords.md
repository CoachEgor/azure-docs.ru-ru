# <a name="set-or-reset-password-for-virtual-machines-in-classroom-labs-instructor"></a>Установка или сброс пароля для виртуальных машин в учебной лаборатории (instructor)
Владелец лаборатории (преподаватель) может установить или сбросить пароль для виртуальных машин во время создания лаборатории (мастер создания лабораторий) или после создания лаборатории (на панели мониторинга). 

## <a name="set-password-at-the-time-of-lab-creation"></a>Задать пароль во время создания лаборатории
Владелец лаборатории (преподаватель) может задать пароль для виртуальных машин в лаборатории на странице **Установка учетных данных** мастера создания лабораторий.

![Задайте учетные данные](../media/tutorial-setup-classroom-lab/set-credentials.png)

Включив или отключив параметр **использовать один и тот же пароль для всех виртуальных машин** на этой странице, преподаватель может выбрать один и тот же пароль для всех виртуальных машин в лаборатории или разрешить учащимся устанавливать пароли для их виртуальных машин. По умолчанию этот параметр включен для всех образов операционных систем Windows и Linux, кроме Ubuntu. Если этот параметр отключен, студентам будет предложено задать пароль при первой попытке подключения к виртуальной машине. 

Владелец лаборатории может сбросить этот пароль (при необходимости) на странице **Настройка шаблона** мастера создания лабораторий. 

![Страница настройки шаблона по завершении процесса](../media/tutorial-setup-classroom-lab/configure-template-after-complete.png)

Владелец лаборатории также может сбросить пароль после создания лаборатории на панели мониторинга. 

## <a name="reset-password-on-the-dashboard"></a>Сброс пароля на панели мониторинга

1. Выберите меню переполнения (вертикально три точки) на плитке лаборатории и щелкните **Сброс пароля**. 

    ![Меню сброса пароля на домашней странице](../media/how-to-set-virtual-machine-passwords/reset-password-menu-dashboard.png)
1. В диалоговом окне **Установка пароля** введите пароль и нажмите кнопку **задать пароль**.
    
    ![Диалоговое окно «Установка пароля»](../media/how-to-set-virtual-machine-passwords/set-password.png)
