# <a name="set-or-reset-password-for-virtual-machines-in-classroom-labs"></a>Установить или сбросить пароль виртуальных машин в лаборатории аудитории
В этой статье предоставляет различные способы настройки и resettings пароли для доступа к ВМ в аудиториях labs. 

## <a name="lab-owners-teachers"></a>Владельцы лаборатории (преподаватели)
Владелец лаборатории (учитель) можно установить на пароль, для виртуальных машин в лаборатории **задать учетные данные** страницы мастера создания лаборатории.

![Настройка учетных данных](../media/tutorial-setup-classroom-lab/set-credentials.png)

Затем владелец лаборатории могут сбрасывать пароли (при необходимости) на **Настройка шаблона** страницы мастера создания лаборатории. 

![Страница настройки шаблона по завершении процесса](../media/tutorial-setup-classroom-lab/configure-template-after-complete.png)

Владелец лаборатории можно также сбросить пароль, после создания лаборатории на панели мониторинга. 

![Меню сброса пароля на домашней странице](../media/how-to-set-virtual-machine-passwords/reset-password-menu-dashboard.png)

Затем введите новый пароль в **пароль набора** страницы и выберите **пароль набора**. 

![Меню сброса пароля на домашней странице](../media/how-to-set-virtual-machine-passwords/set-password.png)

## <a name="lab-users-students"></a>Пользователи лаборатории (для учащихся)
Во время создания лаборатории. В то же время владелец лаборатории можно включить или отключить позволяют учащимся возможность задавать свои собственные пароли для виртуальных машин. Если владелец лаборатории (учитель) включает этот параметр, пользователь лаборатории (учащегося) будет иметь возможность установить пароль для виртуальной Машины, когда ученик выполняет вход на виртуальную машину в первый раз. На **виртуальной Машины Windows**, нажмите клавишу **CTRL + ALT + END**и выберите **Смена пароля**.
