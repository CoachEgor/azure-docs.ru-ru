# <a name="set-or-reset-password-for-virtual-machines-in-classroom-labs"></a>Установить или сбросить пароль виртуальных машин в лаборатории аудитории
В этой статье предоставляет различные способы настройки и resettings пароли для доступа к ВМ в аудиториях labs. 

## <a name="lab-owners-teachers"></a>Владельцы лаборатории (преподаватели)
Владелец лаборатории (учитель) можно установить или сбросить пароль для виртуальных машин во время создания лаборатории (мастер создания лаборатории) или после создания лаборатории (на панели мониторинга). 

### <a name="set-password-at-the-time-of-lab-creation"></a>Установка пароля во время создания лаборатории
Владелец лаборатории (учитель) можно установить на пароль, для виртуальных машин в лаборатории **задать учетные данные** страницы мастера создания лаборатории.

![Настройка учетных данных](../media/tutorial-setup-classroom-lab/set-credentials.png)

При включении или отключении **использовать тот же пароль для всех виртуальных машин** на этой странице, преподаватель можно использовать тот же пароль для всех виртуальных машин в лаборатории или разрешить участникам задавать пароли для своих виртуальных машин. По умолчанию этот параметр включен для всех Windows и Linux образов операционных систем, за исключением Ubuntu. Если этот параметр отключен, учащихся будет предложено задать пароль при попытке подключиться к виртуальной Машине в первый раз. 

Владелец лаборатории можно сбросить этот пароль (при необходимости) на **Настройка шаблона** страницы мастера создания лаборатории. 

![Страница настройки шаблона по завершении процесса](../media/tutorial-setup-classroom-lab/configure-template-after-complete.png)

Владелец лаборатории можно также сбросить пароль, после создания лаборатории на панели мониторинга. 

### <a name="reset-password-on-the-dashboard"></a>Сброс пароля на панели мониторинга

1. Выберите меню переполнения (вертикальное многоточие) на плитке "лаборатории" и выберите **сброс пароля**. 

    ![Меню сброса пароля на домашней странице](../media/how-to-set-virtual-machine-passwords/reset-password-menu-dashboard.png)
1. На **пароль набора** диалоговом окне введите пароль и выберите **пароль набора**.
    
    ![Диалоговое окно пароля набора](../media/how-to-set-virtual-machine-passwords/set-password.png)

## <a name="lab-users-students"></a>Пользователи лаборатории (для учащихся)
Во время создания лаборатории, владелец лаборатории можно включить или отключить **использовать тот же пароль для всех виртуальных машин**. Если этот параметр включен, учащихся невозможно сбросить пароль. Все виртуальные машины в лабораторном занятии будет иметь тот же пароль, который задается параметром преподаватель. 

Если этот параметр отключен, пользователям придется установить пароль при попытке подключения к виртуальной Машине в первый раз. Когда пользователи (учащиеся) выбирают **Connect** кнопку на плитке лаборатории на **Мои виртуальные машины** страницы, пользователь увидит следующее диалоговое окно позволяет задать пароль для виртуальной Машины: 

![Сброс пароля для учащегося](../media/how-to-set-virtual-machine-passwords/student-set-password.png)

Учащегося также можно задать пароль, щелкнув меню переполнения (**вертикальное многоточие**) на плитке "лаборатории" и выбрав **сброс пароля**.
