# <a name="enable-automatic-shutdown-of-vms-on-disconnect"></a>Включить автоматическое выключение ВМ при отключении
В этой статье показано, как можно включить или отключить автоматическое отключение vMs-технологий для лаборатории **Windows 10** (шаблон или студент) после отключения удаленного соединения рабочего стола. Вы также можете указать, как долго вспыхнуть в ожидании повторного подключения пользователя, прежде чем автоматически выключиться.

Администратор учетной записи лаборатории может настроить этот параметр для учетной записи лаборатории, в которой вы создаете лаборатории. Для получения дополнительной информации [см. Налаживание автоматического выключения ВМ при отключении для лабораторной учетной записи.](how-to-configure-lab-accounts.md#automatic-shutdown-of-vms-on-disconnect) Как владелец лаборатории, вы можете переопределить настройки при создании лаборатории или после создания лаборатории. 

## <a name="configure-when-creating-a-lab"></a>Настройка при создании лаборатории
На странице мастера создания лаборатории можно включить или отключить эту функцию, а также указать, как долго VM должен ждать повторного подключения пользователя перед автоматическим завершением. 

![Настройка во время создания лаборатории](../media/how-to-enable-shutdown-disconnect/configure-lab-creation.png)

## <a name="configure-after-the-lab-is-created"></a>Настройка после создания лаборатории
Эту настройку можно настроить на странице **«Настройки»,** как показано на следующем изображении: 

![Настройка после создания лаборатории](../media/how-to-enable-shutdown-disconnect/configure-lab-automatic-shutdown.png)

> [!WARNING]
> При отключении операционной системы (ОС) Windows на VM перед отключением сеанса RDP к VM функция автоматического выключения не будет работать должным образом.
