# <a name="use-a-shared-image-gallery-in-azure-lab-services"></a>Использование общей галереи изображений в службах лаборатории Azure
В этой статье показано, как преподаватели/лабораторные админ могут сохранить изображение виртуальной машины шаблона, чтобы оно было повторно использовано другими пользователями. Эти изображения сохраняются в [общей галерее изображений](../../virtual-machines/windows/shared-image-galleries.md)Azure. В качестве первого шага лабораторный админ прикрепляет существующую общую галерею изображений к учетной записи лаборатории. После присоединения общей галереи изображений лаборатории, созданные в учетной записи лаборатории, могут сохранять изображения в общей галерее изображений. Другие преподаватели могут выбрать это изображение из общей галереи изображений, чтобы создать шаблон для своих классов. 

> [!NOTE]
> В настоящее время Azure Lab Services поддерживает создание шаблонных VM на основе только **обобщенных** VM-изображений (не специализированных изображений) в общей галерее изображений. 

## <a name="prerequisites"></a>Предварительные требования
- Создайте общую галерею изображений, используя либо [Azure PowerShell,](../../virtual-machines/windows/shared-images.md) либо [Azure CLI.](../../virtual-machines/linux/shared-images.md)
- Вы прикрепили общую галерею изображений к учетной записи лаборатории. Для пошаговых инструкций [смотрите, как прикрепить или отсоединить общую галерею изображений.](how-to-attach-detach-shared-image-gallery.md)


## <a name="save-an-image-to-the-shared-image-gallery"></a>Сохранить изображение в общей галерее изображений
После присоединения общей галереи изображений админ или преподаватель лаборатории могут сохранить изображение в общей галерее изображений, чтобы его можно было повторно использовать другими преподавателями. 

1. На странице **шаблона** для лаборатории выберите Экспорт в **общую галерею изображений** на панели инструментов.

    ![Сохранить кнопку изображения](../media/how-to-use-shared-image-gallery/export-to-shared-image-gallery-button.png)
2. На **диалоге «Экспорт в общую галерею изображений»** введите **имя для изображения,** а затем выберите **Export.** 

    ![Экспорт в диалог общей галереи изображений](../media/how-to-use-shared-image-gallery/export-to-shared-image-gallery-dialog.png)
3. Вы можете увидеть ход этой операции на странице **шаблона.** Эта операция может занять некоторое время. 

    ![Экспорт в процессе](../media/how-to-use-shared-image-gallery/exporting-image-in-progress.png)
4. Когда операция экспорта выполняется успешно, вы видите следующее сообщение:

    ![Экспорт завершен](../media/how-to-use-shared-image-gallery/exporting-image-completed.png)

Вы также можете загрузить изображение в общую галерею изображений вне контекста лаборатории. Для получения дополнительной информации смотрите [обзор галереи общих изображений](../../virtual-machines/windows/shared-images.md). 

## <a name="use-an-image-from-the-shared-image-gallery"></a>Используйте изображение из общей галереи изображений
Преподаватель/профессор может выбрать пользовательское изображение, доступное в общей галерее изображений, для шаблона во время создания новой лаборатории.

![Используйте изображение виртуальной машины из галереи](../media/how-to-use-shared-image-gallery/use-shared-image.png)
