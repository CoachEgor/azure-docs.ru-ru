# <a name="create-and-manage-schedules-for-classroom-labs-in-azure-lab-services"></a>Создание и управление расписанием лабораторий для аудитории в Службах лабораторий Azure 
Расписания позволяют настроить лабораторию аудитории, например, виртуальные машины в лаборатории автоматически начинают и завершают работу в указанное время. Вы можете определить разовое расписание или повторяющееся расписание. Из следующих процедур можете узнать шаги, которые необходимо выполнить для создания и управления расписаниями для лаборатории аудитории. 

> [!IMPORTANT]
> Запланированное время запуска виртуальных машин не учитывается в [квоте, выделенной пользователю](how-to-configure-student-usage.md#set-quotas-per-user). Квота рассчитывается на время, которое выходит за пределы расписания и которое учащийся тратит на виртуальные машины. 

## <a name="add-a-schedule-once"></a>Добавление расписания (однократное)

1. Перейдите на страницу **Расписания** и на панели инструментов выберите **Добавить расписание**. 

    ![Добавление кнопки расписания на страницу "Расписания"](../media/how-to-create-schedules/add-schedule-button.png)
2. На странице **Добавить расписание** подтвердите, что вверху был выбран параметр **Однократно**. Если нет, то выберите **Однократно**. 
3. В поле **Schedule date (required)** (Запланированная дата (обязательно)) введите дату или нажмите значок календаря, а затем выберите дату. 
4. В разделе **Время начала** выберите время, когда должны запускаться виртуальные машины. Время начала требуется, если не указано время остановки. Если необходимо указать только время остановки, выберите **Remove start event** (Удалить событие начала). Если **Время начала** отключено, то, чтобы включить его, выберите **Add start event** (Добавить событие начала) рядом с раскрывающимся списком. 
5. В поле **Время остановки** выберите время, когда виртуальные машины должны остановиться. Время остановки необходимо, когда время начала не указано. Если необходимо указать только время начала, выберите **Remove stop event** (Удалить событие остановки). Если **Время остановки** отключено, то, чтобы его включить, выберите **Add stop event** (Добавить событие остановки) рядом с раскрывающимся списком.
6. В поле **Time zone (required)** (Часовой пояс (обязательно)) выберите часовой пояс для указанного времени начала и окончания. 
7. В разделе **Примечания** введите описание или заметки, которые соответствуют расписанию. 
8. Щелкните **Сохранить**. 

    ![Однократное расписание](../media/how-to-create-schedules/add-schedule-page.png)

## <a name="add-a-recurring-schedule-weekly"></a>Добавление повторяющегося расписания (еженедельно)

1. Перейдите на страницу **Расписания** и на панели инструментов выберите **Добавить расписание**. 

    ![Добавление кнопки расписания на страницу "Расписания"](../media/how-to-create-schedules/add-schedule-button.png)
2. На странице **Добавить расписание** переключитесь на **Еженедельно** вверху. 
3. В разделе **Schedule days (required)** (Запланированные дни (обязательно)) выберите дни, необходимые, чтобы расписание вступило в силу. В следующем примере выбран период "Понедельник — пятница". 
4. В поле **Из** введите **дату начала расписания** или выберите ее, нажав кнопку **Календарь**. Это поле обязательно к заполнению. 
5. В поле **Дата окончания плана** введите или выберите дату окончания, когда виртуальные машины должны прекратить работу. 
6. В разделе **Время начала** выберите время запуска виртуальных машин. Время начала требуется, если не указано время остановки. Если необходимо указать только время остановки, выберите **Remove start event** (Удалить событие начала). Если **Время начала** отключено, то, чтобы включить его, выберите **Add start event** (Добавить событие начала) рядом с раскрывающимся списком. 
7. В разделе **Время остановки** выберите время завершения работы виртуальных машин. Время остановки необходимо, когда время начала не указано. Если необходимо указать только время начала, выберите **Remove stop event** (Удалить событие остановки). Если **Время остановки** отключено, то, чтобы его включить, выберите **Add stop event** (Добавить событие остановки) рядом с раскрывающимся списком.
8. В поле **Time zone (required)** (Часовой пояс (обязательно)) выберите часовой пояс для указанного времени начала и окончания.  
9. В разделе **Примечания** введите описание или заметки, которые соответствуют расписанию. 
10. Щелкните **Сохранить**. 

    ![Еженедельное расписание](../media/how-to-create-schedules/add-schedule-page-weekly.png)

## <a name="view-schedules-in-calendar"></a>Просмотр расписаний в календаре
Запланированные даты и время будут выделены в представлении календаря, как показано на следующем изображении, указанном ниже.

![Расписания в представлении календаря](../media/how-to-create-schedules/schedules-in-calendar.png)

Чтобы переключиться на текущую дату в календаре, нажмите кнопку **Сегодня**. Выберите **стрелку влево**, чтобы переключиться на предыдущую неделю, а чтобы переключиться на следующую неделю, выберите **стрелку вправо**. 

## <a name="edit-a-schedule"></a>Изменение расписания
Если дважды щелкнуть на выделенном расписании в календаре или нажать кнопку **Карандаш** на панели инструментов, появится страница **Изменить расписание**. Обновление параметров на этой странице ничем не отличается от обновления параметров на странице **Добавить расписание**, что описано в разделе [Добавление повторяющегося расписания (еженедельно)](#add-a-recurring-schedule-weekly). 

![Изменение страницы расписания](../media/how-to-create-schedules/edit-schedule-page.png)

## <a name="delete-a-schedule"></a>Удаление расписания

1. Чтобы удалить расписание, нажмите кнопку корзины (удалить) на панели инструментов, как показано на следующем изображении.

    ![Кнопка "Удалить" на панели инструментов](../media/how-to-create-schedules/delete-schedule-button.png)

    Кнопку "Удалить" можно использовать в календаре на любых запланированных датах и времени, выбрав **Удалить**. 
2. На странице **Удалить расписания** выберите **Да**.

    ![Подтверждение "Удалить расписания"](../media/how-to-create-schedules/delete-schedules-confirmation.png)
