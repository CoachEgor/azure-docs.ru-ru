# <a name="configure-automatic-shutdown-of-vms-on-disconnect-setting-for-a-lab-account"></a>Настройка автоматического завершения работы виртуальных машин в при отключении учетной записи лаборатории
После отключения подключения к удаленному рабочему столу вы можете включить или отключить автоматическое завершение работы виртуальных машин лаборатории Windows (шаблон или учащийся). Также можно указать, как долго Службы лабораторий должны ожидать повторного подключения пользователя до автоматического завершения работы.

## <a name="enable-automatic-shutdown"></a>Включение автоматического завершения работы

1. На странице **Учетная запись лаборатории** в меню слева выберите **Labs settings** (Параметры лаборатории).
2. Выберите параметр **Автоматически завершать работу виртуальных машин при отключении пользователей**.
3. Укажите, как долго Службы лабораторий должны ожидать повторного подключения пользователя до автоматического завершения работы виртуальной машины.

    ![Параметр автоматического завершения работы в учетной записи лаборатории](../media/how-to-configure-lab-accounts/automatic-shutdown-vm-disconnect.png)

    Этот параметр применяется ко всем лабораториям, созданным в учетной записи лаборатории. Создатель лаборатории (преподаватель) может переопределить этот параметр на уровне лаборатории. Изменение этого параметра в учетной записи лаборатории будет влиять только на лаборатории, созданные после внесения изменений.

    Чтобы отключить этот параметр, снимите флажок **Автоматически завершать работу виртуальных машин при отключении пользователей** на этой странице.
