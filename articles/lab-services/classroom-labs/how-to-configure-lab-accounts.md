# <a name="configure-lab-accounts-in-azure-lab-services"></a>Настройка учетных записей лаборатории в службах лаборатории Azure 
В службах лаборатории Azure лабораторная учетная запись — это контейнер для управляемых лабораторных типов, таких как учебные лаборатории. Администратор задает учетную запись лаборатории с помощью Служб лабораторий Azure и предоставляет доступ владельцам лабораторий, которые могут создать лаборатории в учетной записи. 

В этой статье показано, как выполнять следующие задачи: 

- Укажите диапазон адресов для виртуальных машин в лаборатории
- Настройка автоматического завершения работы виртуальных машин при отключении

## <a name="specify-an-address-range-for-vms-in-the-lab"></a>Укажите диапазон адресов для виртуальных машин в лаборатории
Ниже описана процедура указания диапазона адресов для виртуальных машин в лаборатории. Если вы обновляете ранее указанный диапазон, измененный диапазон адресов применяется только к виртуальным машинам, созданным после внесения изменений. 

Ниже приведены некоторые ограничения, которые следует учитывать при указании диапазона адресов. 

- Префикс должен быть меньше или равен 23. 
- Если виртуальная сеть является одноранговой для учетной записи лаборатории, указанный диапазон адресов не может перекрывать диапазон адресов от одноранговой виртуальной сети.

1. На странице **учетная запись лаборатории** в меню слева выберите **Параметры лабораторий** .
2. В поле **диапазон адресов** укажите диапазон адресов для виртуальных машин, которые будут созданы в лаборатории. Диапазон адресов должен находиться в нотации с независимым междоменным маршрутизацией (CIDR) (например: 10.20.0.0/23). В этом диапазоне адресов будут создаваться виртуальные машины в лаборатории.
3. На панели инструментов щелкните **Сохранить**. 

    ![Настройка диапазона адресов](../media/how-to-manage-lab-accounts/labs-configuration-page-address-range.png)


## <a name="automatic-shutdown-of-vms-on-disconnect"></a>Автоматическое завершение работы виртуальных машин при отключении
После отключения подключения к удаленному рабочему столу можно включить или отключить автоматическое завершение работы виртуальных машин Windows Lab (шаблон или учащийся). Вы также можете указать, как долго виртуальные машины должны ожидать повторного подключения пользователя перед автоматическим завершением работы.

![Параметр автоматического завершения работы в учетной записи лаборатории](../media/how-to-configure-lab-accounts/automatic-shutdown-vm-disconnect.png)

Этот параметр применяется ко всем лабораторным занятиям, созданным в учетной записи лаборатории. Владелец лаборатории может переопределить этот параметр на уровне лаборатории. Изменение этого параметра в учетной записи лаборатории будет влиять только на лаборатории, созданные после внесения изменений.

Сведения о том, как владелец лаборатории может настроить этот параметр на уровне лаборатории, см. в [этой статье](how-to-enable-shutdown-disconnect.md) .
