# <a name="attach-or-detach-a-shared-image-gallery-in-azure-lab-services"></a>Прикрепите или отсоедините общую галерею изображений в лабораторных службах Azure
Преподаватели/лабораторные админ могут сохранить изображение VM-шаблона в [общей галерее изображений](../../virtual-machines/windows/shared-image-galleries.md) Azure, чтобы его можно было повторно использовать другими пользователями. В качестве первого шага лабораторный админ прикрепляет существующую общую галерею изображений к учетной записи лаборатории. После присоединения общей галереи изображений лаборатории, созданные в учетной записи лаборатории, могут сохранять изображения в общей галерее изображений. Другие преподаватели могут выбрать это изображение из общей галереи изображений, чтобы создать шаблон для своих классов. 

При сохранении изображения в общей галерее изображений службы Azure Lab воспроизводит сохраненное изображение в другие регионы, доступные в той же [географии.](https://azure.microsoft.com/global-infrastructure/geographies/) Это гарантирует, что изображение доступно для лабораторий, созданных в других регионах в той же географии. Сохранение изображений в общей картинной галерее несет дополнительные расходы, которые включают в себя расходы на все реплицированные изображения. Эта стоимость отделена от стоимости использования услуг Azure Lab. Для получения дополнительной информации о ценах на галерею общих изображений [см.]( https://docs.microsoft.com/azure/virtual-machines/windows/shared-image-galleries#billing)

В этой статье показано, как прикрепить или отсоединить общую галерею изображений к лабораторной учетной записи. 

> [!NOTE]
> В настоящее время Azure Lab Services поддерживает создание шаблонных VM на основе только **обобщенных** VM-изображений (не специализированных изображений) в общей галерее изображений. 


## <a name="configure-at-the-time-of-lab-account-creation"></a>Настройка во время создания лабораторного счета
При создании лабораторной учетной записи можно прикрепить общую галерею изображений к учетной записи лаборатории. Вы можете выбрать существующую общую галерею изображений из списка выпадающих изображений или создать новую. Чтобы создать и прикрепить общую галерею изображений к учетной записи лаборатории, выберите **Создать новый,** введите имя для галереи и введите **OK.** 

![Настройка общей галереи изображений во время создания лабораторного счета](../media/how-to-use-shared-image-gallery/new-lab-account.png)

## <a name="configure-after-the-lab-account-is-created"></a>Настройка после создания учетной записи лаборатории
После создания учетной записи лаборатории можно высделать следующие задачи:

- Создание и присоединение общей галереи изображений
- Прикрепите общую галерею изображений к учетной записи лаборатории
- Отделите общую галерею изображений из лабораторного счета

## <a name="create-and-attach-a-shared-image-gallery"></a>Создание и присоединение общей галереи изображений
1. Войдите на [портал Azure](https://portal.azure.com).
2. В меню слева выберите **Все службы**. Выберите **Службы лабораторий** в разделе **DevOps**. При выборе звездочки (`*`) рядом с полем **Службы лабораторий** оно добавляется в раздел **Избранное** в меню слева. В следующий раз выберите **Службы лабораторий** в разделе **Избранное**.

    ![Все службы -> Службы лабораторий](../media/tutorial-setup-lab-account/select-lab-accounts-service.png)
3. Выберите учетную запись лаборатории, чтобы просмотреть страницу **учетной записи лаборатории.** 
4. Выберите **общую галерею изображений** в левом меню и выберите **«Создание** на панели инструментов».  

    ![Создание общей кнопки галереи изображений](../media/how-to-use-shared-image-gallery/new-shared-image-gallery-button.png)
5. В окне **галереи изображений Create,** введите **имя** галереи и введите **OK**. 

    ![Создание окна общей галереи изображений](../media/how-to-use-shared-image-gallery/create-shared-image-gallery-window.png)

    Службы Azure Lab создают общую галерею изображений и прикрепляют ее к учетной записи лаборатории. Все лаборатории, созданные в этой учетной записи лаборатории, имеют доступ к присоединенной общей галерее изображений. 

    ![Прилагаемые картины галерея](../media/how-to-use-shared-image-gallery/image-gallery-in-list.png)

    В нижней панели вы видите изображения в общей галерее изображений. В этой новой галерее нет изображений. При загрузке изображений в галерею вы видите их на этой странице.     

    Все изображения в прилагаемых общих галереях изображений включены по умолчанию. Вы можете включить или отключить выбранные изображения, выбрав их в списке и используя кнопку **Включить выбранные изображения** или кнопку **«Отключить выбранные изображения».**

## <a name="attach-an-existing-shared-image-gallery"></a>Прикрепите существующую общую галерею изображений
Следующая процедура показывает, как прикрепить существующую общую галерею изображений к лабораторной учетной записи. 

1. На странице **учетной записи Lab** выберите **галерею общих изображений** в левом меню и **прикрепите на** панели инструментов. 

    ![Общая галерея изображений - Добавить кнопку](../media/how-to-use-shared-image-gallery/sig-attach-button.png)
5. На **странице Attach существующая** общая страница галереи изображений выберите общую галерею изображений и выберите **OK.**

    ![Выберите существующую галерею](../media/how-to-use-shared-image-gallery/select-image-gallery.png)
6. Вы видите следующий экран: 

    ![Моя галерея в списке](../media/how-to-use-shared-image-gallery/my-gallery-in-list.png)
    
    В этом примере изображения в общей галерее пока нет.

    Инообразительная идентификация Azure Lab Services добавляется в качестве участника общей галереи изображений, присоединенной к лаборатории. Это позволяет учителям/ ИТ-админу сохранять изображения виртуальных машин в общей галерее изображений. Все лаборатории, созданные в этой учетной записи лаборатории, имеют доступ к присоединенной общей галерее изображений. 

    Все изображения в прилагаемых общих галереях изображений включены по умолчанию. Вы можете включить или отключить выбранные изображения, выбрав их в списке и используя кнопку **Включить выбранные изображения** или кнопку **«Отключить выбранные изображения».** 

## <a name="detach-a-shared-image-gallery"></a>Отсоедините общую галерею изображений
К лаборатории может быть прикреплена только одна общая галерея изображений. Если вы хотите прикрепить другую общую галерею изображений, отсоедините текущую галерею, прежде чем прикрепить новую галерею. Чтобы отделить общую галерею изображений из лаборатории, выберите **Detach** на панели инструментов и подтвердите операцию отсоединить. 

![Отделите общую галерею изображений из лабораторного счета](../media/how-to-use-shared-image-gallery/detach.png)
