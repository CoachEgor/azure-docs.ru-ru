# <a name="how-to-add-additional-owners-to-an-existing-lab-in-azure-lab-services"></a>Как добавить дополнительных владельцев в существующую лабораторию в Лабораторных службах Azure
В этой статье показано, как вы, как администратор, можете добавить дополнительных владельцев в существующую лабораторию.

## <a name="add-user-to-the-reader-role-for-the-lab-account"></a>Добавление пользователя в роль читателя для учетной записи лаборатории
Чтобы добавить пользователя в качестве дополнительного владельца в существующую лабораторию, необходимо сначала дать пользователю разрешения на **чтение** в учетной записи лаборатории.

1. Войдите на [портал Azure](https://portal.azure.com).
2. В меню слева выберите **Все службы**. Поиск **лабораторных служб,** а затем выберите его.
3. Выберите **учетную запись лаборатории** из списка. 
2. На **странице учетной записи Lab**выберите **элемент управления доступом (IAM)** в левом меню. 
2. На странице **управления доступом (IAM)** выберите **Добавить** на панели инструментов и выбрать **назначение роли Добавить.**

    ![Назначение ролей для учетной записи лаборатории ](../media/how-to-add-user-lab-owner/lab-account-access-control-page.png)
3. На странице **назначения ролей** сделайте следующие шаги: 
    1. Выберите **Читателя** для **роли**. 
    2. Выберите пользователя. 
    3. Нажмите кнопку **Сохранить**. 

        ![Добавление пользователя в роль читателя для учетной записи лаборатории ](../media/how-to-add-user-lab-owner/reader-lab-account.png)

## <a name="add-user-to-the-owner-role-for-the-lab"></a>Добавление пользователя к роли владельца для лаборатории

1. Вернитесь на страницу **учетной записи Lab,** выберите **все лаборатории** в левом меню.
2. Выберите **лабораторию,** в которую вы хотите добавить пользователя в качестве владельца. 
    
    ![Выбор лаборатории ](../media/how-to-add-user-lab-owner/select-lab.png)    
3. На странице **Lab** выберите **элемент управления доступом (IAM)** в левом меню.
4. На странице **управления доступом (IAM)** выберите **Добавить** на панели инструментов и выбрать **назначение роли Добавить.**
5. На странице **назначения ролей** сделайте следующие шаги: 
    1. Выберите **владельца** для **роли.** 
    2. Выберите пользователя. 
    3. Нажмите кнопку **Сохранить**.
