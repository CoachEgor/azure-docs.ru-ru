# <a name="firewall-settings-for-azure-lab-services"></a>Параметры брандмауэра для Служб лабораторий Azure

Каждая организация или школа будет создать свою собственную сеть таким образом, чтобы наилучшим образом соответствовали их потребностям.  Иногда это включает в себя установление правил брандмауэра, которые блокируют удаленный протокол рабочего стола (rdp) или безопасные подключения Shell (ssh) к машинам за пределами их собственной сети.  Поскольку Службы Лаборатории Лазурного Излза работают в общедоступном облаке, может потребоваться дополнительная конфигурация, чтобы позволить студентам получить доступ к VM при подключении из сети кампуса.

Каждая лаборатория использует единый публичный IP-адрес и несколько портов.  Все ВМ, как шаблон VM, так и студенческие VMs, будут использовать этот общедоступный IP-адрес.  Общественный IP-адрес не изменится за всю жизнь лаборатории.  Тем не менее, каждый VM будет иметь другой номер порта.  Номера портов варьируются от 49152 до 65535.  Сочетание общедоступного IP-адреса и номера порта используется для подключения инструктора и студентов к правильному VM.  В этой статье будет рассмотрен способ поиска конкретного общедоступного IP-адреса, используемого лабораторией.  Эта информация может быть использована для обновления входящих и исходящих правил брандмауэра, чтобы студенты могли получить доступ к своим вс-чам.

>[!IMPORTANT]
>Каждая лаборатория будет иметь свой общедоступный IP-адрес.

## <a name="find-public-ip-for-a-lab"></a>Найти общедоступный IP для лаборатории

Общедоступные IP-адреса для каждой лаборатории перечислены на странице **All Labs** в учетной записи лаборатории Lab Services.  Для направления, как найти страницу **Все лаборатории,** [см.](how-to-manage-lab-accounts.md#view-and-manage-labs-in-the-lab-account)  

> [!div class="mx-imgBorder"]
> ![Страница всех лабораторий](../media/how-to-configure-firewall-settings/all-labs-properties.png)

>[!NOTE]
>Вы не увидите общедоступный IP-адрес, если шаблон машины для вашей лаборатории еще не опубликован.

## <a name="conclusion"></a>Заключение

Теперь мы знаем публичный IP-адрес для лаборатории.  Правила входящих и исходящих могут быть созданы для брандмауэра организации для общедоступного IP-адреса и диапазона портов 49152-65535.  После обновления правил учащиеся могут получить доступ к своим вс-чам без сетевого брандмауэра, блокирующего доступ к ним.
