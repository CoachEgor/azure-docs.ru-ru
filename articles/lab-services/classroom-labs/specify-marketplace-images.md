# <a name="specify-marketplace-images-available-to-lab-creators"></a>Выбор образов Marketplace, доступных для создателей лаборатории
Являясь владельцем учетной записи лаборатории, вы можете указать образы Marketplace, которые можно использовать при создании лабораторий в учетной записи лаборатории. 

## <a name="select-images-available-for-labs"></a>Выбор образов, доступных для лабораторных занятий
В меню слева выберите **Образы Marketplace**. По умолчанию отображается полный список образов (включенных и отключенных). В раскрывающемся списке вверху можно отфильтровать список, чтобы увидеть только включенные или отключенные образы, выбрав **Только включенные**/**Только отключенные**. 
    
![Страница с образами Marketplace](../media/tutorial-setup-lab-account/marketplace-images-page.png)

В списке отображаются только образы Marketplace, которые соответствуют следующим условиям:
    
- позволяют создать одну виртуальную машину;
- используют диспетчер ресурсов Azure для подготовки виртуальных машин;
- не требуют приобретения дополнительного плана лицензирования.

## <a name="disable-images-for-a-lab"></a>Отключение образов для лаборатории 
Чтобы отключить один образ для лаборатории, выберите.. **. (многоточие)** в последнем столбце и выберите **отключить изображение**. 

![Отключение одного образа](../media/tutorial-setup-lab-account/disable-one-image.png) 

Кроме того, можно установить флажок перед именем образа и выбрать **Отключить выбранные изображения** на панели инструментов. 

Чтобы отключить несколько образов одновременно, установите флажки перед именами образов и выберите **Отключить выбранные изображения** на панели инструментов. 

![Отключение нескольких образов](../media/tutorial-setup-lab-account/disable-multiple-images.png) 


## <a name="enable-images-for-a-lab"></a>Включение образов для лаборатории
Чтобы включить отключенный образ, выберите **... (многоточие)** в последнем столбце выберите **включить изображение**. Кроме того, можно установить флажок перед именем образа и выбрать **включить выбранные изображения** на панели инструментов. 

Чтобы отключить несколько образов одновременно, установите флажки перед именами образов и выберите **включить выбранные изображения** на панели инструментов.
