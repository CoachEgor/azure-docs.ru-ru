# <a name="specify-marketplace-images-available-to-lab-creators"></a>Выбор образов Marketplace, доступных для создателей лаборатории
Являясь владельцем учетной записи лаборатории, вы можете указать образы Marketplace, которые можно использовать при создании лабораторий в учетной записи лаборатории. 

## <a name="select-images-available-for-labs"></a>Выберите изображения, доступные для лабораторий
В меню слева выберите **Образы Marketplace**. По умолчанию отображается полный список образов (включенных и отключенных). Вы можете отфильтровать список, чтобы увидеть только включенные/отключенные изображения, выбрав только опцию/ **«Включенные только****инвалиды»** из списка выпадающих в верхней части списка. 
    
![Страница с образами Marketplace](../media/tutorial-setup-lab-account/marketplace-images-page.png)

В списке отображаются только образы Marketplace, которые соответствуют следующим условиям:
    
- позволяют создать одну виртуальную машину;
- используют диспетчер ресурсов Azure для подготовки виртуальных машин;
- не требуют приобретения дополнительного плана лицензирования.

## <a name="disable-images-for-a-lab"></a>Отогиваемые изображения для лаборатории 
Чтобы отключить одно изображение для лаборатории, выберите **... (эллипсис)** в последнем столбце и выберите **изображение От".** 

![Отключение одного образа](../media/tutorial-setup-lab-account/disable-one-image.png) 

Кроме того, вы выбираете флажок перед именем изображения и выберете **выбранные изображения** на панели инструментов. 

Чтобы отключить несколько изображений одновременно, выберите флажки перед именами изображений и выберите **выбранные изображения** на панели инструментов. 

![Отключение нескольких образов](../media/tutorial-setup-lab-account/disable-multiple-images.png) 


## <a name="enable-images-for-a-lab"></a>Включить изображения для лаборатории
Для включения инвалидного изображения выберите **... (эллипсис)** в последнем столбце и выберите **изображение Включить.** Кроме того, вы выбираете флажок перед именем изображения и выберите **Выбранные изображения на** панели инструментов. 

Чтобы отключить несколько изображений одновременно, выберите флажки перед именами изображений и выберите **Выбранные изображения** на панели инструментов. 

## <a name="enable-images-at-the-time-of-lab-creation"></a>Включить изображения во время создания лаборатории
Вы можете включить больше изображений во время создания лаборатории: 

1. Войти на [веб-сайт Azure Lab Services](https://labs.azure.com) с использованием учетных данных владельца **учетной записи лаборатории**
2. Выберите изображение виртуальной машины по умолчанию или стрелку вниз. 
3. Выберите **Включить больше вариантов изображения.** 

    ![Включить больше параметров изображения](../media/specify-marketplace-images/enable-more-images-menu.png)
4. Следуйте инструкциям из предыдущего раздела, чтобы включить выбранные изображения. 
5. Возможно, потребуется закрыть **окно Новой лаборатории** и вновь открыть его, чтобы увидеть изображения, выбранные на предыдущем этапе.
