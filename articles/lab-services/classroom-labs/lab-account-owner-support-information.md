# <a name="set-up-support-information-lab-account-owner-in-azure-lab-services"></a>Настройка сведений о поддержке (владелец учетной записи лаборатории в службах лаборатории Azure)
В этой статье объясняется, как вы (в качестве владельца учетной записи лаборатории) можете указать сведения о поддержке, которые авторы лабораторий (преподаватели) и пользователи лаборатории (учащиеся) могут использовать для получения справки, если они работают с техническими проблемами с использованием учетной записи лаборатории или лаборатории. 

Сведения о поддержке включают:

- URL-адрес
- Электронная почта
- Номер телефона
- Дополнительные инструкции 

## <a name="specify-support-information"></a>Указание сведений о поддержке
1. Войдите на [портал Azure](https://portal.azure.com).
2. В строке поиска введите **службы лаборатории**и выберите **службы лаборатории** в результатах поиска. 
3. Выберите свою учетную запись лаборатории из списка учетных записей лаборатории. 
4. Перейдите на страницу **внутренняя поддержка** и выполните следующие действия.
    1. Введите **URL-адрес поддержки**. 
     2. Введите **адрес электронной почты для поддержки**. 
     3. Введите **Телефон службы поддержки**.
     4. Введите подробные **инструкции по поддержке** (необязательно). Владельцы и пользователи лаборатории увидят этот текст вместе с контактной информацией службы поддержки. URL-адреса будут автоматически включены в ссылки. 
     5. На панели инструментов щелкните **Сохранить**.
