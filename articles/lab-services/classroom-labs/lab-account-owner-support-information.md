# <a name="set-up-support-information-lab-account-owner-in-azure-lab-services"></a>Настройка информации о поддержке (владелец учетной записи лаборатории в Лабораторных службах Azure)
В этой статье объясняется, как вы (как владелец учетной записи лаборатории) можете указать информацию о поддержке, которую создатели лаборатории (инструкторы) и пользователи лаборатории (студенты) могут использовать для получения помощи, если они столкнулись с какими-либо техническими проблемами с использованием учетной записи лаборатории или лаборатории. 

Информация о поддержке включает в себя:

- URL-адрес
- Email
- Номер телефона
- Дополнительные инструкции 

## <a name="specify-support-information"></a>Указать информацию о поддержке
1. Войти на [портал Azure](https://portal.azure.com).
2. В панели поиска введите **лабораторные службы**и выберите **лабораторные службы** в результатах поиска. 
3. Выберите учетную запись лаборатории из списка лабораторных учетных записей. 
4. Переключитесь на страницу **внутренней поддержки,** сделайте следующие шаги:
    1. Введите **URL-адрес поддержки.** 
     2. Введите **электронную почту поддержки**. 
     3. Введите **телефон поддержки.**
     4. Введите **подробные инструкции поддержки** (необязательно). Владельцы лабораторий и пользователи увидят этот текст вместе с контактной информацией поддержки. URL-адреса будут автоматически превращаться в ссылки. 
     5. На панели инструментов щелкните **Сохранить**.
