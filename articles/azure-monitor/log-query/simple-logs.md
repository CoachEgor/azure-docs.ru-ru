# <a name="simple-logs-experience-in-azure-monitor-preview"></a>Простые журналы в Azure Monitor (Предварительный просмотр)
Azure Monitor предоставляет [богатый опыт](get-started-portal.md) создания [журналов с](log-query-overview.md) использованием языка КЗЛ. Вы не можете требовать полной мощности КЗЛ, хотя и предпочитают упрощенный опыт для основных требований запроса. Опыт простых журналов позволяет создавать основные запросы без прямого взаимодействия с КЗЛ. Вы также можете использовать простые журналы в качестве учебного пособия для КЗЛ, поскольку вам требуются более сложные запросы.

> [!NOTE]
> Простые журналы в настоящее время реализованы в качестве теста только для Космос DB и Key Vaults. Пожалуйста, поделитесь своим опытом с корпорацией Майкрософт через [User Voice,](https://feedback.azure.com/forums/913690-azure-monitor) чтобы помочь нам определить, будем ли мы расширять и выпускать эту функцию.


## <a name="scope"></a>Область
Интерфейс «Простые журналы» получает данные из таблицы *AzureDiagnostics,* *AzureMetrics*и *AzureActivity* для выбранного ресурса. 

## <a name="using-simple-logs"></a>Использование простых журналов
Перейдите к любому Космическому DB или Key Vault в подписке Azure с [диагностическими настройками, настроенными для сбора журналов в рабочем пространстве Log Analytics.](../platform/resource-logs-collect-storage.md) Нажмите **журналы** в меню **мониторинга,** чтобы открыть простой опыт журналов.

![Меню](media/simple-logs/menu.png)

Выберите **поле** и **оператора** и укажите **значение** для сравнения. Нажмите **+** и укажите **И/Или** добавить дополнительные критерии.

![Критерии](media/simple-logs/criteria.png)

Нажмите **Выполнить,** чтобы просмотреть результаты запроса.

## <a name="view-and-edit-kql"></a>Просмотр и отодевать КЗЛ
Выберите **редактор запроса,** чтобы открыть запрос «Простые журналы» в полном опыте анализа журналов. 

![Редактор запросов](media/simple-logs/query-editor.png)

Вы можете непосредственно отсеивать КЗЛ и использовать другие функции в журнале Analytics, такие как фильтры для дальнейшего уточнения результатов.

![Элит КЗЛ](media/simple-logs/edit-kql.png)
