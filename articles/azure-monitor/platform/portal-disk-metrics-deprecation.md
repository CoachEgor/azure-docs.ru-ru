# <a name="disk-metrics-deprecation-in-the-azure-portal"></a>Амортизация дисковых метрик на портале Azure

Удаляемые метрики, связанные с диском, вскоре будут удалены с портала Azure. Доступна новая версия каждой униженные метрики. В этой статье показано, какие метрики являются новыми и как обновить метик оповещения, чтобы использовать их.

## <a name="list-of-new-metrics"></a>Список новых метрик

Эта таблица отображает каждый изунимых метрик и соответствует новой метрике. 

|Обезображенный метрики|Новая (замена) метрики|
|----|----|
|Диск данных QD (Обезвращенный)|Глубина очереди диска данных (Предварительный просмотр)|
|Диск данных Читайте Байты/Сек (Обезвращенный)|Скорость чтения с диска данных (байт/с) (предварительная версия)|
|Операции чтения диска данных/Sec (Обезвращено)|Скорость чтения с диска данных (операций/с) (предварительная версия)|
|Диск данных Написать Байты / Sec (Обезвреженный)|Скорость записи на диск данных (байт/с) (предварительная версия)|
|Операции записи диска данных/Sec (Обезвреженные)|Скорость записи на диск данных (операций/с) (предварительная версия)|
|Os QD (Обезображено)|Глубина очереди ОС (Предварительный просмотр)|
|OS Читать Байты / Sec (Обезвоченный)|OS Читать Байты / Sec (Предварительный просмотр)|
|Операции чтения OS/Sec (Унипрачено)|Операции чтения OS/Sec (Предварительный просмотр)|
|OS Написать Байты / Sec (Обезображенные)|OS Написать Байты / Sec (Предварительный просмотр)|
|ОПЕРАЦИИ записи OS/Sec (Обезвреженные)|Операции записи OS/Sec (Предварительный просмотр)|

## <a name="migrate-metrics-in-your-metric-alerts"></a>Мигрировать метрики в метеоданных

Обновите метеосигналы, чтобы использовать новые метрики.

1. На портале Azure **ищите оповещения.** Затем, в разделе **Услуги,** выберите оповещения . **Services**

   > [!div class="mx-imgBorder"]
   > ![Описание изображения](./media/portal-disk-metrics-deprecation/alert-service-azure-portal.png)

2. На странице **оповещения** выберите кнопку **«Правила управления».** 

   > [!div class="mx-imgBorder"]
   > ![Описание изображения](./media/portal-disk-metrics-deprecation/manage-alert-rules-button.png)

3. В списке выпадающих групп **ресурсов** выберите флажок **виртуальных машин,** а в списке выпадающих сигналов **—** выберите флажок **Метрики.** 

   > [!div class="mx-imgBorder"]
   > ![Описание изображения](./media/portal-disk-metrics-deprecation/filter-alerts.png)

4. В списке метрик определите условия, связанные с дисками. Нажмите на название правила. 

   Имя отображается как гиперссылка в столбце **«Имя»** таблицы.

   > [!div class="mx-imgBorder"]
   > ![Описание изображения](./media/portal-disk-metrics-deprecation/find-disk-conditions.png)

5. В разделе **«Условия»** на странице **управления Правилами** щелкните состояние оповещения. 

   Условие отображается как гиперссылка.  

   > [!div class="mx-imgBorder"]
   > ![Описание изображения](./media/portal-disk-metrics-deprecation/adjust-condition.png)

   Появляется страница **логики сигнала Настройка и** настройки состояния в разделе **логики оповещения** на этой странице.

6. Сделайте запись этих параметров, как они исчезнут, когда вы удалите изунимые метрики.

   > [!div class="mx-imgBorder"]
   > ![Описание изображения](./media/portal-disk-metrics-deprecation/condition-rules.png)

   > [!TIP] 
   > Рассмотрите возможность захвата этих параметров в скриншоте или в текстовом файле. 

7. Нажмите на ссылку **обратного выбора сигнала.**

   > [!div class="mx-imgBorder"]
   > ![Описание изображения](./media/portal-disk-metrics-deprecation/back-to-signal-selection.png)

8. На странице **логики сигнала Настройка** выбрать соответствующую метрику замены (новая метрика). Используйте [таблицу,](#update-metrics) которая появляется ранее в этой статье, чтобы установить имя новой метрики.

   > [!TIP] 
   > Начните вводить в бар поиска, чтобы сузить список имен метрик. 

   > [!div class="mx-imgBorder"]
   > ![Описание изображения](./media/portal-disk-metrics-deprecation/choose-new-metric.png)

9. Выберите кнопку **«Сделано».** 

   > [!div class="mx-imgBorder"]
   > ![Описание изображения](./media/portal-disk-metrics-deprecation/set-new-metric.png)

10. Совершайте изменения, выбрав кнопку **«Сохранить».** 

    > [!div class="mx-imgBorder"]
    > ![Описание изображения](./media/portal-disk-metrics-deprecation/save-new-metric.png)
