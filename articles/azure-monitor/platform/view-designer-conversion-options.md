# <a name="azure-monitor-view-designer-to-workbooks-conversion-options"></a>Дизайнер представления Azure Monitor для вариантов преобразования трудовых книжек
[Дизайнер представления](view-designer.md) — это функция Azure Monitor, которая позволяет создавать пользовательские представления, чтобы помочь вам визуализировать данные в рабочем пространстве Log Analytics с диаграммами, списками и сроками. Они постепенно сворачиваются и заменяются трудовыми книжеками, которые обеспечивают дополнительную функциональность. В этой статье сравниваются фундаментальные понятия между ними и варианты преобразования представлений в трудовые книжки.

## <a name="basic-workbook-designs"></a>Основные проекты книжки

Дизайнер представления имеет фиксированный статический стиль представления, в то время как трудовые книжки позволяют свободно включать и изменять представление данных. На опубликованных ниже изображениях показаны два примера того, как можно организовать трудовые книжки при преобразовании представлений.

## <a name="tile-conversion"></a>Преобразование плитки
Дизайнер представления использует функцию плитки обзора для представления и обобщения общего состояния. Они представлены в семи плитках, начиная от чисел и диаграмм. В трудовых книжках пользователи могут создавать аналогичные визуализации и прикреплять их к оригинальному стилю обзорных плиток. 

![Коллекции](media/view-designer-conversion-options/overview.png)


## <a name="view-dashboard-conversion"></a>Просмотр преобразования панели мониторинга
Тип дизайнера плитки обычно состоят из двух разделов, визуализации и списка, который соответствует данным из визуализации, **например, плитки Donut & List.**

![кольцевой график;](media/view-designer-conversion-options/donut-example.png)

С помощью трудовых книжек мы позволяем пользователю выбрать запрос одного или обоих разделов представления. Формулирование запросов в трудовых книжках является простым двухэтапным процессом. Во-первых, данные генерируются из запроса, а во-вторых, данные отображаются как визуализация.  Пример того, как это представление будет воссоздано в трудовых книжках, заключается в следующем:
