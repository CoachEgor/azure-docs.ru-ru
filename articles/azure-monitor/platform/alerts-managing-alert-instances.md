# <a name="manage-alert-instances"></a>Управление экземплярами оповещений
С помощью [унифицированного интерфейса оповещений](https://aka.ms/azure-alerts-overview) Azure Monitor можно увидеть все типы оповещений Azure, охватывая несколько подписок, на одной панели. В этой статье рассказывается, как можно просмотреть свои экземпляры оповещения и где на портале найти конкретные экземпляры оповещений для устранения неполадок.

1. Есть три способа перехода на 	страницу "Оповещения"

   + На [портале](https://portal.azure.com/) выберите **Мониторинг**, а затем в разделе "Мониторинг" выберите **Оповещения**.  
     ![Мониторинг](media/alerts-managing-alert-instances/monitoring-alerts-managing-alert-instances-toc.jpg)
  
   + К странице"Оповещения" можно перейти из контекста определенного **ресурса**. Как только ресурс откроется, перейдите по его содержимому в раздел "Мониторинг" и выберите **Оповещения**, при этом целевая страница будет предварительно отфильтрована для оповещений конкретно этого ресурса.
   
     ![Мониторинг](media/alerts-managing-alert-instances/alert-resource.JPG)
    
   + К странице "Оповещения" можно перейти из контекста определенной **группы ресурсов**. Как только откроется группа ресурсов, перейдите по ее содержимому в раздел "Мониторинг" и выберите **Оповещения**, при этом целевая страница будет предварительно отфильтрована для оповещений конкретно этой группы ресурсов.    
   
     ![Мониторинг](media/alerts-managing-alert-instances/alert-rg.JPG)

1. Вы перейдете на страницу **Сводка по оповещениям**, которая предоставляет общие сведения о всех экземплярах оповещений в Azure. Режим общего просмотра можно изменить, выбрав **несколько подписок** (максимум 5) или путем фильтрации по **группам ресурсов**, определенным **ресурсам** или **диапазонам времени**. Чтобы перейти к списку просмотра оповещений, щелкните "Все оповещения" или любую группу серьезности.     
   ![Сводка по оповещениям](media/alerts-managing-alert-instances/alerts-summary.jpg)
 
1. На странице **Все оповещения** можно увидеть все перечисленные оповещения в Azure. На портале с уведомлением об оповещениях можно использовать фильтры, чтобы сократить их количество для этого конкретного экземпляра оповещения. (**Примечание**. Щелкнув по любой из групп серьезности при входе на страницу, список предварительно будет отфильтрован по критерию серьезности). Помимо фильтров, доступных на предыдущей странице, вы также можете фильтровать на основе службы мониторинга (например, Платформа для метрики), системы контроля (сработано или разрешено), степени серьезности, состояния оповещения (новое/известное/закрытое) или идентификатора интеллектуальной группы.

   ![Все оповещения.](media/alerts-managing-alert-instances/all-alerts.jpg)

   > [!NOTE]
   >  Щелкнув по любой из групп серьезности при входе на страницу, список на этой странице предварительно отфильтруется по критерию серьезности.
 
1. Щелкнув на любой экземпляр оповещения, откроется страница **Сведения об оповещении**, позволяя в деталях узнать информацию о конкретном экземпляре оповещения.   
   ![Сведения об оповещении](media/alerts-managing-alert-instances/alert-details.jpg)
