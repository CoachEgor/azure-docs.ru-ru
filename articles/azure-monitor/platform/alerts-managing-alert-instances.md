# <a name="manage-alert-instances-with-unified-alerts"></a>Управление экземплярами оповещения с помощью унифицированных оповещений

С [помощью унифицированных оповещений](https://aka.ms/azure-alerts-overview) в Azure Monitor можно увидеть все различные типы оповещений в Azure. Это охватывает несколько подписок, в одном стекол. В этой статье показано, как можно просматривать экземпляры оповещения и как найти конкретные экземпляры оповещения для устранения неполадок.

> [!NOTE]
> Вы можете получить доступ только к оповещениям, генерируемым за последние 30 дней.

## <a name="go-to-the-alerts-page"></a>Перейдите на страницу оповещения

Вы можете перейти на страницу оповещения любым из следующих способов:

- На [портале Azure](https://portal.azure.com/)выберите **Monitor** > **Alerts.**  

     ![Скриншот монитора оповещения](media/alerts-managing-alert-instances/monitoring-alerts-managing-alert-instances-toc.jpg)
  
- Используйте контекст определенного ресурса. Откройте ресурс, перейдите в раздел **Мониторинг** и выберите **оповещения.** Целевая страница предварительно отфильтровывается для оповещений на этом конкретном ресурсе.

     ![Скриншот оповещения о мониторинге ресурсов](media/alerts-managing-alert-instances/alert-resource.JPG)

- Используйте контекст определенной группы ресурсов. Откройте группу ресурсов, перейдите в раздел **Мониторинг** и выберите **оповещения.** Целевая страница предварительно отфильтровывается для оповещений в этой конкретной группе ресурсов.    

     ![Скриншот ресурсной группы Мониторинг оповещения](media/alerts-managing-alert-instances/alert-rg.JPG)

## <a name="find-alert-instances"></a>Поиск экземпляров оповещения

Страница **«Резюме предупреждений»** дает обзор всех экземпляров оповещения в Azure. Можно изменить сводной представление, выбрав **несколько подписок** (максимум до 5) или отфильтровав **группы ресурсов,** определенные **ресурсы**или **временные диапазоны.** Выберите **Всего оповещения**, или любой из диапазонов серьезности, чтобы перейти к представлению списка для ваших предупреждений.     

![Скриншот страницы "Оповещение"](media/alerts-managing-alert-instances/alerts-summary.jpg)
 
На странице **All Alerts перечислены** все экземпляры оповещения в Azure. На портале с уведомлением об оповещениях можно использовать фильтры, чтобы сократить их количество для этого конкретного экземпляра оповещения.

> [!NOTE]
> Если вы пришли на страницу, выбрав какую-либо из полос серьезности, список предварительно отфильтровывается для этой серьезности.

Помимо фильтров, доступных на предыдущей странице, можно также фильтровать на основе службы мониторинга (например, платформы для метрик), состояния монитора (выстрелил или решен), тяжести, состояния оповещения (новый/признанный/закрытый) или идентификатор смарт-группы.

![Скриншот страницы «Все оповещения»](media/alerts-managing-alert-instances/all-alerts.jpg)

> [!NOTE]
> Если вы пришли на страницу, выбрав какую-либо из полос серьезности, список предварительно отфильтровывается для этой серьезности.

Выбор любого экземпляра оповещения открывает страницу **«Подробности оповещения»,** что позволяет увидеть более подробную информацию об этом конкретном экземпляре оповещения.   

![Скриншот страницы оповещения Подробная информация](media/alerts-managing-alert-instances/alert-details.jpg)
