# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Просмотр аналитических данных для метрик ресурсов веб-приложений Azure

![Символ веб-приложений](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Решение по анализу веб-приложений Azure перешло на поддержку сообщества.  
>- Это решение больше недоступно в Azure Marketplace, но его можно установить из [шаблонов быстрого запуска Azure](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) (его поддержка осуществляется сообществом).
>- Пользователи, которые уже установили это решение, смогут работать с ним с определенными изменениями.  
>- Корпорация Майкрософт рекомендует выполнять мониторинг веб-приложений с помощью [Application Insights](../../azure-monitor/app/app-insights-overview.md).
