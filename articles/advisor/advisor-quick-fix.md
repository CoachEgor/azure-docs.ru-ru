# <a name="quick-fix-remediation-for-advisor"></a>Быстрая исправление исправление для советника
**Быстрый Fix** позволяет быстрее и проще способ исправления для рекомендации на нескольких ресурсах. Он обеспечивает возможность массовых исправлений ресурсов и помогает вам оптимизировать подписку быстрее с исправлением в масштабе для ваших ресурсов.
Функция доступна только для определенных рекомендаций, через портал Azure.


## <a name="steps-to-use-quick-fix"></a>Шаги, чтобы использовать 'Быстрый Fix'

1. Из списка рекомендаций, которые имеют ярлык **Быстрого Fix,** нажмите на рекомендацию.

   ![Советник быстро исправить](./media/quick-fix-1.png)
   
   *Цены на изображении, например, только*

2. На странице рекомендаций по деталям вы увидите список ресурсов, для которых у вас есть эта рекомендация. Выберите все ресурсы, которые вы хотите исправления для рекомендации.

   ![Советник быстро исправить](./media/quick-fix-2.png)
   
   *Цены на изображении, например, только*

3. После того как вы выбрали ресурсы, нажмите на кнопку **Быстрый Fix** навалом исправление.

   > [!NOTE]
   > Некоторые из перечисленных ресурсов могут быть отключены, так как у вас нет соответствующих разрешений на их изменение.
   
   > [!NOTE]
   > Если есть и другие последствия, в дополнение к преимуществам, упомянутым в Advisor, вы будете сообщены в опыте, чтобы помочь вам принять обоснованные решения по исправлению положения.
   
4. Вы получите уведомление о завершении исправления. Вы увидите ошибку, если в представлении списка ресурсов есть ресурсы, которые не исправлены, и ресурсы в выбранном режиме.
