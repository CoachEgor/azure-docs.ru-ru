# <a name="configure-periodic-summary-for-recommendations"></a>Настройка периодического резюме для рекомендаций

**Рекомендации** советника обеспечивают простой и упреждающий способ оставаться на вершине ваших активных рекомендаций, в различных категориях. Функция обеспечивает возможность настройки периодических уведомлений для резюме всех ваших активных рекомендаций, в разных категориях. Вы можете выбрать нужный канал для уведомлений, таких как электронная почта, sms или другие, используя группы действий. В этой статье показано, как настроить **рекомендации дайджесты** для рекомендаций консультанта.


## <a name="setting-up-your-recommendation-digest"></a>Настройка рекомендательного дайджеста 

Опыт создания **рекомендаций для дайджеста** поможет настроить резюме. Вы можете выбрать ниже параметры для конфигураций:
1. Категория: У нас есть категории рекомендаций, такие как стоимость, высокая доступность, производительность и операционное совершенство. Возможности пока не доступны для рекомендаций по безопасности.
2. Частота дайджеста: Частота сводных уведомлений может быть еженедельной, раз в две недели и ежемесячно.
3. Группа действий: Вы можете выбрать существующую группу действий или создать новую группу действий. Чтобы узнать больше о группах действий, [см.](https://docs.microsoft.com/azure/azure-monitor/platform/action-groups)
4. Язык для дайджеста
5. Рекомендация дайджест имя: Вы можете использовать удобный строку, чтобы лучше отслеживать и контролировать дайджесты.

## <a name="steps-to-create-recommendation-digest-in-azure-portal"></a>Шаги для создания рекомендательного дайджеста на портале Azure

Вот шаги, чтобы создать **рекомендацию дайджест:**
* **Шаг 1:** На портале Azure перейдите в разделе **«Советник»** и раздел **«Мониторинг»,** выберите **дайджест Рекомендации** 

   ![Рекомендация дайджест точки входа](./media/digest-0.png)

* **Шаг 2:** Выберите **новый рекомендательный дайджест** из верхней панели, как ниже:

   ![Создание рекомендательного дайджеста](./media/digest-5.png)

* **Шаг 3:** В разделе **области** выберите **подписку** для дайджеста

   ![Предоставление рекомендаций дайджестов](./media/digest-1.png)

* **Шаг 4:** В разделе **состояние** выберите конфигурации, такие как **категория,** **частота** и **язык**

   ![Обеспечить рекомендации дайджест условий ввода](./media/digest-2.png)

* **Шаг 5:** В разделе **группы действий** выберите **группу действий** для дайджеста. Вы можете узнать больше здесь - [Создание и управление группами действий](https://docs.microsoft.com/azure/azure-monitor/platform/action-groups)

   ![Обеспечить рекомендации дайджест группы действий действий](./media/digest-3.png)

* **Шаг 6:** В этом заключительном разделе для **дайджеста детали**, вы можете назначить имя и состояние в ашего рекомендации дайджест. Пресс **создать рекомендации дайджест** для завершения настройки.
   ![Полное создание рекомендаций дайджест](./media/digest-4.png)
