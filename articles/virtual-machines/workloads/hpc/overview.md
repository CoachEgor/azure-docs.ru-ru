# <a name="high-performance-computing-on-h-series-vms"></a>Высокопроизводительные вычисления на виртуальных машинах серии H

Высокопроизводительные вычисления (HPC) на виртуальных машинах серии HB, и HC обеспечивают наиболее оптимизированную производительность HPC на любых виртуальных машинах в Azure. Виртуальные машины, оптимизированные с помощью HPC, используются для решения некоторых из самых сложных математических задач, таких как: гидродинамика, модели для нефтяной и газовой промышленности и моделирование погоды.

В этой статье рассматриваются некоторые ключевые функции виртуальных машин серии HB и HC, почему эти виртуальные машины хорошо работают в сценариях HPC и как начать работу.

## <a name="features-and-capabilities"></a>Функции и возможности

Виртуальные машины серии HB и HC предназначены, чтобы обеспечить максимальную производительность HPC, масштабируемость интерфейса передачи сообщений (MPI) и эффективность затрат для рабочих нагрузок HPC.

### <a name="message-passing-interface"></a>Интерфейс передачи сообщений

Серии HB и HC поддерживают практически все типы и версии MPI. Некоторые из наиболее распространенных поддерживаемых типов MPI: OpenMPI, MVAPICH2, платформа MPI, Intel MPI и все команды удаленного доступа к памяти (RDMA). Дополнительные сведения см. в разделе [Настройка интерфейса передачи сообщений для HPC](setup-mpi.md).

### <a name="rdma-and-infiniband"></a>RDMA и InfiniBand

Интерфейс RDMA входит в стандартную комплектацию виртуальных машин серии HB и HC. Экземпляры с поддержкой RDMA обмениваются данными через сеть InfiniBand, работая для виртуальных машин серии HB и HC на повышенных скоростях передачи данных (EDR). Экземпляры с поддержкой RDMA позволяют повысить масштабируемость и производительность некоторых приложений MPI.

Виртуальные машины серии HB и HC, поддерживающие конфигурацию InfiniBand, являются неблокируемыми утолщенными деревьями с небольшим размером программной архитектуры для согласованной производительности RDMA.

Дополнительные сведения о настройке InfiniBand на виртуальных машинах серии HB или HC, см. в разделе [Включение InfiniBand](enable-infiniband.md).

## <a name="get-started"></a>Начало работы

Во-первых решите, какие виртуальные машины серии H вы собираетесь использовать. Сведения о виртуальных машинах, оптимизированных с помощью HPC, см. в разделе [Обзор серии HB](hb-series-overview.md) и [Обзор серии HC](hc-series-overview.md). Спецификации см. в разделе [Размеры виртуальных машин, оптимизированных для высокопроизводительных вычислений](https://docs.microsoft.com/azure/virtual-machines/linux/sizes-hpc).

После выбора и создания виртуальной машины для приложения, необходимо настроить ее, включив InfiniBand. Сведения о включении InfiniBand на виртуальных машинах Linux и Windows, см. в разделе [Включение InfiniBand](enable-infiniband.md).

MPI является важной составляющей рабочих нагрузок HPC. Серии HB и HC поддерживают практически все типы и версии MPI. Дополнительные сведения см. в разделе [Настройка интерфейса передачи сообщений для HPC](setup-mpi.md).

После выбора серии виртуальных машин, настройте Infiniband и MPI. Теперь вы готовы приступить к созданию рабочих нагрузок HPC.
