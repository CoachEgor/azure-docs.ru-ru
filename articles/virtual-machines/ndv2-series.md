# <a name="updated-ndv2-series"></a>Обновленная серия NDv2

Виртуальная машина серии NDv2 является новым дополнением к семейством GPU, предназначенному для нужд самых требовательных GPU-ускоренный ИИ, машинного обучения, моделирования и рабочих нагрузок HPC.

NDv2 оснащается 8 NVIDIA Tesla V100 NVLINK подключенных gpUs, каждый с 32 ГБ памяти GPU. Каждый NDv2 VM также имеет 40 ядер Intel Xeon Platinum 8168 (Skylake) и 672 GiB системной памяти.

Экземпляры NDv2 обеспечивают отличную производительность для hPC и AI рабочих нагрузок с использованием CUDA GPU оптимизированных вычислительных ядер, а также многих инструментов ИИ, ML и аналитики, которые поддерживают ускорение GPU 'out-of-box', такие как TensorFlow, Pytorch, Caffe, RAPIDS и другие Рамки.

Критически, NDv2 построен как для вычислительно интенсивного масштабирования (использование 8 графических процессоров на ВМ) и масштабирования (использование нескольких ВМ, работающих вместе) рабочих нагрузок. Серия NDv2 теперь поддерживает 100-гигабит InfiniBand EDR бэкэнд сетей, аналогичные той, которая доступна на HB серии HPC VM, чтобы высокопроизводительные кластеризации для параллельных сценариев, включая распределенное обучение для ИИ и ML. Эта бэкэнд-сеть поддерживает все основные протоколы InfiniBand, включая те, которые используются в библиотеках NVIDIA NCCL2, что позволяет беспрепятственно кластерить графические процессоры.

> При [включении InfiniBand](https://docs.microsoft.com/azure/virtual-machines/workloads/hpc/enable-infiniband) на ND40rs_v2 VM, пожалуйста, используйте 4.7-1.0.0.1 Mellanox OFED водителя.
>
> Из-за увеличения памяти GPU, новый ND40rs_v2 VM требует использования [поколения 2 VMs](https://docs.microsoft.com/azure/virtual-machines/windows/generation-2) и изображений рынка. 
>
> Пожалуйста, обратите внимание: ND40s_v2 с 16 ГБ памяти за gPU больше не доступен для предварительного просмотра и был заменен обновленной ND40rs_v2.

## <a name="supported-operating-systems-and-drivers"></a>Поддерживаемые операционные системы и драйверы

Чтобы воспользоваться возможностями графического процессора VMs серии Azure N, необходимо установить драйверы графического процессора NVIDIA.

[Расширение драйвера GPU NVIDIA](./extensions/hpccompute-gpu-linux.md) устанавливает необходимые драйверы CUDA или GRID NVIDIA на виртуальную машину серии N. Для установки расширения и управления им можно использовать портал Azure или такие инструменты, как Azure PowerShell и шаблоны Azure Resource Manager. Общие сведения о расширениях виртуальных машин см. в статье [Расширения и компоненты виртуальных машин Azure](./extensions/overview.md).

Если вы решите установить драйверы NVIDIA GPU вручную, [см.](./linux/n-series-driver-setup.md)
