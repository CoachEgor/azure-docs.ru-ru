# <a name="av2-series"></a>Серия Av2

Виртуальные машины серии Av2 можно развертывать с использованием оборудования и процессоров разных типов. VMs серии Av2 имеют конфигурации производительности процессора и памяти, наиболее подходящие для рабочих нагрузок начального уровня, таких как разработка и тестирование. Размер регулируется, чтобы обеспечить согласованную производительность процессора для запущенного экземпляра, независимо от используемого оборудования. Чтобы определить физическое оборудование, на котором развернут определенный размер, отправьте запрос к виртуальному оборудованию из виртуальной машины. Некоторые примеры использования включают серверы разработки и тестирования, веб-серверы с низким трафиком, базы данных малого и среднего, коррективы и репозитории кода.
