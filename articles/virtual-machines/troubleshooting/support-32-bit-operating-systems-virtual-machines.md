# <a name="support-for-32-bit-operating-systems-in-azure-virtual-machines"></a>Поддержка 32-разрядных операционных систем на виртуальных машинах Azure

Microsoft Azure теперь позволяет пользователям приносить свои 32-разрядные операционные системы Windows в Azure. Поскольку некоторые из этих операционных систем уже достигли соглашения о жизнеобеспечении, корпорация Майкрософт может не предложить им дополнительную поддержку. Поддержка также не предлагается для операционных систем на основе Linux или Berkeley Software Distribution (BSD), работающих на виртуальной машине Microsoft Azure (VM).
