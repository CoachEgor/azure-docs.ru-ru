# <a name="prepay-for-azure-software-plans"></a>Предварительная оплата планов программного обеспечения Azure

Когда предоплаты за использование программного обеспечения SUSE и RedHat в Azure, можно сэкономить по сравнению с вашей оплатой. Скидки применяются только в том случае, SUSE и RedHat метров, а не на использование виртуальных машин. Вы можете приобрести резервирование для виртуальных машин отдельно для дополнительной экономии.

Вы можете приобрести SUSE и RedHat планы программного обеспечения на портале Azure. Чтобы приобрести план нужно сделать следующее.

- Необходимо иметь роль владельца для по крайней мере один Enterprise или отдельные подписки с оплатой.
- Для подписок Enterprise **Добавление зарезервированных экземпляров** параметр должен быть включен в [портал EA](https://ea.azure.com/). Если этот параметр отключен, необходимо быть администратором EA для подписки.
- Для программы поставщика облачных решений (CSP) агенты администрирования или агентов по продажам можно приобрести планы программного обеспечения.

## <a name="buy-a-software-plan"></a>Приобрести план программного обеспечения

1. Войдите на портал Azure и перейдите к [резервирования](https://portal.azure.com/#blade/Microsoft_Azure_Reservations/ReservationsBrowseBlade).
2. Нажмите кнопку **добавить** и затем выберите план программного обеспечения, который вы хотите приобрести.
Заполните обязательные поля. Любой виртуальной Машины SUSE Linux или виртуальной Машины RedHat, соответствующий атрибуты вы приобретаете получает скидку. Фактическое количество развертываний, получающих скидку, зависит от области и выбранного количества.
3. Выберите подписку. Он используется для оплаты для плана.
Метод оплаты подписки будет взиматься плата первоначальных затрат для резервирования. Тип подписки должно быть соглашение Enterprise Agreement (предлагают номера: MS-AZR-0017P или MS-AZR - 0148 P) или отдельного соглашения с оплатой по (предлагают номера: MS-AZR-0003P или MS-AZR 0023P).
    - Для подписки с соглашением Enterprise плата вычитается из баланса денежных обязательств или относится к избыточным расходам.
    - Для отдельной подписки с оплатой плата взимается на подписку метод оплаты кредитной карты или по счету.
4. Выберите область. Область резервирования может охватывать одну или несколько подписок (общая область).
    - Одна подписка — скидка на план применяется к соответствующим использования в подписке.
    - Shared - применении скидки на план для поиска совпадающих экземпляров во всех подписках в текущем контексте выставления счетов. Для корпоративных клиентов контекст выставления счетов регистраций и включает все подписки для регистрации. Для отдельных плана с оплатой по мере использования клиентами ценообразования контекст выставления счетов — это все отдельные планы с оплатой по мере использования ценообразования подписки, созданные администратором учетной записи.
5. Выберите продукт, чтобы выбрать размер виртуальной Машины и тип изображения. Скидка применяется к только выбранный размер виртуальной Машины.
6. Выбор условия один год или три года.
7. Выберите количество, являющееся числом экземпляров предоплаченные виртуальных Машин, которые могут получить скидку.
8. Добавьте продукт для покупок, проверку и покупки.

Скидка на резервирование автоматически применяется к индикаторе программного обеспечения, который вы предварительно оплачиваете. Плата за вычислительные ресурсы виртуальной Машины, этот план не действителен. Вы можете приобрести резервирование виртуальной Машины отдельно.

## <a name="discount-applies-to-different-suse-vm-sizes"></a>Скидка применяется к различных размеров виртуальных Машин SUSE

Как зарезервированные экземпляры виртуальных Машин SUSE Linux: планы обеспечивают гибкость размер экземпляра. Скидку применяется даже в том случае, если развернуть виртуальную Машину, которая является другой размер из SUSE плана, который вы приобрели. Дополнительные сведения см. в разделе [сведения о применении скидки план программного обеспечения](../../billing/billing-understand-suse-reservation-charges.md).

## <a name="redhat-plan-discount"></a>Скидка плана RedHat

Планы доступны только для виртуальных машин Red Hat Enterprise Linux. Скидка не применяется к RedHat Enterprise Linux SAP HANA виртуальных машин или RedHat Enterprise Linux бизнеса приложения виртуальных машин SAP.

RedHat плана скидки применяются только к размер виртуальной Машины, выбранный во время покупки. Невозможно компенсируются или обмен после приобретения планы RHEL.


## <a name="cancellation-and-exchanges-not-allowed"></a>Невозможность отмены и обмена

Ее нельзя отменить или exchange SUSE или RedHat план, который вы приобрели. Проверьте использование, чтобы убедиться в том, что вы приобретаете подходящей план. Чтобы определить, что нужно приобрести помощь, см. в разделе [сведения о применении скидки план программного обеспечения](../../billing/billing-understand-suse-reservation-charges.md).
