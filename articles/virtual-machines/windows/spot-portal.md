# <a name="preview-deploy-spot-vms-using-the-azure-portal"></a>Предварительный просмотр: Развертывание Spot VMs с помощью портала Azure

Использование [Spot VM](spot-vms.md) позволяет воспользоваться нашими неиспользованными мощностями при значительной экономии средств. В любой момент времени, когда Azure требуется обратной мощности, инфраструктура Azure будет выселять Spot VMs. Таким образом, Spot VMs отлично подходят для рабочих нагрузок, которые могут обрабатывать перерывы, такие как задания по обработке пакетов, среды для разработки разработчиков/тестов, большие вычислительные нагрузки и многое другое.

Цены на Spot VMs является переменной, на основе региона и SKU. Для получения дополнительной информации [Linux](https://azure.microsoft.com/pricing/details/virtual-machines/linux/) см. [Windows](https://azure.microsoft.com/pricing/details/virtual-machines/windows/) Для получения дополнительной информации об установлении максимальной цены, см [Spot VMs - Цены](spot-vms.md#pricing).

У вас есть возможность установить максимальную цену, которую вы готовы платить, в час, для VM. Максимальная цена на Spot VM может быть установлена в долларах США (USD), используя до 5 десятичных мест. Например, значение `0.05701`будет максимальной ценой $0.05701 USD в час. Если вы установите максимальную цену, чтобы быть, `-1`VM не будет выселен в зависимости от цены. Цена для VM будет текущая цена на спотовую или цену за стандартный VM, который всегда меньше, до тех пор, пока есть емкость и квота.

## <a name="create-the-vm"></a>Создание виртуальной машины

Процесс создания VM, используюго Spot VMs, такой же, как и подробный в [quickstart.](quick-create-portal.md) При развертывании VM можно использовать экземпляр пятна Azure.


На **вкладке «Основы»** в разделе **детали instance** **—«Нет»** — это значение по умолчанию для использования экземпляра пятна Azure.

![Захват экрана для выбора нет, не используйте экземпляр пятна Azure](media/spot-portal/no.png)

Это вы выбираете **Да**, раздел расширяется, и вы можете выбрать [свой тип выселения и политики выселения.](spot-vms.md#eviction-policy) 

![Захват экрана для выбора да, используйте экземпляр пятна Azure](media/spot-portal/yes.png)
