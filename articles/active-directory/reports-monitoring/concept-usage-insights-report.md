# <a name="usage-and-insights-report-in-the-azure-active-directory-portal"></a>Отчет об использовании и информации на портале Active Directory Azure

С помощью отчета об использовании и информации можно получить ориентированное на приложение представление данных о входе в систему. Ответы на следующие вопросы можно найти:

*   Каковы наиболее часто используемые приложения в моей организации?
*   Какие приложения имеют наиболее неудачных входе в систему? 
*   Каковы основные ошибки ввходе для каждого приложения?

## <a name="prerequisites"></a>Предварительные требования 

Для доступа к данным из отчета об использовании и информации необходимо:

* клиент Azure AD;
* Премиум-премия Azure AD (P1/P2) для просмотра данных ввне
* Пользователь глобального администратора, администратор безопасности, читатель безопасности или отчетостого чтения ролей. Кроме того, любой пользователь (не-админы) может получить доступ к своим собственным входным входов. 

## <a name="access-the-usage-and-insights-report"></a>Доступ к отчету об использовании и информации

1. Перейдите на [портал Azure](https://portal.azure.com).
2. Выберите правильный каталог, а затем выберите **Активный каталог Azure** и выберите **приложения Enterprise.**
3. Из раздела **Activity** выберите **сведения об использовании &,** чтобы открыть отчет. 

![Отчет об использовании и аналитике](./media/concept-usage-insights-report/main-menu.png)
                                     

## <a name="use-the-report"></a>Использование отчета

Отчет об использовании и информации показывает список приложений с одним или несколькими в попытке знака и позволяет сортировать по количеству успешных вводов, неудачным входе и показателю успеха.

Нажатие нагрузки больше в нижней части списка позволяет просматривать дополнительные приложения на странице. Вы можете выбрать диапазон дат для просмотра всех приложений, которые были использованы в диапазоне.

Вы также можете настроить фокус на конкретном приложении. Выберите **действие ввходной** связи представления, чтобы увидеть знак в действии с течением времени для приложения, а также верхние ошибки.  

При выборе дня в графике использования приложения вы получаете подробный список действий для приложения.  

![Отчет об использовании и аналитике](./media/concept-usage-insights-report/usage-and-insights-report.png)
