# <a name="single-page-application-app-registration"></a>Одностраничное приложение: Регистрация приложений

На этой странице объясняется специфика регистрации приложения для одностраничного приложения (SPA).

Выполните последующие действия по [регистрации нового приложения на платформе майкрософт](quickstart-register-app.md)и выберите поддерживаемые учетные записи для вашего приложения. Сценарий SPA может поддерживать аутентификацию с помощью учетных записей в вашей организации или любой организации и личных учетных записей Майкрософт.

Далее, узнайте о конкретных аспектах регистрации заявлений, которые применяются к одностраничным приложениям.

## <a name="register-a-redirect-uri"></a>Регистрация перенаправления URI

Неявный поток отправляет токены в перенаправлении на одностраничное приложение, работая на веб-браузере. Поэтому важно зарегистрировать перенаправление URI, где ваше приложение может получить токены. Убедитесь, что перенаправление URI точно соответствует URI для вашего приложения.

На [портале Azure](https://go.microsoft.com/fwlink/?linkid=2083908)перейдите на зарегистрированное заявление. На странице **аутентификации** приложения выберите **веб-платформу.** Введите значение перенаправления URI для вашего приложения в поле **Redirect URI.**

## <a name="enable-the-implicit-flow"></a>Включить неявный поток

На той же странице **аутентификации,** в **расширенных настройках,** вы также должны включить **неявное предоставление.** Если ваше приложение только подписывается в пользователях и получает идентификационные токены, достаточно выбрать флажок **маркеров ID.**

Если приложению также необходимо получить токены доступа для вызова AIS, также выберите флажок **токенов Access.** Для получения дополнительной информации смотрите [токены ID](./id-tokens.md) и [токены Access.](./access-tokens.md)

## <a name="api-permissions"></a>Разрешения API

Одностраничные приложения могут вызывать AA от имени пользователя, вписавого в нее. Они должны запрашивать делегированные разрешения. Для получения подробной информации [см.](quickstart-configure-app-access-web-apis.md#add-permissions-to-access-web-apis)
