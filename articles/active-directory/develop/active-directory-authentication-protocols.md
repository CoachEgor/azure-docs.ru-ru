# <a name="microsoft-identity-platform-authentication-protocols"></a>Протоколы аутентификации платформы идентификации Майкрософт

Платформа идентификации Майкрософт поддерживает несколько наиболее широко используемых протоколов проверки подлинности и авторизации. Темы в этом разделе описывают поддерживаемые протоколы и их реализацию в платформе идентификации Майкрософт. Эти разделы содержат обзор поддерживаемых типов утверждений, основные сведения об использовании метаданных федерации, подробную документацию по протоколам OAuth 2.0. и SAML 2.0, а также советы по устранению неполадок.
