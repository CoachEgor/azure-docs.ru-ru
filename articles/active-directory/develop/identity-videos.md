# <a name="microsoft-identity-platform-videos"></a>Видео по платформе удостоверений Майкрософт

Изучите основы современной проверки подлинности, платформы удостоверений Майкрософт и библиотек проверки подлинности Майкрософт (MSAL).

## <a name="microsoft-identity-platform-for-developers"></a>Платформа удостоверений Майкрософт для разработчиков

В серии видеороликов *Удостоверения для разработчиков* Маттийс Хоекстра (Matthijs Hoekstra) и Кайл Марш (Kyle Marsh) знакомят зрителей с платформой удостоверений Майкрософт. Узнайте о ключевых компонентах и возможностях платформы, а также о том, как использовать библиотеки проверки подлинности, чтобы приступить к добавлению современной и безопасной проверки подлинности в приложения.

Благодаря содержимому, которое было рассмотрено во многих учебных семинарах, проведенных специалистами, эти видеоматериалы являются отличной отправной точкой для любого, кто решил разобраться с удостоверениями в Azure.
