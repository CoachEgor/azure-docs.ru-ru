# <a name="azure-ad-connect-cloud-provisioning-new-configuration"></a>Azure AD Connect облачная подготовка новой конфигурации

После установки агента необходимо выполнить вход в портал Azure и настроить подготовку.  Чтобы включить агент, выполните следующие действия.

## <a name="configure-provisioning"></a>Настройка подготовки
Чтобы настроить подготовку, выполните следующие действия.

1.  На портале Azure AD щелкните **Azure Active Directory**
2.  Щелкните **Azure AD Connect**
3.  Выберите **Управление подготовка (Предварительная версия)** 
![](media/how-to-configure/manage1.png)

4.  Щелкните **Новая конфигурация**.
5.  На экране конфигурации локальный домен предварительно заполнен.
6. Введите **уведомление по электронной почте**. Это сообщение электронной почты будет уведомлено о неработоспособности подготовки.  
8. Переместите селектор, чтобы **включить** его, и нажмите кнопку **сохранить**.
![](media/tutorial-single-forest/configure2.png)

## <a name="scoping-provisioning-to-specific-users-and-groups"></a>Определение области подготовки для конкретных пользователей и групп
Если вы хотите ограничить агентом синхронизацию только конкретных пользователей и групп, это можно сделать. Область можно использовать в локальных группах AD или подразделениях. Нельзя настроить группы и подразделения в конфигурации. 

1.  На портале Azure AD щелкните **Azure Active Directory**
2.  Щелкните **Azure AD Connect**
3.  Выберите **Управление подготовкой (Предварительная версия)**
4.  В разделе **Конфигурация** щелкните конфигурацию.  
![](media/how-to-configure/scope1.png)

5.  В разделе **Настройка**выберите **все пользователи** , чтобы изменить область действия правила конфигурации.
![](media/how-to-configure/scope2.png)

6. Справа можно изменить область, чтобы она включала только группы безопасности, введя различающееся имя группы и нажав кнопку **Добавить**.
![](media/how-to-configure/scope3.png)

7. Или измените его, чтобы включить только определенные подразделения. Нажмите кнопку **Готово** и **сохранить**.
![](media/how-to-configure/scope4.png)


## <a name="restart-provisioning"></a>Перезапустить подготовку 
Если вы не хотите ждать следующего запланированного выполнения, можно запустить запуск подготовки с помощью кнопки перезапустить подготовку. 
1.  На портале Azure AD щелкните **Azure Active Directory**
2.  Щелкните **Azure AD Connect**
3.  Выберите **Управление подготовкой (Предварительная версия)**
4.  В разделе **Конфигурация** щелкните конфигурацию.  
![](media/how-to-configure/scope1.png)

5.  В верхней части щелкните **перезапустить подготовку**.

## <a name="removing-a-configuration"></a>Удаление конфигурации
Если вы хотите удалить конфигурацию, это можно сделать, выполнив следующие действия.

1.  На портале Azure AD щелкните **Azure Active Directory**
2.  Щелкните **Azure AD Connect**
3.  Выберите **Управление подготовкой (Предварительная версия)**
4.  В разделе **Конфигурация** щелкните конфигурацию.  
![](media/how-to-configure/scope1.png)

5.  В верхней части щелкните **Удалить**.
![](media/how-to-configure/remove1.png)

>[!IMPORTANT]
>Перед удалением конфигурации нет подтверждения, поэтому перед нажатием кнопки **Удалить**убедитесь, что это действие необходимо выполнить.
