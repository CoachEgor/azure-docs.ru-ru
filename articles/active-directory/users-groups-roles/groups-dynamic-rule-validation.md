# <a name="validate-a-dynamic-group-membership-rule-preview-in-azure-active-directory"></a>Проверка динамического правила членства в группе (предварительный просмотр) в Active Directory Azure

Активный каталог Azure (Azure AD) теперь предоставляет средства для проверки динамических правил группы (в общедоступном предварительном просмотре). На вкладке **правил проверки** можно проверить динамическое правило в отношении членов группы выборки, чтобы подтвердить, что правило работает, как ожидалось. При создании или обновлении динамических правил группы администраторы хотят знать, будет ли пользователь или устройство членом группы. Это помогает оценить, соответствует ли пользователь или устройство критериям правила и помогает в устранении неполадок, когда членство не ожидается.

## <a name="step-by-step-walk-through"></a>Шаг за шагом проходной

Чтобы начать работу, перейдите в **группы активных каталогов** > **Groups**Azure. Выберите существующую динамическую группу или создайте новую динамическую группу и нажмите на правила членства Dynamic. Затем можно увидеть вкладку **«Правила проверки».**

![Найдите вкладку правил проверки и начните с существующего правила](./media/groups-dynamic-rule-validation/validate-tab.png)

На вкладке **правил проверки** можно выбрать пользователей для проверки их членства. 20 пользователей или устройств могут быть выбраны одновременно.

![Добавление пользователей для проверки существующего правила против](./media/groups-dynamic-rule-validation/validate-tab-add-users.png)

После выбора пользователей или устройств из сборщика, и **выберите,** проверка автоматически начнется и результаты проверки будут отображаться.

![Просмотр результатов проверки правил](./media/groups-dynamic-rule-validation/validate-tab-results.png)

Результаты говорят о том, является ли пользователь членом группы или нет. Если правило не действительно или есть сетевая проблема, результат будет отображаться как **Неизвестный**. В случае **неизвестного**, подробное сообщение об ошибке будет описывать проблему и действия, необходимые.

![Просмотр сведений о результатах проверки правил](./media/groups-dynamic-rule-validation/validate-tab-view-details.png)

Вы можете изменить правило и проверка членства будет запущена. Чтобы узнать, почему пользователь не является членом группы, нажмите на кнопку "Просмотр деталей" и детали проверки будут отображать результат ы каждого выражения, составляющего правило. Нажмите **OK,** чтобы выйти.
