# <a name="search-groups-and-members-preview-in-azure-active-directory"></a>Группы поиска и участники (предварительный просмотр) в Active Directory Azure

В этой статье рассказывается о том, как искать членов и владельцев группы и как использовать фильтры поиска в рамках предварительного просмотра групп на портале Azure Active Directory (Azure AD). Есть много улучшений в группах опытом, чтобы помочь вам управлять своими группами, в том числе членов и владельцев, быстро и легко. Для получения дополнительной информации о предварительных просмотрах [см.](https://azure.microsoft.com/support/legal/preview-supplemental-terms/)

Изменения в этом предварительном просмотре включают в себя:

- Новые возможности поиска групп, такие как поиск подстроки в группах
- Новые варианты фильтрации и сортировки в списках членов и владельцев
- Новые возможности поиска для списков членов и владельцев
- Более точные групповые подсчеты для больших групп

## <a name="enabling-and-managing-the-preview"></a>Включение и управление предварительным просмотром

Мы упростили присоединение к предварительному просмотру:

  1. Вопийте на [портале Azure AD](https://portal.azure.com)и выберите **группы.**
  2. Из групп - Все группы страницы, выберите баннер в верхней части страницы, чтобы присоединиться к предварительному просмотру.

Вы также можете проверить последние функции и улучшения, выбрав **Информация Предварительного просмотра** на странице **Всех групп.** После присоединения к предварительному просмотру можно увидеть тег предварительного просмотра на всех страницах групп, которые имеют улучшения и являются частью предварительного просмотра. Не каждая страница группы была обновлена в рамках этого предварительного просмотра.

Если у вас возникли какие-либо проблемы, вы можете переключить обратно устаревший опыт, выбрав баннер в верхней части **страницы Все группы.** Мы ценим ваши отзывы, чтобы мы могли улучшить наш опыт.

## <a name="group-search-and-sorting"></a>Групповой поиск и сортировка

Поиск списка групп был расширен, так что при вводе строки `startswith` поиска, поиск автоматически выполняет поиск и подстроку в списке имен групп. Поиск подстроки выполняется только на целых словах и не включает специальные символы. Поиск подстроки является чувствительным к случаям.

![новые поиски подстроки на странице All Groups](./media/groups-members-owners-search/groups-search-preview.png)

Например, поиск "политики" теперь будет возвращать как "политика МДМ - Запад" и "Политическая группа". Группа под названием "New_policy" не будет возвращена.

- Вы также можете выполнять такой же поиск в списках участников группы.
- Теперь можно сортировать список групп по имени, используя стрелки справа от столбца имен, направляющегося для сортировки списка в порядке восходящего или убывающего.

## <a name="group-member-search-and-filtering"></a>Поиск и фильтрация участников группы

### <a name="search-group-member-and-owner-lists"></a>Список участников группы поиска и владельцев

Теперь вы можете искать членов определенной группы по имени, и выполнять тот же поиск в списке владельцев группы, а также. В новом опыте, если вы введете строку в поле поиска, автоматически выполняется поиск. Например, поиск "Скотт" вернет Скотта Уилкинсона.

![новые поиски подстроки в списках членов и владельцев группы](./media/groups-members-owners-search/members-list.png)

### <a name="filter-member-and-owners-list"></a>Список членов и владельцев фильтров

В дополнение к поиску, теперь вы можете фильтровать списки членов и владельцев по типу пользователя. Это информация, сохноченная в столбце типа пользователя списка. Таким образом, вы можете отфильтровать список членов и гостей, чтобы определить, есть ли гости в группе.

### <a name="view-and-manage-membership"></a>Просмотр и управление членством

В дополнение к просмотру прямых членов определенной группы, теперь вы можете просматривать список всех членов группы на странице Членов. Список участников включает в себя всех уникальных членов группы, включая любых транзитных членов.

Вы также можете искать и фильтровать список прямых членов и список всех членов по отдельности. Фильтрация списка всех членов не влияет на фильтры, которые применяются в списке прямых членов.

## <a name="improved-group-member-counts"></a>Улучшенные количество членов группы

Мы улучшили страницу **обзора** группы, чтобы обеспечить количество членов группы для групп всех размеров. Вы можете видеть количество участников даже для групп с более чем 1000 членов. Теперь вы можете увидеть общее количество прямых членов для группы и общее количество членов (все уникальные члены группы, включая транзитных членов) на странице **Обзор.**

![Более высокая точность в подсчетах членства в группе](./media/groups-members-owners-search/member-numbers.png)
