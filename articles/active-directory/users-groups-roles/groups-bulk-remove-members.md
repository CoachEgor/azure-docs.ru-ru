# <a name="bulk-remove-group-members-preview-in-azure-active-directory"></a>Массовое удаление членов группы (предварительный просмотр) в Active Directory Azure

Используя портал Azure Active Directory (Azure AD), можно удалить большое количество участников из группы, используя файл, разделенный на запятую (CSV), для массового удаления членов группы.

## <a name="to-bulk-remove-group-members"></a>Для массового удаления членов группы

1. Вопием на [портал Azure](https://portal.azure.com) с учетной записью администратора пользователя в организации. Владельцы групп также могут навалом удалить членов групп, которые они владеют.
1. В Azure AD выберите **группы** > **все группы.**
1. Откройте группу, из которой вы удаляете участников, а затем выберите **участников.**
1. На странице **Участников** выберите **Удалить участников**.
1. На странице **Bulk удалить членов группы (Preview)** выберите **Download,** чтобы получить шаблон файла CSV с требуемыми свойствами членов группы.

   ![Команда Удаления участников находится на странице профиля группы](./media/groups-bulk-remove-members/remove-panel.png)

1. Откройте файл CSV и добавьте строку для каждого участника группы, которого вы хотите удалить из группы (требуемые значения — идентификатор объекта участника или основное имя пользователя). Затем сохраните файл.

   ![Файл CSV содержит имена и идентионные данные для удаления](./media/groups-bulk-remove-members/csv-file.png)

1. На странице **Bulk удалить членов группы (Preview)** под **загрузить файл csv,** просматривать файл. При выборе файла запускается проверка CSV-файла.
1. Когда содержимое файла проверяется, страница импорта навалом отображает **файл, загруженный успешно.** Если будут обнаружены ошибки, их нужно исправить для отправки задания.
1. Когда файл проходит проверку, выберите **отправку,** чтобы начать операцию навалом Azure, которая удаляет членов группы из группы.
1. Когда операция удаления завершится, вы увидите уведомление о том, что основная операция прошла успешно.

## <a name="check-removal-status"></a>Проверка состояния удаления

Вы можете увидеть состояние всех ожидающих натковых запросов на странице **результаты (предварительный просмотр) результатов операции Bulk.**

   ![Страница результатов результатов массовых операций показывает состояние запроса навалом](./media/groups-bulk-remove-members/bulk-center.png)

Для получения подробной информации о каждом элементе строки в ходе основной операции выберите значения в строках **«Успех» («Успех»** или **«Общие запросы».** **# Failure** При возникновении сбоя здесь будут указаны его причины.

## <a name="bulk-removal-service-limits"></a>Ограничения службы массового удаления

Каждое массовое действие для удаления списка членов группы может работать до одного часа. Это позволяет удалить список не менее 40 000 членов.
