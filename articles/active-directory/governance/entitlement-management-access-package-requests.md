# <a name="view-requests-for-an-access-package-in-azure-ad-entitlement-management"></a>Просмотр запросов на пакет доступа в управлении правами Azure AD

В управлении правами Azure AD можно увидеть, кто запросил пакеты доступа, их политику и статус. В этой статье описывается, как просматривать запросы на пакеты доступа.

## <a name="view-requests"></a>Просмотр запросов

**Роль предпосылки:** Глобальный администратор, администратор пользователей, владелец каталога или менеджер пакетов доступа

1. На портале Azure щелкните **Azure Active Directory** и выберите **Управление удостоверениями**.

1. В левом меню щелкните **пакеты Access,** а затем откройте пакет доступа.

1. Нажмите **Запросы**.

1. Нажмите на конкретный запрос, чтобы узнать дополнительные сведения.

    ![Список запросов на пакет доступа](./media/entitlement-management-access-package-requests/requests-list.png)
