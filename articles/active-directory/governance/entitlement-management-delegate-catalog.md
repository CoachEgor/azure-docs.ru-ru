# <a name="delegate-access-governance-to-catalog-creators-in-azure-ad-entitlement-management"></a>Делегируйте управление доступом создателям каталогов в управлении правами Azure AD

Каталог представляет собой контейнер ресурсов и пакетов доступа. Вы создаете каталог, когда вы хотите группировать связанные ресурсы и пакеты доступа. По умолчанию глобальный администратор или администратор пользователя может [создать каталог](entitlement-management-catalog-create.md)и добавить дополнительных пользователей в качестве владельцев каталогов.

Чтобы делегировать пользователей, которые не являются администраторами, чтобы они могли создавать свои собственные каталоги, можно добавить этих пользователей в роль создателя каталога, определяемого управлением Azure AD. Вы можете добавить отдельных пользователей, или вы можете добавить группу, члены которой затем могут создавать каталоги.  После создания каталога они могут впоследствии добавлять ресурсы, которые у них есть, в свой каталог.

## <a name="as-an-it-administrator-delegate-to-a-catalog-creator"></a>Как ИТ-администратор, делегируйте создателю каталога

Выполните следующие действия, чтобы назначить пользователя роли создателя каталога.

**Роль предпосылки:** Глобальный администратор или администратор пользователя

1. На портале Azure щелкните **Azure Active Directory** и выберите **Управление удостоверениями**.

1. В левом меню, в разделе **Управления правами,** нажмите **Настройки**.

1. Щелкните **Правка**.

    ![Настройки для добавления создателей каталога](./media/entitlement-management-delegate-catalog/settings-delegate.png)

1. В разделе **управления правами делегата** щелкните **создателей каталога Add,** чтобы выбрать пользователей или группы, которым вы хотите делегировать эту роль управления правами.

1. Нажмите **Выберите**.

1. Нажмите **Сохранить**.

## <a name="allow-delegated-roles-to-access-the-azure-portal"></a>Разрешить делегированные роли для доступа к порталу Azure

Чтобы разрешить делегированным ролям, таким как создатели каталогов и менеджеры пакетов доступа, получить доступ к порталу Azure для управления пакетами доступа, необходимо проверить настройки портала администрирования.

**Роль предпосылки:** Глобальный администратор или администратор пользователя

1. На портале Azure щелкните **Active Directory Azure,** а затем нажмите **на пользователей.**

1. В левом меню щелкните **настройки пользователя.**

1. Убедитесь, что **ограничение доступа к порталу администрирования Azure AD** настроено **на нет.**

    ![Настройки пользователя Azure AD - Административный портал](./media/entitlement-management-delegate-catalog/user-settings.png)
