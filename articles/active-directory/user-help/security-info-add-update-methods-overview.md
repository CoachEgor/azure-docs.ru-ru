# <a name="overview-about-changing-your-security-info-preview-methods-and-information"></a>Обзор вариантов изменения методов проверки и информации в сведениях для защиты (предварительная версия)

На страницах **сведений для защиты** можно добавлять, изменять и удалять метод проверки сведений для защиты, используемый по умолчанию. Дополнительные сведения о каждом методе и способах изменения информации см. в соответствующей статье для каждого метода.

## <a name="two-factor-verification-versus-password-reset-authentication"></a>Сравнение двухфакторной проверки и аутентификации для сброса пароля

Методы проверки сведений для защиты используются как при двухфакторной проверке, так и при сбросе пароля. При этом не все методы можно использовать для обоих видов аутентификации.

| Метод | Область использования |
| ------ | -------- |
| Приложение Authenticator | Двухфакторная проверка и аутентификация для сброса пароля. |
| Текстовые сообщения | Двухфакторная проверка и аутентификация для сброса пароля. |
| Телефонные звонки | Двухфакторная проверка и аутентификация для сброса пароля. |
| Учетная запись электронной почты | Только аутентификация для сброса пароля. Для двухфакторной проверки необходимо выбрать другой метод. |
| Контрольные вопросы | Только аутентификация для сброса пароля. Для двухфакторной проверки необходимо выбрать другой метод. |
