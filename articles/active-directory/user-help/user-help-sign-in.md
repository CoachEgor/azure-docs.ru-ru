# <a name="sign-in-using-two-step-verification-or-security-info"></a>Вход с помощью двухфакторной проверки подлинности или сведений для защиты
После настройки двухфакторной проверки подлинности или сведений для защиты вы сможете войти в учетную запись, используя указанный метод аутентификации.

> [!Note]
> Если вы используете процедуру двухфакторной проверки подлинности, необходимо настроить методы проверки подлинности, следуя указаниям в [этой статье](multi-factor-authentication-end-user-first-time.md).
> 
> Если администратор включил функцию использования сведений для защиты, необходимо настроить методы проверки подлинности, выполняя пошаговые инструкции из этих статей:<ul><li>[Настройка использования приложения проверки подлинности в сведениях для защиты (предварительная версия)](security-info-setup-auth-app.md)</li><li>[Настройка использования обмена текстовыми сообщениями в сведениях для защиты (предварительная версия)](security-info-setup-text-msg.md)</li><li>[Настройка использования телефонного звонка в сведениях для защиты (предварительная версия)](security-info-setup-phone-number.md)</li></ul>

## <a name="sign-in-using-an-authenticator-app-notification-on-your-mobile-device"></a>Вход с помощью уведомления от приложения проверки подлинности на мобильном устройстве

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. В уведомлении, которое отправилось на мобильное устройство, выберите **Утвердить**.

## <a name="sign-in-using-an-authenticator-app-code-on-your-mobile-device"></a>Вход с помощью кода в приложении проверки подлинности на мобильном устройстве

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. Откройте приложение проверки подлинности и введите случайным образом сформированный код для своей учетной записи в поле **Введите код**.

## <a name="sign-in-using-your-phone-number"></a>Вход с использованием номера телефона

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. Ответьте на вызов, а затем следуйте инструкциям.

## <a name="sign-in-using-a-text-message"></a>Вход с использованием текстового сообщения

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. Откройте сообщение и введите код из текста в поле **Введите код**.

## <a name="sign-in-using-another-verification-method"></a>Выполнение входа с использованием другого способа проверки
Если по некоторым причинам вы не удалось использовать основной метод, можно использовать другой ранее настроить способ проверки.

1. Обычно войдите свою учетную запись и выберите **войдите другим способом** ссылку на **двухфакторную проверку подлинности** страницы.

    ![Изменение метода проверки при входе](media/security-info/two-factor-auth-signin-another-way.png)

    >[!Note]
    >Если вы не видите **войдите другим способом** ссылку, это означает, что вы еще не настроили другие методы проверки, и вы получите обратитесь к администратору для получения справки, вход в учетную запись. После вашего администратора поможет вам выполнить вход, убедитесь, что добавление дополнительных способов проверки подлинности. Дополнительные сведения о добавлении методов проверки см. в статье [Управление параметрами двухфакторной проверки подлинности](multi-factor-authentication-end-user-manage-settings.md).
    > 
    >Если вы видите **войдите другим способом** связать, но по-прежнему не видите других методов проверки, вам придется обратитесь к администратору за помощью, вход учетную запись.

2. Выберите альтернативный метод проверки подлинности и продолжайте процесс двухфакторной проверки подлинности.

3. При необходимости можно обновить свои методы проверки, как только вы вернетесь в свою учетную запись. Дополнительные сведения о добавлении или изменении методов см. в статье [Управление параметрами двухфакторной проверки подлинности](multi-factor-authentication-end-user-manage-settings.md).
