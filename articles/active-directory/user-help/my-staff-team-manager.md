# <a name="delegate-user-management-with-my-staff-preview"></a>Делегирование управления пользователями с моим персоналом (Предварительная версия)

Ваша организация может использовать **Мои сотрудники** для делегирования задач управления пользователями в области полномочий, такие как менеджер магазина или руководитель группы, чтобы помочь сотрудникам получить доступ к нужным приложениям. Если член команды не может получить доступ к приложению из-за того, что он забыл пароль, производительность теряется. Это также повышает затраты на поддержку и приводит к узким местом в административных процессах.  Благодаря моему персоналу член команды, который не может получить доступ к своей учетной записи, может получить доступ всего лишь через несколько щелчков мыши, при этом Справка администратора не требуется.

## <a name="manage-your-staff-in-my-staff"></a>Управляйте своими сотрудниками в моем штате

Управлять членами команды в моем штате очень просто. Чтобы начать, [перейдите в раздел "Мои сотрудники"](https://aka.ms/mystaff), выберите команду или расположение, а затем выберите пользователя. Расположения и члены группы в расположении определяются ИТ-администратором и не могут быть изменены.

Если вы управляете более чем одним расположением, при переходе к моему персоналу необходимо выбрать расположение, чтобы увидеть участника команды, назначенного этому расположению.

Если у вас еще нет достаточных разрешений для доступа к моим сотрудникам, вы увидите следующее сообщение: "вероятно, у вас нет прав на Просмотр моих сотрудников в настоящее время. Для получения дополнительных сведений обратитесь к администратору. "

### <a name="find-a-staff-member-in-my-staff"></a>Поиск сотрудника в моем штате

Прежде чем приступить к управлению, необходимо открыть профиль сотрудника.

1. [Откройте Мой персонал](https://aka.ms/mystaff) и при необходимости выберите расположение.

    ![Выбор расположения для члена команды в моем штате](media/my-staff-team-manager/allaus.png)

1. Откройте профиль участника команды.

    ![Выбор одного из пользователей в расположении в моем штате](media/my-staff-team-manager/aupage.png)

## <a name="reset-a-user-password"></a>Сброс пароля пользователя

Если ваша организация предоставила вам разрешение, вы можете сбросить пароли сотрудников.

1. [Откройте "Мои сотрудники](https://aka.ms/mystaff)".
1. Откройте профиль сотрудника.
1. Выберите **Сбросить пароль**.

    ![Сброс пароля пользователя в моих персонале](media/my-staff-team-manager/resetpassword1.png)

1. Создайте или введите новый пароль. Может отображаться автоматически созданный временный пароль, или вам может быть предложено ввести временный пароль для пользователя.

    ![Копировать временный пароль пользователя после сброса в моем штате](media/my-staff-team-manager/resetpassword2.png)

После сброса пароля пользователя присвойте ему временный пароль. Когда пользователь входит в систему с временным паролем, он должен изменить их.

## <a name="manage-a-users-phone-number"></a>Управление номером телефона пользователя

Если ваша организация предоставила вам разрешение, вы можете управлять телефонными номерами сотрудников.

### <a name="add-a-phone-number"></a>Добавление номера телефона

1. [Откройте "Мои сотрудники](https://aka.ms/mystaff)".
1. Откройте профиль сотрудника.
1. Выберите **Добавить номер телефона**.

    ![Добавить номер телефона пользователя в мой штат](media/my-staff-team-manager/addphone1.png)

1. Добавьте номер телефона и нажмите кнопку **сохранить**.

    ![Сохранить добавленный номер телефона пользователя в моем штате](media/my-staff-team-manager/addphone2.png)

После регистрации номера телефона для пользователя он может использовать его для входа с помощью SMS, выполнения двухфакторной проверки подлинности или сброса пароля самостоятельно, в зависимости от параметров Организации.

![Новый номер телефона, зарегистрированный моим сотрудникам](media/my-staff-team-manager/addphone3.png)

### <a name="edit-a-phone-number"></a>Изменение номера телефона

1. [Откройте "Мои сотрудники](https://aka.ms/mystaff)".
1. Откройте профиль сотрудника.
1. Выберите **изменить номер телефона**.

    ![Выберите Изменить в профиле пользователя в моем персонале.](media/my-staff-team-manager/editphone2.png)

1. Введите новый номер телефона и нажмите кнопку **сохранить**.

    ![Изменение телефонного номера сотрудника в моем штате](media/my-staff-team-manager/editphone1.png)

### <a name="enable-phone-number-sign-in-for-a-user"></a>Включить вход с помощью номера телефона для пользователя

Если для входа используется номер телефона в качестве имени пользователя (вход в SMS) включен в вашей организации, можно добавить эту проверку подлинности в существующий номер телефона пользователя.

1. [Откройте "Мои сотрудники](https://aka.ms/mystaff)".
1. Откройте профиль сотрудника.
1. Если в нижней части экрана появляется сообщение о том, что для вас доступна учетная запись с номером телефона в качестве имени пользователя, выберите **включить** , чтобы начать процесс. Это сообщение появляется, если пользователь включил вход с помощью номера телефона.

    ![Ознакомьтесь с сообщением, когда вход с помощью телефона поддерживается в расположении в моем штате](media/my-staff-team-manager/enableforms1.png)

1. По завершении щелкните **ОК**.

    ![Удалить номер телефона сотрудника в моем штате](media/my-staff-team-manager/enableforms2.png)

### <a name="remove-a-phone-number"></a>Удаление номера телефона

1. [Откройте "Мои сотрудники](https://aka.ms/mystaff)".
1. Откройте профиль сотрудника.
1. Выберите **удалить номер телефона**.
1. Когда все будет готово, выберите **Удалить** .

    ![Удалить номер телефона сотрудника в моем штате](media/my-staff-team-manager/deletephone1.png)
