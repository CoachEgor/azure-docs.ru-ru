# <a name="change-your-two-factor-verification-method-and-settings"></a>Измените двухфакторный метод проверки и настройки

После настройки методов проверки безопасности для вашей учетной записи или школы вы можете обновить любую из связанных с ней деталей, в том числе:

- Метод проверки безопасности по умолчанию

- Детали метода проверки безопасности, например номер телефона

- Настройка или удаляемое устройство из приложения authenticator

## <a name="using-the-additional-security-verification-page"></a>Использование страницы дополнительной проверки безопасности

Если ваша организация предоставила вам конкретные шаги о том, как включить и управлять двухфакторной проверкой, вы должны сначала следовать этим инструкциям. В противном случае вы можете перейти к настройкам метода проверки безопасности со страницы [Дополнительной проверки безопасности.](https://docs.microsoft.com/azure/active-directory/user-help/multi-factor-authentication-end-user-first-time)

>[!Note]
>Если то, что вы видите на экране, не соответствует тому, что описано в этой статье, это означает, что либо администратор включил информацию о **безопасности (предварительный просмотр),** либо что ваша организация предоставила свой собственный пользовательский портал. Для получения дополнительной информации о новом опыте информации о безопасности смотрите [обзор информации о безопасности (предварительный просмотр).](user-help-security-info-overview.md) Для получения дополнительной информации о пользовательском портале организации необходимо связаться со справочной службой организации.

### <a name="to-get-to-the-additional-security-verification-page"></a>Чтобы попасть на страницу дополнительной проверки безопасности

Вы можете перейти по этой ссылке на [страницу дополнительной проверки безопасности.](https://account.activedirectory.windowsazure.com/proofup.aspx?proofup=1)

![Дополнительная страница проверки безопасности с информацией о доступном методе проверки безопасности](./media/multi-factor-authentication-end-user-manage-settings/mfa-security-verification-page.png)

Вы также можете перейти на страницу **дополнительной проверки безопасности,** выяснив следующие действия:

1. Войти [https://myapps.microsoft.com](https://myapps.microsoft.com)на ввесенку .

1. В правом верхнем углу выберите имя учетной записи и **профиль**.

1. Выберите **Дополнительную проверку безопасности.**  

    ![Ссылка на приложения для дополнительной страницы проверки безопасности](./media/multi-factor-authentication-end-user-manage-settings/mfa-myapps-link.png)

>[!Note]
>Для получения информации об использовании **раздела паролей приложений** на странице **дополнительной проверки безопасности** см. [Manage app passwords for two-factor verification](multi-factor-authentication-end-user-app-passwords.md) Пароли приложений следует использовать только для приложений, которые не поддерживают двухфакторную проверку.

## <a name="change-your-default-security-verification-method"></a>Измените метод проверки безопасности по умолчанию

После того, как вы вопиете в своей рабочей или школьной учетной записи с вашим именем пользователя и паролем, вам автоматически представят выбранный метод проверки безопасности. В зависимости от требований организации это может быть код уведомления или проверки через приложение для проверки подлинности, текстовое сообщение или телефонный звонок.

Если вы решили изменить используемый метод проверки безопасности по умолчанию, вы можете сделать это здесь.

### <a name="to-change-your-default-security-verification-method"></a>Изменить метод проверки безопасности по умолчанию

1. На странице **дополнительной проверки безопасности** выберите метод для использования из списка **предпочтительных вариантов.** Вы увидите все варианты, но вы можете выбрать только те, которые доступны для вас вашей организацией.

    - **Сообщите мне через приложение**: Вы будете уведомлены через приложение аутентификатора, что у вас есть запрос проверки ожидания.

    - **Позвоните на мой телефон аутентификации:** Вы получите телефонный звонок на мобильном устройстве, с просьбой проверить вашу информацию.

    - **Текстовый код на мой телефон проверки подлинности:** Вы получите код проверки как часть текстового сообщения на мобильном устройстве. Вы должны ввести этот код в запрос проверки для вашей работы или школьной учетной записи.

    - **Позвоните на мой офисный телефон**: Вы получите телефонный звонок на свой офисный телефон, с просьбой проверить вашу информацию.

    - **Используйте код проверки из приложения:** Вы будете использовать приложение для проверки подлинности, чтобы получить код проверки, который вы введете в запрос из вашей учетной записи работы или школы.

2. Нажмите кнопку **Сохранить**.

## <a name="add-or-change-your-phone-number"></a>Добавление или изменение номера телефона

Вы можете добавить новые номера телефонов или обновить существующие номера со страницы **дополнительной проверки безопасности.**

>[!Important]
>Мы настоятельно рекомендуем вам добавить вторичный номер телефона, чтобы предотвратить блокировку из вашей учетной записи, если ваш основной телефон потерян или украден, или если вы получаете новый телефон и больше не имеют свой оригинальный, основной номер телефона.

### <a name="to-change-your-phone-numbers"></a>Изменить номера телефонов

1. С раздела **«Как вы хотели бы ответить?»** на странице **дополнительной проверки безопасности,** обновите информацию о номере телефона для **вашего телефона аутентификации** (основного мобильного устройства) и **телефона Office.**

1. Выберите поле рядом с опцией **альтернативной проверки подлинности телефона,** а затем введите вторичный номер телефона, где вы можете получать текстовые сообщения или телефонные звонки, если вы не можете получить доступ к основному устройству.

1. Нажмите кнопку **Сохранить**.

## <a name="add-a-new-account-to-the-microsoft-authenticator-app"></a>Добавление новой учетной записи в приложение microsoft authenticator

Вы можете настроить свою рабочую или школьную учетную запись в приложении Microsoft Authenticator для [Android](https://play.google.com/store/apps/details?id=com.azure.authenticator) или [iOS.](https://apps.apple.com/app/microsoft-authenticator/id983156458)

Если вы уже создали свою учетную запись в приложении Microsoft Authenticator, вам не нужно делать это снова.

1. Из раздела **«Как вы хотели бы ответить?»** раздел **аутов дополнительной проверки безопасности,** выберите **Настройка приложения Authenticator.**

    ![Настройка рабочей или школьной учетной записи в приложении Microsoft Authenticator](./media/multi-factor-authentication-end-user-manage-settings/mfa-security-verification-page-auth-app.png)

1. Следуйте инструкциям на экране, в том числе с помощью мобильного устройства для сканирования кода, а затем выберите **Следующий.**

    Вам будет предложено утвердить уведомление через приложение Microsoft Authenticator для проверки вашей информации.

1. Нажмите кнопку **Сохранить**.

## <a name="delete-your-account-or-device-from-the-microsoft-authenticator-app"></a>Удалить свою учетную запись или устройство из приложения Microsoft Authenticator

Вы можете удалить свою учетную запись из приложения Microsoft Authenticator, а также удалить устройство из рабочей или школьной учетной записи. Обычно вы удаляете устройство, чтобы навсегда удалить потерянное, украденное или старое устройство из учетной записи, и вы удаляете свою учетную запись, чтобы попытаться исправить некоторые проблемы с подключением или устранить изменение учетной записи, например, новое имя пользователя.

### <a name="to-delete-your-device-from-your-work-or-school-account"></a>Удаление устройства из рабочей или школьной учетной записи

1. Из раздела **«Как бы вы хотели ответить?»** **раздела Дополнительной страницы проверки безопасности** выберите кнопку **«Настройка аутентификатора».**

1. Нажмите кнопку **Сохранить**.

### <a name="to-delete-your-account-from-the-microsoft-authenticator-app"></a>Удалить учетную запись из приложения Microsoft Authenticator

Из приложения Microsoft Authenticator выберите кнопку **«Удалить»** рядом с устройством, которое вы хотите удалить.

## <a name="turn-on-two-factor-verification-prompts-on-a-trusted-device"></a>Включите двухфакторные запросы проверки на надежном устройстве

В зависимости от настроек организации, вы можете увидеть флажок, который **говорит, что не просите снова X дней,** когда вы выполняете двухфакторную проверку в вашем браузере. Если вы выбрали эту опцию, чтобы остановить двухфакторные запросы проверки, а затем вы потеряете устройство или ваше устройство потенциально скомпрометировано, вы должны включить двухфакторные запросы проверки обратно, чтобы помочь защитить вашу учетную запись. Вы должны включить запросы для всех ваших устройств в то же время. К сожалению, вы не можете включить подсказки обратно только для конкретного устройства.

### <a name="to-turn-two-factor-verification-prompts-back-on-for-your-devices"></a>Включить двухфакторные запросы проверки для ваших устройств

На [странице **дополнительной проверки безопасности** ](#to-get-to-the-additional-security-verification-page)выберите **восстановление многофакторной аутентификации на ранее доверенных устройствах.** При следующем входе на любое устройство вам будет предложено выполнить двухфакторную проверку.
