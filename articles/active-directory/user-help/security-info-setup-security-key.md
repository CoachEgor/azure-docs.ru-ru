# <a name="set-up-a-security-key-as-your-verification-method"></a>Настройка ключа безопасности в качестве метода проверки

Вы можете использовать ключи безопасности в качестве безпадоного метода вхвнутри организации. Ключ безопасности — это физическое устройство, которое используется с уникальным PIN-кодом для вхвнутри своей учетной записи или школы. Поскольку ключи безопасности требуют, чтобы у вас было физическое устройство и что-то, что только вы знаете, это считается более сильным методом проверки, чем имя пользователя и пароль.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-security-key.md)]

>[!Note]
> Если вы не видите опцию ключа безопасности, возможно, что ваша организация не позволит использовать эту опцию для проверки. В этом случае вам необходимо выбрать другой метод или обратиться в службу поддержки вашей организации для получения дополнительной помощи.

## <a name="security-verification-versus-password-reset-authentication"></a>Сравнение проверки безопасности и аутентификации для сброса пароля

Методы проверки сведений для защиты используются как при двухфакторной проверке безопасности, так и при сбросе пароля. При этом не все методы можно использовать для обоих видов аутентификации.

| Метод | Используется для |
| ------ | -------- |
| Приложение Authenticator | Двухфакторная проверка и аутентификация для сброса пароля. |
| Текстовые сообщения | Двухфакторная проверка и аутентификация для сброса пароля. |
| Телефонные звонки | Двухфакторная проверка и аутентификация для сброса пароля. |
| Ключ безопасности | Двухфакторная проверка и аутентификация для сброса пароля. |
| Учетная запись электронной почты | Только аутентификация для сброса пароля. Для двухфакторной проверки необходимо выбрать другой метод. |
| Контрольные вопросы | Только аутентификация для сброса пароля. Для двухфакторной проверки необходимо выбрать другой метод. |

## <a name="what-is-a-security-key"></a>Что такое ключ безопасности?

В настоящее время мы поддерживаем несколько проектов и поставщиков ключей безопасности, используя протоколы проверки подлинности [Fast Identity Online (FIDO)](https://fidoalliance.org/fido2/) (FIDO2). Эти ключи позволяют вам войти в свою рабочую или школьную учетную запись для доступа к облачным ресурсам организации, когда они работают на поддерживаемом устройстве и веб-браузере.

Ваш администратор или ваша организация предоставит вам ключ безопасности, если он потребуется для вашей работы или школьной учетной записи. Существуют различные типы ключей безопасности, которые можно использовать, например, ключ USB, который вы подключаете к устройству, или ключ NFC, который вы нажмете на считыватель NFC. Вы можете узнать больше о ключе безопасности, в том числе о том, какой тип он имеет, из документации производителя.

> [!Note]
> Если вы не можете использовать ключ безопасности FIDO2, есть и другие методы проверки без паролей, которые можно использовать, такие как приложение Microsoft Authenticator или Windows Hello. Для получения дополнительной информации о приложении Microsoft Authenticator, смотрите [Что такое приложение Microsoft Authenticator?](user-help-auth-app-overview.md). Для получения дополнительной информации о Windows Hello смотрите [обзор Windows Hello.](https://www.microsoft.com/windows/windows-hello)

## <a name="before-you-begin"></a>Перед началом

Прежде чем вы сможете зарегистрировать ключ безопасности, должно быть правдой:

- Администратор включил эту функцию для использования в организации.

- Вы находитесь на устройстве под управлением Windows 10 Май 2019 Обновление и с помощью поддерживаемого браузера.

- У вас есть ключ физической безопасности, одобренный администратором или организацией. Ключ безопасности должен соответствовать требованиям FIDO2 и Microsoft. Если у вас есть вопросы о ключе безопасности и совместимли ли он, обратитесь в службу поддержки организации.

## <a name="register-your-security-key"></a>Зарегистрируйте ключ безопасности

Вы должны создать свой ключ безопасности и дать ему уникальный PIN-код, прежде чем вы сможете войти в свой рабочий или школьный счет с помощью ключа. У вас может быть зарегистрировано до 10 ключей. 

1. Перейти на страницу https://myprofile.microsoft.com **"Мой профиль"** и войти, если вы еще этого не сделали.

2. Выберите **информацию о безопасности,** выберите **метод добавления,** а затем выберите **ключ безопасности** из списка **«Добавить выпадение» метода.**

    ![Добавить поле метода с выбранным ключом безопасности](media/security-info/security-info-security-key-add-method.png)

3. Выберите **Добавить,** а затем выберите тип ключа безопасности у вас есть, либо **USB-устройство** или **NFC устройства.**

    ![Выберите, есть ли у вас ключ безопасности USB или NFC](media/security-info/security-info-security-key-choose-type.png)

    > [!Note]
    > Если вы не уверены, какой тип ключа безопасности у вас есть, обратитесь к документации производителя. Если вы не уверены в производителе, обратитесь за помощью в службу поддержки вашей организации.

4. Иметь ключ безопасности физически доступным, а затем в поле **ключа безопасности,** выберите **Следующий.**

    ![Окно регистрации ключа безопасности](media/security-info/security-info-security-key-start-setup.png)

    Новая коробка, как представляется, поможет вам через настройку нового метода вхастывки.

5. При **настройке нового окна метода ввоза** выберите **Следующую,** а затем:

    - Если ключом безопасности является USB-устройство, вставьте ключ безопасности в порт USB вашего устройства.

    - Если ключом безопасности является устройство NFC, нажмите ключ безопасности к читателю.

6. Введите уникальный PIN-код ключа безопасности в поле **безопасности Windows,** а затем выберите **OK.**

    Вы вернетесь к настройке новой коробки **метода ввоза.**

7. Нажмите кнопку **Далее**.

8. Вернуться на страницу **сведения о безопасности,** введите имя, которое вы узнаете позже для нового ключа безопасности, а затем выберите **Следующий.**

    ![Страница информации о безопасности, именуя ключ безопасности](media/security-info/security-info-security-key-name.png)

    Ваш ключ безопасности зарегистрирован и готов к тому, чтобы вы могли войти в свой рабочий или школьный аккаунт.

9. Выберите **Готово,** чтобы закрыть поле **ключа безопасности.**

    Страница **информации о безопасности** обновляется с информацией о ключевых элементах безопасности.

    ![Страница информации о безопасности со всеми зарегистрированными методами, показанными](media/security-info/security-info-security-key-configured.png)

## <a name="delete-a-security-key-from-your-security-info"></a>Удалить ключ безопасности из вашей информации о безопасности

Если вы неуместны или больше не хотите использовать ключ безопасности, вы можете удалить ключ из вашей информации о безопасности. В то время как это останавливает использование ключа безопасности в вашей учетной записи работы или школы, ключ безопасности продолжает хранить ваши данные и учетные данные. Чтобы удалить данные и учетные данные из самого ключа безопасности, необходимо следовать инструкциям в разделе [сбросить ключевой элемент безопасности, совместимый с корпорацией Майкрософт,](#reset-your-security-key) в этой статье.

1. Выберите ссылку **Удалить** из ключа безопасности для удаления.

2. Выберите **Ok** из **окна ключа безопасности Delete.**

    Ключ безопасности удален, и вы больше не сможете использовать его для вху-на-айбордину в свою учетную запись.

>[!Important]
>Если вы удалите ключ безопасности по ошибке, вы можете зарегистрировать его снова, используя инструкции в разделе [Ключа безопасности](#register-your-security-key) этой статьи.

## <a name="manage-your-security-key-settings-from-windows-settings"></a>Управление настройками ключей безопасности из настроек Windows

Вы можете управлять настройками ключей безопасности из приложения **«Настройки Windows»,** включая сброс ключа безопасности и создание нового PIN-кода ключа безопасности.

### <a name="reset-your-security-key"></a>Сбросить ключ безопасности

Если вы хотите удалить всю информацию об учетной записи, хранящуюся на ключе физической безопасности, вы должны вернуть ключ обратно в заводские по умолчанию. Сброс ключа безопасности удаляет все из ключа, что позволяет начать все сначала.

>[!IMPORTANT]
>Сброс ключа безопасности удаляет все, от ключа, сбросив его на заводские по умолчанию.
>
> **Все данные и учетные данные будут очищены.**

#### <a name="to-reset-your-security-key"></a>Сбросить ключ безопасности

1. Откройте приложение «Настройки Windows», выберите **учетные записи,** выберите **параметры вхаса»,** выберите **ключ безопасности,** а затем выберите **Управление.**

2. Вставьте ключ безопасности в порт USB или коснитесь считывателя NFC, чтобы проверить вашу личность.

3. Следуйте инструкциям на экране, основанным на вашем конкретном производителе ключей безопасности. Если ваш производитель ключей не указан в инструкциях на экране, обратитесь к сайту производителя для получения дополнительной информации.

4. Выберите **Близко,** чтобы закрыть экран **Управления.**

### <a name="create-a-new-security-key-pin"></a>Создание нового PIN-кода безопасности

Можно создать новый PIN-код ключа безопасности для ключа безопасности.

#### <a name="to-create-a-new-security-key-pin"></a>Создание нового PIN-кода ключа безопасности

1. Откройте приложение «Настройки Windows», выберите **учетные записи,** выберите **параметры вхаса»,** выберите **ключ безопасности,** а затем выберите **Управление.**

2. Вставьте ключ безопасности в порт USB или коснитесь считывателя NFC, чтобы проверить вашу личность.
3. Выберите **Добавить** из области **PIN-кода security Key,** введите и подтвердите свой новый PIN-код ключа безопасности, а затем выберите **OK.**

     Ключ безопасности обновляется с новым PIN-кодом ключа безопасности для использования в вашей работе или школьной учетной записи. Если вы решите изменить PIN-код снова, вы можете выбрать кнопку **Изменения.**
4. Выберите **Близко,** чтобы закрыть экран **Управления.**
