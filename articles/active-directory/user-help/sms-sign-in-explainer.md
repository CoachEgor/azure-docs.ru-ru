# <a name="use-your-phone-number-as-a-user-name-preview"></a>Использование номера телефона в качестве имени пользователя (Предварительная версия)

Регистрация устройства предоставляет вашему телефону доступ к службам Организации и не разрешает вашей организации доступ к телефону. Если вы являетесь администратором, дополнительные сведения см. в подокне [Настройка и включение пользователей для проверки подлинности на основе SMS](../authentication/howto-authentication-sms-signin.md).

Если ваша организация не предоставила доступ к серверу SMS, вы не увидите параметр для него при регистрации телефона в вашей учетной записи.  

## <a name="when-you-have-a-new-phone-number"></a>При наличии нового номера телефона

Если вы получаете новый телефон или номер и регистрируете его в Организации, для которой доступен вход в систему SMS, вы получаете стандартный процесс регистрации на телефоне:

1. Выберите **Добавить метод**.
1. Выберите **Телефон**.
1. Введите номер телефона и выберите **текст я код**.
1. После ввода кода нажмите кнопку **Далее**.
1. Появится запрос с сообщением "SMS проверено. Ваш телефон успешно зарегистрирован. "

> [!Important]
> Из-за известной проблемы в предварительной версии в течение короткого времени при добавлении номера телефона номер для входа в SMS не будет зарегистрирован. Вам потребуется выполнить вход, используя добавленный номер, а затем следовать инструкциям, чтобы зарегистрировать номер для входа в SMS.

### <a name="when-the-phone-number-is-in-use"></a>При использовании номера телефона

Если вы попытаетесь использовать номер телефона, который использует кто-то другой в вашей организации, вы увидите следующее сообщение:

![Сообщение об ошибке, если ваш номер телефона уже используется](media/sms-sign-in-explainer/sms-sign-in-error.png)

Чтобы исправить проблему, обратитесь к администратору.

## <a name="when-you-have-an-existing-number"></a>При наличии существующего числа

Если вы уже используете номер телефона в Организации и используете номер телефона в качестве имени пользователя, приведенные ниже инструкции помогут вам войти в систему.

1. Когда вход в SMS будет доступен, отобразится баннер с вопросом о том, хотите ли вы включить номер телефона для входа в SMS:

    [![](media/sms-sign-in-explainer/sms-sign-in-banner.png "Banner to enable SMS sign-in for a phone number")](media/sms-sign-in-explainer/sms-sign-in-banner.png#lightbox)

1. Кроме того, при выборе курсора на плитке метод телефона появляется кнопка **включить** .

    [![](media/sms-sign-in-explainer/sms-sign-in-phone-method.png "Banner to enable SMS sign-in for a phone number")](media/sms-sign-in-explainer/sms-sign-in-phone-method.png#lightbox)

1. Чтобы включить метод, выберите **включить**. Вам будет предложено подтвердить действие:

    ![Диалоговое окно подтверждения включения входа SMS для номера телефона](media/sms-sign-in-explainer/sms-sign-in-confirmation.png)

1. Нажмите кнопку **Включить**.

## <a name="when-you-remove-your-phone-number"></a>При удалении номера телефона

1. Чтобы удалить номер телефона, нажмите кнопку Удалить на плитке метода входа на телефон SMS.

    [![](media/sms-sign-in-explainer/sms-sign-in-delete-method.png "Banner to delete SMS sign-in for a phone number")](media/sms-sign-in-explainer/sms-sign-in-delete-method.png#lightbox)

2. При появлении запроса на подтверждение действия нажмите кнопку **ОК**.

Нельзя удалить номер телефона, используемый в качестве метода входа по умолчанию. Чтобы удалить число, необходимо изменить метод входа по умолчанию, а затем снова удалить номер телефона.
