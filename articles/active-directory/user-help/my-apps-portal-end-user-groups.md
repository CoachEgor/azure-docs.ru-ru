# <a name="update-your-groups-info-from-the-my-apps-portal"></a>Обновите сведения учетной записи группы с портала Мои приложения
На веб-портале **Мои приложения** можно использовать рабочую или учебную учетную запись, чтобы просматривать и запускать многие облачные приложения вашей организации, обновлять сведения о профиле и учетной записи, просматривать сведения о **группах** и выполнять **проверки доступа** к приложениям и группам. Если у вас нет доступа к порталу **Мои приложения**, обратитесь за разрешением в службу технической поддержки.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

>[!Important]
>Эти материалы предназначены для пользователей. Администраторы могут найти дополнительные сведения о настройке облачных приложений и управлении ими в [документации по управлению приложениями](https://docs.microsoft.com/azure/active-directory/manage-apps).

## <a name="view-your-groups-information"></a>Просмотреть сведения о группах
Если администратор предоставил вам разрешения на просмотр **группы** плитку, вы можете:

- **Как член группы.** Просмотреть подробные сведения или выйти из любой группы.

- **В качестве владельца группы.** Просмотреть подробные сведения, создайте новую группу, добавить или удалить участников или удалите группу.

### <a name="to-view-your-groups-information"></a>Чтобы просмотреть сведения о группах

1.  Войдите в рабочую или учебную учетную запись.

2.  Откройте браузер и перейдите к https://myapps.microsoft.com, или воспользоваться ссылкой, предоставляемую вашей организацией. Например, вам могут направляться на настроенной страницы для вашей организации, такие как https://myapps.microsoft.com/contoso.com.

    **Приложений** появится страница, отображение всех облачных приложений, владельцем которых является вашей организации и доступны для использования.

    ![Страницы приложений на портале Мои приложения](media/my-apps-portal/my-apps-portal-apps-page-groups-tile.png)

3. Выберите **группы** плитку, чтобы просмотреть данные для связанных с группой.

    ![Страница "группы" с обоими принадлежит и группы элементов](media/my-apps-portal/my-apps-portal-groups-page.png)

4. На основании вашего разрешения, можно использовать **группы** страницы:

    - **Проверьте группы, которыми вы владеете.** Просмотрите сведения обо всех группах вы владеете в вашей организации из **группы в моем владении** области. Выбрав имя конкретной группы предоставляет дополнительные сведения о группе, включая тип группы, количество элементов, политику присоединения и список активных участников.

    - **Создайте группу.** Создать новую группу с вами как владелец из **группы в моем владении** области. Дополнительные сведения см. в разделе [создать группу](#create-a-new-group) этой статьи.

    - **Измените существующую группу.** Измените сведения о любой из собственных групп. Дополнительные сведения см. в разделе [изменить существующую группу](#edit-an-existing-group) этой статьи.

    - **Добавление или удаление членов.** Добавлять или удалять членов групп, которыми вы владеете. Дополнительные сведения см. в разделе [Добавление или удаление члена](#add-or-remove-a-member) этой статьи.

    - **Обновите группу Office 365.** Если ваша организация разрешает его, можно обновить группами Office 365. Дополнительные сведения см. в разделе [обновить группу Office 365](#renew-an-office-365-group) этой статьи. 

    - **Удаление группы.** Удалите все группы, которыми вы владеете. Дополнительные сведения см. в разделе [удаление группы](#delete-a-group) этой статьи.

    - **Просмотрите группы, которые вы работаете.** Просмотреть имена из любой группы, к которой вы входите из **групп, я нахожусь в** области. Выбрав имя конкретной группы предоставляет дополнительные сведения о группе, включая тип группы, количество элементов, политику присоединения и список активных участников.

    - **Присоединение к группе.** Присоединиться к существующей группе, для которого вы еще не член, из **групп, я нахожусь в** области. Дополнительные сведения см. в разделе [присоединиться к существующей группе](#join-an-existing-group).

## <a name="create-a-new-group"></a>Создание группы
1. На **группы** выберите **создайте группу** из **группы в моем владении** области.

    **Создать группу** появится поле.

    ![Создание группы](media/my-apps-portal/my-apps-portal-create-group-page.png)

2. Введите необходимые сведения:

    - **Тип группы.**
        
        - **Безопасность.** Используется для управления доступом участников и компьютеров к общим ресурсам группы пользователей. Например, можно создать группу безопасности для определенной политики безопасности. Поступив так, можно сразу предоставить набор разрешений всем участникам, а не добавлять эти разрешения каждому участнику по отдельности.

        - **Office 365.** Обеспечивает возможности совместной работы, предоставляя участникам доступ к общему почтовому ящику, календарю, файлам, сайту SharePoint и многое другое. Этот параметр также позволяет предоставить доступ к группе пользователям за пределами вашей организации.

    - **Имя группы.** Добавьте осмысленное имя группы, которое легко запомнить.

    - **Описание группы (необязательно).** Добавьте дополнительное описание группы.

    - **Групповая политика.** Выбор — Разрешить любому присоединиться к группе или только разрешения владельца группы для добавления членов.

3. Нажмите кнопку **Создать**.

    Будет создана новая группа с вами как владелец и отображается в вашей **группы в моем владении** списка. Поскольку вы являетесь владельцем, эта группа появилась в **групп, я нахожусь в** списка.

## <a name="edit-an-existing-group"></a>Изменить существующую группу
После создания группы, можно изменить сведения о нем, включая обновление существующей сведениями.

### <a name="to-edit-your-details"></a>Чтобы изменить свои данные
1. Выберите группу, для редактирования из **группы** странице, а затем выберите **изменения сведений о** на *&lt;имя_группы&gt;* страницы.

    **Изменения сведений о** появится окно, и вы можете обновить данные, которые были добавлены при создании группы.

2. Внести все изменения, а затем выберите **обновления**.

## <a name="add-or-remove-a-member"></a>Добавить или удалить элемент
Можно добавить или удалить членов для любой собственные группы.

### <a name="to-add-or-remove-a-member"></a>Чтобы добавить или удалить элемент
1. Выберите группу, нужно добавить элементы, а затем выберите **+** на *&lt;имя_группы&gt;* страницы.

    ![Добавление члена группы с + выделенным входа](media/my-apps-portal/my-apps-portal-add-member-link.png)

2. Найдите элемент, который вы хотите добавить, из **добавление членов** поле, а затем выберите **добавить**.

    ![Добавить поле члены, с помощью нового элемента для добавления](media/my-apps-portal/my-apps-portal-add-member-page.png)

    Отправлено приглашение на новый элемент, чтобы приступить к работе, получает доступ к приложениям организации.

3. Если вы добавили членом по ошибке или если член покинул вашей организации, можно удалить элемент, выбрав **удаление члена** рядом с именем члена на *&lt;имя_группы&gt;* страницы.

    ![Удаление члена, с выделенной ссылкой удаления](media/my-apps-portal/my-apps-portal-remove-member-link.png)

## <a name="renew-an-office-365-group"></a>Обновить группу Office 365
Если ваша организация разрешает его, можно обновить группу Office 365, расширение истечения срока.

### <a name="to-renew-a-group"></a>Чтобы обновить группу
1. Выберите группу Office 365, вы хотите обновить, а затем выберите **продлении действия группы**.

    ![Обновить группу Office 365, продлевая срок действия](media/my-apps-portal/my-apps-portal-renew-group-link.png)

2. Нажмите кнопку **ОК** чтобы закрыть сообщение с подтверждением.

    После обновления страницы вы увидите обновленное **последнего обновления** и **группе истечения срока действия** даты.

## <a name="delete-a-group"></a>Удаление группы
Вы можете удалить любой собственных групп в любое время. Тем не менее если вы по ошибке удалите группу необходимо создать его и снова добавьте членов.

### <a name="to-delete-the-group"></a>Чтобы удалить группу
1. Выберите группу, вы хотите удалить без возможности восстановления, а затем выберите **удалить группу** на *&lt;имя_группы&gt;* страницы.

    ![Страница < имя_группы > с выделенной ссылкой группы Delete](media/my-apps-portal/my-apps-portal-delete-group-link.png)

2. Выберите **Да** в сообщении с подтверждением.

    Окончательно удалить группу. 
    
## <a name="join-an-existing-group"></a>Присоединиться к существующей группе
Вы можете присоединиться к уже существующей группы из **группы** страницы.

### <a name="to-join-or-leave-a-group"></a>Чтобы присоединиться к или выход из группы

1. На **группы** выберите **присоединиться к группе** из **групп, я нахожусь в** области.

    **Присоединиться к группам** появится страница.

    ![Присоединяйтесь к странице "группы" с выделенной кнопкой группы соединения](media/my-apps-portal/my-apps-portal-join-group-link.png)

2. На **присоединиться к группам** выберите имя группы, вы хотите присоединиться, сведений о связанных группы и затем, если группа будет доступна, выберите **присоединиться к группе**.

    Если группе требуется владельцу одобрять членство в группе, будет предложено ввести коммерческое обоснование для почему нужно присоединиться к группе и выберите **запроса**. Если группы не требует утверждения, вы будете немедленно добавлены как член и группа отображается в вашей **групп, я нахожусь в** списка.

3. Если вы по ошибке присоединены к группе, или если вам больше не нужно входить в его, можно выбрать имя группы из **присоединиться к группам** странице, а затем выберите **выйти из группы**.

    ![Присоединяйтесь к странице "группы" с выделенной кнопкой "оставьте группы"](media/my-apps-portal/my-apps-portal-leave-group-link.png)
