# <a name="register-your-verification-method-info-to-reset-your-own-password"></a>Зарегистрируйте информацию о методе проверки для сбросить свой пароль

Если вы забыли пароль рабочей или учебной учетной записи, никогда не получали его от службы поддержки своей компании или ваша учетная запись была заблокирована, используйте сведения для защиты и мобильное устройство, чтобы сбросить пароль рабочей или учебной учетной записи.

Администратор должен включить эту функцию, чтобы вы могли зарегистрировать свою информацию и сбросить свой собственный пароль. Если вы не видите опцию **Забыли мой пароль,** это означает, что администратор не включил функцию для вашей организации. Если вы полагаете, что это неправильно, обратитесь в службу технической поддержки за помощью.

>[!Important]
>Эта статья предназначена для пользователей, пытающихся использовать регистрацию для сбросить пароля самообслуживания. Это означает, что вы сможете сбросить свой собственный рабочий или школьный пароль (например, alain@contoso.comбез помощи администратора. Если вы администратор ищете информацию о том, как включить сбросить пароля самообслуживания для ваших сотрудников или других пользователей, см. [сбросить пароля самообслуживания Deploy Azure AD и другие статьи.](https://docs.microsoft.com/azure/active-directory/authentication/howto-sspr-deployment)

## <a name="set-up-your-password-reset-verification-method"></a>Настройка метода проверки спаза паролей

1. Откройте веб-браузер на устройстве и перейдите на [страницу информации о безопасности.](https://account.activedirectory.windowsazure.com/PasswordReset/Register.aspx?regref=ssprsetup)

2. В зависимости от того, как администратор настроил организацию, один или несколько из следующих вариантов будут доступны для настройки в качестве метода проверки безопасности. При наличии нескольких опций мы настоятельно рекомендуем использовать несколько вариантов в качестве метода проверки безопасности в случае, если один из ваших методов становится недоступным.

    - **Приложение аутентификации.** В качестве метода проверки безопасности выберите приложение Microsoft Authenticator или другое приложение для проверки подлинности. Для получения дополнительной информации о настройке приложения можно [ознакомиться с приложением Microsoft Authenticator в качестве метода проверки.](security-info-setup-auth-app.md)

    - **Текстовые сообщения.** Выберите отправить себе текстовые сообщения на мобильное устройство. Для получения дополнительной информации о настройке текстовых сообщений, смотрите [Настройка текстовых сообщений в качестве метода проверки](security-info-setup-text-msg.md).

    - **Телефонные звонки.** Выберите телефонный звонок на свой зарегистрированный номер телефона. Для получения дополнительной информации о настройке телефонных звонков можно [настроить номер телефона в качестве метода проверки.](security-info-setup-phone-number.md)

    - **Ключ безопасности.** Выберите ключ безопасности, совместимый с корпорацией Майкрософт. Для получения дополнительной информации смотрите [Направьте ключ безопасности в качестве метода проверки.](security-info-setup-security-key.md)

    - **Адрес электронной почты.** Выберите альтернативный адрес электронной почты, который можно использовать, не требуя вашего забытого или отсутствующих паролей. Это работает только для сбросить пароль, а не как метод проверки безопасности. Для получения дополнительной информации о настройке адреса электронной почты, смотрите [Настроите адрес электронной почты в качестве метода проверки.](security-info-setup-email.md)

    - **Вопросы безопасности.** Выберите настройку и ответ на заранее определенные вопросы безопасности, настроенные администратором. Это работает только для сбросить пароль, а не как метод проверки безопасности. Для получения дополнительной информации о вопросах безопасности, [пересмотрите Вопросы безопасности как метод проверки.](security-info-setup-questions.md)

3. После выбора и настройки методов выберите **finish** для завершения процесса.

    > [!Note]
    > Информация, добавленная для вашего номера телефона или адреса электронной почты, не передается в глобальный каталог организации. Единственные люди, которые могут видеть эту информацию, это вы и ваш администратор. Ответы на ваши секретные вопросы видите только вы.

## <a name="common-problems-and-their-solutions"></a>распространенные проблемы и способы их устранения

 Ниже описаны ошибки, которые встречаются чаще всего, и подсказки по их устранению.

| Сообщение об ошибке |  Возможное решение |
| --- | --- | --- |
| Обратитесь к администратору.<br>Мы выяснили, что корпорация Майкрософт не управляет паролем от вашей учетной записи. Соответственно, мы не можем автоматически сбросить ваш пароль.<br>Если вам нужна дополнительная информация, обратитесь к ИТ-специалистам.| Если вы получили сообщение об ошибке после ввода идентификатора пользователя, это означает, что ваша организация внутренне управляет вашим паролем и не хочет, чтобы вы сбросить пароль из ссылки **Can't access your account.** Чтобы сбросить пароль в этой ситуации, необходимо обратиться за помощью в службу поддержки организации или к администратору. |
| Для вашей учетной записи запрещен сброс пароля.<br>К сожалению, ИТ-специалисты не установили для вашей учетной записи возможность использования этой службы.<br>Если хотите, мы можем обратиться к администратору вашей организации для сброса вашего пароля. | Если вы получили это сообщение об ошибке после ввода идентификатора пользователя, это означает, что либо ваша организация не включила функцию сбросить пароль, либо вы не можете использовать ее. Чтобы сбросить пароль в этой ситуации, необходимо выбрать **ссылку Администратора Контакта.** После нажатия на ссылку в службу поддержки или администратору организации отправляется электронное письмо, сообщающее им, что вы хотите сбросить пароль. |
| Не удалось проверить учетную запись.<br>Если хотите, мы можем обратиться к администратору вашей организации для сброса вашего пароля. | Если вы получили это сообщение об ошибке после ввода идентификатора пользователя, это означает, что ваша организация включила сбросить пароль и что вы можете использовать его, но вы не зарегистрировались для службы. В этой ситуации необходимо связаться со службой поддержки организации или администратором для сбросить пароль. Для получения информации о регистрации для сбросить пароль после того, как вы вернулись на устройство, смотрите процесс выше в этой статье. |
