# <a name="access-and-use-apps-on-the-my-apps-portal"></a>Доступ и использование приложений на портале Мои приложения
На веб-портале **Мои приложения** можно использовать рабочую или учебную учетную запись, чтобы просматривать и запускать многие облачные приложения вашей организации, обновлять сведения о профиле и учетной записи, просматривать сведения о **группах** и выполнять **проверки доступа** к приложениям и группам. Если у вас нет доступа к порталу **Мои приложения**, обратитесь за разрешением в службу технической поддержки.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

>[!Important]
>Эти материалы предназначены для пользователей. Администраторы могут найти дополнительные сведения о настройке облачных приложений и управлении ими в [документации по управлению приложениями](https://docs.microsoft.com/azure/active-directory/manage-apps).

## <a name="supported-browsers"></a>Поддерживаемые браузеры
Можно перейти к **Мои приложения** портала из любого из следующих веб-браузеров:

- Google Chrome

- Mozilla Firefox 26.0 или более поздней версии

- Microsoft Edge

- Internet Explorer версии 11 (ограниченная поддержка).

## <a name="download-and-install-the-my-apps-secure-sign-in-extension"></a>Скачайте и установите Мои расширение защищенного приложения входа в систему
Скачайте и установите расширение My защищенного приложения входа в систему, при появлении запроса. Это расширение помогает запустить любой из вашей организации облачных приложений, которые требуется использовать единый вход в систему. Если ваша организация уже задал вам для использования единого входа, модуль устанавливается автоматически, и этот раздел можно пропустить.

Это расширение помогает: 

- Вход непосредственно в приложения на странице входа в систему.

- Запуск любого приложения, с помощью **быстрого поиска** функции.

- См. в разделе последнего приложения, используемые в **недавно использованные** раздел.

- Использовать URL-адреса внутренней компании при удаленном подключении с [прокси приложения](https://docs.microsoft.com/azure/active-directory/active-directory-application-proxy-get-started).

### <a name="to-download-and-install-the-extension"></a>Чтобы скачать и установить расширение
Скачайте и установите расширение, на основе браузера, которую вы используете:

- **Google Chrome.** Из Store Web Chrome, перейдите к [Мои расширение защищенного приложения вход](https://chrome.google.com/webstore/detail/my-apps-secure-sign-in-ex/ggjhpefgjjfobnfoldnjipclpcfbgbhl) компонентов, а затем выберите **добавьте Chrome**.

- **Mozilla Firefox** из **надстройки Firefox** перейдите к странице [Мои расширение защищенного приложения вход](https://addons.mozilla.org/firefox/addon/access-panel-extension/) компонентов, а затем выберите **добавьте Firefox**.

- **Microsoft Edge** из Microsoft Store, перейдите в раздел [Мои расширение защищенного приложения вход](https://www.microsoft.com/p/my-apps-secure-sign-in-extension/9pc9sckkzk84?rtc=1&activetab=pivot%3Aoverviewtab) компонентов, а затем выберите **получить**.

Значок добавляется справа от вашей **адрес** панели, позволяя входа в систему и настроить его.

#### <a name="to-change-your-my-apps-portal-using-the-extension"></a>Чтобы изменить с помощью расширения портала Мои приложения
Вы можете выбрать, сколько приложений для просмотра в **недавно использованные** раздела и решить, следует ли разрешить внутренние URL-адреса вашей организации, для перенаправления.

1. Выберите новый **Мои расширение защищенного приложения вход** значок ![значок расширения](media/my-apps-portal/my-apps-portal-extension-icon.png) справа от вашей **адрес** панели, а затем выберите **войти систему, для получения работы**.

2. Щелкните правой кнопкой мыши **параметры** значок ![значок "Параметры"](media/my-apps-portal/my-apps-portal-extension-settings-icon.png), а затем выберите **параметры**.

3. В **параметры** поле, укажите число последних приложений, которые вы хотите увидеть на портале и чтобы разрешить организации, внутренние URL-адреса для перенаправления, можно ли использовать их удаленно.

    ![Параметры модуля, показывающий доступные настройки](media/my-apps-portal/my-apps-portal-extension-settings-page.png)

## <a name="access-and-use-the-my-apps-portal-by-device"></a>Доступ и использование на портале "Мои приложения" устройством
Можно получить доступ к и использование портала Мои приложения на компьютере под управлением Intune браузером или из мобильных устройств Android или iOS.

![Страницы приложений на портале Мои приложения](media/my-apps-portal/my-apps-portal-apps-page.png)

### <a name="access-and-use-the-my-apps-portal-on-your-computer"></a>Доступ и использование на портале "Мои приложения" на компьютере
Если у вас есть разрешение на доступ и использовать приложения на основе облака в вашей организации, вы можете получить к ним через **Мои приложения** портала.

1.  Войдите в рабочую или учебную учетную запись.

2.  Откройте браузер и перейдите к https://myapps.microsoft.com, или воспользоваться ссылкой, предоставляемую вашей организацией. Например, вам могут направляться на настроенной страницы для вашей организации, такие как https://myapps.microsoft.com/contoso.com.

    **Приложений** появится страница, отображение всех облачных приложений, владельцем которых является вашей организации и доступны для использования.

3. Из **приложений** выберите приложение, вы хотите начать использовать.
    
    Откроется новая страница для приложения, где можно войти (при необходимости) или приступить к использованию приложения.

### <a name="access-and-use-the-my-apps-portal-on-an-intune-managed-browser"></a>Доступ и использование портала Мои приложения в браузере под управлением Intune
Просмотр и использование приложения вашей организации из Intune Managed Browser в iOS и Android.

1. На мобильном устройстве, скачайте и установите приложение Intune Managed Browser из [Apple App Store](https://itunes.apple.com/us/app/microsoft-intune-managed-browser/id943264951?mt=8) и [Google Play Store](https://play.google.com/store/apps/details?id=com.microsoft.intune.mam.managedbrowser).

2. Откройте приложение Intune Managed Browser, перейдите к https://myapps.microsoft.com, или воспользоваться ссылкой, предоставляемую вашей организацией. Например, вам могут направляться на настроенной страницы для вашей организации, такие как https://myapps.microsoft.com/contoso.com.

    **Приложений** появится страница, отображение всех облачных приложений, владельцем которых является вашей организации и доступны для использования.

3. Из **приложений** выберите приложение, вы хотите начать использовать.
    
    Откроется новая страница для приложения, где можно войти (при необходимости) или приступить к использованию приложения.

## <a name="add-a-new-app-to-the-my-apps-portal"></a>Добавить новое приложение на портале Мои приложения
Если администратор предоставил вам разрешения, вы можете добавить новое приложение для **приложений** страницы.

1. Из **приложений** выберите **добавить приложение**.

    **Добавление приложений** появится страница.

    ![Добавление страницы приложения на портале Мои приложения](media/my-apps-portal/my-apps-portal-add-apps-page.png)

2. Выберите приложение, необходимо добавить в предоставленном списке, а затем выберите **добавить**.

3. Приложение добавляется в список на **приложений** страницы.

    Некоторые приложения могут требовать утверждения администратора, перед добавлением. В этом случае приложение не добавляется **приложений** странице, пока администратор должен утвердить его.
