# <a name="move-all-apps-assigned-to-a-connector-group-to-another-connector-group"></a>Перемещение всех приложений, назначенных группе соединителей, в другую группу соединителей

Этот пример сценария PowerShell перемещает все приложения прокси приложения Azure Active Directory (Azure AD), в настоящее время назначенные группе соединителей, в другую группу соединителей.
