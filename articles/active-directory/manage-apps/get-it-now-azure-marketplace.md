# <a name="get-it-now---add-an-app-from-the-azure-marketplace"></a>Добавление приложения из Azure Marketplace с помощью кнопки "Получить"

Почти готово! 

Эта статья содержит сведения об использовании Azure AD в качестве поставщика удостоверений для приложения. Вам нужно просто добавить его в клиент Azure AD. Чтобы узнать, как это сделать, ознакомьтесь с рядом руководств здесь: [Просмотр приложений в клиенте Azure AD](view-applications-portal.md).

Эта статья не содержит сведений об установке приложения на локальном компьютере или мобильном устройстве. Azure Marketplace предназначен для организаций, использующих приложения с клиентами Azure. Информацию о персональных компьютерах и устройствах можно найти в [магазине приложений Microsoft](https://www.microsoft.com/store/apps).
