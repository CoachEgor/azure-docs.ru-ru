# <a name="azure-ad-application-proxy-version-release-history"></a>Характеристики прокси приложения Azure AD: Журнал выпуска версий
В этой статье перечислены версии и возможности прокси приложения Azure Active Directory (Azure AD), которые были выпущены. Команда Azure AD регулярно обновляет прокси приложения, добавляя новые функции и возможности. Соединители прокси приложения обновляются автоматически при выпуске новой версии.

Ниже приведен список связанных ресурсов:

Resource |  Сведения
--------- | --------- |
Включение прокси приложения | Описываются предварительные требования для включения прокси приложения, Установка и регистрация соединителя в этом [руководстве](application-proxy-add-on-premises-application.md).
Сведения о соединителях прокси приложения Azure AD | Дополнительные сведения о [управления соединителя](application-proxy-connectors.md) и как соединители [автоматическое обновление](application-proxy-connectors.md#automatic-updates).
Скачивание соединителя прокси приложения Azure AD |  [Скачайте последнюю версию connector](https://download.msappproxy.net/subscription/d3c8b69d-6bf7-42be-a529-3fe9c2e70c90/connector/download).

## <a name="156120"></a>1.5.612.0

### <a name="release-status"></a>Состояние выпуска

20 сентября 2018 г.: выпущена версия для скачивания.

### <a name="new-features-and-improvements"></a>Новые функции и внесенные улучшения

- Добавлена поддержка WebSocket для приложения QlikSense. Дополнительные сведения о том, как интегрировать QlikSense с помощью прокси приложения, см. в этой [Пошаговое руководство](application-proxy-qlik.md). 
- Улучшенная мастер установки, чтобы было проще настроить исходящий прокси-сервер. 
- Установить TLS 1.2 в качестве протокола по умолчанию для соединителей. 
- Добавлен новый лицензионного соглашения (EULA).  

### <a name="fixed-issues"></a>Исправленные проблемы

- Исправлена ошибка, вызывавшая некоторые утечек памяти в соединителе.
- Обновленная версия служебной шины Azure, включая исправление проблемы с временем ожидания соединителя.

## <a name="154020"></a>1.5.402.0

### <a name="release-status"></a>Состояние выпуска

19 января 2018 г.: выпущена версия для скачивания.

### <a name="fixed-issues"></a>Исправленные проблемы

- Добавлена поддержка для пользовательских доменов, которым требуется перевода домена в файле cookie.

## <a name="151320"></a>1.5.132.0

### <a name="release-status"></a>Состояние выпуска 

25 мая 2017 г.: выпущена версия для скачивания. 

### <a name="new-features-and-improvements"></a>Новые функции и внесенные улучшения 

Улучшенный контроль над устройствами ограничения исходящих подключений соединители. 

## <a name="15360"></a>1.5.36.0

### <a name="release-status"></a>Состояние выпуска

15 апреля 2017 г.: выпущена версия для скачивания.

### <a name="new-features-and-improvements"></a>Новые функции и внесенные улучшения

- Упрощенное адаптации и управления с меньшим числом требуемых портов. Прокси приложения теперь требует открытия только два стандартных исходящие порты: 443 и 80. Прокси приложения по-прежнему используют только исходящие подключения, поэтому все компоненты в сети Периметра не требуется. Дополнительные сведения см. в разделе наших [документации по конфигурации](application-proxy-add-on-premises-application.md).  
- Если поддерживается внешние прокси-сервер или брандмауэр, теперь можно открыть в сети, DNS, вместо диапазона IP-адресов. Службы прокси приложения требуются подключения к *. msappproxy.net и *. servicebus.windows.net только.


## <a name="earlier-versions"></a>Более ранние версии

Если вы используете более ранних, чем 1.5.36.0 версия соединителя прокси приложения, обновление до последней версии, чтобы гарантировать использование последней полностью поддерживаемые функции.
