# <a name="change-the-name-or-logo-of-an-enterprise-application-in-azure-active-directory"></a>Изменение имени или логотипа корпоративного приложения в active-каталоге Azure

Вы можете легко изменить имя или логотип настраиваемого корпоративного приложения в Azure Active Directory (Azure AD). Вы должны иметь соответствующие разрешения, чтобы сделать эти изменения, и вы должны быть создателем пользовательского приложения.

## <a name="how-do-i-change-an-enterprise-applications-name-or-logo"></a>Как изменить название или логотип корпоративного приложения?

1. Вопием на [порталaz Active Directory](https://aad.portal.azure.com/) с учетной записью, которая является глобальным амином для каталога. Отображается страница **центра adin-центра Azure Active Directory.**
2. В области слева выберите **Корпоративные приложения**. Отображается список корпоративных приложений.
3. Выберите приложение. Отображается страница обзора приложения.
4. В панели обзора приложения под заголовком **«Управление»** выберите **свойства.** Отображается страница **Свойства.**
5. Если вы хотите изменить имя, выберите поле **имени,** введите новое имя и нажмите **Enter.**
6. Если вы хотите изменить логотип, найдите поле **логотипа** и выберите значок папки рядом с **файловым** полем, которое находится ниже текущего изображения логотипа приложения.

   ![Выбор команды "Свойства"](./media/change-name-or-logo-portal/change-logo.png)

   В противном случае, если вы не меняете логотип, перейдите к шагу 8.
7. В сборщике файлов выберите нужный файл в качестве нового логотипа. Имя файла отображается в поле под текущим изображением логотипа.

   > [!NOTE]
   > Azure требует, чтобы изображение логотипа было файлом PNG, и оно применяет ограничения по ширине, высоте и размеру файла. Пользовательские логотипы должны быть &times; точно 215 215 пикселей в размерах и быть в формате PNG. Мы рекомендуем использовать сплошной цветной фон без прозрачности логотипа приложения, чтобы казаться лучшим для пользователей.
8. Нажмите кнопку **Сохранить**. Если вы выбрали новый логотип, изображение поля **логотипа** изменяется, чтобы отразить новый файл логотипа.
