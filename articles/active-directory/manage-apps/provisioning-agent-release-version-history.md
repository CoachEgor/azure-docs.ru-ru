# <a name="azure-ad-connect-provisioning-agent-version-release-history"></a>Azure AD Connect агент подготовки: Журнал выпуска версий
В этой статье перечислены версии и возможности выпущенного агента подготовки Azure AD Connect. Команда Azure AD регулярно обновляет агент подготовки с помощью новых функций и функций. Агенты подготовки обновляются автоматически при выпуске новой версии. 

Мы рекомендуем включить автоматическое обновление для агентов, чтобы обеспечить наличие новейших компонентов и исправлений ошибок. Корпорация Майкрософт предоставляет прямую поддержку последней версии агента и одной версии.

## <a name="11670"></a>1.1.67.0

### <a name="release-status"></a>Состояние выпуска

9 сентября 2019 г.: Выпущено для автоматического обновления

### <a name="new-features-and-improvements"></a>Новые функции и внесенные улучшения

* Возможность настройки дополнительной трассировки и ведения журнала для отладки проблем с агентом подготовки
* Возможность выбирать только те атрибуты AD, которые настроены в сопоставлении для повышения производительности синхронизации.

### <a name="fixed-issues"></a>Исправленные проблемы

* Исправлена ошибка, при которой агент перешел в состояние без реагирования, если возникли проблемы с ошибками подключения к Active Directory.
* Исправлена ошибка, которая вызывала проблемы при чтении двоичных данных из Active Directory
* Исправлена ошибка, при которой агенту не удалось продлить отношение доверия с облачной службой идентификации в облаке.

## <a name="11300"></a>1.1.30.0

### <a name="release-status"></a>Состояние выпуска

23 января 2019 г.: выпущена версия для скачивания.

### <a name="new-features-and-improvements"></a>Новые функции и внесенные улучшения

* Переработанный агент подготовки & архитектура соединителя для повышения производительности, стабильности и надежности 
* Упрощенная конфигурация агента подготовки с помощью мастера установки на основе пользовательского интерфейса 
* Добавлена поддержка автоматического обновления агента.
