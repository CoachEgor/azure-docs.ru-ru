# <a name="servicenow-and-azure-active-directory"></a>ServiceNow и Azure Active Directory

Ускорьте внедрение ServiceNow в своей организации и повысьте внедрение пользователей с помощью предварительно созданной интеграции. Оптимизируйте управление ИТ-услугами, подключив ServiceNow к Azure Active Directory (Azure AD).

![Отображение логотипа приложения ServiceNow](./media/servicenow.png)

> [!div class="nextstepaction"]
> [Попробуйте использовать ServiceNow с Azure AD >](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>Что можно сделать с помощью ServiceNow и Azure AD?

- **Простой единый вход** . Исключите несколько имен пользователей и пароли и упростите доступ, позволяя пользователям входить в ServiceNow с помощью учетных записей Azure AD.
- **Автоматизация подготовки пользователей** . Автоматизируйте создание, Обслуживание и удаление учетных записей пользователей в ServiceNow при присоединении, перемещении или выходе пользователей из Организации.
- **Обеспечьте безопасность, управление и мониторинг** доступа — безопасный доступ к ServiceNow, применяя политики условного доступа, контролируя доступ с помощью проверок доступа и отслеживая доступ с помощью журналов входа.

## <a name="get-started"></a>Начало работы

Настройте [подготовку пользователей](https://aka.ms/ServiceNowProvisioning) и [единый вход](https://aka.ms/ServiceNowSSO) для ServiceNow с помощью наших простых руководств по настройке.
