# <a name="box-and-azure-active-directory"></a>Box и Azure Active Directory

Ускорьте внедрение Box в своей организации и повысьте внедрение пользователей с помощью предварительно созданной интеграции. Разрешить пользователям быстро сотрудничать, подключившись к Azure Active Directory (Azure AD).

![Показывает эмблему приложения Box](./media/box.png)

> [!div class="nextstepaction"]
> [Поле try с Azure AD >](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Что можно сделать с помощью Box и Azure AD?

- **Простой единый вход** . исключайте несколько имен пользователей и паролей и упростите доступ, позволяя пользователям входить в Box с помощью учетных записей Azure AD.
- **Автоматизация подготовки пользователей** . Автоматизируйте создание, Обслуживание и удаление учетных записей пользователей в Box, когда пользователи присоединяются, переходят или оставляют свою организацию.
- **Обеспечьте безопасность, контролируйте и отслеживайте** доступ к Box, применяя политики условного доступа, контролируя доступ с помощью проверок доступа и отслеживая доступ с помощью журналов входа.
