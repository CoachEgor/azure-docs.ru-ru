# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet и Azure Active Directory

Ускорьте внедрение Smartsheet в своей организации и повысьте внедрение пользователей с помощью предварительно созданной интеграции. Разрешите пользователям быстро сотрудничать, подключив Smartsheet к Azure Active Directory (Azure AD).

![Показывает логотип приложения Smartsheet](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Попробуйте Smartsheet с помощью Azure AD >](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>Что можно сделать с помощью Smartsheet и Azure AD?

- **Простой единый вход** . исключайте несколько имен пользователей и паролей и упростите доступ, позволяя пользователям входить в Smartsheet с помощью учетной записи Azure AD.
- **Автоматизация подготовки пользователей** . Автоматизируйте создание, Обслуживание и удаление учетных записей пользователей в Smartsheet при присоединении, перемещении или выходе пользователей из Организации.
- **Обеспечьте безопасность, контролируйте и отслеживайте** доступ к Smartsheet путем применения политик условного доступа, контроля доступа через проверки доступа и мониторинга доступа с помощью журналов входа.

## <a name="get-started"></a>Начало работы

Настройте [подготовку пользователей](https://aka.ms/SmartsheetProvisioningDoc) и [единый вход](https://aka.ms/SmartsheetSSODoc) для Smartsheet с помощью наших простых руководств по настройке.
