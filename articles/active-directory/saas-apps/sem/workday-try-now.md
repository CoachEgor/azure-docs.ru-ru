# <a name="workday-and-azure-active-directory"></a>Workday и Azure Active Directory

Подключайте Workday к Azure Active Directory (Azure AD) с помощью встроенной интеграции и автоматически подготавливайте или отзывайте учетные записи пользователей при присоединении, перемещении или выходе сотрудников.

![Показывает эмблему приложения Workday](./media/workday.png)

> [!div class="nextstepaction"]
> [Попробуйте использовать Workday с Azure AD >](https://aka.ms/WorkdayTryItNow)

## <a name="what-can-you-do-with-workday-and-azure-ad"></a>Что можно делать с Workday и Azure AD?

- **Автоматизация подготовки пользователей** — автоматическое назначение приложений сотрудникам на основе их группы, роли или других атрибутов в Workday.
- **Синхронизация профилей сотрудников** . Обеспечьте актуальность данных о сотрудниках между Azure AD и Workday.
- **Упростите подключение и** отключение. автоматически подготавливайте учетные записи пользователей для новых сотрудников, чтобы их можно было проработать на один день и уменьшить риски безопасности, автоматически выполнив автоматическую подготовку учетных записей пользователей при выходе сотрудников.
- **Простой единый вход** Исключите имя пользователя и пароли и разрешите пользователям входить в систему с помощью учетных записей организации, размещенных в Azure AD.
