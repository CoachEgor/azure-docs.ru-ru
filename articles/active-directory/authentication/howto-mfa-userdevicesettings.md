# <a name="manage-user-settings-with-azure-multi-factor-authentication-in-the-cloud"></a>Управление параметрами пользователей с помощью Многофакторной идентификации Azure в облаке

Как администратор вы можете управлять следующими параметрами пользователей и устройств.

* Параметр "Требовать у выбранных пользователей сообщать о способах связи еще раз"
* Удалять пароли приложений.
* Требовать Многофакторную идентификацию на всех доверенных устройствах.

## <a name="manage-authentication-methods"></a>Управление методами проверки подлинности

В качестве администратора, назначенного роль администратора аутентификации, вы можете потребовать от пользователей сбросить свой пароль, перерегистрироваться на МИД или отозвать существующие сеансы MFA с объекта пользователя.

![Управление методами проверки подлинности с портала Azure](./media/howto-mfa-userdevicesettings/manage-authentication-methods-in-azure.png)

1. Войдите на [портал Azure](https://portal.azure.com).
1. Слева выберите**пользователей** >  **Azure Active Directory** > **Все пользователи.**
1. Выберите пользователя, на который вы хотите выполнить действие, и выберите **методы аутентификации.**
   - **Сбросить пароль** сбросить пароль пользователя и назначить временный пароль, который должен быть изменен при следующем входе.
   - **Требование перерегистрации МИД** сделает так, что, когда пользователь подписывает в следующий раз, они будут предложено настроить новый метод проверки подлинности МИД.
   - **Отмена сессии МИД** очищает запоминаемых сессий МИД пользователя и требует, чтобы он выполнил МИД в следующий раз, когда это требуется по полису на устройстве.

## <a name="delete-users-existing-app-passwords"></a>Удалять существующие пароли приложений пользователей

Этот параметр удаляет все созданные пользователем пароли приложений. Работа не использующих браузер приложений, связанных с этими паролями, приостанавливается, пока не будут созданы новые пароли. Для выполнения этого действия требуются глобальные разрешения администратора.

### <a name="how-to-delete-users-existing-app-passwords"></a>Как удалить существующие пароли приложений пользователей

1. Войдите на [портал Azure](https://portal.azure.com).
2. Слева выберите**пользователей** >  **Azure Active Directory** > **Все пользователи.**
3. Справа на панели инструментов выберите **Многофакторная идентификация**. Откроется страница Многофакторной идентификации.
4. Установите флажок рядом с именем пользователя или пользователей, которыми требуется управлять. Справа отображается список вариантов быстрого шага.
5. Выберите **Управление параметрами пользователя**.
6. Установите флажок **Удалить все существующие пароли приложений, созданные выбранными пользователями**.
   ![Удалить все существующие пароли приложений](./media/howto-mfa-userdevicesettings/deleteapppasswords.png)
7. Нажмите кнопку **Сохранить**.
8. Нажмите **близко**.
