# <a name="start-using-privileged-identity-management"></a>Начало работы с управлением привилегированными пользователями

С помощью Privileged Identity Management (PIM) вы можете управлять, управлять и контролировать доступ в рамках организации Azure Active Directory (Azure AD). Эта область включает доступ к ресурсам Azure, Azure AD и другим онлайн-сервисам Майкрософт, таким как Office 365 или Microsoft Intune.

В этой статье описывается, как включить и начать использовать Privileged Identity Management.

## <a name="prerequisites"></a>Предварительные требования

Чтобы использовать Privileged Identity Management, необходимо иметь одну из следующих лицензий:

- Azure AD Premium P2
- Enterprise Mobility + Security (EMS) E5.

Для получения дополнительной информации см. [Требования к лицензиям для использования Privileged Identity Management.](subscription-requirements.md)

## <a name="sign-up-pim-for-azure-ad-roles"></a>Регистрация PIM для ролей Azure AD

После включения privileged Identity Management для каталога необходимо зарегистрироваться в Privileged Identity Management для управления ролями Azure AD.

1. Открытое **управление привилегированным удостоверением личности Azure AD.**

1. Выберите **роли AD Azure**.

    ![Зарегистрируйтесь в привилегированном управлении идентификацией для ролей Azure AD](./media/pim-getting-started/sign-up-pim-azure-ad-roles.png)

1. Выберите **Зарегистрироваться.**

1. В сообщении, которое появляется, нажмите **«Да»,** чтобы зарегистрироваться в привилегированном управлении идентификационными данными для управления ролями Azure AD.

    ![Подпишитесь на привилегированное управление идентификацией для сообщения ролей Azure AD](./media/pim-getting-started/sign-up-pim-message.png)

    После завершения регистрации станут доступны параметры Azure AD. Вам может потребоваться обновить портал.

    Для получения информации о том, как обнаружить и выбрать [Discover Azure resources to manage in Privileged Identity Management](pim-resource-roles-discover-resources.md)ресурсы Azure для защиты с помощью Privileged Identity Management, см.

## <a name="navigate-to-your-tasks"></a>Переход к задачам

После настройки privileged Identity Management можно приступить к выполнению задач управления удостоверениями.

![Навигационное окно в привилегированном управлении идентификацией, отображающее задачи и параметры управления](./media/pim-getting-started/pim-quickstart-tasks.png)

| Задача и управление | Описание |
| --- | --- |
| **Мои роли**  | Отображается список назначенных вам действительных и активных ролей. Здесь можно активировать все действительные назначенные роли. |
| **Мои запросы** | Отображаются все запросы, ожидающие активации назначения действительных ролей. |
| **Утверждение запросов** | Отображается список запросов на активацию действительных ролей в вашем каталоге для пользователей, которых вы должны подтвердить. |
| **Проверка доступа** | Приведены активные проверки доступа, которые вы должны выполнить (для себя или для кого-то другого). |
| **Роли Azure AD** | Отображает панель мониторинга и настройки для привилегированных администраторов ролей для управления назначениями ролей Azure AD. Эта панель мониторинга отключена для всех, кто не является администратором привилегированных ролей. Эти пользователи имеют доступ к специальной панели мониторинга My view (Мое представление). На панели мониторинга My view (Мое представление) отображаются только сведения о пользователе, который получает доступ к панели мониторинга, а не о клиенте. |
| **Ресурсы Azure** | Отображаются панель мониторинга и параметры для администраторов привилегированных ролей, чтобы они могли управлять назначениями ролей ресурса Azure. Эта панель мониторинга отключена для всех, кто не является администратором привилегированных ролей. Эти пользователи имеют доступ к специальной панели мониторинга My view (Мое представление). На панели мониторинга My view (Мое представление) отображаются только сведения о пользователе, который получает доступ к панели мониторинга, а не о клиенте. |

## <a name="add-a-pim-tile-to-the-dashboard"></a>Добавление плитки PIM на панель мониторинга

Чтобы упростить открытие Privileged Identity Management, добавьте плитку Privileged Identity Management в панель мониторинга портала Azure.

1. Войдите на [портал Azure](https://portal.azure.com/).

1. Выберите **все службы** и найдите службу **управления привилегированными удостоверениями личности Azure AD.**

    ![Azure AD Privileged Identity Management в разделе "Все службы"](./media/pim-getting-started/pim-all-services-find.png)

1. Выберите Быстрый запуск управления привилегированными удостоверениями.

1. Проверьте **лезвие Pin на приборной панели,** чтобы прикрепить лезвие privileged Identity Management к приборной панели.

    ![Значок Pushpin прикрепить лезвие управления привилегированными удостоверениями к приборной панели](./media/pim-getting-started/pim-quickstart-pin-to-dashboard.png)

    На панели мониторинга Azure вы увидите такие плитки:

    ![Привилегированного управления идентификацией Квикстарт плитка на приборной панели](./media/pim-getting-started/pim-quickstart-dashboard-tile.png)
