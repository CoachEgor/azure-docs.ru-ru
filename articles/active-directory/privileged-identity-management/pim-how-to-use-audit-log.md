# <a name="view-audit-history-for-azure-ad-roles-in-pim"></a>Просмотреть журнал аудита для ролей Azure AD в PIM

Журнал аудита Azure Active Directory (Azure AD) Privileged Identity Management (PIM) можно использовать для просмотра всех назначений ролей и активаций за последние 30 дней для всех привилегированных ролей. Если вы хотите просмотреть весь журнал аудита действий в каталоге, а также администратор, конечный пользователь и действия при синхронизации, можно использовать [отчеты безопасности и действия Azure Active Directory](../reports-monitoring/overview-reports.md).

## <a name="view-audit-history"></a>Просмотр журнала аудита

Выполните следующие действия, чтобы просмотреть журнал аудита для ролей Azure AD.

1. Войдите на [портал Azure](https://portal.azure.com/) с учетными данными пользователя, который является участником роли [Администратор привилегированных ролей](../users-groups-roles/directory-assign-admin-roles.md#privileged-role-administrator).

1. Откройте страницу **Azure AD Privileged Identity Management**.

1. Выберите **Роли Azure AD**.

1. Нажмите кнопку **журнал аудита ролей каталога**.

    В зависимости от журнала аудита Гистограмма отображается вместе с всего активаций, максимального и среднего количества активаций в день.

    ![Журнал аудита ролей каталога](media/pim-how-to-use-audit-log/directory-roles-audit-history.png)

    В нижней части страницы таблицы отображается с информацией о каждом действии в журнале аудита доступны. Столбцы имеют следующий смысл:

    | столбец | Описание |
    | --- | --- |
    | Время | Когда произошло действие. |
    | Запрашивающая сторона | Пользователь, запросивший активацию или изменение роли. Если значение равно **система Azure**, проверьте наличие в журнале аудита Azure Дополнительные сведения. |
    | Действие | Действия, предпринятые запросившей стороной. Операции могут включать назначение, снять, активировать, деактивировать или AddedOutsidePIM. |
    | Участник | Пользователь, который активируется или роли. |
    | Роль | Роль, назначенная или активированная пользователем. |
    | Мотивация | Текст, введенный в поле Причина во время активации. |
    | Окончание срока действия | После активации роли истечения срока действия. Применяется только к соответствующих назначений ролей. |

1. Чтобы отсортировать журнал аудита, щелкните **время**, **действие**, и **роли** кнопки.

## <a name="filter-audit-history"></a>Фильтрация журнала аудита

1. В верхней части страницы журнала аудита, щелкните **фильтра** кнопки.

    **Обновить параметры диаграммы** откроется панель.

1. В **диапазон времени**, выберите диапазон времени.

1. В **ролей**, добавление меток для ролей, которые вы хотите просмотреть.

    ![Область параметров диаграммы обновления](media/pim-how-to-use-audit-log/update-chart-parameters.png)

1. Нажмите кнопку **сделать** для просмотра журнала отфильтрованный аудита.
