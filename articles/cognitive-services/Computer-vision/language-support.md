# <a name="language-support-for-computer-vision"></a>Языковая поддержка компьютерного зрения

Некоторые функции компьютерного зрения поддерживают несколько языков; любые функции, не упомянутые здесь, поддерживают только английский язык.

## <a name="text-recognition"></a>Распознавание текста

Компьютерное зрение может распознавать текст на многих языках. В частности, [API OCR](https://westus.dev.cognitive.microsoft.com/docs/services/5adf991815e1060e6355ad44/operations/56f91f2e778daf14a499e1fc) поддерживает различные языки, в то время как [Read](https://westus.dev.cognitive.microsoft.com/docs/services/5adf991815e1060e6355ad44/operations/2afb498089f74080d7ef85eb) API и [Распознаваемый текстовый](https://westus.dev.cognitive.microsoft.com/docs/services/5adf991815e1060e6355ad44/operations/587f2c6a154055056008f200) API поддерживают только английский язык. Для получения дополнительной информации об этой функциональности и преимуществах каждого API можно ознакомиться с [печатным и рукописным текстом.](concept-recognizing-text.md)

OCR автоматически обнаруживает язык вхоточного материала, поэтому нет необходимости указывать языковой код в вызове API. Однако языковые коды всегда возвращаются как значение `"language"` узла в ответе JSON.
