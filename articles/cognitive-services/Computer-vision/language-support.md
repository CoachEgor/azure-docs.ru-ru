# <a name="language-support-for-computer-vision"></a>Поддержка языков для компьютерного зрения

Некоторые функции компьютерного поддержку нескольких языков; все функции, не упомянутые здесь поддерживают только на английском языке.

## <a name="text-recognition"></a>Распознавание текста

Компьютерное зрение можно распознать множества языков. В частности [OCR](https://westus.dev.cognitive.microsoft.com/docs/services/5adf991815e1060e6355ad44/operations/56f91f2e778daf14a499e1fc) API поддерживает широкий спектр языков, тогда как [чтения](https://westus.dev.cognitive.microsoft.com/docs/services/5adf991815e1060e6355ad44/operations/2afb498089f74080d7ef85eb) API и [распознавание текста](https://westus.dev.cognitive.microsoft.com/docs/services/5adf991815e1060e6355ad44/operations/587f2c6a154055056008f200) API поддерживают только на английском языке. См. в разделе [распознавание текста печатных и рукописные](concept-recognizing-text.md) Дополнительные сведения о этой функции и преимущества каждого API-интерфейса.

OCR автоматически определяет язык ввода материала, поэтому нет необходимости указывать код языка в вызове API. Тем не менее, всегда возвращаются коды языка для параметра `"language"` узел в ответе JSON.
