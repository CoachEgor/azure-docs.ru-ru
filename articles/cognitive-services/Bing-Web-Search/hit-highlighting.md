# <a name="using-decoration-markers-to-highlight-text"></a>Использование маркеров оформления для выделения текста

Bing поддерживает выделение хита, которое отмечает термины запроса (или другие термины, которые Bing считает актуальными) в строках отображения некоторых ответов. Например, результат веб-страницы `displayUrl`и `snippet` поля могут содержать отмеченные термины запроса. `name` 

По умолчанию Bing не включает маркеры выделения в отображаемых строках. Чтобы включить маркеры, `textDecorations` включите параметр запроса `true`в запрос и установите его на .

## <a name="hit-highlighting-example"></a>Пример подсветки хита

Следующий пример показывает веб-результат для `Sailing Dinghy`. Bing ознаменовал начало и конец термина запроса с использованием символов E000 и E001 Unicode.
  
![Выделение совпадений](./media/cognitive-services-bing-web-api/bing-hit-highlighting.png) 

Прежде чем отображать результат в пользовательском интерфейсе, замените символы Unicode на те, которые подходят для формата дисплея.

## <a name="marker-formatting"></a>Форматирование маркеров

Bing предоставляет возможность использования символов Unicode или HTML-тегов в качестве маркеров. Чтобы указать, какие маркеры использовать, включите параметр запроса [textFormat:](https://docs.microsoft.com/rest/api/cognitiveservices-bingsearch/bing-web-api-v7-reference#textformat) 

| Значение             | Маркер                       |
|-------------------|------------------------------|
| `textFormat=Raw`  | Символы Unicode (по умолчанию) |
| `textFormat=HTML` | HTML символы              |

## <a name="additional-text-decorations"></a>Дополнительные текстовые украшения

Bing может вернуть несколько различных текстовых украшений. Например, `Computation` ответ может содержать маркеры subscript `log(2)` для `expression` термина запроса в поле.

![Маркеры вычислений](./media/cognitive-services-bing-web-api/bing-markers-computation.png) 

Если в запросе не указаны украшения, `expression` поле будет содержать `log10(2)`. 

Если `textDecorations` `true`это , Bing может включать следующие маркеры в строках отображения ответов. Если эквивалентного HTML-тега нет, ячейка таблицы пуста.

|Юникод|HTML|Описание
|-|-|-

|U+E001|\</b>|Помечает окончание термина запроса
|U+E002|\<i>|Помечает начало содержимого, выделенного курсивом 
|U+E003|\</i>|Помечает конец содержимого, выделенного курсивом
|U+E004|\<br/>|Помечает разрыв строки
|U+E005||Помечает начало номера телефона

|U+E008||Помечает конец адреса
|U+E009|\&nbsp;|Помечает неразрывный пробел
|U+E00C|\<strong>|Помечает начало содержимого, выделенного полужирным шрифтом
|U+E00D|\</strong>|Помечает конец содержимого, выделенного полужирным шрифтом
|U+E00E||Помечает начало содержимого, фон которого светлее, чем фон окружающего содержимого
