# <a name="resize-and-crop-thumbnail-images"></a>Изменение размера и обрезка эскизов

Некоторые ответы с помощью API поиска Bing содержат URL-адреса эскизов изображений, обслуживаемых Bing, в котором можно изменить размер и обрезка и может содержать параметры запроса. Пример:

`https://<host>/th?id=AMMS_92772df988...&w=110&h=73&rs=1&qlt=80&cdv=1&pid=16.1`

Если отобразить подмножество этих эскизов, предоставляет возможность просматривать остальные изображения.

> [!NOTE]
> Убедитесь, что Обрезка и изменение размера эскизов изображений будет предоставлять сценарий поиска, который учитывает права третьих лиц, требуемые для API Bing для поиска [использовать и требований к отображению](use-display-requirements.md).

## <a name="resize-a-thumbnail"></a>Изменение размера эскизов 

Чтобы изменить размер эскиза, Bing рекомендует указывать только один `w` (ширина) или `h` параметров запроса (высота) в URL-адрес эскиза. Задание высоты или ширины позволяет сохранить исходные пропорции изображения Bing. Укажите ширину и высоту в пикселях. 

Например, если исходный эскиз — 480 x 620:

`https://<host>/th?id=JN.5l3yzwy%2f%2fHj59U6XhssIQ&pid=Api&w=480&h=620`

И вы хотите уменьшить его размер, задайте `w` параметра новое значение (например `336`) и удалите `h` параметр:

`https://<host>/th?id=JN.5l3yzwy%2f%2fHj59U6XhssIQ&pid=Api&w=336`

Если указать только высоту или ширину эскиза, исходные пропорции изображения сохранятся. Если указать оба параметра и пропорции не поддерживается, Bing добавит белые внутренние поля к границе изображения.

Например при изменении размера образа 480 x 359 до размера 200 x 200 без обрезки, полная ширина будет содержать изображение, но высоту будет содержать 25 пикселей белые в верхней и нижней части изображения. Если образ был 359 x 480, левую и правую границы будет содержать белые поля. При обрезке изображения белые поля не добавляются.  

Ниже показано изображение эскиза в исходном размере (480 x 300).  
  
![Исходное изображение пейзажа](./media/resize-crop/bing-resize-crop-landscape.png)  
  
Ниже показано изображение, размер которого изменен до 200 x 200. Пропорции сохранены и верхней и нижней границ, дополняются белым (здесь черной рамкой показывают заполнение).  
  
![Изображение пейзажа с измененным размером](./media/resize-crop/bing-resize-crop-landscape-resized.png)  

Если указать размеры, превышающие исходной ширины и высоты изображения, Bing добавит белые внутренние поля левой и верхней границ.  

## <a name="request-different-thumbnail-sizes"></a>Разные размеры эскизов запросов

Чтобы запросить размер различных эскиз, удалите все параметры запроса из URL-адрес эскиза, за исключением `id` и `pid` параметров. Добавьте одно `&w` (ширина) или `&h` параметра запроса (высота) с размером нужный образ в точках, но не оба. Bing сохранят исходные пропорции изображения. 

Чтобы увеличить ширину изображения, указанного выше URL-адресом на 165 пикселей ниже, необходимо использовать следующий URL-адрес:

`https://<host>/th?id=AMMS_92772df988...&w=165&pid=16.1`

Если вы запрашивали изображение, которое больше, чем исходный размер изображения, Bing добавляет белые поля вокруг изображения, при необходимости. Например, если исходное изображение размер составляет 474 x 316 и установке `&w` 500, Bing будет возвращать образ 500 x 333. Этот образ будет иметь 8,5 пикселей белый внутренних вдоль верхней и нижней границ и 13 точек для заполнения для левого и правого краев.

Чтобы предотвратить добавление белые поля, если запрошенный размер больше исходного размера изображения в Bing, задайте `&p` параметр 0 запроса. Например, если включить `&p=0` параметра выше URL-адреса Bing вернет образ 474 x 316 вместо 500 x 333 изображения:

`https://<host>/th?id=AMMS_92772df988...&w=500&p=0&pid=16.1`

Если задан и `&w` и `&h` параметры запроса, Bing будет сохраняться соотношение сторон изображения и добавляет белые поля по мере необходимости. Например, если исходное изображение размер составляет 474 x 316 и задать параметры ширины и высоты до размера 200 x 200 (`&w=200&h=200`), Bing возвращает изображение, которое содержит 33 точек белые поля сверху и снизу. При включении `&p` параметр запроса Bing возвращает 200 x 134 изображения.

## <a name="crop-a-thumbnail"></a>Обрезать эскиза 

Чтобы обрезать изображение, включите `c` параметра запроса (обрезать). Можно использовать следующие значения:
  
- `4` &mdash; Отношение для людей с нарушениями зрения  
- `7` &mdash; Смарт-отношение  

### <a name="smart-ratio-cropping"></a>Интеллектуальная Обрезка отношение

При запросе обрезки Smart Ratio (задавая `c` параметр `7`), Bing будет обрезать изображение в центре его интересующей области за ее пределами, сохраняя пропорции изображения. Интересующая область — это область изображения, где по определению Bing содержится самая важная часть. Ниже показан пример интересующей области.  
  
![Интересующая область](./media/resize-crop/bing-resize-crop-regionofinterest.png)

Изменение размера изображения и запросе обрезки Smart Ratio, Bing позволяет сократить изображение запрошенный размер сохранением пропорций. Bing затем обрезает его, в зависимости от измененного размера. Например если размер по ширине меньше или равно размеру, Bing будет обрезать изображение, слева и справа от центра интересующей области. В противном случае Bing позволяет обрезать его сверху и снизу центра интересующей области.  
  
 
Ниже показано изображение, уменьшенное до размера 200 x 200 с использованием обрезки Smart Ratio. Так как Bing измеряет изображения из верхнего левого угла, в нижней части окна изображение обрезается. 
  
![Изображение пейзажа, обрезанное до размера 200 x 200](./media/resize-crop/bing-resize-crop-landscape200x200c7.png) 
  
Ниже показано изображение, уменьшенное до размера 200 x 100 с использованием обрезки Smart Ratio. Так как Bing измеряет изображения из верхнего левого угла, в нижней части окна изображение обрезается. 
   
![Изображение пейзажа, обрезанное до размера 200 x 100](./media/resize-crop/bing-resize-crop-landscape200x100c7.png)
  
Ниже показано изображение, уменьшенное до размера 100 x 200 с использованием обрезки Smart Ratio. Так как Bing измеряет образа в центре, обрезаются левую и правую части изображения.
  
![Изображение пейзажа, обрезанное до размера 100 x 200](./media/resize-crop/bing-resize-crop-landscape100x200c7.png) 

Если Bing не может определить интересующую вас область изображения, которую служба будет использовать Обрезка Blind отношение.  

### <a name="blind-ratio-cropping"></a>Обрезка Blind отношение

При запросе обрезки Blind Ratio (задавая `c` параметр `4`), Bing использует следующие правила для обрезки изображения.  
  
- Если `(Original Image Width / Original Image Height) < (Requested Image Width / Requested Image Height)`, изображение измеряется от верхнего левого угла и обрезается внизу.  
- Если `(Original Image Width / Original Image Height) > (Requested Image Width / Requested Image Height)`, изображение измеряется от центра и обрезается слева и справа.
