# <a name="project-acoustics-unity-integration"></a>Интеграция проектов с акустическими характеристиками Unity
В этом разделе описывается интеграция подключаемого модуля Unity в проект Unity.

Требования к программному обеспечению:
* [Unity 2018.2+](https://unity3d.com) для Windows;
* [Пакет Unity для акустических характеристик проекта](https://www.microsoft.com/download/details.aspx?id=57346)

## <a name="import-the-plugin"></a>Импорт подключаемого модуля
Импортируйте пакет акустики UnityPackage в свой проект. 
* В Unity перейдите в раздел **Assets > Import Package > Custom Package...** (Ресурсы > Импорт пакета > Пользовательский пакет...).

    ![Снимок экрана: меню пакета импорта Unity](media/import-package.png)  

* Выберите пакет **ProjectAcoustics.unitypackage**.

* Нажмите кнопку " **Импорт** ", чтобы интегрировать пакет Unity в свой проект.  

    ![Снимок экрана: диалоговое окно импорта пакета Unity](media/import-dialog.png)  

Если вы импортируете подключаемый модуль в имеющийся проект, в его корне уже может присутствовать файл **mcs.rsp**, в котором указаны параметры для компилятора C#. Вам необходимо объединить содержимое этого файла с файлом mcs.rsp, входящим в состав подключаемого модуля Project Acoustics.

## <a name="enable-the-plugin"></a>Включение подключаемого модуля
Часть сборки инструментария акустики требует версии среды выполнения сценариев .NET 4.x. В результате импорта пакетов обновятся параметры проигрывателя Unity. Перезапустите Unity, чтобы этот параметр вступил в силу.

![Снимок экрана с панелью параметров проигрывателя в Unity](media/player-settings.png)

![Снимок экрана с выбором .NET 4.5 на панели параметров проигрывателя в Unity](media/net45.png)

## <a name="set-up-audio-dsp"></a>Настройка DSP для аудио
Акустические характеристики проекта включают в себя DSP среды выполнения аудио, которая интегрируется в платформу Unity Audio Engine спатиализер. Он включает пространственное ХРТФ и панорамирование. Включите поддержку по проектам DSP, открыв параметры звука Unity с помощью команды **Edit > Project settings > Audio**, а затем выбрав в качестве **подключаемого модуля Спатиализер** для проекта **акустические характеристики проекта** . Убедитесь, что **Размер буфера DSP** имеет значение лучшая производительность.

![Снимок экрана: панель параметров проекта Unity](media/project-settings.png)  

![Снимок экрана: панель параметров Unity Спатиализер с выбранными акустическими характеристиками проекта спатиализер](media/choose-spatializer.png)

Затем откройте микшер звука (**окно > звуковой микшер**). Убедитесь, что у вас есть хотя бы один микшер с одной группой. Если его нет, нажмите кнопку "+" справа от раздела **Mixers** (Микшеры). Щелкните правой кнопкой мыши нижнюю часть полосы каналов в разделе "эффекты" и добавьте эффект микшера " **акустические акустики проекта** ". Обратите внимание, что одновременно поддерживается только один микшер Project Acoustics.

![Снимок экрана с микшером "звуковые характеристики проекта Unity" размещения](media/audio-mixer.png)

## <a name="enable-acoustics-on-sound-sources"></a>Включить акустические характеристики в звуковых источниках
Создайте источник звука. Установите флажок в нижней части панели проверки AudioSource **Spatialize** (Пространственный звук). Для параметра **Spatial Blend** (Пространственное смешивание) выберите значение 3D.  

![Снимок экрана: панель источника данных Unity](media/audio-source.png)

## <a name="enable-acoustic-design"></a>Включить проектирование акустических характеристик
Присоедините **акаустиксаджуст** сценария к звуковому источнику в сцене, чтобы включить дополнительные параметры исходного кода, нажав кнопку **Добавить компонент** и выбрав **скрипты > Акустические настройки**:

![Снимок экрана со скриптом AcousticsAdjust в Unity](media/acoustics-adjust.png)
