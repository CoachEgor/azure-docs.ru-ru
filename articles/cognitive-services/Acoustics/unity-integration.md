# <a name="project-acoustics-unity-integration"></a>Интеграция проектов с акустическими характеристиками Unity
В этой статье описывается, как интегрировать подключаемый модуль Unity в проект Unity.

Требования к программному обеспечению:
* [Unity 2018.2+](https://unity3d.com) для Windows;
* [Пакет Unity для акустических характеристик проекта](https://www.microsoft.com/download/details.aspx?id=57346)

## <a name="import-the-plug-in"></a>Импорт подключаемого модуля
1. Импортируйте акустические пакет unitypackage в свой проект. 
 В Unity перейдите к разделу **Ресурсы** > **Import Package** (Импорт пакета)  > **Custom Package** (Пользовательский пакет).

    ![Меню пакета импорта Unity](media/import-package.png)  

1. Выберите пакет **ProjectAcoustics.unitypackage**.

1. Нажмите кнопку **Import** (Импорт), чтобы интегрировать пакет Unity в свой проект.

    ![Диалоговое окно Unity Import Package (Импорт пакетов в Unity)](media/import-dialog.png)  

Если подключаемый модуль импортируется в существующий проект, в корневом каталоге проекта может уже быть файл *MCS. rsp* . В этом файле указываются параметры C# компилятора. Объедините содержимое этого файла с файлом MCS. rsp, который поставляется с подключаемым модулем "акустические характеристики проекта".

## <a name="enable-the-plug-in"></a>Включение подключаемого модуля
Часть сборки инструментария акустики требует версии среды выполнения сценариев .NET 4.*x*. При импорте пакетов обновляются параметры проигрывателя Unity. Перезапустите Unity, чтобы этот параметр вступил в силу.

![Панель параметров проигрывателя Unity](media/player-settings.png)

![Выбранная панель параметров проигрывателя Unity с выбранной .NET 4,5](media/net45.png)

## <a name="set-up-audio-dsp"></a>Настройка DSP для аудио
Акустические характеристики проекта включают в себя DSP среды выполнения аудио, которая интегрируется в платформу Unity Audio Engine спатиализер. Он включает пространственное ХРТФ и панорамирование. Чтобы включить DSP по проектам, выберите **изменить** > **Параметры проекта** > **звук** , чтобы открыть параметры звука Unity. Выберите в качестве **подключаемого модуля спатиализер** для вашего проекта **акустические характеристики проекта** . Убедитесь, что **Размер буфера DSP** имеет значение *лучшая производительность*.

![Меню параметров проекта Unity](media/project-settings.png)  

![Панель параметров Unity Спатиализер с выбранными акустическими характеристиками проекта спатиализер](media/choose-spatializer.png)

Затем откройте микшер звука (**окно** > **аудио микшер**). Убедитесь, что у вас есть по крайней мере один микшер с одной группой. Если у вас ее нет, нажмите кнопку **+** справа от **миксерс**. Щелкните правой кнопкой мыши нижнюю часть полосы канала в разделе эффектов и добавьте эффект **Microsoft Acoustics Mixer**. В каждый момент времени поддерживается только один микшер для акустических характеристик проекта.

![Микшер звука Unity, в котором размещается микшер акустических характеристик проекта](media/audio-mixer.png)

## <a name="enable-acoustics-on-sound-sources"></a>Включить акустические характеристики в звуковых источниках
Создание источника аудио: установите флажок **спатиализе** в нижней части панели инспектора аудиосаурце. Убедитесь, что для **пространственного смешения** задано значение Full *3D*.  

![Панель источника звука Unity](media/audio-source.png)

## <a name="enable-acoustic-design"></a>Включить проектирование акустических характеристик
Присоедините сценарий *акаустиксаджуст* к звуковому источнику в сцене, чтобы включить дополнительные параметры проектирования источника: выберите **Добавить компонент** и выберите **скрипты** , > **акустические настройки**.

![Скрипт AcousticsAdjust в Unity](media/acoustics-adjust.png)
