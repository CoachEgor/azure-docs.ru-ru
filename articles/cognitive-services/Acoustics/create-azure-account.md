# <a name="project-acoustics-azure-batch-account-setup"></a>Настройка акустических параметров учетной записи пакетной службы Azure
В этом практическом задании описывается настройка учетной записи пакетной службы Azure для использования со акустическими характеристиками Unity и нереальными модулями интеграции.

## <a name="get-an-azure-subscription"></a>Получить подписку Azure
До настройки учетных записей хранения и учетной записи пакетной службы необходимо получить [подписку Azure](https://azure.microsoft.com/free/). Если регистрируетесь впервые, Azure предоставляет несколько бесплатных ресурсов, которые ограничены по времени, и кредит в сумме 200 долл. США.

## <a name="create-azure-batch-and-storage-accounts"></a>Создание учетных записей хранения и учетной записи пакетной службы Azure
Далее следуйте [этим инструкциям](https://docs.microsoft.com/azure/batch/batch-account-create-portal), чтобы настроить учетную запись пакетной службы Azure и связанные учетные записи хранения Azure.

Выберите параметры по умолчанию для учетных записей хранения и учетной записи пакетной службы:
  
  ![Снимок экрана: параметры новых учетных записей Azure Batch, отображающие параметры по умолчанию](media/new-batch-account-create.png)

  ![Снимок экрана: параметры новых учетных записей службы хранилища Azure с параметрами по умолчанию](media/batch-storage-account-create.png)

Развертывание учетных записей в Azure занимает несколько минут. В правом верхнем углу портала появится уведомление о завершении.
  
  ![Снимок экрана с развернутым уведомлением учетных записей Azure](media/batch-accounts-deploy-notification.png)

Теперь учетные записи должны отображаться на панели мониторинга.
  
  ![Снимок экрана панели мониторинга портал Azure, показывающей пакет и учетную запись хранения](media/azure-portal-dashboard.png)

## <a name="set-up-acoustics-bake-ui-with-azure-credentials"></a>Настройка пользовательского интерфейса акустической обработки с использованием учетных данных Azure
Щелкните ссылку на учетную запись пакетной службы на панели мониторинга, а затем на открывшейся странице щелкните ссылку **Ключи** для доступа к своим учетным данным.
  
  ![Снимок экрана: учетная запись пакетной службы Azure со ссылкой на страницу выбранных ключей](media/batch-access-keys.png)

  ![Снимок экрана со страницей ключей учетной записи пакетной службы Azure с ключами доступа](media/batch-keys-info.png)

Щелкните ссылку **Учетная запись хранения** на странице, чтобы получить доступ к учетным данным учетной записи хранения Azure.
  
  ![Снимок экрана: страница ключей учетной записи хранения Azure с ключами доступа](media/storage-keys-info.png)

Введите эти учетные данные в подключаемом модуле [Unity внедрить](unity-baking.md) или в нереальном [внедрить](unreal-baking.md).

## <a name="node-types-and-region-support"></a>Типы узлов и поддержка регионов
Для акустики проекта требуются оптимизированные для вычислений виртуальные машины Azure серии серия fsv2 и H, которые могут не поддерживаться во всех регионах Azure. Просмотрите [эту таблицу](https://azure.microsoft.com/global-infrastructure/services), чтобы выбрать правильное расположение для учетной записи пакетной службы.
![Снимок экрана, показывающий виртуальные машины Azure по регионам](media/azure-regions.png) 

## <a name="upgrading-your-quota"></a>Обновление квоты
Учетные записи пакетной службы Azure подготавливаются с ограничением в 20 вычислительных ядер. Мы можем увеличить это ограничение для более быстрого внедрить времени, так как вы можете распараллелить рабочую нагрузку акустических данных на нескольких узлах, вплоть до количества точек пробы в сцене. Вы можете запросить увеличение квоты, щелкнув ссылку **Квота** на странице портала для пакетной службы Azure, а затем щелкните **Запросить увеличение квоты**:

![Снимок экрана: страница квоты Azure](media/azure-quotas.png)
