# <a name="project-url-preview-use-and-display-requirements"></a>Требования к использованию и отображению для API предварительного просмотра URL-адресов в проекте

Используйте только данные из API предварительного просмотра URL-адресов в проекте для предварительного отображения фрагментов кода и эскизов изображений, связанных гиперссылками с исходными сайтами, при инициированном пользователем предоставлении общего доступа к URL-адресам в социальных сетях, чат-ботах и подобных предложениях. Не копируйте, не сохраняйте и не кэшируйте данные, полученные из API предварительного просмотра URL-адресов в проекте. Необходимо учитывать любые запросы на отключение службы предварительного просмотра, которые вы можете получить от владельцев веб-сайтов или содержимого.
