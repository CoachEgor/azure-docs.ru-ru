# <a name="how-to-add-pre-trained-entities"></a>Добавление предварительно обученных сущностей
В этом руководстве описано, как добавить предварительно обученные сущности в модель Conversation Learner.

## <a name="video"></a>Видео

[![Предварительная версия учебника по предварительно обученным сущностям](https://aka.ms/cl_Tutorial_v3_PreTrainedEntities_Preview)](https://aka.ms/cl_Tutorial_v3_PreTrainedEntities)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот обучения общего назначения.

    npm run tutorial-general

## <a name="details"></a>Сведения

Предварительно обученные сущности распознают общие типы сущностей, например числа, даты, денежные суммы и другие.  Они работают сразу же, не требуют никакого обучения, и их поведение не может быть изменено в отличие от пользовательских сущностей.  По умолчанию предварительно обученные сущности многозначны и содержат все обнаруженные экземпляры сущности.

## <a name="steps"></a>Действия

Процедура начинается на домашней странице веб-интерфейса.

### <a name="create-the-model"></a>Создание модели

1. Выберите **новой модели**.
2. Введите **PretrainedEntities** для **имя**.
3. Нажмите кнопку **Создать**.

### <a name="entity-creation"></a>Создание сущности

1. Выберите **сущностей** на левой панели, затем **новую сущность**.
2. Выберите **Trained/datetimeV2** для **тип сущности**.
3. Проверьте **многозначных** включить сущность накопления один или несколько значений. Обратите внимание, что Pre-Trained сущностей не может быть операция логического отрицания.
4. Нажмите кнопку **Создать**.

![](../media/T08_entity_create.png)

1. Выберите **действия** на левой панели, затем **новое действие**.
2. Введите **дата является $builtin-datetimev2** для **Bot ответа...** .
3. Нажмите кнопку **Создать**.

![](../media/T08_action_create_1.png)

### <a name="create-the-second-action"></a>Создание второго действия

1. Выберите **действия** на левой панели, затем **новое действие**.
2. Введите **что такое Дата?** для **Bot ответа...** . Предварительно обученный сущностей не может быть **необходимые сущности** как они распознаются по умолчанию для всех фразы.
3. Введите **builtin datetimev2** для **дисквалификации сущностей**.
4. Нажмите кнопку **Создать**.

![](../media/T08_action_create_2.png)

### <a name="train-the-model"></a>Обучение модели

1. Выберите **Train диалоговые окна** на левой панели, затем **новое диалоговое окно Train**.
2. Введите **hello** для пользователя utterance чата на левой панели.
3. Выберите **оценка действия**.
4. Выберите **что такое Дата?** из списка действий
5. Введите **сегодня** для пользователя utterance чата на левой панели.
    - **Сегодня** utterance автоматически распознается предварительно обученных моделей в LUIS.
    - При наведении указателя мыши на значения предварительно обученных сущностей отображаются дополнительные данные, предоставляемые LUIS.

![](../media/T08_training.png)
