# <a name="introduction-to-training"></a>Введение в обучение

Мы продемонстрируем основы обучения модели, создание ветвей нового обучения на основе предыдущего обучения и изменение ответов бота.

## <a name="video"></a>Видео

[![Введение в предварительную версию учебника по обучению](https://aka.ms/cl_Tutorial_v3_IntroTraining_Preview)](https://aka.ms/cl_Tutorial_v3_IntroTraining)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот обучения общего назначения.

    npm run tutorial-general

## <a name="details"></a>Сведения

- Действия: Ответ бота на введенные пользователем данные.
- Обучение: способ научить бота реагировать на введенные пользователем данные.
- Создание ветви: изменение вводимых пользователем данных в сохраненном обучающем диалоге для создания нового обучающего диалога, который начинается точно так же, как оригинал, но затем развивается в другом направлении.

## <a name="steps"></a>Действия

### <a name="create-a-new-model"></a>Создание модели

1. В пользовательском веб-интерфейсе щелкните "Новая модель".
2. В качестве имени введите Inspire Bot. Затем щелкните "Create" (Создать).

### <a name="create-an-action"></a>Создание действия

1. На левой панели щелкните "Действия", а затем нажмите кнопку "Создать действие".
2. В поле "Ответ бота" введите Hi! Would you like to be inspired today? (Здравствуйте! Хотите вдохновиться?).
    - Оставьте для остальных полей и флажков параметры по умолчанию.
3. Щелкните Создать.

### <a name="first-training-and-creating-another-action-while-training"></a>Первое обучение и создание другого действия во время обучения

1. На левой панели щелкните "Обучающие диалоги" и нажмите кнопку "Создать обучающий диалог".
2. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст hello (привет). 
    - Эта процедура имитирует реплику пользователя.
3. Щелкните "Оценить действия".
4. Выберите ответ Hi! Would you like to be inspired today? (Здравствуйте! Хотите вдохновиться?).
5. Ответьте за пользователя yes (да).
6. Щелкните "Оценить действия".
7. Нажмите кнопку "+ Действие". 
    - Откроется знакомое диалоговое окно "Создать действие".
8. Введите ответ бота You're awesome! (Вы великолепны!).
9. Щелкните Создать.
10. Обратите внимание, что бот реагирует сразу.
11. Нажмите кнопку "Сохранить".

### <a name="branch-a-second-training-off-of-the-first-training"></a>Создание ветви второго обучения на основе первого
1. Щелкните строку сетки, которая обобщает первое обучение. 
    - Это позволит просматривать и изменять существующее обучение.
2. Щелкните ответ пользователя yes (да). 
    - Откроются элементы управления редактированием.
3. Щелкните значок ветви. 
    - Откроется запрос на ввод пользователем другой фразы для нового диалога.
4. Введите no (нет), нажмите клавишу ВВОД или щелкните кнопку "Создать". 
    - На этом этапе вы получите новый экземпляр обучающего диалога, а оригинал останется без изменений.
5. Щелкните "Оценить действия".
6. Щелкните только что появившийся неверный ответ бота.
7. Нажмите кнопку "+ Действие", 
    - чтобы создать новое действие, с помощью которого бот будет отвечать.
8. Введите ответ бота No problem! Have a great day! (Хорошо! Удачного дня!).
9. Щелкните Создать.
10. Обратите внимание, что бот реагирует сразу.
11. Нажмите кнопку "Сохранить".

### <a name="test-the-trainings"></a>Тестирование обучения
1. На левой панели щелкните "Записываемые диалоги" и нажмите "Создать записываемый диалог".
2. Введите сообщение hi (привет). 
3. Обратите внимание, что бот отреагирует автоматически, как мы его обучили.
4. Введите ответ пользователя yes (да).
5. Обратите внимание на ответ бота — он показывает, что первое обучение работает.
6. Нажмите кнопку "Время ожидания сеанса". Это сообщит Conversation Learner, что мы хотим начать заново, игнорируя существующий ход разговора.
7. Введите сообщение hi (привет). 
8. Обратите внимание, что бот отреагирует автоматически, как мы его обучили.
9. Введите ответ пользователя no (нет).
10. Обратите внимание на ответ бота — он показывает, что второе обучение работает.
11. Нажмите кнопку Done Testing (Тестирование завершено).
