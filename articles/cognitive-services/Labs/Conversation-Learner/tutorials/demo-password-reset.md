# <a name="demo-password-reset"></a>Демонстрация. Сброс паролей
В этом руководстве показана простая программ-бот технической поддержки, которая может помочь ученику в процессе смены паролей на основе Conversation Learner. Модель бота может изучать нестандартные диалоговые потоки информации, сложные последовательности, включая классы за пределами домена. Задачу можно выполнить без кода или дополнительных сущностей.

[![Предварительная версия демонстрации сброса пароля](https://aka.ms/cl_Tutorial_v3_DemoPassword_Preview)](https://aka.ms/cl_Tutorial_v3_DemoPassword)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот сброса пароля.

    npm run demo-password

### <a name="open-the-demo"></a>Запуск демонстрационного примера

В списке моделей пользовательского веб-интерфейса выберите "Tutorial Demo Password Reset" (Демонстрация сброса пароля для работы с руководством). 

### <a name="actions"></a>Действия

Модель содержит набор действий, которые призваны помочь пользователям устранять распространенные проблемы с паролями.

![](../media/tutorial_pw_reset_actions.PNG)

### <a name="training-dialogs"></a>Обучающие диалоги

Модель также содержит несколько обучающих диалогов, включая те, которые демонстрируют обучение за пределами предметной области. Например, для пользователей, которые могут запросить направление движения по маршруту. Пример бота обучен на нескольких из них для демонстрационных целей и просто отвечает, говоря, что не может с этим помочь. Список существующих диалогов находится в разделе "Обучающие диалоги" на левой панели.

![](../media/tutorial_pw_reset_entities.PNG)

1. На левой панели щелкните "Обучающие диалоги" и нажмите кнопку "Создать обучающий диалог".
2. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст "i lost my password" (я забыл пароль).
3. Нажмите кнопку "Оценить действия".
4. Выберите ответ "Is that for your local account or Microsoft account?" (Это пароль для локальной учетной записи или учетной записи Майкрософт?).
5. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст "local account please" (локальная учетная запись).
6. Нажмите кнопку "Оценить действия".
7. Выберите ответ "Which version of Windows do you have?" (Какая у вас версия Windows?).
8. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст "windows xp".
9. Нажмите кнопку "Оценить действия".
10. Нажмите кнопку "+ Действие".
11. В поле "Ответ бота" введите "SOLUTION: how to reset password on Windows XP." (РЕШЕНИЕ. Как сбросить пароль в Windows XP.).
12. Нажмите кнопку "Создать".

### <a name="training-dialogs-for-out-of-domain-scenarios"></a>Обучающие диалоги для сценариев вне предметной области

1. На левой панели щелкните "Обучающие диалоги" и выберите диалог "toy stores".
2. На панели "Чат" нажмите фразу "toy stores" (магазины игрушек).
3. В поле "Добавить альтернативный ввод…" введите тип "web search" (поиск в Интернете) и нажмите ВВОД.
4. В поле "Добавить альтернативный ввод…"введите тип "flight booking" (бронирование авиабилетов) и нажмите ВВОД.
5. Нажмите кнопку "Сохранить изменения".
6. Нажмите кнопку "Сохранить правки".
7. На левой панели щелкните "Записываемые диалоги" и нажмите кнопку "Создать записываемый диалог".
8. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст "i can’t find my password" (я не могу найти пароль).
9. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст "Microsoft account" (учетная запись Майкрософт).
10. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите текст "thanks" (спасибо).
11. Нажмите кнопку "Завершить тестирование".
12. Щелкните диалог "i can't find my password" в представлении сетки.
13. На панели "Чат" щелкните неправильно отображаемый ответ "Solution: How to reset a Microsoft Account Password" (Решение. Как сбросить пароль учетной записи Майкрософт).
14. Нажмите кнопку "+ Действие".
15. В поле "Ответ бота" введите "You are welcome" (Пожалуйста).
16. Нажмите кнопку "Создать".
17. Нажмите кнопку "Сохранить как обучающий диалог".
