# <a name="how-to-use-multi-value-entities-with-a-conversation-learner-model"></a>Использование многозначных сущностей в модели Conversation Learner
В этом руководстве показано, как использовать многозначные свойства для сущностей.

## <a name="video"></a>Видео

[![Предварительная версия учебника по сущности с несколькими значениями](https://aka.ms/cl_Tutorial_v3_MultiValued_Preview)](https://aka.ms/cl_Tutorial_v3_MultiValued)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот обучения общего назначения.

    npm run tutorial-general

## <a name="details"></a>Сведения
Многозначные сущности накапливают значения в список, а не хранят только одно значение.  Эти сущности полезны в тех случаях, когда пользователи могут указать более одного значения. Начинки для пиццы, например.

Сущности, помеченные как многозначные, будут содержать каждый распознанный экземпляр сущности в списке в памяти бота. Последующие распознавания обновляют значение сущности, а не перезаписывают его.

## <a name="steps"></a>Действия

Процедура начинается на домашней странице веб-интерфейса.

### <a name="create-the-model"></a>Создание модели

1. Выберите **новой модели**.
2. Введите **MultiValueEntities** для **имя**.
3. Нажмите кнопку **Создать**.

### <a name="entity-creation"></a>Создание сущности

1. Выберите **сущностей** на левой панели, затем **новую сущность**.
2. Выберите **Custom обучена** для **тип сущности**.
3. Введите **начинки** для **имя сущности**.
4. Проверьте **многозначных** включить сущность накопления один или несколько значений.
5. Проверьте **операция логического отрицания**.
6. Нажмите кнопку **Создать**.

![](../media/T07_entity_create.png)

### <a name="create-the-first-action"></a>Создание первого действия

1. Выберите **действия** на левой панели, затем **новое действие**.
2. Введите **Вот вашей начинки: $toppings** для **Bot ответа...** . Знак доллара в начале указывает ссылку на сущность.
3. Нажмите кнопку **Создать**.

![](../media/T07_action_create_1.png)

### <a name="create-the-second-action"></a>Создание второго действия

1. Выберите **действия** на левой панели, затем **новое действие**.
2. Введите **какие начинки вы хотите?** для **Bot ответа...** .
3. Введите **начинки** для **дисквалификации дает право**.
4. Нажмите кнопку **Создать**.

Теперь у вас есть два действия.

![](../media/T07_action_create_2.png)

### <a name="train-the-model"></a>Обучение модели

1. Выберите **Train диалоговые окна** на левой панели, затем **новое диалоговое окно Train**.
2. Введите **Здравствуйте** для пользователя utterance чата на левой панели.
3. Выберите **оценка действия**.
4. Выберите **какие начинки вы хотите?** из списка действий. Процентиль — 100% как единственное допустимое действие на основе ограничений.
5. Введите **сыр и грибы** для пользователя utterance чата на левой панели.
6. Выделите **сыр** выберите **+ начинки**.
7. Выделите **грибы** выберите **+ начинки**.
8. Выберите **оценка действия**.
9. Выберите **Вот вашей начинки: $toppings** из списка действий.
10. Введите **добавить перец** для пользователя Далее utterance чата на левой панели.
11. Выделите **Перец** выберите **+ начинки**.
12. Выберите **оценка действия**.
13. Выберите **Вот вашей начинки: $toppings** из списка действий.
14. Введите **удалить сыр** для третьего utterance пользователя на панели слева чата.
15. Выделите **сыр** выберите **-начинки**.
16. Выберите **оценка действия**.
17. Выберите **Вот вашей начинки: $toppings** из списка действий.

![](../media/T07_training.png)
