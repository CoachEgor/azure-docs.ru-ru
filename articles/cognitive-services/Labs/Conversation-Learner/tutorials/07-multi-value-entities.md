# <a name="how-to-use-multi-value-entities-with-a-conversation-learner-model"></a>Использование многозначных сущностей в модели Conversation Learner
В этом руководстве показано, как использовать многозначные свойства для сущностей.

## <a name="video"></a>Видео

[![Учебник по многозначным сущностям Предварительная версия](https://aka.ms/cl_Tutorial_v3_MultiValued_Preview)](https://aka.ms/cl_Tutorial_v3_MultiValued)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот обучения общего назначения.

    npm run tutorial-general

## <a name="details"></a>Подробнее
Многозначные сущности накапливают значения в список, а не хранят только одно значение.  Эти сущности полезны в тех случаях, когда пользователи могут указать более одного значения. Начинки для пиццы, например.

Сущности, помеченные как многозначные, будут содержать каждый распознанный экземпляр сущности в списке в памяти бота. Последующие распознавания обновляют значение сущности, а не перезаписывают его.

## <a name="steps"></a>Шаги

Процедура начинается на домашней странице веб-интерфейса.

### <a name="create-the-model"></a>Создание модели

1. Выберите **создать модель**.
2. Введите **мултивалуинтитиес** в качестве **имени**.
3. Нажмите кнопку **Создать**.

### <a name="entity-creation"></a>Создание сущности

1. На левой панели выберите **сущности** , а затем — **Новая сущность**.
2. Выберите **настраиваемое обучение** для **типа сущности**.
3. Введите **«** наименование» в качестве **имени сущности**.
4. Установите флажок с **несколькими** значениями, чтобы включить функцию накопления одного или нескольких значений.
5. Проверка **отрицания**.
6. Нажмите кнопку **Создать**.

![](../media/T07_entity_create.png)

### <a name="create-the-first-action"></a>Создание первого действия

1. Выберите **действия** на левой панели, а затем — **новое действие**.
2. Введите **: $toppings** для **ответа Bot...** . Начальный знак доллара указывает на ссылку на сущность.
3. Нажмите кнопку **Создать**.

![](../media/T07_action_create_1.png)

### <a name="create-the-second-action"></a>Создание второго действия

1. Выберите **действия** на левой панели, а затем — **новое действие**.
2. Укажите **, что вы хотите?** для **ответа Bot...**
3. Введите **«** находящие», чтобы не подсчитать права.
4. Нажмите кнопку **Создать**.

Теперь у вас есть два действия.

![](../media/T07_action_create_2.png)

### <a name="train-the-model"></a>Обучение модели

1. Выберите пункт **обучение диалоговых окон** на левой панели, а затем **новое диалоговое окно "обучение**".
2. Введите **Hi** для utterance пользователя на левой панели разговора.
3. Выберите **действия оценки**.
4. Выберите из списка действий, **какие из них вам хотелось бы?** Значение процентиля равно 100% в качестве единственного допустимого действия на основе ограничений.
5. Введите **Мак и грибы** для utterance пользователя на левой панели разговора.
6. Выделите **Мак** , а затем выберите **+** пересветки.
7. Выделите **грибы** , а затем выберите **+** пересветки.
8. Выберите **действия оценки**.
9. Выберите **: $toppings** из списка действий.
10. Введите **добавить перца** для следующего utterance пользователя на левой панели чата.
11. Выделите **перца** , а затем выберите **+** пересветки.
12. Выберите **действия оценки**.
13. Выберите **: $toppings** из списка действий.
14. Введите **Удалить Мак** для третьего utterance пользователя на левой панели разговора.
15. Выделите **Мак** **, а**затем выберите «на».
16. Выберите **действия оценки**.
17. Выберите **: $toppings** из списка действий.

![](../media/T07_training.png)
