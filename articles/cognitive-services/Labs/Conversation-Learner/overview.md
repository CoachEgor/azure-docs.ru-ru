# <a name="what-is-conversation-learner"></a>Что такое Conversation Learner?

Conversation Learner позволяет создавать и обучать разговорные интерфейсы, которые учатся на примере взаимодействий. 

В отличие от традиционных подходов, Conversation Learner учитывает весь контекст диалога, чтобы улучшить ответы и сделать работу пользователей более удобной. Охватывая широкий набор ориентированных на задачи вариантов использования, Conversation Learner применяет алгоритмы машинного обучения, чтобы боты и интеллектуальные агенты меньше раздражали пользователей, не вызывали роста расходов на службу поддержки и были более понятными.

Разработчики могут начать с введения прототипов диалогов, которые нужно имитировать. Эффективное обучение модели зависит от числа диалогов. Когда модель начнет работать достаточно хорошо, бот можно будет развернуть для пользователей. Conversation Learner записывает диалоги с пользователями, и разработчик может просмотреть их. При обнаружении ошибок разработчик может тут же внести исправление, после чего модель переобучается и немедленно становится доступной для использования.

Этот подход уменьшает потребность в ручном программировании логики управления диалогом и позволяет владельцам организаций или экспертам в предметной области участвовать в разработке разговорного интерфейса, не обладая знаниями о машинном обучении. Будучи развернутым в боте, на интеллектуальном устройстве или интеллектуальном агенте, Conversation Learner может с помощью итеративного подхода быстро обучаться новым навыкам, поведению или компетенциям и повышать качество работы. 

Conversation Learner дает разработчикам возможность ускорить выход на рынок и использовать успешные диалоги в разных каналах общения посредством Microsoft Bot Framework или самостоятельно, с помощью собственно инфраструктуры.

Сводные сведения и особенности.

- Conversation Learner — передовое решение для разработки ориентированных на задачи ботов на основе ИИ.

- В нем используется комплексная нейронная сеть с обратными связями (LSTM), которая обеспечивает обучение непосредственно на примерах общения со множеством реплик. 

- С помощью этого решения конструкторы, разработчики, бизнес-пользователи и сотрудники центра обработки вызовов могут разрабатывать и обслуживать боты. 

- Предоставляет возможность выражать бизнес-правила и здравый смысл в коде.

- Во время сеансов обучения используются модель нейронной сети для оценки следующего набора предполагаемых действий в диалоге. Затем разработчик бота может выбрать необходимое действие и обучить сеть правильному реагированию.
 
- После завершения обучения разработчик может использовать записанные диалоги пользователей, чтобы внести исправления в ответы бота и переобучить модель. 

- Для выполнения задач может вызывать интерфейсы API различных предметных областей и сторонних разработчиков.
