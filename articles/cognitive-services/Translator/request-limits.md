# <a name="request-limits-for-translator-text"></a>Ограничения запросов API перевода текстов

В этой статье перечислены ограничения регулирования для API перевода текстов. Услуги включают перевод, транслитерацию, определение длины предложения, определение языка и альтернативные переводы.

## <a name="character-and-array-limits-per-request"></a>Символ и массив ограничений на запрос

Каждый запрос Translate ограничена 5000 символов. Плата взимается на основе количества знаков, а не запросов. Рекомендуется отправлять более короткие запросы.

Следующие таблицы списки массив элементов и символ ограничения для каждой операции API перевода текстов.

| Операция | Максимальный размер элемента массива |   Максимальное число элементов массива |  Максимальный размер запроса (символов) |
|:----|:----|:----|:----|
| Перевод | 5 000 | 100   | 5 000 |
| Транслитерация | 5 000 | 10    | 5 000 |
| Определение | 10 000 | 100 |   50 000 |
| BreakSentence | 10 000    | 100 | 5,0000 |
| Поиск по словарю| 100 |  10  | 1000 |
| Примеры словарей | 100 для текста и 100 для перевода (200 общее)| 10|   2 000 |

## <a name="character-limits-per-hour"></a>Ограничение по количеству знаков за час

Ограничение по количеству знаков за час определяется уровнем подписки API перевода текстов. Почасовая Квота должны будут потребляться равномерно на протяжении этого часа. Если вы или обойти эти ограничения либо отправьте слишком большой части квоты за короткий период времени, скорее всего появится готовую квоты ответа.

| Уровень | Допустимое число знаков |
|------|-----------------|
| F0 | 2 миллиона знаков в час |
| S1 | 40 миллионов знаков в час |
| S2 / C2 | 40 миллионов знаков в час |
| S3 / C3 | 120 миллионов знаков в час |
| S4 / C4 | 200 миллионов знаков в час |

Ограничения для [нескольких служб подписки](https://docs.microsoft.com/azure/cognitive-services/translator/reference/v3-0-reference#authentication) совпадают с уровня S1.

Эти ограничения, ограничены модели перевода с использованием стандартных корпорации Майкрософт. Пользовательские преобразования моделей, использующих пользовательский преобразователь ограничены 1800 символ в секунду.

## <a name="latency"></a>Задержка

API перевода текстов имеет максимальную задержку 15 секунд, используя стандартные модели. Преобразование, с помощью пользовательских моделей имеет максимальную задержку, равную 25 секунд. За это время вы получите результат или ответ об истечении времени ожидания. Как правило, ответы возвращаются в таком диапазоне времени: 150–300 миллисекунд. Время отклика зависит от размера пару из запроса и языка. Если вы не получите перевод или [сообщение об ошибке](https://docs.microsoft.com/azure/cognitive-services/translator/reference/v3-0-reference#errors) в течение этого времени, следует Проверьте сетевое подключение и повторите попытку.

## <a name="sentence-length-limits"></a>Допустимая длина предложения

При использовании функции [BreakSentence](https://docs.microsoft.com/azure/cognitive-services/translator/reference/v3-0-break-sentence) допустимая длина предложения составляет 275 знаков. Исключения применимы для следующих языков:

| Язык | Код | Допустимое число знаков |
|----------|------|-----------------|
| Китайский | zh | 132 |
| Немецкий | de | 290 |
| Итальянский | it | 280 |
| Японский | ja | 150 |
| Португальский | pt | 290 |
| Испанский | es | 280 |
| Итальянский | it | 280 |
| Тайский | th | 258 |

> [!NOTE]
> Это ограничение не применяется к переводам.
