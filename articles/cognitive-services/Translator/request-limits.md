# <a name="request-limits-for-translator-text"></a>Ограничения запросов API перевода текстов

В этой статье перечислены ограничения регулирования для API перевода текстов. Услуги включают перевод, транслитерацию, определение длины предложения, определение языка и альтернативные переводы.

## <a name="character-and-array-limits-per-request"></a>Ограничения характера и массива на запрос

Каждый запрос на перевод ограничен 5000 символов, на всех целевых языках, на которых вы переводите. Например, отправка запроса на перевод 1500 символов для перевода на 3 разных языках приводит к размеру запроса 1500x3 и 4500 символов, что удовлетворяет лимитзапросу. Плата взимается на основе количества знаков, а не запросов. Рекомендуется отправлять более короткие запросы.

В следующей таблице перечислены элемент массива и ограничения символов для каждой операции API текста переводчика.

| Операция | Максимальный размер элемента array |   Максимальное количество элементов массива |  Максимальный размер запроса (символы) |
|:----|:----|:----|:----|
| Перевод | 5 000 | 100   | 5 000 |
| Транслитерация | 5 000 | 10    | 5 000 |
| Определение | 10 000 | 100 |   50 000 |
| BreakSentence | 10 000    | 100 | 50 000 |
| Поиск по словарю| 100 |  10  | 1000 |
| Примеры словарей | 100 для текста и 100 для перевода (200 всего)| 10|   2 000 |

## <a name="character-limits-per-hour"></a>Ограничение по количеству знаков за час

Ограничение по количеству знаков за час определяется уровнем подписки API перевода текстов. 

Почасовая квота должна потребляться равномерно в течение часа. Например, на пределе уровня F0 в 2 миллиона символов в час символы должны потребляться не быстрее, чем примерно 33 300 символов в минуту раздвижного окна (2 миллиона символов, разделенных на 60 минут).

Если вы достигнете или превысите эти пределы, или отправите слишком большую часть квоты в течение короткого периода времени, вы, вероятно, получите ответ из квоты. Для одновременных запросов ограничений нет.

| Уровень | Допустимое число знаков |
|------|-----------------|
| F0 | 2 миллиона знаков в час |
| S1 | 40 миллионов знаков в час |
| S2 / C2 | 40 миллионов знаков в час |
| S3 / C3 | 120 миллионов знаков в час |
| S4 / C4 | 200 миллионов знаков в час |

Ограничения для [многофункциональных подписок](https://docs.microsoft.com/azure/cognitive-services/translator/reference/v3-0-reference#authentication) такие же, как уровень S1.

Эти ограничения ограничены стандартными моделями перевода корпорации Майкрософт. Пользовательские модели перевода, которые используют пользовательский переводчик, ограничены 1800 символами в секунду.

## <a name="latency"></a>Задержка

API Translator Text API имеет максимальную задержку в 15 секунд с использованием стандартных моделей и 120 секунд при использовании пользовательских моделей. Как правило, ответы *на текст в пределах 100 символов* возвращаются за 150 миллисекунд до 300 миллисекунд. Пользовательские модели переводчика имеют схожие характеристики задержки по устойчивому тарифу запроса и могут иметь более высокую задержку, когда частота запроса прерывистая. Время отклика будет варьироваться в зависимости от размера пары запроса и языка. Если вы не получили перевод или [ответ на ошибку](https://docs.microsoft.com/azure/cognitive-services/translator/reference/v3-0-reference#errors) в течение этого срока, пожалуйста, проверьте свой код, подключение к сети и повторную попытку. 

## <a name="sentence-length-limits"></a>Допустимая длина предложения

При использовании функции [BreakSentence](https://docs.microsoft.com/azure/cognitive-services/translator/reference/v3-0-break-sentence) допустимая длина предложения составляет 275 знаков. Исключения применимы для следующих языков:

| Язык | Код | Допустимое число знаков |
|----------|------|-----------------|
| Китайский | zh | 132 |
| Немецкий | de | 290 |
| Итальянский | it | 280 |
| Японский | ja | 150 |
| Португальский | pt | 290 |
| Испанский | es | 280 |
| Итальянский | it | 280 |
| Тайский | th | 258 |

> [!NOTE]
> Это ограничение не применяется к переводам.
