## <a name="bounding-boxes-and-hot-spots"></a>Ограничивающие прямоугольники и гиперобъекты

Теги не по умолчанию включают в себя ограничивающий прямоугольник, определяющий интересующую вас область изображения, к которой применяется тег. Если ограничивающий прямоугольник не идентифицирует изображение целиком, то создайте на изображении гиперобъект, используя ограничивающий прямоугольник. Пользователь может щелкнуть такой гиперобъект, чтобы просмотреть сведения, связанные с содержимым, найденным в зоне этого гиперобъекта (или прямоугольника). Например если используется образ высокой манере, результаты может содержать теги (и ограничивающие прямоугольники) на аксессуары, показано на рисунке, например парковке, ювелирный, scarfs и т. д. В следующем примере показано прямоугольник активной зоны для очков, показано на рисунке:

![Ограничивающий прямоугольник и гиперобъект](./media/click-to-search.PNG)
