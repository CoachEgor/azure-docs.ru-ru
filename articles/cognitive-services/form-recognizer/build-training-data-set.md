# <a name="build-a-training-data-set-for-a-custom-model"></a>Построение обучающий набор данных для пользовательской модели

При использовании пользовательской модели распознаватель формы, поэтому модель можно обучить в конкретной отрасли формы предоставляют обучающих данных. Вы можете обучить модель с пять заполненные формы или как пустая форма (включают слово «empty» в имени файла) и две формы заполнены по умолчанию. Даже если у вас есть достаточно заполненные формы для обучения на основе, Добавление пустой формы для набора данных для обучения может улучшить точность модели.

## <a name="training-data-tips"></a>Полезные советы данных

Очень важно использовать набор данных, которая оптимизирована для обучения. Используйте приведенные ниже советы, чтобы получить наилучший результат из [Обучение модели](https://westus2.dev.cognitive.microsoft.com/docs/services/form-recognizer-api/operations/TrainCustomModel) операции:

* По возможности используйте текстовых документов PDF, а не документов на основе образа. Отсканированные документы PDF организованы в виде изображения.
* Используйте одну пустую форму и две формы заполнены по умолчанию, если они доступны.
* Для заполнения форм используйте примеры, которые имеют все свои поля заполнены.
* Используйте формы с разными значениями в каждом поле.
* Если форма образы имеют более низкое, используйте большего набора данных (например, изображений 10 – 15).
