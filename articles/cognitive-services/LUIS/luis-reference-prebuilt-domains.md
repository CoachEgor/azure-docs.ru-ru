# <a name="prebuilt-domain-reference-for-your-luis-app"></a>Справочник по предварительно созданным доменам для приложения LUIS
В этом справочнике представлены сведения о [предварительно созданных доменах](luis-how-to-use-prebuilt-domains.md), которые представляют собой предварительно созданные коллекции намерений и сущностей, предлагаемые приложением LUIS.

[Пользовательские домены](luis-how-to-start-new-app.md), напротив, сначала не имеют намерений и моделей. Вы можете добавить любые намерения и сущности предварительно созданных доменов в пользовательскую модель.

# <a name="supported-domains-across-cultures"></a>Поддерживаемые домены различных региональных параметров

Единственный поддерживаемый язык и региональные параметры на английском языке.

|Тип сущности|description|
|--|--|
|Календарь|Календарь является ничего о личных встреч и встречи, _не_ открытые события (например world cup расписания, Сиэтл события календарей или универсального календарей (например, какой день ее уже сегодня, что осенью начинается, при Labor Day).|
|Обмен данными|Запросы для выполнения вызовов, отправки текстовых сообщений или мгновенных сообщений, поиск и добавление контактов и различные другие связи запросы (обычно исходящие). Имя контактного лица только запросы не принадлежат к домену связи.|
|Email|Электронная почта — поддомен домена обмен данными. Главным образом в нем запросов для отправки и получения сообщений через сообщения электронной почты.|
|HomeAutomation|Домен HomeAutomation предоставляет намерения и сущности, связанные с Управление смарт-домашних устройств. Он главным образом поддерживает команды управления, относящиеся к источников света и кондиционер, но имеет некоторые возможности обобщение для других электрического устройств.|
|Примечания|Примечание домен предоставляет намерения и сущности для создания заметок и записывать элементы для пользователей.|
|Места|Окружение включают предприятий, учреждения, рестораны, общедоступным и адреса. Домен поддерживает поиск и запросом об информации общественных местах, таких как местоположение, операционная часов и расстояния.|
|RestaurantReservation|Ресторан резервирования домен поддерживает способы для обработки резервирования для ресторанов.|
|ToDo|ToDo домена предоставляет типы списков задач для пользователей, для добавления, пометить и удалить их элементы списка поручений.|
|ToDo_IPA|Исходя из домена ToDo, ToDo_IPA — настраиваемой версии для развертывания сущностей в ToDo. Версия IPA предоставляет намерения и сущности, для поддержки распознавания типа списка todo. Модель определяет три вида списки дел: списком покупок, списка покупок и списка дел, охватывающий других случаях.|
|Техническое оборудование|Служебные программы является доменом общие среди всех LUIS предварительно созданные модели, содержащих общие намерения и фразы в сценариях различие.|
|Weather|Домен погоды посвящена проверки погодных условий и рекомендации с расположением и времени или времени при проверке погодные условия.|
|Интернет|Домен Web предоставляет намерения и сущности для поиска веб-сайта.|
