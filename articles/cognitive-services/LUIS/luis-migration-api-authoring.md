# <a name="authoring-api-changes"></a>Изменения API создания 

API-интерфейсы разработки обновляются для версии 3, чтобы разрешить [декомпозицию модели с помощью сущностей, занятых компьютером](luis-concept-data-extraction.md). 

Вы можете создать в API v2 или Preview v3. Это же приложение также может использовать ответы на конечную точку прогнозирования v2 или v3. Нет необходимости иметь одинаковую версию API для создания и прогнозирования ответа конечной точки.

Портал LUIS использует только API-интерфейсы разработки версии 3.

## <a name="next-steps"></a>Дальнейшие действия

Проверьте [изменения конечной точки прогнозирования](luis-migration-api-v3.md) для API V3.
