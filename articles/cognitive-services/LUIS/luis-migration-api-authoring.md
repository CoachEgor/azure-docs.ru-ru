# <a name="authoring-api-changes"></a>Авторирование изменений API 

Авторство AIS обновляется, для V3, чтобы обеспечить [разложение модели с машино-изученных объектов.](luis-concept-data-extraction.md) 

Вы можете автор в V2 или предварительный просмотр V3 APIS. То же приложение может также выбрать для использования V2 или V3 прогнозирования конечных точек ответов. Вам не нужно иметь ту же версию API для авторизации и прогнозирования конечной точки ответа.

Портал LUIS использует только AA, авторство AIS V3.
