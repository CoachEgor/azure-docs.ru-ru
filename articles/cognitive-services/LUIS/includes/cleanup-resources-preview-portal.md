Удалите приложение LUIS, если оно больше не нужно. Для этого выберите **Мои приложения** в меню слева вверху. Щелкните имя приложения в списке приложений и выберите **Удалить** из контекстного меню. Во всплывающем диалоговом окне **Delete app** (Удаление приложения) щелкните **ОК**.
