1. Войдите в предварительную версию портала LUIS по URL-адресу [https://preview.luis.ai](https://preview.luis.ai). 

1. Выберите **Создать приложение**.  

    [![Снимок экрана страницы "My Apps" (Мои приложения) Распознавания речи (LUIS)](../media/create-app-in-portal.png "Снимок экрана страницы "My Apps" (Мои приложения) Распознавания речи (LUIS)")](../media/create-app-in-portal.png#lightbox)

1. Во всплывающем диалоговом окне введите имя `Pizza tutorial` и сохраните язык и региональные параметры по умолчанию (**на английском языке**). Другие поля являются необязательными, не задавайте их значения. Нажмите кнопку **Готово**. 

    ![Введите имя нового приложения LUIS.](../media/create-pizza-tutorial-app-in-portal.png)
