Доступ к ключам и конечным точкам предоставляется на портале LUIS в разделе "Управление" на странице ресурсов Azure. 
 
1. Войдите на портал LUIS. 
1. Выберите приложение из списка приложений.
1. В разделе **Управление** выберите **Ресурсы Azure**. 
1. Скопируйте значения для ключа и конечной точки для ресурса начала работы.

    Ключ для **начала работы** позволяет бесплатно получить запросы к API для разработки и до 1000 запросов к API конечной точки прогнозирования в месяц для всех приложений LUIS.
