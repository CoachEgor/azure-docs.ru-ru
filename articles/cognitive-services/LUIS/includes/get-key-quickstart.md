Ключи и конечные точки предоставляются на портале LUIS в разделе **Управление** на странице **ресурсов Azure**. 
 
1. Войдите на портал [LUIS](https://preview.luis.ai/). 
1. Создайте приложение или выберите имеющееся приложение из списка.
1. В разделе **Управление** выберите **Ресурсы Azure**. 
1. Скопируйте значения для ключа и конечной точки для ресурса начала работы. Они понадобятся для выполнения примера ниже.
   > [!NOTE]
   > Ключ для **начала работы** позволяет бесплатно получить запросы к API для разработки и до 1000 запросов к API конечной точки прогнозирования в месяц для всех приложений LUIS.
