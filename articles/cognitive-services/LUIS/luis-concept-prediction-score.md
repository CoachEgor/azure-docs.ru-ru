# <a name="prediction-scores-indicate-prediction-accuracy-for-intent-and-entities"></a>Оценки прогнозирования отражают точность прогнозирования намерения и сущностей

Оценка прогноза указывает степень достоверности LUIS для результатов прогнозирования высказывания пользователя.

Оценка прогнозирования находится в диапазоне от нуля (0) до единицы (1). Пример оценки прогнозирования LUIS с высокой степенью достоверности — 0,99. Пример оценки прогнозирования LUIS с низкой достоверностью — 0,01. 

|Значение оценки|Достоверность|
|--|--|
|1|Точное соответствие|
|0.99|Высокая степень достоверности|
|0,01|Низкая степень достоверности|
|0|Точное несоответствие|

## <a name="top-scoring-intent"></a>Намерение с максимальной оценкой

Для каждого прогнозирования высказывание возвращает намерение с максимальной оценкой. Этот прогноз представляет собой численное сравнение показателей предсказаний. 

## <a name="proximity-of-scores-to-each-other"></a>Близость партитур друг к другу

Лучшие 2 балла могут иметь очень небольшую разницу между ними. LUIS не указывает на эту близость, кроме возвращения наивысшего балла.  

## <a name="return-prediction-score-for-all-intents"></a>Возврат оценки прогнозирования для всех намерений

Результат для теста или конечной точки может содержать все намерения. Эта конфигурация устанавливается на конечную точку с использованием правильной пары имен/значений запроса.

|API прогнозирования|Имя запроса|
|--|--|
|V3|`show-all-intents=true`|
|V2|`verbose=true`|

## <a name="review-intents-with-similar-scores"></a>Просмотр намерений с похожими оценками

Просмотр оценки для всех намерений является хорошим способом проверить, что не только правильное намерение определены, но что следующий выявленных намерения оценка значительно и последовательно ниже для высказываний.

Если у нескольких намерений есть близкие оценки прогнозирования, то на основе контекста высказывания LUIS может переключаться между намерениями. Чтобы исправить эту ситуацию, продолжайте добавлять высказывания к каждому намерению с более широким разнообразием контекстуальных различий, или вы можете иметь клиентское приложение, например, чат-бот, сделать программный выбор о том, как справиться с 2 верхними намерениями.

2 намерения, которые слишком тесно забил, может инвертировать **из-за недетерминированной подготовки**. Первая оценка может стать второй, а вторая может стать первой. Для того, чтобы предотвратить эту ситуацию, добавьте пример высказывания к каждому из двух верхних намерений для этого высказывания с выбором слова и контекстом, который отличает 2 намерения. Для двух намерений нужно добавить приблизительно одинаковое число примеров фрагментов речи. Общее правило для разделения для предотвращения изменения порядка намерений из-за обучение заключается в том, что разница в оценках должна составлять 15 %.

Вы можете отключить **недетерминированное обучение,** [тренируясь со всеми данными.](luis-how-to-train.md#train-with-all-data)

## <a name="differences-with-predictions-between-different-training-sessions"></a>Различия с прогнозами между различными учебными занятиями

Когда вы тренируете ту же модель в другом приложении, и оценки не то же самое, эта разница в том, что есть **недетерминированной подготовки** (элемент случайности). Во-вторых, любое перекрытие высказывания более чем на одно намерение означает, что намерение с максимальной оценкой для того же высказывания может измениться в зависимости от обучения.

Если ваш чат-бот требует определенного балла LUIS, чтобы указать на уверенность в намерениях, вы должны использовать разницу в оценках между двумя верхними намерениями. Такая ситуация обеспечивает гибкость для вариаций в обучении.

Вы можете отключить **недетерминированное обучение,** [тренируясь со всеми данными.](luis-how-to-train.md#train-with-all-data)

## <a name="e-exponent-notation"></a>Запись E (экспоненциальная запись)

Оценки прогнозирования можно записывать в форме экспоненциальной записи, _над_ диапазоном 0–1, например `9.910309E-07`. Эта оценка свидетельствует об очень **маленьком** числе.

|Экспоненциальная запись оценки |Фактическая оценка|
|--|--|
|9.910309E-07|0,0000009910309|
