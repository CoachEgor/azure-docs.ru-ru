# <a name="use-phrase-lists-to-boost-signal-of-word-list"></a>Используйте списки фраз для повышения эффективности работы списка слов.

В приложение LUIS можно добавить функции для повышения его точности. Признаки подсказывают приложению LUIS, что некоторые слова и фразы относятся к словарю предметной области приложения. 

В [список фраз](luis-concept-feature.md) входит группа значений (слов или фраз), принадлежащих одному классу и обрабатываемых аналогичным образом (например, названия городов или продуктов). Характеристики одного значения, полученные LUIS, автоматически применяются к остальным значениям. Этот список не то же самое, что и [сущность списка](reference-entity-list.md) (точное совпадение текста) сопоставленных слов.

Список фраз добавляется в словарь домена приложений в качестве второго сигнала LUIS об этих словах.

## <a name="add-phrase-list"></a>Добавление списка фраз

LUIS позволяет добавлять до 10 списков фраз в одно приложение. 

1. Откройте приложение, щелкнув его имя на странице **Мои приложения**, щелкните **Сборка**, а затем на левой панели приложения щелкните **Phrase lists** (Списки фраз). 

1. На странице **Списки фраз** щелкните **Create new phrase list** (Создать список фраз). 
 
1. В диалоговом окне **Добавление списка фраз** введите `Cities` в поле имя списка фраз. В поле **Значение** введите значения списка фраз. Можно ввести только одно значение или набор значений, разделенные запятыми и нажать **ВВОД**.

    ![Добавление списка фраз "Города"](./media/luis-add-features/add-phrase-list-cities.png)

1. Интеллектуальная служба распознавания речи может предложить связанные значения для добавления в список фраз. Щелкните **Рекомендовать**, чтобы получить группу предложенных значений, которые семантически связаны с добавленными значениями. Щелкните любое предложенное значение или нажмите **Add All** (Добавить все), чтобы добавить все.

    ![Добавление всех предложенных значений списка фраз](./media/luis-add-features/related-values.png)

1. Щелкните **These values are interchangeable** (Эти значения являются взаимозаменяемыми), если добавленные значения списка фраз являются альтернативными, которые можно использовать как взаимозаменяемые.

    ![Назначение предложенных значений списка фраз взаимозаменяемыми](./media/luis-add-features/interchangeable.png)

1. Нажмите кнопку **Done**(Готово). Список фраз "Города" добавляется на страницу **Список фраз**.
