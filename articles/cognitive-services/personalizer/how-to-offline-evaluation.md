# <a name="analyze-your-learning-loop-with-an-offline-evaluation"></a>Проанализируйте цикл обучения с помощью оценки в автономном режиме

Сведения о том, как завершить автономную оценку и истолковать результаты.

Оффлайн-оценки позволяют измерить эффективность Personalizer по сравнению с поведением приложения по умолчанию, узнать, какие функции вносят наибольший вклад в персонализацию, и автоматически открывать новые значения машинного обучения.

Дополнительные сведения см. в разделе [Автономные оценки](concepts-offline-evaluation.md).

## <a name="prerequisites"></a>Предварительные требования

* Настроенная петля Personalizer
* Цикл Personalizer должен иметь репрезентативный объем данных - в качестве стадиона мы рекомендуем по крайней мере 50 000 событий в своих журналах для значимых результатов оценки. При необходимости ранее экспортированные файлы _политики обучения_ можно сравнить и протестировать в одном и том же процессе оценивания.

## <a name="run-an-offline-evaluation"></a>Выполнение автономной оценки

1. На [портале Azure](https://azure.microsoft.com/free/)найдите ресурс Personalizer.
1. На портале Azure перейдите в раздел **Оценки** и выберите **«Создать оценку».**
    ![На портале Azure перейдите в раздел «Оценки» и выберите «Оценка».](./media/offline-evaluation/create-new-offline-evaluation.png)
1. Задайте следующие значения.

    * Имя оценки.
    * Дата начала и окончания - это даты, которые определяют диапазон данных для использования в оценке. Эти данные должны присутствовать в журналах, как указано в значении [хранения данных.](how-to-settings.md)
    * Оптимизация Discovery установить **да**.

    > [!div class="mx-imgBorder"]
    > ![Выберите настройки оценки в автономном режиме](./media/offline-evaluation/create-an-evaluation-form.png)

1. Начните оценку, выбрав **Ok**.

## <a name="review-the-evaluation-results"></a>Обзор результатов оценки

Выполнение оценки может занять много времени в зависимости от объема обрабатываемых данных, количества сравниваемых политик обучения и того, было ли запрошено оптимизацию.

После завершения, вы можете выбрать оценку из списка оценок, а затем выберите **Сравнить оценка вашего приложения с другими потенциальными настройками обучения.** Выберите эту функцию, когда вы хотите увидеть, как работает текущая политика обучения по сравнению с новой политикой.

1. Просмотрите эффективность [политики обучения.](concepts-offline-evaluation.md#discovering-the-optimized-learning-policy)

    > [!div class="mx-imgBorder"]
    > [![Обзор результатов оценки](./media/offline-evaluation/evaluation-results.png)](./media/offline-evaluation/evaluation-results.png#lightbox)

1. Выберите **применить** политику, которая улучшает модель лучше для ваших данных.
