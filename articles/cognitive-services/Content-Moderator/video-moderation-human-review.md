# <a name="video-moderation-with-human-review"></a>Модерация видео с помощью пользовательских проверок

Используйте машинную [модерацию видео](video-moderation-api.md) и [инструмент пользовательской проверки](Review-Tool-User-Guide/human-in-the-loop.md) Content Moderator для модерации видео и расшифровок для выявления содержимого для взрослых (откровенного содержимого) и непристойного содержимого, чтобы получить наилучшие результаты для своего бизнеса.

## <a name="video-trained-classifier-preview"></a>Классификатор, обученный с помощью видео, (предварительная версия)

Машинная классификации видео реализуется с помощью обученных на изображениях или видео моделей. В отличие от обученных на изображениях классификаторов видео, классификатор видео для взрослых и непристойного видео, предлагаемый корпорацией Майкрософт, обучен на видео. Этот метод обеспечивает более высокое качество совпадения.

## <a name="shot-detection"></a>Обнаружение сюжетов

При выводе данных классификации дополнительные средства аналитики видео обеспечивают большую гибкость в анализе видео. Вместо вывода только кадров служба модерации видео Майкрософт также предоставляет сведения уровня сюжета. Теперь у вас есть возможность анализировать видео на уровне сюжета и кадра.
 
## <a name="key-frame-detection"></a>Обнаружение опорных кадров

Вместо вывода кадров с регулярными интервалами служба модерации видео идентифицирует и выводит только потенциально полные (эффективные) кадры. Это позволяет создавать эффективные кадры для анализа на уровне кадра на наличие содержимого для взрослых и непристойного содержимого.

В следующем извлеченном фрагменте показан неполный ответ с потенциальными сюжетами, опорными кадрами оценками зрелости и неприличности.

## <a name="visualization-for-human-reviews"></a>Визуализация для пользовательских проверок

Для получения более точных сведений организациям нужно решение для пользовательской проверки, обеспечивающее преобразование для просмотра видео, кадров и тегов машинной модерации. Модераторы-пользователи, проверяющие видео и кадры, получают полное представление об аналитических сведениях, изменяют теги и отправляют принятые ими решения.

![Представление по умолчанию инструмента проверки видео](images/video-review-default-view.png)

## <a name="player-view-for-video-level-review"></a>Представление проигрывателя для проверки на уровне видео

Принятие логических решений на уровне видео возможно благодаря представлению видеопроигрывателя, в котором отображаются потенциальные кадры для взрослых и непристойные кадры. Рецензенты-пользователи просматривают видео с разной скоростью для изучения сцен. Они подтверждают свои решения, переключая теги.

![Представление проигрывателя инструмента проверки видео](images/video-review-player-view.PNG)

## <a name="frames-view-for-detailed-reviews"></a>Представление кадров для тщательной проверки

Тщательная проверка видео при покадровом анализе возможна благодаря представлению кадров. Рецензенты-пользователи просматривают и выбирают один или несколько кадров, переключая теги для подтверждения принятых решений. Необязательный следующий шаг — редактирование оскорбительных кадров или содержимого.

![Представление кадров инструмента проверки видео](images/video-review-frames-view-apply-tags.PNG)

## <a name="transcript-moderation"></a>Модерация расшифровки

Видео обычно содержит закадровый голос, который также требует модерации на наличие оскорбительной речи. С помощью службы индексатора мультимедийных данных Azure можно преобразовать речь в текст, а затем использовать API проверки Content Moderator для отправки расшифровки на модерацию текста в инструменте проверки.

![Представление расшифровки инструмента проверки видео](images/video-review-transcript-view.png)
