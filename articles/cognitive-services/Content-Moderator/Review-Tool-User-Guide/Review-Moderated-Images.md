# <a name="create-human-reviews"></a>Создание пользовательской проверки

В этом руководстве вы узнаете, как настроить [просматривает](../review-api.md#reviews) средства проверки веб-сайта. Обзоры, хранения и отображения содержимого для человека модераторов для оценки. Модераторы можно изменить примените теги и применять собственные пользовательские теги, соответствующим образом. Когда пользователь завершает проверку, результаты отправляются в конечную точку заданный обратный вызов, и содержимое удаляется с сайта.

## <a name="prerequisites"></a>Технические условия

- Войдите или создайте учетную запись на Content Moderator [средство проверки](https://contentmoderator.cognitive.microsoft.com/) сайта.

## <a name="image-reviews"></a>Оценка изображений

1. Перейдите к [средство проверки](https://contentmoderator.cognitive.microsoft.com/)выберите **попробуйте** вкладке и передачи некоторых изображений для просмотра.
1. После завершения обработки передаваемых изображений, перейдите к **проверки** и выберите **изображение**.

    ![Средство проверки, открытое в браузере Chrome, с выделенным параметром Image (Изображение) на вкладке Review (Проверка)](images/review-images-1.png)

    Подписями, которые были назначены в процессе умеренность автоматического отображения изображения. Изображения, которые вы отправили через средства проверки не видимы для других рецензентов.

1. При необходимости переместите **обзоры для отображения** "ползунок" (1), чтобы настроить число изображений, отображаемых на экране. Щелкните **с тегами** или **непомеченный** кнопки (2) для сортировки изображений соответствующим образом. Щелкните панель тега (3) для его включения или отключения.

    ![Браузер Chrome с открытым средством проверки, в котором показаны изображения с тегами, выбранные для проверки](images/review-images-2.png)

1. Чтобы просмотреть дополнительные сведения на изображении, нажмите кнопку с многоточием в эскиз и выберите **просмотреть сведения о**. Изображение можно назначить подгруппам с **переместить** параметр (см. в разделе [команды](./configure.md#manage-team-and-subteams) раздел, чтобы узнать больше о подгруппы).

    ![Изображение с выделенным параметром View details (Просмотреть сведения)](images/review-images-3.png)

1. Просмотрите информацию о модерации изображения на странице сведений.

    ![Изображение с информацией о модерации в отдельной области](images/review-images-4.png)

1. После просмотра и изменения назначенных тегов (если это необходимо) нажмите кнопку **Next** (Далее), чтобы отправить результаты проверки. После отправки у вас будет примерно пять секунд на то, чтобы нажать кнопку **Prev** (Назад) и вернуться к предыдущему экрану для повторной проверки изображений. После этого изображения исчезнут из очереди отправки и кнопка **Prev** (Назад) станет неактивной.

## <a name="text-reviews"></a>Оценка текста

Текст рассматривает функции точно так же для проверки образа. Вместо отправки содержимого, вы просто написать или вставьте текст (до 1024 символов). Затем Content Moderator анализирует текст и применяет тега (вместе с другими сведениями модерации, например ненормативной лексики и личные данные). При проверке текста можно переключить примените теги или применять пользовательские теги перед отправкой на проверку.

![Снимок экрана с отображением в средстве проверки отмеченного текста в окне браузера Chrome](../images/reviewresults_text.png)
