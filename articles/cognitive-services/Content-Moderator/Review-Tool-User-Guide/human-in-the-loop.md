# <a name="content-moderator-review-tool"></a>Содержимое средства проверки модератора

Azure Content Moderator предоставляет службы для объединения машины обучения модерации содержимого с помощью пользовательской проверки и [средство проверки](https://contentmoderator.cognitive.microsoft.com) веб-сайта является понятным внешний интерфейс, который предоставляет подробный доступ к этим службам.

![Панели мониторинга средство просмотра в браузере](./images/0-dashboard.png)

## <a name="what-it-does"></a>Действие

[Средство проверки](https://contentmoderator.cognitive.microsoft.com), при использовании в сочетании с машинная Модерация API-интерфейсы, позволяет выполнять следующие задачи в процессе модерации содержимого:

- Используйте один набор средств для модерировать содержимое в различных форматах (текст, изображения и видео).
- Автоматизируйте human [просматривает](../review-api.md#reviews) при API модерации результатов вернитесь в.
- Назначить их или передавайте содержимого обзоры, которые несколько команд проверки, упорядоченные по уровень категории или качества содержимого.
- Использовать по умолчанию или настраиваемую логику фильтры ([рабочие процессы](../review-api.md#workflows)) для сортировки и отслеживать содержимое, без написания кода.
- Используйте [соединители](./configure.md#connectors) для обработки содержимого с помощью Microsoft PhotoDNA, анализа текста и API-интерфейсы распознавания лиц, помимо API-интерфейсы Content Moderator.
- Создание соединителя для создания рабочих процессов для любого API или бизнес-процесса.
- Получение ключевых метрик производительности для процессов модерации контента.

## <a name="review-tool-dashboard"></a>Панель мониторинга средство проверки

На **панели мониторинга** вкладку, вы увидите ключевые метрики для проверки содержимого, выполняемая в средство. Узнать, сколько всего, завершения и ожидающих обзоры для изображений, текста и видео-контента. Также вы увидите разбивку пользователей и групп, которые были завершены, обзоры, а также теги "умеренность", которые были применены.

![Просмотр панели мониторинга](images/0-dashboard.png)

## <a name="review-tool-credentials"></a>Учетные данные для средства проверки

При регистрации с [средство проверки](https://contentmoderator.cognitive.microsoft.com), вам будет предложено выбрать регион Azure для вашей учетной записи. Это обусловлено [средство проверки](https://contentmoderator.cognitive.microsoft.com) создает ключ бесплатной пробной версии для служб Azure Content Moderator; вам потребуется этот ключ для доступа к какой-либо службы из вызова REST или пакета SDK клиента. Ключ и API конечной точки URL-адрес можно просмотреть, выбрав **параметры** > **учетные данные**.
