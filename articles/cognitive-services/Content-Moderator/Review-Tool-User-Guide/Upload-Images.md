# <a name="upload-images"></a>Передача изображений

Чтобы изучить пользовательский интерфейс, передайте до пяти изображений за раз, при этом размер каждого файла не должен превышать 2 МБ. Обязательно проверьте размер файлов перед передачей.

![Передача изображений](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Отправка на модерацию
Если вы передали больше пяти изображений одновременно, используйте значок корзины для удаления лишних изображений. Затем отправьте изображения на автоматическую модерацию.

![Отправка изображений](images/1-try-image-2.png)
