# <a name="use-active-learning-to-improve-your-knowledge-base"></a>Использование активного обучения для расширения базы знаний

[Активное обучение](../Concepts/active-learning-suggestions.md) позволяет улучшить качество вашей базы знаний, предлагая альтернативные вопросы. Представления пользователей принимаются во внимание и отображаются в виде предложений в списке альтернативных вопросов. У вас есть гибкость, чтобы либо добавить эти предложения в качестве альтернативных вопросов или отклонить их.

База знаний не изменяется автоматически. Для того, чтобы любое изменение вступило в силу, вы должны принять предложения. Так вы добавите новые вопросы, не меняя и не удаляя существующие.


## <a name="upgrade-runtime-version-to-use-active-learning"></a>Обновление версии времени выполнения для использования активного обучения

Активное обучение поддерживается в среде выполнения версии 4.4.0 и выше. Если ваша база знаний создана с помощью более ранней версии, [обновите среду выполнения](set-up-qnamaker-service-azure.md#get-the-latest-runtime-updates), чтобы воспользоваться этой функцией.

## <a name="turn-on-active-learning-for-alternate-questions"></a>Включите активное обучение для альтернативных вопросов

По умолчанию активное обучение отключено. Чтобы просматривать предлагаемые вопросы, включите эту функцию. После того, как вы включите активное обучение, вам необходимо отправить информацию из клиентского приложения в SnA Maker. Для получения дополнительной информации [см. Архитектурный поток для использования GenerateAnswer и Train AIS от бота.](improve-knowledge-base.md#architectural-flow-for-using-generateanswer-and-train-apis-from-a-bot)

1. Выберите **Опубликовать** для публикации базы знаний. Активные запросы обучения собираются только из конечных точек прогнозирования GenerateAnswer API. Запросы на тестовую панель на портале «NA Maker» не влияют на активное обучение.

1. Чтобы включить активное обучение на портале «NA Maker», перейдите в правый верхний угол, выберите **имя,** перейдите в [**настройки сервиса.**](https://www.qnamaker.ai/UserSettings)

    ![Включите предлагаемые варианты вопросов активного обучения со страницы настроек Службы. Выберите имя пользователя в правом верхнем меню, а затем выберите настройки службы.](../media/improve-knowledge-base/Endpoint-Keys.png)


1. Найдите службу QnA Maker, а затем активируйте переключатель **Active Learning** (Активное обучение).

    > [!div class="mx-imgBorder"]
    > [![На странице настроек Службы переключайся на функции активного обучения. Если вы не в состоянии переключить функцию, возможно, потребуется обновить службу.](../media/improve-knowledge-base/turn-active-learning-on-at-service-setting.png)](../media/improve-knowledge-base/turn-active-learning-on-at-service-setting.png#lightbox)

    > [!Note]
    > Точная версия на предыдущем изображении показана только в качестве примера. Ваша версия может быть другой.

    После включения **active Learning** база знаний предлагает новые вопросы через регулярные промежутки времени на основе вопросов, представленных пользователем. Вы можете отключить **активное обучение** с помощью этого же переключателя.
