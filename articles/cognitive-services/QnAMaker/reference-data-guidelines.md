# <a name="import-and-export-data-reference"></a>Ссылка на данные по импорту и экспорту

Просмотрите эту ссылку на импорт и экспорт, чтобы получить наилучшие результаты для резервного копирования, хранения и замены базы знаний.

## <a name="import-and-export-knowledge-base"></a>База знаний об импорте и экспорте

**Файлы TSV и XLS**из экспортируемых баз знаний могут использоваться только путем импорта файлов со **страницы «Настройки»** на портале «NA Maker». Они не могут быть использованы в качестве источников данных во время создания базы знаний или из **файла «Добавить»** или **добавить** функцию URL на странице **«Настройки».**
