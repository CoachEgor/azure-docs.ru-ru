# <a name="design-knowledge-base-for-content-language"></a>База знаний о дизайне языка контента

Ресурс «NA Maker» и все базы знаний внутри этого ресурса поддерживают единый язык. Единый язык необходим для предоставления наилучших результатов ответа для запроса.

## <a name="single-language-per-resource"></a>Единый язык на ресурсе

Соображения для языковой поддержки:

* Сервис SnA Maker и все ее базы знаний поддерживают только один язык.
* Язык явно устанавливается при создании первой базы знаний службы
* Язык определяется из файлов и URL-адресов, добавленных при создании базы знаний
* Язык не может быть изменен для любых других баз знаний в службе
* Язык используется службой Cognitive Search (ranker #1) и сервисом «NA Maker» (ranker #2) для создания наилучшего ответа на запрос

## <a name="supporting-multiple-languages"></a>Поддержка нескольких языков

Если вам необходимо поддерживать базу знаний, которая включает в себя несколько языков, вы можете выбрать один из следующих методов:

* Воспользуйтесь [службой Translation Text](../../translator/translator-info-overview.md) для перевода вопроса на один язык, прежде чем отправлять вопрос в базу знаний. Это позволяет сосредоточиться на качестве одного языка и качестве альтернативных вопросов и ответов.
* Создание ресурса «NA Maker» и базы знаний внутри этого ресурса для каждого языка. Это позволяет управлять отдельными альтернативными вопросами и отвечать на текст, который является более тонким для каждого языка. Это дает вам гораздо больше гибкости, но требует гораздо более высоких затрат на техническое обслуживание, когда вопросы или ответы меняются на всех языках.

Языки рецензя, [поддерживаемые](../overview/language-support.md) для qnA Maker.

### <a name="support-each-language-with-a-qna-maker-resource"></a>Поддержка каждого языка с помощью ресурса «NA Maker»

* Создание ресурса для каждого языка для создателя
* Только добавляйте файлы и URL-адреса для этого языка
* Используйте конвенцию именования для ресурса для определения языка. Примером `qna-maker-fr` является все базы знаний для французских документов
