# <a name="label-images-faster-with-smart-labeler"></a>Этикетка изображения быстрее с Smart Labeler

В этом руководстве вы узнаете, как использовать Smart Labeler для создания предлагаемых тегов для изображений. Это позволяет более быстро маркировать большое количество изображений при обучении модели Custom Vision.

При пометке изображений для модели Custom Vision служба использует последнюю обученную итерацию модели для прогнозирования меток немаркированных изображений. Затем он показывает эти прогнозы в виде предлагаемых тегов, основанных на выбранном пороге доверия и неопределенности прогнозирования. Затем можно либо подтвердить, либо изменить предложения, ускорив процесс ручной пометки изображений для обучения.

## <a name="when-to-use-smart-labeler"></a>Когда использовать Smart Labeler

Имейте в виду следующие ограничения:

* Вы должны запросить только предлагаемые теги для изображений, содержание которых уже было обучено один раз. Не получайте предложений для нового тега, который вы только начинаете тренировать.

> [!IMPORTANT]
> Функция Smart Labeler использует ту же [модель ценообразования,](https://azure.microsoft.com/pricing/details/cognitive-services/custom-vision-service/) что и регулярные прогнозы. При первом запуске предлагаемых тегов для набора изображений с вас будет взиматься такая же плата, как и за вызовы прогнозирования. После этого служба хранит результаты для выбранных изображений в базе данных в течение 30 дней, и вы можете получить к ним доступ в любое время бесплатно в течение этого периода. Через 30 дней с вас снова будет взиматься плата, если вы снова попросите их предложенные теги.

## <a name="smart-labeler-workflow"></a>Рабочий процесс Smart Labeler

Следующие шаги показывают, как использовать Smart Labeler:

1. Загрузите все обучающие изображения в проект Custom Vision.
1. Метка часть вашего набора данных, выбирая равное количество изображений для каждого тега.
    > [!TIP]
    > Убедитесь, что вы используете все теги, для которых вы хотите предложения позже.
1. Начните тренировочный процесс.
1. Когда обучение завершено, перейдите к **немаркированному** представлению и выберите кнопку **«Предлагаемые теги»** на левой панели.
    > [!div class="mx-imgBorder"]
    > ![Предлагаемая кнопка тегов отображается под вкладкой немаркированных изображений.](./media/suggested-tags/suggested-tags-button.png)
1. В всплывающем окне всплывающее окно установите количество изображений, для которых нужны предложения. Вы должны получить только первоначальные предложения тега для части немаркированных изображений. Вы получите лучшие предложения тегов, как вы итерации через этот процесс.
1. Подтвердите предложенные теги, исправив все, что не правильно.
    > [!TIP]
    > Изображения с предлагаемыми тегами сортируются по неопределенности прогноза (более низкие значения указывают на более высокую достоверность). Можно изменить порядок сортировки с **помощью опции неопределенности.** Если вы установите заказ до **максимума до минимума,** вы можете сначала исправить прогнозы с высокой неопределенностью, а затем быстро подтвердить низконеопределенностные.
    * В проектах классификации изображений можно выбрать и подтвердить теги в пакетах. Фильтр вид за данный тег, отменить изображения, которые помечены неправильно, а затем подтвердить остальные в пакете.
        > [!div class="mx-imgBorder"]
        > ![Предлагаемые теги отображаются в пакетном режиме для IC с фильтрами.](./media/suggested-tags/ic-batch-mode.png)

        Вы также можете использовать предлагаемые теги в индивидуальном режиме изображения, выбрав изображение из галереи.

        ![Предлагаемые теги отображаются в индивидуальном режиме изображения для IC.](./media/suggested-tags/ic-individual-image-mode.png)
    * В проектах обнаружения объектов пакетные подтверждения не поддерживаются, но все равно можно фильтровать и сортировать по предлагаемым тегам для более организованного опыта маркировки. Thumbnails ваших untagged изображений покажет наложение ограничивающих коробок, указывающих расположение предлагаемых тегов. Если вы не выберете предложенный фильтр тегов, все ваши немаркированные изображения будут отображаться без наложения ограничивающих коробок.
        > [!div class="mx-imgBorder"]
        > ![Предлагаемые теги отображаются в пакетном режиме для OD с фильтрами.](./media/suggested-tags/od-batch-mode.png)

        Чтобы подтвердить теги обнаружения объектов, необходимо применить их к каждому отдельному изображению в галерее.

        ![Предлагаемые теги отображаются в индивидуальном режиме изображения для OD.](./media/suggested-tags/od-individual-image-mode.png)
1. Начните тренировочный процесс заново.
1. Повторите предыдущие шаги до тех пор, пока вы не будете удовлетворены качеством предложения.
