# <a name="test-and-retrain-a-model-with-custom-vision-service"></a>Тестирование и переобучение модели с помощью Пользовательской службы визуального распознавания

После обучения модели ее можно быстро протестировать с помощью хранящегося локально или в сети изображения. Для тестирования используется итерация самого последнего обучения.

## <a name="test-your-model"></a>Тестирование модели

1. На [веб-странице Пользовательской службы визуального распознавания](https://customvision.ai) выберите свой проект. Выберите **Quick Test** (Быстрый тест) справа в верхней строке меню. Откроется окно **Quick Test** (Быстрый тест).

    ![В правом верхнем углу окна отображена кнопка "Quick Test" (Быстрый тест).](./media/test-your-model/quick-test-button.png)

2. В окне **Quick Test** (Быстрый тест) щелкните поле **Submit Image** (Отправка изображения) и введите URL-адрес изображения, которое вы хотите использовать для теста. Если вместо этого вы хотите использовать локальное изображение, щелкните **Browse local files** (Обзор локальных файлов) и выберите локальный файл изображения.

    ![Изображение страницы отправки изображения](./media/test-your-model/submit-image.png)

Выбранное изображение появится посредине страницы. Затем под изображением отобразятся результаты в виде таблицы с двумя столбцами, **Tags** (Теги) и **Confidence** (Достоверность). После просмотра результатов окно **Quick Test** (Быстрый тест) можно будет закрыть.

Теперь вы можете добавить протестированное изображение в модель и затем переобучить ее.

## <a name="use-the-predicted-image-for-training"></a>Использование прогнозируемого образа для обучения

Чтобы использовать отправленное ранее изображение для обучения, выполните следующие действия.

1. Чтобы просмотреть отправленные в классификатор изображения, откройте [веб-страницу Пользовательской службы визуального распознавания](https://customvision.ai) и выберите вкладку __Predictions__ (Прогнозы).

    ![Изображение вкладки прогнозов](./media/test-your-model/predictions-tab.png)

    > [!TIP]
    > Представление по умолчанию содержит изображения из текущей итерации. Поле с раскрывающимся списком __Iteration__ (Итерация) позволяет перейти к изображениям, переданным в предыдущих итерациях.

2. Наведите указатель мыши на изображение, чтобы увидеть спрогнозированные классификатором теги.

    > [!TIP]
    > Изображения ранжируются по полезности для классификатора, то есть самые полезные расположены вверху списка. Чтобы выбрать другой вариант сортировки, используйте раздел __Sort__ (Сортировка).

    Чтобы добавить изображение в набор данных для обучения, выберите нужное изображение, затем тег и щелкните __Save and close__ (Сохранить и закрыть). Выбранное изображение удаляется из списка __Predictions__ (Прогнозы) и добавляется к изображениям для обучения. Теперь его можно просмотреть на вкладке __Training Images__ (Изображения для обучения).

    ![Изображение страницы для управления тегами](./media/test-your-model/tag-image.png)

3. Нажмите кнопку __Train__ (Обучение), чтобы переобучить классификатор.
