## <a name="clean-up-resources"></a>Очистка ресурсов

Если нужно реализовать собственный проект классификации изображений (или использовать вместо него проект [обнаружения объектов](../quickstarts/object-detection.md)), вы можете удалить проект определения видов деревьев из этого примера. При работе в бесплатной пробной версии Пользовательской службы визуального распознавания можно использовать два проекта.

На веб-сайте [Пользовательской службы визуального распознавания](https://customvision.ai) перейдите в раздел **проектов** и щелкните значок корзины под заголовком "My New Project" (Мой новый проект).
