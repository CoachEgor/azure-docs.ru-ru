## <a name="get-the-training-and-prediction-keys"></a>Получение ключей обучения и прогнозирования

Для взаимодействия со службой вам нужно добавить в проект допустимый набор ключей подписки. Элементы можно найти на веб-сайте [службы "Пользовательское визуальное распознавание"](https://customvision.ai). Войдите с учетной записью, связанной с учетной записью Azure, которая использовалась для создания ресурсов службы "Пользовательское визуальное распознавание". Выберите __значок шестеренки__ в правом верхнем углу. В разделе __учетных записей__ содержатся раскрывающиеся списки для ресурсов Azure, которые позволяют увидеть значения для полей __ключа обучения__, __ключа прогнозирования__ и __идентификатора ресурса прогнозирования__.

![Изображение пользовательского интерфейса с ключами](../media/csharp-tutorial/training-prediction-keys.png)

 Кроме того, эти ключи и идентификаторы можно получить на [портале Azure](https://www.portal.azure.com), просмотрев ресурсы обучения и прогнозирования службы "Пользовательское визуальное распознавание" и вкладку __Ключи__. Там можно найти __ключ обучения__ или __ключ прогнозирования__. Перейдите на вкладку __Свойства__ ресурса прогнозирования, чтобы получить __идентификатор ресурса прогнозирования__.
