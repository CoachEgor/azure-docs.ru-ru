# <a name="how-to-sign-up-for-the-text-analytics-api"></a>Сведения о том, как зарегистрироваться для использования API анализа текста

В облаке ресурсы API анализа текста доступны круглосуточно. Прежде чем передавать содержимое для анализа, вы должны зарегистрироваться, чтобы получить ключ доступа. В каждом запросе для вызова API требуется ключ доступа.

+ Начните с создания подписки Azure. Чтобы поэкспериментировать, вы можете создать [бесплатную учетную запись](https://azure.microsoft.com/free/).

+ Создайте [учетную запись API Cognitive Services](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account), выбрав **API анализа текста**. Ключ создается при регистрации.

Для API анализа текста есть уровень "Бесплатный" для анализа и вычисления, а также платные уровни для рабочих нагрузок в производственной среде. Может иметь несколько регистрационные данные в каждой подписке: один free, одно платное и т. д. Вы можете переключиться на уровень, предлагающий больше транзакций при увеличении объема запросов.

Для служб, доступных в режиме предварительной версии или на уровне "Бесплатный", соглашение об уровне обслуживания отсутствует. Дополнительные сведения см. в статье [Соглашение об уровне обслуживания для Cognitive Services](https://azure.microsoft.com/support/legal/sla/cognitive-services/v1_1/).

## <a name="how-to-change-tiers"></a>Как изменить уровни

Начните с уровня "Бесплатный", а затем перейдите на платный уровень для рабочих нагрузок в производственной среде. На дифференцированных уровнях предлагается стандартное выставление счетов. Вы можете переключаться между уровнями и сохранять одну и ту же конечную точку и ключи доступа.

1. Войдите на [портал Azure](https://portal.azure.com) и [найдите свою службу](text-analytics-how-to-access-key.md).

2. Щелкните **Ценовая категория**.

   ![Команда "Ценовая категория" в меню навигации слева](../media/portal-pricing-tier.png)

3. Выберите уровень и нажмите кнопку **Выбрать**.  Новые ограничения вступят в силу сразу же после обработки выбора. 

   ![Плитки и кнопка "Выбрать" на странице выбора уровня](../media/portal-choose-tier.png)

## <a name="how-billing-works"></a>Как работает выставление счетов

Выставление счетов основано на количестве транзакций. Вы можете приобрести блок транзакций для определенного уровня в месячном цикле выставления счетов, а затем при переходе на другой уровень для каждой транзакции применяется небольшая плата за избыток. Если вы регулярно превышаете допустимое ограничение, подумайте, чтобы перейти на более высокий уровень.

Дополнительные сведения см. на [странице цен](https://azure.microsoft.com/pricing/details/cognitive-services/text-analytics/).

### <a name="what-constitutes-a-transaction-in-the-text-analytics-api"></a>Из чего состоит транзакция в API анализа текста?
Любая заметка к документу считается транзакцией. При расчете стоимости для вызова пакетной оценки учитывается количество документов, которые нужно оценить в соответствующей транзакции. Так, например, если в одном вызове API для анализа тональности отправляется 1000 документов, это расценивается как 1000 транзакций.
