# <a name="supported-types-for-named-entity-recognition-version-3-preview"></a>Поддерживаемые типы для распознавания именованных сущностей версии 3 (Предварительная версия)

API анализа текста предоставляет возможность определения и устранения неоднозначности сущностей, найденных в тексте. [Распознавание имен сущностей](how-tos/text-analytics-how-to-entity-linking.md) (NER) — это возможность определять различные сущности в тексте и классифицировать их в заранее определенные классы.
