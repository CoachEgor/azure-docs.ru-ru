> [!TIP]
> Чтобы найти ключ и конечную точку на портале Azure, сделайте следующее:
> 1. Перейдите к ресурсу Azure по адресу: https://portal.azure.com/
> 2. Щелкните элемент **Быстрый запуск**, расположенный в разделе **Управление ресурсами**.
>
> Не забудьте удалить суффикс `/text/analytics/<version>` в конечной точке при его наличии. Этот суффикс не требуется для использования с клиентскими библиотеками.
