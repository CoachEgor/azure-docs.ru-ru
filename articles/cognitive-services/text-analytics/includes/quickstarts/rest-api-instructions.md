1. Скопируйте команду в текстовый редактор.
2. При необходимости внесите следующие изменения в команду.
    1. Замените значение `<your-text-analytics-key-here>` собственным ключом.
    2. Замените первую часть URL-адреса запроса `<your-text-analytics-endpoint-here>` URL-адресом своей конечной точки.
3. Откройте окно командной строки.
4. Вставьте команду из текстового редактора в окно командной строки и выполните команду.
