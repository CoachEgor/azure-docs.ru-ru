В API "Анализ текста" версии 3 можно выбрать наиболее актуальную версию модели для данных. Чтобы выбрать необходимую версию модели для запросов, используйте необязательный параметр `model-version`. Если этот параметр не указан, интерфейс API по умолчанию использует значение `latest`, то есть последнюю стабильную версию. Вы можете использовать последнюю версию модели в любом запросе, хотя в каждой версии обновляются только лишь некоторые функции. В следующей таблице приводятся обновленные функции для каждой версии модели.

| Версия модели           | Обновленные функции         | Последняя версия для функции           |
|-------------------------|--------------------------|--------------------------|
| `2020-02-01`            | Распознавание сущностей                      | Распознавание сущностей                      |
| `2019-10-01`            | Распознавание сущностей, анализ тональности  | Распознавание языка, извлечение ключевых фраз, анализ тональности|


Каждый ответ от конечных точек версии 3 включает в себя поле `model-version`, в котором указывается используемая версия модели.

```json
{
    "documents": […]
    "errors": []
    "model-version": "2019-10-01"
}
```
Дополнительные сведения об обновлениях для этих версий модели см. в статье с описанием [новых возможностей](../whats-new.md).
