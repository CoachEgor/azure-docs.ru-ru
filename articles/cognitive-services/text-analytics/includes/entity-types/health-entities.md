## <a name="health-entity-categories"></a>Категории сущностей работоспособности:

[Анализ текста для работоспособности](../../how-tos/text-analytics-for-health.md)возвращаются следующие категории сущностей.  Обратите внимание, что в этом предварительном просмотре контейнера поддерживается только текст на английском языке, и в каждом образе контейнера предоставляется только одна версия модели.

### <a name="named-entity-recognition"></a>Распознавание именованных сущностей

|Категория  |Описание   |
|----------|--------------|
| Возраст | Устаревают. Например, *30 лет*. |
| административивент | Административное событие. |
| бодиструктуре | Части пользовательского текста, включая Органс и другие структуры. Например, *ARM*или *сердце*. | 
| каринвиронмент | Среда, в которой осуществляется управление с осторожностью или лечения. Например, *экстренная комната* | 
| кондитионкуалифиер | Уровни условий. Например, *умеренное*, *Расширенное*или *рассеянное*. | 
| Diagnosis | Медицинские условия. Например, « *натяжение*». | 
| Направление | По. Например, *Left* или *Антериор*. | 
| досаже | Размер или количество лечения. Например, *25mg*.  | 
| ексаминатионнаме | Метод или процедура проверки. Например, *X-Ray*. | 
| релатионалоператор | Оператор, определяющий связь между двумя сущностями. Например, *меньше*или `>=` .  | 
| MeasurementUnit | Единица измерения (например, процент). | 
| меасурементвалуе | Числовое значение единицы измерения. | 
| фамилирелатион | Отношение фамилиал. Например, *связанное*.  | 
| Частота | Частоты.   | 
| пол; | Разные половые принадлежности. | 
| Ген | Сущность ген, например *TP53*.   | 
| хеалскарепрофессион | Метод администрирования лечения. Например, *устные Administration*. | 
| медикатионкласс | Классы лечения. Например, *антибиотикс*.  | 
| медикатионформ | Форма лечения. Например, *капсула*. | 
| медикатионнаме  | Универсальная и фирменная символика с именем медикатионс. Например, *ибупрофен*. | 
| медикатионрауте | Метод администрирования лечения. Например, *устные Administration*. | 
| симптоморсигн  | Признаков заболеваний. Например, очень *сроат*. | 
| Time | Указанное. Например, *8 лет* или *2:10:30 это утро* |
| треатментнаме  | Имена лечения. Например, *серапи*. | 
| Variant | Вариант генетических сущности ген. | 

### <a name="relation-extraction"></a>Извлечение связей

Извлечение связей определяет осмысленные соединения между понятиями, упомянутыми в тексте. Например, связь "время условия" обнаруживается путем связывания имени условия с временем. Анализ текста для работоспособности может обозначать следующие отношения:

|Категория  |Описание   |
|----------|--------------|
| директионофбодиструктуре | Направление структуры текста. |
| директионофкондитион | Направление условия. |
| директионофексаминатион | Направление проверки. |
| директионофтреатмент | Направление обработки. |
| тимеофкондитион | Время, связанное с SES условия. |
| куалифиерофкондитион | Связанный квалификатор для условия. |
| досажеофмедикатион | Досаже лечения. |
| формофмедикатион | Форма лечения. |
| раутеофмедикатион | Маршрут или режим использования медицина. Например, *устные*. |
| фрекуенциофмедикатион | Частота, с которой потребляется лечения. | 
| валуеофкондитион | Числовое значение, связанное с условием. |
| унитофкондитион | Единица (например, время), связанная с условием. |
| тимеофмедикатион | Время использования лечения. |
| тимеофтреатмент | Время администрирования обработки. | 
| фрекуенциофтреатмент | Частота, с которой администрируется обработка. |
| валуеофексаминатион | Числовое значение, связанное с проверкой. | 
| унитофексаминатион | Единица (например, процентная доля), связанная с проверкой. |
| релатионофексаминатион | Отношение между сущностью и проверкой. | 
| тимеофексаминатион | Время, связанное с проверкой. |
| Сокращение | Аббревиатура.  |
