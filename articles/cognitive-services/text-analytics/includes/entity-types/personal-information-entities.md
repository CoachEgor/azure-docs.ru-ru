> [!NOTE]
> Чтобы определить защищенную информацию о работоспособности (фи), используйте `domain=phi` параметр и версию модели `2020-04-01` или более поздней версии.
>
> Пример: `https://<your-custom-subdomain>.cognitiveservices.azure.com/text/analytics/v3.1-preview.2/entities/recognition/pii?domain=phi&model-version=2020-07-01`

|------------|-------------|--------------------------------------|------------------------|---|
| Человек     | Недоступно         | Имена людей.  | `2019-10-01`  | Также возвращается с `domain=phi` . |

## <a name="azure-information"></a>Сведения об Azure

Эта категория сущностей включает в себя идентифицируемые сведения об Azure, включая сведения о проверке подлинности и строки подключения. Доступно начиная с версии модели `2019-10-01` . Не возвращается с `domain=phi` параметром.

| Подкатегория                           | Описание                                                                 |
|---------------------------------------|-----------------------------------------------------------------------------|
| ключ проверки подлинности Azure DocumentDB;             | Ключ авторизации для сервера Azure Cosmos DB.                           |
| строка подключения Базы данных Azure IAAS и строка подключения Azure SQL; | Строка подключения для базы данных инфраструктуры Azure в виде службы (IaaS) и строки подключения SQL. |
| Строка подключения SQL Azure           | Строка подключения для базы данных в базе данных SQL Azure.                                |
| строка подключения Azure IoT;           | Строка подключения для Azure IoT.                        |
| пароль для публикации Azure;        | Пароль для параметров публикации Azure.                                        |
| строка подключения кэша Redis для Azure;   | Строка подключения для кэша Redis.                             |
| Azure SAS;                             | Строка подключения для программного обеспечения Azure как услуги (SaaS).                     |
| строка подключения Служебной шины Azure;   | Строка подключения для служебной шины Azure.                                 |
| Ключ учетной записи хранения Azure             | Ключ учетной записи хранения Azure.                                   |
| ключ учетной записи хранилища Azure (универсальный).   | Универсальный ключ учетной записи для учетной записи хранения Azure.                           |
| строка подключения SQL Server;          | Строка подключения для компьютера, на котором работает SQL Server.                                         |
