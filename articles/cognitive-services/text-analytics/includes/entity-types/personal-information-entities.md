## <a name="personal-information-entity-types"></a>Типы сущностей личных сведений:

### <a name="phone-number"></a>номер телефона;

Номера телефонов. 

Языки:

* Общедоступная Предварительная версия: `English`

| Имя подтипа           | Description (Описание)                                           |
|------------------------|-------------------------------------------------------|
| Недоступно                    | Номера телефонов, например `+1 123-123-123`.          |
| Номер телефона ЕС        | Номера телефонов, относящиеся к Европейского союза.         |
| Номер мобильного телефона в ЕС | Номера мобильных телефонов, характерные для Европейского союза. |

### <a name="eu-gps-coordinates"></a>Координаты GPS в ЕС

 Координаты GPS для мест в Европейского союза. 

Языки:

* Общедоступная Предварительная версия: `English`

| Имя подтипа | Description (Описание)                               |
|--------------|-------------------------------------------|
| Недоступно          | Координаты GPS в Европейского союза |

### <a name="azure-information"></a>Сведения об Azure

Идентифицируемые сведения об Azure, включая сведения о проверке подлинности и строки подключения. 

Языки:

* Общедоступная Предварительная версия: `English`

| Имя подтипа                          | Description (Описание)                                                                 |
|---------------------------------------|-----------------------------------------------------------------------------|
| Ключ проверки подлинности Azure DocumentDB             | Ключ авторизации для сервера Azure DocumentDB.                           |
| Строка подключения к базе данных IAAS Azure | Строка подключения для базы данных инфраструктуры Azure в виде службы (IaaS). |
| Строка подключения SQL Azure           | Строка подключения для базы данных SQL Azure.                                |
| Строка подключения к Azure IoT           | Строка подключения для Интернета вещей Azure (IoT).                        |
| Пароль параметра публикации Azure        | Пароль для параметров публикации Azure.                                        |
| Строка подключения кэша Redis для Azure   | Строка подключения для кэша Azure для Redis.                             |
| SAS Azure                             | Строка подключения для службы программного обеспечения Azure (SAS).                     |
| Строка подключения служебной шины Azure   | Строка подключения для служебной шины Azure.                                |
| Ключ учетной записи хранения Azure             | Ключ учетной записи хранения Azure.                                   |
| Ключ учетной записи хранения Azure (Универсальный)   | Универсальный ключ учетной записи для учетной записи хранения Azure.                           |
| Строка подключения SQL Server          | Строка подключения для SQL Server.                                         |
