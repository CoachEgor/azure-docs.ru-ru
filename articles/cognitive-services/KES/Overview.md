# <a name="knowledge-exploration-service"></a>Служба изучения знаний

Вас приветствует Служба поиска и распознавания данных (Майкрософт).  Служба поиска и распознавания данных (KES) позволяет быстро и эффективно добавлять возможности интерактивного поиска и уточнения в приложения.  С помощью KES можно создать сжатый индекс из структурированных данных, создать грамматику для интерпретации запросов на естественном языке, а также обеспечить интерактивное составление запросов с вариантами автозавершения.

В разделе [Приступая к работе](GettingStarted.md) представлен пример с пошаговыми инструкциями по использованию KES для создания интерфейса интерактивного поиска для области научных публикаций.

![AcademicAutoComplete](AutoComplete.png)

Служба поиска и распознавания данных перемещена из предварительной версии Cognitive Services в Cognitive Services Labs. Теперь домашняя страница этого проекта имеет адрес [https://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration](https://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration).
