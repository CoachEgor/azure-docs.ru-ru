# <a name="scenario-availability"></a>Доступность сценариев

Speech SDK содержит множество сценариев в самых разных языках и средах программирования. Пока не все сценарии доступны на всех языках программирования или во всех средах. Ниже приводится доступность каждого сценария.

- **Распознавание речи (SR), Список фраз, намерения, перевод и контейнеры на территории**
  - СЗ/Windows & Linux & macOS
  - СЗ (Рамочное & .NET Core)/Windows & UWP & Unity & Xamarin & Linux & macOS
  - Java (Jre и Android)
  - JavaScript (Брауэр и NodeJS)
  - Python
  - Swift
  - Objective-C  
- **Текст к речи (TTS)**
  - СЗ/Windows & Linux
  - C/Windows & UWP & unity
  - Java (Jre и Android)
  - Python
  - Swift
  - Objective-C
  - TTS REST API может быть использован в любой другой ситуации.
- **Ключевое слово Spotting (KWS)**
  - СЗ/Windows & Linux
  - СЗ/Windows & Linux
  - Python/Windows & Linux
  - Java/Windows & Linux & Android (Речевые устройства SDK)
  - Функциональность пятнистости ключевых слов (KWS) может работать с любым типом микрофона, официальная поддержка KWS, однако, в настоящее время ограничена массивами микрофонов, найденными в оборудовании Azure Kinect DK или SDK-устройствах речи
- **Голосовые помощники**
  - СЗ/Windows & Linux & macOS
  - C/Windows
  - Java/ Windows & Linux & macOS & Android (Речевые устройства SDK)
- **Транскрибирование бесед**
  - СЗ/Windows & Linux
  - СЗ (Рамочное & .NET Core)/Windows & UWP & Linux
  - Java/Windows & Linux & Android (Речевые устройства SDK)
- **Ведение беседы с использованием нескольких устройств**
  - СЗ/Окна
  - СЗ (Рамочное & .NET Core)/Windows
- **Стенограмма колл-центра**
  - REST API и может быть использован в любой ситуации
- **Кодек Сжатый аудио вход**
  - СЗ/Linux
  - СЗ/Linux
  - Java/Linux, Android и iOS
