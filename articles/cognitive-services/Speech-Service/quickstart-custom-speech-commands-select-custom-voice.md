# <a name="quickstart-use-custom-commands-with-custom-voice-preview"></a>Быстрый запуск: Используйте пользовательские команды с пользовательским голосом (Предварительный просмотр)

В [предыдущей статье](./quickstart-custom-speech-commands-create-parameters.md)мы создали новый проект пользовательских команд для реагирования на команды с параметрами.

В этой статье мы выберем пользовательский голос вывода для созданного приложения.

## <a name="select-a-custom-voice"></a>Выберите пользовательский голос

1. Откройте проект, [который мы создали ранее](./quickstart-custom-speech-commands-create-parameters.md)
1. Выберите **настройки** из левого стекла
1. Выберите **пользовательский голос** из среднего стекла
1. Выберите нужный пользовательский или публичный голос из таблицы
1. Выберите **Сохранить**

> [!div class="mx-imgBorder"]
> ![Примеры предложений с параметрами](media/custom-speech-commands/select-custom-voice.png)

> [!NOTE]
> Пользовательские голоса могут быть созданы со страницы проекта Custom Voice. Выберите ссылку **Студии речи,** а затем **пользовательский голос,** чтобы начать работу.

Теперь приложение будет отвечать выбранным голосом, а не голосом по умолчанию.
