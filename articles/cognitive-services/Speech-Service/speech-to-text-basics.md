# <a name="get-started-with-speech-to-text"></a>Начало работы с преобразованием речи в текст

Одной из основных функций службы распознавания речи является способность распознать и преобразовать речь (часто это называется преобразованием речи в текст). Из этого краткого руководство вы узнаете, как использовать пакет SDK для службы "Речь" в приложениях и продуктах для выполнения высококачественного преобразования речи в текст.

> [!TIP]
> Если у вас нет возможности воспользоваться одним из наших кратких руководств, мы рекомендуем вам начать работу и попробовать распознавание речи самостоятельно.
> * [Распознавание речи с микрофона](quickstarts/speech-to-text-from-microphone.md)
