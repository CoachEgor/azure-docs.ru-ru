# <a name="learn-the-basics-of-speech-recognition"></a>Основные сведения о распознавании речи

Одной из основных функций службы распознавания речи является способность распознать и преобразовать речь (часто это называется преобразованием речи в текст). В этой статье вы узнаете, как использовать пакет SDK для службы "Речь" в приложениях и продуктах для выполнения высококачественного распознавания речи.

> [!TIP]
> Если у вас нет возможности воспользоваться одним из наших кратких руководств, мы рекомендуем вам начать работу и попробовать распознавание речи самостоятельно.
> * [Распознавание речи с микрофона](quickstarts/speech-to-text-from-microphone.md)
