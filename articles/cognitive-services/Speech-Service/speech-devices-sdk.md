# <a name="about-the-speech-devices-sdk"></a>Сведения о пакете SDK для устройств в службе "Речь"

[Служб речи](overview.md) работают с самых разнообразных устройств и источников аудио. Теперь ваши речевые приложения могут стать еще эффективнее благодаря соответствующему оборудованию и программному обеспечению. Пакет SDK для речевых устройств — это предварительно настроенная библиотека, связанная со специально созданными пакетами разработки для микрофонной решетки.

Пакет SDK для речевых устройств помогает решать следующие задачи:

* быстрое тестирование новых сценариев голосовой связи;
* Более легко интегрируйте облачных служб речи в устройстве.
* создание удобного интерфейса пользователя для ваших клиентов.

Пакет SDK для речевых устройств использует [пакет SDK для распознавания речи](speech-sdk.md). Она использует Speech SDK для отправки аудио, обрабатываемого наш алгоритм расширенная обработка аудио из массива микрофон устройства для [служб речи](overview.md). Он использует многоканальный звук для обеспечения более точного [распознавания речи](speech-to-text.md) благодаря шумоподавлению, подавлению эха, обнаружению голоса в дальней зоне, формированию луча и подавлению реверберации.

Speech SDK устройства можно также использовать для создания окружения устройств, которые имеют собственные [настроены wake word](speech-devices-sdk-create-kws.md) , подсказки, которая инициирует взаимодействие с пользователем является уникальным для вашего бренда.

Speech SDK устройств обеспечивает широкий спектр сценариев с голосовым управлением, такие как [виртуальные помощники Voice-First Custom](https://aka.ms/bots/speech/va), диск сквозной упорядочение систем, [расшифровка дикторского текста для диалога](conversation-transcription-service.md)и смарт- докладчики. На реплики пользователей можно отвечать текстом, общаться с ними с помощью [настраиваемых голосов](how-to-customize-voice-font.md) или голосов по умолчанию, предоставлять результаты поиска, [переводить](speech-translation.md) на другие языки и т. д. Мы будем рады видеть вашу сборку!

## <a name="get-the-speech-devices-sdk"></a>Получение пакета SDK для устройств, подключаемых к службе "Речь"

### <a name="android"></a>Android

Устройства Android скачивать последнюю версию [пакета SDK для устройств Android речи](https://aka.ms/sdsdk-download-android).

### <a name="windows"></a>Windows

Пример приложения для Windows предоставляется кросс платформенные приложения Java. Скачайте последнюю версию [JRE Speech устройств SDK](https://aka.ms/sdsdk-download-JRE).
Приложение создается с использованием пакета SDK для распознавания речи и среду разработки Java Eclipse (версия 4) на 64-разрядной Windows. Оно работает в 64-разрядной среде выполнения Java 8 (JRE).

### <a name="linux"></a>Linux

Пример приложения для Linux предоставляется кросс платформенные приложения Java. Скачайте последнюю версию [JRE Speech устройств SDK](https://aka.ms/sdsdk-download-JRE).
Приложение создается с использованием пакета SDK для распознавания речи и среду разработки Java Eclipse (версия 4) на 64-разрядная версия Linux (Ubuntu 16.04, Ubuntu 18.04, Debian 9). Оно работает в 64-разрядной среде выполнения Java 8 (JRE).
