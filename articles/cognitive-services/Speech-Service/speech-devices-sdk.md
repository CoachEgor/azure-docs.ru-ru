# <a name="what-is-the-speech-devices-sdk"></a>Что такое SDK речевых устройств?

Служба [речевой речи](overview.md) работает с широким спектром устройств и аудиоисточников. Теперь ваши речевые приложения могут стать еще эффективнее благодаря соответствующему оборудованию и программному обеспечению. Пакет SDK для устройств, подключаемых к службе "Речь" — это предварительно настроенная библиотека, связанная со специально созданными пакетами разработки для микрофонных массивов.

Пакет SDK для речевых устройств помогает решать следующие задачи:

- быстрое тестирование новых сценариев голосовой связи;
- более простая интеграция облачной службы речи в устройства;
- создание удобного интерфейса пользователя для ваших клиентов.

Пакет SDK для речевых устройств использует [пакет SDK для распознавания речи](speech-sdk.md). Использование наших передовых алгоритмов обработки аудио с микрофоном устройства массивдля для отправки аудио в [службу речи](overview.md). Он обеспечивает точное [распознавание речи](speech-to-text.md) на дальнем поле с помощью подавления шума, отмены эхо, балки и дереверберации.

Вы также можете использовать Устройства речи SDK для создания окружающих устройств, которые имеют свой собственный [индивидуальный ключевое слово.](speech-devices-sdk-create-kws.md) Пользовательские ключевое слово обеспечивает сигнал, который начинает взаимодействие с пользователем, который является уникальным для вашего бренда.

SDK Speech Devices позволяет использовать различные сценарии с поддержкой голоса, такие как [голосовые помощники,](https://aka.ms/bots/speech/va)системы заказа на диске, [транскрипция разговоров](conversation-transcription-service.md)и интеллектуальные динамики. На реплики пользователей можно отвечать текстом, общаться с ними с помощью [настраиваемых голосов](how-to-customize-voice-font.md) или голосов по умолчанию, предоставлять результаты поиска, [переводить](speech-translation.md) на другие языки и т. д. Мы будем рады видеть вашу сборку!

## <a name="get-the-speech-devices-sdk"></a>Получение пакета SDK для устройств, подключаемых к службе "Речь"

### <a name="android"></a>Android

Для Android, устройства скачать последнюю версию [Android Speech Devices SDK](https://aka.ms/sdsdk-download-android).

### <a name="windows"></a>Windows

Для Windows пример приложения предоставляется как кроссплатформенный Java-приложение. Скачать последнюю версию [JRE Речевые устройства SDK](https://aka.ms/sdsdk-download-JRE).
Приложение создается с использованием пакета SDK для распознавания речи и интегрированной среды разработки Eclipse Java (версия 4) в 64-разрядной версии Windows. Оно работает в 64-разрядной среде выполнения Java 8 (JRE).

### <a name="linux"></a>Linux

Для Linux пример приложения предоставляется как кросс-платформенный Java-приложение. Скачать последнюю версию [JRE Речевые устройства SDK](https://aka.ms/sdsdk-download-JRE).
Приложение построено с пакетом Speech SDK и Eclipse Java IDE (v4) на 64-битном Linux (Ubuntu 16.04, Ubuntu 18.04, Debian 9, RHEL 8, CentOS 8). Оно работает в 64-разрядной среде выполнения Java 8 (JRE).

Дополнительные бинарные файлы предоставляются для поддержки предстоящих устройств, [Roobo v2 DDK](https://aka.ms/sdsdk-download-roobov2), [Urbetter DDK](https://aka.ms/sdsdk-download-urbetter), и [GGEC спикер](https://aka.ms/sdsdk-download-speaker).
