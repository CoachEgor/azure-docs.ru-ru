# <a name="what-is-direct-line-speech"></a>Что такое Direct Line Speech?

**Direct Line Speech** — это надежное сквозное решение для создания гибкого, расширяемого голосового помощника. Он питается от Bot Framework и его прямой линии речи канала, который оптимизирован для голоса в, голос из взаимодействия с ботами.

[Голосовые помощники](voice-assistants.md) слушают пользователей и принимают меры в ответ, часто выступая назад. Они используют [речевой текст](speech-to-text.md) для расшифровки речи пользователя, а затем принимают меры по пониманию текста на естественном языке. Это действие часто включает в себя устный вывод от помощника, генерируемого [текстом к речи.](text-to-speech.md)

Прямая линия речи предлагает самый высокий уровень настройки и сложности для голосовых помощников. Он предназначен для разговорных сценариев, которые являются открытыми, естественными или гибридами двух с завершением задачи или использованием командии и управления. Такая высокая степень гибкости сопряжена с большей сложностью, и сценарии, которые предназначены для четко определенных задач с использованием ввода естественного языка, могут потребоваться для рассмотрения [пользовательских команд (Preview)](custom-commands.md) для обтекаемого решения.

## <a name="customization"></a>Настройка

Голосовые помощники, созданные с помощью службы speech, могут использовать весь спектр вариантов настройки, доступных для [от речи к тексту,](speech-to-text.md)текста [к речи](text-to-speech.md)и [выбора пользовательских ключевых слов.](speech-devices-sdk-create-kws.md)

> [!NOTE]
> Параметры настройки варьируются в зависимости от языка/локализации (см. [Поддерживаемые языки).](supported-languages.md)

Прямая линия речи и связанные с ней функции для голосовых помощников являются идеальным дополнением к [виртуальному помощнику решения и шаблона предприятия](https://docs.microsoft.com/azure/bot-service/bot-builder-enterprise-template-overview). Хотя Direct Line Speech может работать с любым совместимым ботом, эти ресурсы обеспечивают многоразовую основу для высококачественного разговорного опыта, а также общих навыков поддержки и моделей для быстрого запуска.
