# <a name="custom-neural-voice-gating-overview"></a>Обзор настраиваемой ограничения нейронной речи

Дополнительные сведения о процессе см. в статье Приступая к работе с настраиваемой нейронной речи.

## <a name="commitment-to-responsible-innovation"></a>Обязательства в отношении ответственных инноваций

В рамках обязательств корпорации Майкрософт по проектированию ответственного искусственного интеллекта мы подготовили набор материалов, чтобы помочь клиентам в использовании пользовательского нейронного голоса. Рекомендации и аналитические сведения, приведенные здесь, основаны на [принципах корпорации Майкрософт для ответственных инноваций в искусственном интеллекте.](https://www.microsoft.com/AI/our-approach-to-ai)

### <a name="guidance-for-deploying-custom-neural-voice"></a>Руководство по развертыванию пользовательского нейронного голоса

- [Рекомендации для ответственного развертывания](concepts-guidelines-responsible-deployment-synthetic.md): наши лучшие рекомендации на основе нашего исследования
- [Раскрытие голоса для специалиста по голосовым](https://aka.ms/disclosure-voice-talent)технологиям: что нужно знать и уведомлять о том, что технологии используют ее в качестве ответственного.
- [Проектирование раскрытия](concepts-disclosure-guidelines.md): как разрабатывать опыт, чтобы пользователи знали, когда используется искусственный Voice и доверяете вашей службе.

### <a name="why-custom-neural-voice-is-a-gated-technology"></a>Почему настраиваемая нейронная речь является многоусловной технологией

Наша цель заключается в том, чтобы защитить права пользователей и общества, способствовать прозрачному взаимодействию с человеком-компьютером и отвлечь за собой распространение вредоносных дипфакес и неправильного содержимого. По этой причине мы настроили использование пользовательского нейронного голоса. Клиенты получают доступ к технологии только после того, как их приложения просматриваются и зафиксировано их использование в соответствии с нашими принципами этического характера.

### <a name="our-gating-process"></a>Наш процесс ограничения

Чтобы получить доступ к настраиваемой нейронной речи, необходимо начать с заполнения интерактивной формы принимать. Начните работу с [приложением.](https://aka.ms/custom-neural-intake-form)

Доступ к пользовательской службе нейронной связи осуществляется по исключительному усмотрению корпорации Майкрософт в зависимости от критериев, заглянув процесса и доступности для поддержки ограниченного числа клиентов во время этой предварительной версии.

В рамках процесса приложения необходимо зафиксироваться, чтобы получить явно написанное письменное разрешение от специалистов голоса перед созданием голосового шрифта, который включает в себя общий доступ к [раскрытию голоса](https://aka.ms/disclosure-voice-talent). Кроме того, необходимо согласиться, что при развертывании голосового шрифта ваша реализация будет [раскрывать искусственный характер](concepts-disclosure-guidelines.md) службы для пользователей, предоставлять атрибуты для синтетических речевых услуг Майкрософт в условиях предоставления услуг и поддерживать канал обратной связи, позволяющий пользователям службы сообщать о проблемах и обмениваться информацией с корпорацией Майкрософт. Дополнительные сведения об условиях использования см. [здесь](https://aka.ms/custom-neural-code-of-conduct).
