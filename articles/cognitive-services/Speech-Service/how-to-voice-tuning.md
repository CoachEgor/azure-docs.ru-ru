# <a name="fine-tune-text-to-speech-output"></a>Настройка преобразования текста в речь

Службы распознавания речи Azure позволяют настраивать скорость, произношение, громкость, шаг и профиль вывода текста в речь с помощью [языка разметки речи (SSML)](speech-synthesis-markup.md). SSML — это язык разметки на основе XML, который использует теги для информирования службы о том, какая функция требует настройки. Затем сообщение SSML отправляется в тексте каждого запроса в службу преобразования текста в речь. Для упрощения процесса настройки службы речи теперь предлагают средство [настройки голосовой связи](https://aka.ms/voicetuning) , которое позволяет визуально проверять и настраивать вывод текста в речь в режиме реального времени.

Средство настройки речи поддерживает [стандартные](language-support.md#standard-voices), [нейронные](language-support.md#text-to-speech) и [пользовательские голоса](how-to-customize-voice-font.md) Майкрософт.

## <a name="get-started-with-the-voice-tuning-tool"></a>Начало работы со средством настройки голоса

Прежде чем начать тонкую настройку вывода текста в речь с помощью средства настройки голосовой связи, необходимо выполнить следующие действия.

1. Создайте [бесплатную учетная запись Майкрософт](https://account.microsoft.com/account) , если у вас его еще нет.
2. Создайте [бесплатную учетную запись Azure](https://azure.microsoft.com/free/) , если у вас ее еще нет. Щелкните **запустить бесплатно**и создайте новую учетную запись Azure с помощью учетная запись Майкрософт.

3. Создайте подписку на службы Speech Services в портал Azure. Доступны пошаговые инструкции по [созданию ресурса речи](https://docs.microsoft.com/azure/cognitive-services/speech-service/get-started#create-a-speech-resource-in-azure) .
   >[!NOTE]
   >При создании ресурса речи в портал Azure сведения о расположении Azure должны совпадать с регионом TTS. Голосовое по нейронного TTS поддерживает подмножество расположений Azure. Полный список поддержки см. в разделе [регионы](regions.md#text-to-speech).

   >[!NOTE]
   >Прежде чем использовать службу, необходимо создать ключ F0 или S0 в портал Azure. Настройка голоса **не** поддерживает [30-дневную бесплатную пробную версию ключа](https://docs.microsoft.com/azure/cognitive-services/speech-service/get-started?branch=release-build-cogserv-speech-services#free-trial).

4. Войдите на портал [настройки голоса](https://aka.ms/voicetuning) и подключите подписку на речевые службы. Выберите одну подписку на службы речи, а затем создайте проект.
5. Выберите **Новая настройка**. Затем выполните следующие действия.

   * Выберите **все подписки**.  
   * Выберите **Connect existing subscription** (Подключение к имеющейся подписке).  
     ![Подключите существующую подписку](./media/custom-voice/custom-voice-connect-subscription.png).
   * Введите ключ подписки Azure Speech Services, а затем нажмите кнопку **Добавить**. Ключи подписки доступны на портале настройки речи на [странице подписки](https://go.microsoft.com/fwlink/?linkid=2090458). Ключи можно также получить на панели "Управление ресурсами" в [портал Azure](https://portal.azure.com/).
   * Если вы планируете использовать более одной подписки на службы речи, повторите эти действия для каждой подписки.

## <a name="customize-the-text-to-speech-output"></a>Настройка вывода текста в речь

Теперь, когда вы создали учетные записи и связались с подпиской, можно приступить к настройке вывода текста в речь.

1. Выберите голосовое.
2. Введите текст, который требуется изменить.
3. Прежде чем вносить изменения, воспроизводите звук, чтобы получить впечатление от выходных данных.
4. Выберите слово или предложение, которое нужно уточнить, и начните экспериментировать с различными функциями на основе SSML.

>[!TIP]
> Дополнительные сведения о настройке SSML и настройке вывода голоса см. в разделе [язык разметки речи (SSML)](speech-synthesis-markup.md).

## <a name="limitations"></a>Ограничения

Настройка нейронных голосов немного отличается от настройки стандартных и пользовательских голосов.

* Интонатион не поддерживается для нейронных голосов.
* Функции «шаг» и «том» работают только с полными предложениями. Эти функции недоступны на уровне слова.
* Для ставки некоторые нейронные голоса можно настраивать на основе слов, а другие — для выбора целых предложений.

> [!TIP]
> Средство настройки речи предоставляет контекстные сведения о компонентах и настройке.
