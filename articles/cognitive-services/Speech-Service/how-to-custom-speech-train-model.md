# <a name="train-a-model-for-custom-speech"></a>Train a model for Custom Speech (Обучение модели для Пользовательского распознавания речи)

Обучение модели речи к тексту может повысить точность распознавания базовой модели Корпорации Майкрософт. Модель обучается с использованием транскрипций с маркировкой человека и связанного с ним текста. Эти наборы данных наряду с ранее загруженными аудиоданными данными используются для уточнения и обучения модели речи к тексту для распознавания слов, фраз, аббревиатур, имен и других терминов, связанных с продуктом. Чем больше наборов данных в домене, которые вы предоставляете (данные, связанные с тем, что скажут пользователи и что вы ожидаете распознать), тем точнее будет ваша модель, что приводит к улучшению распознавания. Имейте в виду, что, подавая несвязанные данные в обучение, вы можете уменьшить или повредить точность вашей модели.

## <a name="use-training-to-resolve-accuracy-issues"></a>Используйте обучение для решения проблем точности

Если вы столкнулись с проблемами распознавания с вашей моделью, использование канделях человека стенограммы и связанные с ними данные для дополнительного обучения может помочь повысить точность. Используйте эту таблицу, чтобы определить, какой набор данных использовать для решения проблемы(ы):

| Вариант использования | Тип данных |
| -------- | --------- |
| Улучшение точности распознавания в области лексики и грамматики, таких как медицинская терминология или ИТ-жаргон. | Похожий текст (предложения/высказывания) |
| Определите фонетическую и отображаемую форму слова или термина, который имеет нестандартное произношение, например названия продуктов или аббревиатуры. | Похожий текст (произношение) |
| Улучшение точности распознавания на говоря стилей, акценты, или конкретные фоновые шумы. | Аудио- и тесконизированные транскрипты с маркировкой человека |

> [!IMPORTANT]
> Если вы еще не загрузили набор данных, пожалуйста, просмотрите [Подготовьте и проверьте свои данные.](how-to-custom-speech-test-data.md) В этом документе содержатся инструкции по загрузке данных и рекомендации по созданию высококачественных наборов данных.

## <a name="train-and-evaluate-a-model"></a>Обучение и оценка модели

Первым шагом для обучения модели является загрузка обучаемых данных. Используйте [Подготовьте и проверьте свои данные](how-to-custom-speech-test-data.md) для пошаговых инструкций для подготовки транскрипций с маркировкой человека и связанного с ним текста (высказывания и произношения). После загрузки обучающие данные следуйте этим инструкциям, чтобы начать обучение модели:

1. Войти на [портал Custom Speech](https://speech.microsoft.com/customspeech).
2. Перейдите к **речи к тексту > пользовательских > речи обучение**.
3. Нажмите **Модель поезда**.
4. Далее, дайте вашей подготовки **имя** и **описание**.
5. Из меню **«Сценарий» и «Базовый модель»** выберите сценарий, который наилучшим образом подходит вашему домену. Если вы не уверены в том, какой сценарий выбрать, выберите **Общий**. Базовая модель является отправной точкой для обучения. Если у вас нет предпочтений, вы можете использовать последние.
6. На странице **данных обучения Select** выберите один или несколько аудио-наборов транскрипции, помеченных человеком, которые вы хотели бы использовать для обучения.
7. После завершения обучения можно выбрать тестирование точности на недавно обученной модели. Это необязательный шаг.
8. Выберите **Создать** для создания пользовательской модели.

Таблица обучения отображает новую запись, соответствующую этой недавно созданной модели. Таблица также отображает статус: Обработка, Успешное, Неудачное.

Вы можете проверить данные и оценить точность модели с помощью следующих документов:

- [Проверка данных](how-to-custom-speech-inspect-data.md)
- [Оценка данных](how-to-custom-speech-evaluate-data.md)
