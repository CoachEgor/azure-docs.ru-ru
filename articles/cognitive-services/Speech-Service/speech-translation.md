# <a name="what-is-speech-translation"></a>Что такое перевода речи

Перевод речи из служб Azure речи, позволяет в режиме реального времени, Многоязычная версия перевода аудиопотоков речи речь и преобразования речи в текст. С помощью Speech SDK приложений, средств и устройств доступны транскрипций источника и выходные данные преобразования для предоставленного аудио. Промежуточные результаты транскрипция и перевод возвращаются обнаруживается речь, и финал результаты могут быть преобразованы в синтезированную речь.

Механизм преобразования корпорации Майкрософт реализована на два различных подхода: статистического машинного перевода (SMT) и нейронный машинный перевод (NMT). SMT использует расширенные статистического анализа для оценки лучшие варианты перевода, заданном контексте несколько слов. С NMT нейронные сети используются для предоставления более точной и звучащие естественного переводы с помощью полного контекста предложений для перевода слов.

В настоящее время корпорация Майкрософт использует NMT для перевода на самых популярных языков. Все [языки, доступные для преобразования речи в речь](language-support.md#speech-translation), обрабатываются на платформе NMT. Преобразование речи в текст может выполняться на SMT или NMT в зависимости от конкретной пары языков. Когда NMT поддерживается целевым языком, полный перевод на основе NMT. Если на целевом языке не поддерживается NMT, перевод является гибридным NMT и SMT, с помощью английский как «сводная» между двумя языками.

## <a name="core-features"></a>Основные функции

Ниже приведены возможности, доступные через Speech SDK и REST API:

| Вариант использования | SDK | REST |
|----------|-----|------|
| Перевод речи в текст с помощью результатов распознавания. | Да | Нет |
| Преобразование речи в речь. | Да | Нет |
| Промежуточные результаты распознавания и перевода. | Да | Нет |
