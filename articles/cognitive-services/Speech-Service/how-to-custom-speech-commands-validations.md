# <a name="how-to-add-validations-to-custom-command-parameters-preview"></a>Как: Добавление валидаций к параметрам custom Command (Предварительный просмотр)

В этой статье вы узнаете, как добавить проверки к параметрам и подскажут для коррекции.

## <a name="prerequisites"></a>Предварительные требования

Вы должны завершить шаги в следующих статьях:

- [Быстрый запуск: Создайте пользовательский командный заказ (Предварительный просмотр)](./quickstart-custom-speech-commands-create-new.md)
- [Быстрый запуск: Создайте пользовательскую команду с параметрами (Предварительный просмотр)](./quickstart-custom-speech-commands-create-parameters.md)

## <a name="create-a-settemperature-command"></a>Создание команды SetTemperature

Чтобы продемонстрировать проверку, давайте создадим новую команду, позволяющую пользователю устанавливать температуру.

1. Откройте ранее созданное приложение Custom Commands в [студии speech Studio](https://speech.microsoft.com/)
1. Создание нового **командного setTemperature**
1. Добавление параметра для целевой температуры
1. Добавить проверку параметра температуры
   > [!div class="mx-imgBorder"]
   > ![Добавление проверки диапазона](media/custom-speech-commands/validations-add-temperature.png)

   | Параметр           | Рекомендуемое значение                                          | Описание                                                                                      |
   | ----------------- | -------------------------------------------------------- | ------------------------------------------------------------------------------------------------ |
   | name              | температура;                                              | Описательное имя для параметра команды                                                    |
   | Обязательно          | Да                                                     | Чекбокс, указывающий, требуется ли значение для этого параметра перед завершением команды |
   | Шаблон ответа | "- Какая температура вы хотели бы?"                     | Запрос для запроса значения этого параметра, когда он не известен                              |
   | Тип              | Number                                                   | Тип параметра, например число, строка или время даты                                      |
   | Проверка        | Мин Значение: 60, Максимальная стоимость: 80                             | Для параметров числа допустимый диапазон значений для параметра                             |
   | Шаблон ответа | "- Извините, я могу установить только между 60 и 80 градусов"      | Попросите запросить обновленное значение, если проверка не удается                                       |

1. Добавить несколько примеров предложений

   ```
   set the temperature to {Temperature} degrees
   change the temperature to {Temperature}
   set the temperature
   change the temperature
   ```

1. Добавление правила завершения для подтверждения результата

   | Параметр    | Рекомендуемое значение                                           | Описание                                        |
   | ---------- | --------------------------------------------------------- | -------------------------------------------------- |
   | Имя правила  | Сообщение с подтверждением                                      | Имя, описывающее цель правила          |
   | Условия | Требуемый параметр - Температура                          | Условия, определяющие, когда правило может быть запущено    |
   | Действия    | SpeechResponse - "- Ок, установка на "Температура" градусов " | Действие, которое необходимо предпринять, когда условие правила истинно |

> [!TIP]
> Этот пример использует речевой ответ для подтверждения результата. Например, о завершении командования действиями клиента [см.: Как: Выполнять команды на клиенте с помощью речи SDK (Предварительный просмотр)](./how-to-custom-speech-commands-fulfill-sdk.md)

## <a name="try-it-out"></a>Попробуйте продукт

Выберите панель тестирования и попробуйте несколько взаимодействий.

- Вход: Установите температуру до 72 градусов
- Выход: "Хорошо, установка до 72 градусов"

- Вход: Установите температуру до 45 градусов
- Выход: "Извините, я могу установить только между 60 и 80 градусов"
- Вход: сделать его 72 градусов вместо
- Выход: "Хорошо, установка до 72 градусов"
