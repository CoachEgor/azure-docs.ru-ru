# <a name="device-roobo-smart-audio-dev-kit-v2"></a>Устройство: Roobo Smart Аудио Dev Kit v2

Эта статья предоставляет устройство конкретных информации для Roobo Smart Аудио Dev Kit2.

## <a name="set-up-the-development-kit"></a>Настройка набора средств разработки

1. Комплект средств разработки содержит два разъема micro USB. Левый разъем предназначен для питания комплекта средств разработки и обозначен надписью "Power" (Питание) на рисунке ниже. Правый разъем предназначен для управления и обозначен на рисунке надписью "Debug" (Отладка). 
    ![Подключение комплекта разработки](media/speech-devices-sdk/roobo-v2-connections.png)
1. С помощью кабеля micro USB подключите разъем питания комплекта средств разработки к компьютеру или адаптеру питания, чтобы обеспечить питание комплекта средств разработки. Под верхней панелью должен засветиться зеленый индикатор питания.
1. Чтобы управлять набором разработки, подключите порт отладки к компьютеру с помощью второго микро-кабель USB. Для обеспечения надежной связи необходимо использовать высококачественный кабель.
1. Ориентируйте свой набор разработки Circularly - вертикально, с микрофонами, обращенными к потолку, как показано выше


## <a name="development-information"></a>Информация о развитии

Для получения дополнительной информации о разработке см. [Roobo development guide](http://dwn.roo.bo/server_upload/ddk/ROOBO%20Dev%20Kit-User%20Guide.pdf)

## <a name="audio-recordplay"></a>Аудиозапись/воспроизведение

Аудиооперации DDK2 могут выполняться следующим образом:
* Используйте библиотеки с открытым исходным кодом ALSA и их приложения.
* Используйте интерфейс appmainprog для разработки приложений. DDK2 аудио - связанные рамки программного обеспечения использует стандартные рамки ALSA, вы можете использовать libasound. Таким образом, чтобы разработать программное обеспечение непосредственно. Таким образом, вы можете использовать alSA в записи и воспроизведения непосредственно для записи и воспроизведения звука.
