# <a name="device-roobo-smart-audio-dev-kit-v2"></a>Устройство: Рубо Smart Audio Dev Kit v2

В этой статье содержатся сведения об устройствах для Рубо Smart Audio dev Kit2.

## <a name="set-up-the-development-kit"></a>Настройка набора средств разработки

1. Комплект средств разработки содержит два разъема micro USB. Левый разъем предназначен для питания комплекта средств разработки и обозначен надписью "Power" (Питание) на рисунке ниже. Правый разъем предназначен для управления и обозначен на рисунке надписью "Debug" (Отладка). 
    ![подключение пакета разработчика](media/speech-devices-sdk/roobo-v2-connections.png)
1. С помощью кабеля micro USB подключите разъем питания комплекта средств разработки к компьютеру или адаптеру питания, чтобы обеспечить питание комплекта средств разработки. Под верхней панелью должен засветиться зеленый индикатор питания.
1. Для управления пакетом средств разработки подключите порт отладки к компьютеру, используя второй разъем Micro-USB. Для обеспечения надежной связи важно использовать высококачественный кабель.
1. Расположить комплект разработки циклически в вертикальном расположении с микротелефонами, как показано выше


## <a name="development-information"></a>Сведения о разработке

Дополнительные сведения о разработке см. в разделе [Рубо Development Guide](http://dwn.roo.bo/server_upload/ddk/ROOBO%20Dev%20Kit-User%20Guide.pdf).

## <a name="audio-recordplay"></a>Звуковая запись/воспроизведение

Операции аудио DDK2 можно выполнять следующими способами.
* Используйте библиотеки с открытым исходным кодом АЛСА и их приложения.
* Используйте интерфейс аппмаинпрог для разработки приложений. В DDK2ной программной инфраструктуре, связанной с аудио, используется стандартная АЛСА Framework, можно использовать либасаунд. Для разработки программного обеспечения напрямую. Поэтому вы можете использовать АЛСА арекорд и Аплай непосредственно для записи и воспроизведения звука.
