В этом кратком руководстве показано, как с помощью интерфейса командной строки службы "Речь" распознать речь, записанную в аудиофайл, и выполнить транскрибирование. С помощью интерфейса командной строки службы "Речь" можно без усилий выполнять распространенные задачи распознавания речи, например транскрибирование бесед. После однократной настройки интерфейс командной строки службы "Речь" поможет в интерактивном режиме преобразовывать в текст поступающую с микрофона или из файлов речь с использованием пакетного сценария.
