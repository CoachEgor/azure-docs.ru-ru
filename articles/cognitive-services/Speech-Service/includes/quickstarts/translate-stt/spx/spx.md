## <a name="run-the-speech-cli"></a>Запуск интерфейса командной строки службы "Речь"

Теперь вы готовы запустить интерфейс командной строки службы "Речь" для преобразования речи в текст на другом языке.

В командной строке перейдите в каталог, в котором содержится двоичный файл интерфейса командной строки службы "Речь", и введите следующее:

```bash
spx translate --microphone --target de-DE
```

Интерфейс командной строки службы "Речь" преобразует фразу на естественном английском языке в печатный текст на немецком языке.
Нажмите ВВОД для остановки средства.

> [!NOTE]
> По умолчанию интерфейс командной строки службы "Речь" использует английский язык. Вы можете выбрать другой язык [из таблицы преобразования речи в текст](../../../../language-support.md).
> Например, для распознавания речи на японском добавьте `--source ja-JP`.
