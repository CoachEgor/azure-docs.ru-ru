Для создания объекта конфигурации службы "Речь" требуется указать регион и ключ подписки на ресурсы службы "Речь". Объект конфигурации нужен для создания экземпляра объекта распознавателя речи.

Экземпляр распознавателя предоставляет несколько способов распознавания речи. В этом примере речь распознается один раз. С помощью этой функции служба "Речь" узнает, что для распознавания отправляется одна фраза и что после идентификации фразы необходимо остановить распознавание речи. После получения результата код выведет в консоль причину остановки распознавания.

> [!TIP]
> В пакете SDK службы "Речь" по умолчанию используется язык `en-us` для распознавания речи. Дополнительные сведения о выборе исходного языка см. в статье [Указание исходного языка для преобразования речи в текст](../../../how-to-specify-source-language.md).
