Замените значения `YourSubscriptionKey` и `YourServiceRegion` фактическими значениями из ресурса службы "Речь".

- Перейдите на <a href="https://portal.azure.com/" target="_blank">портал Azure <span class="docon docon-navigate-external x-hidden-focus"></span></a> и откройте раздел ресурса службы "Речь".
- Выберите слева раздел **Ключи**, в нем вы увидите два ключа подписки.
    - Используйте один из них в качестве замены значения `YourSubscriptionKey`.
- Выберите слева раздел **Обзор**, запишите имя региона и сопоставьте его с <a href="https://aka.ms/speech/sdkregion" target="_blank">идентификатором региона <span class="docon docon-navigate-external x-hidden-focus"></span></a>.
    - Используйте **идентификатор региона** для замены значения `YourServiceRegion`, например: `"westus"` для **Западная часть США**.
