Вы можете выполнить синтез речи из текста с помощью пакета SDK службы "Речь" для Swift и Objective-C.

## <a name="prerequisites"></a>Предварительные требования

В приведенных ниже примерах предполагается, что у вас есть учетная запись Azure и подписка на службу "Речь". Если у вас нет учетной записи и подписки, [попробуйте службу "Речь" бесплатно](../../../overview.md#try-the-speech-service-for-free).

## <a name="install-speech-sdk-and-samples"></a>Установка пакета SDK и примеров для службы "Речь"

[Пакет SDK для службы "Речь" из Cognitive Services](https://github.com/Azure-Samples/cognitive-services-speech-sdk) включает примеры, написанные на Swift и Objective-C для iOS и Mac. Щелкните ссылку, чтобы просмотреть инструкции по установке для каждого примера:

* [Синтез речи в macOS с помощью Objective-C](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/objectivec/macos/text-to-speech)
* [Синтез речи в macOS с помощью Swift](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/swift/macos/text-to-speech)
* [Синтез речи в iOS с помощью Objective-C](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/objectivec/ios/text-to-speech)
* [Синтез речи в iOS с помощью Swift](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/swift/ios/text-to-speech)
* [Дополнительные примеры для Objective-C в iOS](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/samples/objective-c/ios)

Мы также предоставляем [онлайн-справку по пакету SDK службы "Речь" для Objective-C](https://docs.microsoft.com/objectivec/cognitive-services/speech/).
