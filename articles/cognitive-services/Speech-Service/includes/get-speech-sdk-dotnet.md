#### <a name="c-nuget-package"></a>Пакет NuGet

SDK речи .NET может быть установлен из **.NET Core CLI** со следующей `dotnet add` командой.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

SDK речи .NET может быть установлен от `Install-Package` **менеджера пакетов** со следующей командой.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
