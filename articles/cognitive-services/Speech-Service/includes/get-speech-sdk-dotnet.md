#### <a name="c-nuget-package"></a>Пакет NuGet C#

Пакет SDK для .NET для распознавания речи можно установить из **.NET Core CLI** с помощью следующей `dotnet add` команды.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

Пакет SDK для .NET для распознавания речи можно установить из **диспетчера пакетов** с помощью следующей `Install-Package` команды.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
