# <a name="get-started-with-custom-voice"></a>Начало работы с настраиваемыми голосовыми моделями

[Custom Voice](https://aka.ms/customvoice) — это набор онлайн-инструментов, которые позволяют создать узнаваемый, единственный в своем роде голос для вашего бренда. Все, что нужно, чтобы начать работу являются несколько аудио файлов и связанных транскрипций. Следуйте ссылкам ниже, чтобы начать создавать пользовательский текст к речи опыт.

## <a name="whats-in-custom-voice"></a>Что в пользовательских голос?

Перед тем, как начать с Custom Voice, вам понадобится учетная запись Azure и подписка на услугу speech. После создания учетной записи вы можете подготовить данные, обучить и протестировать модели, оценить качество голоса и в конечном итоге развернуть пользовательскую голосовую модель.

На приведенной ниже диаграмме показаны шаги по созданию пользовательской голосовой модели с помощью [портала Custom Voice.](https://aka.ms/customvoice) Используйте ссылки, чтобы узнать больше.

![Диаграмма архитектуры пользовательского голоса](media/custom-voice/custom-voice-diagram.png)

1. [Подписаться и создать проект](#set-up-your-azure-account) - Создайте учетную запись Azure и создайте подписку на службу речевой связи. Эта единая подписка дает вам доступ к речевому тексту, тексту к речи, речевому переводу и порталу Custom Voice. Затем, используя подписку на услугу Speech, создайте свой первый проект Custom Voice.

2. [Загрузить данные](how-to-custom-voice-create-voice.md#upload-your-datasets) - Загрузить данные (аудио и текст) с помощью портала Custom Voice или custom Voice API. С портала можно исследовать и оценивать показатели произношения и соотношения сигнала к шуму. Для получения дополнительной информации [см.](how-to-custom-voice-prepare-data.md)

3. [Обучите свою модель](how-to-custom-voice-create-voice.md#build-your-custom-voice-model) — используйте данные для создания пользовательской голосовой модели «текст к речи». Вы можете обучать модель на разных языках. После тренировки проверьте модель, и если вы удовлетворены результатом, вы можете развернуть модель.

4. [Развертывание модели](how-to-custom-voice-create-voice.md#create-and-use-a-custom-voice-endpoint) - Создайте пользовательскую конечную точку для голосовой модели текста в речи и используйте ее для синтеза речи в продуктах, инструментах и приложениях.

## <a name="set-up-your-azure-account"></a>Настройка учетной записи Azure

Прежде чем вы сможете использовать портал Custom Speech для создания пользовательской модели, требуется подписка на обслуживание речевых услуг. Следуйте этим инструкциям, чтобы создать подписку службы речевой речи в Azure. Если у вас нет учетной записи Azure, вы можете подписаться на новую.  

После создания учетной записи Azure и подписки на услуги speech необходимо войти на портал Custom Voice и подключить подписку.

1. Получите ключ подписки на службу speech на портале Azure.
2. Войти на [портал Custom Voice](https://aka.ms/custom-voice).
3. Выберите подписку и создайте речевой проект.
4. Если вы хотите переключиться на другую подписку на речь, используйте значок винтика, расположенный в верхней навигации.

> [!NOTE]
> Служба Custom Voice НЕ поддерживает 30-дневный бесплатный пробный ключ. Прежде чем воспользоваться службой, необходимо создать ключ F0 или S0 в Azure.

## <a name="how-to-create-a-project"></a>Создание проекта

Такие материалы, как данные, модели, тесты и конечные точки, организуются в **проекты** на портале Custom Voice. Каждый проект специфичен для страны/языка и пола голоса, который вы хотите создать. Например, вы можете создать проект для женского голоса для чат-ботов вашего колл-центра, которые используют английский язык в США (en-US).

Чтобы создать свой первый проект, выберите вкладку **Text-to-Speech/Custom Voice,** а затем нажмите **New Project.** Следуйте инструкциям, предоставленным мастером для создания проекта. После создания проекта вы увидите четыре вкладки: **данные,** **обучение,** **тестирование**и **развертывание.** Используйте ссылки, приведенные в [следующих шагах,](#next-steps) чтобы узнать, как использовать каждую вкладку.

> [!IMPORTANT]
> Портал [Custom Voice](https://aka.ms/custom-voice) был недавно обновлен! Если вы создали предыдущие данные, модели, тесты и опубликованные конечные точки на портале CRIS.ai или с помощью AI, необходимо создать новый проект на новом портале для подключения к этим старым объектам.
