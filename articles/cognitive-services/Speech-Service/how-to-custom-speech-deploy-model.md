# <a name="deploy-a-custom-model"></a>Развертывание пользовательской модели

После отправки и проверять данные, вычисляется точность и обучение пользовательскую модель, вы можете развернуть настраиваемую конечную точку для использования с приложений, средств и продуктов. В этом документе вы узнаете, как создать и развернуть конечную точку с помощью речи пользовательского портала.

## <a name="create-a-custom-endpoint"></a>Создание пользовательской конечной точки

Чтобы создать настраиваемую конечную точку, выберите **развертывания** меню настраиваемое преобразование речи в верхней части страницы. Если это первое, вы заметите, что существуют конечные точки, не перечисленные в таблице. После создания конечной точки, эта страница используется для отслеживания каждого развернутую конечную точку.

Затем выберите **добавить конечную точку** и введите **имя** и **описание** для пользовательской конечной точке. Затем выберите пользовательской модели, который вы хотите связать с этой конечной точкой. На этой странице можно также включить ведение журнала. Ведение журнала позволяет отслеживать трафик для конечной точки. Если отключен, трафик не будет сохранен.

![Развертывание модели](./media/custom-speech/custom-speech-deploy-model.png)

> [!NOTE]
> Не забудьте принять условия использования и сведения о ценах.

Далее выберите **Создать**. Это действие позволит вернуться на **развертывания** страницы. Таблица теперь содержит запись, которая соответствует вашей пользовательской конечной точки. Состояние конечной точки отображает его текущее состояние. Может занять до 30 минут для создания экземпляра новой конечной точки с помощью пользовательских моделей. При изменении состояния развертывания для **завершить**, конечная точка готова к использованию.

После развертывания конечной точки, отображается имя конечной точки в виде ссылки. Щелкните ссылку, чтобы отобразить сведения, относящиеся к конечной точке, например ключа конечной точки, URL-адрес конечной точки и примеры кода.

## <a name="view-logging-data"></a>Просмотр ведения журнала данных

Ведение журнала данных, доступных для скачивания в разделе **конечной точки > сведения о**.
