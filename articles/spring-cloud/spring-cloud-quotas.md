# <a name="quotas-and-service-plans-for-azure-spring-cloud"></a>Квоты и планы обслуживания для Azure Веснного облака

Все службы Azure устанавливают ограничения по умолчанию и квоты для ресурсов и компонентов.  В течение периода действия предварительной версии Azure Веснное облако предлагает только один план обслуживания.

В этой статье описываются квоты службы, предлагаемые в течение текущего периода предварительной версии.

## <a name="azure-spring-cloud-service-tiers-and-quotas"></a>Уровни облачной службы и квоты Azure весны

В течение периода действия предварительной версии Azure Веснное облако предлагает только один уровень служб.

Ресурс | Сумма
------- | -------
vCPU | 4\.
Память | 8 ГБ
Подписка Azure пружины Cloud | 1
Экземпляры облачной службы Azure весны по регионам на одну подписку | 2
Всего экземпляров приложений на экземпляр облачной службы Azure весны | 50
Всего экземпляров приложения на пружинное приложение | 20
Постоянные тома | 10 x 50 ГБ

При достижении квоты вы получите сообщение об ошибке 400, которое считывает: "квота превышает лимит *подписки подписки в* *регионе региона, в котором создана облачная служба Azure" Весна*".

## <a name="next-steps"></a>Дальнейшие действия

Вы можете повысить лимит нескольких ограничений и квот. Если ресурс требует увеличения, [Создайте запрос в службу поддержки](https://docs.microsoft.com/azure/azure-supportability/how-to-create-azure-support-request).
