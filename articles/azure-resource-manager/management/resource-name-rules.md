# <a name="naming-rules-and-restrictions-for-azure-resources"></a>Правила и ограничения именования для ресурсов Azure

В этой статье кратко излагаются правила именования и ограничения для ресурсов Azure. Рекомендации о том, как назвать ресурсы, [см.](/azure/cloud-adoption-framework/ready/azure-best-practices/naming-and-tagging)

Имена ресурсов нечувствительны, если конкретно не указано в столбце допустимых символов.

В следующих таблицах термин «буквенное» относится к:

* **a** через **z** (нижние буквы)
* **A** через **буквы (верхние** буквы)
* **От 0** до **9** (числа)
