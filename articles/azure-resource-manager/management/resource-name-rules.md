# <a name="naming-rules-and-restrictions-for-azure-resources"></a>Правила и ограничения именования для ресурсов Azure

В этой статье перечислены правила именования и ограничения для ресурсов Azure. Рекомендации по именам ресурсов см. в разделе [Рекомендуемые правила именования и добавления тегов](/azure/cloud-adoption-framework/ready/azure-best-practices/naming-and-tagging).

Имена ресурсов не учитывают регистр, если только они не указаны в столбце допустимые символы.

В следующих таблицах термин буквенно-цифровых символов относится к:

* **от a** до **z** (строчные буквы)
* **От A** до **Z** (прописные буквы)
* от **0** до **9** (числа)
