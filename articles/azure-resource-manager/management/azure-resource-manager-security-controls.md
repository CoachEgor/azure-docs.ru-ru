# <a name="security-controls-for-azure-resource-manager"></a>Элементы управления безопасностью для менеджера ресурсов Azure

В этой статье документируются элементы управления безопасностью, встроенные в диспетчер ресурсов Azure.

[!INCLUDE [Security controls Header](../../../includes/security-controls-header.md)]

## <a name="data-protection"></a>Защита данных

| Контроль безопасности | Да/нет | Примечания |
|---|---|--|
| Шифрование на стороне сервера в состоянии покоя: клавиши, управляемые Корпорацией Майкрософт | Да |  |
| Шифрование в пути (например, шифрование ExpressRoute, шифрование VNet и шифрование VNet-VNet)| Да | HTTPS/TLS. |
| Шифрование на стороне сервера в состоянии покоя: ключи, управляемые клиентом (BYOK) | Недоступно | Менеджер ресурсов Azure не хранит контент клиента, а только контролирует данные. |
| Шифрование уровня столбцов (Службы данных Azure)| Да | |
| Вызовы API в зашифрованном виде| Да | |

## <a name="network"></a>Сеть

| Контроль безопасности | Да/нет | Примечания |
|---|---|--|
| Поддержка конечных точек обслуживания| нет | |
| Поддержка инъекций VNet| Да | |
| Поддержка сетевой изоляции и брандмауэра| нет |  |
| Принудительная поддержка туннелирования| нет |  |

## <a name="monitoring--logging"></a>Мониторинг & лесозаготовки

| Контроль безопасности | Да/нет | Примечания|
|---|---|--|
| Поддержка мониторинга Azure (аналитика журналов, информация о приложениях и т.д.)| нет | |
| Управление и управление самолетом лесозаготовки и аудит| Да | Журналы активности разоблачают все операции записи (PUT, POST, DELETE), выполняемые на ваших ресурсах; просмотреть [журналы действий View для аудита действий на ресурсах.](view-activity-logs.md) |
| Регистрация и аудит плоскости данных| Недоступно | |
