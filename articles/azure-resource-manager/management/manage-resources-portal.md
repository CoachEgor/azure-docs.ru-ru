## <a name="open-resources"></a>Открытые ресурсы

Ресурсы Azure организуются службами Azure и группами ресурсов. Следующие процедуры показывают, как открыть учетную запись хранения под названием **mystorage0207**. Виртуальная машина находится в группе ресурсов под названием **mystorage0207rg**.

Чтобы открыть ресурс по типу службы:

1. Войдите на [портал Azure](https://portal.azure.com).
2. В левом стеле выберите службу Azure. В этом случае **учетные записи хранилища**.  Если вы не видите в списке службы, выберите **все службы,** а затем выберите тип службы.

    ![открытый лазурный ресурс на портале](./media/manage-resources-portal/manage-azure-resources-portal-open-service.png)

3. Выберите ресурс, который вы хотите открыть.

    ![открытый лазурный ресурс на портале](./media/manage-resources-portal/manage-azure-resources-portal-open-resource.png)

    Учетная запись хранилища выглядит следующим:

    ![открытый лазурный ресурс на портале](./media/manage-resources-portal/manage-azure-resources-portal-open-resource-storage.png)

Чтобы открыть ресурс по группе ресурсов:

1. Войдите на [портал Azure](https://portal.azure.com).
2. В левой панели выберите **группы ресурсов,** чтобы перечислить ресурс в группе.
3. Выберите ресурс, который вы хотите открыть. 

## <a name="manage-resources"></a>Управление ресурсами

На портале для конкретного ресурса можно просмотреть параметры управления.

![управлять ресурсами Azure](./media/manage-resources-portal/manage-azure-resources-portal-manage-resource.png)

На скриншоте показаны варианты управления виртуальной машиной Azure. Вы можете выполнять такие операции, как запуск, перезагрузка и остановка виртуальной машины.

## <a name="delete-resources"></a>Удаление ресурсов

1. Откройте ресурс на портале. Для шагов [см.](#open-resources)
2. Выберите **Удалить**. На следующем скриншоте показаны варианты управления виртуальной машиной.

    ![удалить лазурный ресурс](./media/manage-resources-portal/manage-azure-resources-portal-delete-resource.png)
3. Введите имя ресурса для подтверждения удаления, а затем выберите **Удалить**.

Для получения дополнительной информации о том, как диспетчер [Azure Resource Manager resource group deletion](delete-resource-group.md)ресурсов Azure заказывает удаление ресурсов, см.

## <a name="move-resources"></a>Перемещение ресурсов

1. Откройте ресурс на портале. Для шагов [см.](#open-resources)
2. Выберите **Переместить**. На следующем скриншоте показаны варианты управления учетной записью хранилища.

    ![переместить лазурный ресурс](./media/manage-resources-portal/manage-azure-resources-portal-move-resource.png)
3. Выберите **Переместить в другую группу ресурсов** или **перейдите на другую подписку** в зависимости от ваших потребностей.

Дополнительные сведения см. в статье [Перемещение ресурсов в новую группу ресурсов или подписку](move-resource-group-and-subscription.md).

## <a name="lock-resources"></a>Блокировка ресурсов

Блокировка предотвращает случайное исключение или изменение критически важных ресурсов, таких как подписка Azure, группа ресурсов или ресурс. 

1. Откройте ресурс на портале. Для шагов [см.](#open-resources)
2. Выберите **замки**. На следующем скриншоте показаны варианты управления учетной записью хранилища.

    ![блокировка лазурного ресурса](./media/manage-resources-portal/manage-azure-resources-portal-lock-resource.png)
3. Выберите **Добавить,** а затем указать свойства блокировки.

Для получения дополнительной информации смотрите [ресурсы блокировки с менеджером ресурсов Azure.](lock-resources.md)

## <a name="tag-resources"></a>Добавление тегов к ресурсам

Метки помогают логически организовать группу ресурсов и ресурсы. 

1. Откройте ресурс на портале. Для шагов [см.](#open-resources)
2. Нажмите **Теги**. На следующем скриншоте показаны варианты управления учетной записью хранилища.

    ![Тег лазурный ресурс](./media/manage-resources-portal/manage-azure-resources-portal-tag-resource.png)
3. Укажите свойства тегов, а затем выберите **Сохранить**.

Для получения [информации см. «Использование тегов для организации ресурсов Azure.](tag-resources.md#portal)

## <a name="monitor-resources"></a>Мониторинг ресурсов

При открытии ресурса портал представляет графики и таблицы по умолчанию для мониторинга этого типа ресурса. На следующем скриншоте показаны графики для виртуальной машины:

![мониторинг лазурного ресурса](./media/manage-resources-portal/manage-azure-resources-portal-monitor-resource.png)

Вы можете выбрать значок штифта в правом верхнем углу графиков, чтобы прикрепить график к приборной панели. Чтобы узнать о том, как работать с панелями мониторинга, ознакомьтесь с разделом [Создание панелей мониторинга и предоставление общего доступа к ним на портале Azure](../../azure-portal/azure-portal-dashboards.md).
