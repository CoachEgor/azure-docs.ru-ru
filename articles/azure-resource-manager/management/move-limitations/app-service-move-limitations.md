# <a name="move-guidance-for-app-service-resources"></a>Руководство по перемещению ресурсов службы приложений

В этой статье описываются действия по перемещению ресурсов службы приложений. Существуют особые требования к перемещению ресурсов службы приложений в новую подписку.

## <a name="move-across-subscriptions"></a>Перемещение между подписками

При перемещении веб-приложения по подпискам применяются следующие рекомендации.

- В целевой группе ресурсов не должно быть существующих ресурсов службы приложений. Ресурсы службы приложений:
    - Веб-приложения
    - Планы службы приложений
    - отправленные или импортированные SSL-сертификаты;
    - Среды службы приложений
- Все ресурсы службы приложений в группе ресурсов необходимо перемещать вместе. Обратите внимание, что среды службы приложений нельзя перемещать в новую группу ресурсов и в новую подписку.
- Можно переместить сертификат, привязанный к сети, без удаления привязок SSL при условии, что сертификат перемещен вместе со всеми остальными ресурсами в группе ресурсов.
- Ресурсы службы приложений можно перемещать только из группы ресурсов, в которой они были созданы. Если ресурс службы приложений больше не находится в исходной группе ресурсов, переместите его обратно в исходную группу ресурсов. Затем переместите ресурс по подпискам.

Если вы не помните исходную группу ресурсов, ее можно найти с помощью диагностики. Для веб-приложения выберите **Диагностика и решение проблем**. Затем выберите **Конфигурация и управление**.

![Выбор диагностики](./media/app-service-move-limitations/select-diagnostics.png)

Выберите **параметры миграции**.

![Выбор параметров миграции](./media/app-service-move-limitations/select-migration.png)

Выберите параметр для рекомендуемых действий, чтобы переместить веб-приложение.

![Выбор рекомендуемых шагов](./media/app-service-move-limitations/recommended-steps.png)

Вы увидите Рекомендуемые действия, которые необходимо выполнить перед перемещением ресурсов. Эти сведения включают в себя исходную группу ресурсов для веб-приложения.

![Рекомендации](./media/app-service-move-limitations/recommendations.png)
