# <a name="move-guidance-for-app-service-resources"></a>Перемещение руководства для ресурсов Службы приложений

В этой статье описаны шаги по перемещению ресурсов Службы приложений. Существуют конкретные требования для перемещения ресурсов Службы Приложений к новой подписке.

## <a name="move-across-subscriptions"></a>Перемещение по подписке

При перемещении web-приложения по подписке применяется следующее руководство:

- В целевой группе ресурсов не должно быть существующих ресурсов службы приложений. Ресурсы службы приложений:
    - Веб-приложения
    - Планы службы приложений
    - отправленные или импортированные SSL-сертификаты;
    - Среда службы приложений
- Все ресурсы службы приложений в группе ресурсов необходимо перемещать вместе. Обратите внимание, что среды службы приложений не могут быть перемещены в новую группу ресурсов или в новую подписку.
- Сертификат, привязанный к сети, можно переместить без удаляния привязок SSL, припойвайте сертификат со всеми другими ресурсами группы ресурсов.
- Ресурсы службы приложений можно перемещать только из группы ресурсов, в которой они были созданы. Если ресурс Службы приложений больше не входит в исходную группу ресурсов, переместите его обратно в исходную группу ресурсов. Затем переместите ресурс по подписке.

Если вы не помните исходную группу ресурсов, вы можете найти ее с помощью диагностики. Для веб-приложения выберите **диагностику и решение проблем.** Затем выберите **Конфигурацию и Управление.**

![Выбрать диагностику](./media/app-service-move-limitations/select-diagnostics.png)

Выберите **параметры миграции.**

![Выберите параметры миграции](./media/app-service-move-limitations/select-migration.png)

Выберите опцию для рекомендуемых шагов для перемещения веб-приложения.

![Выберите рекомендуемые шаги](./media/app-service-move-limitations/recommended-steps.png)

Вы видите рекомендуемые действия, которые необходимо предпринять перед перемещением ресурсов. Информация включает в себя исходную группу ресурсов для веб-приложения.

![Рекомендации](./media/app-service-move-limitations/recommendations.png)
