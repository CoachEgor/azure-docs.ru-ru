# <a name="streaming-catalog-views-transact-sql"></a>Представления каталога потоковой передачи (Transact-SQL)

В этом разделе описаны доступные представления каталога и функции, которые связаны с потоковой передачей T-SQL.
  
## <a name="in-this-section"></a>в этом разделе  
  
|Представление|Описание|  
|:---|:---|
|[sys.external_streams](sys-external-streams.md) |Возвращает строку для каждого объекта внешнего потока, созданного в области базы данных.|
|[sys.external_streaming_jobs](sys-external-streaming-jobs.md) |Возвращает строку для каждого задания внешней потоковой передачи, созданного в области базы данных.|
|[sys.external_job_streams](sys-external-job-streams.md)|Возвращает строку для каждого входного или выходного объекта внешнего потока, сопоставленного с заданием внешней потоковой передачи.|
