# <a name="geomatch-custom-rules-preview"></a>Пользовательские правила Geomatch (предварительный просмотр)

Пользовательские правила позволяют создавать индивидуальные правила в соответствии с точными потребностями приложений и политики безопасности. Теперь вы можете ограничить доступ к веб-приложениям по странам/регионам. Как и во всех пользовательских правилах, эта логика может быть усложнена другими правилами в соответствии с потребностями вашего приложения.

Чтобы создать пользовательское правило геофильтрации, просто выберите *геолокацию* в качестве типа матча, а затем выберите страну или страны, которые вы хотите разрешить/заблокировать из приложения. [Узнайте, как создать пользовательские правила в Powershell](configure-waf-custom-rules.md) и больше примеров пользовательских правил (создать-обычай-waf-rules.md) для получения дополнительной информации.

> [!IMPORTANT]
> Эта общедоступная предварительная версия предоставляется без соглашения об уровне обслуживания и не должна использоваться для производственных рабочих нагрузок. Некоторые функции могут не поддерживаться, иметь ограничения и быть доступными не во всех расположениях Azure. См. [дополнительные условия использования для предварительных версий Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).
