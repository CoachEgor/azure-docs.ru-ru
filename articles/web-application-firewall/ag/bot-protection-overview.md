# <a name="azure-web-application-firewall-on-azure-application-gateway-bot-protection-overview"></a>Обзор брандмауэра веб-приложения Azure на сайте шлюза приложений Azure: Общие сведения о защите

Примерно 20% всего трафика интернета приходят от неисправного программы-роботы. Они выполняют такие действия, как отходы, сканирование и поиск уязвимостей в веб-приложении. Если эти программы-роботы остановлены в брандмауэре веб-приложения (WAF), они не смогут атаковать вас. Они также не могут использовать ваши ресурсы и службы, такие как ваш интерфейс и другая базовая инфраструктура.

Вы можете включить управляемый набор правил защиты Bot для WAF, чтобы блокировать или регистрировать запросы от известных вредоносных IP-адресов. Эти IP-адреса передаются из канала Microsoft Threat Intelligence. Intelligent Security Graph поддерживает Microsoft Threat Intelligence и используется во множестве служб, включая Центр безопасности Azure.

> [!IMPORTANT]
> Набор правил защиты Bot в настоящее время находится в общедоступной предварительной версии и предоставляется с соглашением об уровне обслуживания предварительной версии. Некоторые функции могут не поддерживаться или их возможности могут быть ограничены. Дополнительные сведения см. в [дополнительных условиях использования Microsoft Azure предварительных версиях.](https://azure.microsoft.com/support/legal/preview-supplemental-terms/) 

## <a name="use-with-owasp-rulesets"></a>Использование с наборами правил OWASP

Можно использовать Bot-правил защиты с помощью набора правил OWASP (2.2.9, 3,0 и 3,1). В любой момент времени может использоваться только один набор правил OWASP. Набор правил для защиты Bot содержит дополнительное правило, которое отображается в его собственном наборе правил. Это название **Microsoft_BotManagerRuleSet_0**очень важно, и вы можете включить или отключить его, как и другие правила OWASP.

![Набор правил для Bot](../media/bot-protection-overview/bot-ruleset.png)

## <a name="ruleset-update"></a>Обновление набора правил

Список набора правил для предотвращения попыток для ленты с известными неправильными IP-адресами обновляется несколько раз в день из веб-канала аналитики угроз Майкрософт для поддержания синхронизации с программы-роботы. Веб-приложения постоянно защищаются даже при изменении векторов атаки Bot.
