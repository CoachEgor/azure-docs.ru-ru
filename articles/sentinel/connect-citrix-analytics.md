# <a name="connect-citrix-analytics-security"></a>Подключение Citrix Analytics (модуль безопасности) 

Интеграция Citrix Analytics (Security) с Azure Sentinel помогает экспортировать данные, проанализированные для рискованных событий, от Citrix Analytics (Security) в Azure Sentinel. Можно создавать пользовательские панели мониторинга, анализировать данные из других источников, а также данные из Citrix Analytics (Security) и создавать пользовательские рабочие процессы с использованием логических приложений для мониторинга и смягчения событий безопасности. 



## <a name="prerequisites"></a>Предварительные требования

- Прочитайте и напишите разрешения в рабочее пространство Azure Sentinel.

- Право на аналитику безопасности Citrix в облаке Citrix.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Подключение Azure Sentinel к аналитике Citrix (безопасность)

1. На портале Azure перейдите на**разъемы данных** **Azure Sentinel** > Data, а затем выберите разъем **Citrix Analytics (Security).**

2. Выберите **страницу Open connector**.

3. Следуйте инструкциям на странице **Citrix Analytics (Security).**
