# <a name="connect-your-zimperium-mobile-threat-defense-to-azure-sentinel"></a>Подключите вашу защиту мобильных угроз «Зимпериум» к Azure Sentinel


> [!IMPORTANT]
> Разъем данных системы «Зимпериум Мобильный угроза обороны» в Azure Sentinel в настоящее время находится в открытом доступе.
> Эта функция предоставляется без соглашения об уровне обслуживания, и она не рекомендуется для производственных нагрузок. Некоторые функции могут не поддерживаться или их возможности могут быть ограничены. Дополнительные сведения см. в статье [Дополнительные условия использования предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).



Разъем smperium Mobile Threat Defense позволяет подключить журнал угроз «Зимпериум» к журналу угроз Azure Sentinel для просмотра панелей мониторинга, создания пользовательских оповещений и улучшения расследования. Это дает вам более глубокое представление о мобильном ландшафте угроз вашей организации и расширяет возможности работы службы безопасности.

> [!NOTE]
> Данные будут храниться в географическом местоположении рабочего пространства, на котором работает Azure Sentinel.

## <a name="configure-and-connect-zimperium-mobile-threat-defense"></a>Налажить и подключить Киберним Мобильная защита угрозы

Компания «Зимпериум Мобильные угрозы обороны» может интегрировать и экспортировать журналы непосредственно в **Azure Sentinel.**

1. На портале Azure Sentinel щелкните разъемы данных и выберите **систему защиты мобильных угроз «Зимпериум».**
2. Выберите **страницу Open connector**.
3. Следуйте инструкциям на странице разъема **мобильных угроз «Зимпериум»,** кратко о чем идет.
 1. В zConsole нажмите **«Управление»** на панели навигации.
 2. Щелкните вкладку **Integrations** (Интеграции).
 3. Нажмите кнопку **«Сообщение об угрозах»,** а затем нажмите кнопку **«Добавить интеграцию».**
 4. Создайте интеграцию, выбрав **Microsoft Azure Sentinel** из доступных интеграций и введите идентификатор рабочего пространства и основной ключ для подключения к Azure Sentinel.
 5. Также доступна возможность выбора уровня фильтра для данных об угрозе для нажатия на Azure Sentinel. 

4. Для получения дополнительной информации, пожалуйста, обратитесь к [порталу поддержки клиентов «Зимпериум».](https://support.zimperium.com)


## <a name="find-your-data"></a>Найти свои данные

После успешного соединения данные отославается в журнале Analytics в соответствии с ZimperiumThreatLog_CL и ZimperiumMitigationLog_CL CustomLogs.

Чтобы использовать соответствующую схему в журнале Analytics для защиты мобильных угроз, поиск ZimperiumThreatLog_CL и ZimperiumMitigationLog_CL.


## <a name="validate-connectivity"></a>Проверка подключения

Это может занять более 20 минут, пока ваши журналы начинают появляться в журнале Analytics.
