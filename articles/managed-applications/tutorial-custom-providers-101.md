# <a name="create-custom-actions-and-resources-in-azure"></a>Создание пользовательских действий и ресурсов в Azure

Настраиваемые поставщики позволяют настраивать рабочие процессы в Azure. Настраиваемый поставщик представляет собой контракт между Azure и `endpoint`. Чтобы включить новые развертывания и управление возможностями он позволяет добавлять новые пользовательские API в Azure Resource Manager. В этом учебнике описывается простой процесс добавления нового действия и ресурса в Azure и как выполнить их интеграцию.

Это руководство разделено на следующие разделы:

- Настройка Функций Azure для настраиваемых поставщиков Azure
- Создание конечной точки RESTful для настраиваемых поставщиков
- Создание и использование настраиваемых поставщиков
