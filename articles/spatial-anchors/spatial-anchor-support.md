# <a name="azure-spatial-anchors-support-options"></a>Варианты поддержки Azure пространственных привязки

## <a name="open-a-tech-support-ticket"></a>Откройте запрос в службу технической поддержки

Чтобы открыть запрос в службу технической поддержки на портале Azure для привязки пространственных Azure:

1. С помощью [портала Azure](https://azure.microsoft.com/account/) открыть, выберите значок справки в верхней строке меню, а затем выберите **Справка и поддержка** кнопки. 

![Azure портала справки и поддержки](./media/spatial-anchor-support.png)

1. С помощью и поддержка открыт, выберите страницу **+ создать запрос в службу поддержки**.

![Azure портала новый запрос в службу поддержки](./media/spatial-anchor-support2.png)

1. При выполнении поля запрос в службу поддержки: 

- Тип проблемы. Технические требования
- Служба: Пространственные привязки

![Поля билет поддержка портала Azure](./media/spatial-anchor-support3.png)
