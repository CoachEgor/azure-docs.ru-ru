# <a name="support-matrix-for-backup-of-sap-hana-databases-on-azure-vms"></a>Матрица поддержки для резервного копирования баз данных SAP HANA на виртуальных машинах Azure

Azure Backup поддерживает резервное копирование баз данных SAP HANA в Azure. В этой статье перечислены поддерживаемые сценарии и ограничения, имеющиеся при использовании Azure Backup для резервного копирования SAP HANA баз данных на виртуальных машинах Azure.

> [!NOTE]
> Теперь периодичность резервного копирования журнала может быть не менее 15 минут. Резервные копии журналов начинают работать только после успешного полного резервного копирования базы данных.
