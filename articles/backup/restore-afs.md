# <a name="restore-azure-file-shares"></a>Восстановление акций файлов Azure

В этой статье объясняется, как использовать портал Azure для восстановления всей совместной части файла или определенных файлов из точки восстановления, созданной [резервным копированием Azure.](https://docs.microsoft.com/azure/backup/backup-overview)

В этой статье вы узнаете, как выполнять следующие задачи.

* Восстановить полный общий объем файлов Azure.
* Восстановление отдельных файлов или папок.
* Отслеживайте состояние операции восстановления.

## <a name="steps-to-perform-a-restore-operation"></a>Шаги для выполнения операции восстановления

Чтобы выполнить операцию восстановления, выполните следующие действия.

### <a name="select-the-file-share-to-restore"></a>Выберите раздел файла для восстановления

1. На [портале Azure](https://portal.azure.com/)откройте хранилище служб восстановления, используемое для настройки резервной системы для общего файла.

1. В панели обзора выберите **элементы резервного копирования** в разделе **Защищенные элементы.**

    ![Выберите элементы резервного копирования](./media/restore-afs/backup-items.png)

1. После выбора **элементов резервного копирования**рядом с панелью обзора открывается новая панель, в которых перечислены все типы управления резервным копированием.

    ![Типы управления резервного копирования](./media/restore-afs/backup-management.png)

1. В панели **элементов резервного копирования** под **типом управления резервным копированием**выберите **хранилище Azure (Azure Files)**. Вы увидите список всех файлов и соответствующие учетные записи хранения, подкрепленные с помощью этого хранилища.

    ![Список всех файловых акций](./media/restore-afs/file-shares.png)

1. Из списка разделов файлов Azure выберите долю файла, для которой требуется выполнить операцию восстановления.

### <a name="full-share-recovery"></a>Полное восстановление доли

Эту опцию восстановления можно использовать для восстановления полной доли файла в исходном месте или в другом месте.

1. Выберите опцию **«Доли восстановления»** в панели **элемента резервного копирования,** которая появится после выбора доли файла для восстановления в шаге 5 [раздела «Выберите файл» для восстановления](#select-the-file-share-to-restore) раздела.

   ![Выберите раздел восстановления](./media/restore-afs/restore-share.png)

1. После выбора **доли восстановления**панель **«Восстановление»** открывается меню **«Точка восстановления»,** в которое отображается список точек восстановления, доступных для выбранного раздела файла.

1. Выберите точку восстановления, которая требуется использовать для выполнения операции восстановления, и выберите **OK.**

    ![Выбор точки восстановления](./media/restore-afs/restore-point.png)

1. После выбора **OK**меню **панели восстановления** переключается на место **восстановления.** В **местоположении восстановления**укажите, где и как восстановить данные. Выберите один из следующих двух вариантов:

    * **Исходное местоположение**: Восстановление полного общего файла в том же месте, что и исходный источник.
    * **Альтернативное местоположение**: Восстановить полную долю файла в альтернативное местоположение и сохранить исходную долю файла как есть.

#### <a name="restore-to-the-original-location"></a>Восстановление исходного местоположения

1. Выберите **исходное местоположение** в качестве **назначения восстановления**и выберите, следует ли пропустить или перезаписать, если есть конфликты. После того как вы сделаете соответствующий выбор, выберите **OK**.

    ![Выберите исходное местоположение](./media/restore-afs/original-location.png)

1. Выберите **Восстановление,** чтобы начать операцию восстановления.

    ![Выберите Восстановление для начала](./media/restore-afs/click-restore.png)

#### <a name="restore-to-an-alternate-location"></a>Восстановление в альтернативном расположении

1. Выберите **альтернативное местоположение** в качестве **назначения восстановления.**
1. Выберите учетную запись хранения назначения, где требуется восстановить резервное содержимое из списка выпадающих данных **учетной записи хранения.**
1. Список выпадающих файлов **Select File Share** отображает доли файлов, присутствующие в выбранном вами в шаге 2. Выберите раздел файла, где требуется восстановить содержимое резервного копирования.
1. В поле **Folder Name** укажите имя папки, которое вы хотите создать в файле назначения, разделитесь с восстановленным содержимым.
1. Выберите, пропустить или перезаписать, если есть конфликты.
1. После ввода соответствующих значений во всех коробках выберите **OK**.

    ![Выберите альтернативное местоположение](./media/restore-afs/alternate-location.png)

1. Выберите **Восстановление,** чтобы начать операцию восстановления.

    ![Выберите Восстановление для начала](./media/restore-afs/click-restore.png)

### <a name="item-level-recovery"></a>Восстановление на уровне элементов

Эту опцию восстановления можно использовать для восстановления отдельных файлов или папок в исходном или альтернативном месте.

1. Выберите опцию **восстановления файла** в панели **элемента резервного копирования,** которая появится после выбора доли файла для восстановления в шаге 5 [раздела «Выберите файл» для восстановления](#select-the-file-share-to-restore) раздела.

    ![Выберите восстановление файла](./media/restore-afs/file-recovery.png)

1. После выбора **восстановления файла**панель **«Восстановление»** открывается меню **«Точка восстановления»,** в которое отображается список точек восстановления, доступных для выбранного раздела файла.

1. Выберите точку восстановления, которая требуется использовать для выполнения операции восстановления, и выберите **OK.**

    ![Выбор точки восстановления](./media/restore-afs/restore-point.png)

1. После выбора **OK**меню восстановления панели переключается на **место восстановления.** В **местоположении восстановления**укажите, где и как восстановить данные. Выберите один из следующих двух вариантов:

    * **Исходное местоположение**: Восстановление выбранных файлов или папок в ту же долю файла, что и исходный источник.
    * **Альтернативное местоположение**: Восстановление выбранных файлов или папок в альтернативном месте и сохранение исходного содержимого файла как есть.

#### <a name="restore-to-the-original-location"></a>Восстановление исходного местоположения

1. Выберите **исходное местоположение** в качестве **назначения восстановления**и выберите, следует ли пропустить или перезаписать, если есть конфликты.

    ![Исходное местоположение для восстановления уровня элемента](./media/restore-afs/original-location-item-level.png)

1. Выберите **файл,** чтобы выбрать файлы или папки, которые вы хотите восстановить.

    ![Выберите файл](./media/restore-afs/select-file.png)

1. После выбора **файла Select**File панель обмена файлами отображает содержимое выбранной для восстановления точки восстановления доли файла.

1. Выберите флажок, соответствующий файлу или папке, которую вы хотите восстановить, и **выберите select.**

    ![Выберите файл или папку](./media/restore-afs/select-file-folder.png)

1. Повторите шаги 2-4, чтобы выбрать несколько файлов или папок для восстановления.
1. После выбора всех элементов, которые вы хотите восстановить, выберите **OK**.

    ![После выбора всех элементов для восстановления выберите OK](./media/restore-afs/after-selecting-items.png)

1. Выберите **Восстановление,** чтобы начать операцию восстановления.

    ![Выберите Восстановление для начала](./media/restore-afs/click-restore.png)

#### <a name="restore-to-an-alternate-location"></a>Восстановление в альтернативном расположении

1. Выберите **альтернативное местоположение** в качестве **назначения восстановления.**
1. Выберите учетную запись хранения назначения, где требуется восстановить резервное содержимое из списка выпадающих данных **учетной записи хранения.**
1. Список выпадающих файлов **Select File Share** отображает доли файлов, присутствующие в выбранном вами в шаге 2. Выберите раздел файла, где требуется восстановить содержимое резервного копирования.
1. В поле **Folder Name** укажите имя папки, которое вы хотите создать в файле назначения, разделитесь с восстановленным содержимым.
1. Выберите, пропустить или перезаписать, если есть конфликты.
1. Выберите **файл,** чтобы выбрать файлы или папки, которые вы хотите восстановить.

    ![Выберите элементы для восстановления в альтернативное местоположение](./media/restore-afs/restore-to-alternate-location.png)

1. При выборе **файла Select File**панель обмена файлами отображает содержимое выбранной для восстановления точки восстановления доли файла.
1. Выберите флажок, соответствующий файлу или папке, которую вы хотите восстановить, и **выберите select.**

    ![Выберите место восстановления](./media/restore-afs/recovery-destination.png)

1. Повторите шаги с 6 по 8, чтобы выбрать несколько файлов или папок для восстановления.
1. После выбора всех элементов, которые вы хотите восстановить, выберите **OK**.

    ![Выберите OK после выбора всех файлов](./media/restore-afs/after-selecting-all-items.png)

1. Выберите **Восстановление,** чтобы начать операцию восстановления.

## <a name="track-a-restore-operation"></a>Отслеживание операции восстановления

Когда вы запускаете операцию восстановления, служба Backup создает задание для отслеживания операции. Azure Backup отображает уведомления о задании на портале. Для просмотра операций для задания выберите гиперссылку уведомлений.

![Выберите гиперссылку уведомлений](./media/restore-afs/notifications-link.png)

Вы также можете контролировать ход восстановления из хранилища служб восстановления:

1. Откройте хранилище служб восстановления, откуда вы сработали операции восстановления.
1. В панели обзора выберите **задания резервного копирования** в разделе **Мониторинг,** чтобы увидеть состояние операций, работающих с различными рабочими нагрузками.

    ![Выберите вакансии резервного копирования](./media/restore-afs/backup-jobs.png)

1. Выберите имя рабочей нагрузки, которое соответствует доле файла, чтобы просмотреть более подробную информацию об операции восстановления, например, **передаваемые данные** и **количество восстановленных файлов.**

    ![Посмотреть восстановленные детали](./media/restore-afs/restore-details.png)
