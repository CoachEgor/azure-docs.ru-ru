# <a name="manage-accounts-on-the-azure-vmware-solutions-avs-portal"></a>Управление учетными записями на портале решений Azure VMware (AVS)

При создании службы AVS она создает учетную запись в AVS. Учетная запись связана с подпиской Azure, в которой находится служба. Все пользователи с ролями владельца и участника в подписке имеют доступ к порталу AVS. Идентификатор подписки Azure и идентификатор клиента, связанные со службой AVS, находятся на странице учетных записей.

Чтобы управлять учетными записями на портале AVS, [откройте портал](access-cloudsimple-portal.md) и выберите **учетная запись** в боковом меню.

Выберите **Сводка** , чтобы просмотреть сведения о конфигурации AVS вашей компании. Отобразится текущая емкость конфигурации облака, включая количество частных облаков AVS, общее хранилище, конфигурацию кластера vSphere, число узлов и число ядер вычислений. Если текущая конфигурация не соответствует всем потребностям, в нее добавляется ссылка для приобретения дополнительных узлов.

## <a name="email-alerts"></a>Оповещения по электронной почте

Вы можете добавить адреса электронной почты всех пользователей, которых вы хотите уведомлять об изменениях в конфигурации частного облака AVS.

1. В области **Дополнительные оповещения по электронной почте** щелкните **Добавить новый**.
2. Введите адрес электронной почты.
3. Нажмите клавишу Return.  

Чтобы удалить запись, нажмите кнопку **X**.

## <a name="avs-operator-access"></a>Доступ к оператору AVS

Параметр доступа "оператор" позволяет AVS помочь в устранении неполадок, разрешая инженеру службы поддержки войти на портал AVS. Этот параметр включен по умолчанию. Все действия, выполняемые инженером службы поддержки при входе в учетную запись клиента, записываются и становятся доступными для проверки на странице **действия** > **аудита** .

Щелкните переключатель **доступ к оператору AVS** , чтобы включить или отключить доступ.
