# <a name="manage-accounts-on-the-azure-vmware-solution-by-cloudsimple-portal"></a>Управление учетными записями на решении Azure VMware с помощью портала CloudSimple

При создании сервиса CloudSimple он создает учетную запись в CloudSimple. Учетная запись связана с подпиской Azure, в которой находится служба. Все пользователи с функциями владельца и автора в подписке имеют доступ к порталу CloudSimple. Идентификатор подписки Azure и идентификатор клиента, связанный с службой CloudSimple, можно найти на странице Счетов.

Для управления учетными записями на портале CloudSimple [получите доступ к порталу](access-cloudsimple-portal.md) и выберите **учетную запись** в меню стороны.

Выберите **Резюме** для просмотра информации о конфигурации CloudSimple вашей компании. Отображается текущая емкость конфигурации облака, включая количество частных облаков, общее хранилище, конфигурацию кластера vSphere, количество узлов и количество вычислительных ядер. Ссылка включена для покупки дополнительных узлов, если текущая конфигурация не отвечает всем вашим потребностям.

## <a name="email-alerts"></a>Оповещения по электронной почте

Вы можете добавить адреса электронной почты любых людей, которых вы хотели бы уведомить об изменениях в конфигурации Private Cloud.

1. В области **дополнительных оповещений по электронной почте** нажмите **«Добавить новое».**
2. Введите адрес электронной почты.
3. Возвращение прессы.  

Чтобы удалить запись, нажмите **X**.

## <a name="cloudsimple-operator-access"></a>Доступ оператора CloudSimple

Настройка доступа оператора позволяет CloudSimple помочь вам с устранением неполадок, разрешив инженеру службы поддержки войти на ваш портал CloudSimple.  Настройка включена по умолчанию. Все действия, выполняемые инженером службы поддержки при входе в вашу учетную запись, записываются и доступны для вашего обзора на странице**аудита** **деятельности.** > 

Нажмите **на доступ оператора CloudSimple, который позволяет** переключиться, чтобы включить или выключить доступ.
