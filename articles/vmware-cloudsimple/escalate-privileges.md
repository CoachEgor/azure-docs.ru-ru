# <a name="escalate-cloudsimple-privileges-to-perform-administrative-functions-in-private-cloud-vcenter"></a>Эскалация привилегий CloudSimple для выполнения административных функций в Private Cloud vCenter

Подход CloudSimple к привилегиям разработан для предоставления пользователям vCenter привилегий, необходимых для выполнения обычных операций. В некоторых случаях пользователю могут потребоваться дополнительные привилегии для выполнения конкретной задачи.  Вы можете обострять привилегии пользователя vCenter SSO в течение ограниченного периода времени.

Причины эскалации привилегий могут включать в себя следующие:

* Конфигурация источников идентификации
* Управление пользователями
* Удаление распределенной группы портов
* Установка решений vCenter (например, приложения для резервного копирования)
* Создание учетных записей служб

> [!WARNING]
> Действия, предпринимаемые в обостренном привилегированном состоянии, могут негативно повлиять на вашу систему и привести к тому, что ваша система станет недоступной. Выполняйте только необходимые действия в период эскалации.

С портала CloudSimple [эскалация привилегий](escalate-private-cloud-privileges.md) для локального пользователя CloudOwner на vCenter SSO.  Вы можете обострить привилегию удаленного пользователя только в том случае, если дополнительный поставщик идентификационных данных настроен на vCenter.  Эскалация привилегий включает в себя добавление выбранного пользователя в группу встроенных администраторов vSphere.  Только один пользователь может иметь эскалацию привилегий.  Если вам необходимо обострить привилегии другого пользователя, сначала деэскалировать привилегии текущих пользователей.

Пользователи из дополнительных источников идентификации должны быть добавлены в качестве членов группы CloudOwner.

> [!CAUTION]
> Новые пользователи должны быть добавлены только в *Cloud-Owner-Group*, *Cloud-Global-Cluster-Admin-Group,* *Cloud-Global-Storage-Admin-Group,* *Cloud-Global-Network-Admin-Group* или, *Cloud-Global-VM-Admin-Group.*  Пользователи, добавленные в группу *администраторов,* будут удалены автоматически.  Только учетные записи служб должны быть добавлены в группу *администраторов,* а учетные записи службы не должны использоваться для входиных в веб-uI vSphere.

В период эскалации CloudSimple использует автоматизированный мониторинг с соответствующими уведомлениями оповещения для выявления любых непреднамеренных изменений в среде.
