# <a name="quickstart---consume-vmware-vms-on-azure"></a>Краткое руководство. Использование виртуальных машин VMware в Azure

Чтобы создать виртуальную машину на портале Azure, используйте шаблоны виртуальных машин, CloudSimple администратор включил для вашей подписки. Эти шаблоны виртуальных Машин находятся на инфраструктура VMware.

## <a name="cloudsimple-vm-creation-on-azure-requires-a-vm-template"></a>Создание виртуальной Машины CloudSimple в Azure требуется шаблон виртуальной Машины

Создание виртуальной машины в частном облаке из vCenter пользовательского интерфейса. Чтобы создать шаблон, следуйте инструкциям в [клонировать виртуальную машину в шаблон в веб-клиенте vSphere](https://docs.vmware.com/en/VMware-vSphere/6.7/com.vmware.vsphere.vm_admin.doc/GUID-FE6DE4DF-FAD0-4BB0-A1FD-AFE9A40F4BFE.html). Store шаблона виртуальной Машины в частном облаке vCenter.

## <a name="create-a-virtual-machine-in-the-azure-portal"></a>Создание виртуальной машины на портале Azure

1. Выбор пункта **Все службы**.

2. Поиск **CloudSimple виртуальные машины**.

3. Щелкните **Добавить**.

    ![Создание виртуальной машины CloudSimple](media/create-cloudsimple-virtual-machine.png)

4. Введите основные сведения, щелкните **далее: размер**.

    ![Создание виртуальной машины CloudSimple — основы](media/create-cloudsimple-virtual-machine-basic-info.png)

    | Поле | Описание |
    | ------------ | ------------- |
    | Подписка | Подписка Azure, связанные с частным облаком.  |
    | Группа ресурсов | Группа ресурсов, к которому будет назначен виртуальной Машине. Можно выбрать существующую группу или создайте новую. |
    | ИМЯ | Имя для идентификации виртуальной Машины.  |
    | Расположение | Регион Azure, в котором размещается эта виртуальная машина.  |
    | Частное облако | CloudSimple частного облака, где вы хотите создать виртуальную машину. |
    | Пул ресурсов | Сопоставить пул ресурсов для виртуальной Машины. Выберите доступные пулы ресурсов. |
    | vSphere шаблона | vSphere шаблона для виртуальной Машины.  |
    | Имя пользователя | Имя пользователя администратора виртуальной Машины (для шаблонов Windows)|
    | Пароль |  Пароль администратора виртуальной Машины (для шаблонов Windows). |
    | Подтвердите пароль | Подтвердите пароль |

5. Выберите количество ядер и объем памяти виртуальной Машины и нажмите кнопку **далее: конфигурации**. Установите флажок, если вы хотите предоставить полной виртуализации ЦП для гостевой операционной системы. Приложения, требующие аппаратную виртуализацию можно запускать на виртуальных машинах без двоичную трансляцию или паравиртуализации. Дополнительные сведения см. в статье VMware <a href="https://docs.vmware.com/en/VMware-vSphere/6.5/com.vmware.vsphere.vm_admin.doc/GUID-2A98801C-68E8-47AF-99ED-00C63E4857F6.html" target="_blank">предоставлять VMware оборудования виртуализации</a>.

    ![Создание виртуальной машины CloudSimple - размер](media/create-cloudsimple-virtual-machine-size.png)

6. Настройка сетевых интерфейсов и диски, как описано в следующих таблицах и нажмите кнопку **проверки + создать**.

    ![Создание виртуальной машины CloudSimple - конфигурации](media/create-cloudsimple-virtual-machine-configurations.png)

    Для сетевых интерфейсов, нажмите кнопку **добавить сетевой интерфейс** и настройте следующие параметры.
    
    | Управление | Описание |
    | ------------ | ------------- |
    | ИМЯ | Введите имя для определения интерфейса.  |
    | Сеть | Выберите из списка настроенный порт распределенной группы в vSphere вашего частного облака.  |
    | Адаптер | Выберите адаптер vSphere в списке доступных типов, которые настроены для виртуальной Машины. Дополнительные сведения см. в статье базы знаний VMware <a href="https://kb.vmware.com/s/article/1001805" target="_blank">Выбор сетевой адаптер для виртуальной машины</a>. |
    | Включение при загрузке | Выберите, следует ли включить оборудовании сетевого Адаптера при загрузке виртуальной Машины. По умолчанию используется **включить**. |

    Для дисков, нажмите кнопку **добавить диск** и настройте следующие параметры.

    | Элемент | Описание | 
    | ------------ | ------------- | 
    | ИМЯ | Введите имя для идентификации диска.  | 
    | Размер | Выберите один из доступных размеров.  | 
    | SCSI-контроллер | Выберите SCSI-контроллер для диска.  |
    | Режим | Определяет, как диск участвует в моментальных снимках. Выберите один из следующих вариантов: <br> -Независимые от постоянных: Все данные, записанные на диск записываются без возможности восстановления.<br> -Независимые непостоянный: Изменения записываются на диск отменяются выключение питания или сброс виртуальной машины.  Независимый режим непостоянный позволяет всегда перезапустите виртуальную Машину в том же состоянии. Дополнительные сведения см. в разделе <a href="https://docs.vmware.com/en/VMware-vSphere/6.5/com.vmware.vsphere.vm_admin.doc/GUID-8B6174E6-36A8-42DA-ACF7-0DA4D8C5B084.html" target="_blank">документации по VMware</a>.

7. После завершения проверки, проверьте параметры и нажмите кнопку **создать**. Чтобы внести изменения, щелкните вкладки в верхней или.

    ![Создание виртуальной машины CloudSimple - просмотрите](media/create-cloudsimple-virtual-machine-review.png)
