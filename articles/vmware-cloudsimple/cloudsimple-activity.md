# <a name="activity-management-overview"></a>Обзор управления деятельностью

CloudSimple отслеживает все действия, которые могут повлиять на функционирование вашей среды Private Cloud. Действия включают оповещения, события, задачи и действия аудита. Страницы [активности](monitor-activity.md) суммируют все текущие действия и позволяют просверлить дополнительные сведения.

## <a name="events"></a>События

События отслеживают активность пользователей и системы на портале CloudSimple. События показывают активность, связанную с определенным ресурсом, и серьезность воздействия.  Ознакомиться с событиями можно с портала CloudSimple.

## <a name="alerts"></a>видны узлы

Оповещения — это уведомления о каких-либо значительных действиях в среде CloudSimple. События, влияющие на выставление счетов или доступ пользователей, отображаются в виде предупреждений.  Вы можете подтвердить оповещения с портала CloudSimple.

## <a name="tasks"></a>Задания

Задачи отслеживают любую операцию пользователя, которая занимает более 30 секунд. Вы можете следить за ходом выполнения задачи с портала CloudSimple.  Для завершенных задач информация включает общее время завершения.

## <a name="audit"></a>Аудит

Записи аудита отслеживают операции пользователей. Аудиторские журналы содержат параметры, предусмотренные для операции пользователем.  Вы можете использовать журналы аудита для мониторинга активности пользователей для всех пользователей.
