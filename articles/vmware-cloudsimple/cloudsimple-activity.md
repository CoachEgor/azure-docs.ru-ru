# <a name="activity-management-overview"></a>Общие сведения об управлении действия

CloudSimple отслеживает все действия, которые могут повлиять на работу ваших частных облаков. Эти действия, включая оповещения, события, задачи и действия аудита. [Страницах действий](https://docs.azure.cloudsimple.com/activity/) суммировать все текущее действие и позволяют проводить детализацию углублением для получения дополнительных сведений.

## <a name="events"></a>События

События отслеживания пользователя и системные действия на портале CloudSimple.  События показаны действия, связанного с определенного ресурса и важности влияния.  Можно просмотреть события на портале CloudSimple.

## <a name="alerts"></a>Оповещения

Оповещения — это уведомления значительные действия, выполняемые в среде CloudSimple.  События, которые влияют на доступ выставления счетов или пользователя, отображаются в виде оповещений.  Можно принять вызове предупреждений с портала CloudSimple.

## <a name="tasks"></a>Задачи

Задачи записывать любые операции пользователя, который занимает более 30 секунд.  Вы можете отслеживать ход выполнения задачи на портале CloudSimple.  Выполнить задачи показывают общее время, необходимое для выполнения задачи.

## <a name="audit"></a>Аудит

Журналы аудита отслеживания операций, выполняемых пользователем.  Журналы содержат параметры, которые пользователь для операции.  Журналы аудита можно использовать для отслеживания активности пользователей для всех пользователей.
