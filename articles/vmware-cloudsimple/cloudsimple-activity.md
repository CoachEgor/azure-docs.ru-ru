# <a name="activity-management-overview"></a>Общие сведения об управлении действиями

Клаудсимпле отслеживает все действия, которые могут повлиять на функционирование частных облаков. Эти действия включают в себя предупреждения, события, задачи и действия аудита. [Страницы действий](https://docs.azure.cloudsimple.com/activity/) обобщаются все текущее действие и позволяют детализировать дополнительные сведения.

## <a name="events"></a>События

События регистрируют действия пользователя и системы на портале Клаудсимпле.  События отображают действие, связанное с определенным ресурсом, и серьезность влияния.  События можно просмотреть на портале Клаудсимпле.

## <a name="alerts"></a>Предупреждения

Оповещения — это уведомления о любых значительных действиях в среде Клаудсимпле.  События, влияющие на выставление счетов или доступ пользователей, отображаются как оповещения.  Созданные оповещения можно подтвердить на портале Клаудсимпле.

## <a name="tasks"></a>Задачи

Задачи отписывают любую операцию пользователя, выполнение которой занимает более 30 секунд.  Ход выполнения задачи можно отслеживать на портале Клаудсимпле.  Завершенные задачи показывают общее время, затраченное на завершение задачи.

## <a name="audit"></a>Аудит

Журналы аудита записывают операции, выполненные пользователем.  Журналы содержат параметры, предоставленные пользователем для операции.  Журналы аудита можно использовать для наблюдения за действиями пользователей для всех пользователей.
