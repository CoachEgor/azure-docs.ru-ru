# <a name="activity-management-overview"></a>Общие сведения об управлении действиями

AVS отслеживает все действия, которые могут повлиять на работоспособность среды частного облака AVS. К действиям относятся предупреждения, события, задачи и действия аудита. [Страницы действий](monitor-activity.md) обобщаются все текущее действие и позволяют детализировать дополнительные сведения.

## <a name="events"></a>Мероприятия

События регистрируют действия пользователя и системы на портале AVS. События отображают действие, связанное с определенным ресурсом, и серьезность влияния.  События можно просмотреть на портале AVS.

## <a name="alerts"></a>Оповещения

Оповещения — это уведомления о любых значительных действиях в среде AVS. События, влияющие на выставление счетов или доступ пользователей, отображаются как оповещения. Оповещения можно подтвердить на портале AVS.

## <a name="tasks"></a>Задачи

Задачи отписывают все операции пользователя, выполнение которых занимает более 30 секунд. Ход выполнения задачи можно отслеживать на портале AVS. Для выполненных задач сведения включают в себя общее время завершения.

## <a name="audit"></a>Аудит

Журналы аудита отслеживают операции пользователя. Журналы аудита содержат параметры, предоставленные пользователем для операции. Журналы аудита можно использовать для наблюдения за действиями пользователей для всех пользователей.
