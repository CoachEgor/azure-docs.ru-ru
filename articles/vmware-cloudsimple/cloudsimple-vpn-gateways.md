# <a name="vpn-gateways-overview"></a>Общие сведения о шлюзах VPN

VPN-шлюз используется для отправки зашифрованного трафика между CloudSimple регион, сеть в локальное расположение или компьютер через общедоступный Интернет.  Каждый регион может иметь только один VPN-шлюз. Однако к одному VPN-шлюзу можно создать несколько подключений. При создании нескольких подключений к одному VPN-шлюзу все VPN-туннели совместно используют доступную для этого шлюза пропускную способность.

CloudSimple имеются два вида VPN-шлюзов:

* Сеть сеть VPN-шлюза
* Шлюз типа "точка сеть"

## <a name="site-to-site-vpn-gateway"></a>VPN-шлюз типа "сеть — сеть";

Сеть сеть VPN-шлюз используется для отправки зашифрованного трафика между сетью CloudSimple регион и локальным центром обработки данных. Это подключение можно используйте для определения диапазона подсети/CIDR, для обмена данными между вашей локальной сетью и сетью CloudSimple регион.

VPN-шлюз позволяет использовать службы из локальной в вашем частном облаке и служб в частном облаке, из локальной сети.  CloudSimple предоставляет VPN-сервера на основе политики для подключения между вашей локальной сети.

Среди вариантов использования VPN типа "сеть сеть":

* Доступность vCenter частного облака с любой рабочей станции в вашей локальной сети.
* Использование локальной Active Directory в качестве источника удостоверений vCenter.
* Удобный передачи шаблоны виртуальных Машин, образы ISO и другие файлы из локальных ресурсов к vCenter, частного облака.
* Доступность рабочих нагрузок, выполняющихся в вашем частном облаке из вашей локальной сети.

![Топологии подключения сеть-сеть VPN](media/cloudsimple-site-to-site-vpn-connection.png)

> [!IMPORTANT]
> Необходимо установить для TCP MSS на 1078 байт или ниже. Или если ваши VPN-устройства не поддерживают фиксацию MSS, можно также задать MTU на интерфейсе туннеля 1118 байт вместо этого. 

### <a name="cryptographic-parameters"></a>Параметры шифрования

VPN-подключение сайт сайт использует следующие параметры шифрования по умолчанию для безопасного подключения.  При создании подключения из локальных VPN-устройства, используйте любой из следующих параметров, поддерживаемых VPN-шлюзу локальной.

#### <a name="phase-1-proposals"></a>Этап 1 предложения

| Параметр | Первый способ. | Второй способ. | Предложение 3 |
|-----------|------------|------------|------------|
| Версия IKE | IKEv1 | IKEv1 | IKEv1 |
| Шифрование | AES 128 | AES 256 | AES 256 |
| Хэш-алгоритм| SHA 256 | SHA 256 | SHA 1 |
| Группа Диффи Хелмана (группа Диффи-Хелмана) | 2 | 2 | 2 |
| Время существования | 28 800 сек | 28 800 сек | 28 800 сек |
| Размер данных | 4 ГБ | 4 ГБ | 4 ГБ |


#### <a name="phase-2-proposals"></a>Этап 2 предложения 

| Параметр | Первый способ. | Второй способ. | Предложение 3 |
|-----------|------------|------------|------------|
| Шифрование | AES 128 | AES 256 | AES 256 |
| Хэш-алгоритм| SHA 256 | SHA 256 | SHA 1 |
| Идеально подходит пересылки (PFS) (группы PFS) | Нет | Нет | Нет |
| Время существования | 1800 секунд | 1800 секунд | 1800 секунд |
| Размер данных | 4 ГБ | 4 ГБ | 4 ГБ |

## <a name="point-to-site-vpn-gateway"></a>VPN-шлюз типа "точка — сеть"

VPN-Подключение между сайтами используется для отправки зашифрованного трафика между сетью CloudSimple регион и клиентский компьютер.  Типа "точка сеть" — это самый простой способ получить доступ к сети частного облака, включая vCenter частного облака и виртуальные машины рабочих нагрузок.  Используйте VPN-подключения между сайтами, если вы подключаетесь к частному облаку, удаленно.
