# <a name="create-a-cloudsimple-private-cloud"></a>Создание частного облака Клаудсимпле

Частное облако — это изолированный стек VMware, который поддерживает узлы ESXi, vCenter, vSAN и НСКС. Управление частными облаками осуществляется через портал Клаудсимпле. Они имеют собственный сервер vCenter в собственном домене управления. Стек работает на выделенных узлах и изолированных аппаратных узлах без операционной системы.

Создание частного облака позволяет решать различные распространенные потребности в сетевой инфраструктуре.

* **Рост**. Если вы достигли точки обновления оборудования для существующей инфраструктуры, частное облако позволяет расширить, не требующее новых инвестиций в оборудование.

* **Быстрое расширение**. Если возникли какие-либо временные или незапланированные потребности в емкости, частное облако позволяет создать дополнительную емкость без задержки.

* **Повышенная защита**. С частным облаком из трех и более узлов вы получаете автоматическую избыточность и защиту высокого уровня доступности.

* **Долгосрочные потребности**в инфраструктуре. Если ваши центры обработки данных находятся в емкости или вы хотите изменить структуру, чтобы снизить затраты, частное облако позволяет выключать центры обработки данных и переходить на облачное решение, в то время как оно остается совместимым с корпоративными операциями.

При создании частного облака вы получаете один кластер vSphere и все виртуальные машины управления, созданные в этом кластере.

## <a name="access-the-cloudsimple-portal"></a>Доступ к порталу CloudSimple

Доступ к [порталу клаудсимпле](access-cloudsimple-portal.md).

## <a name="create-a-new-private-cloud"></a>Создание частного облака

1. Выбор пункта **Все службы**.
2. Выполните поиск по запросу **Клаудсимпле Services**.
3. Выберите службу Клаудсимпле, для которой нужно создать частное облако.
4. В разделе **Обзор**щелкните **создать частное облако** , чтобы открыть новую вкладку браузера для клаудсимпле Portal. При появлении запроса войдите с помощью учетных данных для входа в Azure.

    ![Создание частного облака из Azure](media/create-private-cloud-from-azure.png)

5. На портале Клаудсимпле укажите имя частного облака.
6. Выберите **Расположение** для частного облака.
7. Выберите **тип узла**в соответствии с тем, что вы приобрели в Azure.  Можно выбрать [параметр CS28 или CS36](cloudsimple-node.md#vmware-solution-by-cloudsimple-nodes-sku). Последний вариант включает максимальный объем вычислений и объем памяти.
8. Укажите **количество узлов**.  Для создания частного облака требуется по крайней мере три узла.

    ![Создание частного облака — основные сведения](media/create-private-cloud-basic-info.png)

9. Щелкните **Далее: Дополнительные параметры**.
10. Введите диапазон CIDR для подсетей vSphere/vSAN. Убедитесь, что диапазон CIDR не пересекается с какими-либо локальными или другими подсетями Azure (виртуальными сетями) или с подсетью шлюза.

    **Параметры диапазона CIDR:** /24,/23,/22 или/21. Диапазон CIDR/24 поддерживает до девяти узлов, а также диапазон CIDR/23 поддерживает до 41 узлов, а диапазон значений CIDR/22 и/21 поддерживает до 64 узлов (максимальное количество узлов в частном облаке).

    > [!IMPORTANT]
    > IP-адреса в диапазоне CIDR vSphere/vSAN зарезервированы для использования в инфраструктуре частного облака.  Не используйте этот IP-адрес в этом диапазоне на любой виртуальной машине.

11. Щелкните **Далее: Проверьте и создайте**.
12. Проверьте параметры. Если необходимо изменить какие бы то ни было параметры, нажмите кнопку **назад**.
13. Нажмите кнопку **Создать**.

Начинается процесс подготовки частного облака. Подготовка частного облака может занять до двух часов.

Инструкции по развертыванию существующего частного облака см. в разделе [расширение частного облака](expand-private-cloud.md).
