# <a name="avs-virtual-machines-overview"></a>Общие сведения о виртуальных машинах AVS

AVS позволяет управлять виртуальными машинами VMware (ВМ) из портал Azure. Кластер или пул ресурсов из кластера vSphere управляются с помощью Azure путем его сопоставления с подпиской.

Чтобы создать виртуальную машину AVS из Azure, необходимо наличие шаблона виртуальной машины в частном облаке "AVS". Шаблон используется для настройки операционной системы и приложений. Шаблон виртуальной машины можно зафиксировать в соответствии с политиками безопасности предприятия. С помощью шаблона можно создавать виртуальные машины, а затем использовать их из портал Azure, используя модель самообслуживания.

## <a name="benefits"></a>Преимущества

Виртуальные машины AVS от портал Azure предоставляют пользователям механизм самообслуживания для создания виртуальных машин VMware и управления ими.

* Создание виртуальной машины AVS в частном облаке "AVS"
* Управление свойствами виртуальной машины
  * Добавление и удаление дисков
  * Добавление и удаление сетевых карт
* Операции с питанием виртуальной машины AVS
  * Включение и выключение питания
  * Сброс виртуальной машины
* Удаление виртуальной машины
