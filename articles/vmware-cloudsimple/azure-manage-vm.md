# <a name="manage-your-cloudsimple-private-cloud-virtual-machines-in-azure"></a>Управление виртуальными машинами CloudSimple Private Cloud в Azure

Чтобы управлять виртуальными машинами, [созданными для облачного частного облака,](azure-create-vm.md)подпишитесь на [портал Azure.](https://portal.azure.com) Поиск и выбор виртуального (поиск по **всем службам** или **виртуальные машины** в боковом меню).

## <a name="control-virtual-machine-operation"></a>Управление виртуальной работой машины

Следующие элементы управления доступны на странице **Обзор** для выбранной виртуальной машины.

| Control | Описание |
| ------------ | ------------- |
| Подключение | Подключитесь к указанному VM.  |
| Запуск | Запустите указанный VM.  |
| Перезагрузить | Выключите, а затем включите указанный VM.  |
| Остановить | Выключите конкретный VM.  |
| Сбор | Захват изображения указанного VM, чтобы он мог быть использован в качестве изображения для создания других VM. Смотрите [Создать управляемое изображение обобщенного VM в Azure.](../virtual-machines/windows/classic/capture-image.md)   |
| Переместить | Перейдите на указанный VM.  |
| DELETE | Удалите указанный VM.  |
| Обновить | Обновление данных на дисплее.  |

### <a name="view-performance-information"></a>Просмотр сведений о производительности

На диаграммах в нижней части страницы **Обзор** представлены данные о производительности для выбранного интервала (последний час продлится 30 дней; по умолчанию — последний час). В пределах каждой диаграммы можно отображать числовые значения в любое время в интервале, перемещая курсор вперед и назад по диаграмме.

Отображаются следующие диаграммы.

| Item | Описание |
| ------------ | ------------- |
| Процессор (средний) | Среднее использование процессора в процентах по сравнению с выбранным интервалом.   |
| Сеть | Трафик в сети (MB) и из нее в течение выбранного интервала.  |
| Диск Байты | Общее количество данных, считаемых с диска и написанных на диск (МБ) за выбранный интервал.  |
| Операции диска | Средняя скорость операций диска (операции/секунда) в течение выбранного интервала. |

## <a name="manage-vm-disks"></a>Управление дисками виртуальной машины

Чтобы добавить VM-диск, откройте страницу **дисков** для выбранного VM. Чтобы добавить диск, нажмите **Добавить диск**. Настроите каждую из следующих настроек, введя или выбрав входящий параметр. Нажмите **Сохранить**.

   | Item | Описание |
   | ------------ | ------------- |
   | name | Введите имя для идентификации диска.  |
   | Размер | Выберите один из доступных размеров.  |
   | SCSI Controller (Контроллер SCSI) | Выберите контроллер SCSI. Доступные контроллеры различаются для различных поддерживаемых операционных систем.  |
   | Режим | Определяет, как диск участвует в моментальных снимках. Выберите один из следующих параметров. <br> - Независимый настойчивый: Все данные, записанные на диск, пишутся постоянно.<br> - Независимые, непостоянные: Изменения, написанные на диске, отбрасываются при отключении или сбросе виртуальной машины.  Этот режим позволяет всегда перезапускать VM в том же состоянии. Дополнительные сведения см. в [документации VMware](https://docs.vmware.com/en/VMware-vSphere/6.5/com.vmware.vsphere.vm_admin.doc/GUID-8B6174E6-36A8-42DA-ACF7-0DA4D8C5B084.html). |

Чтобы удалить диск, выберите его и нажмите **Удалить**.

## <a name="change-the-capacity-of-the-vm"></a>Изменение емкости ВМ

Чтобы изменить емкость VM, откройте страницу **Размера** для выбранного VM. Укажите любое из следующих, и нажмите **Сохранить**.

| Item | Описание |
| ------------ | ------------- |
| Число ядер | Количество ядер, назначенных VM.  |
| Виртуализация оборудования | Выберите флажок, чтобы виртуализация оборудования виртуализации гостевой ОС. Смотрите статью VMware [Expose VMware Аппаратная вспомогательная виртуализация](https://docs.vmware.com/en/VMware-vSphere/6.5/com.vmware.vsphere.vm_admin.doc/GUID-2A98801C-68E8-47AF-99ED-00C63E4857F6.html). |
| Размер памяти | Выберите объем памяти для выделения VM.  

## <a name="manage-network-interfaces"></a>Управление сетевыми интерфейсами

Чтобы добавить интерфейс, нажмите **Добавить сетевой интерфейс**. Настроили каждую из следующих настроек, введя или выбрав входящий параметр. Нажмите **Сохранить**.

   | Control | Описание |
   | ------------ | ------------- |
   | name | Введите имя для идентификации интерфейса.  |
   | Сеть | Выберите из списка настроенных сетей в вашем private Cloud vSphere.  |
   | Адаптер | Выберите адаптер vSphere из списка доступных типов, настроенных для VM. Для получения дополнительной информации смотрите статью базы знаний VMware [Выбор сетевого адаптера для виртуальной машины.](https://kb.vmware.com/s/article/1001805) |
   | Power on at Boot (Включать при загрузке) | Выберите, следует ли включить оборудование сетевого адаптера при загрузке виртуальной машины. По умолчанию этот параметр имеет значение **Enable** (Включить). |

Чтобы удалить сетевой интерфейс, выберите его и нажмите **Удалить**.
