# <a name="monitor-vmware-solution-by-cloudsimple-activity"></a>Мониторинг решения VMware с помощью деятельности CloudSimple

Журналы активности CloudSimple дают представление об операциях, выполненных на портале CloudSimple.  Список включает оповещения, события, задачи и аудит.  Используйте журналы активности, чтобы определить, кто, когда и какие операции были выполнены.  В журналы активности не содержится никаких операций чтения, выполненных пользователем.

## <a name="sign-in-to-azure"></a>Вход в Azure

Войти на портал Azure по адресу [https://portal.azure.com](https://portal.azure.com).

## <a name="access-the-cloudsimple-portal"></a>Доступ к порталу CloudSimple

Доступ к [порталу CloudSimple](access-cloudsimple-portal.md).

## <a name="activity-information"></a>Информация о деятельности

Чтобы получить доступ к страницам Activity, выберите **«Действие»** в боковом меню.

![Обзор страницы активности](media/activity-page-overview.png)

Чтобы просмотреть сведения о любом из действий на странице действия, выберите действие. Справа открывается панель деталей. Действия в панели зависят от вида деятельности. Нажмите **X,** чтобы закрыть панель.

Нажмите на заголовок столбца для сортировки дисплея.  Можно фильтровать столбцы для просмотра определенных значений.  Скачать отчет о деятельности, нажав на **Скачать как значок CSV.**

## <a name="alerts"></a>видны узлы

Оповещения — это уведомления о каких-либо значительных действиях в среде CloudSimple.  Оповещения включают события, влияющие на выставление счетов или доступ пользователей.

Чтобы подтвердить предупреждения и удалить их из списка, выберите один или несколько из списка и нажмите **«Подтверждение».**

Следующие столбцы информации доступны для оповещений. Нажмите на **столбцы edit** и выберите столбцы, которые вы хотите просмотреть.

| Столбец | Описание |
------------ | ------------- |
| Тип оповещения | Категория оповещения.|
| Time | Время оповещения произошло. |
| Severity | Значение оповещения.|
| Имя ресурса | Имя, присвоенное ресурсу, например имя частного облака. |
| Тип ресурса | Категория ресурсов: Частное облако, Облачный стойка. |
| Идентификатор ресурса | Идентификатор ресурса. |
| Описание | Описание того, что вызвало предупреждение. |
| Подтверждено | Указание на то, признается ли предупреждение. |

## <a name="events"></a>События

События показывают активность пользователей и систем на портале CloudSimple. Страница События перечисляет действия, связанные с определенным ресурсом, и серьезность воздействия.

Следующие столбцы информации доступны для оповещений. Нажмите на **столбцы edit** и выберите столбцы, которые вы хотите просмотреть.

| Столбец | Описание |
------------ | ------------- |
| Time | Дата и время события. |
| Тип события | Числовый код, идентифицирующие событие. |
| Severity | Тяжесть событий.|
| Имя ресурса | Имя, присвоенное ресурсу, например имя частного облака. |
| Тип ресурса | Категория ресурсов: Частное облако, Облачный стойка. |
| Описание | Описание того, что вызвало предупреждение. |

## <a name="tasks"></a>Задания

Задачи являются действиями Private Cloud, которые, как ожидается, займут 30 секунд или более. (Действия, которые, как ожидается, займет менее 30 секунд, сообщаются только как события.) Откройте страницы задач для отслеживания хода выполнения задач для вашего частного облака.

Следующие столбцы информации доступны для оповещений. Нажмите на **столбцы edit** и выберите столбцы, которые вы хотите просмотреть.

| Столбец | Описание |
------------ | ------------- |
| Идентификатор задачи | Уникальный идентификатор для задачи. |
| Операция | Действие, выполняемые задачей. |
| Пользователь | Пользователь, назначенный для выполнения задачи. |
| Имя ресурса | Имя, присвоенное ресурсу. |
| Тип ресурса | Категория ресурсов: Частное облако, Облачный стойка. |
| Идентификатор ресурса | Идентификатор ресурса. |
| Запуск | Время начала выполнения задачи. |
| Конец | Время окончания задачи. |
| Состояние | Текущее состояние задачи. |
| Время, прошедшее | Время выполнения задачи (если выполнено) или в настоящее время принимается (если оно выполняется). |
| Описание | Описание задачи. |

## <a name="audit"></a>Аудит

Записи аудита отслеживают активность пользователей. Вы можете использовать журналы аудита для мониторинга активности пользователей для всех пользователей.

Следующие столбцы информации доступны для оповещений. Нажмите на **столбцы edit** и выберите столбцы, которые вы хотите просмотреть.

| Столбец | Описание |
------------ | ------------- |
| Time | Время ввода аудита. |
| Операция | Действие, выполняемые задачей. |
| Пользователь | Пользователь, назначенный для задачи. |
| Имя ресурса | Имя, присвоенное ресурсу. |
| Тип ресурса | Категория ресурсов: Частное облако, Облачный стойка. |
| Идентификатор ресурса | Идентификатор ресурса. |
| Результат | Результат деятельности, такой как **Успех**. |
| Заехано время | Время для выполнения задачи. |
| Описание | Описание действия. |
