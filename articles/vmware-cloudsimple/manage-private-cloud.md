# <a name="manage-avs-private-cloud-resources-and-activities"></a>Управление ресурсами и действиями частного облака AVS

Частные облака AVS управляются на портале AVS. Проверьте состояние, доступные ресурсы, действия в частном облаке AVS и другие параметры на портале AVS.

## <a name="sign-in-to-azure"></a>Войдите в Azure

Войдите на портал Azure по адресу [https://portal.azure.com](https://portal.azure.com).

## <a name="access-the-avs-portal"></a>Доступ к порталу AVS

Доступ к [порталу AVS](access-cloudsimple-portal.md).

## <a name="view-the-list-of-avs-private-clouds"></a>Просмотр списка частных облаков AVS

На вкладке **частные облака AVS** на странице **ресурсы** ПЕРЕЧИСЛЕНЫ все частные облака AVS в вашей подписке. Сведения включают имя, число кластеров vSphere, расположение, текущее состояние частного облака AVS и сведения о ресурсах.

![Страница частного облака AVS](media/manage-private-cloud.png)

Выберите частное облако AVS для получения дополнительных сведений и действий.

## <a name="avs-private-cloud-summary"></a>Сводка частного облака AVS

Просмотрите подробную сводку по выбранному частному облаку AVS. Страница сводка включает DNS-серверы, развернутые в частном облаке AVS. Вы можете настроить перенаправление DNS с локальных DNS-серверов на DNS-серверы частного облака AVS. Дополнительные сведения о пересылке DNS см. в [статье Настройка DNS для разрешения имен из локального каталога AVS для частного облака](https://docs.azure.cloudsimple.com/on-premises-dns-setup/).

![Сводка частного облака AVS](media/private-cloud-summary.png)

### <a name="available-actions"></a>Доступные действия

* [Запустите клиент vSphere](https://docs.azure.cloudsimple.com/vsphere-access/). Доступ к vCenter для этого частного облака AVS.
* [Приобретение узлов](create-nodes.md). Добавьте узлы в это частное облако AVS.
* [Разверните узел](expand-private-cloud.md). Добавьте узлы в это частное облако AVS.
* **Обновить**. Обновите сведения на этой странице.
* **Удалить**. Частное облако AVS можно удалить в любое время. **Перед удалением убедитесь, что вы выполнили резервное копирование всех систем и данных.** При удалении частного облака AVS удаляются все виртуальные машины, конфигурация vCenter и данные. Нажмите кнопку **Удалить** в разделе Сводка для выбранного частного облака AVS. После удаления все данные частного облака AVS удаляются в безопасном, строго соответствующем очистки процессе.
* [Измените привилегии vSphere](escalate-private-cloud-privileges.md). Эскалировать свои привилегии в частном облаке AVS.

## <a name="avs-private-cloud-vlanssubnets"></a>Виртуальные ЛС и подсети в частном облаке AVS

Просмотрите список определенных виртуальных ЛС или подсетей для выбранного частного облака AVS. Список включает в себя виртуальные ЛС и подсети управления, созданные при создании частного облака AVS.

![Частное облако AVS — виртуальные ЛС и подсети](media/private-cloud-vlans-subnets.png) 

### <a name="available-actions"></a>Доступные действия

* [Добавьте виртуальные ЛС и подсети](https://docs.azure.cloudsimple.com/create-vlan-subnet/). Добавьте виртуальную ЛС или подмножество в это частное облако AVS.

Выберите виртуальную ЛС или подсеть для следующих действий
* [Присоединение таблицы брандмауэра](https://docs.azure.cloudsimple.com/firewall/). Подключите таблицу брандмауэра к этому частному облаку AVS.
* **Edit** (Изменение)
* **Удаление** (только определяемые пользователем виртуальные ЛС и подсети)

## <a name="avs-private-cloud-activity"></a>Действие частного облака AVS

Просмотрите следующие сведения о выбранном частном облаке AVS. Сведения о действии представляют собой отфильтрованный список всех действий для выбранного частного облака AVS. На этой странице отображается до 25 последних действий.

* Недавние оповещения
* Недавние события
* Недавние задачи
* Последний аудит

![Частное облако AVS — активность](media/private-cloud-activity.png)

## <a name="cloud-racks"></a>Облачные стойки

Облачные стойки — это стандартные блоки частного облака AVS. Каждая стойка предоставляет единицу мощности. AVS автоматически настраивает облачные стойки в соответствии с выбранными параметрами при создании или расширении частного облака AVS. Просмотрите полный список облачных стоек, включая частное облако AVS, которому назначено каждое из них.

![Частное облако AVS — облачные стойки](media/private-cloud-cloudracks.png)

## <a name="vsphere-management-network"></a>Сеть управления vSphere

Список ресурсов управления VMware и виртуальных машин, настроенных в частном облаке AVS. Сведения включают в себя версию программного обеспечения, полное доменное имя (FQDN) и IP-адрес ресурсов.

![Частное облако AVS — сеть управления vSphere](media/private-cloud-vsphere-management-network.png)
