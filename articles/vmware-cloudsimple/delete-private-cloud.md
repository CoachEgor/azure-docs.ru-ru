# <a name="delete-a-cloudsimple-private-cloud"></a>Удалить облачное частное облако

CloudSimple обеспечивает гибкость для удаления частного облака.  Частное облако состоит из одного или нескольких кластеров vSphere. Каждый кластер может иметь от 3 до 16 узлов. При удалении приватного облака все кластеры будут удалены.

## <a name="before-you-begin"></a>Перед началом

Удаление приватного облака удаляет все приватное облако.  Все компоненты приватного облака будут удалены.  Если вы хотите сохранить какие-либо данные, убедитесь, что вы резервное копирование данных на хранение на месте или хранилище Azure.

Компоненты частного облака включают в себя:

* Облачныепростые узлы
* Виртуальные машины
* Виртуальные ЛС и подсети
* Все пользовательские данные, хранящиеся в приватном облаке
* Все вложения правил брандмауэра в VLAN/Subnet

## <a name="sign-in-to-azure"></a>Вход в Azure

Войти на портал Azure по адресу [https://portal.azure.com](https://portal.azure.com).

## <a name="delete-a-private-cloud"></a>Удаление частного облака

1. [Доступ к порталу CloudSimple](access-cloudsimple-portal.md).

2. Откройте страницу **Ресурсов.**

3. Нажмите на приватное облако, но вы хотите удалить

4. На сводной странице нажмите **«Удалить».**

    ![Удаление частного облака](media/delete-private-cloud.png)

5. На странице подтверждения введите название частного облака и нажмите **Удалить**. 

    ![Удаление частного облака - подтвердите](media/delete-private-cloud-confirm.png)

Частное облако помечено для удаления.  Процесс удаления начинается через три часа и удаляет частное облако.

> [!CAUTION]
> Узлы должны быть удалены после удаления частного облака.  Измерение узлов будет продолжаться до тех пор, пока узлы не будут удалены из подписки.
