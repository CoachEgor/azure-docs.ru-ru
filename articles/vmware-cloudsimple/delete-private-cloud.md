# <a name="delete-an-avs-private-cloud"></a>Удаление частного облака AVS

AVS предоставляет гибкие возможности для удаления частного облака AVS. Частное облако AVS состоит из одного или нескольких кластеров vSphere. Каждый кластер может иметь от 3 до 16 узлов. При удалении частного облака AVS все кластеры будут удалены.

## <a name="before-you-begin"></a>Перед началом работы

При удалении частного облака AVS удаляется все частное облако AVS. Все компоненты частного облака AVS будут удалены. Если вы хотите хранить данные, убедитесь, что вы заархивированы данные в локальное хранилище или в хранилище Azure.

Компоненты частного облака AVS включают:

* Узлы AVS
* Виртуальные машины
* Виртуальные ЛС и подсети
* Все пользовательские данные, хранящиеся в частном облаке AVS
* Все вложения правил брандмауэра в виртуальную ЛС или подсеть

## <a name="sign-in-to-azure"></a>Войдите в Azure

Войдите на портал Azure по адресу [https://portal.azure.com](https://portal.azure.com).

## <a name="delete-an-avs-private-cloud"></a>Удаление частного облака AVS

1. [Доступ к порталу AVS](access-cloudsimple-portal.md).

2. Откройте страницу **ресурсы** .

3. Щелкните частное облако AVS, которое вы хотите удалить.

4. На странице Сводка нажмите кнопку **Удалить**.

    ![Удалить частное облако AVS](media/delete-private-cloud.png)

5. На странице Подтверждение введите имя частного облака AVS и нажмите кнопку **Удалить**. 

    ![Удаление частного облака AVS — подтверждение](media/delete-private-cloud-confirm.png)

Частное облако AVS помечено для удаления. Процесс удаления начинается через три часа и удаляет частное облако AVS.

> [!CAUTION]
> После удаления частного облака AVS необходимо удалить узлы. Измерение узлов будет продолжаться до тех пор, пока узлы не будут удалены из подписки.
