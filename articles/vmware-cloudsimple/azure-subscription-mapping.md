# <a name="map-resource-pools-from-your-private-cloud-to-your-azure-subscription"></a>Сопоставьте пулы ресурсов из частного облака с подпиской Azure.

Сопоставление подписок Azure позволяет сопоставлять пулы ресурсов из вашего частного облака vCenter с подпиской Azure. Вы можете сопоставлять только подписку, в которой вы создали службу Клаудсимпле.  Создание виртуальной машины VMware из портал Azure развертывает виртуальную машину в сопоставленном пуле ресурсов.  На портале Клаудсимпле вы можете просматривать подписку Azure для частных облаков и управлять ею.

Подписка может быть сопоставлена с несколькими пулами ресурсов vCenter частного облака.  Необходимо сопоставлять пулы ресурсов для каждого частного облака.  Для создания виртуальной машины VMware из портал Azure будут доступны только сопоставленные пулы ресурсов.

> [!IMPORTANT]
> Сопоставление пула ресурсов также сопоставляет все дочерние пулы ресурсов. Родительский пул ресурсов не может быть сопоставлен, если уже сопоставлены все дочерние пулы ресурсов.

## <a name="before-you-begin"></a>Перед началом работы

В этой статье предполагается, что в вашей подписке есть служба Клаудсимпле и частное облако.  Сведения о создании службы Клаудсимпле см. в разделе [Краткое руководство. Создание](quickstart-create-cloudsimple-service.md)службы.  Если вам нужно создать частное облако, см. раздел [Краткое руководство. Настройка среды частного облака](quickstart-create-private-cloud.md).

Кластер vCenter (пул корневых ресурсов) можно преобразовать в подписку.  Если вы хотите создать новый пул ресурсов, см. статью [Создание пула ресурсов](https://docs.vmware.com/en/VMware-vSphere/6.7/com.vmware.vsphere.resmgmt.doc/GUID-0F6C6709-A5DA-4D38-BE08-6CB1002DD13D.html) на сайте документации по VMware.

## <a name="default-resource-group"></a>Группа ресурсов по умолчанию

Создание виртуальной машины Клаудсимпле из портал Azure позволяет выбрать группу ресурсов.  Виртуальная машина, созданная с помощью частного облака vCenter в сопоставленном пуле ресурсов, будет отображаться на портал Azure.  Обнаруженная виртуальная машина будет помещена в группу ресурсов Azure по умолчанию.  Вы можете изменить имя группы ресурсов по умолчанию.

## <a name="map-azure-subscription"></a>Сопоставьте подписку Azure

1. Доступ к [порталу клаудсимпле](access-cloudsimple-portal.md).

2. Откройте страницу **ресурсы** и выберите частное облако, которое нужно подключить.

3. Выберите **сопоставление подписок Azure**.

4. Щелкните **изменить сопоставление подписок Azure**.

5. Чтобы отобразить доступные пулы ресурсов, выберите их слева и щелкните стрелку вправо.

6. Чтобы удалить сопоставления, выберите их справа и щелкните стрелку влево.

    ![Подписки Azure](media/resources-azure-mapping.png)

7. Нажмите кнопку **ОК**.

## <a name="change-default-resource-group-name"></a>Изменить имя группы ресурсов по умолчанию

1. Доступ к [порталу клаудсимпле](access-cloudsimple-portal.md).

2. Откройте страницу **ресурсы** и выберите частное облако, которое нужно подключить.

3. Выберите **сопоставление подписок Azure**.

4. Щелкните **изменить** в разделе Имя группы ресурсов Azure.

    ![Изменение имени группы ресурсов](media/resources-edit-resource-group-name.png)

5. Введите новое имя для группы ресурсов и нажмите кнопку **Отправить**.

    ![Введите имя новой группы ресурсов](media/resources-new-resource-group-name.png)
