# <a name="create-resource-pools-for-your-private-cloud-with-azure-subscription-mapping"></a>Создание пулов ресурсов для приватного облака с помощью отображения подписки Azure
Отображение подписок Azure позволяет создавать пулы ресурсов для частного облака из доступных пулов ресурсов vSphere. На портале CloudSimple вы можете просматривать и управлять подпиской Azure для ваших частных облаков.

> [!NOTE]
> Сопоставление пула ресурсов также отображает все пулы ресурсов детей. Родительский пул ресурсов не может быть отображен, если какие-либо пулы детских ресурсов уже отображены.

1. [Доступ к порталу CloudSimple](access-cloudsimple-portal.md).
2. Откройте страницу **ресурсов** и выберите **отображение подписок Azure.**  
3. Нажмите **отображение подписки Edit Azure.**  
4. Чтобы сопоставить доступные пулы ресурсов, выберите их слева и щелкните правой стрелкой. 
5. Чтобы удалить отображение, выберите их справа и нажмите на левую стрелку. 

    ![Подписки Azure](media/resources-azure-mapping.png)

6. Нажмите кнопку **ОК**.
