# <a name="map-resource-pools-from-your-private-cloud-to-your-azure-subscription"></a>Сопоставление пулов ресурсов с частным облаком в вашей подписке Azure

Сопоставление подписки Azure позволяет сопоставить пулов ресурсов из частного облака vCenter к своей подписке Azure. Можно сопоставить только подписки, где вы создали службу CloudSimple.  Создание виртуальной машины VMware на портале Azure развертывает виртуальную машину в пуле сопоставленный ресурс.  В портале CloudSimple можно просматривать и управлять подпиской Azure для вашего частного облака.

Подписки могут сопоставляться с нескольких пулов ресурсов vCenter частного облака.  Необходимо сопоставить пулов ресурсов для каждого частного облака.  Только пулы сопоставленный ресурс будет доступен для создания виртуальной машины VMware на портале Azure.

> [!IMPORTANT]
> Сопоставление пулу ресурсов сопоставляется любой дочерний пулов ресурсов. Родительский пул ресурсов не может быть сопоставлен, если все дочерние пулы ресурсов уже сопоставлены.

## <a name="before-you-begin"></a>Перед началом работы

В этой статье предполагается, что у вас есть служба CloudSimple и частного облака в вашей подписке.  Чтобы создать службу CloudSimple, см. в разделе [краткое руководство. Создание службы](quickstart-create-cloudsimple-service.md).  Если вам нужно создать частное облако, см. в разделе [краткое руководство. Настройка среды с частным облаком](quickstart-create-private-cloud.md).

Можно сопоставить кластера vCenter (корневой пул ресурсов) для вашей подписки.  Если вы хотите создать новый пул ресурсов, см. в разделе [создать пул ресурсов](https://docs.vmware.com/en/VMware-vSphere/6.7/com.vmware.vsphere.resmgmt.doc/GUID-0F6C6709-A5DA-4D38-BE08-6CB1002DD13D.html) статьи на сайте документации по VMware.

## <a name="default-resource-group"></a>Группы ресурсов по умолчанию

Создание новой виртуальной машины CloudSimple на портале Azure можно выбрать группу ресурсов.  Виртуальная машина, созданная на сервере vCenter частного облака в пуле сопоставленный ресурс снова появится на портале Azure.  Обнаруженной виртуальной машины будут размещены в группе ресурсов Azure по умолчанию.  Можно изменить имя группы ресурсов по умолчанию.

## <a name="map-azure-subscription"></a>Сопоставить подписки Azure

1. Доступ [CloudSimple портала](access-cloudsimple-portal.md).

2. Откройте **ресурсы** страницы и выберите частное облако, которое требуется сопоставить.

3. Выберите **сопоставление подписок Azure**.

4. Нажмите кнопку **сопоставления подписки Azure, изменить**.

5. Чтобы сопоставить доступные пулы ресурсов, выберите его в левой части и нажмите кнопку со стрелкой вправо.

6. Чтобы удалить сопоставления, выберите их справа и щелкните стрелку влево.

    ![Подписки Azure](media/resources-azure-mapping.png)

7. Последовательно выберите **ОК**.

## <a name="change-default-resource-group-name"></a>Изменение имени группы ресурсов по умолчанию

1. Доступ [CloudSimple портала](access-cloudsimple-portal.md).

2. Откройте **ресурсы** страницы и выберите частное облако, которое требуется сопоставить.

3. Выберите **сопоставление подписок Azure**.

4. Нажмите кнопку **изменить** с именем группы ресурсов Azure.

    ![Изменение имени группы ресурсов](media/resources-edit-resource-group-name.png)

5. Введите новое имя группы ресурсов и нажмите кнопку **отправить**.

    ![Введите имя новой группы ресурсов](media/resources-new-resource-group-name.png)
