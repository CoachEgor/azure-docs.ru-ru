# <a name="account-management-overview"></a>Общие сведения об управлении учетной записи

При создании службы CloudSimple, он создает учетную запись на CloudSimple.  Учетная запись связана с вашей подпиской Azure, где расположена служба.  Все пользователи, имеющие **владельца** и **участник** роли в подписке, имеют доступ к порталу CloudSimple.  Идентификатор подписки Azure и идентификатор клиента, связанный со службой CloudSimple находятся на [страница учетных записей](https://docs.azure.cloudsimple.com/account/).

## <a name="additional-alert-emails"></a>Дополнительные оповещения по электронной почте

Идентификаторы электронной почты можно настроить в CloudSimple для получения оповещений:

* Относящиеся к службе
* Для автоматической обработки

## <a name="cloudsimple-operator-access"></a>Оператор доступа CloudSimple

Можно контролировать доступ к порталу CloudSimple персонала.  Персонала Войдите на портал при отправке запроса в службу поддержки.  Операции службы, чтобы исправить сообщаемые проблемы и действия, предпринятые доступны в журналах аудита.

## <a name="users"></a>Пользователи

Все пользователи, имеющие **владельца** и **участник** роль в подписке имеет доступ к порталу CloudSimple.  При доступе к порталу, пользователь будет создан в CloudSimple учетной записи.  На странице учетных записей можно отключить доступ к порталу CloudSimple для конкретных пользователей.
