# <a name="quickstart---create-cloudsimple-service"></a>Краткое руководство. Создание службы Клаудсимпле

Чтобы приступить к работе, создайте решение Azure VMware, Клаудсимпле в портал Azure.

Служба Клаудсимпле позволяет использовать решение Azure VMware с помощью Клаудсимпле.  Создание службы позволяет покупать узлы, резервировать узлы и создавать частные облака.  Вы добавляете службу Клаудсимпле в каждый регион Azure, где доступна служба Клаудсимпле.  Служба определяет пограничной сеть решения Azure VMware, Клаудсимпле.  Эта пограничная сеть используется для служб, которые включают VPN, ExpressRoute и подключение к Интернету к частным облакам.

Чтобы добавить службу Клаудсимпле, необходимо создать подсеть шлюза. Подсеть шлюза используется при создании пограничной сети и требует блока/28 CIDR. Адресное пространство подсети шлюза должно быть уникальным. Он не может пересекаться с любыми адресными пространствами локальной сети или адресным пространством виртуальной сети Azure.

## <a name="sign-in-to-azure"></a>Войдите в Azure

Войдите на портал Azure по адресу [https://portal.azure.com](https://portal.azure.com).

## <a name="enable-microsoftvmwarecloudsimple-resource-provider"></a>Включение поставщика ресурсов Microsoft. Вмвареклаудсимпле

Выполните следующие действия, чтобы включить поставщик ресурсов для службы Клаудсимпле.

1. Выбор пункта **Все службы**.
2. Найдите и выберите **подписки**.

    ![Выбрать подписки](media/cloudsimple-service-select-subscriptions.png)

3. Выберите подписку, для которой необходимо включить службу Клаудсимпле.
4. Щелкните **поставщики ресурсов** для подписки.
5. Используйте **Microsoft. вмвареклаудсимпле** для фильтрации поставщика ресурсов.
6. Выберите поставщик ресурсов **Microsoft. вмвареклаудсимпле** и нажмите кнопку **зарегистрировать**.

    ![Зарегистрировать поставщик ресурсов](media/cloudsimple-service-enable-resource-provider.png)

## <a name="create-the-service"></a>Создание службы

1. Выбор пункта **Все службы**.
2. Выполните поиск **службы клаудсимпле**.

    ![Поиск службы Клаудсимпле](media/create-cloudsimple-service-search.png)

3. Выберите **Клаудсимпле Services (службы**).
4. Нажмите кнопку **Добавить** , чтобы создать новую службу.

    ![Добавление службы Клаудсимпле](media/create-cloudsimple-service-add.png)

5. Выберите подписку, в которой вы хотите создать службу Клаудсимпле.
6. Выберите группу ресурсов для службы. Чтобы добавить новую группу ресурсов, щелкните **создать**.
7. Введите имя для идентификатора службы.
8. Введите CIDR для шлюза службы. Укажите подсеть/28, которая не пересекается с любыми локальными подсетями, подсетями Azure или запланированными подсетями Клаудсимпле. После создания службы невозможно изменить CIDR.

    ![Создание службы Клаудсимпле](media/create-cloudsimple-service.png)

9. Нажмите кнопку **ОК**.

Служба будет создана и добавлена в список служб.

## <a name="purchase-nodes"></a>Покупка узлов

Чтобы настроить емкость с оплатой по мере использования для среды частного облака Клаудсимпле, сначала подготавливает узлы в портал Azure.

1. Выбор пункта **Все службы**.
2. Выполните поиск **узлов клаудсимпле**.

    ![Поиск узлов Клаудсимпле](media/create-cloudsimple-node-search.png)

3. Выберите **узлы клаудсимпле**.
4. Нажмите кнопку **Добавить** , чтобы создать узлы.

    ![Добавление узлов Клаудсимпле](media/create-cloudsimple-node-add.png)

5. Выберите подписку, в которой вы хотите приобрести узлы Клаудсимпле.
6. Выберите группу ресурсов для узлов. Чтобы добавить новую группу ресурсов, щелкните **создать**.
7. Введите префикс для обозначения узлов.
8. Выберите расположение ресурсов узла.
9. Выберите выделенное расположение для размещения ресурсов узла.
10. Выберите тип узла. Можно выбрать [параметр CS28 или CS36](cloudsimple-node.md). Последний вариант включает максимальный объем вычислений и объем памяти.
11. Выберите число узлов для инициализации.
12. Выберите **Review + Create** (Просмотреть и создать).
13. Проверьте параметры. Чтобы изменить параметры, нажмите кнопку **назад**.
14. Нажмите кнопку **Создать**.
