# <a name="cloudsimple-public-ip-address-overview"></a>Обзор общедоступных IP-адресов CloudSimple

Публичный IP-адрес позволяет интернет-ресурсам общаться с входящими на ресурсы Private Cloud по частному IP-адресу. Частный IP-адрес — это виртуальная машина или балансизатор загрузки программного обеспечения на вашем Private Cloud vCenter. Общедоступный IP-адрес позволяет выставлять в Интернет услуги, работающие в личном облаке.

Публичный IP-адрес предназначен для частного IP-адреса до тех пор, пока вы его не откажете. Публичный IP-адрес может быть присваивается только одному частному IP-адресу.

Ресурс, связанный с публичным IP-адресом, всегда использует общедоступный IP-адрес для доступа в Интернет. По умолчанию только исходящий доступ в Интернет разрешен на общедоступном IP-адресе.  Входящий трафик на общедоступный IP-адрес отказано.  Чтобы разрешить входящий трафик, создайте правило брандмауэра для общедоступного IP-адреса в конкретном порту.

## <a name="benefits"></a>Преимущества

Использование общедоступного IP-адреса для связи с входящими обеспечивает:

* Распределенный отказ в обслуживании (DDoS) предотвращение атак. Эта защита автоматически включена для общедоступного IP-адреса.
* Всегда на мониторинг трафика и в режиме реального времени смягчение общих атак сетевого уровня. Эти средства защиты являются теми же защиты, используемые microsoft онлайн-сервисов.
* Весь масштаб глобальной сети Azure. Сеть может использоваться для распределения и уменьшения трафика атак в разных регионах.
