# <a name="cloudsimple-public-ip-address-overview"></a>Общие сведения об CloudSimple адрес общедоступных IP-адрес

Общедоступный IP-адрес позволяет Интернет-ресурсам для обмена данными входящего трафика к ресурсам частного облака в частный IP-адрес. Частный IP-адрес является виртуальной машины или подсистемы балансировки нагрузки. Частный IP-адрес включен vCenter частного облака. Общедоступный IP-адрес позволяет предоставлять службы, запущенные в вашем частном облаке, к Интернету.

Общедоступный IP-адрес выделяется частный IP-адрес, пока не отменить ее назначение. Общедоступный IP-адрес можно назначить только один частный IP-адрес.

Ресурс, связанный с общедоступного IP-адреса всегда использует общедоступный IP-адрес для доступа к Интернету. По умолчанию только исходящий доступ к Интернету разрешен на общедоступный IP-адрес.  Трафик, поступающий на общедоступный IP-адрес отклонен.  Чтобы разрешить входящий трафик, создайте правило брандмауэра для общедоступных IP-адрес, определенный порт.

## <a name="benefits"></a>Преимущества

С помощью общедоступного IP-адреса для входящего подключения обеспечивает:

* Распределенный отказ в обслуживании (DDoS) предотвращение атак. Такая защита включается автоматически для общедоступного IP-адреса.
* Всегда-трафика в режиме реального времени и мониторинга защита от распространенных атак сетевого уровня. Средства защиты встроены же средства защиты, используемые службами Microsoft online services.
* Масштаб всей глобальной сети Azure. Сети можно использовать для снижения трафика атаки в регионах.
