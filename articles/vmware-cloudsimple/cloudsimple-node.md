# <a name="cloudsimple-nodes-overview"></a>Общие сведения о CloudSimple узлов

Узел — это:

* Выделенный исходного состояния системы вычислительных узлов, где установлен гипервизор VMware ESXi  
* Единица вычислений, которую вы можете приобрести или резерв для создания частных облаков  
* Для приобретения или зарезервировать в регионе, где доступна служба CloudSimple

Узлы являются стандартными блоками частного облака.  Чтобы создать частное облако, требуется по крайней мере из трех узлов одного номера SKU.  Чтобы развернуть в частном облаке, добавьте дополнительные узлы.  Можно добавить узлы в существующий кластер. Или можно создать новый кластер приобретения узлов на портале Azure, и связав их со службой CloudSimple.  Все узлы, которые приобрели отображаются в службе CloudSimple.  Создание частного облака от приобретенных узлов на портале CloudSimple.

## <a name="purchased-nodes"></a>Приобретенных узлов

Приобретенных узлов предоставляют емкость по мере использования. Приобретение узлы поможет вам быстро масштабировать кластер VMware по запросу. Можно добавить узлы, при необходимости, или удалить приобретенных узел масштабировать кластер VMware. Приобретенных узлы выставлен счет на ежемесячной основе и счет подписки, где их можно приобрести сроком:

* Если вы платите за подписку Azure, с помощью кредитной карты, карточки оплачивается немедленно.
* Если вы используете по счету-фактуре, плата за отражается в ближайшую накладную.

## <a name="vmware-solution-by-cloudsimple-nodes-sku"></a>Решение VMware CloudSimple узлами SKU

Для приобретения или резервирования доступны следующие типы узлов.

| SKU | CS28 - узел | CS36 - узел |
|-----|-------------|-------------|
| ЦП | 2x2.2 ГГц, 28 ядер (56 HT) | 2x2.3 ГГц, 36 ядер (72 HT) |
| ОЗУ | 256 ГБ | 512 ГБ |
| Диск кэша |  NVMe 1,6 ТБ | NVMe 3.2 ТБ |
| Емкость диска | 5.625 необработанные ТБ | 11,25 необработанные ТБ |
| тип хранилища; | All Flash | All Flash |

## <a name="limits"></a>Ограничения

Применяются следующие ограничения узел в частных облаках.

| Ресурс | Ограничение |
|----------|-------|
| Минимальное количество узлов для создания частного облака | 3 |
| Максимальное количество узлов в кластере в частном облаке | 16 |
| Максимальное количество узлов в частном облаке | 64 |
| Минимальное количество узлов в новом кластере | 3 |
