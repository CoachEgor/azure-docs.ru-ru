# <a name="cloudsimple-nodes-overview"></a>Обзор узлов CloudSimple

Узлы являются строительными блоками частного облака. Узла:

* Специализированный голый металлический вычислительный хост, где установлен гипервизор VMware ESXi  
* Единица вычислений, которые можно предоставить или зарезервировать для создания частных облаков
* Доступно для обеспечения или резервирования в регионе, где доступна услуга CloudSimple

Вы создаете частное облако из подготовленных узлов. Для создания частного облака требуется как минимум три узла одного и того же SKU. Чтобы расширить частное облако, добавьте дополнительные узлы.  Можно добавить узлы в существующий кластер или создать новый кластер, подготавлив узлы в портале Azure и связав их с службой CloudSimple.  Все готовые узлы видны в сервисе CloudSimple.  

## <a name="provisioned-nodes"></a>Предусмотренные узлы

Предусмотренные узлы обеспечивают пропускную способность с оплатой по мере вхад. Подготовка узлов позволяет быстро масштабировать кластер VMware по требованию. При необходимости можно добавить узлы или удалить подготовленный узлы для масштабирования кластера VMware. Предусмотренные узлы выставляются на ежемесячной основе и взимаются с подписки, где они подготовлены.

* Если вы оплачиваете подписку Azure кредитной картой, счет на карту немедленно выставляется.
* Если вы оплачены счетом-фактурой, сборы отображаются на следующем счете-фактуре.

## <a name="vmware-solution-by-cloudsimple-nodes-sku"></a>Решение VMware от CloudSimple узлов SKU

Для подготовки или резервирования доступны следующие типы узлов.

| номер SKU           | CS28 - Узла                 | CS36 - Узла                 | CS36m - Узло                |
|---------------|-----------------------------|-----------------------------|-----------------------------|
| Регион        | Восточная часть США, западная часть США            | Восточная часть США, западная часть США            | Западная Европа                 |
| ЦП           | 2x2,2 ГГц, 28 ядер (56 HT) | 2x2,3 ГГц, 36 ядер (72 HT) | 2x2,3 ГГц, 36 ядер (72 HT) |
| ОЗУ           | 256 ГБ                      | 512 ГБ                      | 576 ГБ                      |
| Кэш-диск    | 1.6-ТБ НВМЕ                 | 3.2-ТБ НВМЕ                 | 3.2-ТБ НВМЕ                 |
| Диск емкости | 5.625 ТБ Сырье                | 11.25 ТБ Сырье                | 15.36 ТБ Сырье                |
| Тип хранения  | All Flash                   | All Flash                   | All Flash                   |

## <a name="limits"></a>Ограничения

Следующие ограничения узлов применяются к частным облакам.

| Ресурс | Ограничение |
|----------|-------|
| Минимальное количество узлов для создания приватного облака | 3 |
| Максимальное количество узлов в кластере в частном облаке | 16 |
| Максимальное количество узлов в частном облаке | 64 |
| Минимальное количество узлов в новом кластере | 3 |
