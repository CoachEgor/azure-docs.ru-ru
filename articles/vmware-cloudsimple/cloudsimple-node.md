# <a name="cloudsimple-nodes-overview"></a>Общие сведения о CloudSimple узлов

Узел — это:

* Выделенный исходного состояния системы вычислительных узлов, где установлен гипервизор VMware ESXi  
* Единица вычислений вы может подготовить или зарезервировать для создания частных облаков  
* Для подготовки или зарезервировать в регионе, где доступна служба CloudSimple

Узлы являются стандартными блоками частного облака.  Чтобы создать частное облако, требуется по крайней мере из трех узлов одного номера SKU.  Чтобы развернуть в частном облаке, добавьте дополнительные узлы.  Можно добавить узлы в существующий кластер. Или можно создать новый кластер подготовки узлов на портале Azure, и связав их со службой CloudSimple.  Все подготовленные узлы отображаются в узле службы CloudSimple.  Создание частного облака из подготовленных узлов на портале CloudSimple.

## <a name="provisioned-nodes"></a>Подготовленные узлы

Подготовленных узлов предоставляют емкость по мере использования. Подготовка узлов поможет вам быстро масштабировать кластер VMware по запросу. Можно добавить узлы, при необходимости, или удалить узлом, подготовленную масштабировать кластер VMware. подготовленные узлы выставлен счет на ежемесячной основе и счет подписки, где они подготовлены:

* Если вы платите за подписку Azure, с помощью кредитной карты, карточки оплачивается немедленно.
* Если вы используете по счету-фактуре, плата за отражается в ближайшую накладную.

## <a name="vmware-solution-by-cloudsimple-nodes-sku"></a>Решение VMware CloudSimple узлами SKU

Для подготовки или резервирования доступны следующие типы узлов.

| SKU | CS28 - узел | CS36 - узел |
|-----|-------------|-------------|
| ЦП | 2x2.2 ГГц, 28 ядер (56 HT) | 2x2.3 ГГц, 36 ядер (72 HT) |
| ОЗУ | 256 ГБ | 512 ГБ |
| Диск кэша |  NVMe 1,6 ТБ | NVMe 3.2 ТБ |
| Емкость диска | 5.625 необработанные ТБ | 11,25 необработанные ТБ |
| тип хранилища; | All Flash | All Flash |

## <a name="limits"></a>Ограничения

Применяются следующие ограничения узел в частных облаках.

| Ресурс | Ограничение |
|----------|-------|
| Минимальное количество узлов для создания частного облака | 3 |
| Максимальное количество узлов в кластере в частном облаке | 16 |
| Максимальное количество узлов в частном облаке | 64 |
| Минимальное количество узлов в новом кластере | 3 |
