# <a name="cloudsimple-nodes-overview"></a>Обзор узлов Клаудсимпле

Узлы — это стандартные блоки частного облака. Узел:

* Выделенный компьютер для вычислений без операционной системы, на котором установлена низкоуровневая оболочка VMware ESXi  
* Единица вычислений, которые можно приобрести или зарезервировать для создания частных облаков
* Доступно для приобретения или резервирования в регионе, где доступна служба Клаудсимпле

Частное облако создается на основе приобретенных узлов. Чтобы создать частное облако, требуется как минимум три узла одного номера SKU. Чтобы развернуть частное облако, добавьте дополнительные узлы.  Вы можете добавить узлы в существующий кластер или создать новый кластер, приобретя узлы в портал Azure и связав их со службой Клаудсимпле.  Все приобретенные узлы отображаются в службе Клаудсимпле.  

## <a name="purchased-nodes"></a>Приобретенные узлы

Приобретенные узлы обеспечивают емкость с оплатой по мере использования. Узлы приобретения помогают быстро масштабировать кластер VMware по требованию. При необходимости можно добавить узлы или удалить приобретенный узел для масштабирования кластера VMware. Счета за приобретенные узлы выставляются ежемесячно и взимается в подписку, в которой они приобретены.

* Если вы платите за подписку Azure по кредитной карте, плата оплачивается немедленно.
* Если счет выставляется по счету, плата отображается в следующем счете.

## <a name="vmware-solution-by-cloudsimple-nodes-sku"></a>SKU решения VMware по Клаудсимпле для узлов

Для покупок или резервирования доступны следующие типы узлов.

| номер SKU | CS28-node | CS36-node |
|-----|-------------|-------------|
| ЦП | 2 2,2 ГГц, 28 ядер (56 HT) | 2 ядра с частотой 2.3 ГГц, 36 ядер (72 HT) |
| ОЗУ | 256 ГБ | 512 ГБ |
| Диск кэша |  1,6-ТБ NVMe | 3,2-ТБ NVMe |
| Диск емкости | 5,625 ТБ необработанных | 11,25 ТБ необработанных |
| тип хранилища; | All Flash | All Flash |

## <a name="limits"></a>Ограничения

Следующие ограничения узла применяются к частным облакам.

| Resource | Ограничение |
|----------|-------|
| Минимальное число узлов для создания частного облака | 3 |
| Максимальное число узлов в кластере в частном облаке | 16 |
| Максимальное число узлов в частном облаке | 64 |
| Минимальное число узлов в новом кластере | 3 |
