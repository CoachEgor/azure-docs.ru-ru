# <a name="view-the-list-of-cloudsimple-portal-users"></a>Посмотреть список пользователей портала CloudSimple

Пользователи добавляются в список пользователей при первом доступе к порталу CloudSimple. Чтобы просмотреть список пользователей, имеющих доступ к порталу CloudSimple через Azure, [получите доступ к порталу CloudSimple,](access-cloudsimple-portal.md)выберите **учетную запись** в боковом меню, а затем выберите **пользователей** на портале CloudSimple.

* Чтобы отобразить данные пользователя, включая подписку Azure, идентионсы клиента и идентиповцев пользователей, нажмите запись на странице **пользователей.**

* Для просмотра журнала действий аудита для пользователя выберите вкладку **журнала аудита.**
* Чтобы заблокировать или разблокировать учетную запись пользователя, нажмите **на заблокированный** переключатель при отображении данных пользователя. Когда учетная запись разблокирована, пользователь может получить доступ к порталу CloudSimple. Когда учетная запись заблокирована, доступ к порталу блокируется.
