# <a name="cloudsimple-node-quota-limits"></a>CloudSimple узел квоту.

Четырех узлов — это количество по умолчанию, можно приобрести, при подписке включено для службы CloudSimple.  Вы можете приобрести любой [тип узла](cloudsimple-node.md) с портала Azure.  Как минимум из трех узлов одного номера SKU необходимы для создания частного облака.  Если вы приобрели узлы, ошибка может появиться при попытке приобрести дополнительные узлы.

## <a name="quota-increase"></a>Увеличение квоты

Квоты узла можно увеличить, отправив запрос в службу поддержки. Рабочая группа службы вычисления запроса и работать с вами, чтобы увеличить квоту на узел.  При открытии новым билетом, выберите следующие параметры:

* Тип проблемы. **Технические требования**
* Подписка: **Идентификатор подписки**
* Тип службы: **Решение VMware, CloudSimple**
* Тип проблемы: **Квота выделенных узлов**
* Подтип проблемы: **Увеличьте квоту выделенных узлов**
* Субъект: **Увеличение квоты**

В области сведений в службу поддержки укажите необходимое количество узлов и узла SKU.

Можно также обратиться к своему представителю корпорации Майкрософт в [ azurevmwaresales@microsoft.com ](mailto:azurevmwaresales@microsoft.com) для увеличения квоты узла в вашей подписке.  Вам потребуется предоставить:

* Идентификатор подписки
* SKU узла
* Количество дополнительных узлов, для которых запрашивается увеличение квоты
