# <a name="cloudsimple-node-quota-limits"></a>Ограничения квоты узла Клаудсимпле

По умолчанию количество узлов доступно для подготовки, если подписка включена для службы Клаудсимпле.  Вы можете подготавливать любой [тип узла](cloudsimple-node.md) из портал Azure.  Для создания частного облака требуется как минимум три узла одного номера SKU.  Если вы подготовили узлы, при попытке подготовить дополнительные узлы может появиться сообщение об ошибке.

## <a name="quota-increase"></a>Увеличение квоты

Вы можете увеличить квоту узла, отправив запрос в службу поддержки. Группа операций службы будет оценивать запрос и работать с вами, чтобы увеличить квоту узла.  При открытии нового билета выберите следующие параметры:

* Тип проблемы. **Технические требования**
* Подписка: **Идентификатор подписки**
* Тип службы: **Решение VMware по Клаудсимпле**
* Тип проблемы: **Квота выделенных узлов**
* Подтип проблемы: **Увеличение квоты выделенных узлов**
* Тема: **Увеличение квоты**

В сведениях о заявке в службу поддержки укажите необходимое число узлов и номер SKU узла.

Вы также можете обратиться к представителям [azurevmwaresales@microsoft.com](mailto:azurevmwaresales@microsoft.com) учетная запись Майкрософт, чтобы увеличить квоту узла на подписку.  Необходимо предоставить:

* Идентификатор подписки
* Номер SKU узла
* Число дополнительных узлов, для которых запрашивается увеличение квоты
