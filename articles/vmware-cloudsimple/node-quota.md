# <a name="cloudsimple-node-quota-limits"></a>Ограничения квот на узлы CloudSimple

Четыре узла — это количество по умолчанию, доступное для покупки, когда ваша подписка включена для службы CloudSimple.  Вы можете приобрести любой [тип узлов](cloudsimple-node.md) на портале Azure.  Для создания частного облака требуется не менее трех узлов одного и того же SKU.  Если узлы приобрели, вы можете увидеть ошибку при попытке приобрести дополнительные узлы.

## <a name="quota-increase"></a>Повышение квоты

Вы можете увеличить квоту узлов, отправив запрос на поддержку. Команда службы будет оценивать запрос и работать с вами, чтобы увеличить квоту узлов.  Выберите следующие варианты при открытии нового билета:

* Тип выпуска: **Технический**
* Подписка: **Ваш идентификатор подписки**
* Тип сервиса: **Решение VMware от CloudSimple**
* Тип проблемы: **Квота выделенных узлов**
* Подтип проблемы: **Увеличение квоты выделенных узлов**
* Тема: **Увеличение квот**

В деталях опорного билета укажите необходимое количество узлов и узлов SKU.

* Узла SKU
* Количество дополнительных узлов, для которых вы запрашиваете увеличение квоты
