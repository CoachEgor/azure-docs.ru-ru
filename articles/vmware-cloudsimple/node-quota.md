# <a name="cloudsimple-node-quota-limits"></a>Ограничения квоты узла Клаудсимпле

По умолчанию количество узлов доступно для покупки, если подписка включена для службы Клаудсимпле.  Вы можете приобрести любой [тип узла](cloudsimple-node.md) из портал Azure.  Для создания частного облака требуется по крайней мере три узла одного номера SKU.  Если вы приобрели узлы, при попытке приобретения дополнительных узлов может появиться сообщение об ошибке.

## <a name="quota-increase"></a>Повышение квоты

Вы можете увеличить квоту узла, отправив запрос в службу поддержки. Группа операций службы будет оценивать запрос и работать с вами, чтобы увеличить квоту узла.  При открытии нового билета выберите следующие параметры:

* Тип проблемы. **Технические требования**
* Подписка: **Идентификатор подписки**
* Тип службы: **Решение VMware по Клаудсимпле**
* Тип проблемы: **Квота выделенных узлов**
* Подтип проблемы: **Увеличение квоты выделенных узлов**
* Субъект: **Увеличение квоты**

В сведениях о заявке в службу поддержки укажите необходимое число узлов и номер SKU узла.

* Номер SKU узла
* Число дополнительных узлов, для которых запрашивается увеличение квоты
