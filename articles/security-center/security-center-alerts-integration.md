# <a name="integration-of-azure-security-products-in-azure-security-center"></a>Интеграция продуктов безопасности Azure в центре безопасности Azure

Центр безопасности Azure предоставляет дополнительные лицензии Майкрософт для работы со следующими продуктами безопасности:

* [Azure WAF](#azure-waf)
* [Защита от атак DDoS Azure](#azure-ddos)

## Azure WAF<a name="azure-waf"></a>

Шлюз приложений Azure предоставляет брандмауэр веб-приложения (WAF), обеспечивающий централизованную защиту веб-приложений от распространенных эксплойтов и уязвимостей.

Веб-приложения все чаще подвергаются вредоносным атакам, использующим общеизвестные уязвимости. WAF шлюза приложений основан на основном наборе правил 3,0 или 2.2.9 из открытого проекта безопасности веб-приложений. WAF обновляется автоматически для защиты от новых уязвимостей без необходимости дополнительной настройки. Оповещения WAF передаются в центр безопасности. Дополнительные сведения о предупреждениях, создаваемых WAF, см. в разделе [правила и правила CR для брандмауэра веб-приложения](../web-application-firewall/ag/application-gateway-crs-rulegroups-rules.md?tabs=owasp31#crs911-31).

## Защита от атак DDoS в Azure<a name="azure-ddos"></a>

Известно, что атаки типа "отказ в обслуживании" (от атак DDoS) являются простыми в выполнении. Они стали отличным вопросом безопасности, особенно если вы перемещаете приложения в облако. 

Целью DDoS-атаки является исчерпание ресурсов приложения, чтобы приложение стало недоступным для обычных пользователей. Атаки от атак DDoS могут ориентироваться на любые конечные точки, к которым можно получить доступ через Интернет.

Защита от атак DDoS Azure в сочетании с рекомендациями по проектированию приложений обеспечивает защиту от атак от атак DDoS. Защита от атак DDoS предоставляет различные уровни служб. Дополнительные сведения см. в статье [Общие сведения о защите Azure от атак DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview).

С помощью службы "Защита от атак DDoS" уровня "Стандартный" можно устранить следующие типы атак.

> [!div class="mx-tableFixed"]

|Предупреждение|ОПИСАНИЕ|
|---|---|
|**Обнаружена атака объемные**|Цель этой атаки — перегрузить сетевой уровень с очень значительным объемом подлинного трафика. Такие атаки включают UDP-флуд, заполнение с усилением и другие типы заполнения с использованием поддельных пакетов. От атак DDoS Protection Standard устраняет эти потенциальные атаки с несколькими гигабайтами, выявляя и вычистяя их с помощью глобальной масштабируемости сети, автоматически.|
|**Обнаружена атака по протоколу**|Эти атаки отображают целевой объект недоступным путем использования слабости в стеках протоколов уровня 3 и 4. Он включает атаки Flood, атаки с отражением и другие атаки протокола. Служба "Защита от атак DDoS" уровня "Стандартный" позволяет устранить эти атаки, разделяя допустимый и вредоносный трафик путем взаимодействия с клиентом и блокировки вредоносного трафика.|
|**Обнаружена атака на уровне ресурсов (приложений)**|Эти атаки нацелены на пакеты веб-приложений, чтобы нарушить передачу данных между узлами. Такие атаки включают в себя нарушение протокола HTTP, атаки путем внедрения кода SQL, межсайтовые сценарии и другие атаки уровня 7. Используйте WAF шлюза приложений Azure со стандартом защиты от атак DDoS, чтобы защититься от этих атак. Кроме того, в Azure Marketplace доступны предложения WAF сторонних производителей.|
