# <a name="cross-tenant-management-in-security-center"></a>Кросс-арендаторное управление в Центре безопасности

Управление кросс-арендаторами позволяет просматривать и управлять позицией безопасности нескольких клиентов в Центре безопасности, используя [делегированные ресурсами Azure.](../lighthouse/concepts/azure-delegated-resource-management.md) Эффективное управление несколькими арендаторами, с одного представления, без необходимости входить в каталог каждого арендатора.

- Поставщики услуг могут управлять безопасностью ресурсов, для нескольких клиентов, из своего собственного арендатора.

- Группы безопасности организаций с несколькими арендаторами могут просматривать и управлять своей позицией безопасности из одного места.

## <a name="set-up-cross-tenant-management"></a>Настройка управления кросс-арендаторами

Настройка управления кросс-арендаторами путем делегирования доступа к ресурсам управляемых арендаторов собственному арендатору с помощью [делегированного ресурсами Azure.](../lighthouse/concepts/azure-delegated-resource-management.md)

> [!NOTE]
> Делегированное управление ресурсами Azure — один из ключевых компонентов Azure Lighthouse.

## <a name="how-does-cross-tenant-management-work-in-security-center"></a>Как работает кросс-арендаторное управление в Центре безопасности

Вы можете просматривать и управлять подписками между несколькими арендаторами таким же образом, что вы управляете несколькими подписками в одном арендаторе.

Из верхней панели меню щелкните значок фильтра и выберите подписку из каталога каждого арендатора, который вы хотите просмотреть.

  ![Арендаторы фильтров](./media/security-center-cross-tenant-management/cross-tenant-filter.png)

Взгляды и действия в основном одинаковы. Ниже приведены некоторые примеры:

- **Управление политиками безопасности:** С одного представления, управлять позицией безопасности многих ресурсов с [помощью политик,](tutorial-security-policy.md)принимать меры с рекомендациями безопасности, а также собирать и управлять данными, связанными с безопасностью.
- **Улучшение безопасной оценки и соответствия позы**: Кросс-арендатор видимость позволяет просматривать общую позицию безопасности всех ваших арендаторов и где и как наилучшим образом улучшить [безопасный счет](security-center-secure-score.md) и [соответствие позы](security-center-compliance-dashboard.md) для каждого из них.
- **Исправление рекомендаций**: Мониторинг и исправления [рекомендации](security-center-recommendations.md) для многих ресурсов от различных арендаторов в одно время. Затем можно немедленно устранить уязвимости, которые представляют наибольший риск для всех арендаторов.
- **Управление оповещениями:** [Обнаружение оповещений](security-center-alerts-overview.md) между различными арендаторами. Принять меры в отношении ресурсов, которые не соответствуют действиям, которые [используются.](security-center-managing-and-responding-alerts.md)

- **Управление расширенными функциями облачной защиты и многое**другое: Управление различными службами защиты от угроз, такими как [просто в срок (JIT) VM доступ,](security-center-just-in-time.md) [Адаптивное укрепление сети,](security-center-adaptive-network-hardening.md) [адаптивные элементы управления приложениями,](security-center-adaptive-application.md)и многое другое.
