# <a name="billing-questions"></a>Вопросы по выставлению счетов

## <a name="how-does-billing-work-for-azure-security-center"></a>Как выставляются счета за использование Центра безопасности Azure?
Центр безопасности предлагается с защитником Azure и без него:

- **Без участия в защитнике Azure**центр безопасности предоставляет сведения о состоянии безопасности ресурсов Azure, базовой политике безопасности, рекомендациях по безопасности и интеграции с продуктами и службами безопасности от партнеров.

- **Защитник Azure** добавляет возможности защиты от угроз, в том числе оповещения системы безопасности, аналитика угроз, анализ поведения, обнаружение аномалий и отчеты о соотношении угроз. Вы можете запустить бесплатную 30-дневную пробную версию защитника Azure. Дополнительные [сведения об обновлении](security-center-pricing.md).

## <a name="how-can-i-track-who-in-my-organization-enabled-azure-defender-changes-in-azure-security-center"></a>Как можно узнать, кто из моих организаций изменил защитник Azure в центре безопасности Azure
Подписки Azure могут иметь несколько администраторов с разрешениями на изменение настроек цен. Чтобы узнать, какой пользователь выполнил изменение, используйте журнал действий Azure. 

Дополнительные сведения см. [здесь](https://techcommunity.microsoft.com/t5/Security-Identity/Tracking-Changes-in-the-Pricing-Tier-for-Azure-Security-Center/td-p/390832).
