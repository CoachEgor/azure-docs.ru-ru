# <a name="set-up-email-notifications-for-security-alerts"></a>Настройка уведомлений по электронной почте для оповещений системы безопасности 

Чтобы сотрудники вашей организации получали уведомления об оповещениях системы безопасности в вашей среде, введите их адреса на странице Параметры **уведомлений по электронной почте** .

При настройке уведомлений можно настроить отправку сообщений электронной почты конкретным лицам или всем пользователям с определенной ролью RBAC для подписки. 

Чтобы избежать выносливости предупреждений, центр безопасности ограничивает объем исходящих сообщений. Для каждой подписки центр безопасности отправляет:

- не более **четырех** сообщений в день для оповещений **с высокой серьезностью**
- не более **двух** сообщений электронной почты в день для предупреждений **среднего уровня серьезности**
- не более **одного** электронного сообщения в день для оповещений **с низкой серьезностью**

## <a name="availability"></a>Доступность

- Состояние выпуска: **Общедоступная версия**
- Обязательные роли: **администратор безопасности** или **владелец подписки** 
- Облака: ✔ коммерческие облака ✔ US Gov (частично) ✘ National/независимых (Китай gov, other gov)


## <a name="set-up-email-notifications-for-alerts"></a>Настройка уведомлений по электронной почте для оповещений<a name="email"></a>

Вы можете отправлять пользователям уведомления по электронной почте или всем пользователям с конкретными ролями RBAC.

1. В области **параметры & цен** центра безопасности выберите соответствующую подписку и щелкните **уведомления по электронной почте**.

1. Определите получателей уведомлений:

    - Из раскрывающегося списка выберите доступные роли.
    - И/или введите конкретные адреса электронной почты, разделенные запятыми. Количество адресов электронной почты, которые можно ввести, не ограничено.

1. Чтобы применить контактную информацию безопасности к своей подписке, нажмите кнопку **сохранить**.
