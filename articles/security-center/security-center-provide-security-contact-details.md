# <a name="set-up-email-notifications-for-security-alerts"></a>Настройка отправки по электронной почте уведомлений об оповещениях системы безопасности 

Чтобы определенные сотрудники вашей организации получали уведомления об оповещениях системы безопасности в вашей среде, укажите их адреса на странице параметров **Уведомления по электронной почте**.

Отправку уведомлений можно настроить так, чтобы их получали определенные сотрудники или все пользователи, которым в подписке назначена определенная роль Azure. 

Чтобы количество уведомлений не стало чрезмерным, в Центре безопасности установлены ограничения на их отправку. Для каждой подписки Центр безопасности отправляет:

- не более **четырех** оповещений с **высоким уровнем серьезности** в день;
- не более **двух** оповещений со **средним уровнем серьезности** в день;
- не более **одного** оповещения с **низким уровнем серьезности** в день.

## <a name="set-up-email-notifications-for-alerts"></a>Настройка отправки по электронной почте уведомлений об оповещениях <a name="email"></a>

Вы можете отправлять уведомления определенным сотрудникам или всем пользователям с определенными ролями Azure.

1. В Центре безопасности на странице **Цены и параметры** выберите нужную подписку и выберите элемент **Уведомления по электронной почте**.

1. Чтобы указать получателей уведомлений, выполните следующие действия:

    - Из раскрывающегося списка доступных ролей выберите нужные.
    - Кроме того, вы можете ввести адреса получателей, разделяя их запятыми. Количество таких получателей неограниченно.

1. Нажмите кнопку **Сохранить**, чтобы сохранить сведения о контактных лицах по вопросам безопасности в своей подписке.
