# <a name="improve-your-secure-score-in-azure-security-center"></a>Улучшение оценки безопасности в центре безопасности Azure

> [!NOTE]
> В предварительной версии доступен расширенный показатель безопасности. Расширенный показатель безопасности в конечном итоге заменит существующий показатель безопасности, но в то время как он будет выполняться параллельно, чтобы упростить переход.
>
> Дополнительные сведения о преимуществах расширенной оценки безопасности см. [здесь](secure-score-security-controls.md).
>
> Чтобы принять участие в предварительной версии, откройте портал Azure, запустите центр безопасности Azure и выберите "безопасность — оценка". Отсюда вы увидите баннер в верхней части страницы, в которой предлагается новый показатель безопасной работы. Можно также щелкнуть [здесь](https://aka.ms/ascnewscore).

Когда так много служб, предлагающих возможности защиты, порой трудно решить, что следует сделать в первую очередь, чтобы повысить безопасность и надежность рабочих нагрузок. Баллы по безопасности проверяют ваши рекомендации по защите и назначают приоритеты для вас, поэтому вы узнаете, какие рекомендации следует выполнить первыми. Это поможет определить, какие уязвимости в системе безопасности являются наиболее серьезными и приоритетными для анализа. Оценка безопасности — это средство, помогающее оценить безопасность рабочей нагрузки.

## <a name="secure-score-calculation"></a>Вычисление безопасной оценки

Центр безопасности имитирует работу аналитика в сфере безопасности, проверяя рекомендации по безопасности и применяя сложные алгоритмы для определения степени важности каждой из них.
Центр безопасности Azure постоянно изучает ваши активные рекомендации и оценивает вашу оценку безопасности на основе этих рекомендаций. Оценка рекомендаций основана на его серьезности и рекомендациях по обеспечению безопасности, которые влияют на безопасность рабочей нагрузки.

Центр безопасности также предоставляет **общий показатель безопасности**. 

**Общий показатель безопасности** — это совокупность всех ваших оценок рекомендаций. Вы можете просмотреть общую оценку безопасности в подписках или группах управления в зависимости от того, что вы выбрали. Она будет зависеть от выбранных подписок и активных рекомендаций в них.

Чтобы узнать, какие рекомендации влияют на вашу оценку, вы можете просмотреть три наиболее важные рекомендации на панели мониторинга центра безопасности или отсортировать рекомендации в колонке список рекомендаций, используя столбец "влияние на **Безопасность оценки** ".

Для просмотра общей оценки безопасности выполните следующие действия.

1. На панели мониторинга Azure щелкните **Центр безопасности** , а затем щелкните **Безопасность оценки**.

2. В верхней части можно увидеть важные показатели безопасности:
   - **Общий показатель безопасности** представляет оценку для каждой политики на выбранную подписку.
   - **Безопасность оценки по категориям** показывает, какие ресурсы требуют наибольшего внимания
   - **Основные рекомендации по влиянию на результаты безопасности** предоставляют список рекомендаций, которые улучшат вашу оценку безопасности.
 
   ![Оценка безопасности](./media/security-center-secure-score/secure-score-dashboard.png)

3. В таблице ниже можно увидеть каждую из подписок и общую оценку безопасности для каждого из них.

   > [!NOTE]
   > Сумма безопасной оценки каждой подписки не равна общей оценке безопасности. Оценка безопасности — это вычисление, основанное на соотношении между работоспособными ресурсами и общим количеством ресурсов по рекомендациям, а не суммой безопасных оценок по подпискам. 
   >
4. Щелкните **просмотреть рекомендации** , чтобы просмотреть рекомендации для этой подписки, которые можно исправить для улучшения оценки безопасности.
4. В списке рекомендаций можно увидеть, что для каждой рекомендации существует столбец, представляющий **влияние на показатель безопасности**. Это число показывает, насколько будет совершенствоваться общий показатель безопасности при соблюдении рекомендаций. Например, на приведенном ниже экране при **исправлении уязвимостей в конфигурациях безопасности контейнеров**Оценка безопасности увеличится на 35 точек.

   ![Оценка безопасности](./media/security-center-secure-score/security-center-secure-score1.png)



## <a name="individual-secure-score"></a>Индивидуальная оценка безопасности

Кроме того, для просмотра отдельных оценок безопасности их можно найти в колонке рекомендации по отдельным рекомендациям.  

**Оценка безопасности рекомендаций** — это вычисление, основанное на соотношении работоспособных ресурсов и общего количества ресурсов. Если число работоспособных ресурсов равно общему числу ресурсов, вы получаете максимальную защищенную оценку рекомендации 50. Чтобы оценить безопасность ближе к максимальному показателю, исправьте неработоспособные ресурсы, следуя рекомендациям.

**Влияние на рекомендацию** позволяет вам понять, насколько надежна Ваша оценка безопасности при применении рекомендаций. Например, если оценка безопасности равна 42, а **рекомендации** имеют значение + 3, выполнение действий, описанных в рекомендациях, улучшите оценку, чтобы она стала 45.

В рекомендации указывается, каким угрозам будет подвергаться рабочая нагрузка, если не выполнить инструкции по исправлению.

![Индивидуальная оценка безопасности рекомендаций](./media/security-center-secure-score/indiv-recommendation-secure-score.png)
