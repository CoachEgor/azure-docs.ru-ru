# <a name="improve-your-secure-score-in-azure-security-center"></a>Повышение оценки безопасности в центре безопасности Azure

> [!NOTE]
> Эта статья о предыдущей версии безопасного балла. Этот безопасный опыт оценки по-прежнему доступен из uI, но будет постепенно прекращена с течением времени. Два безопасных опыта оценки работают бок о бок, чтобы обеспечить более плавный переход.
>
> Подробную информацию о новом безопасном счете смотрите [здесь](secure-score-security-controls.md).
>

Когда так много служб, предлагающих возможности защиты, порой трудно решить, что следует сделать в первую очередь, чтобы повысить безопасность и надежность рабочих нагрузок. Secure Score рассматривает ваши рекомендации по безопасности и распоряжается их приоритетами для вас, так что вы знаете, какие рекомендации выполнить в первую очередь. Это поможет определить, какие уязвимости в системе безопасности являются наиболее серьезными и приоритетными для анализа. Secure Score — это инструмент, который поможет вам оценить вашу позу безопасности рабочей нагрузки.

## <a name="secure-score-calculation"></a>Расчет безопасного счета

Центр безопасности имитирует работу аналитика в сфере безопасности, проверяя рекомендации по безопасности и применяя сложные алгоритмы для определения степени важности каждой из них.
Центр безопасности Azure постоянно просматривает ваши активные рекомендации и вычисляет ваш Secure Score на их основе, оценка рекомендации вытекает из его серьезности и безопасности передового опыта, которые повлияет на безопасность рабочей нагрузки больше всего.

Центр безопасности также предоставляет вам **общий безопасный балл.** 

**В целом Безопасный результат** является накопление всех ваших рекомендаций баллов. Вы можете просматривать общий безопасный счет в группах подписки или управления в зависимости от выбранного вами показателя. Она будет зависеть от выбранных подписок и активных рекомендаций в них.

Чтобы проверить, какие рекомендации больше всего влияют на ваш Secure Score, вы можете просмотреть три наиболее эффективных рекомендации в панели мониторинга Центра безопасности или сортировать рекомендации в лезвии списка рекомендаций с помощью столбца **воздействия Secure Score.**

Для просмотра вашего общего безопасного результата:

1. В панели мониторинга Azure щелкните **центр безопасности,** а затем нажмите **Secure Score.**

2. В верхней части вы можете увидеть Безопасный Оценка подчеркивается:
   - **Общий безопасный балл** представляет счет за полисы, по выбранной подписке
   - **Безопасный результат по категориям** показывает, какие ресурсы нуждаются в наибольшем внимании
   - **Лучшие рекомендации по воздействию Secure Score** предоставляет вам список рекомендаций, которые улучшат ваш Secure Score больше всего, если вы их реализуете.
 
   ![Оценка безопасности](./media/security-center-secure-score/secure-score-dashboard.png)

3. В таблице ниже вы можете увидеть каждую из ваших подписок и общий безопасный балл для каждой из них.

   > [!NOTE]
   > Сумма Secure Score каждой подписки не равна общему secure Score. Безопасный балл — это расчет, основанный на соотношении между вашими здоровыми ресурсами и общими ресурсами в зависимости от рекомендации, а не суммабезопасных результатов по подписке. 
   >
4. Нажмите **Посмотреть рекомендации,** чтобы увидеть рекомендации для этой подписки, которые вы можете исправить, чтобы улучшить ваш безопасный результат.
4. В списке рекомендаций можно увидеть, что для каждой рекомендации есть столбец, представляющий **влияние Secure Score.** Это число представляет, насколько ваш общий безопасный балл улучшится, если вы будете следовать рекомендациям. Например, на экране ниже, если вы **исправите уязвимости в конфигурациях безопасности контейнеров,** ваш Secure Score увеличится на 35 пунктов.

   ![Оценка безопасности](./media/security-center-secure-score/security-center-secure-score1.png)



## <a name="individual-secure-score"></a>Индивидуальный безопасный счет

Кроме того, для просмотра отдельных Безопасные результаты, вы можете найти их в рамках индивидуальной рекомендации лезвия.  

**Рекомендация Безопасный оценка** является расчет на основе соотношения между здоровых ресурсов и общих ресурсов. Если количество здоровых ресурсов равно общему количеству ресурсов, вы получите максимальный безопасный балл рекомендации 50. Чтобы попытаться получить безопасный балл ближе к максимальной оценки, исправить нездоровые ресурсы, следуя рекомендациям.

**Рекомендация воздействия** позволяет узнать, насколько ваш Безопасный оценка улучшается, если вы применяете рекомендации шаги. Например, если ваш Secure Score 42, а **влияние Рекомендации** - 3, выполнение шагов, изложенных в рекомендации, улучшает ваш балл и становится 45.

В рекомендации указывается, каким угрозам будет подвергаться рабочая нагрузка, если не выполнить инструкции по исправлению.

![оценка безопасности для отдельной рекомендации](./media/security-center-secure-score/indiv-recommendation-secure-score.png)
