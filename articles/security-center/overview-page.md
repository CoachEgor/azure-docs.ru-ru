# <a name="azure-security-centers-overview-page"></a>Обзорная страница центра безопасности Azure

При открытии центра безопасности Azure Первая отображаемая страница является страницей обзора. 

:::image type="content" source="media/overview-page/overview.png" alt-text="Страница обзора в Центре безопасности":::

Выявление и оценка безопасности рабочих нагрузок, а также определение и устранение рисков с помощью страницы обзора центра безопасности.

В обзоре представлено единое представление о безопасности гибридных облачных рабочих нагрузок. Кроме того, в нем отображаются оповещения системы безопасности, сведения о покрытии и многое другое.


## <a name="features-of-the-overview-page"></a>Возможности страницы "Обзор"

:::image type="content" source="media/overview-page/top-bar-of-overview.png" alt-text="Верхняя панель страницы обзора центра безопасности":::

**Верхняя строка меню** предлагает:
- **Подписки** . Вы можете просмотреть и отфильтровать список подписок, нажав эту кнопку. Центр безопасности отобразит уровень безопасности для выбранных подписок.
- **Новые** возможности. открывает [заметки о выпуске](release-notes.md) , чтобы вы могли оставаться в курсе новых возможностей, исправлений ошибок и устаревших функций.
- **Высокоуровневые номера** для подключенных облачных учетных записей для отображения контекста сведений на основных плитках ниже. А также количество активных рекомендаций и оповещений.
    Дополнительные сведения о подключении [учетных записей AWS](quickstart-onboard-aws.md) и [проектов обеспечить](quickstart-onboard-gcp.md).


В центре страницы расположены **четыре центральных плитки**, каждая из которых связывается с выделенной панелью мониторинга для получения дополнительных сведений:
- **Оценка** безопасности. Центр безопасности постоянно оценивает ресурсы, подписки и Организации на предмет проблем безопасности. Затем он объединяет все результаты в одну оценку, чтобы вы могли понять текущее состояние системы безопасности: чем выше оценка, тем ниже выявленный уровень риска. [Подробнее](secure-score-security-controls.md).
- **Соответствие** — центр безопасности предоставляет аналитические сведения о соответствии требованиям, основанным на непрерывных оценках среды Azure. Центр безопасности анализирует факторы риска в гибридной облачной среде в соответствии с рекомендациями по безопасности. Эти оценки сопоставляются с элементами управления соответствием из поддерживаемого набора стандартов. Дополнительные [сведения](security-center-compliance-dashboard.md).
- **Защитник Azure** — это платформа облачной защиты рабочих нагрузок (квпп), интегрированная в центр безопасности для расширенной и интеллектуальной защиты рабочих нагрузок Azure и гибридной рабочей нагрузки. На плитке отображается покрытие подключенных ресурсов (для выбранных в настоящее время подписок) и последние оповещения, запрограммированные по степени серьезности. [Подробнее](azure-defender.md).
- **Инвентаризация** . на плитке отображается количество неотслеживаемых виртуальных машин и простой барометер ресурсов, отслеживаемых центербен безопасности. [Подробнее](asset-inventory.md).


Панель **Insights** предлагает настраиваемые элементы для вашей среды, в том числе:
- Самые неатакованные ресурсы
- Элементы управления безопасностью с наибольшим возможным увеличением уровня безопасности
- Активные рекомендации с наибольшим влиянием на ресурсы
- Последние записи блога экспертов Центра безопасности Azure
