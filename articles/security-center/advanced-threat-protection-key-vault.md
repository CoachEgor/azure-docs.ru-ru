# <a name="threat-protection-for-azure-key-vault-preview"></a>Защита от угроз для убежища ключей Azure (предварительный просмотр)

Расширенная защита от угроз для Azure Key Vault обеспечивает дополнительный уровень безопасности. Этот инструмент обнаруживает потенциально опасные попытки получить доступ к учетным записям Key Vault или использовать их. Используя наиболее расширенную защиту угроз в Центре безопасности Azure, вы можете устракиваться в угрозах, не будучи экспертом по безопасности и не изучая дополнительные системы мониторинга безопасности.

Когда Центр безопасности обнаруживает аномальную активность, он отображает оповещения. Он также по электронной почте администратор подписки с подробной информацией о подозрительной деятельности и рекомендации о том, как исследовать и устранять выявленные угрозы.

## <a name="configuring-threat-protection-from-security-center"></a>Настройка защиты от угроз из Центра безопасности

По умолчанию расширенная защита от угроз включена для всех учетных записей Key Vault при подписке на стандартный уровень ценообразования Центра безопасности. Дополнительные сведения см. на странице [цен](security-center-pricing.md).

Для включения или отключителя защиты для конкретной подписки:

1. Из левой панели в Центре безопасности выберите **настройки & цен.**

1. Выберите подписку с учетными записями хранения, для которых вы хотите включить или отключить защиту от угроз.

1. Выберите **уровень ценообразования**.

1. На **уровне выбора цен по** группе типов ресурсов найдите строку **Key Vaults** и выберите **Enabled** или **Disabled.**

    [![Включение или отключение расширенной защиты от угроз для Убежища ключей в Центре безопасности Azure](media/advanced-threat-protection-key-vault/atp-for-akv-enable-atp-for-akv.png)](media/advanced-threat-protection-key-vault/atp-for-akv-enable-atp-for-akv.png#lightbox)

1. Нажмите кнопку **Сохранить**.
