# <a name="security-recommendations-in-azure-security-center"></a>Рекомендации по безопасности в Центре безопасности Azure 
В этом разделе объясняется, как просмотреть и изучить рекомендации в центре безопасности Azure, которые помогут защитить ресурсы Azure.

> [!NOTE]
> В документе приводится обзор службы с помощью примера развертывания.  Этот документ не является пошаговым руководством.
>

## <a name="what-are-security-recommendations"></a>Что такое рекомендации по безопасности?

Рекомендации — это действия, которые необходимо выполнить для защиты ресурсов.

Центр безопасности периодически анализирует состояние безопасности ресурсов Azure для выявления потенциальных уязвимостей системы безопасности. Затем он предоставляет рекомендации по их удалению.

Каждая рекомендация предоставляет следующие сведения.

- Краткое описание рекомендуемых возможностей.
- Действия по исправлению, которые необходимо выполнить для реализации рекомендации. <!-- In some cases, one-click remediation is available. -->
- Какие ресурсы требуются для выполнения рекомендуемых действий с ними.
- **Влияние на показатель безопасности**, которое является суммой, которую будет использовать ваш показатель безопасности при реализации этой рекомендации.

## Мониторинг рекомендаций<a name="monitor-recommendations"></a>

Центр безопасности анализирует состояние безопасности ресурсов для выявления потенциальных уязвимостей. На плитке **рекомендации** в разделе **Обзор** отображается общее число рекомендаций, определенных центром безопасности.

![Обзор центра безопасности](./media/security-center-recommendations/asc-overview.png)

1. Выберите **плитку рекомендации** в разделе **Обзор**. Откроется список **рекомендаций** .

      ![Просмотр рекомендаций](./media/security-center-recommendations/view-recommendations.png)

    Вы можете отфильтровать рекомендации. Для этого выберите **Фильтр** в колонке **Рекомендации**. Откроется колонка **Фильтр** , где можно выбрать значения серьезности и состояния для просмотра.

   * **РЕКОМЕНДАЦИИ**: Рекомендация.
   * **ВЛИЯНИЕ НА ПОКАЗАТЕЛЬ БЕЗОПАСНОСТИ**: Оценка, созданная центром безопасности с использованием ваших рекомендаций по безопасности, и применение расширенных алгоритмов для определения важности каждой рекомендации. Дополнительные сведения см. в разделе [Вычисление безопасной оценки](security-center-secure-score.md#secure-score-calculation).
   * **Ресурс**. Список ресурсов, к которым применяется эта рекомендация.
   * **СТРОКИ СОСТОЯНИЯ**:  Описание уровня серьезности конкретной рекомендации.
       * **Высокий (красный)** : Существует требующая внимания уязвимость важных ресурсов (таких, как приложение, виртуальная машина или группа безопасности сети).
       * **Средний (оранжевый)** : Существует уязвимость, и для ее устранения либо для завершения процесса требуются некритические или дополнительные действия.
       * **Низкий (синий)** : Уязвимость нужно устранить, но она не требует немедленного вмешательства. (По умолчанию низкий уровень не представлен, однако вы можете выполнить фильтрацию по этому значению, чтобы просмотреть соответствующие рекомендации). 
       * **Исправен (зеленый)** :
       * **Недоступно (серый)** :

1. Чтобы просмотреть сведения о каждой рекомендации, щелкните рекомендацию.
