# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>Подключение к порталу Azure Certified for IoT

Вы можете использовать [сертифицированный портал Azure для IoT](https://aka.ms/ACFI) для:

- Завершите процесс сертификации устройства IoT.
- Найдите модели устройств IoT Plug и Play Preview. Эти модели можно использовать для быстрого создания устройств, готовых к Работе с IoT, и их интеграции с решениями.
- Опубликовать модель возможностей устройства в общедоступном репозитории.

## <a name="sign-in"></a>Вход

Используйте _рабочую учетную запись_ Майкрософт для вхотливки на портал [Azure Certified для портала IoT.](https://aka.ms/ACFI) Портал проверяет ваше членство в Партнерском центре Майкрософт при входе в систему:

- Если ваша организация уже является членом [Партнерского центра Майкрософт,](https://docs.microsoft.com/partner-center/)вам предоставляется доступ ко всем функциям портала.
- Если ваша организация не является членом [Партнерского центра Майкрософт,](https://docs.microsoft.com/partner-center/)вы можете провести сертификационные тесты и просмотреть репозитории моделей возможностей устройств. Вы не можете опубликовать свое устройство в каталоге сертифицированных устройств IoT.
- Если вы видите сообщение об идентификаторе сети Microsoft Partner Network, необходимо [создать учетную запись Центра партнеров.](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account) После утверждения учетной записи вы можете вернуться на портал и опубликовать свое устройство. Для получения дополнительной информации, [см.](https://support.microsoft.com/help/4340639/partner-center-account-faqs)

Перед публикацией устройства в каталоге сертифицированных устройств IoT прочитайте и примите соглашение о программе. Соглашение можно найти на странице настроек на портале Azure Certified for IoT.
