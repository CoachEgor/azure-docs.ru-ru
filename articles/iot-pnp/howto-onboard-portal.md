# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>Подключение к порталу Azure Certified for IoT

[Портал сертификации Azure для Интернета вещей](https://aka.ms/ACFI) можно использовать для следующих действий:

- Завершите процесс сертификации для вашего устройства IoT.
- Найдите Plug and Play Предварительная версия моделей возможностей устройства для Интернета вещей. Эти модели можно использовать для быстрого создания готовых устройств IoT и их интеграции с решениями.
- Опубликуйте модель возможностей устройства в общедоступном репозитории.

## <a name="sign-in"></a>Вход

Используйте свою _рабочую учетную запись_ Майкрософт для входа на портал [сертификации Azure для центра Интернета вещей](https://aka.ms/ACFI) . При входе на портал проверяется членство в центре партнеров Майкрософт:

- Если ваша организация уже является участником [центра партнеров Майкрософт](https://docs.microsoft.com/partner-center/), вам предоставляется доступ ко всем функциям на портале.
- Если ваша организация не является членом [центра партнеров Майкрософт](https://docs.microsoft.com/partner-center/), можно запустить тесты сертификации и просмотреть репозитории модели возможностей устройства. Вы не можете опубликовать устройство в каталоге сертифицированных устройств IoT.
- Если появится сообщение с уведомлением о том, что ваш идентификатор Microsoft Partner Network отсутствует, следует [создать учетную запись центра партнеров](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account). После утверждения учетной записи можно вернуться на портал и опубликовать устройство. Дополнительные сведения см. в разделе [часто задаваемые вопросы центра партнеров](https://support.microsoft.com/help/4340639/partner-center-account-faqs).

Перед публикацией устройства в каталоге сертифицированных устройств IoT прочтите и примите соглашение об использовании программы. Соглашение можно найти на странице Параметры на портале сертификации Azure для IoT.
