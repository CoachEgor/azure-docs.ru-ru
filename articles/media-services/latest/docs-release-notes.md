# <a name="azure-media-services-v3-documentation-updates"></a>Обновления документации служб мультимедиа Azure v3

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

>Получайте уведомления о том, когда следует повторно посетить эту страницу для обновлений, скопировав и вставив URL-адрес `https://docs.microsoft.com/api/search/rss?search=%22Azure+Media+Services+v3+documentation+updates%22&locale=en-us` в средство чтения RSS-канала.

В этой статье рассказывается о последних обновлениях документации по службам мультимедиа версии 3.

## <a name="june-2020"></a>Июнь 2020 г.

* Предварительная версия функции Live Video Analytics на IoT Edge стала общедоступной. Дополнительные сведения см. в разделе " [интерактивная аналитика видео" в](../live-video-analytics-edge/index.yml) документации по IOT Edge.
* Новые краткие руководства:

    * [Отправка, кодирование и потоковая передача содержимого с помощью портала](manage-assets-quickstart.md)
    * [Шифрование содержимого с помощью портала](encrypt-content-quickstart.md)

## <a name="april-2020"></a>Апрель 2020 г.

* Документация по Проигрывателю мультимедиа Azure перенесена в [документацию по Azure](../azure-media-player/azure-media-player-overview.md).
* Добавлен [динамический Потоковая трансляция с помощью краткого руководства по Open широковещания Studio (OBS)](live-events-obs-quickstart.md) . В нем показано, как создать динамический поток служб мультимедиа с помощью портал Azure и OBS.

## <a name="march-2020"></a>Март 2020 г.

Добавлен [динамический Потоковая трансляция с помощью краткого руководства Wirecast для телепотока](live-events-wirecast-quickstart.md) . Здесь показано, как создать динамический поток служб мультимедиа Azure с помощью портал Azure и Wirecastного потока.
