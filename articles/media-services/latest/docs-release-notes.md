# <a name="azure-media-services-v3-documentation-updates"></a>Обновления документации Azure Media Services v3

>Получайте уведомления о том, когда вернуться на эту страницу `https://docs.microsoft.com/api/search/rss?search=%22Azure+Media+Services+v3+documentation+updates%22&locale=en-us` для обновления, копируя и вставляя этот URL: в ваш rsS-канал читателя.

В этой статье рассказывается о последних обновлениях документации Media Services v3.

## <a name="april-2020"></a>Апрель 2020 г.

* Документы мультиплеера Azure были перенесены в [документацию Azure.](../azure-media-player/azure-media-player-overview.md)
* [Прямая трансляция с Open Broadcasting Studio (OBS)](live-events-obs-quickstart.md) quickstart был добавлен. Он показывает, как создать прямую трансляцию служб мультимедиа с помощью портала Azure и OBS.

## <a name="march-2020"></a>Март 2020 г.

[Прямая трансляция с Telestream Wirecast](live-events-wirecast-quickstart.md) quickstart был добавлен. Он показывает, как создать прямую трансляцию медиаслужбы Azure с помощью портала Azure и Telestream Wirecast.
