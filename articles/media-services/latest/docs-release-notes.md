# <a name="azure-media-services-v3-documentation-updates"></a>Обновления документации служб мультимедиа Azure v3

>Получите уведомления о том, когда следует повторно посетить эту страницу для получения обновлений путем копирования и вставления `https://docs.microsoft.com/api/search/rss?search=%22Azure+Media+Services+v3+documentation+updates%22&locale=en-us` этого URL-адреса в средство чтения канала RSS.

В этой статье рассказывается о последних обновлениях документации по службам мультимедиа версии 3.

## <a name="april-2020"></a>Апрель 2020 г.

* Проигрыватель мультимедиа Azure документы перенесены в [документацию Azure](../azure-media-player/azure-media-player-overview.md).
* Добавлен [динамический Потоковая трансляция с помощью краткого руководства по Open широковещания Studio (OBS)](live-events-obs-quickstart.md) . В нем показано, как создать динамический поток служб мультимедиа с помощью портал Azure и OBS.

## <a name="march-2020"></a>Март 2020 г.

Добавлен [динамический Потоковая трансляция с помощью краткого руководства Wirecast для телепотока](live-events-wirecast-quickstart.md) . Здесь показано, как создать динамический поток служб мультимедиа Azure с помощью портал Azure и Wirecastного потока.
