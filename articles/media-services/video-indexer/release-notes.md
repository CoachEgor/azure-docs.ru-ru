# <a name="azure-media-services-video-indexer-release-notes"></a>Заметки о выпуске видеоиндекса медиаиндекса От службы мультимедиа Azure

## <a name="january-2020"></a>Январь 2020 г.
 
### <a name="custom-language-support-for-additional-languages"></a>Пользовательская языковая поддержка дополнительных языков

Video Indexer теперь поддерживает `ar-SY` `en-UK`пользовательские `en-AU` модели языка для, и (только API).
 
### <a name="delete-account-timeframe-action-update"></a>Удаление обновления таймфрейма учетной записи

Удалить действие учетной записи теперь удаляет учетную запись в течение 90 дней вместо 48 часов.
 
### <a name="new-video-indexer-github-repository"></a>Новый видеоиндекс GitHub репозиторий

Новый видео индексер GitHub с различными проектами, получение начал руководства и образцы кода теперь доступны:https://github.com/Azure-Samples/media-services-video-indexer
 
### <a name="swagger-update"></a>Обновление Swagger

Video Indexer унифицирован **аутентификации** и **операции** в единый [видео индексер OpenAPI спецификации (swagger)](https://api-portal.videoindexer.ai/docs/services/Operations/export?DocumentFormat=OpenApiJson). Develpers можно найти AA в [Video Indexer Developer Portal](https://api-portal.videoindexer.ai/).

## <a name="december-2019"></a>Декабрь 2019 г.

### <a name="update-transcript-with-the-new-api"></a>Стенограмма обновления с новым API

Обновление конкретного раздела в стенограмме с помощью API [обновления-видео-индекса.](https://api-portal.videoindexer.ai/docs/services/Operations/operations/Update-Video-Index?&pattern=update)

### <a name="fix-account-configuration-from-the-video-indexer-portal"></a>Исправлена конфигурация учетной записи с портала Video Indexer

Теперь вы можете обновить конфигурацию подключения к media Services, чтобы самостоятельно помочь с такими проблемами, как: 

* неправильный ресурс Медиа-услуг Azure
* изменения пароля
* Ресурсы медиасервисов перемещались между подписками  

Чтобы исправить конфигурацию учетной записи, на портале Video Indexer перейдите на вкладку «Настройки > учетной записи» (как владелец).

### <a name="configure-the-custom-vision-account"></a>Настройка учетной записи пользовательского видения

Настроили учетную запись пользовательского видения на платных счетах с помощью портала Video Indexer (ранее это поддерживалось только API). Для этого вопийте на портале Video Indexer, выберите настройку модели > анимированные символы > Configure. 

### <a name="scenes-shots-and-keyframes--now-in-one-insight-pane"></a>Сцены, кадры и ключевые кадры - теперь в одном размахе понимания

Сцены, кадры и ключевые кадры теперь объединены в одно понимание для облегчения потребления и навигации. При выборе желаемой сцены вы можете увидеть, из каких снимков и ключевых кадров она состоит. 

### <a name="notification-about-a-long-video-name"></a>Уведомление о длинном имени видео

Когда имя видео длиннее 80 символов, Video Indexer показывает описательную ошибку при загрузке.

### <a name="streaming-endpoint-is-disabled-notification"></a>Конечная точка потоковой передачи отключена

Когда конечная точка потоковой передачи отключена, Video Indexer покажет описательную ошибку на странице игрока.

### <a name="error-handling-improvement"></a>Улучшение обработки ошибок

Код статуса 409 теперь будет возвращен из [Re-Index Video](https://api-portal.videoindexer.ai/docs/services/Operations/operations/Re-Index-Video?https://api-portal.videoindexer.ai/docs/services/Operations/operations/Re-Index-Video?) и [Update Video Index](https://api-portal.videoindexer.ai/docs/services/Operations/operations/Update-Video-Index?) AIS в случае активной индексации видео, чтобы предотвратить изменение текущих изменений реиндекса случайно.

## <a name="november-2019"></a>Ноябрь 2019 г.
 
* Поддержка корейских моделей пользовательских языков

    Видео индексер теперь поддерживает пользовательские модели языка на корейском языке ()`ko-KR`как в API, так и на портале. 
* Новые языки, поддерживаемые для речевого текста (STT)

    Video Indexer AIS теперь поддерживают STT на арабском языке Левантин (ar-SY), английском диалекте Великобритании (en-GB) и английском австралийском диалекте (en-AU).
    
    Для загрузки видео, мы заменили zh-HANS на zh-CN, оба поддерживаются, но zh-CN рекомендуется и более точно.
    
## <a name="october-2019"></a>Октябрь 2019 г.
 
* Поиск анимированных персонажей в галерее

    При индексации анимированных персонажей теперь можно искать их в видео-камбузе аккаунта. Для получения дополнительной информации, см [Анимированные символы признание](animated-characters-recognition.md).

## <a name="september-2019"></a>Сентябрь 2019 г.
 
Несколько достижений, объявленных на IBC 2019:
 
* Распознавание анимированных символов (публичный предварительный просмотр)

    Возможность обнаружения группового рекламного распознавания символов в анимированном контенте, через интеграцию с пользовательским видением. Для получения дополнительной информации [см.](animated-characters-recognition.md)
* Многоязычная идентификация (публичный предварительный просмотр)

    Обнаружить сегменты на нескольких языках в звуковой дорожке и создать многоязычную стенограмму на их основе. Первоначальная поддержка: английский, испанский, немецкий и французский. Для получения дополнительной информации [см. Автоматическая идентификация и транскрибирует многоязычный контент.](multi-language-identification-transcription.md)
* Названое извлечение сущности для людей и местоположения

    Извлекает бренды, места и людей из речи и визуального текста с помощью обработки естественного языка (NLP).
* Классификация типа редакционного выстрела

    Метки выстрелов с редакционными типами, такими как крупным планом, средний выстрел, два выстрела, в помещении, на открытом воздухе и т.д. Для получения дополнительной информации [см.](scenes-shots-keyframes.md#editorial-shot-type-detection)
* Тема вывод повышения - в настоящее время охватывающих уровень 2
    
    Тема вывод модели в настоящее время поддерживает более глубокую детализацию таксономии IPTC. Читайте полную информацию на [Лазурном Медиа Сервисе новых инноваций с помощью ИИ.](https://azure.microsoft.com/blog/azure-media-services-new-ai-powered-innovation/)

## <a name="august-2019"></a>Август 2019 г.
 
### <a name="video-indexer-deployed-in-uk-south"></a>Видео индексер развернуты в Великобритании на юге

Теперь вы можете создать платный счет Video Indexer в южном регионе Великобритании.

### <a name="new-editorial-shot-type-insights-available"></a>Новые редакционные Shot Тип идеи доступны

Новые теги, добавленные в видеоснимки, обеспечивают редакционные "типы выстрелов", чтобы идентифицировать их с общими редакционными фразами, используемыми в рабочем процессе создания контента, таких как: экстремальный крупный крупный бизнес, крупный крупный, широкий, средний, два выстрела, открытый, закрытый, левое лицо и правое лицо (доступно в JSON).

### <a name="new-people-and-locations-entities-extraction-available"></a>Доступно извлечение новых объектов и сущностей

Video Indexer определяет названные места и людей с помощью обработки естественного языка (NLP) из OCR и транскрипции видео. Video Indexer использует алгоритм машинного обучения, чтобы распознавать, когда конкретные места (например, Эйфелева башня) или люди (например, Джон Доу) вызываются в видео.

### <a name="keyframes-extraction-in-native-resolution"></a>Извлечение ключевых кадров в родном разрешении

Ключевые кадры, извлеченные Video Indexer, доступны в исходном разрешении видео.
 
### <a name="ga-for-training-custom-face-models-from-images"></a>GA для обучения пользовательских моделей лица из изображений

Обучающие лица из изображений перемещаются из режима Preview в GA (доступны через API и на портале).

> [!NOTE]
> В переходе "Предварительный просмотр к ГА" не оказывается никакого влияния на ценообразование.

### <a name="hide-gallery-toggle-option"></a>Скрыть опцию переключения галереи

Пользователь может скрыть вкладку галереи от портала (по аналогии с сокрытием вкладки образцов).
 
### <a name="maximum-url-size-increased"></a>Увеличен максимальный размер URL

Поддержка строки запроса URL 4096 (вместо 2048) при индексации видео.
 
### <a name="support-for-multi-lingual-projects"></a>Поддержка многоязычных проектов

Проекты теперь могут создаваться на основе видео, индексируемого на разных языках (только API).

## <a name="may-2019"></a>Май 2019 г.

### <a name="update-custom-language-model-from-closed-caption-file"></a>Обновление пользовательской модели языка из закрытого файла заголовка

[Создавайте пользовательские языковые модели](https://api-portal.videoindexer.ai/docs/services/Operations/operations/Create-Language-Model?&groupBy=tag) и [обновляйте пользовательские языковые модели](https://api-portal.videoindexer.ai/docs/services/Operations/operations/Update-Language-Model?&groupBy=tag) AI, теперь поддерживайте форматы файлов VTT, SRT и TTML в качестве ввода для языковых моделей.

При вызове [API стенограммы обновления видео](https://api-portal.videoindexer.ai/docs/services/Operations/operations/Update-Video-Transcript?&pattern=transcript), стенограмма добавляется автоматически. Модель обучения, связанная с видео, также обновляется автоматически. Для получения информации о том, как настроить и обучить свои языковые модели, [см.](customize-language-model-overview.md)

### <a name="new-download-transcript-formats--txt-and-csv"></a>Новые форматы загрузки - TXT и CSV

В дополнение к закрытому формату субтитров, уже поддержанным (SRT, VTT и TTML), Video Indexer теперь поддерживает загрузку стенограммы в форматах TXT и CSV.
