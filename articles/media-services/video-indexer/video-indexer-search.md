# <a name="find-exact-moments-within-videos"></a>Поиск конкретных кадров в видео

В этом разделе описываются параметры поиска, с помощью которых можно находить нужные кадры в видео.

1. Откройте веб-сайт [Индексатора видео](https://www.videoindexer.ai/) и выполните вход.
2. Выполните поиск среди всех видео в учетной записи.

    В следующем примере выполняется поиск всех видео, поговорим о безопасности и в котором содержится Сатья,

    ![поиска](./media/video-indexer-search/video-indexer-search01.png)
3. Выполните поиск сводных аналитических сведений о видео.

    После этого можно осуществлять поиск по конкретному видео, нажав кнопку **воспроизведения** для нужного ролика. Затем можно выполнять поиск по видео, открыв вкладку **Search** (Поиск). 

    В следующем примере мы отслеживаем «безопасное» внутри выбранного видео.

    ![поиска](./media/video-indexer-search/video-indexer-search02.png)

    Если щелкнуть один из результатов, проигрыватель откроет соответствующий кадр видео. В приложении можно настроить представление проигрывателя или аналитических сведений, а также синхронизацию. Дополнительные сведения см. в руководстве по [внедрению мини-приложений Индексатора видео в приложение](video-indexer-embed-widgets.md). 
4. Поиск подробной разбивки видео.
    
    Если вы хотите создать свои собственные коллекции на основании видео, которое можно найти, нажмите клавишу **изменить** кнопки. На этой странице отображаются видео вместе с его insights как фильтры. Дополнительные сведения см. в руководстве по [просмотру и редактированию аналитических сведений Индексатора видео](video-indexer-view-edit.md). 

    Вы можете выполнять поиск по видео для отображения только тех строк, которые интересуют и использовать их на стороне для фильтрации частей, которые вы хотите см. в разделе. Когда вы закончите, можно выполнить предварительный просмотр клипов и нажмите клавишу **публикации** для создания нового клип, который отображается в вашей коллекции.
    
    В следующем примере мы провели поиск текста «смешанной реальности». Кроме того, мы применили дополнительные фильтры, как показано на следующем рисунке.
    
    ![поиска](./media/video-indexer-search/video-indexer-search03.png)
