# <a name="find-exact-moments-within-videos"></a>Поиск конкретных кадров в видео

В этом разделе описываются параметры поиска, с помощью которых можно находить нужные кадры в видео.

1. Откройте веб-сайт [Индексатора видео](https://www.videoindexer.ai/) и выполните вход.
2. Выполните поиск среди всех видео в учетной записи.

    В следующем примере мы будем искать все видео, которые говорят о безопасности и в каком Сатья отображается,

    ![Поиск](./media/video-indexer-search/video-indexer-search01.png)
3. Выполните поиск сводных аналитических сведений о видео.

    После этого можно осуществлять поиск по конкретному видео, нажав кнопку **воспроизведения** для нужного ролика. Затем можно выполнять поиск по видео, открыв вкладку **Search** (Поиск). 

    В следующем примере мы будем искать «Secure» в выбранном видео.

    ![Поиск](./media/video-indexer-search/video-indexer-search02.png)

    Если щелкнуть один из результатов, проигрыватель откроет соответствующий кадр видео. В приложении можно настроить представление проигрывателя или аналитических сведений, а также синхронизацию. Дополнительные сведения см. в руководстве по [внедрению мини-приложений Индексатора видео в приложение](video-indexer-embed-widgets.md). 
4. Поиск подробной разбивки видео.
    
    Если вы хотите создать собственный клип на основе найденного видео, нажмите кнопку **изменить** . На этой странице показано видео с подробными сведениями о фильтрах. Дополнительные сведения см. в руководстве по [просмотру и редактированию аналитических сведений Индексатора видео](video-indexer-view-edit.md). 

    Вы можете выполнить поиск в видео, чтобы показать только интересующие вас строки, и использовать эту информацию для фильтрации нужных частей. По завершении вы можете просмотреть клип и нажать кнопку **опубликовать** , чтобы создать новый клип, который появится в галерее.
    
    В следующем примере мы искали текст "Mixed Reality". Кроме того, мы применили дополнительные фильтры, как показано на следующем рисунке.
    
    ![Поиск](./media/video-indexer-search/video-indexer-search03.png)
