# <a name="customize-a-language-model-with-the-video-indexer-website"></a>Настройка языковой модели с помощью веб-сайта Индексатора видео

Индексатор видео позволяет создавать пользовательские языковые модели для настройки распознавания речи путем передачи текста адаптации, а именно текста из предметной области, к словарю которой вы хотите адаптировать подсистему. Как только вы обучите свою модель, новые слова, появляющиеся в тексте адаптации, будут распознаваться.

Подробный обзор и рекомендации для пользовательских моделей языков можно найти в [модели Language с помощью Video Indexer.](customize-language-model-overview.md)

Вы можете использовать веб-сайт Индексатора видео для создания и изменения пользовательских языковых моделей в своей учетной записи, как описано в этом разделе. Кроме того, вы можете применять API, как описано в [этой статье](customize-language-model-with-api.md).

## <a name="create-a-language-model"></a>Создание языковой модели

1. Зайдите на веб-сайт [Video Indexer](https://www.videoindexer.ai/) и войдите в систему.
2. Чтобы настроить модель в учетной записи, выберите кнопку **настройки модели Content** в правом верхнем углу страницы.

   ![Настройка модели контента в Video Indexer](./media/content-model-customization/content-model-customization.png)

3. Выберите вкладку **Язык**.

    Откроется список поддерживаемых языков.

    ![Список моделей языков в Video Indexer](./media/customize-language-model/customize-language-model.png)

4. Под нужным языком выберите **модель Add.**
5. Введите имя языковой модели и нажмите клавишу ВВОД.

    Этот шаг создает модель и дает возможность загружать текстовые файлы в модель.

6. Чтобы добавить текстовый файл, выберите **Добавить файл**. Ваш исследователь файлов откроется.

7. Перейдите к текстовому файлу и выберите его. Вы можете добавить несколько текстовых файлов в языковую модель.

    Вы также можете добавить текстовый файл, выбрав кнопку **на** правой стороне модели Language и выбрав **файл Add.**

8. После того, как вы закончите загружать текстовые файлы, выберите зеленый вариант **Train.**

    ![Модель языка поезда в видео индексере](./media/customize-language-model/train-model.png)

Процесс обучения может занять несколько минут. По завершении рядом с моделью появится значение состояния **Trained** (Обученная). Вы можете просматривать, скачивать и удалять файл из модели.

![Обученная языковая модель в Video Indexer](./media/customize-language-model/preview-model.png)

### <a name="using-a-language-model-on-a-new-video"></a>Использование языковой модели в новом видео

Чтобы использовать модель Language в новом видео, сделайте одно из следующих действий:

* Выберите кнопку **Загрузка** в верхней части страницы.

    ![Кнопка загрузки Видео Индексер](./media/customize-language-model/upload.png)

* Бросьте аудио- или видеофайл в круг или просмотрите файл.

    ![Загрузить медиафайл Видео индексер](./media/customize-language-model/upload2.png)

Вам предоставляется возможность выбрать **язык исходного кода видео.** Выберите выпадение и выберите модель языка, созданную из списка. В ней должны быть указаны язык вашей языковой модели и заданное имя в скобках.

Выберите опцию **загрузки** в нижней части страницы, и новое видео будет проиндексировано с помощью модели Language.

### <a name="using-a-language-model-to-reindex"></a>Повторная индексация с помощью языковой модели

Чтобы использовать языковую модель для повторной индексации видео в вашей коллекции, перейдите в раздел **Account videos** (Видеозаписи в учетной записи) на домашней странице [Индексатора видео](https://www.videoindexer.ai/) и наведите указатель мыши на название нужного видео.

Вы увидите варианты отсвазаний видео, удаления видео и переиндексации видео. Выберите опцию для переиндексации видео.

![Переиндексировать с помощью видеоиндекса](./media/customize-language-model/reindex1.png)

Вам предоставляется возможность выбрать **язык исходного кода видео** для переиндексации видео. Выберите выпадение и выберите модель языка, созданную из списка. В ней должны быть указаны язык вашей языковой модели и заданное имя в скобках.

![Выберите язык исходного кода видео-Переиндексвидео видео с видео-индексом](./media/customize-language-model/reindex.png)

Выберите кнопку **Re-index,** и ваше видео будет переиндексировано с помощью модели Language.

## <a name="edit-a-language-model"></a>Изменение языковой модели

Вы можете отредактировать языковую модель, изменив ее имя, добавив в нее файлы и удалив их из нее.

Если вы добавите или удалите файлы из модели Language, вам придется обучить модель снова, выбрав опцию green **Train.**

### <a name="rename-the-language-model"></a>Переименование языковой модели

Вы можете изменить название модели Language, выбрав кнопку ellipsis **(...**) на правой стороне модели Language и выбрав **Rename.**

Введите новое имя и нажмите клавишу ВВОД.

### <a name="add-files"></a>Добавление файлов

Чтобы добавить текстовый файл, выберите **Добавить файл**. Ваш исследователь файлов откроется.

Перейдите к текстовому файлу и выберите его. Вы можете добавить несколько текстовых файлов в языковую модель.

Вы также можете добавить текстовый файл, выбрав кнопку ellipsis **(...)** на правой стороне модели Языка и выбрав **файл Add.**

### <a name="delete-files"></a>Удаление файлов

Чтобы удалить файл из модели Language, выберите кнопку ellipsis **(...**) на правой стороне текстового файла и выберите **Удалить**. Всплывает новое окно, сообщающее вам, что удаление не может быть отменено. Выберите опцию **«Удалить»** в новом окне.

Это действие полностью удалит файл из языковой модели.

## <a name="delete-a-language-model"></a>Удаление языковой модели

Чтобы удалить модель Языка из учетной записи, выберите кнопку ellipsis **(...**) на правой стороне модели Language и выберите **Удалить.**

Всплывает новое окно, сообщающее вам, что удаление не может быть отменено. Выберите опцию **«Удалить»** в новом окне.

Это действие полностью удалит языковую модель из вашей учетной записи. Любое видео, используюееееееееееееееееетумодельную модель Language, будет сохранять тот же индекс до тех пор, пока вы не проиндексировать видео. При переиндексации видео можно назначить новую модель языка для видео. В противном случае Video Indexer будет использовать свою модель по умолчанию для переиндексации видео.

## <a name="customize-language-models-by-correcting-transcripts"></a>Настройка языковых моделей путем исправления стенограмм

Video Indexer поддерживает автоматическую настройку языковых моделей на основе фактических исправлений, которые пользователи внося в транскрипции своих видео.

1. Чтобы внести коррективы в стенограмму, откройте видео, которое вы хотите отсеить из видео с вашей учетной записи. Выберите вкладку **Хронология.**

    ![Настройка вкладки Временной шкалы языковой модели-Видео Индексер](./media/customize-language-model/timeline.png)

1. Выберите значок карандаша, чтобы отсеять транскрипт вашей транскрипции.

    ![Настройка транскрипции модели для отодвилость от видео-индексатора](./media/customize-language-model/edits.png)

    Video Indexer фиксирует все строки, которые исправлены вами в транскрипции вашего видео, и автоматически добавляет их в текстовый файл под названием "Из редактирования транскриптов". Эти веки используются для переподготовки конкретной модели Language, которая использовалась для индексирования этого видео.
    
    Если при индексации этого видео не указана модель Language, все изменения для этого видео будут храниться в модели языка по умолчанию под названием "Адаптация учетной записи" в обнаруженном языке видео.
    
    В случае, если в одну строку внесено несколько прав, для обновления модели Language будет использоваться только последняя версия исправленной строки.  
    
    > [!NOTE]
    > Для настройки используются только текстовые исправления. Исправления, не связанные с фактическими словами (например, знаками препинания или пробелами), не включены.
    
1. В вкладке Language модели настройки модели «Язык» будут отображаться исправления транскрипта.

    ![Настройка языковой модели-Видео Индексер](./media/customize-language-model/customize.png)

   Чтобы посмотреть файл "Из взлома" для каждой из ваших моделей Языка, выберите его, чтобы открыть его.

    ![Из стенограммы edits-Video Indexer](./media/customize-language-model/from-transcript-edits.png)
