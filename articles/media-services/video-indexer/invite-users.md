# <a name="quickstart-invite-users-to-video-indexer"></a>Краткое руководство. Приглашение пользователей в Индексатор видео

Для совместной работы с коллегами вы можете пригласить их в свою учетную запись Индексатора видео. 

> [!NOTE]
> Добавлять или удалять пользователей может только администратор учетной записи.

## <a name="invite-new-users"></a>Приглашение новых пользователей

1. Войдите на веб-сайт [Индексатора видео](https://www.videoindexer.ai/). Убедитесь, что при подключении использовалась учетная запись администратора.
1. Если у вас есть права администратора, в правом верхнем углу вы увидите кнопку **Share account** (Общий доступ к учетной записи). Нажмите эту кнопку, чтобы пригласить пользователей. 

   ![Приглашение новых пользователей](./media/invite-users/share-account.png)
1. Добавьте адреса электронной почты пользователей, которых хотите добавить в свою учетную запись Индексатора видео:

    ![Приглашение пользователей в эту учетную запись](./media/invite-users/invite-to-account.png)
        
    >[!NOTE]
    > У всех приглашенных пользователей будет разрешение на чтение и запись для всех видео в вашей учетной записи.
1. Приглашенные пользователи получат по электронной почте сообщение со ссылкой и смогут получить доступ к учетной записи, после того как перейдут по ссылке для **присоединения к Индексатору видео** :

    ![Подтверждение](./media/invite-users/invite-msg.png)

    Чтобы получить доступ к учетной записи, пользователю следует нажать ссылку и присоединиться. 

## <a name="removing-existing-users"></a>Удаление существующих пользователей

Если вы хотите удалить пользователей, имеющих доступ к вашей учетной записи, нажмите значок **X** рядом с их именами:

![Удаление пользователей](./media/invite-users/remove-users.png)

При удалении пользователи не получают уведомление. После удаления пользователи не будут иметь права на вход.
