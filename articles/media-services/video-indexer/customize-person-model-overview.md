# <a name="customize-a-person-model-in-video-indexer"></a>Настройка модели Person в Индексаторе видео

Индексатор видео поддерживает распознавание знаменитостей в видео. Функция распознавания знаменитостей охватывает около миллиона лиц на основе часто запрашиваемых источников данных, таких как IMDB, Википедия и влиятельные лица LinkedIn. Фрагменты, которые не распознаются в индексатор видео по-прежнему обнаружены, но остаются без имени. Клиенты могут создавать пользовательские модели пользователя и включить индексатора видео для распознавания лиц, которые не распознаются по умолчанию. Клиенты могут создавать эти модели лица, задавая имя человека с файлы изображений лиц человека.  

Если ваша учетная запись предусматривает разные варианты использования, можно воспользоваться не сможет создать несколько моделей пользователя на одну учетную запись. Например если содержимое в вашей учетной записи должен быть распределены по разным каналам, можно создать отдельную модель пользователя для каждого канала. 

> [!NOTE]
> Каждая модель Person поддерживает до 1 миллиона людей и каждая учетная запись имеет ограничение в 50 человек моделей. 

После создания модели вы можете использовать ее указав идентификатор конкретной модели пользователя при отправке, индексации или переиндексации видео. Учебные курсы новое лицо видео, обновляет определенной пользовательской модели, который видео был связан с. 

Если вам не требуется поддержка нескольких моделей пользователя, не присваивайте вашему видео идентификатор модели пользователя во время отправки, индексации или переиндексации. В этом случае индексатора видео будет использовать модель по умолчанию пользователь в вашей учетной записи. 

Можно использовать индексатор видео веб-сайта можно изменить лиц, которые были обнаружены в видео и управлять несколькими пользовательские модели лица в вашей учетной записи, как описано в разделе [Настройка модели человека на веб-сайт](customize-person-model-with-website.md) раздела. Можно также использовать API, как описано в разделе [настроить модель человека, с помощью интерфейсов API](customize-person-model-with-api.md).
