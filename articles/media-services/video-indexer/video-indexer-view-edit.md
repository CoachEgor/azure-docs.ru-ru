# <a name="view-and-edit-video-indexer-insights"></a>Просмотр и редактирование аналитических сведений Индексатора видео

В этом разделе показано, как просматривать и редактировать аналитические сведения о видео в Индексаторе видео.

1. Откройте веб-сайт [Индексатора видео](https://www.videoindexer.ai/) и выполните вход.
2. Найдите видео, из которого необходимо создать аналитические сведения в Индексаторе видео. Дополнительные сведения см. в руководстве по [поиску определенных фрагментов в видео](video-indexer-search.md).
3. Нажмите кнопку **Воспроизвести**.

    На странице отображаются сводные аналитические сведения о видео. 

    ![Аналитика](./media/video-indexer-view-edit/video-indexer-summarized-insights.png)

4. Просмотрите сводные аналитические сведения о видео. 

    Сводные аналитические сведения отображают агрегированное представление данных: лица, ключевые слова, тональности. Например, вы увидите лица людей, диапазон времени, в течение которого отображается каждое лицо, и значение времени отображения в процентах.

    Проигрыватель и аналитические сведения синхронизируются. Например, если щелкнуть ключевое слово или строку расшифровки, проигрыватель откроет видео именно в этом месте. В приложении можно настроить представление проигрывателя или аналитических сведений, а также синхронизацию. Дополнительные сведения см. в руководстве по [внедрению мини-приложений Индексатора видео Azure в приложение](video-indexer-embed-widgets.md).
