# <a name="automatically-identify-the-spoken-language-with-language-identification-model"></a>Автоматизия разговорного языка с моделью идентификации языка

Video Indexer поддерживает автоматическую идентификацию языка (LID), которая представляет собой процесс автоматической идентификации содержимого разговорного языка из аудио и отправки файла мультимедиа для транскрибирования на доминирующем идентифицированном языке. 

В настоящее время LID поддерживает: английский, испанский, французский, немецкий, итальянский, китайский, японский, русский и португальский (бразильский). 

Убедитесь в том, чтобы просмотреть [Руководящие принципы и ограничения](#guidelines-and-limitations) раздел ниже.

## <a name="choosing-auto-language-identification-on-indexing"></a>Выбор автоматической идентификации языка при индексации

При индексации или [переиндексации](https://api-portal.videoindexer.ai/docs/services/operations/operations/Re-Index-Video?) видео с использованием `auto detect` API `sourceLanguage` выберите опцию в параметре.

При использовании портала перейдите на **сайт аккаунта** на главной странице [Video Indexer](https://www.videoindexer.ai/) и нависнете над названием видео, которое вы хотите переиндексировать. Справа внизу нажмите кнопку повторного индекса. В диалоге **видео Re-index** выберите *Автоматическое обнаружение* из окна для выпадения **языка исходного кода Видео.**

![автоматическое обнаружение](./media/language-identification-model/auto-detect.png)

## <a name="model-output"></a>Выходные данные модели

Видео Indexbes видео в соответствии с наиболее вероятным языком, если уверенность в том, что язык `> 0.6`. Если язык не может быть идентифицирован с уверенностью, он предполагает, что разговорный язык является английским. 

Модель доминирующего языка доступна в идеи `sourceLanguage` JSON как атрибут (под корень / видео / понимание). Соответствующий балл доверия также `sourceLanguageConfidence` доступен под атрибутом.

```json
"insights": {
        "version": "1.0.0.0",
        "duration": "0:05:30.902",
        "sourceLanguage": "fr-FR",
        "language": "fr-FR",
        "transcript": [...],
        . . .
        "sourceLanguageConfidence": 0.8563
      },
```

## <a name="guidelines-and-limitations"></a>Рекомендации и ограничения

* Автоматическая идентификация языка (LID) поддерживает следующие языки: 

    Английский, Испанский, Французский, Немецкий, Итальянский, Китайский, Японский, Русский и Португальский (бразильский).
* Несмотря на то, что Video Indexer поддерживает арабский (современный стандарт и левантийский), хинди и корейский языки, эти языки не поддерживаются в LID.
* Если аудио содержит языки, не ведомые выше, результат является неожиданным.
* Если Video Indexer не может идентифицировать язык`>0.6`с достаточно высокой уверенностью ( ), резервный язык английский язык.
* В настоящее время нет поддержки файла со смешанными языками аудио. Если аудио содержит смешанные языки, результат неожиданный. 
* Низкое качество звука может повлиять на результаты модели.
* Модель требует по крайней мере одну минуту речи в аудио.
* Модель предназначена для распознавания спонтанной разговорной речи (не голосовых команд, пения и т.д.).
