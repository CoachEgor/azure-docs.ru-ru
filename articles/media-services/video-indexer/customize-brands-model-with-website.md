# <a name="customize-a-brands-model-with-the-video-indexer-website"></a>Настройка модели торговых марок с помощью веб-сайта Индексатора видео

Индексатор видео поддерживает распознавание торговых марок в речи и визуальном тексте во время индексирования и повторного индексирования видео- и аудиосодержимого. Функция распознавания торговых марок идентифицирует упоминания о продуктах, службах и компаниях, предлагаемых базой данных торговых марок Bing. Например, если корпорацию Майкрософт упоминают в видео- или аудиосодержимом или она отображается в визуальном тексте видео, Индексатор видео распознает ее как торговую марку в содержимом. Настраиваемая модель торговых марок позволяет выбрать, будет ли Индексатор видео распознавать торговые марки из базы данных торговых марок Bing, исключать определенные торговые марки из обнаружения (по существу создавая список блокировок торговых марок) и включать торговые марки, которые должны быть частью вашей модели, но могут отсутствовать в базе данных торговых марок Bing (фактически создавая утвержденный список торговых марок).

Дополнительные сведения см. в [этой статье](customize-brands-model-overview.md).

Вы можете использовать веб-сайт Индексатора видео для создания, использования и изменения моделей торговых марок, обнаруженных в видео, как описано в этом разделе. Кроме того, вы можете использовать API, как описано в [этой статье](customize-brands-model-with-api.md).

## <a name="edit-the-settings-of-the-brands-model"></a>Изменение параметров модели торговых марок  

Вы можете указать, необходимо ли распознавать торговые марки из базы данных Bing. Для этого нужно изменить параметры модели торговых марок.

1. Откройте веб-сайт [Индексатора видео](https://www.videoindexer.ai/) и выполните вход.
2. Чтобы настроить модель в учетной записи, в правом верхнем углу страницы нажмите кнопку **Content model customization** (Настройка модели содержимого).
 
   ![Настройка модели содержимого](./media/content-model-customization/content-model-customization.png) 
3. Чтобы изменить торговую марку, выберите вкладку **Торговые марки**.

    ![Настройка модели торговых марок](./media/customize-brand-model/customize-brand-model.png)
4. Установите флажок **Показать торговые марки, предложенные Bing**, если вы хотите, чтобы Индексатор видео включил торговые марки, предлагаемые Bing. Если не нужно, чтобы Индексатор видео распознавал торговые марки, предложенные Bing в вашем содержимом, не устанавливайте этот флажок. 

## <a name="include-brands-in-the-model"></a>Включение торговых марок в модель

Раздел **Включить торговые марки** представляет торговые марки, которые должен распознавать Индексатор видео, даже если они не предлагаются Bing.  

### <a name="add-a-brand"></a>Добавление торговой марки

1. Щелкните "+ Добавить торговую марку".

    ![Настройка модели торговых марок](./media/customize-brand-model/add-brand.png)

    Укажите имя (обязательно), категория (необязательно), описание (необязательно) и URL-адрес ссылки (необязательно).
    Поле категории позволяет отметить тегом ваши торговые марки. В этом поле отображаются *теги* торговой марки при использовании API Индексатора видео. Например, торговая марка "Azure" может быть отмечена тегом или отнесена к категории "Облако".

    В поле URL-адреса ссылки может содержаться ссылка на любой справочный веб-сайт для торговой марки, например, ссылка на страницу в Википедии.
2. Щелкните "Добавить торговую марку", и вы увидите, что в список **Включить торговые марки** была добавлена торговая марка.

### <a name="edit-a-brand"></a>Изменение торговой марки

1. Щелкните значок карандаша рядом с торговой маркой, которую требуется изменить.

    Вы можете обновить категорию, описание или URL-адрес ссылки торговой марки. Изменить название торговой марки невозможно, так как ее имя является уникальным. Если вам нужно изменить название торговой марки, полностью удалите ее (перейдите к следующему разделу) и создайте новую торговую марку с новым именем.
2. Нажмите кнопку **Обновить**, чтобы обновить информацию о торговой марке.

### <a name="delete-a-brand"></a>Удаление торговой марки

1. Щелкните значок корзины рядом с торговой маркой, которую требуется удалить.
2. Щелкните "Удалить", после чего торговая марка больше не будет отображаться в списке *Включить торговые марки*.

## <a name="exclude-brands-from-the-model"></a>Исключение торговых марок из модели

Раздел **Исключить торговые марки** представляет торговые марки, которые Индексатор видео не должен распознавать.

### <a name="add-a-brand"></a>Добавление торговой марки

1. Щелкните "+ Добавить торговую марку".

    Укажите имя (обязательно) и категорию (необязательно).
2. Щелкните "Добавить торговую марку", и вы увидите, что в список *Исключить торговые марки* была добавлена торговая марка.

### <a name="edit-a-brand"></a>Изменение торговой марки

1. Щелкните значок карандаша рядом с торговой маркой, которую требуется изменить.

    Вы можете обновить только категорию торговой марки. Изменить название торговой марки невозможно, так как ее имя является уникальным. Если вам нужно изменить название торговой марки, полностью удалите ее (перейдите к следующему разделу) и создайте новую торговую марку с новым именем.
2. Нажмите кнопку **Обновить**, чтобы обновить информацию о торговой марке.

### <a name="delete-a-brand"></a>Удаление торговой марки

1. Щелкните значок корзины рядом с торговой маркой, которую требуется удалить.
2. Щелкните "Удалить", после чего торговая марка больше не будет отображаться в списке *Исключить торговые марки*.
