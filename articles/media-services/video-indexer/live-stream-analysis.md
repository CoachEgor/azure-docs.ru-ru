# <a name="live-stream-analysis-with-video-indexer"></a>Анализ потоковой передачи с помощью видеоиндекса

Video Indexer Media Services Azure — это сервис Azure, предназначенный для извлечения глубоких аналитических данных из видео- и аудиофайлов в автономном режиме. Это делается для анализа данного файла мультимедиа, уже созданного заранее. Тем не менее, для некоторых случаях использования важно, чтобы получить информацию о средствах массовой информации из живой корма как можно быстрее, чтобы разблокировать операционные и другие случаи использования нажата во времени. Например, такие богатые метаданные в прямом эфире могут быть использованы производителями контента для автоматизации производства ТВ.

Решение, описанное в этой статье, позволяет клиентам использовать Video Indexer в практически в режиме реального времени в реальных моментах на прямых каналах. Задержка с индексацией может составить всего четыре минуты с помощью этого решения, в зависимости от проиндексированных фрагментов данных, разрешения ввода, типа содержимого и вычислительной мощности, используемой для этого процесса.

![Метаданные видеоиндекса в прямом эфире](./media/live-stream-analysis/live-stream-analysis01.png)

*Рисунок 1 - Пример игрока, отображающего метаданные видеоиндекса в прямом эфире*

[Решение анализа потоков,](https://aka.ms/livestreamanalysis) с помощью azure Functions и двух логических приложений для обработки живой программы с живого канала в Службах мультимедиа Azure с помощью Video Indexer и отображает результат с помощью медиаплеера Azure, показывающего практически реальный поток.

На высоком уровне он состоит из двух основных шагов. Первый шаг выполняется каждые 60 секунд, и занимает подклип из последних 60 секунд играл, создает актив из него и индексирует его через Video Indexer. Затем второй шаг называется после завершения индексации. Полученные сведения обрабатываются, отправляются в Azure Cosmos DB, а индексируемая подклипа удаляется.

Образец игрока играет в прямом эфире и получает информацию от Azure Cosmos DB, используя специальную функцию Azure. Он отображает метаданные и эскизы в синхронизации с живым видео.

![Два логических приложения, обрабатывающие прямую трансляцию каждую минуту в облаке](./media/live-stream-analysis/live-stream-analysis02.png)
