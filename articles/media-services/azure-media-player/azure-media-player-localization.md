# <a name="localization"></a>Локализация #

Многоязычная поддержка позволяет пользователям неанглоязычных языков взаимодействовать с игроком. Azure Media Player мгновенно получит глобальный словарь языков, который будет локализовать сообщения об ошибках на основе языка страницы. Разработчик также может создать экземпляр игрока с языком принудительного набора. Языком по умолчанию является английский (en).

> [!NOTE]
> Эта функция все еще проходит некоторое тестирование и как таковая подвержена ошибкам.

```html
    <video id="vid1" class="azuremediaplayer amp-default-skin" data-setup='{"language":"es"}'>...</video>
```
