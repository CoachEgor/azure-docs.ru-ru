# <a name="azure-blockchain-workbench-database-views"></a>Представления базы данных Azure Blockchain Workbench

Azure Blockchain Workbench Preview предоставляет данные из распределенных реестров в *внецепную* базу данных S'L DB. Внецеповая база данных позволяет использовать S'L и существующие инструменты, такие как [S'L Server Management Studio,](https://docs.microsoft.com/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-2017)для взаимодействия с данными блокчейна.

Azure Blockchain Workbench содержит набор представления баз данных, который дает возможность получить доступ к данным. Это будет полезно при пополнении запросов. Эти представления слишком сильно денормализованные, чтобы облегчить быстрое создание отчетов, аналитики и других пользовательских данных блокчейна с помощью существующих инструментов и без необходимости повторного обучения сотрудников базы данных.

В этом разделе находятся обзоры представления баз данных и данные, которые они содержат.

> [!NOTE]
> Любое прямое использование таблиц баз данных, найденных в базе данных за пределами этих представлений, не поддерживается, хотя это и возможно.
>
