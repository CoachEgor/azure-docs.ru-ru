# <a name="what-is-azure-blockchain-tokens"></a>Что собой представляет служба блокчейн-токенов Azure?

[!INCLUDE [Preview note](./includes/preview.md)]

Токены блокчейна Azure — это служба PaaS (платформа как услуга) для стандартизированной выдачи токенов и управления ими по всем реестрам блокчейна в Azure.

С помощью службы "Токены блокчейна Azure" вы можете создавать стандартизированные токены для любых решений блокчейна на основе предварительно созданного шаблона токена. Кроме того, служба позволяет создавать собственные шаблоны токенов. Созданную службу "Токены блокчейна Azure" можно подключить к блокчейну, чтобы выдавать для него токены. Выданными токенами можно управлять сразу в нескольких блокчейн-сетях.

## <a name="templates"></a>Шаблоны

Выбирайте предварительно созданный шаблон токена или создавайте собственный с помощью службы "Токены блокчейна Azure". Служба "Токены блокчейна Azure" поддерживает концепцию компоновки шаблона токена, которая позволяет создать собственный шаблон токена на основе поддерживаемых схем поведения. Шаблоны токенов можно использовать для большинства решений блокчейна, так как они сопоставляются с большей частью распространенных токенов. Вы можете настроить готовый шаблон в соответствии со своими потребностями и развернуть токены для решения.

Дополнительные сведения о шаблонах службы "Токены блокчейна Azure" см. в [этой статье](templates.md).

## <a name="management"></a>Управление

Служба "Токены блокчейна Azure" предоставляет интерфейс управления на портале Azure и интерфейсы API для подключения к существующей сети блокчейна. Сейчас можно подключаться к [службе "Блокчейн Azure"](../service/overview.md) или другому блокчейну из семейства Ethereum.

После подключения к одной или нескольким блокчейн-сетям вы можете выдавать токены и управлять ими в составе решения блокчейна с помощью интерфейсов API службы "Токены блокчейна Azure". Интерфейсы API позволяют интегрировать управление токенами в бизнес-приложения и бизнес-логику. Например, можно управлять токенами через REST API, а не напрямую в блокчейн-системе.

## <a name="blockchains-and-accounts"></a>Блокчейны и учетные записи

Служба "Токены блокчейна Azure" предоставляет интерфейс управления на портале Azure и интерфейсы API для создания новых групп и учетных записей для существующих блокчейн-сетей. Вы можете создавать новые учетные записи непосредственно в подключенных сетях, а служба "Токены блокчейна Azure" будет управлять для них закрытыми ключами от вашего имени. С помощью групп можно объединять учетные записи блокчейна из нескольких сетей и управлять доступом на уровне групп.

Дополнительные сведения об управлении учетными записями в службе "Токены блокчейна Azure" см. в [этой статье](account-management.md).

## <a name="token-taxonomy-framework"></a>Платформа таксономии токенов

Токены блокчейна Azure основываются на стандартизированной платформе таксономии токенов. Платформа таксономии токенов предоставляет материалы, созданные рабочей группой [инициативы по таксономии токенов](https://entethalliance.org/participate/token-taxonomy-initiative/). Рабочая группа инициативы по таксономии токенов определяет бизнес-классификацию токенов и возможные схемы поведения, которые могут применяться ко всем основным реестрам (в том числе Ethereum, Quorum, Corda и Hyperledger Fabric). Цель рабочей группы — создание платформы для стандартизации использования токенов с точки зрения бизнеса. Такая платформа позволит сделать разработку на основе токенов более простой и доступной. Позволяя отрасли определять токены и их поведение на уровне бизнеса, мы отделяем систему реализации токенов от бизнес-логики для управления ими.
