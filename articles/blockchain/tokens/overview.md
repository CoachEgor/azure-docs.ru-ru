# <a name="what-is-azure-blockchain-tokens"></a>Что такое токены Azure Блокчейн?

[!INCLUDE [Preview note](./includes/preview.md)]

Токены Azure Блокчейн — это платформа как услуга (PaaS) для стандартизированного выдачи и управления маркерами в книгах блокчейн в Azure.

С помощью маркеров Azure Блокчейн вы можете создавать стандартизированные токены для решения блокчейн с помощью предварительно созданного шаблона токена. Вы также можете создать собственный шаблон токена с помощью службы. После создания Используйте токены Azure Блокчейн, чтобы подключить и выдать токены в блокчейн. После выпуска можно управлять маркерами в нескольких блокчейн сетях.

## <a name="templates"></a>Шаблоны

Используйте токены Azure Блокчейн для выбора предварительно созданного шаблона токена или создания собственного шаблона токена. Токены Azure Блокчейн поддерживают шаблон маркера компонуемости, который позволяет создать собственный шаблон токена на основе поддерживаемых поведений. Шаблоны маркеров можно использовать для большинства блокчейн решений, так как они сопоставляются с наиболее распространенными маркерами. Вы можете начать с шаблона, персонализировать его и развернуть маркеры для решения.

Дополнительные сведения о шаблонах маркеров Azure Блокчейн см. в статье [шаблоны токенов Блокчейн Azure](templates.md).

## <a name="management"></a>управления

Маркеры Блокчейн Azure предоставляют Управление портал Azure и интерфейсы API для подключения к существующей сети блокчейн. Сейчас вы можете подключиться к [службе Блокчейн Azure](../service/overview.md) или другому семейству Ethereum блокчейн.

После подключения к одной или нескольким блокчейн сетям можно использовать интерфейсы API маркеров Azure Блокчейн, чтобы выдать маркеры и управлять ими для использования в решении блокчейн. С помощью интерфейсов API можно интегрировать управление маркерами в бизнес-приложениях и логике. Например, можно использовать REST API для управления маркерами вместо управления токенами непосредственно в блокчейн.

## <a name="blockchains-and-accounts"></a>Блокчейна и учетные записи

Маркеры Блокчейн Azure предоставляют Управление портал Azure и интерфейсы API для создания новых групп и учетных записей блокчейн в подключенных сетях блокчейн. Вы можете создавать новые учетные записи непосредственно в подключенных сетях, а Azure Блокчейн tokens управляет закрытыми ключами вашей учетной записи от вашего имени. С помощью групп можно объединять разные учетные записи блокчейн из нескольких сетей и управлять доступом через группы.

Дополнительные сведения об управлении учетными записями маркеров Azure Блокчейн см. в статье [Управление учетными записями маркеров Блокчейн Azure](account-management.md).

## <a name="token-taxonomy-framework"></a>Инфраструктура таксономии токенов

Маркеры Блокчейн для Azure основаны на основанной на стандартах базе, именуемой структурой таксономии маркеров (TTF). TTF — это набор конечных результатов, созданный на основе рабочей группы токена " [инициатива по классификации маркеров](https://entethalliance.org/participate/token-taxonomy-initiative/) " (TTI). Рабочая группа TTI определяет бизнес-классификацию для маркеров и их поведения, которые могут применяться ко всем основным книгам, включая Ethereum, кворум, Кабели и структуру в главной книге. Цель Рабочей группы — создать платформу, которая стандартизует использование токенов из бизнес-перспективы для упрощения и разработки на основе маркеров более демократичным. Позволяя отрасли определять эти маркеры и их поведение на уровне бизнеса, детальная реализация маркеров отделяется от бизнес-логики, управляющей маркерами.
