# <a name="azure-blockchain-tokens-account-management"></a>Управление счетами блокчейн-токенов Azure

[!INCLUDE [Preview note](./includes/preview.md)]

Для решения блокчейна пользователям может потребоваться различные уровни доступа к токенам, созданным с помощью сервиса Blockchain Tokens Azure. В большинстве сценариев блокчейна необходимо планировать и развертывать различные блокчейн-аккаунты, которые существуют в книге. Вам также необходимо управлять доступом между участниками.Используя управление счетами аккаунтов Azure Blockchain Tokens, можно создавать группы и связывать блокчейн-аккаунты для контроля доступа к действиям блокчейна.

## <a name="blockchain-networks"></a>Блокчейн-сети

Токены Azure Blockchain позволяют развертывать и управлять токены в наборе блокчейн-сетей. Вы можете подключить к сервису единый блокчейн-реестр или несколько блокчейн-книг.

## <a name="accounts"></a>Учетные записи

Для блокчейн-сетей, подключенных к токенам Azure Blockchain, сервис создает и управляет парой ключей частного и публичного управления и выполняет подписание и отправку транзакций. Azure Blockchain Tokens также предоставляет отображение идентификационных данных для сопоставления учетных записей с идентификацией ключей общего пользования в книге.

## <a name="groups"></a>Группы

Группы позволяют управлять большим количеством блокчейн-аккаунтов в подключенных сетях. Вы можете отслеживать и проверять, какие приложения и пользователи в каталоге имеют возможность использовать учетные записи через ApIS Azure Blockchain Tokens. Например, можно сгруппировать набор учетных записей, представляющих различные направления бизнеса или различные роли и доступ к токенам блокчейна.

Вы также можете связать группу с пользователем или основным сервисом Azure Active Directory, и у этого принципала есть разрешения на группу и связанные с ней учетные записи.
