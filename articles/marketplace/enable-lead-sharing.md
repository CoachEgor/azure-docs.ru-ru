# <a name="enable-lead-sharing"></a>Возможность общего доступа к данным о потенциальных клиентах
Убедитесь, что управление потенциальными клиентами включено для вашего предложения в [Azure Marketplace](https://azuremarketplace.microsoft.com) или [Microsoft AppSource](https://appsource.microsoft.com), чтобы вы могли получать контактную информацию для следующих категорий аудитории.

| Онлайн-магазин | Аудитория |
|:--- |:--- |
| AppSource | Бизнес-клиенты |
| Azure Marketplace | Все клиенты |

Потенциальные клиенты обеспечивают базу для кампаний, перечисленных в следующей таблице, соответствующих рекламных акций ваших специалистов по продажам, а также позволяют получить сведения об эффективности приложения или предложения.

| Онлайн-магазин | Кампания |
|:--- |:--- |
| AppSource | Развитие |
| Azure Marketplace | Создание спроса |

Используйте следующие рекомендации по работе с интересами.
*   Определите потенциальных клиентов и оцените их в качестве возможностей для продаж.
*   Развивайте потенциальных клиентов для активирования канала продаж.
*   Совмещайте инициативы с общей стратегией маркетинговой кампании.

Потенциальные клиенты демонстрируют заинтересованность в следующих предложениях.

| Онлайн-магазин | Вывод списка |
|:--- |:--- |
| AppSource | <table> <tr><td>приложение</td></tr> <tr><td>Консультационные услуги</td></tr> </table> |
| Azure Marketplace | <table> <tr><td>Предложение</td></tr> <tr><td>Технология</td></tr> </table> |

У вас должен быть налажен процесс для работы с потенциальными клиентами, как описано в следующей таблице:

| Онлайн-магазин | Потенциальные клиенты |
|:--- |:--- |
| AppSource | <table> <tr><td>Система управления отношениями с клиентами (CRM)</td></tr> <tr><td>Специалист по продажам Microsoft CRM</td></tr> </table> |
| Azure Marketplace | <table> <tr><td>Azure Marketplace</td></tr> <tr><td>Специалист по продажам Microsoft CRM</td></tr> </table> |

Потенциальные клиенты онлайн-магазинов в Marketplace отличаются от потенциальных продавцов Майкрософт, что является отличительной чертой программы совместных продаж.

Дополнительные сведения о получении доступа к программе совместных продаж см. в статье [Развитие бизнеса с помощью Майкрософт](./promote-your-business-with-microsoft.md).
