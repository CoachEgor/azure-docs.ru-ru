# <a name="consulting-services-for-azure-marketplace-and-appsource"></a>Консультационные услуги для Azure Marketplace и AppSource
Консультационные услуги в Azure Marketplace и AppSource позволяют партнерам корпорации Майкрософт организовать подключение клиентов к службам, поддерживающим и расширяющим взаимодействие с Azure и связанные с ним бизнес-задачи. 

Azure Marketplace и AppSource поддерживают публикацию предложений с консультационными услугами системных интеграторов, сотрудничающих с корпорацией Майкрософт, чтобы они могли предоставить клиентам качественные консалтинговые услуги для взаимодействия с нужными экспертами. Они помогают клиентам оценить, изучить и развернуть правильные решения и реализации, помогающие реализовать бизнес-цели и выполнить поставленные задачи.  

Эти предложения консультационных услуг строго привязаны к конкретным заказчикам. Издатели имеют возможность выбрать тип приложения, определить область действия и продолжительность, стоимость (фиксированная или бесплатно), а также описать программу передачи клиенту результатов работы. Обращения потенциальных клиентов поступают в виде уведомлений в системе CRM партнера. На запросы клиентов следует реагировать не позднее 48 часов.

Полный контрольный список требований к регистрации издатели можно найти в статье [Become a Publisher in the Marketplace](https://docs.microsoft.com/azure/marketplace/become-publisher) (Как стать издателем в Marketplace).

[Портал Cloud Partner](https://cloudpartner.azure.com) используется издателями для предоставления информации и заявок по своим предложениям. 

>[!Note]
>Партнеру достаточно зарегистрироваться однократно, чтобы стать издателем. Получив по электронной почте приветственное сообщение с информацией для входа на портал Cloud Partner, издателю больше не нужно регистрироваться повторно для публикации предложений других типов.

## <a name="publishing-process"></a>Процесс публикации
Процесс публикации предложения консультационных услуг в Azure Marketplace и AppSource включает следующие этапы.

1. **Листинг**. Чтобы зарегистрировать новое предложение, войдите в учетную запись на [Портале Cloud Partner](https://cloudpartner.azure.com). Выберите в раскрывающемся меню "Новое предложение", а затем "Consulting Services". Введите артефакты для предложения и опубликуйте его.
2. **Проверка**. Ваш запрос будет проверен на соответствие требованиям к публикации. Затем мы отправим вам инструкции по размещению предложения консультационных услуг. 
3. **Отправка**. Предоставьте необходимую информацию в соответствии с инструкциями. Все содержимое и вспомогательные материалы должны быть на английском языке. 
4. **Проверка**. Наша группа маркетологов и редакторов изучит предоставленные материалы и скорректирует их для повышения разборчивости, грамотности и эффективности. 
5. **Промежуточный процесс**. Отправка является промежуточным процессом вашего отзыва. 
6. **Публикация**. Когда все промежуточное содержимое будет полностью приемлемым для вас и корпорации Майкрософт, листинг будет готовым для запуска в Marketplace. Дата активации: go-live зависит от страны или региона, который указан в ваше предложение. Если каталог для выбранной страны или региона уже активирован в Azure Marketplace или AppSource, оно появится в течение нескольких часов. В противном случае предложение станет доступным с момента публикации каталога.

## <a name="offer-listings-and-content-review-criteria"></a>Перечисление предложений и критерии проверки содержимого
Принимаются к рассмотрению предложения услуг следующих типов (вы можете разместить сразу несколько типов). 

* **Оценка**. Вычисление среды клиента, чтобы определить применимость решения и предоставить оценки затрат и времени. 
* **Совещание**. Представление решения или консультационной услуги для привлечения клиента с использованием схем, демонстрационных материалов и историй клиентов. 
* **Реализация**. Полная установка, предоставляющая клиенту полностью рабочее решение. Мы рекомендуем использовать только такие решения, которые реализуются не дольше двух недель. 
* **Подтверждение концепции**. Ограниченная область реализации, позволяющая определить применимость решения к требованиям клиента. 
* **Семинар**. Интерактивная вовлеченность на территории клиента. Сюда может входить обучение, брифинги, оценки и демонстрации, основанные на данных или рабочей среде клиента. 

**Примечание.** Все типы услуг, за исключением *совещаний*, можно предоставлять в личном порядке (в расположении клиента или на объекте партнера) или виртуально (с применением телеконференций, веб-конференций, удаленной реализации и т. д.). Брифинги должны проводиться в личном порядке.

После того как вы настроите необходимые элементы, перечисленные выше, команда проверки содержимого проверит, чтобы ваше описание предложения было информативным и исчерпывающим для целевой аудитории.
