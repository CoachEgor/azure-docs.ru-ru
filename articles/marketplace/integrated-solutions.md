# <a name="integrated-solutions-in-azure-marketplace-and-appsource"></a>Интегрированные решения в Azure Marketplace и AppSource

Интегрированное решение объединяет в себе технологии из лучших партнеров корпорации Майкрософт, а также консультации, развертывания и предоставляемые экспертные системы интеграторов (SI) поддерживает.

Эти решения предназначены для конкретного клиента требуется адрес или сценарий, который начинается с понимания отрасли запрос или болевые точки. Два или более из лучших технологий затем интегрированы с SI организовать работу единой и воспроизводимого развертывания для клиентов, для их настройки и запуска быстро. Системы, в которые интегратор также предоставляет единую точку для клиента, указав соответствующие настройки и консультации, обратитесь в службу и поддерживают вокруг решения.  Результат представляет собой интегрированное решение, включая технологии и службы.

## <a name="advantages"></a>Преимущества

Достоинства интегрированный подход

* *Опыт*: Системных интеграторов, предоставляя решения для определенных отраслей и знаний
* *Оставшееся время*: Интегрированные развернутые подход помогут вам понять преимущества
* *Из лучших решений*: Обнаружение простоту интегрированного приобретение и реализация интерфейса
