# <a name="azure-marketplace-and-appsource-roadmap"></a>Стратегия развития Azure Marketplace и AppSource

Этот документ содержит представление о новых возможностях для Microsoft коммерческих marketplace для издателей, продавцов и клиентов. Он охватывает некоторые из важных функций, мы были зафиксированы, приблизительные сроки при можно ожидать их и будет обновлен каждый месяц. Он не полный список всех новых функций, но предназначен для обеспечения видимости неполный. Эти наборы возможностей и интервалы времени реализации отражают текущие планы и могут быть изменены.

Ждем ваших предложений! Присоединяйтесь к обсуждениям в [сообществе партнеров Майкрософт](https://www.microsoftpartnercommunity.com/), чтобы получать последние новости о новых возможностях, программах и событиях.
