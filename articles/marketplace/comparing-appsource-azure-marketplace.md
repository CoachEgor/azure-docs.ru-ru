# <a name="comparing-microsoft-appsource-and-azure-marketplace"></a>Сравнение Microsoft AppSource и Azure Marketplace

В состав коммерческого рынка Майкрософт входит [Microsoft AppSource](https://appsource.microsoft.com) и [Azure Marketplace](https://azuremarketplace.microsoft.com). Каждый Интернет-магазин предлагает различные типы решений с уникальными сценариями, поэтому вы можете убедиться, что ваше решение или служба доступны для нужных клиентов.

## <a name="understanding-the-differences-between-online-stores"></a>Основные сведения о различиях между Интернет-магазинами

Чтобы выбрать Интернет-магазин, сначала укажите тип предлагаемого решения. Если ваше предложение является корпоративным и ИТ-решением, вы можете опубликовать один раз для продажи в нескольких магазинах.

| Azure Marketplace | AppSource |
|:--- |:--- |
| ИТ – решения | Бизнес-решения |

В следующей таблице описаны преимущества использования Интернет-магазина.

| Преимущества | Azure Marketplace | AppSource |
|:--- |:--- | :--- |
| **Гибкость выставления счетов** | Azure Marketplace поддерживает типы предложений Free, пробного, BYOL и Transact. Предложения Transact-based — на основе использования или подписки. Для предложений на основе потребления выставляются счета за почасовое использование и может быть предоставлен начальный бесплатный период. Счета за службы на основе подписок выставляются ежемесячно или ежегодно по тарифу на рабочее место или по фиксированной ставке.| AppSource поддерживает типы предложений Free, пробного, BYOL и Transact. Предложения Transact-это продукты SaaS на основе подписок, которые можно ежемесячно или ежегодно использовать для каждого места или для фиксированной ставки. |
|**Связи с другими партнерами**  | Сейчас вы не можете связать поставщика службы или партнеров доставки с вашим предложением. | Связывает независимых поставщиков программного обеспечения, системных интеграторов и поставщиков управляемых служб в рамках конкретных сценариев реализации. Вы можете совместно продавать службы или предложения новым клиентам. |
| **Служба автоматизации** | Сейчас вы не можете связать поставщика службы или партнеров доставки с вашим предложением. Используйте автоматизированную подготовку SaaS с помощью надстройки, а также возможность публикации шаблонов решений для приложений Azure. Эти шаблоны позволяют автоматизировать сценарии сбора данных и развертывания на основе SaaS. | Связывает независимых поставщиков программного обеспечения, системных интеграторов и поставщиков управляемых служб в рамках конкретных сценариев реализации. Вы можете совместно продавать службы или предложения новым клиентам. |
| **Несколько типов облака** | Публикуйте решения для общедоступного облака и локальной среды, используя следующие типы:<ul> <li>Azure Stack</li> <li>Azure для государственных организаций</li> <li>региональные облака, включая Китай и Германию.</li></ul> | Сейчас не поддерживает Azure Stack, Azure для государственных организаций или региональных облаков. |
| **Презентация в контексте для клиентов** | Сделайте свое решение доступным для контекстного поиска с использованием интерфейса Azure. Используйте возможности публикации шаблонов решений для виртуальных машин и приложений Azure. | Вы узнаете большему кругу клиентов, используя взаимодействие в приложении для продуктов Майкрософт, таких как Dynamics 365, Power BI и Microsoft 365. |
