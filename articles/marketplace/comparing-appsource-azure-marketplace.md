# <a name="comparing-microsoft-appsource-and-azure-marketplace"></a>Сравнение Microsoft AppSource и Azure Marketplace

Коммерческий рынок Майкрософт включает [microsoft AppSource](https://appsource.microsoft.com) и [Azure Marketplace.](https://azuremarketplace.microsoft.com) Каждый магазин обслуживает уникальные требования клиентов и различные целевые аудитории, так что вы можете убедиться, что ваше решение или услуга доступна для нужных клиентов.

## <a name="understanding-the-differences-between-storefronts"></a>Понимание различий между онлайн-магазинами

Чтобы выбрать витрину магазина, сначала определите целевую аудиторию для вашего предложения. Если вы ориентируетесь на аудиторию в нескольких онлайн-магазинах, достаточно выполнить однократную публикацию.

| Azure Marketplace | AppSource |
|:--- |:--- |
| ИТ-специалисты и разработчики | Бизнес-пользователи и лица, принимающие решения |

В следующей таблице описываются преимущества использования онлайн-магазина:

| Преимущества | Azure Marketplace | AppSource |
|:--- |:--- | :--- |
| **Гибкость выставления счетов** | Azure Marketplace поддерживает типы бесплатных, пробных, BYOL и серантов. Предложения о сделке основаны либо на основе потребления, либо на основе подписки. Предложения на основе потребления выставляются путем почасового использования и могут обеспечить первоначальный бесплатный период. Услуги на основе подписки выставляются ежемесячно или ежегодно по ставке на одно место или по фиксированной ставке.| AppSource поддерживает бесплатные, пробные, BYOL и сделки типов предложений. Предложения по сделке – это продукты SaaS на основе подписки, которые могут выставлять счета ежемесячно или ежегодно по месту или по фиксированной ставке. |
|**Связи с другими партнерами**  | Сейчас вы не можете связать поставщика службы или партнеров доставки с вашим предложением. | Связывает независимых поставщиков программного обеспечения, системных интеграторов и поставщиков управляемых служб в рамках конкретных сценариев реализации. Вы можете совместно продавать службы или предложения новым клиентам. |
| **Служба автоматизации** | Сейчас вы не можете связать поставщика службы или партнеров доставки с вашим предложением. Используйте автоматизированную подготовку SaaS с помощью надстройки, а также возможность публикации шаблонов решений для приложений Azure. Эти шаблоны позволяют автоматизировать сценарии сбора данных и развертывания на основе SaaS. | Связывает независимых поставщиков программного обеспечения, системных интеграторов и поставщиков управляемых служб в рамках конкретных сценариев реализации. Вы можете совместно продавать службы или предложения новым клиентам. |
| **Несколько типов облаков** | Публикуйте решения для общедоступного облака и локальной среды, используя следующие типы:<ul> <li>Azure Stack</li> <li>Azure для государственных организаций</li> <li>региональные облака, включая Китай и Германию.</li></ul> | Сейчас не поддерживает Azure Stack, Azure для государственных организаций или региональных облаков. |
| **Презентация в контексте для клиентов** | Сделайте свое решение доступным для контекстного поиска с использованием интерфейса Azure. Используйте возможности публикации шаблонов решений для виртуальных машин и приложений Azure. | Привлекайте большее количество клиентов через взаимодействие в приложении для продуктов Майкрософт, таких как Dynamics 365, Power BI и Office 365. |
