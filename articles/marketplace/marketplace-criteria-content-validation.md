# <a name="azure-marketplace-listing-guidelines"></a>Рекомендации по публикации в Azure Marketplace

Этот документ содержит требования и рекомендации по новым предложениям и службам, которые будут перечислены в Azure Marketplace.

Все предложения должны соответствовать требованиям, перечисленным в разделе "перечень требований для всех предложений".

Дополнительные требования и контрольные списки предоставляются для конкретных типов списков, в том числе: 
* [Требования к пробной версии](#trial-offer-requirements)
* [Требования к приложениям SaaS](#saas-app-requirements)
* [Требования к контейнерному предложению](#container-offer-requirements)
* [Требования к предложению консультационных услуг](#consulting-offer-requirements)
