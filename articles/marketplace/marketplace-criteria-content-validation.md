# <a name="azure-marketplace-listing-guidelines"></a>Рекомендации по публикации в Azure Marketplace

В этой статье описаны требования и рекомендации по публикации новых предложений и служб в Azure Marketplace.

Все предложения должны соответствовать требованиям, перечисленным в разделе [Требования к публикациям всех предложений](#listing-requirements-for-all-offers).

Дополнительные требования и контрольные списки предоставляются для определенных типов публикаций, в том числе: 
* [требования к пробной версии](#trial-offer-requirements);
* [требования к приложению SaaS](#saas-application-requirements);
* [требования к контейнерному предложению](#container-offer-requirements);
* [требования к предложению консультационных услуг](#consulting-offer-requirements).
