# <a name="publishing-checklist-for-appsource"></a>Публикация контрольных списков для AppSource  

Проверьте необходимые компоненты, прежде чем приступать к процессу публикации.  

Чтобы опубликовать приложение в AppSource, необходимы следующие артефакты.  

## <a name="checklist"></a>Контрольный список  

| Тип приложения | Тип предложения | Публикация артефакта |   
|:--- |:--- |:--- |  
| список | Консультационная услуга | <table> <tr><th>Подробные сведения об онлайн-магазинах</th></tr> <tr><td>Имя предложения (200 знаков)</td></tr> <tr><td>Описание (2000 знаков)</td></tr> <tr><td>Идентификатор MPN</td></tr> <tr><td>Доступность по странам и (или) регионам</td></tr> <tr><td>Применимые отрасли, категории и ключевые слова для поиска</td></tr> <tr><td>Снимки экрана (1280 х 720, максимум 5)</td></tr> <tr><td>Маркетинговые документы (максимум 3)</td></tr> <tr><td>Назначение потенциального клиента</td></tr> <tr><td>Логотипы компании (48 x 48, 216 x 216)</td></tr> <tr><td>Компетенции</td></tr> <tr><td>Видео с обзором продукта (необязательно)</td></tr> </table> |
