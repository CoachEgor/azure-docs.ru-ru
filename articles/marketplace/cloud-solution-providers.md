# <a name="cloud-solution-providers"></a>Поставщики облачных решений

Предложения программного обеспечения могут обращаться к миллионам квалифицированных клиентов Майкрософт, обслуживаемых партнерами в программе поставщика облачных решений (CSP), а также предоставлять общедоступные предложения через [Интернет-магазины Майкрософт](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Издатели настраивают предложения для обеспечения доступности в программе CSP на основе согласия (по умолчанию исключаются) для нового предложения или существующего, позволяя партнерам продавать продукты и создавать Объединенные решения для клиентов.

Издатели отвечают за предоставление поддержки по устранению неисправностей для конечных клиентов и на предоставление партнерам и/или клиентам механизма связи с вами для получения поддержки. Рекомендации по предоставлению партнеров CSP с пользовательской документацией, обработкой и уведомлениями о работоспособности и сбоях службы (как применимо), чтобы партнеры CSP могли обслуживать запросы на поддержку уровня 1 от клиентов.

В следующей таблице приведены типы предложений и модели ценообразования, которые могут использовать программу CSP:

| **Тип предложения**    | **BYOL**  |  **Бесплатный** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| Виртуальные машины  | Да | Да | Да |
| Контейнеры | Недоступно | Да | Недоступно |
| Шаблоны решений | Недоступно | Да | Недоступно |
| Управляемые приложения | Недоступно | Да | Да |
| Программное обеспечение как услуга (SaaS) | Недоступно | Да | Да |
|   |   |   |

## <a name="how-to-configure-an-offering"></a>Настройка предложения

Параметр "участие в программе CSP" настраивается в центре партнеров или Портал Cloud Partner предлагает возможность создания. (Дополнительные[сведения об изменении взаимодействия с издателем.](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/Cloud-Marketplace-In-Partner-Center/m-p/9738#M293))

В Портал Cloud Partner он задается на вкладке Marketplace или витрина. В центре партнеров он задается на вкладке Настройка предложения в разделе сведения о настройке.

![Возможности использования CSP в CPP](media/marketplace-publishers-guide/csp-opt-in.png)

Как показано ниже, в центре партнеров издатели могут принять участие в доступе к своим предложениям в программе CSP, установив флажок **сделать предложение доступным в программе CSP**.

![Участие в работе CSP в центре партнеров](media/marketplace-publishers-guide/pc-csp-opt-in.png)

## <a name="sharing-sales-and-support-materials-with-csp-partners"></a>Совместное использование материалов по продажам и поддержке с партнерами CSP

Чтобы позволить партнерам в программе поставщика облачных решений наиболее эффективно представить ваше предложение и привлечься к Организации, отправьте материалы по продажам и поддержке, которые будут доступны торговым посредникам. Эти ресурсы не будут доступны клиентам в витринах Marketplace.

Как показано ниже, в Портал Cloud Partner издатели могут ввести URL-адрес, который содержит релевантные маркетинговые материалы и контактные данные канала, в канал CSP:

![Вспомогательные сведения CSP](media/marketplace-publishers-guide/cpp-csp-information.png)
