# <a name="cloud-solution-providers"></a>Поставщики облачных решений

Предложения по теперь может насчитывать миллионы квалифицированные клиенты Майкрософт, обслуживаемых партнеров поставщика облачных решений (CSP) с минимальными инвестиций маркетинга и продаж.  Издатели могут теперь использовать каналы партнера CSP, помимо общей доступности предложений через [корпорации Майкрософт Интернет-магазины](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Чтобы подписаться на предложение для доступности в канале партнера CSP, для нового предложения или обновляете старую можно выбрать издателей. Этот параметр позволяет партнерам CSP продавать свои продукты, а также создавать пакетные решения для клиентов.

Издатели отвечают за предоставление поддержки для устранения неисправностей для конечных пользователей, а также для предоставления механизма для партнеров CSP и/или клиенты для связи с вами для поддержки. Это рекомендации для предоставления партнеров CSP с помощью пользовательской документации, обучения и службы уведомления о работоспособности/сбой (если применимо), чтобы партнеры CSP несут широкие возможности для обработки запросов поддержки 1 уровня от клиентов.

Ниже приведены типы предложений и модели ценообразования, можно выбрать канал партнера CSP:

| **Тип предложения**    | **BYOL**  |  **Бесплатный** | **(PAYG)**   |
| :---------------- | :---------|:----------|:-----------|
| Виртуальные машины  | Да | Да | Да |
| Контейнеры | Н/Д | Да | Н/Д |
| Шаблоны решений | Н/Д | Да | Н/Д |
| Управляемые приложения | Н/Д | Да | Да |
| ИТ-решение SaaS (модель ценообразования сайта) | Н/Д | Да | Да |
|   |   |   |
