# <a name="cloud-solution-providers"></a>Поставщики облачных решений

Предложения по может насчитывать миллионы квалифицированные клиенты Майкрософт, обслуживаемых партнеров по программе поставщика облачных решений (CSP), помимо общей доступности предложений через [корпорации Майкрософт Интернет-магазины](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Издатели Настройка предложения для доступности в программе CSP на основе согласием (по умолчанию не исключается), для нового предложения или обновляете старую, позволяя предоставлять партнерам продавать свои продукты и создайте упакованные этих решений для клиентов.

Издатели отвечают за предоставление поддержки для устранения неисправностей для конечных пользователей, а также для предоставления механизма для партнеров CSP и/или клиенты для связи с вами для поддержки. Это рекомендации для предоставления партнеров CSP с помощью пользовательской документации, обучения и службы уведомления о работоспособности/сбой (если применимо), чтобы партнеры CSP несут широкие возможности для обработки запросов поддержки 1 уровня от клиентов.

Ниже приведены типы предложений и модели ценообразования, можно включить в программу CSP:

| **Тип предложения**    | **BYOL**  |  **Бесплатный** | **(PAYG)**   |
| :---------------- | :---------|:----------|:-----------|
| Виртуальные машины  | Да | Да | Да |
| Контейнеры | Н/Д | Да | Н/Д |
| Шаблоны решений | Н/Д | Да | Н/Д |
| Управляемые приложения | Н/Д | Да | Да |
| Программное обеспечение как услуга (SaaS) | Н/Д | Да | Да |
|   |   |   |
