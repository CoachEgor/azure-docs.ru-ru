# <a name="cloud-solution-providers"></a>Поставщики облачных решений

Программное обеспечение предлагает может охватить миллионы квалифицированных клиентов Microsoft, обслуживаемых партнерами в программе Cloud Solution Provider (CSP), в дополнение к общедоступной доступности предложений через [веб-магазины Майкрософт.](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace)

Издатели настраивать предложения для доступности в программе CSP на основе выбора, для нового предложения или существующего, что позволяет партнерам продавать вашу продукцию и создавать комплексные решения для клиентов.

Издатели несут ответственность за предоставление поддержки устранения сбоа конечным клиентам и за предоставление механизма для партнеров в программе CSP и/или клиентов, чтобы связаться с вами для поддержки. Лучше всего предоставлять партнерам программы CSP пользовательские документы, обучение и уведомления о работоспособности/отключении услуг (по мере необходимости), чтобы партнеры в программе CSP были оснащены для обработки запросов поддержки первого уровня от клиентов.  

Следующие предложения могут быть выбраны для продажи партнерами в программе CSP:

- Программное обеспечение как-услуга (SaaS) сделки предложения
- Виртуальные машины
- Шаблоны решений
- Управляемые приложения

> [!NOTE]
> Контейнеры и принести свою собственную лицензию (BYOL) VM SKUs выбираются для продажи партнерами в программе CSP по умолчанию.

## <a name="how-to-configure-an-offering"></a>Как настроить предложение

Настройка отказа по программе CSP настроена в Партнерской центре или облачном партнерской портале, предлагая опыт создания. [Узнайте больше о меняющемся опыте издателя](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/Cloud-Marketplace-In-Partner-Center/m-p/9738#M293).

### <a name="partner-center-opt-in"></a>Отказ в участии в партнерской центре

В Partner Center вы найдете опыт выбора в рамках модуля аудитории CSP Reseller.

![Аудитория реселлеров CSP](media/marketplace-publishers-guide/csp-reseller-audience.png)

В модуле аудитории CSP Reseller у вас есть три варианта на выбор:

- Вариант первый: Любой партнер в программе CSP
- Вариант второй: Конкретные партнеры в программе CSP, которые я выбираю
- Вариант третий: Нет партнеров в программе CSP

#### <a name="option-one-any-partner-in-the-csp-program"></a>Вариант первый: Любой партнер в программе CSP

![Любой партнер программы CSP](media/marketplace-publishers-guide/csp-reseller-option-one.png)

 Выбирая этот вариант, все партнеры в программе CSP имеют право перепродать ваше предложение своим клиентам.

#### <a name="option-two-specific-partners-in-the-csp-program-i-select"></a>Вариант второй: Конкретные партнеры в программе CSP, которые я выбираю

![Конкретные партнеры в программе CSP, которые я выбираю](media/marketplace-publishers-guide/csp-reseller-option-two.png)

Выбирая этот вариант, вы разрешаете, какие партнеры в программе CSP имеют право перепродать ваше предложение.

Чтобы авторизовать партнеров, нажмите **Select CSP Partners** и появится меню, которое позволяет искать по имени партнера или идентификатору клиента CSP Azure Active Directory (AAD).

![Выберите меню CSP](media/marketplace-publishers-guide/csp-pop-up-module.png)

Вы можете применить фильтры поиска, такие как **страна,** **компетенция**или **навык.**

![Фильтры для поиска партнеров по стране, компетенции и навыкам](media/marketplace-publishers-guide/csp-add-resellers.png)

После того как вы выбрали список партнеров, выберите **Добавить.**

![Пример списка авторизованных партнеров в программе CSP](media/marketplace-publishers-guide/csp-add-resellers-details.png)

Таблица, отображая список выбранных партнеров, отображается на странице аудитории CSP Reseller.

![Таблица со списком партнеров на странице аудитории CSP Reseller](media/marketplace-publishers-guide/csp-option-two-add-reseller.png)

Выберите **проект Сохранить** для регистрации изменений.

Если это предложение не опубликовано, вам нужно опубликовать свое предложение, чтобы сделать его доступным для выбранных партнеров.

>[!NOTE]
>Если вы уполномочите партнера в программе CSP в данном регионе, они могут продать предложение любому клиенту, который принадлежит к этому конкретному региону. Подробнее о том, как классифицируются предложения CSP по регионам, читайте в материале [программы Cloud Solution Provider по региональным рынкам и валюте.](https://docs.microsoft.com/partner-center/regional-authorization-overview)

Если вы обновляете список CSP уже опубликованного предложения, добавьте дополнительных партнеров и выберите **аудиторию Sync CSP.**

Если у вас есть предложение, которое уже имеет список авторизованных партнеров, и вы хотели бы использовать тот же список для другого предложения, используйте **Импорт / Экспорт**. Перейдите к предложению, которое имеет список CSP и выберите **экспортных cSP.** Функция разрабатывает файл .csv, который может быть импортирован в другое предложение.

#### <a name="option-three-no-partners-in-the-csp-program"></a>Вариант третий: Нет партнеров в программе CSP

![Нет партнеров в программе CSP](media/marketplace-publishers-guide/csp-reseller-option-three.png)

Выбрав этот вариант, вы выбираете свое предложение из программы CSP. Вы можете изменить этот выбор в любое время.

### <a name="cloud-partner-portal-opt-in"></a>Отказ в отказе от облачного партнера

На портале Cloud Partner отказ установлен на вкладке Marketplace или Storefront. Возможность выбора конкретных партнеров в программе CSP доступна только в Partner Center.

![Опыт отказа cSP в CPP](media/marketplace-publishers-guide/csp-opt-in.png)

## <a name="deauthorize-partners-in-the-csp-program"></a>Деавторизовать партнеров в программе CSP

Если вы уполномочили партнера в программе CSP и этот партнер уже перепродал продукт своим клиентам, вам не будет разрешено деавторизовать этого партнера.

Если партнер в программе CSP не продал ваш продукт своим клиентам, и вы хотите удалить CSP после того, как ваше предложение было опубликовано, используйте следующие инструкции:

1. Перейдите на [страницу запроса поддержки](https://aka.ms/marketplacepublishersupport). Первые несколько меню выпадения автоматически заполняются для вас.

   > [!NOTE]
   > Не изменяйте преднаселенные выборы меню выпадающих.

2. Для **выбора версии продукта**, выберите Live управление **предложением**.
3. Для **выбора категории, которая лучше всего описать проблему,** выберите категорию, которая относится к вашему предложению.
4. Для **выбора проблемы, которая лучше всего описывает проблему,** выберите **Обновление существующее предложение.**
5. Выберите **Далее,** чтобы быть направленным на **страницу подробностей проблемы,** чтобы ввести более подробную информацию о вашей проблеме.
6. Используйте **Deauthorize CSP** в качестве заголовка проблемы и заполните остальные необходимые разделы.

## <a name="sharing-sales-and-support-materials-with-partners-in-the-csp-program"></a>Обмен материалами по продажам и поддержке с партнерами по программе CSP

Чтобы помочь партнерам в программе Cloud Solution Provider наиболее эффективно представлять ваше предложение и взаимодействовать с вашей организацией, необходимо представить материалы продаж и поддержки, которые будут доступны посредникам. Эти ресурсы не будут подвергаться воздействию клиентов на витринах маркетплейсов.
