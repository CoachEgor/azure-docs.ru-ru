# <a name="seller-insights-release-notes"></a>Заметки о выпуске функции Seller Insights 

(Дата выпуска: 1 марта 2019 г.)

Эта статья содержит сведения об изменениях в функции Seller Insights в [портале Microsoft Cloud Partner](https://cloudpartner.azure.com/#insights).

## <a name="release-highlights-for-march-1-2019"></a>Основные сведения о выпуске для 1 марта 2019 г.

* *Тренд клиента* добавлен к сводке
* *Первые пять заказчиков* на Сводка отражают всех подписок Azure, у клиента есть
* *Нормализовать тенденции использования & Active тренда заказов* на Сводка перемещен *число заказов с первого взгляда*
* *Отчет о выплатах сверки* обновлены
* *Первые пять заказчиков* на сведения о выплатах отражают всех подписок Azure, у клиента есть
* *Отчет об использовании* обновлены с Идентификатором клиента
* *Работы клиента* на заказы и использование отражает всех подписок Azure, у клиента есть


(Дата выпуска: 28 июля 2018 г.)

## <a name="release-highlights-for-july-28-2018"></a>Основные сведения о выпуске для 28 июля 2018 г.


-   *Оценка цен* обеспечивает клиентам представление о платежах с учетом последствий конвертации валют.
-   *Прогнозируемые выплаты* обеспечивают более раннее представление о потенциальных выплатах.
-  *Идентификаторы ссылок потребления* обеспечивают качество данных использования и заказов клиентов с выплатами
-   *Использование в ежедневном интервале* обеспечивает большую степень детализации и лучшие сведения об использовании клиентом.


### <a name="changes-to-data-structure-and-taxonomy"></a>Изменения в структуре данных и классификации

В следующей таблице перечислены метрики, которые были добавлены, или существенно изменены в этой версии. 

| **Создать термин**                   |    **Определение**                                                             |
|--------------------------------|  ---------------------------------------------------------------------------- |
| Цена (в валюте клиента)                     | Цена за единицу потребления заданного номера SKU (в валюте клиента).       |
| Оценка расширенных расходов (CC) | Оценка расширенных расходов за количество единиц потребления для данного номера SKU (в валюте клиента). Это значение может быть неточным из-за ошибок округления или усечения.   |
| Валюта издателя (PC)        | Валюта, предпочитаемая издателем для выплаты.                               |
| Оценка цены (PC)           | Оценка цены за единицу потребления для данного номера SKU, основанная на расчете конвертации иностранной валюты на дату потребления (в валюте издателя). Это значение может быть неточным из-за ошибок округления или усечения.   |
| Оценка расширенных расходов (PC) | Оценка расширенных расходов за количество единиц потребления для данного номера SKU, основанная на расчете конвертации иностранной валюты на дату потребления (в валюте издателя). Это значение может быть неточным из-за ошибок округления или усечения. |
| Оценка выплаты (PC)          | Оценка платежа за количество единиц потребления для данного номера SKU, основанная на расчете конвертации иностранной валюты на дату потребления (в валюте издателя). Это значение может быть неточным из-за ошибок округления или усечения.   |
| Ссылка на потребление                | Идентификатор одного или нескольких дней потребления клиентом заданного номера SKU, связанный с записью в отчете о выплатах. |
