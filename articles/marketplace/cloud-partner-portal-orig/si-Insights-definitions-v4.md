<a name="seller-insights-definitions"></a>Определения Seller Insights
=======================

В следующей таблице представлены определения многих терминов, используемых в Seller Insights.

|  **Термин**                 |  **Определение**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| **Тип лицензии Azure**                               | Тип лицензионного соглашения, используемого клиентами для покупок в Azure.  Другое название — канал.                                                  |
| **Тип лицензии Azure: поставщик облачных решений**      | Конечный пользователь покупает Azure и ваше предложение в Marketplace через своего поставщика облачных решений, который выступает в качестве торгового посредника.                 |
| **Тип лицензии Azure: Корпоративный**                   | Конечный пользователь покупает Azure и ваше предложение в Marketplace, подписывая корпоративное соглашение непосредственно с Майкрософт.                  |
| **Тип лицензии Azure: Корпоративный через торгового посредника**  | Клиент предоставляет Azure и предложение Marketplace через торгового посредника, который упрощает их Соглашение Enterprise с корпорацией Майкрософт.     |
| **Тип лицензии Azure: оплата по мере использования**                | Конечному пользователю необходимо приобрести Azure и предложение Marketplace с помощью соглашения с оплатой по мере использования, подписанного непосредственно с помощью корпорации Майкрософт.                |
| **Сумма оплаты (CC)**                              | Сумма, выплаченная клиенту в *валюте клиента* (CC) для выставления счетов.                                                                 |
| **Сумма оплаты (ПК)**                               | Сумма, выплаченная клиенту, в *валюте выплаты* (ПК).                                                                      |
| **Дата оплаты**                                      | Дата расчета оплаты клиента (обычно сразу после периода использования).                                             |
| **Имя облачного экземпляра**                              | Экземпляр Microsoft Cloud, в котором была развернута виртуальная машина                                   |
| **Имя облачного экземпляра: Azure Global**                | Общедоступное глобальное облако Майкрософт.                           |
| **Имя облачного экземпляра: Azure для государственных организаций**        | Облако Майкрософт для государственных организаций одной из следующих стран: Китай, Германия или США.                           |
| **Состояние коллекции**  | Для конкретной оплаты — Последнее состояние жизненного цикла выставления счетов и сбора.  Например, состояние может быть "сбор", "сбор" или "возврат денег".                  |
| **Размер ядра**  | Число виртуальных ядер, представленных виртуальной машиной, на которой работает предложение.               |
| **Город клиента**  | Название города, предоставленное клиентом. Это может отличаться от города в подписке Azure клиента.  |
| **Язык связи с клиентами**        | Язык, который клиент предпочитает при общении.                                  |
| **Название компании клиента**                  | Название компании, предоставленное клиентом. Это может отличаться от города в подписке Azure клиента.                                  |
| **Страна клиента**                       | Название страны или региона, предоставленное клиентом.  Это может отличаться от страны или региона в подписке Azure клиента.                               |
| **Валюта клиента (CC)**                 | Валюта, которую клиент предпочитает использовать для оплаты и выставления счетов.                            |
| **Адрес электронной почты клиента**                         | Адрес электронной почты, предоставленный конечным пользователем.  Это может отличаться от адреса электронной почты в подписке Azure клиента.                   |
| **Имя клиента**                              | Первое имя, предоставленное клиентом.  Это может отличаться от имени, указанного в подписке Azure клиента.                    |
| **Фамилия клиента**                               | Последнее имя, предоставленное клиентом.  Это может отличаться от имени, указанного в подписке Azure клиента.                    |
| **Идентификатор клиента**                               | Уникальный идентификатор, назначенный клиенту.  У клиента может быть не более одной подписки Azure Marketplace.                    |
| **Тип оплаты клиента**                            | Тип платежного средства, которое использует клиент.  Например, это может быть карта, счет или электронная проверка.                             |
| **Почтовый индекс клиента**                             | Почтовый индекс, предоставленный клиентом.  Это может отличаться от почтового кода, предоставленного в подписке Azure клиента.                                                                 |
| **Состояние клиента**                                   | Состояние (адрес), предоставленное клиентом.  Это может отличаться от фамилии, предоставленной в подписке Azure клиента.                                                                 |
| **Дата получения**                                    | Первая дата, когда клиент приобрел предложение, опубликованное Вами.                 |
| **Дата потери**                                        | Последняя дата, когда Клиент отменил Последнее предложение, приобретенное ранее.                                                         |
| **Предполагаемая Расширенная плата (CC)**                   | Оценка расширенной оплаты за количество единиц использования для данного номера SKU (в валюте клиента). Это значение может быть неточным из-за округления или связанных с усечением ошибок.                            |
| **Предполагаемая Расширенная плата (ПК)**                   | Расчетная сумма расширенной оплаты за количество единиц использования для данного номера SKU, основанная на конвертации внешнего обмена по дате использования, вычисляется (в валюте издателя). Это значение может быть неточным из-за округления или связанных с усечением ошибок.                  |
| **Предполагаемая выплата (ПК)**                            | Оценочный платеж по количеству единиц использования для данного номера SKU на основе конвертации внешнего обмена на дату, когда потребление вычисляется (в валюте издателя). Это значение может быть неточным из-за округления или связанных с усечением ошибок.                        |
| **Предполагаемая цена (ПК)**                             | Оценочная цена за единицу использования для данного номера SKU, основанная на конвертации внешнего обмена по дате использования, вычисляется (в валюте издателя). Это значение может быть неточным из-за округления или связанных с усечением ошибок.                                             |
| **Новый клиент**                                  | Это значение позволит вам узнать, используется ли новый клиент для приобретения одного или нескольких предложений Marketplace в первый раз (или нет).  Значение будет равно "Да", если это тот же календарный месяц, что и для "Дата получения".  Если клиент приобрел какие бы то ни было предложения до того, как отчетный месяц был приобретен, значение будет равно «нет».                                                       |
| **SKU предварительной версии**                                  | Это значение позволит вам выяснить, помечен ли SKU как "Предварительная версия". Значение будет равно "Да", если номер SKU указан соответствующим образом, и только подписки Azure, прошедшие авторизацию, могут развернуть и использовать этот образ.  Если номер SKU не был определен как "предварительный просмотр", значение будет равно "нет".                                          |
| **Согласие на участие в маркетинговых контактах**                   | Это значение позволит вам выяснить, был ли клиент заранее выбран для рекламного контакта от издателей. Сейчас этот параметр недоступен клиентам, поэтому повсеместно указано значение "Нет". Когда эта функция будет развернута, мы внесем соответствующие изменения в настройки.                                          |
| **Тип лицензии Marketplace**                         | Метод выставления счетов за предложение в Marketplace.                             |
| **Тип лицензии Marketplace: счет выставляется через Azure**   | Для такого предложения в Marketplace Майкрософт выступает в качестве вашего агента и выставляет клиентам счета от вашего имени. (Оплата по мере использования кредитной картой или счет-фактура на предприятие.)       |
| **Тип лицензии Marketplace: использование собственной лицензии** | Для развертывания виртуальной машины требуется лицензионный ключ, предоставленный клиентом. Майкрософт не выставляет клиентам счета за использование таких предложений в Marketplace. |
| **Тип лицензии Marketplace: бесплатный**                   | Предложение в Marketplace предоставляется бесплатно всем пользователям. Майкрософт не выставляет клиентам счета за использование таких предложений в Marketplace.    |
| **Тип лицензии Marketplace: Майкрософт в качестве торгового посредника**  | Для таких предложений в Marketplace Майкрософт выступает вашим торговым посредником.                               |
| **Название организации подписки Marketplace**  | Название компании, предоставленное клиентом в подписке Azure.                           |
| **Идентификатор подписки Marketplace**            | Уникальный идентификатор, связанный с подпиской Azure, которую клиент использовал для покупки вашего предложения Marketplace.  Раньше он был GUID подписки Azure.  В некоторых случаях, например в порядке SAAS, подписка не требуется.           |
| **Плата за Майкрософт (CC)**                               | Вознаграждение Майкрософт за транзакцию в валюте клиента.                              |
| **Название предложения**                                | Имя предложения Marketplace.                                                             |
| **Тип предложения**                                | Тип предложения Microsoft Marketplace.  Например, это может быть управляемое приложение, лицензии виртуальной машины или контейнер.                           |
| **Дата отмены заказа**                         | Дата, когда заказ в Marketplace был отменен.                                                       |
| **Идентификатор заказа**                                  | Уникальный идентификатор заказа клиента для службы Marketplace.  Предложения на основе использования виртуальных машин не связаны с заказом.                 |
| **Дата покупки заказа**                       | Дата создания заказа Marketplace.                                                        |
| **Количество заказов**                       | Количество заказов для одного номера SKU, связанного с подпиской Azure.  Для заказов SAAS он представляет количество рабочих мест, связанных с этим экземпляром заказа.                                            |
| **Состояние заказа**                              | Состояние заказа в Marketplace на момент последнего обновления данных.  Например, заказ может быть активным или отмененным.             |
| **Состояние заказа: активно**                           | Клиент приобрел заказ и не отменил свой заказ.   |
| **Состояние заказа: отменено**                        | Клиент ранее приобрел заказ и впоследствии отменил его заказ.        |
| **Сумма выплата (ПК)**                             | Сумма, выплачиваемая Вам, в предпочтительной валюте выплаты (ПК).                                       |
| **Валюта выплаты (ПК)**                           | Валюта, используемая для выплат на ваш счет.                                                                |
| **Дата выплаты**                                    | Дата отправки запроса на оплату от корпорации Майкрософт в выбранное финансовое учреждение.                                                                         |
| **Состояние вывыплаты**                                  | Состояние выплаты на момент последнего обновления данных.                                   |
| **Состояние выплаты: выполняется**               | Транзакция не готова к выплате. (Дополнительные сведения см. в разделе состояние коллекции.)                             |
| **Состояние вывыплаты: "оплачено"**                          | Транзакция была включена в прошлый расчет выплаты. Положительные значения выплачиваются, а отрицательные — вычитаются из общей суммы к оплате. |
| **Состояние вывыплаты: предстоящие выплаты**                   | Транзакция готова к выплате и будет включена в следующий расчет выплаты.                      |
| **Цена (CC)**                                   | Цена за единицу потребления определенного номера SKU в валюте клиента.                                                                  |
| **Электронная почта поставщика**                                 | Адрес электронной почты поставщика, участвующего в связи между корпорацией Майкрософт и конечным пользователем. Если клиент является предприятием через торгового посредника, он будет торговым посредником.  Если используется поставщик облачных решений (CSP), он будет CSP.                                                                 |
| **Имя поставщика**                                 | Имя поставщика, участвующего в связи между корпорацией Майкрософт и конечным пользователем. Если клиент является предприятием через торгового посредника, он будет торговым посредником.  Если используется поставщик облачных решений (CSP), он будет CSP.                                                                 |
| **URI ресурса**                                  | Путь, по которому был развернут артефакт развертывания.                                            |
| **SKU**                                           | Имя SKU, определенное во время публикации. Предложение может иметь много номеров SKU, но номер SKU может быть связан только с одним предложением.              |
| **Тип счета SKU**                                 | Метод выставления счетов SKU.                                                                                                             |
| **Тип счета SKU: использование собственной лицензии**         | Для развертывания виртуальной машины или службы требуется лицензионный ключ, предоставленный клиентом. Корпорация Майкрософт не оплачивает счета клиентов за использование этих предложений Marketplace.   |
| **Тип выставления счетов на SKU: от корпорации Майкрософт в качестве торгового посредника**          | Корпорация Майкрософт является вашим торговым посредником для этого SKU.   |
| **Тип выставления счетов SKU: бесплатный**                           | Номер SKU настраивается бесплатно для всех пользователей. Корпорация Майкрософт не оплачивает счета клиентам за использование этого SKU.                           |
| **Тип счета SKU: оплачено**                           | Корпорация Майкрософт является вашим агентом для клиентов SKU и выставляет счета от вашего имени. (Оплата по мере использования кредитной картой или счет-фактура на предприятие.)                   |
| **Тип счета SKU: пробная версия**                          | Клиент использует предложение в течение пробного периода и перейдет на платную версию, если не отменит или не удалит его.                                         |
| **Сумма налогов (CC)**                                  | Сумма налогов, применяемая к счету клиента в валюте клиента (CC).                                                               |
| **Дата транзакции**                                 | Дата транзакции, записанная в отчете о выплатах на ваш счет.                                                                           |
| **Тип транзакции**                               | Тип транзакции, о которой создается запись в отчете.  Например, это может быть оплата, возмещение или корректировка выплат.                                                                                        |
| **Дата окончания срока действия пробной версии**                                 | Дата, когда пробный период для определенного заказа закончится или закончился.                                                                           |
| **Дата использования**                                       | Дата, когда клиент потреблял услуги.                                                                                                             |
| **Количество использований**                                       | Сведения о потреблении клиентом номера SKU. Для образов виртуальных машин записи об использовании представляют сведения об использовании указанного периода для этого размера и SKU виртуальной машины.                                               |
| **Справочник по использованию**                                       | Идентификатор одного или нескольких дней потребления клиентом заданного номера SKU, связанный с записью в отчете о выплатах.                                               |
| **Usage Type** (Тип использования)                                       | Описание измеряемого потребления (Нормализованное использование или необработанное использование)                                                              |
| **Тип использования: нормализованное использование**                | Часы потребления, нормализованные с учетом количества использованных ядер виртуальных машин: [количество ядер виртуальных машин] x [количество часов фактического потребления]. Для виртуальных машин, используемых в режиме общего ядра (SHAREDCORE), в качестве множителя [количество ядер виртуальной машины] используется значение 1/6 (или 0,1666).                                        |
| **Тип использования: необработанное использование**                        | Количество часов потребления вашего предложения в Marketplace.  Она представлена с точки зрения "календаря" часов.  Это число запущенных виртуальных машин.                           |
| **Единицы использования**                                   | Единица измерения для указанного потребления. Использование виртуальных машин всегда измеряется в часах.                               |
| **Размер виртуальной машины**                                       | Представляет "размер" оборудования виртуальной машины, соответствующий определенному предложению в Azure, например `Basic_A0`, `Standard_A11`, `Standard_D12` или `Standard_G4`.   |
|  |  |
