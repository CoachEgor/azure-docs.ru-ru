<a name="seller-insights-definitions"></a>Определения Seller Insights
=======================

В следующей таблице представлены определения многих терминов, используемых в Seller Insights.

|  **Термин**                 |  **Определение**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| **Тип лицензии Azure**                               | Тип лицензионного соглашения, используемого клиентами для покупок в Azure.  Другое название — канал.                                                  |
| **Тип лицензии Azure: поставщик облачных решений**      | Конечный пользователь покупает Azure и ваше предложение в Marketplace через своего поставщика облачных решений, который выступает в качестве торгового посредника.                 |
| **Тип лицензии Azure: корпоративная**                   | Конечный пользователь покупает Azure и ваше предложение в Marketplace, подписывая корпоративное соглашение непосредственно с Майкрософт.                  |
| **Тип лицензии Azure: корпоративная, через торгового посредника**  | Конечный клиент закупает предложение Azure и Marketplace через реселлера, который облегчает их корпоративное соглашение с корпорацией Майкрософт.     |
| **Тип лицензии Azure: оплата по мере использования**                | Конечный клиент закупает предложение Azure и Marketplace в рамках соглашения «Оплата, как вы едете», подписанного непосредственно с корпорацией Майкрософт.                |
| **Начисленная сумма (в валюте клиента)**                              | Сумма, взимаемая с клиента, в *клиентской валюте* (CC) для выставления счетов.                                                                 |
| **Начисленная сумма (в валюте выплаты)**                               | Сумма, взимаемая с клиента, в *валюте выплат* (PC).                                                                      |
| **Дата начисления платы**                                      | Дата расчета платежа клиента (обычно сразу после периода использования).                                             |
| **Имя облачного экземпляра**                              | Экземпляр Microsoft Cloud, в котором была развернута виртуальная машина                                   |
| **Название экземпляров облака: Azure Global**                | Общедоступное глобальное облако Microsoft.                           |
| **Название экземпляра облака: Правительство Azure**        | Специфические для правительства Microsoft облака для одного из следующих правительств: Китай, Германия или Соединенные Штаты Америки.                           |
| **Статус коллекции**  | Для конкретного заряда, последний статус цикла биллинга и сбора.  Например, статус может быть сбор в процессе, сбор или возврат.                  |
| **Размер ядра**  | Количество виртуальных ядер, представленных виртуальной машиной, на которую работает предложение.               |
| **Город клиента**  | Название города предоставлено заказчиком. Это может отличаться от города в подписке Azure клиента.  |
| **Язык общения с клиентом**        | Язык, который клиент предпочитает при общении.                                  |
| **Название компании клиента**                  | Название компании предоставлено заказчиком. Это может отличаться от города в подписке Azure клиента.                                  |
| **Страна клиента**                       | Название страны/региона, предоставляемое заказчиком.  Это может отличаться от страны/региона в подписке Azure клиента.                               |
| **Валюта клиента**                 | Валюта, которую клиент предпочитает использовать для оплаты и выставления счетов.                            |
| **Электронная почта клиента**                         | Адрес электронной почты, предоставленный конечным клиентом.  Это может отличаться от адреса электронной почты в подписке Azure клиента.                   |
| **Имя клиента**                              | Имя, предоставленное клиентом.  Это может отличаться от имени, предусмотренного в подписке Azure клиента.                    |
| **Фамилия клиента**                               | Фамилия, предоставленная клиентом.  Это может отличаться от имени, предусмотренного в подписке Azure клиента.                    |
| **Идентификатор клиента**                               | Уникальный идентификатор, присвоенный клиенту.  Клиент может иметь нулевую или более подписку На Azure Marketplace.                    |
| **Тип оплаты клиента**                            | Тип платежного средства, которое использует клиент.  Например, это может быть карта, счет-фактура или электронный чек.                             |
| **Почтовый индекс клиента**                             | Почтовый индекс, предоставленный клиентом.  Это может отличаться от почтового индекса, предусмотренного в подписке На Azure клиента.                                                                 |
| **Регион клиента**                                   | Состояние (адрес), предоставленное клиентом.  Это может отличаться от фамилии, представленной в подписке Azure клиента.                                                                 |
| **Дата приобретения**                                    | В первый день клиент приобрел любое предложение, опубликованное вами.                 |
| **Дата отмены**                                        | Последняя дата, когда клиент отменил последнее из всех ранее приобретенных предложений.                                                         |
| **Расширенная смета затрат (в валюте клиента)**                   | Предполагаемая расширенная плата за количество единиц использования для данного СКУ (в валюте клиента). Это значение может быть неточным из-за округления или связанных с усечением ошибок.                            |
| **Расширенная смета затрат (в валюте издателя)**                   | Расчетная расширенная плата за количество единиц использования данного СКУ, основанная на конвертации иностранной валюты на дату использования рассчитывается (в валюте издателя). Это значение может быть неточным из-за округления или связанных с усечением ошибок.                  |
| **Сметная сумма выплаты (в валюте издателя)**                            | Расчетный платеж за количество единиц использования данного СКУ, основанный на конвертации иностранной валюты на дату расчета использования (в валюте издателя). Это значение может быть неточным из-за округления или связанных с усечением ошибок.                        |
| **Ориентировочная цена (в валюте издателя)**                             | Рассчитывается ориентировочная цена за единицу использования данного СКУ, основанная на конвертации иностранной валюты на дату использования (в валюте издателя). Это значение может быть неточным из-за округления или связанных с усечением ошибок.                                             |
| **Является ли новый клиент**                                  | Значение даст вам знать, если это новый клиент потребления или покупки одного или нескольких ваших предложений Marketplace в первый раз (или нет).  Значение будет "Да", если это тот же календарный месяц для "Дата приобретена".  Значение будет "Нет", если клиент приобрел любое из ваших предложений до календарного месяца сообщили.                                                       |
| **Является Предварительный SKU**                                  | Значение даст вам знать, если вы отметили SKU как "предварительный просмотр". Значение будет "Да", если SKU был помечен соответствующим образом, и только авторизованные вами подписки Azure могут развернуть и использовать это изображение.  Значение будет "Нет", если SKU не был определен как "предварительный просмотр".                                          |
| **Является ли рекламный контакт Opt In**                   | Значение даст вам знать, если клиент активно выбрал для рекламных контактов с издателями. Сейчас этот параметр недоступен клиентам, поэтому повсеместно указано значение "Нет". Когда эта функция будет развернута, мы внесем соответствующие изменения в настройки.                                          |
| **Тип лицензии Marketplace**                         | Метод выставления счетов за предложение в Marketplace.                             |
| **Тип лицензии Marketplace: выставление счета через Azure**   | Для такого предложения в Marketplace Майкрософт выступает в качестве вашего агента и выставляет клиентам счета от вашего имени. (Оплата по мере использования кредитной картой или счет-фактура на предприятие.)       |
| **Тип лицензии Marketplace: использование собственной лицензии (BYOL)** | Для развертывания VM требуется ключ лицензии, предоставленный заказчиком. Майкрософт не выставляет клиентам счета за использование таких предложений в Marketplace. |
| **Тип лицензии Marketplace: бесплатная**                   | Предложение в Marketplace предоставляется бесплатно всем пользователям. Майкрософт не выставляет клиентам счета за использование таких предложений в Marketplace.    |
| **Тип лицензии Marketplace: Майкрософт в качестве торгового посредника**  | Для таких предложений в Marketplace Майкрософт выступает вашим торговым посредником.                               |
| **Название компании-подписки Marketplace**  | Название компании, представленное клиентом в подписке Azure.                           |
| **Идентификатор подписки Marketplace**            | Уникальный идентификатор, связанный с подпиской Azure, который клиент использовал для покупки предложения Marketplace.  Ранее это был ГРАФИЧЕСКИй интерфейс подписки Azure.  В некоторых случаях, как заказ SAAS, подписка не требуется.           |
| **Вознаграждение Майкрософт (в валюте клиента)**                               | Вознаграждение Майкрософт за транзакцию в валюте клиента.                              |
| **Название предложения**                                | Название предложения Marketplace.                                                             |
| **Тип предложения**                                | Тип предложения Microsoft Marketplace.  Например, это может быть управляемое приложение, виртуальные лицензии машины или контейнер.                           |
| **Дата отмены заказа**                         | Дата, когда заказ в Marketplace был отменен.                                                       |
| **Идентификатор заказа**                                  | Уникальный идентификатор заказа клиента для вашего сервиса Marketplace.  Предложения на основе использования виртуальной машины не связаны с заказом.                 |
| **Дата покупки по заказу**                       | Дата, когда был создан заказ в Marketplace.                                                        |
| **Заказанное количество**                       | Количество заказов на тот же SKU, связанное с подпиской Azure.  Для заказов SAAS он представляет количество мест, связанных с экземпляром заказа.                                            |
| **Состояние заказа**                              | Состояние заказа в Marketplace на момент последнего обновления данных.  Например, заказ может быть активным или отмененным.             |
| **Состояние заказа: активен**                           | Клиент приобрел заказ и не отменил свой заказ.   |
| **Состояние заказа: отменен**                        | Клиент ранее приобрел заказ и впоследствии отменил свой заказ.        |
| **Сумма выплаты (в валюте выплаты)**                             | Выплачиваемая вам сумма в вашей предпочитаемой валюте.                                       |
| **Валюта выплаты**                           | Валюта, используемая для выплат на ваш счет.                                                                |
| **Дата выплаты**                                    | Дата отправки платежного запроса от корпорации Майкрософт выбранному вами финансовому учреждению.                                                                         |
| **Состояние выплаты**                                  | Состояние выплаты на момент последнего обновления данных.                                   |
| **Статус выплат: В процессе**               | Транзакция не готова к выплате. (Для получения дополнительной информации см. Статус коллекции)                             |
| **Состояние выплаты: выплачено**                          | Транзакция была включена в прошлый расчет выплаты. Положительные значения выплачиваются, а отрицательные — вычитаются из общей суммы к оплате. |
| **Состояние выплаты: предстоящая выплата**                   | Транзакция готова к выплате и будет включена в следующий расчет выплаты.                      |
| **Цена (в валюте клиента)**                                   | Цена за единицу потребления определенного номера SKU в валюте клиента.                                                                  |
| **Поставщик электронной почты**                                 | Адрес электронной почты поставщика, участвующего в отношениях между корпорацией Майкрософт и конечным клиентом. Если клиент является предприятием через реселлера, он будет реселлером.  Если в этом участвует поставщик облачных решений (CSP), это будет CSP.                                                                 |
| **Имя поставщика**                                 | Имя поставщика, участвующего в отношениях между корпорацией Майкрософт и конечным клиентом. Если клиент является предприятием через реселлера, он будет реселлером.  Если в этом участвует поставщик облачных решений (CSP), это будет CSP.                                                                 |
| **Универсальный код ресурса (URI)**                                  | Путь развертывания артефакта развертывания.                                            |
| **номер SKU**                                           | Указанное в момент публикации имя SKU. У предложения может быть несколько номеров SKU, но номер SKU может быть связан только с одним предложением.              |
| **Тип начисления платы за SKU**                                 | Метод начисления платы за определенный номер SKU.                                                                                                             |
| **SKU Биллинг Тип: Принесите свою собственную лицензию**         | Для развертывания VM или сервиса требуется ключ лицензии, предоставленный клиентом. Корпорация Майкрософт не выставляет счета клиентам за использование этих предложений Marketplace.   |
| **Тип выставления счетов SKU: Microsoft как реселлер**          | Для таких SKU Майкрософт выступает вашим торговым посредником.   |
| **Тип начисления платы за SKU: бесплатно**                           | SKU предоставляется бесплатно всем пользователям. Майкрософт не выставляет клиентам счета за использование таких SKU.                           |
| **Тип начисления платы за SKU: платно**                           | Для таких SKU Майкрософт выступает в качестве вашего агента и выставляет клиентам счета от вашего имени. (Оплата по мере использования кредитной картой или счет-фактура на предприятие.)                   |
| **Тип начисления платы за SKU: пробная версия**                          | Клиент использует предложение в течение пробного периода и перейдет на платную версию, если не отменит или не удалит его.                                         |
| **Сумма налога (в валюте клиента)**                                  | Сумма налогов, включенных в счет в валюте клиента.                                                               |
| **Дата сделки**                                 | Дата транзакции, записанная в отчете о выплатах на ваш счет.                                                                           |
| **Тип транзакции**                               | Тип транзакции, о которой создается запись в отчете.  Например, это может быть корректировка на сбор, возврат или выплата.                                                                                        |
| **Дата окончания срока действия пробной версии**                                 | Дата, когда пробный период для определенного заказа закончится или закончился.                                                                           |
| **Дата использования**                                       | Дата, когда клиент потреблял услуги.                                                                                                             |
| **Количество использования**                                       | Заявленное потребление SKU клиентом. Для образов виртуальных машин записи о потреблении представляют потребление за отчетный период для соответствующего размера виртуальной машины и SKU.                                               |
| **Ссылка на данные о потреблении**                                       | Идентификатор одного или нескольких дней потребления определенного номера SKU, связанный с записью в отчете о выплатах.                                               |
| **Тип использования**                                       | Описание измеряемого потребления (нормализованное потребление или фактическое потребление).                                                              |
| **Тип использования: Нормализованное использование**                | Часы потребления, нормализованные с учетом количества использованных ядер виртуальных машин: [количество ядер виртуальных машин] x [количество часов фактического потребления]. Для виртуальных машин, используемых в режиме общего ядра (SHAREDCORE), в качестве множителя [количество ядер виртуальной машины] используется значение 1/6 (или 0,1666).                                        |
| **Тип использования: Использование сырья**                        | Количество часов потребления вашего предложения в Marketplace.  Он представлен с точки зрения "календарных" часов.  Это количество времени виртуальные машины были запущены.                           |
| **Единицы потребления**                                   | Единица измерения для указанного потребления. Использование виртуальных машин всегда измеряется в часах.                               |
| **Размер VM**                                       | Представляет "размер" оборудования виртуальной машины, соответствующий определенному предложению в Azure, например `Basic_A0`, `Standard_A11`, `Standard_D12` или `Standard_G4`.   |
|  |  |
