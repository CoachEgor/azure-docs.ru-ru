# <a name="dynamics-365-solution-preparation"></a>Подготовка решения Dynamics 365

Система решений Dynamics 365 является платформой для формирования пакетов, установки и удаления компонентов, которые обеспечивают конкретные бизнес-функции. Решения используются независимыми поставщиками программного обеспечения и другими партнерами Microsoft Dynamics 365 для распределения расширений, которые они создают.

Если вы являетесь существующим независимым поставщиком программного обеспечения Dynamics 365 (xRM), скорее всего, вы уже создали управляемое решение и имеете файл solution.zip. В решении убедитесь, что поля "Отображаемое имя" и "Описание" отображают необходимое для пользователей содержимое. Эти данные отображаются в центре администрирования CRM Online.

![CRMScreenShot1](media/CRMScreenShot1.png)

_**Примечание**. В следующем примере пакета предполагается, что имя решения — "SampleSolution.zip"_

Если вы являетесь новым независимым поставщиком программного обеспечения, дополнительные сведения о создании решения можно получить здесь:[https://msdn.microsoft.com/library/gg334530.aspx](https://msdn.microsoft.com/library/gg334530.aspx)

Если ваше решение требует поддержки данных выполните следующие действия.

* Создайте демонстрационные данные в среде тестирования.
* Используйте средство переноса конфигурации для создания схемы с правилами сравнения данных.
* Сохраните схему конфигурации с файлами проекта. Вам понадобится это позже при обновлении данных конфигурации.
* Экспортируйте данные конфигурации. Обязательно укажите имя для файла, имеющее значение для экспорта.
