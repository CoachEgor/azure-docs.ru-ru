# <a name="cloud-partner-portal-migration-to-partner-center"></a>Миграция Портала Cloud Partner в Центр партнеров

Чтобы упростить управление учетной записью и предложить опыт управления, корпорация Майкрософт выполняет миграцию предложений с Портал Cloud Partner в центр партнеров. Если вы в настоящее время управляете сопутствующими материалами в одном коммерческом каталоге для партнеров (OCP GTM), вы сможете сразу же обновить свои материалы по совместной продаже в центре партнеров.

| **Тип предложения** | **Управление предложениями, перенесенное из Портал Cloud Partner в центр партнеров** | **Возможность обновления материалов для совместной продажи в центре партнеров** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| Программное обеспечение как услуга (SaaS) |     Уже доступна       |   Общая доступность, 2020 апреля      |
| Dynamics 365 для участия клиентов & PowerApps      |     Уже доступна        |   Уже доступна      |
| Dynamics 365 for Operations     |     Уже доступна       |   Уже доступна     |
| Dynamics 365 Business Central      |     Уже доступна        |   Общая доступность, 2020 апреля     |
| Приложение Azure     |     Уже доступна       |   Общая доступность, 2020 апреля     |
| Виртуальная машина Azure      |     Общедоступная доступность — Май 2020       |   Будет объявлено позднее    |
| Контейнер Azure     |     Общедоступная доступность — Май 2020       |   Будет объявлено позднее     |
| Консультационная услуга      |     Общедоступная доступность — Май 2020       |   Будет объявлено позднее     |
| Модуль IoT Edge      |     Общедоступная доступность — Май 2020       |   Будет объявлено позднее     |
| Приложение службы Power BI     |     Общедоступная доступность — Май 2020       |   Будет объявлено позднее     |
