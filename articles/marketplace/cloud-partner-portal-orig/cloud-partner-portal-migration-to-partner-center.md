# <a name="cloud-partner-portal-migration-to-partner-center"></a>Миграция Портала Cloud Partner в Центр партнеров

Чтобы упростить управление учетной записью и предложить опыт управления, корпорация Майкрософт выполняет миграцию предложений с Портал Cloud Partner в центр партнеров. Если вы в настоящее время управляете сопутствующими материалами в одном коммерческом каталоге для партнеров (OCP GTM), вы сможете сразу же обновить свои материалы по совместной продаже в центре партнеров.

| **Тип предложения** | **Управление предложениями, перенесенное из Портал Cloud Partner в центр партнеров** | **Возможность обновления материалов для совместной продажи в центре партнеров** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| Программное обеспечение как услуга (SaaS) |     Доступно сейчас       |   Общая доступность около февраля. 2020      |
| Dynamics 365 для участия клиентов & PowerApps      |     Доступно сейчас        |   Доступно сейчас      |
| Dynamics 365 for Operations     |     Доступно сейчас       |   Доступно сейчас     |
| Dynamics 365 Business Central      |     Доступно сейчас        |   Общая доступность около февраля. 2020     |
| Приложение Azure     |     Общедоступная доступность, Декабрь 2019       |   Общая доступность около февраля. 2020     |
| Виртуальная машина Azure      |     Будет известно позже       |   Будет известно позже    |
