# <a name="cloud-partner-portal-migration-to-partner-center"></a>Миграция Портала Cloud Partner в Центр партнеров

Чтобы оптимистить свою учетную запись и предложить опыт управления, корпорация Майкрософт переиздает предложения из облачного портала партнеров в партнерский центр. Если в настоящее время вы управляете залогом со-продажи в каталоге One Commercial Partner Go-to-Market (OCP GTM), вы сможете обновить свой сопутствующий залог в Partner Center в ближайшее время.

| **Тип предложения** | **Управление предложениями перенесено с облачного партнерского портала в партнерский центр** | **Возможность обновления сопродающего залога в Партнерской центре** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| Программное обеспечение как услуга (SaaS) |     Уже доступна       |   Общая доступность апреля 2020 г.      |
| Динамика 365 для взаимодействия с клиентами & PowerApps      |     Уже доступна        |   Уже доступна      |
| Dynamics 365 for Operations     |     Уже доступна       |   Уже доступна     |
| Dynamics 365 Business Central      |     Уже доступна        |   Общая доступность апреля 2020 г.     |
| Приложение Azure     |     Уже доступна       |   Общая доступность апреля 2020 г.     |
| Виртуальная машина Azure      |     Общая доступность апреля-май 2020       |   Будет объявлено позднее    |
| Контейнер Azure     |     Общая доступность апреля-май 2020       |   Будет объявлено позднее     |
| Консультационная услуга      |     Общая доступность апреля-май 2020       |   Будет объявлено позднее     |
