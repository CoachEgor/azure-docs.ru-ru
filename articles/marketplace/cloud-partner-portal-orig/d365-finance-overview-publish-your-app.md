# <a name="bring-your-microsoft-dynamics-365-business-central-app-into-microsoft-appsource"></a>Перенос приложения Microsoft Dynamics 365 Business Central в Microsoft AppSource

Существует множество преимуществ использования Dynamics 365 Business Central в качестве платформы для создателей приложений:

-   Расширение Dynamics 365 Business Central, проверенного интернет-решения Майкрософт, с помощью собственного опыта.
-   Использование торговой марки Dynamics 365 — известной торговой марки, которой доверяют миллионы пользователей.
-   Расширение круга пользователей бизнес-приложений с помощью Microsoft AppSource.
-   Достижение лучших результатов с помощью платформы, которая предоставляет современный интерфейс и возможность масштабирования.
-   Объединение с интеллектуальными бизнес-приложениями, такими как Microsoft PowerApps, Microsoft Flow, Power BI, Cortana Intelligence и многими другими.

Для переноса приложения Business Central в Microsoft AppSource выполните следующее.

1.  Создайте учетные записи.
2.  Обратитесь к нам со своей идеей приложения.
    - Разработайте технические аспекты приложения.
    - Разработайте маркетинговые аспекты приложения.
4.  *Опубликуйте приложение*.
