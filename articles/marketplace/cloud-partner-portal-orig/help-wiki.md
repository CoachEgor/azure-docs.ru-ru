# <a name="dynamics-365-for-operations-help-wiki"></a>Вики-сайт справки Dynamics 365 for Operations

Microsoft Dynamics 365 for Operations является бизнес-приложением корпорации Майкрософт\' для компаний. С помощью более "интеллектуального" пользовательского интерфейса, который обеспечивает доступ к аналитическим сведениям и данным в реальном времени, пользователи могут принимать более обоснованные решения. Используя обновленный пользовательский интерфейс, компании могут изменяться и использовать новаторский подход путем внедрения признанной бизнес-логики, что позволит им быстрее изменять их бизнес-процессы. Это позволяет компаниям расти в своем собственном темпе и вести бизнес практически в любом месте и в любое время. С его помощью компании также могут глобально масштабировать свои операции, что позволяет соответствовать потребностями бизнеса.

В данном руководстве приведены сведения о том, как использовать Dynamics 365 for Operations для бизнеса.

Дополнительные сведения см. в статье [Документация по Microsoft Dynamics 365 for Finance and Operations](https://docs.microsoft.com/dynamics365/operations/).
