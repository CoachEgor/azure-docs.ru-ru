# <a name="containers"></a>Контейнеры

[Портал Cloud Partner](https://cloudpartner.azure.com/) больше не поддерживает создание предложений контейнеров Azure и управление ими. Эта функция была перемещена в [Центр партнеров](https://partner.microsoft.com/pcv/) Майкрософт. Дополнительные сведения см. в статье [Создание предложения по контейнерам Azure](../../partner-center-portal/create-azure-container-offer.md).
