# <a name="container-support-tab"></a>Вкладка "Поддержка контейнеров"

Используйте вкладку **Поддержка** на странице **Новое предложение**, чтобы предоставить технические ресурсы и пользовательскую поддержку для предложения.  Поля ввода будут предоставляться по форма службы поддержки: **Проектирование контакт**, **поддержки**, и **поддерживает URL-адреса**. Символ звездочки (*) после имени поля указывает, что оно является обязательным.

## <a name="support-form-fields"></a>Поля формы службы поддержки

Введите требуемые сведения в полях "Контактные данные технической службы" и "Поддержка клиентов".

### <a name="engineering-contact"></a>Контактные данные технической службы

Контактные данные технической службы — это канал для связи между вашей организацией и компанией Майкрософт для решения технических вопросов. Требуется указать следующие сведения:

- **Имя** — имя лица или группы, предоставляющих техническую поддержку.
- **Адрес электронной почты** — адрес электронной почты для связи по техническим вопросам.
- **Телефон** — Номер телефона для получения технической поддержки.

### <a name="customer-support"></a>Служба поддержки пользователей

Служба поддержки пользователей получает соответствующие запросы, открытые клиентами в Azure Требуется указать следующие сведения:

- **Имя** — имя лица или группы, отвечающих за поддержку клиентов.
- **Адрес электронной почты** — адрес электронной почты для связи по техническим вопросам.
- **Телефон** — номер телефона службы поддержки клиентов.

## <a name="next-steps"></a>Дальнейшие действия
