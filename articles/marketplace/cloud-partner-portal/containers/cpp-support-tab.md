# <a name="container-support-tab"></a>Вкладка "Поддержка контейнеров"

Используйте вкладку **Поддержка** на странице **Новое предложение**, чтобы предоставить технические ресурсы и пользовательскую поддержку для предложения.  Поля ввода будут предоставляться по форма службы поддержки: **Проектирование контакт**, **поддержки**, и **поддерживает URL-адреса**. Символ звездочки (*) после имени поля указывает, что оно является обязательным.


## <a name="support-form-fields"></a>Поля формы службы поддержки

Введите требуемые сведения в полях "Контактные данные технической службы" и "Поддержка клиентов".


### <a name="engineering-contact"></a>Контактные данные технической службы

Контактные данные технической службы — это канал для связи между вашей организацией и компанией Майкрософт для решения технических вопросов. Обязательные поля помечены звездочкой (*).

- **Имя\***  -имя пользователя или группы, который служит в качестве technical/проектирование поддержки.
- **Адрес электронной почты\***  -адрес этого техническим вопросам электронной почты.
- **Телефон\***  -номер телефона для технической поддержки.


### <a name="customer-support"></a>Поддержка клиентов

Служба поддержки пользователей получает соответствующие запросы, открытые клиентами в Azure  Обязательные поля помечены звездочкой (*).

- **Имя\***  -имя пользователя или группы, который служит в качестве поддержки клиентов.
- **Адрес электронной почты\***  -адрес этого контакта поддержка электронной почты.
- **Телефон\***  -номер телефона для поддержки клиентов.
