# <a name="publish-container-offer"></a>Публикация предложения контейнера

> [!IMPORTANT]
> С 13 апреля 2020 года мы начнем перемещение управления предложениями Azure Container в партнерский центр. После миграции вы будете создавать и управлять своими предложениями в Partner Center. Следуйте инструкциям в [предложении Create an Azure Container](https://aka.ms/CreateContainerOffer) для управления мигрированными предложениями.

 Опубликовать предложение можно после его создания на странице **Новое предложение**. Выберите **Публиковать**, чтобы начать процесс публикации.

На следующей схеме представлены основные шаги в процессе публикации для запуска предложения.

![Этапы публикации предложения контейнера](./media/offer-publishing-steps.png)

## <a name="detailed-description-of-publishing-steps"></a>Подробное описание шагов публикации

В приведенной ниже таблице описан каждый этап публикации. Также здесь указано предполагаемое время завершения каждого этапа.


|  **Шаг публикации**           | **Время**    | **Описание**                                                            |
|  -------------------           | --------    | ---------------                                                            |
| Проверка необходимых компонентов         | 15 мин.   | Информация и параметры предложения — проверенные.                        |
| Сертификация                  | 1 неделя | Команда сертификации Azure анализирует предложение. Предложение проверяется на предмет вирусов, вредоносных программ, соответствия стандартам безопасности и проблем безопасности. Это позволяет определить, соответствует ли предложение всем критериям применимости. Дополнительные сведения см. в статье с [обязательными требованиями](./cpp-prerequisites.md) и [инструкциях по подготовке технических ресурсов](./cpp-create-technical-assets.md). При обнаружении проблем отправляется отзыв. |
| Упаковка | 1 час  | Технические активы предложения упакованы для использования клиентами, а свинцовые системы настроены и настроены. |
|  Утверждение издателем             |  -        | Просмотрите отзыв издателя и подтверждение активации предложения. Убедитесь, что предложение соответствует всем требованиям. Для этого разверните его в выбранной подписке (на шагах сведений о предложении).  Выберите **Go Live** (Запустить), чтобы сделать следующий шаг. |
| Упаковка                 | 1 час | Готовое предложение реплицируется в производственные системы и регионы Marketplace. | 
| Реальные данные                           | 4 дня |Предложение выпущено, реплицировано в необходимые регионы и доступно для просмотра. |

Предоставляется до 10 рабочих дней для завершения процесса публикации и выпуска предложения. После публикации предложение контейнера будет добавлено в список в [Microsoft Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/category/internet-of-things?page=1&subcategories=iot-edge-modules).
