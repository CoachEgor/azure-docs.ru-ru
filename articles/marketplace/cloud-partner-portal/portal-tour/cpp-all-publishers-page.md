# <a name="all-publishers-page"></a>Страница "Все издатели"

На странице [**Все издатели**](https://cloudpartner.azure.com/#publishers) отображается список всех зарегистрированных издателей в облаке Microsoft с уникальными и отображаемыми именами издателей. Вы можете упорядочить результаты по столбцу или указать строку для фильтрации результатов.  На этой странице также отображается кнопка **Добавить издателя**, но она доступна только для администраторов портала.  При выборе издателя в этом списке отображается соответствующий профиль.  См. дополнительные сведения об [управлении профилем издателя](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-publisher-profile.md).

На следующем рисунке показан список издателей, отфильтрованных по строке contoso.  

![Страница "Все издатели"](./media/all-publishers-page1.png)

Щелкнув первую запись, вы увидите страницу профиля для Contoso.  Вы можете изменить только собственный профиль издателя.
