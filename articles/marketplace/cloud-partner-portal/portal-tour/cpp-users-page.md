# <a name="users-page"></a>Страница "Пользователи"

На странице [**Пользователи**](https://cloudpartner.azure.com/#users) отображается список всех зарегистрированных пользователей Портала Cloud Partner.  Здесь есть столбцы для таких данных, как адрес электронной почты каждого из пользователей, роль на портале, тип предложения и организация-издатель. Результаты можно упорядочить по столбцу или отфильтровать по определенной строке.  

На этой странице также есть кнопка **Добавить пользователя** для добавления пользователей в организацию-издатель.  Дополнительные сведения см. в статье [Управление пользователями на облачном портале для партнеров](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-users.md).
