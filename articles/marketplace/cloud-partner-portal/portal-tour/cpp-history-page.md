# <a name="audit-history-page"></a>Страница "Журнал аудита"

На странице [**Журнал аудита**](https://cloudpartner.azure.com/#history) отображается список событий публикации для всех предложений Marketplace.  Здесь есть столбцы для таких данных, как тип события, дата и время события, имя предложения, издатель и пользователь, который инициировал действие.  Результаты можно упорядочить по столбцу или отфильтровать по определенной строке.  На этой странице пользователь также может скачать файл текущего списка аудита с разделителями-запятыми (CSV).
