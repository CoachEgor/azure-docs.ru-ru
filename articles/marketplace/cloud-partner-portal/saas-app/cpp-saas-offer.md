# <a name="azure-saas-application-offer"></a>Предложение приложения Azure SaaS

[Облачный портал для партнеров](https://cloudpartner.azure.com/) больше не поддерживает создание и управление предложениями приложения SaaS.  Вместо этого, эта функция была перенесена в Microsoft [портал центра партнеров](https://partner.microsoft.com/pcv/).  Дополнительные сведения см. в разделе [создайте новое предложение SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Дальнейшие действия

Перед созданием нового предложения приложения SaaS, которое необходимо прочитать [приложения SaaS предоставляют руководство по публикации](../../marketplace-saas-applications-technical-publishing-guide.md) и просмотрите шаги портала центра партнеров, перечисленные в [контрольный список для создания предложения](../../partner-center-portal/offer-creation-checklist.md).
