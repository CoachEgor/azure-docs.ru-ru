# <a name="azure-saas-application-offer"></a>Предложение приложения Azure SaaS

[Портал Cloud Partner](https://cloudpartner.azure.com/) больше не поддерживает создание и Управление предложениями приложений SaaS.  Вместо этого эта функция была перемещена на [портал центра партнеров](https://partner.microsoft.com/pcv/)Майкрософт.  Дополнительные сведения см. [в разделе Создание нового предложения SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Следующие шаги

Перед созданием нового предложения приложения SaaS следует ознакомиться с [руководством по публикации приложений SaaS](../../marketplace-saas-applications-technical-publishing-guide.md) и просмотреть шаги на портале центра партнеров, перечисленные в контрольном [списке создания предложения](../../partner-center-portal/offer-creation-checklist.md).
