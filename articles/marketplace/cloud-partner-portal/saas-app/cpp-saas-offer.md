# <a name="azure-saas-application-offer"></a>Предложение приложения Azure SaaS

[Облачный партнерский портал](https://cloudpartner.azure.com/) больше не поддерживает создание и управление предложениями приложений SaaS.  Вместо этого эта функция была перенесена на [портал Microsoft Partner Center.](https://partner.microsoft.com/pcv/)  Для получения дополнительной информации [см.](../../partner-center-portal/create-new-saas-offer.md) 


## <a name="next-steps"></a>Дальнейшие действия

Прежде чем создать новое предложение приложения SaaS, вы должны прочитать [приложения SaaS Предложение Издательский путеводитель](../../marketplace-saas-applications-technical-publishing-guide.md) и просмотреть шаги портала Partner Center, перечисленные в [контрольном списке создания предложения.](../../partner-center-portal/offer-creation-checklist.md)
