# <a name="saas-application-contacts-tab"></a>Вкладка "Контакты" приложения SaaS

В этой статье описывается, как применять вкладку "Контакты" для определения контактов для клиентов, использующих ваше предложение.

## <a name="identify-contacts"></a>Определение контактов

При предоставлении необходимых сведений используйте в качестве руководства следующий снимок экрана. Все поля, отмеченные звездочкой (*), должны быть заполнены.

![Форма контактных данных технических специалистов и службы поддержки](./media/saas-offer-contacts.png)

1. В разделе "Контактные данные технического специалиста" укажите сведения для следующих полей:

   - Имя — введите имя технического специалиста, ответственного за ваше приложение. Этому человеку будет передаваться техническая информация от корпорации Майкрософт.
   - Адрес электронной почты — введите адрес электронной почты технического специалиста, ответственного за ваше приложение.
   - Телефон — введите телефон технического специалиста, ответственного за ваше приложение. Поддерживаются обозначения номеров телефона в формате ISO. Дополнительные сведения можно найти в разделе [E.123](https://en.wikipedia.org/wiki/E.123)
  
2. В разделе "Контактные данные службы поддержки" укажите сведения для следующих полей.

   - Имя — введите имя сотрудника службы поддержки, ответственного за ваше приложение. Этому человеку корпорация Майкрософт будет передавать сведения, касающиеся поддержки.
   - Адрес электронной почты — введите адрес электронной почты сотрудника службы поддержки, ответственного за ваше приложение.
   - Телефон — введите телефон сотрудника службы поддержки, ответственного за ваше приложение. Поддерживаются обозначения номеров телефона в формате ISO. Дополнительные сведения можно найти в разделе [E.123](https://en.wikipedia.org/wiki/E.123)
   - URL-адрес для связи со службой поддержки — введите URL-адрес страницы для обращения в службу поддержки.

3. Выберите **Сохранить** после заполнения формы.
