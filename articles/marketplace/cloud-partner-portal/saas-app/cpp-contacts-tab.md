# <a name="saas-application-contacts-tab"></a>Вкладка "Контакты" приложения SaaS

В этой статье описывается, как применять вкладку "Контакты" для определения контактов для клиентов, использующих ваше предложение.


## <a name="identify-contacts"></a>Определение контактов

При предоставлении необходимых сведений используйте в качестве руководства следующий снимок экрана. Все поля, отмеченные звездочкой (*), должны быть заполнены.

![Вкладка "Контакты"](./media/saas-offer-contacts.png)

1. В разделе "Контактные данные технического специалиста" укажите сведения для следующих полей:

   - **Имя\***  -имя Инженерный для вашего приложения. Этому человеку будет передаваться техническая информация от корпорации Майкрософт.
   - **Адрес электронной почты\***  -Инженерный адрес электронной почты.
   - **Телефон\***  -Инженерный номер телефона. Поддерживаются обозначения номеров телефона в формате ISO. Дополнительные сведения можно найти в разделе [E.123](https://en.wikipedia.org/wiki/E.123)
  
2. В разделе "Контактные данные службы поддержки" укажите сведения для следующих полей.

   - **Имя\***  -имя контактного лица по поддержке для приложения. Этому человеку корпорация Майкрософт будет передавать сведения, касающиеся поддержки.
   - **Адрес электронной почты\***  -электронный адрес контактного лица поддержки для вашего приложения.
   - **Телефон\***  -телефонный номер контактного лица по поддержке. Поддерживаются обозначения номеров телефона в формате ISO. Дополнительные сведения можно найти в разделе [E.123](https://en.wikipedia.org/wiki/E.123)
   - **URL-адрес поддержки\***  -URL-адрес на страницу поддержки.

3. Выберите **Сохранить** после заполнения формы.
