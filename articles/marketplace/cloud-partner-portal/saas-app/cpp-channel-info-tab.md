# <a name="saas-application-channel-info-tab"></a>Вкладка "Информация о канале" приложения SaaS

Использование вкладки "Информация о канале" для предоставления информации о канале для приложения, в том числе материалов по выходу на рынок и контактных данных.

![Форма информации о канале](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>Настройка сведений о канале

1. В разделе **Channel Info > GTM materials** (Информация о канале > Материалы GTM) введите URL-адрес для веб-сайта, на котором размещены материалы по выходу на рынок.
2. В разделе **Channel Manager Contact** (Контактные данные менеджера канала) содержатся контактные данные менеджера, с которым корпорация Майкрософт сможет связаться по вопросам техподдержки и коммерческим вопросам. Введите сведения для этих полей: "Имя", "Адрес электронной почты" и "Номер телефона".
3. Выберите **Сохранить**, чтобы завершить настройку сведений о канале.
