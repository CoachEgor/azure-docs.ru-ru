# <a name="create-a-new-saas-application-offer"></a>Создание предложения приложения SaaS

В этой статье описывается, как создать и опубликовать запись предложения приложения SaaS для Azure Marketplace.

## <a name="offer-process"></a>Процесс предложения

Ниже показан процесс создания предложения приложения SaaS.

![Процесс создания предложения SaaS](./media/saas-offer-process-overview.png)

## <a name="offer-components"></a>Компоненты предложения

Предложение приложения SaaS состоит из пяти разделов, описанных в следующей таблице:

|  **Группа ресурсов**   |  **Описание**  |
|  ---------------   |  ---------------  |
|    Параметры предложения  |  Используется для настройки уникального идентификатора для приложения SaaS.                 |
|  Технические сведения    |  Используется для настройки типа решения SaaS, а также указания сведений о подключении для вашего приложения.                |
|  Информация о канале      |   Предоставляет сведения о канале, например материалы по выходу на рынок и контактные данные.                |
|  Тестовый выпуск        |   Дополнительный раздел для определения службы, которая позволит клиентам протестировать вашего предложение перед его приобретением.                |
|  Подробные сведения об онлайн-магазинах       | Содержит маркетинговые, юридические и ведущие ресурсы и спецификации управления.  <ul><li> Маркетинговые ресурсы включают имя предложения, описание и логотипы.</li> <li> Юридические ресурсы включают политику конфиденциальности, условия использования и другие юридические документы.</li>  <li> Политика управления потенциальными клиентами позволяет указать способ обработки потенциальных клиентов пользовательского портала Azure Marketplace.</li> </ul> |
| Контакты            | Содержит сведения о поддержке контактных данных и политики. |

## <a name="new-offer-form"></a>Форма нового предложения

Войдите на [Портал Cloud Partner](https://cloudpartner.azure.com/), а затем в строке меню слева выберите **+Новое предложение**. Чтобы отобразить форму"Новое предложение" и начать процесс определения ресурсов для нового приложения SaaS, в меню "Новое предложение" выберите **Приложения SaaS**.

![Меню "Новое предложение" для приложений SaaS](./media/azure-new-saas-offer.png)
