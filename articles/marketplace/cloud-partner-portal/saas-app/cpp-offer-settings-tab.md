# <a name="saas-application-offer-settings-tab"></a>Вкладка параметров предложения приложения SaaS

В этой статье описано, как настроить параметры предложения.

Откроется страница **Приложение SaaS > Новое предложение** с фокусом на вкладке **Параметры предложения**. 

На вкладке "Параметры предложения" настройте **идентификатор предложения**, как показано на следующем снимке экрана. Символ звездочки (*) после имени поля указывает, что оно является обязательным.

![Вкладка "Параметры предложения"](./media/saas-new-offer.png)

## <a name="offer-identity-settings"></a>Параметры удостоверений предложения

В разделе "Идентификатор предложения" необходимо предоставить сведения для полей, описанных в следующей таблице. 


|  **Имя поля**   |  **Описание**  |
|  ---------------   |  ---------------  |
|    ИД предложения  |  Уникальный идентификатор предложения в профиле издателя. Он будет отображаться в URL-адресах продукта и отчетах о выставлении счетов. Идентификатор может содержать только строчные буквы, цифры или дефисы (-). Он не может заканчиваться дефисом, а его длина не должна превышать 50 знаков. Обратите внимание, что после активации предложения это поле блокируется. Например, если издатель Contoso опубликует предложение с идентификатором sample-vm, оно отобразится в Azure Marketplace как https://azuremarketplace.microsoft.com/marketplace/apps/contoso.sample-vm?tab=Overview.                 |
|  Publisher ID (Идентификатор издателя)    |  Идентификатор издателя — это ваш уникальный идентификатор в Marketplace. Для всех предложений необходимо указывать свой идентификатор издателя. Его нельзя изменить после того, как предложение будет сохранено.                |
|  ИМЯ      |   Это отображаемое имя для предложения. Это имя будет отображаться в Azure Marketplace и на портале Azure. Его длина не должна превышать 50 символов. Укажите для своего продукта узнаваемое название торговой марки. Не включайте в него название своей организации, если только это не требуется для продвижения продукта. Если вы продвигаете продукт через свой веб-сайт, убедитесь, что это имя полностью совпадает с именем, используемым на сайте.               |

Щелкните **Сохранить**, чтобы сохранить настройки.
