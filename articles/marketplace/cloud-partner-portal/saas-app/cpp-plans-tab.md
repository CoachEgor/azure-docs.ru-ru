# <a name="saas-application-plans-tab"></a>Вкладка SaaS application Plans (Планы приложения SaaS)

Для создания нового плана используйте вкладку "Планы". При использовании опции Sell through Microsoft (Продажа через Майкрософт) для приложения SaaS, необходимо добавить как минимум 1 план.

![Создание нового плана](./media/saas-plans-new-plan.png)

## <a name="create-a-new-plan"></a>Создание нового плана

Чтобы создать новый план, выполните следующие действия.

1. В разделе **Планы** выберите **+ Создать план**.
2. Во всплывающем окне **Создать план** введите **идентификатор плана**. Максимальная длина составляет 50 символов. Этот идентификатор должен содержать только строчные буквы, цифры, дефисы и символы подчеркивания. Вы не сможете изменить этот идентификатор после публикации предложения.
3. Нажмите кнопку **ОК**, чтобы сохранить этот идентификатор плана.

   ![Идентификатор плана для нового плана](./media/saas-plans-plan-ID.png)

### <a name="to-configure-the-plan"></a>Настройка плана

1. В разделе **Сведения о плане** заполните следующие поля.

   - Название. Укажите название плана. Название может содержать до 50 символов.
   - Описание. Добавьте описание. Описание может содержать до 500 символов.
   - Это частный план? — Если план доступен только выбранной группе клиентов, выберите **Да**.
   - Доступность по странам и (или) регионам. План должен быть доступен по крайней мере для 1 страны или региона. Щелкните **Select regions** (Выбор регионов). Выберите страну (регион) из списка **Select Country/Region availability** (Выбор доступности по странам и регионам), а затем щелкните **ОК**. 
   - Устаревшие цены. Укажите стоимость в долларах США в месяц.

2. В разделе **Simplified Currency Pricing** (Упрощенные цены в разных валютах) укажите следующую информацию.

   - Срок выставления счетов. Ежемесячная цена выбрана по умолчанию. Можно также предоставить ежегодную цену.
   - Ежемесячная цена. Укажите ежемесячную цену, которая должна соответствовать ценам прежних версий.

   >[!NOTE] 
   >Если вы добавите ежегодную цену к сроку выставления счета, вам будет предложено указать **Ежегодную цену** в долларах США в год.

3. Выберите **Сохранить**, чтобы завершить настройку плана.

   >[!NOTE] 
   >После сохранения изменений в ценах вы можете экспортировать или импортировать данные о цене.
