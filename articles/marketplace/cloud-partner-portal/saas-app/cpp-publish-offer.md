# <a name="publish-a-saas-application-offer"></a>Публикация предложения приложения SaaS

Опубликовать предложение можно после создания нового предложения с информацией на странице **Новое предложение**. Выберите **Публиковать**, чтобы начать процесс публикации.

В примере ниже показаны общие действия для публикации нового предложения приложения SaaS.

![Шаги публикации предложения](./media/offer-publishing-steps.png)

## <a name="detailed-description-of-publishing-steps"></a>Подробное описание шагов публикации

В следующей таблице описывается каждый шаг публикации с оценкой времени (максимум) для завершения каждого шага.

|     **Step**       |     **Time**      |  **Описание**  |
|  ---------------   |  ---------------  |  ---------------  |
|         Сертификация           |       2 нед.            |          Команда сертификации Azure анализирует предложение. На этом шаге выполняется проверка на вирусы, вредоносные программы, соответствие стандартам безопасности и проблемы безопасности. Он также выполнит проверку того, что это предложение отвечает всем критериям (см. [предварительные требования](./cpp-prerequisites.md)). Отзывы предоставляются в случае обнаружения проблемы.         |
|           Упаковка         |       1 час            |       Технические ресурсы предложения упакованы для использования клиентами, а ведущие системы настроены и установлены.            |
|        Выход издателя            |         -          |        Просмотрите отзыв издателя и подтверждение активации предложения. Убедитесь, что предложение соответствует всем требованиям. Для этого разверните его в выбранной подписке (на шагах сведений о предложении). Выберите **Go Live** (Запустить), чтобы сделать следующий шаг.           |
|        Упаковка            |        1 час           |        Окончательное предложение реплицируется в производственные системы и регионы Marketplace.           |
|        В реальном времени            |       4 дня            |         Предложение выпущено, реплицировано в необходимые регионы и доступно для просмотра.          |

Предоставляется до 10 рабочих дней для завершения процесса публикации и выпуска предложения. После публикации предложение SaaS будет добавлено в список в [Microsoft Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/category/internet-of-things?page=1&subcategories=iot-edge-modules).
