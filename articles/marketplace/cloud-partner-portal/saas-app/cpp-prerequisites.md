# <a name="saas-application-publishing-prerequisites"></a>Предварительные условия публикации приложения SaaS

В этой статье описаны предварительные требования для публикации предложения приложения SaaS в Azure Marketplace.  Если вы еще не сделано, просмотрите [приложения SaaS предоставляют руководство по публикации](../../marketplace-saas-applications-technical-publishing-guide.md). 


## <a name="publishing-prerequisites"></a>Обязательные требования для публикации

Чтобы опубликовать новое предложение приложения SaaS, необходимо выполнить следующие предварительные условия:
- Получите доступ к [порталу Cloud Partner](https://cloudpartner.azure.com/).  На портале Cloud Partner предоставляет доступ на основе ролей, несколько отдельных сотрудников для совместной работы для публикации предложения. 
- Прежде чем опубликовать предложение от имени учетной записи издателя, одному из пользователей с ролью владельца необходимо согласиться с [условиями использования](https://azure.microsoft.com/support/legal/website-terms-of-use/), [заявлением о конфиденциальности Майкрософт](https://www.microsoft.com/privacystatement/default.aspx) и [соглашением по программе сертификации Microsoft Azure](https://azure.microsoft.com/support/legal/marketplace/certified-program-agreement/).
- Все содержимое неанглийские должен прилагаться с английской версией. Это содержимое включает в себя текст, документы, снимки экрана, условия использования и политику конфиденциальности для интернет-магазина. Можно также указать URL-адрес ссылки на локализованное приложение. 
- Ваши метаданные должны быть готовы к использованию. Следующий список (не исчерпывающий) содержит примеры этих метаданных:
  - название;
  - описание (в формате HTML);
  - Изображение логотипа (в формате PNG) таких фиксированных размеров: 40 x 40 пикселей, 90 x 90 пикселей, 115 x 115 пикселей и 255 x 115 пикселей.
  - Объект *условия использования* и *политику конфиденциальности* инструкций
  - Документация по Application
  - контактные данные службы поддержки.


## <a name="next-steps"></a>Дальнейшие действия

После выполнения этих предварительных требованиях, вы можете [создайте предложение SaaS](./cpp-create-offer.md).
