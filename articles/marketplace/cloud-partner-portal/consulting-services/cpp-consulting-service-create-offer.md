# <a name="create-a-consulting-offer"></a>Создание консалтингового предложения

В этой статье описывается, как создать и опубликовать предложение консультационных услуг.

## <a name="define-your-offer"></a>Описание своего предложения

Определите свое пакетное предложение консультационных услуг. Сосредоточьтесь на фиксированном объеме, фиксированной продолжительности, оценке, цене (или бесплатном предложении) и в первую очередь на предложениях, ориентированных на предварительные продажи, для отдельного клиента. Выберите повторяемые пакетные вовлеченности пользователей, которые оказались наиболее популярными и эффективными для введения новых дел для вас.

## <a name="create-a-new-offer"></a>Создание нового предложения

Чтобы создать новое предложение, выполните следующие действия.
1. В главном меню на портале Cloud Partner выберите **New offer** (Создать предложение).
1. В меню **New Offer** (Новое предложение) выберите **Consulting Service** (Консультационная услуга).

    ![Создание нового предложения консультационных служб](media/cppselectnewconsultingoffer.png)
