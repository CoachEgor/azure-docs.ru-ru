# <a name="publish-a-consulting-service-offer"></a>Публикация предложения консультационных услуг

После заполнения строк **Параметры предложения**, **Сведения об онлайн-магазине** и **Контакты**, выберите **Опубликовать** и укажите адрес электронной почты. Когда корпорация Майкрософт будет готова опубликовать ваше предложение, вы получите по электронной почте сообщение для его предварительного просмотра перед запуском. В любой момент можно вернуться на портал, чтобы проверить состояние вашего предложения.

Предложения в процессе публикации могут отображаться в состоянии Publish canceled (Публикация отменена) или Publish failed (Опубликовать не удалось). Эти обозначения являются нормальной частью процесса, который позволяет корпорации Майкрософт внести изменения в ваше предложение. Если ваше предложение отображается в состоянии "Публикация отменена", оставьте его в этом состоянии.

## <a name="lead-destination"></a>Назначение потенциального клиента

Выберите систему CRM, в которой будут храниться сведения о потенциальных клиентах. В выбранной вами системе CRM мы будем записывать сведения о пользователях, которые тестируют ваше приложение в AppSource.

В зависимости от выбранной системы CRM выберите соответствующий URL-адрес из следующего списка, чтобы получить сведения о том, как заполнять следующий набор полей.

Выберите [таблицу Azure](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-lead-management-instructions-azure-table), если у вас одна из следующих систем CRM:  
* [Marketo](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-lead-management-instructions-marketo)
* [Microsoft Dynamics CRM](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-lead-management-instructions-dynamics)
* [Salesforce](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-lead-management-instructions-salesforce)
