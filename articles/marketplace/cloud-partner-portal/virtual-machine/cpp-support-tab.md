# <a name="virtual-machine-support-tab"></a>Вкладка "Поддержка" для виртуальной машины

> [!IMPORTANT]
> Начиная с 13 апреля 2020 г. Мы начинаем перемещать Управление предложениями виртуальных машин Azure в центр партнеров. После миграции вы создадите предложения в центре партнеров и будете управлять ими. Следуйте инструкциям в разделе [Создание предложения виртуальной машины Azure](https://docs.microsoft.com/azure/marketplace/partner-center-portal/azure-vm-create-offer) для управления перенесенными предложениями.

Вкладка **Поддержка** на странице **Новое предложение** позволяет предоставить технические ресурсы и пользовательскую поддержку для предложения.  Она разделена на три области: **Engineering Contact** (Контакт по техническим вопросам), **Customer Support** (Поддержка клиентов) и **Support URLs** (URL-адреса для получения поддержки).

![Форма для виртуальных машин: окно "Новое предложение" и вкладка "Поддержка"](./media/publishvm_012.png)

## <a name="field-values"></a>Значения полей

В следующей таблице описаны назначение и содержимое этих полей. Обязательные поля помечены звездочкой (*).

|         Поле                 |       Описание                                                        |
|        -------                |       ------------                                                       |
| *Контактные данные технической службы*     | Контактное лицо для связи по техническим вопросам между Майкрософт и вашей организацией. | 
| **Безымян\***                | Имя лица или группы, предоставляющих техническую поддержку.     |
| **Электронная почта\***               | Адрес электронной почты для связи по техническим вопросам.                                      |
| **Факс\***               | Номер телефона для получения технической поддержки.                                           |
| *Поддержка клиентов*        | Лицо, принимающее запросы на поддержку, открытые клиентами в Azure. |
| **Безымян\***                | Имя лица или группы, отвечающих за поддержку клиентов.                  |
| **Электронная почта\***               | Адрес электронной почты службы поддержки клиентов                                            |
| **Факс\***               | Номер телефона службы поддержки клиентов.                                            |
| *URL-адреса для получения поддержки*            | Сайты поддержки, которые будет использовать Майкрософт, когда ваши клиенты будут отправлять запросы в службу поддержки. |
| **Public Azure (Общедоступные Azure)**          | URL-адрес общедоступного веб-сайта поддержки.                                         |
| **Azure Government Cloud (Облако Azure для государственных организаций)**| URL-адрес сайта поддержки облака для государственных организаций.                                        |
|  |  |
