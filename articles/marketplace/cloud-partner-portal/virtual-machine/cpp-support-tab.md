# <a name="virtual-machine-support-tab"></a>Вкладка "Поддержка" для виртуальной машины

Вкладка **Поддержка** на странице **Новое предложение** позволяет предоставить технические ресурсы и пользовательскую поддержку для предложения.  Он состоит из трех областей: **Проектирование**, **поддержки**, и **поддерживает URL-адреса**.

![Форма для виртуальных машин: окно "Новое предложение" и вкладка "Поддержка"](./media/publishvm_012.png)

## <a name="field-values"></a>Значения полей

В следующей таблице описаны назначение и содержимое этих полей. Обязательные поля помечены звездочкой (*).

|         Поле                 |       ОПИСАНИЕ                                                        |
|        -------                |       ------------                                                       |
| *Контакт по техническим вопросам*     | Контактное лицо для связи по техническим вопросам между Майкрософт и вашей организацией. | 
| **Имя\***                | Имя лица или группы, предоставляющих техническую поддержку.     |
| **Электронная почта\***               | Адрес электронной почты для связи по техническим вопросам.                                      |
| **Телефон\***               | Номер телефона для получения технической поддержки.                                           |
| *Поддержка клиентов*        | Лицо, принимающее запросы на поддержку, открытые клиентами в Azure. |
| **Имя\***                | Имя лица или группы, отвечающих за поддержку клиентов.                  |
| **Электронная почта\***               | Адрес электронной почты службы поддержки клиентов                                            |
| **Телефон\***               | Номер телефона службы поддержки клиентов.                                            |
| *URL-адреса для получения поддержки*            | Сайты поддержки, которые будет использовать Майкрософт, когда ваши клиенты будут отправлять запросы в службу поддержки. |
| **Общедоступное облако Azure**          | URL-адрес общедоступного веб-сайта поддержки.                                         |
| **Облако Azure для государственных организаций**.| URL-адрес сайта поддержки облака для государственных организаций.                                        |
|  |  |
