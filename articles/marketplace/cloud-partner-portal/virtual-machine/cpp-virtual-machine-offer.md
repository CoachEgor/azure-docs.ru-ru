# <a name="azure-virtual-machine-offer"></a>Предложение виртуальной машины Azure

[Портал Cloud Partner](https://cloudpartner.azure.com/) больше не поддерживает создание предложений виртуальных машин Azure и управление ими. Эта функция перемещена в [Центр партнеров](https://partner.microsoft.com/pcv/) Майкрософт. Дополнительные сведения см. в статье [Создание предложения виртуальной машины Azure в Azure Marketplace](../../partner-center-portal/azure-vm-create-offer.md).
