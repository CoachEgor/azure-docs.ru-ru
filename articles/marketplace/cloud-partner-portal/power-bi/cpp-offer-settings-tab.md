# <a name="power-bi-apps-offer-settings-tab"></a>Вкладка параметров предложения для приложения Power BI

При открытии новой страницы **предложения** для приложений службы сначала откроется вкладка **параметры предложения** . На этой вкладке вы предоставляете основные идентификаторы и имя для предложения. Звездочка (*) обозначает обязательное поле.

![Вкладка "Параметры предложения"](./media/offer-settings-tab.png)


## <a name="offer-settings-fields"></a>Поля параметров предложения 

На вкладке **параметры предложения** необходимо ввести данные в следующие обязательные поля. Обязательные поля помечены звездочкой (*).

|  Поле        |  Description (Описание)                                                               |
|---------------|----------------------------------------------------------------------------|
| **Идентификатор предложения\***  | Уникальный идентификатор предложения (в профиле издателя). Он отображается в URL-адресах продукта, шаблонах Azure Resource Manager и отчетах о выставлении счетов. Максимальная длина составляет 50 символов. Он может содержать только буквы в нижнем регистре, цифры и дефисы (-). Оно не может заканчиваться тире. Этот идентификатор нельзя изменить после того, как предложение пройдет в реальном времени. Если Contoso публикует предложение с ИДЕНТИФИКАТОРом предложения `sample-SvcApp`, для предложения назначается URL-адрес AppSource `https://appsource.microsoft.com/marketplace/apps/contoso.sample-SvcApp`.      |
| **\* издателя** | Уникальный идентификатор организации в [AppSource](https://appsource.microsoft.com). Для всех предложений необходимо указывать свой идентификатор издателя. Это значение нельзя изменить после сохранения предложения.                         |
| **Имя\***      | Отображаемое имя для предложения. Это имя будет отображаться в AppSource и на Портал Cloud Partner. Максимальная длина составляет 50 символов. Используйте фирменное название, распознаваемое для вашего продукта. Не включайте здесь имя вашей организации, если оно не отображается на рынке с этим именем. Если вы предоставляете это предложение на других веб-сайтах и публикациях, используйте одно и то же имя во всех публикациях.    <br/>Если предложение выпускается в период действия предварительной версии для Power BI приложений, добавьте строку `(Preview)` в конце имени приложения, например: `Sample Scv App (Preview)`. |
|     |     |
