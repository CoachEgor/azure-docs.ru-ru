# <a name="power-bi-apps-offer-settings-tab"></a>Вкладка параметров предложения для приложения Power BI

При открытии **новое предложение** странице для службы приложений, вы увидите **параметров предложения** вкладки. На этой вкладке укажите идентификаторов первичных и имя для вашего предложения. Символ звездочки (*) обозначает обязательное поле.

![Вкладка "Параметры предложения"](./media/offer-settings-tab.png)


## <a name="offer-settings-fields"></a>Поля параметров предложения 

На **параметров предложения** вкладку, необходимо ввести сведения в следующих обязательных полей. Обязательные поля помечены звездочкой (*).

|  Поле        |  Описание                                                               |
|---------------|----------------------------------------------------------------------------|
| **Идентификатор предложения\***  | Уникальный идентификатор предложения (в профиле издателя). Он отображается в URL-адресах продукта, шаблонах Azure Resource Manager и отчетах о выставлении счетов. Максимальная длина составляет 50 символов. Он может содержать только строчные буквы, цифры и дефисы (-). Он не может заканчиваться дефисом. Этот идентификатор нельзя изменить после активации предложения. Если Contoso публикует предложение с идентификатор предложения `sample-SvcApp`, предложение назначается URL-адреса AppSource `https://appsource.microsoft.com/marketplace/apps/contoso.sample-SvcApp`.      |
| **Издатель\*** | Уникальный идентификатор организации в [AppSource](https://appsource.microsoft.com). Для всех предложений необходимо указывать свой идентификатор издателя. Это значение нельзя изменить после сохранения предложения.                         |
| **Имя\***      | Отображаемое имя для вашего предложения. Это имя будет отображаться на AppSource и на портале Cloud Partner. Максимальная длина составляет 50 символов. Используйте имя бренда, распознаваемого продукта. Не включайте здесь название вашей организации, если только приложение, продаваемые с таким именем. Если вы создаете в рамках данного предложения от других веб-сайтов и публикации, используйте имя, совпадающее во всех публикациях.    <br/>Если предложение во время действия предварительной версии выпуска для приложения Power BI, добавьте строку `(Preview)` в конце имени приложения, следующим образом: `Sample Scv App (Preview)`. |
|     |     |
