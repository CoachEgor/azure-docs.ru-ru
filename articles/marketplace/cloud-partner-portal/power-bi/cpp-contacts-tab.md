# <a name="power-bi-app-contacts-tab"></a>Вкладка контактов приложения Power BI

>[!Important]
>Начиная с 13 апреля 2020 г. Мы начинаем перемещать Управление предложениями Power BI приложений в центр партнеров. После миграции вы создадите предложения в центре партнеров и будете управлять ими. Следуйте инструкциям в разделе [Обзор создания приложений Power BI](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer) , чтобы управлять перенесенными предложениями.

Используйте вкладку **Контакты** на странице **Новое предложение**, чтобы предоставить технические ресурсы и пользовательскую поддержку для предложения. Он делится на два раздела: **проектирование** и **Поддержка клиентов**.

![Вкладка "Контакты"](media/contacts-tab.png)


## <a name="contacts-fields"></a>Поля контактов 

В следующей таблице описывается назначение каждого поля.  Обязательные поля помечены звездочкой (*). 

|   Поле               |   Описание                                                            |
|-----------------------|--------------------------------------------------------------------------|
| *Контактные данные технической службы* | Контактное лицо для связи по техническим вопросам между AppSource и вашей организацией.    |
| **Имя\***              | Имя лица или группы, предоставляющей техническую или инженерную поддержку. |
| **Электронная почта\***             | Адрес электронной почты для связи по техническим вопросам.                                  |
| **Факс\***             | Номер телефона технической поддержки.                                       |
| *Контактные данные службы поддержки*     | Лицо, принимающее запросы в службу поддержки от клиентов в AppSource.            |
| **Имя\***              | Имя лица или группы, отвечающих за поддержку клиентов.              |
| **Электронная почта\***             | Адрес электронной почты службы поддержки клиентов.                                        |
|   |   |


## <a name="next-steps"></a>Дальнейшие шаги

После создания предложения в приложении Power BI, необходимо подтвердить создание всех связанных [технических ресурсов](./cpp-create-technical-assets.md) для этого предложения.
