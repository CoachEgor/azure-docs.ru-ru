# <a name="power-bi-app-contacts-tab"></a>Вкладка контактов приложения Power BI

Используйте вкладку **Контакты** на странице **Новое предложение**, чтобы предоставить технические ресурсы и пользовательскую поддержку для предложения. Она разделена на две секции: **инженерное дело** и **поддержка клиентов.**

![Вкладка "Контакты"](media/contacts-tab.png)


## <a name="contacts-fields"></a>Поля контактов 

В следующей таблице описывается цель каждого поля.  Обязательные поля помечены звездочкой (*). 

|   Поле               |   Описание                                                            |
|-----------------------|--------------------------------------------------------------------------|
| *Контактные данные технической службы* | Контактное лицо для связи по техническим вопросам между AppSource и вашей организацией.    |
| **name\***              | Имя лица или группы, предоставляющей техническую или инженерную поддержку. |
| **Отправить по электронной почте\***             | Адрес электронной почты для связи по техническим вопросам.                                  |
| **Телефон\***             | Номер телефона технической поддержки.                                       |
| *Контактные данные службы поддержки*     | Лицо, принимающее запросы в службу поддержки от клиентов в AppSource.            |
| **name\***              | Имя лица или группы, отвечающих за поддержку клиентов.              |
| **Отправить по электронной почте\***             | Адрес электронной почты службы поддержки клиентов.                                        |
|   |   |


## <a name="next-steps"></a>Дальнейшие действия

После создания предложения в приложении Power BI, необходимо подтвердить создание всех связанных [технических ресурсов](./cpp-create-technical-assets.md) для этого предложения.
