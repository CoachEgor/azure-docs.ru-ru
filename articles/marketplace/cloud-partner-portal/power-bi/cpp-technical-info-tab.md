# <a name="power-bi-apps-technical-info-tab"></a>Вкладка технической информации о приложениях Power BI

На странице **«Новое предложение»** используйте вкладку **«Техническая информация»,** чтобы предоставить URL-адрес пакета установки Power BI и другую информацию, необходимую для проверки нового предложения.  Для первоначального релиза все приложения Power BI бесплатны и доступны для скачивания с AppSource. Из-за этого вы не можете определить единицы учета запасов (SKUs) для этого типа предложения.

![Вкладка «Техническая информация»](./media/technical-info-tab.png)


## <a name="technical-info-fields"></a>Поля на вкладке с техническими сведениями 

На вкладке **«Техническая информация»** заполните поля, описанные в следующей таблице. Звездочка (кв) в конце метки поля означает, что поле требуется.

|        Поле          |  Описание                                                                 |
|    ---------------    |  ----------------------------------------------------------------------------|
| **URL-адрес установки\***     | Power BI генерирует этот URL при публикации приложения и продвигаете его к производству.  Для получения дополнительной информации смотрите [приложения Publish с панелей мониторинга и отчетами в Power BI](https://docs.microsoft.com/power-bi/service-create-distribute-apps).  |
|  **Validation instructions** (Инструкции по проверке)  |  Если вы хотите, добавьте инструкции (до 3000 символов), чтобы помочь группе проверки Майкрософт настроить, подключить ся и протестировать приложение. Включите типичные настройки конфигурации, учетные записи, параметры или другую информацию, которая может быть использована для тестирования опции Connect Data. Эта информация видна только группе проверки и используется только для целей проверки.  |
| **Is this app created as a Power BI content pack?** (Создается ли это приложение в виде пакета содержимого Power BI?) | В настоящее время это поле используется только внутренне. Оставьте настройку по умолчанию **No**. Если вы измените настройку **на Yes,** вы можете остановить процесс публикации.  |  
|  |  |
