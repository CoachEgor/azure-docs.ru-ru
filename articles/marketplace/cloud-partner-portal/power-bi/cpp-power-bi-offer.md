# <a name="power-bi-app-offer"></a>Предложение приложения Power BI

|              |                                |
|--------------|--------------------------------|
| В этой статье объясняется, как опубликовать приложение Power BI на рынке Microsoft [AppSource](https://appsource.microsoft.com/). Затем приложение можно развернуть другим арендаторам Power BI через AppSource, выполнить корректировки и настройки, разрешенные разработчиком, и подключить его к собственным данным. | ![Значок Power BI](./media/powerbi-icon.png) |

## <a name="publishing-steps"></a>Этапы публикации

Вот шаги высокого уровня для публикации предложения Power BI App:

![Мощность BI App предлагают публикации шаги](media/publishing-steps.png)

Вот Power BI App предлагают процесс публикации:

1. Создайте шаблонное приложение в Power BI. Это действие генерирует URL-адрес установки пакета, который представляет собой основной технический актив для предложения. Также в это время продвигайте тестовый пакет до предварительного производства. Для получения дополнительной информации, смотрите [Что такое Power BI шаблон приложений?](https://docs.microsoft.com/power-bi/service-template-apps-overview). 
2. Собирайте или создавайте маркетинговые материалы предложения, в том числе: официальное название, описание, логотипы и т.д. 
3. Соберите или создайте юридические и вспомогательные документы предложения: *условия использования,* *политика конфиденциальности,* *политика поддержки,* помощь пользователя и т.д.
4. Создайте предложение: используйте облачный партнерский портал для настройки деталей предложения, включая описание предложения, маркетинговые материалы, правовую информацию, информацию о поддержке и спецификации активов.  После того, как предложение полностью определено, отправьте его для публикации.
5. Мониторинг процесса публикации на портале Cloud Partner.  На этом этапе команда AppSource, которая тестирует, проверяет и удостоверяет ваше приложение. 
6. После успешной сертификации приложения просмотрите его в тестовой среде и отпустите. 
7. Power BI App указан на AppSource (он "идет жить").
8. В Power BI продвигают предсерийный пакет к производству. Для получения дополнительной информации [см.](https://docs.microsoft.com/power-bi/service-template-apps-create#manage-the-template-app-release)
