# <a name="publish-a-power-bi-app-offer"></a>Публикация предложения Power BI приложения

>[!Important]
>Начиная с 13 апреля 2020 г. Мы начинаем перемещать Управление предложениями Power BI приложений в центр партнеров. После миграции вы создадите предложения в центре партнеров и будете управлять ими. Следуйте инструкциям в разделе [Обзор создания приложений Power BI](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer) , чтобы управлять перенесенными предложениями.

Последним шагом после определения предложения в Портал Cloud Partner и создания связанных технических ресурсов является отправка предложения для публикации. Чтобы запустить этот процесс, в левой области окна **новое предложение** выберите **опубликовать**. Дополнительные сведения см. в статье [Публикация предложений Azure Marketplace и AppSource](../manage-offers/cpp-publish-offer.md).

Для завершения этого процесса можно использовать до восьми дней. После выполнения этих шагов публикации предложение Power BI приложения будет указано в разделе [AppSource](https://appsource.microsoft.com/marketplace/apps?product=power-bi%20) Power BI Apps.


### <a name="app-certification-process"></a>Процесс сертификации приложения

Команда адаптации Майкрософт использует этот процесс для проверки отправки предложения приложения Power BI.

1. Ознакомьтесь с юридическими документами и ссылками на справочные материалы.
2. Проверка контактных данных службы поддержки.
3. Используйте URL-адрес установщика для проверки правильности установки.
4. Проверка приложения на наличие вредоносных программ и другого вредоносного содержимого.
5. Убедитесь, что отображаемое содержимое соответствует описанию приложения.
6. Убедитесь, что операции, связанные с приложением, работают ожидаемым образом в Power BI. Команда открывает отчеты и панели мониторинга с примерами данных, подключается к пользовательским источникам данных, обновляет данные и т. д.

Команда сертификации предоставляет обратную связь, если обнаруживает какие-либо проблемы.  Дополнительные сведения о требованиях к приложениям Power BI см. в [документации по приложению Power BI](https://go.microsoft.com/fwlink/?linkid=2028636).
