# <a name="power-bi-app-offer-prerequisites"></a>Приложение Power BI предоставляют необходимые компоненты

Эта статья содержит технические и бизнес-требований, которые должны быть выполнены перед публикацией приложение Power BI предлагать магазина AppSource.  Если вы еще не сделано, просмотрите [Office 365, Dynamics 365, PowerApps и Power BI предоставляют руководство по публикации](../../appsource-offer-publishing-guide.md).


## <a name="technical-requirements"></a>Технические требования

Основные технические активов, вам потребуется для этого типа предложения, [приложение Power BI](https://go.microsoft.com/fwlink/?linkid=2028636). Дополнительные сведения см. в статье [Создание технических ресурсов приложения Power BI](./cpp-create-technical-assets.md).

После создания приложения-службы и протестированы в Power BI, следует сохранить URL-адрес установки приложения, создающий Power BI. Вам потребуется его [Создание предложения приложение Power BI](./cpp-create-offer.md).

## <a name="business-requirements"></a>Бизнес-требования

Бизнес-требования включают процедурного контрактные и юридические обязательства.  Необходимо следующее:

- Быть издателя marketplace зарегистрированное облако. Если вы не зарегистрирован, выполните действия, описанные в [стать издателем Cloud Marketplace](https://docs.microsoft.com/azure/marketplace/become-publisher).

    > [!NOTE] 
    > Использовать одну и ту же учетную запись регистрации центра разработчиков Майкрософт для входа [облачный портал для партнеров](https://cloudpartner.azure.com). У вас должна быть только одна учетная запись Майкрософт для публикации предложений в AppSource. Учетные записи не должно быть только к отдельным службам или предложениям.
 
- Предоставьте клиентам экономически обоснованную техническую поддержку.
- Предоставляют материалы, отвечающие критериям для вашего предложения в AppSource. Дополнительные сведения см. в записи блога [Have an app to list on AppSource? Here’s how](https://appsource.microsoft.com/blogs/have-an-app-to-list-on-appsource-here-s-how) (Есть приложение для публикации в AppSource? Вот как это можно сделать).
- Соответствовать [заявление о конфиденциальности Майкрософт](https://privacy.microsoft.com/privacystatement).
