# <a name="manage-azure-and-appsource-marketplace-offers"></a>Управление предложениями Azure и AppSource Marketplace

В этом разделе представлены общие операции, используемые для управления предложениями для Azure Marketplace и AppSource Marketplace.  Например, здесь объясняется, как использовать [Портал Cloud Partner](https://cloudpartner.azure.com/), чтобы создать и опубликовать предложение в Marketplace.  Этот раздел содержит объяснение технических проблем, которые являются общими для большинства типов предложений, например текстовые файлы и файлы изображений.


## <a name="standard-offer-operations"></a>Стандартные операции предложения

На Портале Cloud Partner издатель может выполнять следующие основные операции над предложением.

|     Операция      |  Описание                                           |
|     ---------      |  -----------                                           |
| [Создание предложения](./cpp-create-offer.md)   | Создает запись предложения ожидания в репозитории подключения, но автоматически не публикует его в Microsoft Marketplace. | 
| [Просмотр сведений о состоянии предложения](./cpp-view-status-offer.md)   | Предоставляет состояние публикации предложения.  На портале существует несколько мест, где отображаются сведения о состоянии. |
| [Публикация предложения](./cpp-publish-offer.md) | Проверяет, а затем публикует запись завершенного предложения на соответствующий Microsoft Marketplace.  Затем предложение считается *активным*. |
| [Обновление предложения](./cpp-update-offer.md)   | Позволяет изменить определенные характеристики существующего предложения.  Применяется к ожидающим и опубликованным предложениям. |
| [Удаление предложения](./cpp-delete-offer.md)   | Удаляет существующее ожидающее или динамическое предложение.  | 
|  |  |
