# <a name="iot-edge-module-support-tab"></a>Модуль IoT Edge: вкладка "Поддержка"

Используйте вкладку **Поддержка** на странице **Новое предложение**, чтобы предоставить технические ресурсы и пользовательскую поддержку для предложения.  В форме поддержки представлены три области для ввода данных: **Engineering Contact** (Контакт по техническим вопросам), **Customer Support** (Поддержка клиентов) и **Support URLs** (URL-адреса для получения поддержки). Символ звездочки (*) после имени поля указывает, что оно является обязательным.

![Вкладка "Поддержка" в форме "Новое предложение" для модулей IoT Edge](./media/iot-edge-module-support-tab.png)

## <a name="support-form-fields"></a>Поля формы службы поддержки

В следующей таблице описаны поля формы и входные данные, необходимые для каждого поля. Обязательные поля помечены звездочкой (*).

|  **Поле**                |     **Описание**                                                          |
|  ---------                |     ---------------                                                          |
| ***Контакт по техническим вопросам***  |  *Контактное лицо для связи по техническим вопросам между Майкрософт и вашей организацией*. | 
| **Имя\***                  | Имя лица или группы, предоставляющих техническую поддержку.     |
| **Электронная почта\***                 | Адрес электронной почты для связи по техническим вопросам.                                      |
| **\* телефона**                 | Номер телефона для получения технической поддержки.                                           |
| ***Поддержка клиентов***  | *Получает запросы в службу поддержки, открытые клиентами в Azure* |
| **Имя\***                  | Имя лица или группы, отвечающих за поддержку клиентов.                  |
| **Электронная почта\***                 | Адрес электронной почты службы поддержки клиентов                                            |
| **\* телефона**                 | Номер телефона службы поддержки клиентов.                                            |
| ***URL-адреса для получения поддержки***  | *Сайты поддержки, которые будет использовать Майкрософт, когда ваши клиенты будут отправлять запросы в службу поддержки*. |
| **Общедоступное облако Azure**          | URL-адрес общедоступного веб-сайта поддержки.                                         |
| **Облако Azure для государственных организаций**| URL-адрес сайта поддержки облака для государственных организаций.                                        |
|   |   |
