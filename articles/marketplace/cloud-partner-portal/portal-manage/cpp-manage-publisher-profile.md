# <a name="manage-your-azure-marketplace-and-appsource-publisher-profile"></a>Управление профилем издателя Azure Marketplace и AppSource

В этой статье объясняется, как использовать [портал Cloud Partner](https://cloudpartner.azure.com/) для просмотра и управления профилем издателя.  Вы уже должны быть зарегистрированным партнером Microsoft Cloud для доступа к этому порталу.  Дополнительные сведения см. [в статье Приступая к работе с облачным порталом для партнеров](../../cloud-partner-portal-orig/cloud-partner-portal-getting-started-with-the-cloud-partner-portal.md).

К этому моменту вы выполнили все шаги, чтобы стать издателем Azure Marketplace и AppSource. Профиль издателя регистрируется на Портал Cloud Partner следующего утверждения запроса партнера. Профиль издателя будет применяться ко всем предложениям и номерам SKU, опубликованным из учетной записи, которая используется при регистрации партнеров.


## <a name="publisher-profile"></a>Профиль издателя

Профиль издателя выделяет вашу организацию на Azure Marketplace и AppSource. Он состоит из идентификатора издателя, отображаемого имени и электронной почты владельца. Хорошо управляемый профиль позволит повысить уровень видимости и поможет пользователям Marketplace легко определять и выбирать соответствующее предложение.

> [!NOTE]
> Идентификатор издателя и зарегистрированную электронную почту владельца нельзя будет изменить после публикации первого предложения. Однако на облачном портале для партнеров можно просматривать свой профиль и редактировать отображаемое имя издателя.  <br/>
> Добавьте пользователей (участников и владельцев) в профиль публикации из раздела "Пользователи" в левой области навигации в разделе "Пользователи" Портал Cloud Partner.


### <a name="view-and-manage-your-publisher-profile"></a>Просмотр профиля издателя и управление им

Чтобы просмотреть профиль издателя и управлять им, в верхней правой строке меню щелкните вкладку **профиль издателя** .

![Вкладка "Профиль публикации"(Профиль публикации)](./media/publisherprofilenew.png)

Откроется диалоговое окно **профиля** .

![Сведения о профиле публикации (Сведения о профиле публикации)](./media/publisherprofiledetails.png)


### <a name="link-your-dev-center-account"></a>Связывание учетной записи центра разработки

На Портале Cloud Partner можно также связать существующую учетную запись Центра разработки с профилем издателя.  Сначала войдите на портал, используя тот же адрес электронной почты, который использовался при регистрации учетной записи Центра разработки. Сведения о состоянии учетной записи Центра разработки, электронная почта владельца учетной записи Центра разработки и имя учетной записи Центра разработки отобразятся на странице профиля издателя.

> [!NOTE]
> Регистрация учетной записи центра разработчиков обязательна для публикации платных номеров SKU для рынка.

Если у вас нет учетной записи центра разработчиков, можно создать учетную запись, как описано в разделе [Регистрация в центре разработки](https://docs.microsoft.com/azure/marketplace/register-dev-center).  Затем добавьте зарегистрированный адрес электронной почты в качестве владельца на Портал Cloud Partner для своего предложения. Владелец может только добавить зарегистрированный адрес электронной почты в профиль публикации из раздела **Пользователь**, а затем войти на портал, используя тот же адрес электронной почты, и привязать свою учетную запись Центра разработки.

![Сведения о профиле публикации (Описание профиля публикации)](./media/publisherprofiledescription.png)
