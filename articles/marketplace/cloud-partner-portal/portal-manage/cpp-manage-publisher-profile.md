# <a name="manage-your-azure-marketplace-and-appsource-publisher-profile"></a>Управление профилем издателя Azure Marketplace и AppSource

В этой статье описываются способы использования [облачный портал для партнеров](https://cloudpartner.azure.com/) для просмотра и управления профилем издателя.  Вам необходимо быть зарегистрированным облачный партнер Майкрософт доступ к этому порталу.  Дополнительные сведения см. в разделе [приступить к работе с облачным порталом для партнеров](../../cloud-partner-portal-orig/cloud-partner-portal-getting-started-with-the-cloud-partner-portal.md).

К этому моменту вы выполнили все шаги, чтобы стать издателем Azure Marketplace и AppSource. Профиль издателя зарегистрирован на портале Cloud Partner, после утверждения запроса партнера. Профиль издателя будет применяться ко всем предложениям и номерам SKU, опубликованным из учетной записи, которая используется при регистрации партнеров.


## <a name="publisher-profile"></a>Профиль издателя

Профиль издателя выделяет вашу организацию на Azure Marketplace и AppSource. Он состоит из идентификатора издателя, отображаемого имени и электронной почты владельца. Хорошо управляемый профиль позволит повысить уровень видимости и поможет пользователям Marketplace легко определять и выбирать соответствующее предложение.

> [!NOTE]
> Идентификатор издателя и зарегистрированную электронную почту владельца нельзя будет изменить после публикации первого предложения. Однако на облачном портале для партнеров можно просматривать свой профиль и редактировать отображаемое имя издателя.  <br/>
> Добавление пользователей (участников и владельцев) в профиль публикации из разделе "пользователи" на левой панели навигации в разделе "пользователи" на портале Cloud Partner.


### <a name="view-and-manage-your-publisher-profile"></a>Просмотр и Управление профилем издателя

Для просмотра и управления профилем издателя, в строке меню в правом верхнем, нажмите кнопку **профиль издателя** вкладки.

![Вкладка "Профиль публикации"(Профиль публикации)](./media/publisherprofilenew.png)

Ваш **профиль** откроется диалоговое окно.

![Сведения о профиле публикации (Сведения о профиле публикации)](./media/publisherprofiledetails.png)


### <a name="link-your-dev-center-account"></a>Привяжите свою учетную запись центра разработки

На Портале Cloud Partner можно также связать существующую учетную запись Центра разработки с профилем издателя.  Сначала войдите на портал, используя тот же адрес электронной почты, который использовался при регистрации учетной записи Центра разработки. Сведения о состоянии учетной записи Центра разработки, электронная почта владельца учетной записи Центра разработки и имя учетной записи Центра разработки отобразятся на странице профиля издателя.

> [!NOTE]
> Регистрация учетной записи центра разработчиков является обязательной для публикации, платные рынке номера SKU.

Если у вас нет учетной записи центра разработчиков, можно создать учетную запись, как описано в разделе [регистрации в центре разработки для](https://docs.microsoft.com/azure/marketplace/register-dev-center).  Затем добавьте зарегистрированный адрес электронной почты в качестве владельца на Портал Cloud Partner для своего предложения. Владелец может только добавить зарегистрированный адрес электронной почты в профиль публикации из раздела **Пользователь**, а затем войти на портал, используя тот же адрес электронной почты, и привязать свою учетную запись Центра разработки.

![Сведения о профиле публикации (Описание профиля публикации)](./media/publisherprofiledescription.png)
