# <a name="cloud-partner-portal-management"></a>Управление облаком портал для партнеров

В этом разделе описываются типы операций, доступных для создания и управления учетными записями, необходимые для использования [облачный портал для партнеров](https://cloudpartner.azure.com).  Эти учетные записи являются обязательными для [Azure Marketplace](https://azuremarketplace.microsoft.com) и [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Справочная информация

Прежде чем использовать облачный портал для партнеров, следует ознакомиться со следующими статьями:

- [Azure Marketplace политики и условия](../../policies-terms.md) перечислены различные юридических и технических требований и правил, которые применяются к пользователям и издателям [Azure Marketplace](https://azuremarketplace.microsoft.com) и [AppSource](https://appsource.microsoft.com).
- [Как стать издателем Cloud Marketplace](https://docs.microsoft.com/azure/marketplace/become-publisher) Обзор все действия, необходимые для станут официальной Microsoft publisher облака.

- [Управление профилем издателя Azure Marketplace и AppSource](./cpp-manage-publisher-profile.md) содержит сведения об использовании на портале Cloud Partner, для просмотра и управления профилем издателя marketplace Microsoft. 
- [Управление пользователями на облачном портале для партнеров](./cpp-manage-users.md) Добавление пользователей в облачной среде организации, публикации учетной записи и как управлять их доступ, чтобы разрешить нескольким пользователям работать на одном предложения.
