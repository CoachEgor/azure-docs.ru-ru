# <a name="cloud-partner-portal-management"></a>Управление Портал Cloud Partner

В этом разделе описываются типы операций, доступных для создания и управления учетными записями, необходимыми для использования [портал Cloud Partner](https://cloudpartner.azure.com).  Эти учетные записи необходимы для [Azure Marketplace](https://azuremarketplace.microsoft.com) и [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Справочная информация

Перед использованием Портал Cloud Partner необходимо ознакомиться со следующими статьями:

- В [политиках и терминах Azure Marketplace](../../policies-terms.md) перечислены различные юридические и технические требования и правила, которые применяются к пользователям и издателям в [Azure Marketplace](https://azuremarketplace.microsoft.com) и [AppSource](https://appsource.microsoft.com).
- [Станьте издателем облачных решений Marketplace](https://docs.microsoft.com/azure/marketplace/become-publisher) , вы подаете обзор всех шагов, необходимых для того, чтобы стать официальным издателем Microsoft Cloud.


## <a name="account-creation-and-management"></a>Создание учетной записи и управление ими

Этот раздел содержит следующие статьи, в которых объясняется, как управлять учетными записями издателя Marketplace.

- [Необходимые условия для управления учетными записями](./cpp-manage-prerequisites.md) поясняют, как стать зарегистрированным издателем облака. это требование для использования облачного портала для партнеров.
- [Управление профилем издателя в Azure Marketplace и AppSource](./cpp-manage-publisher-profile.md) объясняется, как использовать портал Cloud Partner для просмотра и управления профилем издателя Microsoft Marketplace. 
- [Управление пользователями в портал Cloud Partner](./cpp-manage-users.md) Добавление пользователей в учетную запись облачной публикации Организации и управление доступом, чтобы позволить нескольким пользователям работать с одними и теми же предложениями.
