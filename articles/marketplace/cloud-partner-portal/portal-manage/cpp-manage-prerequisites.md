# <a name="prerequisites-for-managing-accounts-on-the-cloud-partner-portal"></a>Необходимые условия для управления учетными записями на облачном портале для партнеров 

Перед использованием [облачный портал для партнеров](https://cloudpartner.azure.com/) для любой функции, включая управление учетными записями, необходимо выполнить ряд шагов для регистрации себя и своей организации в качестве издателей облака Microsoft.


## <a name="account-terms-and-guidelines"></a>Правила и условия учетной записи

Внимательно просмотрите ваши права и обязанности в группе [соглашение об использовании служб Майкрософт](https://www.microsoft.com/servicesagreement) и [условия использования Майкрософт](https://www.microsoft.com/legal/intellectualproperty/copyright) перед созданием учетной записи.  

Тщательно выбирайте имя вашей учетной записи.  Он представляет вашей организации, поэтому не должны содержать слов авторским правом или Соревнующиеся за ресурсы.  Например Корпорация Майкрософт запрещает использование авторским правом условия, такие как «Azure» и «Microsoft», в именах учетных записей.  Дополнительные сведения см. в разделе [рекомендации по Azure Marketplace](https://docs.microsoft.com/azure/marketplace/guidelines)


## <a name="registration-steps"></a>Действия для регистрации

В следующей таблице перечислены действия, необходимые для полностью зарегистрироваться как издатель облака Microsoft.  Полные инструкции см. в разделе [стать издателем Cloud Marketplace](https://docs.microsoft.com/azure/marketplace/become-publisher
). 


|                   Шаг                   |  Итоговый регистрации или ресурс                     |
|                  ------                  |  -----------------------------------                    |
| 1. Регистрация в партнерской сети Microsoft Partner Network |  Идентификатор Microsoft Partner Network для вашей организации |
| 2. Создание учетной записи Майкрософт *           |  Объект проверки учетной записи Майкрософт, заканчивающиеся на @outlook.com, @live.comи т. д. |
| 3. Подача формы регистрации Marketplace | Учетная запись издателя на облачном портале для партнеров      |
| 4. Вход на Портал Cloud Partner        | Права на создание и публикация предложений, управлять пользователей, marketplace сбор метрик, и т.д. |
| 5. Регистрация в Microsoft Developer Center * | Проверенную учетную запись центра разработки; полномочий для проведения транзакций marketplace  |
|   |   |

\* Эти шаги требуются для Azure Marketplace transact предложений, но настоятельно рекомендуется для всех предложений.  После установления учетную запись Майкрософт и зарегистрировать в качестве разработчика Майкрософт, доступны дополнительные ресурсы клиента и разработчика Майкрософт, а также предложения партнеров необязательно.  


## <a name="next-steps"></a>Дальнейшие действия

После завершения действия по регистрации, можно выполнить вход [облачный портал для партнеров](https://cloudpartner.azure.com/).  Этот портал можно использовать для управления профилями издателя и управлять учетными записями пользователей.
