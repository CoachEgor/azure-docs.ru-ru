# <a name="managing-users-on-the-cloud-partner-portal"></a>Управление пользователями на облачном портале для партнеров

[Облачный портал для партнеров](https://cloudpartner.azure.com) позволяет добавлять пользователей с доступом на основе ролей к предложениям виртуальных машин. Это позволяет управлять доступом и разрешениями, пока над предложениями работают несколько пользователей.

## <a name="add-users-and-assign-roles"></a>Добавление пользователей и назначение ролей 

Следуйте инструкциям ниже, чтобы добавить пользователей и назначение ролей для публикации предложения.

1. Нажмите кнопку **пользователей** вкладка в левой части области навигации.

    ![Вкладка "Пользователи"](./media/userstab.png)


2. Нажмите кнопку **Add User**(Добавить пользователя).

    ![Выбор элемента "Добавить пользователя"](./media/adduser.png)


3. Введите адрес электронной почты пользователя и выберите назначение роли.  Можно добавить нового пользователя как «владелец» или «участник».

    **Электронная почта:** Добавьте адреса электронной почты коллег, которые будут работать над публикацией предложения. Поддерживаются учетные записи Майкрософт (Outlook, Hotmail и Live) и идентификаторы организации.

    - Добавьте «команду, групповой адрес электронной почты или группу безопасности» для обеспечения защиты от лица из организации.
    - Убедитесь, что электронной почты, предоставленным на облачном портале для партнеров отслеживаются для любого взаимодействия от корпорации Майкрософт.
    
    ![Назначение роли](./media/assignrole.png)

    **Роль:** Просмотрите таблицу ниже, чтобы определить соответствующий тип роли пользователя.

    ![Уровни ролей](./media/roleaccesslevel.png)

    Только владельцы могут получить доступ к Azure выплаты и клиентов Azure вкладки в [Seller Insights](../../cloud-partner-portal-orig/si-getting-started.md).


4. Выберите имя издателя, которую вы хотите добавить нового пользователя, а затем щелкните "Добавить", чтобы завершить добавление пользователя.

    > [!NOTE]
    > Как владелец или участник, для которых добавляется адрес электронной почты «издатель имена» будут отображаться в раскрывающемся списке **издателя** для выбора.


    ![Назначение роли — выбор издателя](./media/assignselectpublisher.png)


## <a name="delete-an-existing-user"></a>Удаление существующего пользователя

Для удаления существующего пользователя, найдите их адрес электронной почты в **пользователей** вкладке, а затем нажмите клавишу **удалить** кнопки в области справа.
