# <a name="azure-application-support-tab"></a>Вкладка "Поддержка" приложения Azure

На вкладке "Поддержка" введите контактные данные технических специалистов и сведения о службе поддержки.
 
## <a name="identify-contacts-and-websites"></a>Определение контактных данных и веб-сайтов

При предоставлении необходимых сведений используйте в качестве руководства следующий снимок экрана. Все поля, имена которых обозначены красной звездочкой (*), должны быть заполнены.

  ![Форма обращения в службу поддержки для предложения](./media/azureapp-support-tab.png)

1. В разделе **Контактные данные технического специалиста** укажите сведения для следующих полей:

   - **Имя\***  -введите имя Инженерный для вашего приложения. Этому человеку будет передаваться техническая информация от корпорации Майкрософт.
   - **Адрес электронной почты\***  -введите адрес электронной почты контактного лица engineering.
   - **Телефон\***  — введите номер телефона контактного лица engineering.
  
2. В разделе **Контактные данные службы поддержки** укажите сведения для следующих полей:

   - **Имя\***  -введите имя контактного лица по поддержке для приложения. Этому человеку корпорация Майкрософт будет передавать сведения, касающиеся поддержки.
   - **Адрес электронной почты\***  -введите адрес электронной почты контактного лица по поддержке приложения.
   - **Телефон\***  — введите номер телефона контактного лица по поддержке.
 
3. В разделе **URL-адреса поддержки** введите соответствующие URL-адреса службы поддержки, которые корпорация Майкрософт будет использовать, когда клиенты будут открывать запросы в службу поддержки. Эти URL-адреса будут использоваться для общедоступной версии Azure или для облачной версии Azure для государственных организаций.
4. Когда все будет готово, нажмите кнопку **Сохранить**.
