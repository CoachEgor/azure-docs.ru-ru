# <a name="standard-contract"></a>Стандартный контракт

Чтобы упростить процесс закупок для клиентов и снизить юридические сложности для поставщиков программного обеспечения, корпорация Майкрософт предлагает шаблона Standard контракта для облегчения транзакции в marketplace. Вместо создания пользовательских условий, издателей Azure Marketplace можно предлагать свое программное обеспечение в стандартный контракт, который пользователю нужно лишь Обсудите и принять один раз. Стандартный контракт можно найти здесь: [ https://go.microsoft.com/fwlink/?linkid=2041178 ](https://go.microsoft.com/fwlink/?linkid=2041178). 

Условия для использования предложения определены на вкладке "Marketplace" при создании предложения на портале Cloud Partner. Параметр стандартный контракт обеспечивается путем изменения параметра Да.

![Включение параметра стандартный контракт](media/marketplace-publishers-guide/standard-contract.png)

>[!Note] 
>Если вы решили использовать стандартный контракт, отдельных условий по-прежнему необходимы для [поставщика облачных решений](./cloud-solution-providers.md) канала.

## <a name="standard-contract-amendments"></a>Изменения стандартной контракта

Стандартный контракт поправки позволяют издателям выберите стандартный контракт для простоты и терминами, настроенный для своих продуктов бизнес.  Пользователю нужно лишь просмотрите поправки к контракту, в том случае, если они уже проверены и приняты стандартный контракт Microsoft.

Существует два вида поправки для издателей Azure Marketplace:

* Универсальные изменения: Эти изменения применяются глобально стандартных контракта для всех клиентов. Универсальные изменения отображаются для каждого клиента в потоке покупки продукта.

![Универсальные поправки](media/marketplace-publishers-guide/universal-amendaments.png)

* Пользовательские изменения: Azure Marketplace есть резерв пользовательские изменения, предназначенные для клиентов. Они являются в стандартный контракт специальные изменения, предназначенные только для определенных клиентов. Издатели можно выбрать клиента, которыми они будете работать. Клиенты из этого клиента будет приобрести работу продукта в стандартный контракт и целевых поправки.

![Пользовательские изменения](media/marketplace-publishers-guide/custom-amendaments.png)

>[!Note] 
>Клиенты, применить пользовательские изменения также будет предоставлена универсальной поправка для стандартных условий во время покупки.

>[!Note]
>Следующие типы предложение поддерживают стандартные изменения контракта: Приложения Azure (шаблоны решений и управляемые приложения), виртуальные машины, контейнеры, приложения-контейнеры.

### <a name="customer-experience"></a>Качество обслуживания

Во время покупки на портале Azure клиенты будут иметь возможность см. в условиях, связанного с продуктом Microsoft стандартный контракт и поправки.

![Azure портала качество обслуживания клиентов.](media/marketplace-publishers-guide/ibiza-customer-experience.png)

### <a name="api"></a>API

Клиенты могут использовать `Get-AzureRmMarketplaceTerms` для получения условия предложения и принять его. Стандартный контракт и связанные изменения возвращается в выходных данных командлета.
