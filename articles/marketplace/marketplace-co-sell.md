# <a name="co-sell-partner-engagement"></a>Участие партнеров в совместных продажах

Программа совместной продажи Microsoft позволяет партнерам работать непосредственно с корпорацией Майкрософт над совместными возможностями продаж. Вы можете охватить огромное сообщество клиентов и создать новые возможности продаж, которые могут помочь ускорить рост бизнеса.

## <a name="partner-selling-motions"></a>Партнер продажи движений

Есть три партнера продажи движений:

* *Продажа через партнера*: Вы можете перепродать технологию Microsoft через собственных продавцов.
* *Совместное продажу услуг:* Продавцы и партнеры сотрудничают для продажи вашего проекта или управляемых сервисов со встроенными технологиями Майкрософт.
* *IP-продажа:* Продавцы/партнеры сотрудничают, чтобы продать ваше повторяемое, упакованное решение IP».

\*Некоторые из этих решений по совместной продаже IP могут получить дополнительные стимулы при успешной продаже и развертывании с помощью групп по продажам на местах корпорации Майкрософт.
