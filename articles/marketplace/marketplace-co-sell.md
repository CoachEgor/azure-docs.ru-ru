# <a name="co-sell-partner-engagement"></a>Участие партнеров в совместных продажах

Вы можете использовать программу совместной продажи Майкрософт для непосредственной работы с нами в совместных продажах. Эта программа поможет вам связаться с обширным сообществом клиентов и создать новые возможности для ускорения роста бизнеса.

## <a name="co-sell-channels"></a>Каналы совместной продажи

Вы можете совместно продавать свои решения по трем каналам:

* Непосредственно клиентам в Microsoft AppSource или Azure Marketplace.
* Совместно с другими партнерами.
* В совместной работе с Microsoft продавцов.

Программа совместной продажи доступна для **служб** с внедренными технологиями Майкрософт и для повторяемых, упакованных **решений интеллектуальной собственности (IP)**.

Некоторые IP-решения могут быть квалифицированы для поощрения продуктов Майкрософт, когда они успешно проданы и развернуты с группами продаж Майкрософт.

![Схема совместных продаж, когда потенциальные клиенты становятся общими, принимаются и заключают сделки с клиентами, управляемыми корпорацией Майкрософт.](./media/marketplace-publishers-guide/marketplace-co-sell.png)

## <a name="azure-ip-co-sell-incentivized-offers"></a>Предложения для совместной продажи Azure IP выгодно

Предложения в программе совместной продажи — это **готовность к продажам** или **Совместная продажа выгодно**в зависимости от требований, которые они выполняют.

Решения Azure IP, которые достигают **выгодно состояния совместной продажи** , разблокируют следующие преимущества:

* Плата за использование службы Marketplace уменьшается с 20 до 10% от выставляемых продаж. Эта скидка распространяется на решения IaaS и SaaS, продаваемые через коммерческий магазин Майкрософт. Дополнительные сведения см. в разделе [Сокращенная плата за обслуживание](marketplace-commercial-transaction-capabilities-and-considerations.md#reduced-service-fee) .
* Партнеры поставщика облачных решений (CSP) получают стимул продаж на 10% для продажи предложений SaaS и IaaS. Узнайте больше о CSP в [программе поставщика облачных решений](cloud-solution-providers.md).
* Предложения по совместной продажам Azure выгодно будут участвовать в обязательстве по потреблению Azure клиентами, когда они будут приобретены на коммерческом рынке. Соответствующие клиенты увидят предложение, помеченное как **преимущество Azure** , которое подходит для Интернет-магазинов. Дополнительные сведения см. в статье [преимущества обязательств по потреблению Azure](azure-consumption-commitment-benefit.md) .
* Предложения получают эмблему предпочтительных решений Майкрософт в Интернет-магазинах. Значок обеспечивает качество, производительность и способность удовлетворить потребности клиентов в определенной отрасли по вертикали или в области решения. Дополнительные сведения о [предпочтительных решениях см. в Microsoft AppSource и Azure Marketplace](preferred-solutions.md).

## <a name="co-sell-your-offer"></a>Совместная продажа предложения

Начните путешествие прямо в центре партнеров, выполнив приведенные ниже действия. Дополнительные сведения см. в статье о [параметрах совместной продажи в центре партнеров](./partner-center-portal/commercial-marketplace-co-sell.md).

1. Войдите в [Центр партнеров](https://partner.microsoft.com/dashboard/home).
1. На левой панели выберите Обзор **коммерческого рынка**  >  **Overview**, а затем выберите предложение для совместной продажи.

    > [!NOTE]
    > Вы можете присоединиться к программе совместной продажи с новым предложением, которое еще не опубликовано, или с уже опубликованным предложением.

1. В меню слева выберите **Совместная продажа с корпорацией Майкрософт**.

    [![На этом рисунке показано расположение совместной продажи на вкладке Майкрософт.](./media/co-sell/co-sell-with-microsoft-tab.png)](./media/co-sell/co-sell-with-microsoft-tab.png#lightbox)

1. Укажите сведения о предложении и отправьте материалы в разделе " **документы** ". Microsoft продавцов будет использовать эти сведения, чтобы рекомендовать и продать предложение.
 
   [![На этом рисунке показан раздел документы на вкладке Совместная продажа.](./media/co-sell/co-sell-documents-section.png)](./media/co-sell/co-sell-documents-section.png#lightbox)

1. Отправка контактных данных по продажам в разделе " **Контакты** ". Microsoft продавцов может обратиться к этим контактам, чтобы запросить дополнительные сведения о предложении.

     [![На этом рисунке показан раздел контакты на вкладке Совместная продажа.](./media/co-sell/co-sell-contacts-section.png)](./media/co-sell/co-sell-contacts-section.png#lightbox)

1. Сохраните черновик, прежде чем продолжить.
1. После того как все необходимые разделы предложения будут настроены, вы сможете отправить его для проверки и публикации. В правом верхнем углу выберите **проверить и опубликовать**. Просмотрите сведения о предложении, а затем выберите **опубликовать**.
1. Мы продемонстрируем загруженные материалы и сможем определить, подходит ли ваше предложение для программы совместной продажи.
