# <a name="co-sell-partner-engagement"></a>Участие партнеров в совместных продажах

Программа корпорации Майкрософт для совместной продажи позволяет партнерам работать непосредственно с корпорацией Майкрософт в совместных сделках. Вы можете обратиться к обширному сообществу клиентов и создать новые возможности для продаж, которые помогут ускорить рост бизнеса.

## <a name="partner-selling-motions"></a>Продажа партнерских движений

Существует три партнера по продажам:

* *Продавать через партнера*: Вы можете перепродать технологии Майкрософт через собственный продавцов.
* *Совместная продажа служб*: Продавцов и партнеры могут сотрудничать для продажи проекта или управляемых служб с внедренной технологией Майкрософт.
* *Совместная продажа по IP*: Продавцов и партнеры могут сотрудничать для продажи повторяемого, упакованного IP-решения *.

\*Некоторые из этих решений для совместной работы на основе IP-продаж могут получить дополнительную поощрения при успешной продаже и развертывании с помощью групп продаж корпорации Майкрософт.
