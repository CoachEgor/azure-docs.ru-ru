# <a name="co-sell-partner-engagement"></a>Участие партнеров в совместных продажах

Если вы являетесь партнером, можете использовать программу совместных продаж Майкрософт для непосредственной работы с корпорацией Майкрософт в совместных сделках. Программа помогает нашим партнерам расширить охват клиентов и создать новые возможности для продаж, которые помогут ускорить развитие бизнеса.

## <a name="partner-selling-motions"></a>Варианты партнерских продаж

Корпорация Майкрософт предлагает три варианта партнерских продаж.

* *Продажа через партнеров.* Вы можете перепродать технологии Майкрософт через собственных продавцов.
* *Совместная продажа служб.* Продавцы и партнеры сотрудничают для продажи вашего проекта или управляемых служб с внедренными технологиями Майкрософт.
* *Совместная продажа IP-решения.* Продавцы и партнеры сотрудничают для продажи стандартного упакованного IP-решения*.

\* Некоторые из этих совместно продаваемых IP-решений могут получать дополнительные поощрения при условии успешных продаж и развертываний с участием групп продаж Майкрософт на местах.
