# <a name="co-sell-partner-engagement"></a>Участие партнеров в совместных продажах

Если вы являетесь партнером, то можете использовать программу совместной продажи Майкрософт для непосредственной работы с корпорацией Майкрософт в совместных сделках. Программа помогает нашим партнерам достичь обширного сообщества клиентов и создавать новые возможности для продаж, которые помогут ускорить рост бизнеса.

## <a name="partner-selling-motions"></a>Продажа партнерских движений

Корпорация Майкрософт предлагает три партнера по продажам:

* *Продавать через партнера*. Вы можете перепродать технологии Майкрософт через собственный продавцов.
* *Совместная продажа служб*: продавцов и партнеры взаимодействуют с вами для продажи проекта или управляемых служб с внедренной технологией Майкрософт.
* *Совместная продажа по IP*: продавцов и партнеры взаимодействуют с возможностью пропродажи повторяемых, УПАКОВАННЫХ IP-решений. *

\*Некоторые из этих решений для совместной работы на основе IP-продаж могут получить дополнительное стимул, когда они успешно проданы и развернуты с группами продаж Майкрософт.
