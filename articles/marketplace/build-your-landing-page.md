# <a name="build-your-landing-page"></a>Создание целевой страницы

Целевая страница на веб-сайте вашей компании — это шлюз от инициатив по созданию спроса до вашего рейтинга в [Azure Marketplace](https://azuremarketplace.microsoft.com).

Начните с определения своих целей. Для своих решений выберите целевого клиента и действие, которое он должен выполнить. Решение может включать в себя следующие действия:
*   пробное использование решения;
*   получение решения сейчас

Вы можете ссылаться на свою целевую страницу из различных источников в соответствии с вашими маркетинговыми тактиками. Вы также можете использовать следующие маркетинговые тактики: 
*   События
*   веб-трансляции;
*   социальные сети;
*   Технические документы
*   сеансы технического обучения;
*   пресс-релизы.

Чем яснее посыл и призыв к действию, тем проще вашим клиентам найти решение.

## <a name="recommendations"></a>Рекомендации

При наличии готового плана кампании создайте целевую страницу в Marketplace. В следующей таблице описаны действия, позволяющие повысить эффективность целевой страницы. 

| Рекомендации | Избегать |
|:--- |:--- |
| **Укажите, какие проблемы клиентов поможет решить ваш продукт и как для этого используется Azure** | Избегайте нечеткой формулировки относительно интеграции вашего решения в Azure. |
| **Создайте короткий URL-адрес, который легко запомнить** | Не используйте длинные URL-адреса. Длинные URL-адреса плохо запоминаются, и их трудно найти. |
| **Добавьте релевантное визуальное содержимое, например видеоролик с отзывами заказчиков или с кратким описанием архитектуры** | Из-за использования слишком большого количества текста клиенты могут игнорировать ваше содержимое и прекращать изучение.|
| **Создайте четкий призыв к действию, чтобы направить посетителей на вашу страницу каталога Marketplace** | Избегайте использования слишком большого количества ссылок или действий на странице. |
| **Поместите свой призыв к действию в заголовок или четко выделяющийся раздел** | Не внедряйте ссылки в текст предложения Marketplace. |
| **Изучите рекомендации по ключевым словам и оптимизируйте страницу для поиска** | Не предполагайте, что название вашего продукта создаст более высокий рейтинг поиска само по себе. |
| **Используйте ключевые слова из своих рекламных кампаний** | Использование разных ключевых слов в ваших веб-ресурсах снизит эффективность ваших рекламных инвестиций. |
| **Помещайте соответствующие названия продуктов и ключевые слова выше сгиба** | Не заставляйте пользователей прокручивать страницу, чтобы увидеть, какой продукт или решение вы представляете. |
| **Используйте изображения торговой марки для проверки вашего решения в соответствии с рекомендациями по фирменной символике корпорации Майкрософт, например "Сертифицировано для Azure"**. | Не используйте неутвержденные изображения фирменной символики корпорации Майкрософт. |
