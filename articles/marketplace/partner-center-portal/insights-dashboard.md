# <a name="marketplace-insights-dashboard-in-commercial-marketplace-analytics"></a>Панель мониторинга Marketplace Insights в аналитике коммерческого рынка

В этой статье содержится информация о панели мониторинга Marketplace Insights в Партнерс-центре. На этой панели мониторинга отображается сводка веб-аналитики маркетплейса, которая позволяет издателям измерять вовлеченность клиентов для своих страниц с деталями продуктов, перечисленных на витринах магазинов: AppSource и Azure Marketplace.

## <a name="marketplace-insights-dashboard"></a>Панель мониторинга аналитики Marketplace

Чтобы получить доступ к **панели мониторинга Marketplace Insights** в Партнерс-центре, откройте **[вкладку «Анализ»](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)** в рамках Коммерческого рынка.

Вы можете просмотреть графические представления следующих элементов:  

- [Сводка «Взгляды на рынок»](#marketplace-insights-summary)
- [Посещения страниц по географии](#page-visits-by-geography)  
- [Посещения страниц по сравнению с уникальными тенденциями посетителей](#page-visits-versus-unique-visitors-trend)
- [Призыв к действию по сравнению с уникальными посетителями с CTAs](#call-to-action-versus-unique-visitors-with-ctas)
- [Посещения страниц ы и призыв к действию по предложениям](#page-visits-and-calls-to-action-by-offers)
- [Тенденция процентного призыва к действию](#call-to-action-percentage-trend)
- [Посещения страниц и призывы к действию по реферальным доменотам](#page-visits-and-calls-to-action-by-referral-domains)
- [Таблица подробностей Marketplace Insights](#marketplace-insights-details-table)

>[!NOTE]
> Для подробных определений терминологии аналитики, [см. Часто задаваемые вопросы и терминологию для коммерческой аналитики рынка](./faq-terminology.md).

### <a name="insights-dashboard-layout"></a>Макет панели мониторинга Insights

Вы можете просматривать метрики Marketplace различными способами:

- Вкладки магазина
- Фильтры страницы
- Фильтры по дате

**Вкладки Storefront:** Вы можете просматривать метрики ваших предложений отдельно через вкладки AppSource & Azure Marketplace. Выберите предложение (ы) из списка отбрасываемых предложений на право увидеть визуализацию метрик для выбранного предложения (ы). По умолчанию все предложения выбираются.

![Партнер Центра Исследования панели мониторинга предлагают выпадение списка](./media/insights-offer-dropdown.png)

**Фильтры страницы Insights**: Эти фильтры применяются на уровне предложения (страница детали продукта). Можно выбрать несколько фильтров для критериев, которые вы хотели бы просмотреть. Этот фильтр применяется ко всему разделу Marketplace Insights, включая диаграммы и детали.

![Фильтры панели мониторинга Partner Center Insights](./media/insights-page-filter.png)

- Имена предложений перечислены только для предложений, которые имеют посещения страниц в выбранном диапазоне дат.  
- Выбор по умолчанию является "Все" для каждого из вариантов фильтра
- Прикладные фильтры показывают количество выбора (ы) для сделанного выбора. Прикладные фильтры не будут отображаться для выбора "Все" по умолчанию.

![Фильтры Partner Center Insights](./media/insights-page-filter-two.png)

**Фильтры даты Insights**: Этот фильтр применяется ко всему разделу Marketplace Insights. Фильтры могут включать заранее определенные диапазоны дат или пользовательский диапазон дат.

![Фильтры даты partner Center Insights](./media/insights-date-range.png)

## <a name="marketplace-insights-summary"></a>Сводка «Взгляды на рынок»

В разделе сводная информация Marketplace отображается количество **посещений Страницы,** **Призывы к действию**и **уникальные посетители** для выбранного диапазона дат.

### <a name="page-visits"></a>Посещения страниц

Это число представляет подсчет различных пользовательских сессий на странице предложения (страница детали продукта) для выбранного диапазона дат. Индикатор красного/зеленого процента представляет собой % роста посещений страниц. Диаграмма тренда представляет количество посещений страниц за месяц до месяца.

### <a name="unique-visitors"></a>Уникальные посетители

Это число представляет собой четкое количество посетителей во время выбранного диапазона дат для предложения (ы), выбранного в фильтре страницы. Посетитель, посетивший одну или несколько страниц с деталями продукта, будет засчитан как один уникальный посетитель.

### <a name="call-to-action"></a>Призыв к действию

Этот номер представляет количество нажатий кнопки **«Призыв к действию»,** выполненной на странице предложения (страница детали продукта). **Звонки к действию** учитываются, когда **Get it now,** **Бесплатная пробная версия,** **Свяжитесь со мной,** и выбраны кнопки **тест-драйва.**

![Партнер Центра Исследования призыв к действию резюме](./media/insights-summary.png)

## <a name="page-visits-by-geography"></a>Посещения страниц по географии

Тепловая карта ниже отображает количество **посещений Страницы,** **Призывы к действию,** и **уникальные посетители**в зависимости от страны клиента . Более высокие посещения страниц представлены более темными цветами карт, а нижние посещения страницы представлены более светлыми цветами карт.

![Географический охват Центра Партнеров](./media/insights-geography.png)

Тепловая карта включает в себя следующие возможности:

- Тепловая карта имеет дополнительную сетку для просмотра деталей **посещений Страницы,** **Призывы к действию** и **уникальных посетителей** в определенном месте; вы можете увеличить в определенном месте, если предпочтительнее.  
- **Страны распространения** является подсчет всех стран, из которых ваши клиенты сообщили посещения страниц во время выбранного диапазона даты.
- Вы можете искать и выбирать страну в сетке, чтобы увеличить местоположение на карте. Вернуться к исходному представлению, выбрав **Дом** на карте.

## <a name="marketplace-insights-details-table"></a>Таблица подробностей Marketplace Insights

В этой таблице приводится представление списка посещений страниц ы и призывы к действию выбранных вами предложений, отсортированных по дате.

![Таблица подробностей Partner Center Insights](./media/insights-details-page.png)

- Данные могут быть извлечены в файл CSV, если количество записей меньше 1000.
- Если количество записей составляет более 1000, экспортированные данные будут асинхронно размещены на странице загрузки в течение следующих 30 дней.
- Фильтры могут быть применены для отображения данных, которые вас интересуют. Данные могут быть отфильтрованы именами предложений и именами кампаний.
