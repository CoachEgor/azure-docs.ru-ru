# <a name="account-migration-from-cloud-partner-portal-to-partner-center"></a>Миграция учетной записи из облачного портала для партнеров в центре партнеров

Если ваши предложения не будут перенесены из портала Cloud Partner (CPP) для центра партнеров (ПК), они блокируются для редактирования в CPP. На этом этапе настройки учетной записи необходимо перенести через Центр партнеров.  В центре партнеров можно управлять как параметры учетной записи, а также ваши предложения.

## <a name="account-migration-process"></a>Процесс миграции учетной записи

Если предложения переносятся из CPP, ваша учетная запись настроена для миграции. 
 
Если вы являетесь пользователем с ролью владельца в CPP для данной учетной записи, желтый баннер отображается на странице профиля издателя.  Вам будет предложено перенести настройки учетной записи центра партнеров. 

Щелкните баннер, чтобы инициировать процесс миграции учетной записи. Вы должны ввести следующее:

### <a name="work-e-mail-address"></a>**Адрес рабочей электронной почты**

В большинстве случаев, войдите с помощью адрес электронной почты, который вы используете для входа в CPP. В некоторых случаях необходимо использовать другой адрес:

* Учетная запись Майкрософт: Если учетная запись CPP учетную запись Майкрософт, необходимо ввести допустимый рабочий адрес электронной почты, связанный с клиентом, для которого регистрируется идентификатор MPN. 

* Несоответствие клиента: Если ваш рабочий адрес электронной почты не принадлежит к клиенту, который связан с идентификатор Microsoft Partner Network в учетной записи CPP, вы увидите ошибку. Перейти на эту ошибку, введите адрес электронной почты, связанный с клиентом. Сообщение об ошибке будет указано имя клиента. 

### <a name="sign-up-for-microsoft-partner-network-program"></a>Зарегистрироваться в программе Microsoft Partner Network

В случае недоступности учетной записи CPP не имеет идентификатор Microsoft Partner Network или имеет, является недопустимым, необходимо зарегистрироваться в программе Microsoft Partner Network в рамках процесса активации.

## <a name="account-activation-is-complete"></a>Активация учетной записи завершена

Миграция учетной записи должно происходить только один раз для данной учетной записи. После завершения миграции учетной записи участника все владельцы увидите это на своей странице профиля издателя:

1. Вы собираетесь см. на странице Параметры партнера в Microsoft Partner Network, где вы можете управлять параметрами учетной записи партнера Майкрософт. 
1. По завершении миграции учетной записи желтый баннер на странице профиля издателя, предлагающее пользователей, принадлежащих к роли владельца в CPP для данной учетной записи, запросом на управление параметрами учетной записи в центре партнеров. 
1. На страницу параметров учетной записи в CPP преобразуется в режиме только для чтения.
