# <a name="consulting-service-prerequisites"></a>Предварительные требования к консультационной услуге

> [!IMPORTANT]
> Мы перемещаем управление предложениями вашего консалтингового сервиса с облачного партнерского портала в партнерский центр. До тех пор, пока ваши предложения не будут перенесены, пожалуйста, следуйте инструкциям в [консалтинговом сервисе Azure и Dynamics 365](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal/consulting-services/cloud-partner-portal-consulting-services-publishing-offer) для Cloud Partner Portal для управления вашими предложениями.

Для получения информации о предложении консультационных услуг, включая требования к этому типу предложения, [см.](https://docs.microsoft.com/azure/marketplace/consulting-services) В следующей таблице показаны дополнительные требования к конкретным типам консультационных услуг.

| Тема консультационного обслуживания | Первичный продукт для выбора в партнерской центре | Требование |
|---------|---------|---------|
| Продукты Azure | Azure | Предложение должно перечислить по крайней мере одну из следующих полностью заработанных компетенций: <ul><li>Разработка приложений</li><li>Интеграция приложений</li><li>Управление жизненным циклом приложения</li><li>Облачная платформа</li><li>Аналитика данных</li><li>Центр обработки данных</li><li>Платформа данных</li><li>DevOps|
| Dynamics 365 Sales <br> Динамика 365 Маркетинг <br> Динамика 365 Обслуживание клиентов <br> Динамика 365 Полевая служба <br> Динамика 365 Кадровые ресурсы | Приложения для вовлечения клиентов | Должно быть золото или серебро сертифицированы в [компетенции облачных бизнес-приложений](https://partner.microsoft.com/membership/cloud-business-applications-competency) для обслуживания клиентов вариант, или совместно ежеспредложение для основного продукта, с помощью службы, связанных с. |

| Dynamics 365 Customer Insights | Dynamics 365 Customer Insights | Должна иметь по крайней мере одну успешную реализацию [Dynamics 365 Customer Insights,](https://dynamics.microsoft.com/ai/customer-insights/) по крайней мере, с пятью показателями и пятью сегментами, или иметь предложение о совместной продаже первичного продукта, с помощью которых связана услуга. |

| Power BI | Power BI | Должны быть перечислены на [Power BI партнера витрина](https://powerbi.microsoft.com/partner-showcase/), или совместно ежепредложение о продаже первичного продукта услуга связана с.|
