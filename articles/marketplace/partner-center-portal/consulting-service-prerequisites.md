# <a name="consulting-service-prerequisites"></a>Предварительные требования к консультационной услуге

Сведения о предложении службы консультации, включая требования для этого типа предложения, см. в статье [консультационные услуги для Microsoft AppSource и Azure Marketplace](../consulting-services.md). В следующей таблице приведены дополнительные требования для конкретных типов службы консультационных услуг.

| Раздел консультационной службы | Основной продукт для выбора в центре партнеров | Требование |
|---------|---------|---------|
| Продукты Azure&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;| Azure | Предложение должно содержать по крайней мере одну из следующих полностью полученных компетенций: <ul><li>Разработка приложений</li><li>Интеграция приложений</li><li>Управление жизненным циклом приложений</li><li>Облачная платформа</li><li>Аналитика данных</li><li>Центр обработки данных</li><li>Платформа данных</li><li>DevOps|
| Dynamics 365 Sales<br>Dynamics 365 Marketing<br>Dynamics 365 Customer Service<br>Dynamics 365 Field Service<br>Dynamics 365 Human Resources | Приложения для сотрудничества с клиентами | В [облачной Business Applications компетенции](https://partner.microsoft.com/membership/cloud-business-applications-competency) для участия в обслуживании клиентов должно быть указано значение Gold или серебро, а также предложение о совместной продаже для основного продукта, с которым связана служба. |
| Dynamics 365 Finance<br>Управление логистическими цепочками Dynamics 365<br>Dynamics 365 Commerce<br>Dynamics 365 Human Resources<br>Dynamics 365 Project Service Automation | Финансовые и операционные приложения | В [облаке Business Applications компетенции](https://partner.microsoft.com/membership/cloud-business-applications-competency) для Объединенных операций должен быть указан золотой или серебряный сертификат, а также предложение о совместной продаже для основного продукта, с которым связана служба. |
| Dynamics 365 Customer Insights | Dynamics 365 Customer Insights | Должна иметь по крайней мере одну успешную производственную реализацию [Dynamics 365 Customer Insights](https://dynamics.microsoft.com/ai/customer-insights/) с не менее чем пятью мерами и пятью сегментами или иметь предложение совместной продажи для основного продукта, с которым связана служба. |
| Dynamics 365 Business Central | Dynamics 365 Business Central | В [компетенции корпоративного планирования ресурсов](https://partner.microsoft.com/membership/enterprise-resource-planning-competency) должна быть указана золотой или серебристая сертификация, которая обслуживает по крайней мере три клиента или опубликовала бизнес-приложение в Microsoft AppSource или предложит предложение для совместной продажи для основного продукта, с которым связана служба. |
| Power BI | Power BI | Должен быть указан в [демонстрации партнера Power BI](https://powerbi.microsoft.com/partner-showcase/)или иметь предложение о совместной продаже для основного продукта, с которым связана служба.|
|Power Apps | Power Apps | Должен иметь право на дополнительные преимущества программы [партнерства Power Apps](https://aka.ms/PowerAppsPartner) или иметь предложение совместной продажи для основного продукта, с которым связана служба. |

Когда вы будете готовы создать предложение, см. статью [Создание предложения консультационных услуг](./create-consulting-service-offer.md).
