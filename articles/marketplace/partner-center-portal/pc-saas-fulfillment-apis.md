# <a name="saas-fulfillment-apis"></a>API исполнения SaaS

AAS Fulfillment AIS позволяют независимым поставщикам программного обеспечения (ISV) интегрировать свои приложения SaaS с Azure Marketplace. Эти AA позволяют приложениям ISV участвовать во всех каналах, включенных в торговлю: прямых, партнерских (реселлерах) и полевых.  Они являются обязательным требованием для перечисления трансформативных предложений SaaS на Рынке Azure.

> [!WARNING]
> Текущая версия этого API является версия 2, которая должна быть использована для всех новых предложений SaaS.  Версия 1 API является амортизированной и поддерживается для поддержки существующих предложений.


## <a name="business-model-support"></a>Поддержка бизнес-модели

Этот API поддерживает следующие возможности бизнес-модели; Вы можете:

* Укажите несколько планов предложения. Эти планы имеют различную функциональность и могут быть оценены по-разному.
* Предоставьте предложение на уровне на сайте или на основе биллинговой модели для одного пользователя.
* Предоставлять ежемесячные и ежегодные (платные авансовые) варианты выставления счетов.
* Предоставлять клиенту частное ценообразование на основе заключенного бизнес-соглашения.


## <a name="next-steps"></a>Дальнейшие действия

Если вы еще этого не сделали, зарегистрируйте свое приложение SaaS на [портале Azure,](https://ms.portal.azure.com) как это объясняется в [Регистрационном приложении Azure AD.](./pc-saas-registration.md)  После этого используйте самую актуальную версию этого интерфейса для разработки: [SaaS Fulfillment API Version 2](./pc-saas-fulfillment-api-v2.md).
