# <a name="payout-reporting"></a>Отчеты о выплатах

[**Резюме выплат**](https://docs.microsoft.com/windows/uwp/publish/payout-summary) показывает вам подробную информацию о деньгах, которые вы заработали с Microsoft. Она также содержит информацию о сроках получения платежей и сумме выплат.

Если вы продаете предложения в Azure Marketplace, вы также увидите информацию об успешных выплатах в **резюме выплат.** Более подробную информацию об оплате Azure Marketplace можно узнать в [политике участия Microsoft Azure Marketplace](https://go.microsoft.com/fwlink/p/?LinkId=722436) и [в соглашении об издателе Microsoft Azure Marketplace.](https://go.microsoft.com/fwlink/p/?LinkID=699560)

> [!NOTE]
> Чтобы иметь право на выплату, ваши доходы должны достичь [порога выплаты](payment-thresholds-methods-timeframes.md) в размере $ 50. Подробную информацию о пороге платежа можно узнать на этой странице и ознакомиться с [соглашением об издателях Microsoft Azure Marketplace.](https://go.microsoft.com/fwlink/p/?LinkID=699560)

## <a name="customer-types"></a>Типы клиентов

### <a name="enterprise-agreement"></a>Корпоративное соглашение

Клиенты без Соглашения Enterprise ежемесячно получают счета за лицензии на программное обеспечение из Marketplace. Клиенты с Соглашением Enterprise ежемесячно оплачивают счета через накладную, представляемую раз в квартал.

### <a name="credit-cards-and-monthly-invoice"></a>Кредитные карты и ежемесячные накладные

Клиенты также могут оплачивать решения с помощью кредитных карт и ежемесячных накладных. В этом случае плата за лицензию на программное обеспечение будет выставляться ежемесячно.

### <a name="csp-and-direct-pay-users"></a>CSP и прямые пользователи оплаты

Например, если клиент покупает с помощью кредитной карты.
