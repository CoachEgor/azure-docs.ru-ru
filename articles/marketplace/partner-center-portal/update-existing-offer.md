# <a name="update-an-existing-offer-in-the-commercial-marketplace"></a>Update an existing offer in the Commercial Marketplace (Обновление имеющегося предложения на коммерческой платформе Marketplace)

Вы можете просмотреть существующие предложения на вкладке **«Обзор»** [портала «Коммерческий рынок»](https://partner.microsoft.com/dashboard/commercial-marketplace/offers) в Партнерс-центре.

Чтобы обновить существующее предложение, которое в настоящее время находится на рынке:

1. Выберите название предложения, которое вы хотели бы обновить. Статус предложения может быть указан как **Live**, **Опубликовать в прогрессе**, **Проект**, **Внимание необходимо**, или **не продаются на рынке** (если вы ранее решили прекратить продажу предложения). После выбора откроется страница **обзора предложения** для этого предложения.
2. Выберите **обновление** из карты обзора или элемент меню из левой навигации для области, которую вы хотели бы обновить. Вы можете обновить **настройку предложения,** **Свойства,** **Предложение листинг**, **Предварительный просмотр**, **Техническая конфигурация**, **Обзор плана**, или **тест-драйв**. 
3. Внося изменения, а затем выберите **Сохранить**. Повторяйте этот процесс до тех пор, пока все ваши изменения не будут завершены.

## <a name="review-and-publish-an-updated-offer"></a>Просмотрите и опубликуйте обновленное предложение

Когда вы будете готовы опубликовать обновленное предложение, выберите **Publish** с любой страницы. **Страница «Обзор и публикация»** откроется. На этой странице доступны следующие возможности:

- Ознакомьтесь со статусом завершения для обновленных разделов предложения: 
    - **Неопубликованные изменения**: Раздел был обновлен и завершен. Все необходимые данные были предоставлены, и в обновлениях не было допущено никаких ошибок.
    - **Неполный**: Обновления, внесенные в раздел, ввели ошибки, которые должны быть исправлены или требует предоставления дополнительной информации.
- Предоставьте дополнительную информацию группе по сертификационным испытаниям, чтобы обеспечить бесперебойное тестирование.
- Отправить обновленное предложение для публикации, выбрав **Отправить**.  Мы отправим вам по электронной почте, когда предваротельная версия обновленного предложения будет доступна для просмотра и регистрации.

> [!IMPORTANT]
> Вы должны просмотреть предварительное просмотр предложения, как только оно будет доступно, и выберите **Go-live,** чтобы опубликовать обновленное предложение для вашей предполагаемой аудитории (публичной или частной).

## <a name="update-a-plan-within-an-existing-offer"></a>Обновление плана в рамках существующего предложения

Чтобы внести изменения в план в рамках уже опубликованного предложения:

- Поскольку страница **обзора предложения** открыта для существующего предложения, выберите план, который вы хотите изменить. Если план недоступен из списка **обзоров Плана,** выберите **все планы.**
- Вы можете выбрать **название**плана, **модель ценообразования**или **доступность.** *В настоящее время планы доступны только на английском языке (Соединенные Штаты Америки).*
- Выберите **Сохранить** после внесения необходимых изменений в название плана, описание, цены или доступность аудитории. 
- Выберите **Publish,** как только вы будете готовы опубликовать свои обновления. **[Страница «Обзор и публикация»](#review-and-publish-an-updated-offer)** откроется и предоставит статус завершения для ваших обновлений.

## <a name="compare-changes-to-marketplace-offers"></a>Сравните изменения в предложениях рынка

Вы можете провести аудит изменений, внесенных в [опубликованное](#compare-changes-to-published-offer) или [предварительное предложение,](#compare-changes-to-a-preview-offer) прежде чем сделать их в прямом эфире с помощью **Compare.**

>[!NOTE]
> Опубликованное предложение — это предложение, которое было успешно опубликовано в журнале Preview или Live.

Ниже приведена общая информация о ревизии:

- Вы можете использовать **Сравнить** в любой момент во время процесса редактирования.
- Выберите поле на странице **Сравнить** для навигации по значению, выбранное для изменения.
- Если вы готовы опубликовать свои обновления, выберите **Обзор и опубликовать**.
- Чтобы увидеть значения для всех полей, даже не обновляемых полей, выберите фильтр **All fields.** Вы можете изменить фильтры в этих полях, выбрав **измененные поля,** а затем выбрав один из фильтров ниже:
    - **Фильтр удаленных значений** отображает поля, которые вы опубликовали и теперь полностью удаляете.
    - Фильтр **добавленных значений** отображает поля, которые вы изначально не публиковали и теперь добавляете.
    - **Фильтр отредактированных значений** отображает поля, которые были опубликованы, но теперь вы обновили содержимое.

      >[!NOTE]
      > Если один из этих фильтров недоступен, это означает, что вы не сделали обновление этого типа.

- Чтобы увидеть только значения, которые не были обновлены, выберите фильтр **неизмененных полей.** Значения поля, показанные для опубликованной и черновой версии, будут одинаковыми.

  ![Фильтры для сравнения обновлений с опубликованным или предварительным предложением](./media/compare-changes-marketplace.png)

>[!NOTE]
> Следующие страницы в настоящее время не поддерживают **Сравнение:**
>- Аудитория реселлеров CSP
>- Техническая конфигурация тест-драйва
>- Тест-драйв Marketplace листинг
>- Со-продажа
>- Дополнительные файлы

### <a name="compare-changes-to-published-offer"></a>Сравните изменения с опубликованным предложением

Следуйте инструкциям ниже, чтобы сравнить ваши изменения из опубликованного предложения:

1. Выберите **Сравнить** в панели команд страницы. Страница **«Сравнение»** предоставляет побочные версии сохраненных изменений этого предложения и опубликованного предложения на рынке.
2. При доступе **К сравниванию** с определенной страницы предложения по умолчанию отображаются только изменения, внесенные на этой странице.
    - Если вы хотите сравнить изменения на всех страницах, измените страницу с **страницы Выберите для сравнения**.  
    - Если вы хотите сравнить изменения в предложении на всех страницах, выберите **все страницы.**

По **умолчанию, Сравните** контрастирует ваши новые изменения с предложением живого рынка.

Не забудьте переиздать свое предложение после внесения обновлений для изменений, которые всприсят в силу.

### <a name="compare-changes-to-a-preview-offer"></a>Сравните изменения с предложением предварительного просмотра

Если у вас есть изменения в предварительном просмотре, которые не живут, вы можете сравнить новые изменения с предложением рынка предварительного просмотра.

Следуйте инструкциям ниже, чтобы сравнить новые изменения с вашим предложением рынка предварительного просмотра:

1. Выберите **Сравнить** в панели команд страницы.
2. Выберите **с** выпадением и изменить его от **Live предложение** **Предварительный предложение**.
3. Страница **«Сравнение»** предоставляет версии, отображающие изменения.

>[!NOTE]
>Если ваше предложение еще не стало живым, но вы опубликовали для предварительного просмотра, у вас нет возможности сравнить с живым предложением.

Не забудьте переиздать свое предложение после внесения обновлений для изменений, которые всприсят в силу.

## <a name="delete-a-draft-offer"></a>Удалить черновик предложения

Вы можете удалить черновик предложения (тот, который не был опубликован), выбрав **проект Delete** со страницы **обзора предложения.** Эта опция не будет доступна для вас, если вы ранее опубликовали предложение.

После подтверждения того, что вы хотите удалить проект предложения, предложение больше не будет видно или доступно в Partner Center и страница **All Offers** откроется.

## <a name="delete-a-draft-plan"></a>Удалить проект плана

Чтобы удалить неопубликованный план, выберите **проект «Удалить»** на странице **обзора Плана.** Эта опция не будет доступна для вас, если вы ранее опубликовали предложение.

После подтверждения того, что вы хотите удалить проект плана, план больше не будет виден или доступен в Partner Center.

## <a name="cancel-publishing"></a>Отмена публикации

Чтобы отменить предложение со статусом **«Публикация в прогрессе»:**

- Выберите имя предложения, чтобы открыть страницу **обзора предложения.**
- Выберите **Отмена опубликовать** из верхнего правого угла страницы.
- Подтвердите, что вы хотите остановить публикацию предложения.

Если вы хотите опубликовать предложение позже, вам нужно начать процесс публикации.

> [!NOTE]
> Вы можете остановить предложение от публикации только в том случае, если предложение еще не продвинулось до знака издателя от шага. После выбора **Go Live** Live у вас не будет возможности отменить публикацию больше.

## <a name="stop-selling-an-offer-or-plan"></a>Прекратить продажу предложения или плана

Чтобы прекратить продажу предложения после того, как вы уже опубликовали его, выберите **Остановить продажу** со страницы **обзора предложения.** Чтобы прекратить продажу плана, выберите **«Остановить продажу»** на странице **обзора плана.** (Возможность прекратить продажу плана доступна только в том случае, если в предложении более одного плана. Вы можете отказаться от продажи одного плана, не влияя на другие планы в рамках вашего предложения.)

Как только вы подтвердите, что вы хотите прекратить продажу предложения или плана, в течение нескольких часов он больше не будет виден на рынке, и никакие новые клиенты не могут скачать его.

Любые клиенты, которые ранее приобрели предложение или план, все еще могут его использовать. Они могут скачать его снова, но не будут получать обновления, если вы обновляете и переиздаете предложение или план на более позднее время. 

После того, как ваш запрос на прекращение продажи предложения/плана будет завершен, вы все равно увидите его на портале Commercial Marketplace в Partner Center. Чтобы удалить черновик, выберите **проект «Удалить»** на странице **обзора предложения** или **плана.** 

Если вы решили перечислить или продать это предложение или план снова, следуйте инструкциям для [обновления существующего предложения](#update-an-existing-offer-in-the-commercial-marketplace). Не забывайте, что вам нужно будет **опубликовать** предложение или план снова после внесения каких-либо изменений.

## <a name="remove-offers-from-existing-customers"></a>Удаление предложений от существующих клиентов

Чтобы удалить предложения от существующих клиентов, [войдите в запрос поддержки.](https://support.microsoft.com/supportforbusiness/productselection?sapId=48734891-ee9a-5d77-bf29-82bf8d8111ff) Выберите свой конкретный тип предложения из списка **типов проблемы,** а затем выберите **Удалить опубликованное предложение** в списке **категорий.** Отправьте запрос. Команда поддержки проведет вас через процесс удаления предложения.

## <a name="sync-private-plan-audiences"></a>Синхронизация аудитории частных планов

Если ваше предложение включает в себя один или несколько планов, которые настроены, чтобы быть доступны только для частной ограниченной аудитории, можно обновить только аудиторию, которая может получить доступ к этому частному плану без публикации других изменений в предложении. 

Обновление и синхронизация частной аудитории для вашего плана(ы):

- Измените аудиторию в одном или нескольких частных планах, используя кнопку **«Добавить ID»** или **«Импорт» (csv),** а затем сохраните изменения.
- Выберите **частную аудиторию Sync** на странице **обзора Плана.**

**Синхронизация частной аудитории** будет публиковать только изменения в вашей частной аудитории, без публикации каких-либо других обновлений, которые вы могли бы сделать для проекта предложения.
