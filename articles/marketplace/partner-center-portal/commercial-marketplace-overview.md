# <a name="overview-of-the-commercial-marketplace-portal-in-partner-center"></a>Обзор портала коммерческого рынка в центре партнеров

Добро пожаловать на портал коммерческого рынка в центре партнеров Майкрософт. [Портал коммерческого рынка](https://partner.microsoft.com/dashboard/commercial-marketplace/) призван помочь вам монетизации ваши продукты и услуги, получить доступ к клиентам и расширить свой бизнес.

В этой документации будут рассмотрены основные сведения о создании и использовании учетной записи центра партнеров, созданию предложений по разработке программного обеспечения как услуги (SaaS), которые могут быть указаны в [Azure Marketplace](https://azuremarketplace.microsoft.com/) или [AppSource](https://appsource.microsoft.com/), управлении существующими предложениями SaaS и предоставление ресурсов для помощи с вопросами и сбора отзывов. 

## <a name="about-microsofts-commercial-marketplace"></a>О коммерческом магазине Майкрософт

Предприятия, государственные учреждения и организации всех размеров находят бизнес-решения, инновационную технологию и службы поддержки через веб-магазины [Azure Marketplace](https://azuremarketplace.microsoft.com/) и [AppSource](https://appsource.microsoft.com/) , [портал Azure в рамках продукта](https://portal.azure.com), [Поставщики облачных решений](https://partner.microsoft.com/cloud-solution-provider)и группы продаж Майкрософт. 

Центр партнеров — это центр работы партнера корпорации Майкрософт. Для организаций, которые переходят на рынок с помощью корпорации Майкрософт и пользуются преимуществами Marketplace, центр партнеров — это портал, с помощью которого издатели создают предложения и управляют ими, предоставляет демонстрационные видеоролики и просматривают аналитику на производительность предложения. 

## <a name="migration-from-cloud-partner-portal"></a>Миграция с Портал Cloud Partner

Центр партнеров развивается для лучшей интеграции всех аспектов связи Майкрософт с издателями программного обеспечения. В рамках этой эволюции мы переносим функции [портал Cloud Partner](https://cloudpartner.azure.com/) в центр партнеров. 

**В настоящее время центр партнеров поддерживает только создание и Управление предложениями Software-as-Service (SaaS).**

Существующие издатели Marketplace найдут знакомый, но Улучшенный интерфейс работы с интуитивно понятными возможностями. Уведомления, отправляемые на портале и через электронную почту, будут информировать издателей о миграции. Пользователи будут легко направляться в соответствующее расположение. Этот процесс не влияет на клиентов. 

### <a name="improvements-on-user-experience-and-billing"></a>Улучшения в работе пользователей и выставлении счетов

Благодаря переходу в центр партнеров новые функции становятся доступными для издателей, а также для улучшения взаимодействия с пользователем. Предложения SaaS, которые могут оплачиваться ежемесячно или ежегодно, поддерживают лицензирование для каждого пользователя (на основе места) в дополнение к лицензированию "на сайт" (плоская ставка). Эта новая модель монетизацию идеально подходит для бизнес-программ. Введение лицензирования SaaS на пользователя соответствует новым функциям, которые позволяют клиентам выполнять покупки в AppSource, коммерческом рынке Майкрософт для бизнес-решений и отраслевых решениях. 

### <a name="account-and-user-management-updates"></a>Обновления управления учетными записями и пользователями

Управление учетными записями и пользователями в центре партнеров упрощает доступ пользователей к порталу и администраторам для применения соответствующих разрешений. По сравнению с более ранними порталами управления, возможности управления доступом на основе ролей в центре партнеров обеспечивают улучшенную безопасность, которая упрощает процесс публикации для больших организаций. 

### <a name="improved-certification-experience"></a>Улучшенные возможности сертификации

Центр партнеров предоставляет улучшенные возможности сертификации. Теперь сертификация будет использовать автоматизацию и машинное обучение для улучшения времени работы и ускорения обработки. Обеспечение точности будет иметь высший приоритет, что обеспечит безопасность облачного магазина и содержит ценные предложения для наших клиентов. Требования не изменятся, но теперь при отправке предложений через центр партнеров вы получите более подробные и описательные отчеты о сертификации, ссылающиеся на [общедоступные политики](https://docs.microsoft.com/legal/marketplace/certification-policies). 

### <a name="analytics-for-performance-reporting"></a>Аналитика для отчетов о производительности

Аналитика центра партнеров предоставляет представление о производительности ваших предложений и улучшенных средствах создания отчетов. Издатели могут отслеживать продажи с помощью графов визуализации и анализа данных для оптимизации списков. Усовершенствованные средства анализа позволяют работать с результатами производительности и обеспечивать лучшую связь с клиентами и торговыми посредниками.
