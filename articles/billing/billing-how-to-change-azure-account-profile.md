# <a name="change-your-azure-account-profile-information"></a>Изменение сведений в профиле учетной записи Azure

Данные профиля используется для связи с вами об учетной записи Azure. Следует помнить о его обновлены, чтобы корпорация Майкрософт может оповещать вас о важных уведомлений.

Адрес электронной почты в профиле администратора учетной записи используется корпорацией Майкрософт для отправки важных выставления счетов, службы и связанных с рекомендация уведомления об учетной записи. Мы рекомендуем указать адрес электронной почты, который будет регулярно. Администратор учетной записи вы также можете обновить свой профиль, чтобы изменить почтовый адрес и номер телефона для учетной записи.

Адрес электронной почты в профиле пользователя, учетная запись пользователя, который не является администратором учетной записи, используется корпорацией Майкрософт для отправки уведомлений, связанных со службами и рекомендации по учетной записи. Все уведомления о выставлении счетов отправляются для администратора учетной записи. Вы также можете обновить свой профиль, чтобы изменить почтовый адрес и номер телефона, но он не изменяет учетную запись почтовый адрес или номер телефона.

## <a name="change-your-profile-information"></a>Изменение сведений в профиле

1. Войдите в [Центр учетных записей Azure](https://account.azure.com/Profile).
1. Нажмите кнопку **Изменить сведения**, а затем обновите сведения в **профиле**.

   ![Пример, демонстрирующий сведения профиля Azure](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>Отправлять уведомления на адрес электронной почты дополнительных

Возможно, служба уведомления и рекомендации по учетной записи, отправленных на адрес электронной почты, а также адрес электронной почты администратора учетной записи, связанные с учетной записью. Адрес электронной почты администратора учетной записи получает важные уведомления обо всех подписках учетной записи, включая обновления, связанные с выставлением счетов.

### <a name="change-the-account-administrator-email-address-in-your-profile"></a>Изменение адреса электронной почты администратора учетной записи в профиль

Администратор учетной записи — это человек, который настраивает учетную запись Azure и получает электронные уведомлений обо всех подписках в учетной записи. Рекомендуется указать контактный адрес электронной почты для регулярной проверки администратором учетной записи.

### <a name="updating-your-profile-email-doesnt-update-your-sign-in-email-address"></a>Обновление профиля электронной почты не обновляет свой адрес электронной почты входа в систему

Обновление адреса электронной почты профиля не обновляет свой адрес электронной почты входа в систему. Чтобы изменить адрес электронной почты входа в систему, необходимо передать права владения учетной записи.

### <a name="updating-your-profile-address-doesnt-update-your-credit-card-billing-address"></a>Обновление адреса профиля не обновить адрес для выставления счетов вашей кредитной карты

Сведения об обновлении данных о счетах см. в статье [Изменение кредитной карты, используемой для оплаты подписки Azure](billing-how-to-change-credit-card.md).

### <a name="you-cant-update-your-countryregion"></a>Не удается обновить страну или регион

Из-за технических ограничений мы нельзя изменить страну существующей учетной записи. Тем не менее можно создать учетную запись в требуемой страны или региона и затем обратитесь в службу поддержки Azure для переноса служб в этой учетной записи.

## <a name="service-and-marketing-emails"></a>Службы и маркетинговых сообщений электронной почты

Команда Microsoft Azure отправляет связанные со службой сообщений электронной почты о том, критических предупреждений. Команда также отправляет рекомендация учетной записи отправляется сообщение электронной почты, также известный как маркетинговых сообщений электронной почты, чтобы помочь вам оптимизировать и масштабы ваших проектов.

В случае необходимости в [портала Azure](https://portal.azure.com) проверка или обновить адрес электронной почты каждые 90 дней. Уведомление предназначена для убедитесь в том, что корпорация Майкрософт обновила контактные данные в случае срочных со службами уведомления. Майкрософт отправляет сообщения электронной почты на адрес электронной почты профиля с помощью Azure сведения, связанные с учетной записью, для:

- Уведомления службы
- Оповещения безопасности
- В целях выставления счетов
- Поддержка
- Распространение маркетинговой информации
- Практические рекомендации, на основе вашего использования Azure

Введите адрес электронной почты, где вы хотите получать сведения о вашей учетной записи. Введя адрес электронной почты, вы желанию в получать сообщения от корпорации Майкрософт.

![Пример, демонстрирующий обновления box контактные данные](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>Изменить адрес контактного адреса электронной почты

Можно изменить адрес контактного адреса электронной почты, с помощью следующих методов:

**Вариант 1**. Для входа на [центре управления учетной записью](https://account.azure.com/Profile). Выберите **изменения сведений о** и обновите сведения о профиле адрес электронной почты. Обновление адреса электронной почты профиля не обновляет свой адрес электронной почты входа в систему.

**Вариант 2**. Перейдите к [контактные данные](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) области на портале Azure и введите адрес электронной почты. Указанные вами адрес получателя, службы и маркетинговых материалов. Адрес электронной почты в профиле Center учетной записи Azure обновляется для соответствия автоматически. Обновление адреса электронной почты профиля не обновляет свой адрес электронной почты входа в систему.

**Вариант 3**. В [портала Azure](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade), щелкните значок с инициалами или изображение. Выберите в контекстном меню (**...** ). Затем выберите **Мои контактные данные** в меню и введите адрес электронной почты. Указанные вами адрес получателя, службы и маркетинговых материалов. Адрес электронной почты в вашем профиле Center учетной записи Azure обновляется для соответствия автоматически. Обновление адреса электронной почты профиля не обновляет свой адрес электронной почты входа в систему.

![Пример, демонстрирующий обновление контактных данных в Azure](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>Отказаться от маркетинговых сообщений электронной почты

Чтобы отказаться от получения маркетинговых сообщений электронной почты:

1. Перейдите к [форма запроса](https://account.microsoft.com/profile/permissions-link-request) отправить запрос, используя свой адрес электронной почты профиля. Вы получите ссылку по электронной почте, чтобы обновить параметры.
2. Эта ссылка показаны типы рекламные сообщения, включаются в адрес электронной почты. Очистить все разделы, которые вы хотите отказаться от, а затем нажмите кнопку **Сохранить**.


![Пример, демонстрирующий параметры разрешений communications управление](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

### <a name="you-still-receive-service-notifications-if-you-opt-out-of-marketing-communications"></a>Вы по-прежнему получать уведомления службы, если вы отказаться от рекламные сообщения

Если вы указали адрес электронной почты для связи, вы будете получать уведомления службы, в зависимости от вашей учетной записи. Когда вы отказаться, вы только отказаться от маркетинговых материалов, вы по-прежнему получать уведомления службы.
