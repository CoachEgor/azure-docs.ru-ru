# <a name="change-the-profile-information-for-your-azure-account"></a>Измените данные профиля для учетной записи Azure

Корпорация Майкрософт использует адрес электронной почты в профиле администратора учетной записи для отправки важных выставления счетов, службы и связанных с рекомендация уведомления об учетной записи Azure. Мы рекомендуем указать адрес электронной почты, который будет регулярно. Администратор учетной записи, вы также можете обновить свой профиль, чтобы изменить почтовый адрес и номер телефона для учетной записи.

Корпорация Майкрософт использует адрес электронной почты в профиле пользователя, учетная запись пользователя, который не является администратором учетной записи, для отправки связанных со службами уведомления и рекомендации по учетной записи. Все уведомления о выставлении счетов отправляются для администратора учетной записи. Вы также можете обновить свой профиль, чтобы изменить почтовый адрес и номер телефона, но он не изменяет учетную запись почтовый адрес или номер телефона.

## <a name="change-your-profile-information"></a>Изменение сведений в профиле

1. Войдите в [Центр учетных записей Azure](https://account.azure.com/Profile).
1. Нажмите кнопку **Изменить сведения**, а затем обновите сведения в **профиле**.

   ![Пример того, сведения о профиле Azure](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>Отправлять уведомления на адрес электронной почты дополнительных

Возможно, служба уведомления и рекомендации по учетной записи, отправленных на адрес электронной почты, помимо адреса электронной почты администратора учетной записи, связанной с вашей учетной записи. Адрес электронной почты администратора учетной записи получает важные уведомления обо всех подписках учетной записи, включая обновления, связанные с выставлением счетов.

### <a name="change-the-account-administrators-email-address-in-your-profile"></a>Изменение адреса электронной почты администратора учетной записи в профиль

Администратор учетной записи — это человек, который настраивает учетную запись Azure и получает электронные уведомлений обо всех подписках в учетной записи. Рекомендуется указать контактный адрес электронной почты для регулярной проверки администратором учетной записи.

### <a name="update-your-sign-in-email-address"></a>Обновить адрес электронной почты входа в систему

Обновление адреса электронной почты профиля не обновляет свой адрес электронной почты входа в систему. Чтобы изменить адрес электронной почты входа в систему, необходимо передать права владения учетной записи.

### <a name="update-the-billing-address-for-your-credit-card"></a>Обновить адрес для выставления счетов для вашей кредитной карты

Сведения об обновлении данных о счетах см. в статье [Изменение кредитной карты, используемой для оплаты подписки Azure](billing-how-to-change-credit-card.md).

### <a name="update-your-countryregion"></a>Обновление вашей страны/региона

Из-за технических ограничений мы нельзя изменить страну существующей учетной записи. Но можно создать учетную запись в требуемой страны или региона и затем обратитесь в службу поддержки Azure для переноса служб в этой учетной записи.

## <a name="service-and-marketing-emails"></a>Службы и маркетинговых сообщений электронной почты

Появится запрос в [портала Azure](https://portal.azure.com) проверка или обновить адрес электронной почты каждые 90 дней. Майкрософт отправляет сообщения электронной почты на адрес электронной почты профиля с помощью Azure сведения, связанные с учетной записью, для:

- Уведомления службы
- Оповещения безопасности
- В целях выставления счетов
- Поддержка
- Распространение маркетинговой информации
- Практические рекомендации, на основе вашего использования Azure

Введите адрес электронной почты, где вы хотите получать сведения о вашей учетной записи. Введя адрес электронной почты, вы согласие на использование получать сообщения от корпорации Майкрософт.

![Примером запроса, чтобы обновить свои контактные данные](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>Изменить адрес контактного адреса электронной почты

Можно изменить адрес контактного адреса электронной почты с помощью одного из следующих методов:

* Войдите в [Центр учетных записей Azure](https://account.azure.com/Profile). Выберите **изменения сведений о** и обновите адрес электронной почты из профиля. Обновление адреса электронной почты профиля не обновляет свой адрес электронной почты входа в систему.

* Перейдите к [контактные данные](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) области на портале Azure и введите адрес электронной почты. Указываемое вами адрес получателя, службы и маркетинговых материалов. Адрес электронной почты в профиле Center учетной записи Azure обновляется для соответствия автоматически. Обновление адреса электронной почты профиля не обновляет свой адрес электронной почты входа в систему.

* В [портала Azure](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade), щелкните значок с инициалами или изображение. Выберите в контекстном меню (**...** ). Затем выберите **Мои контактные данные** в меню и введите адрес электронной почты. Указываемое вами адрес получателя, службы и маркетинговых материалов. Адрес электронной почты в вашем профиле Center учетной записи Azure обновляется для соответствия автоматически. Обновление адреса электронной почты профиля не обновляет свой адрес электронной почты входа в систему.

![Примеры обновления адрес электронной почты в Azure](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>Отказаться от маркетинговых сообщений электронной почты

Чтобы отказаться от получения маркетинговых сообщений электронной почты:

1. Перейдите к [форма запроса](https://account.microsoft.com/profile/permissions-link-request) отправить запрос, используя свой адрес электронной почты профиля. Вы получите ссылку по электронной почте, чтобы обновить параметры.
2. Выберите ссылку, чтобы открыть **Управление разрешениями связи** страницы. На этой странице отображаются типы рекламные сообщения, адрес электронной почты включено. Очистить все разделы, которые вы хотите отказаться от, а затем выберите **Сохранить**.


![Пример страницы для управления разрешениями, связи](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

Если отказаться рекламные сообщения, вы по-прежнему получать уведомления службы, на основе вашей учетной записи.
