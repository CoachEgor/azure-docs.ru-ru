# <a name="change-the-profile-information-for-your-azure-account"></a>Изменение сведений о профиле для учетной записи Azure

Корпорация Майкрософт использует адрес электронной почты в профиле администратора учетной записи для отправки важных уведомлений, связанных с выставлением счетов, службами и рекомендациями об учетной записи Azure. Рекомендуется указывать регулярно проверяемый адрес электронной почты. Администратор учетной записи также может обновить профиль, чтобы изменить почтовый адрес и номер телефона для учетной записи.

Корпорация Майкрософт использует адрес электронной почты в профиле пользователя, пользователя учетной записи, который не является администратором учетной записи, для отправки уведомлений, относящихся к службе, и рекомендаций по учетным записям. Все уведомления о выставлении счетов отправляются администратору учетной записи. Вы также можете обновить профиль, чтобы изменить почтовый адрес и номер телефона, но не изменить почтовый адрес или номер телефона учетной записи.

## <a name="change-your-profile-information"></a>Изменение сведений о профиле

1. Войдите в [Центр учетных записей Azure](https://account.azure.com/subscriptions).
1. Нажмите кнопку **Изменить сведения**, а затем обновите сведения в **профиле**.

   ![Пример сведений о профиле Azure](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>Отправлять уведомления на дополнительный адрес электронной почты

В дополнение к адресу электронной почты администратора учетной записи, связанному с вашей учетной записью, можно отправлять уведомления о службах и рекомендации учетной записи на адрес электронной почты. Адрес электронной почты администратора учетной записи получает важные уведомления обо всех подписках в учетной записи, включая обновления, связанные с выставлением счетов.

### <a name="change-the-account-administrators-email-address-in-your-profile"></a>Изменение адреса электронной почты администратора учетной записи в профиле

Администратор учетной записи — это человек, который настраивает учетную запись Azure и получает электронные уведомлений обо всех подписках в учетной записи. Рекомендуется указать контактный адрес электронной почты для регулярной проверки администратором учетной записи.

### <a name="update-your-sign-in-email-address"></a>Обновление адреса электронной почты для входа

Обновление адреса электронной почты профиля не приводит к обновлению адреса электронной почты для входа. Чтобы изменить адрес электронной почты для входа, необходимо перенести владение учетной записью.

### <a name="update-the-billing-address-for-your-credit-card"></a>Обновление адреса выставления счетов для кредитной карты

Сведения об обновлении данных о счетах см. в статье [Изменение кредитной карты, используемой для оплаты подписки Azure](billing-how-to-change-credit-card.md).

### <a name="update-your-countryregion"></a>Обновите страну или регион

Из-за технических ограничений мы не можем изменить страну или регион для существующей учетной записи. Но вы можете создать учетную запись в нужной стране или регионе, а затем обратиться в службу поддержки Azure, чтобы перенести службы в эту учетную запись.

## <a name="service-and-marketing-emails"></a>Услуги и маркетинговые сообщения электронной почты

Появится запрос на [портал Azure](https://portal.azure.com) проверки или обновления адреса электронной почты каждые 90 дней. Корпорация Майкрософт отправляет сообщения электронной почты на ваш почтовый адрес профиля с помощью сведений об учетной записи Azure:

- Уведомления службы
- Оповещения безопасности
- Выставление счетов
- Поддержка
- Маркетинговые коммуникации
- Рекомендации на основе использования Azure

Введите адрес электронной почты, по которому вы хотите получить сообщения о вашей учетной записи. Вводя адрес электронной почты, вы можете принять участие в программе для получения сообщений от корпорации Майкрософт.

![Пример запроса на обновление контактных данных](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>Изменение адреса электронной почты контакта

Чтобы изменить адрес электронной почты контакта, используйте один из следующих методов.

* Войдите в [Центр учетных записей Azure](https://account.azure.com/subscriptions). Выберите **изменить сведения** , а затем обновите адрес электронной почты профиля. При обновлении адреса электронной почты профиля ваш адрес электронной почты для входа не обновляется.

* Перейдите в область [контактные данные](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) в портал Azure и введите адрес электронной почты. Предоставленный адрес получает услуги и маркетинговые коммуникации. Адрес электронной почты в профиле Центр управления учетной записью Azure обновляется для автоматического сопоставления. При обновлении адреса электронной почты профиля ваш адрес электронной почты для входа не обновляется.

* В [портал Azure](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)выберите значок с вашими инициалами или изображением. Затем выберите контекстное меню (.. **.** ). Затем в меню выберите **Мои контактные данные** и введите адрес электронной почты. Предоставленный адрес получает услуги и маркетинговые коммуникации. Адрес электронной почты в профиле Центр управления учетной записью Azure обновляется для автоматического сопоставления. При обновлении адреса электронной почты профиля ваш адрес электронной почты для входа не обновляется.

![Пример обновления адреса электронной почты в Azure](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>Отказаться от маркетинговых сообщений

Чтобы отказаться от получения маркетинговых сообщений:

1. Перейдите в [форму запроса](https://account.microsoft.com/profile/permissions-link-request) , чтобы отправить запрос с использованием адреса электронной почты профиля. Вы получите ссылку по электронной почте, чтобы обновить свои предпочтения.
2. Щелкните ссылку, чтобы открыть страницу **Управление** разрешениями на подключение. На этой странице отображаются типы маркетинговых коммуникаций, в которые входит адрес электронной почты. Очистите все разделы, которые нужно отказаться от, а затем нажмите кнопку **сохранить**.

![Пример страницы для управления разрешениями на взаимодействие](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

Если вы отказываетесь от маркетинговых коммуникаций, вы по-прежнему получаете уведомления службы на основе вашей учетной записи.

## <a name="change-the-subscription-name"></a>Изменение имени подписки

1. Войдите в [портал Azure](https://portal.azure.com), выберите пункт **Подписка** в левой области, а затем выберите подписку, которую требуется переименовать.
1. Выберите **Обзор**, а затем выберите **Переименовать** на панели команд.
    ![Пример переименования подписки Azure](./media/billing-how-to-change-azure-account-profile/rename-sub.png)
1. После изменения имени нажмите кнопку **сохранить**.
