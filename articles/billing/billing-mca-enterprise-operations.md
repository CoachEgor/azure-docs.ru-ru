# <a name="complete-enterprise-agreement-tasks-in-your-billing-account-for-a-microsoft-customer-agreement"></a>Выполните задачи соглашения Enterprise в учетной записи выставления счетов для клиента соглашение Microsoft

Если ваша организация подписала соглашение клиента, чтобы продлить соглашение Enterprise, учетную запись выставления счетов создается для соглашения. Выставление счетов в учетной записи новый организованы иначе, чем соглашение Enterprise. В этой статье описывается, как можно использовать новую учетную запись выставления счетов для выполнения задач, которые вы выполняли в соглашение Enterprise.

## <a name="how-billing-is-organized-in-the-new-account"></a>Организация выставление счетов в новой учетной записи

Следующая схема описывает организацию выставление счетов в новой учетной записи выставления счетов.

![Снимок экрана: ea mca-post перехода иерархию](./media/billing-mca-setup-account/mca-post-transition-hierarchy.png)

| Соглашение Enterprise   | Клиентское соглашение Майкрософт    |
|------------------------|--------------------------------------------------------|
| Регистрация            | Использовать профиль выставления счетов для управления выставлением счетов для вашей организации, аналогичную соглашение Enterprise. Администраторы предприятия становятся владельцы профиль выставления счетов. Дополнительные сведения о профилях выставления счетов, см. в разделе [особенностей выставления счетов](billing-mca-overview.md#understand-billing-profiles).
| Department            | Раздел счета позволяет упорядочить свои затраты на аналогичную отделов в соглашение Enterprise. Отдел становится разделы счетов, а администраторам отдела становятся владельцами разделов соответствующие счета. Дополнительные сведения о разделах счета, см. в разделе [разделы счетов понимание](billing-mca-overview.md#understand-invoice-sections). |
| Учетная запись.               | Учетные записи, которые были созданы в соответствии с соглашением Enterprise не поддерживаются в новой учетной записи выставления счетов. Учетной записи подписки принадлежат разделе соответствующих счета для своего факультета. Владельцы учетных записей можно создавать и управлять подписками своих разделов счета. |

## <a name="changes-for-enterprise-administrators-in-the-new-billing-account"></a>Изменения для администраторов предприятия в новой учетной записи выставления счетов

Следующие изменения применяются к администраторам предприятия в рамках соглашения Enterprise есть обновление с соглашением Microsoft Customer.

- Для вашей учетной записи создается профиль выставления счетов. Профиль выставления счетов будет использовать для управления выставлением счетов для вашей организации, таких как соглашение Enterprise. Дополнительные сведения о профилях выставления счетов, [особенностей выставления счетов](billing-mca-overview.md#understand-billing-profiles).
- Раздел счета создается для каждого из отделов в соглашение Enterprise. Разделы счетов будет использоваться для управления факультетов. Вы можете создать новые разделы счетов для настройки дополнительных отделов. Дополнительные сведения о разделах счета, см. в разделе [понять разделы счетов](billing-mca-overview.md#understand-invoice-sections).
- Используйте роль "Автор" подписки Azure в разделы счетов для передачи другим пользователям разрешение на создание подписки Azure, как и учетные записи, которые были созданы при регистрации соглашения Enterprise.
- Вы будете использовать [портала Azure](https://portal.azure.com) для управления выставлением счетов для вашей организации, а не на портале Azure EA.

На новой учетной записи выставления счетов, получают следующие роли:

**Владелец профиль выставления счетов** -вам назначена роль владельца выставления счетов профиля в профиль выставления счетов, который был создан при подписании соглашения. Роль позволяет управлять выставление счетов для вашей организации. Может просматривать расходы и счета-фактуры, упорядочить затраты в счете, управление методами оплаты и управления доступом к выставлении счетов для вашей организации.

**Владелец раздел счета** -вам назначена роль владельца раздел счета на все разделы счетов, которые создаются для отделов в соглашение Enterprise. Роль позволяет управлять тем, кто может создавать подписки Azure и приобрести другие продукты.

### <a name="view-charges-and-credits-balance-for-your-organization"></a>Просмотреть расходы и деньги на счете баланса для вашей организации

Использовать профиль выставления счетов для отслеживания расходов и сальдо, кредиты Azure для вашей организации, аналогичную соглашение Enterprise.

Сведения о просмотре пополните баланс для профиля выставления счетов, см. в разделе [отслеживать счета на счете в Azure для выставления счетов профиля](billing-mca-check-azure-credits-balance.md).

Узнайте, как просмотр затрат для профиля выставления счетов, см. в разделе [основная информация о расходах в соглашении клиента Microsoft счет](billing-mca-understand-your-bill.md).

### <a name="view-charges-for-a-department"></a>Просмотр расходов отдела

Раздел счета создается для каждого отдела, которые были в соглашение Enterprise. Плата за счет раздела можно просмотреть на портале Azure. Дополнительные сведения см. в разделе [просматривать транзакции по разделы счетов](billing-mca-understand-your-bill.md#view-transactions-by-invoice-sections).

### <a name="view-charges-for-an-account"></a>Просмотр расходов для учетной записи

Учетные записи, которые были созданы в соглашение Enterprise не поддерживаются в новой учетной записи выставления счетов. Учетной записи подписки принадлежат разделе соответствующих счета для своего факультета. Владельцы учетных записей можно создавать и управлять подписками своих разделов счета.

Чтобы просмотреть общие затраты для подписки, принадлежащие учетной записи, необходимо задать центр затрат для каждой подписки. После этого можно использовать Azure CSV-файл об использовании и затратах для фильтрации подписок по месту возникновения затрат.

### <a name="download-usage-and-charges-csv-price-sheet-and-tax-documents"></a>Скачивание csv об использовании и затратах, прайс-лист и налоговых документов

Ежемесячный счет создается для каждого профиля выставления счетов в учетной записи выставления счетов. Для каждой накладной можно загрузить Azure об использовании и затратах CSV-файл, прайс-лист и налогового документа (если применимо). Вы также можете скачать Azure об использовании и затратах CSV-файл для оплаты за текущий месяц.

Чтобы узнать, как загрузить Azure об использовании и затратах CSV-файл, см. в разделе [скачать сведения об использовании для соглашение с клиентом Microsoft](billing-download-azure-daily-usage.md#download-usage-for-your-microsoft-customer-agreement).

Чтобы узнать, как загрузить прайс-лист, см. в разделе [скачать цены на соглашение с клиентом Microsoft](billing-ea-pricing.md#view-and-download-pricing-for-your-microsoft-customer-agreement).

Чтобы узнать, как загрузить налоговых документов, см. в разделе [просмотреть налоговых документов для соглашение с клиентом Microsoft](billing-mca-download-tax-document.md#view-and-download-tax-documents).

### <a name="add-an-additional-enterprise-administrator"></a>Добавление дополнительных корпоративных администратора

Предоставляют пользователям доступ к профиль выставления счетов, чтобы разрешить ему представление и управление выставлением счетов для вашей организации. Можно использовать **управление доступом (IAM)** страница на портале Azure, чтобы предоставить доступ.  Дополнительные сведения о выставлении счетов ролями профилей, см. в разделе [выставления счетов профиля роли и задачи](billing-understand-mca-roles.md#billing-profile-roles-and-tasks).

Чтобы узнать, как для предоставления, доступ к профиль выставления счетов, см. в разделе [управление ролями выставления счетов на портале Azure](billing-understand-mca-roles.md#manage-billing-roles-in-the-azure-portal).

### <a name="create-a-new-department"></a>Создать новое подразделение

Создайте раздел счета, чтобы упорядочить затраты, в зависимости от ваших потребностей, например отделов в соглашение Enterprise. На портале Azure можно создать новый раздел счета. Дополнительные сведения см. в разделе [создания разделов на вашем счете, чтобы упорядочить свои затраты на](billing-mca-section-invoice.md).

### <a name="create-a-new-account"></a>Создание учетной записи

Назначьте пользователей роли создателя Azure подписки на разделы счетов, чтобы предоставить им разрешения на создание подписки Azure, как и учетные записи, которые создаются при регистрации соглашения Enterprise. Дополнительные сведения см. в разделе [предоставить другим пользователям разрешение на создание подписок Azure](billing-mca-create-subscription.md#give-others-permission-to-create-azure-subscriptions).

## <a name="changes-for-department-administrators-in-the-new-billing-account"></a>Изменения для администраторам отдела в новой учетной записи выставления счетов

Следующие изменения применяются к администраторам отдела на соглашение Enterprise Agreement, есть обновление с соглашением Microsoft Customer.

- Раздел счета создается для каждого из отделов в соглашение Enterprise. Вы используете разделы счета для управления вашей отделы. Дополнительные сведения о разделах счета, см. в разделе [понять разделы счетов](billing-mca-overview.md#understand-invoice-sections).
- Используйте роль "Автор" подписки Azure в разделе счета для передачи другим пользователям разрешение на создание подписки Azure, как и учетные записи, которые создаются при регистрации соглашения Enterprise.
- Вы используете портал Azure для управления выставлением счетов для вашей организации, а не на портале Azure EA.

На новой учетной записи выставления счетов, получают следующие роли:

**Владелец раздел счета** -вам назначена роль владельца раздела счета в разделе счета, создаваемый для отделы, имеется в рамках соглашения Enterprise. Роль позволяет Просмотр и отслеживание расходов и управления, можно создавать подписки Azure и приобрести другие продукты для раздела счета.

### <a name="view-charges-for-your-department"></a>Просмотр расходов вашего отдела

Вы можете просмотреть взимает плату за счет раздел, который создается для отдела, на портале Azure [управление затратами + выставление счетов страницы](https://portal.azure.com/#blade/Microsoft_Azure_Billing/ModernBillingMenuBlade/Overview).

### <a name="add-an-additional-department-administrator"></a>Добавление дополнительных отдела администратора

Раздел счета создается для каждого отдела, которые были в соглашение Enterprise. Можно использовать **доступа (IAM)** страница на портале Azure, чтобы предоставить общий доступ для просмотра и управления в разделе счета. Дополнительные сведения о ролях разделе счета, см. в разделе [выставляется счет разделе ролей и задач](billing-understand-mca-roles.md#invoice-section-roles-and-tasks).

Чтобы узнать, как предоставить, получать доступ к в раздел счета, см. в разделе [управление ролями выставления счетов на портале Azure](billing-understand-mca-roles.md#manage-billing-roles-in-the-azure-portal).

### <a name="create-a-new-account-in-your-department"></a>Создать учетную запись вашего отдела

Назначьте пользователей роли создателя подписки Azure в разделе "счета", созданный для вашего отдела. Дополнительные сведения см. в разделе [предоставить другим пользователям разрешение на создание подписок Azure](billing-mca-create-subscription.md#give-others-permission-to-create-azure-subscriptions).

### <a name="view-charges-for-accounts-in-your-departments"></a>Просмотр расходов для учетных записей в факультетов

Учетные записи, которые были созданы в соглашение Enterprise не поддерживаются в новой учетной записи выставления счетов. Учетной записи подписки принадлежат разделе соответствующих счета для своего факультета. Владельцы учетных записей можно создавать и управлять подписками своих разделов счета.

Чтобы просмотреть общие затраты для подписки, принадлежащие учетной записи в отделе, необходимо задать центр затрат для каждой подписки. Об использовании и затратах службы файлов Azure можно использовать для фильтрации подписок по месту возникновения затрат.

## <a name="changes-for-account-owners-in-the-new-billing-account"></a>Изменения для владельца учетной записи в новую учетную запись выставления счетов

Владельцы учетных записей на соглашение Enterprise получить разрешение на создание подписок Azure в новую учетную запись выставления счетов. Существующие подписки Azure принадлежат раздел счета, который создается для своего отдела. Если ваша учетная запись не принадлежит отдел, ваши подписки принадлежат раздел счета, с именем раздела счета по умолчанию.

Чтобы создать дополнительные подписки Azure, пользователь получает следующие роли на новую учетную запись выставления счетов.

**Создатель подписки Azure** -вам назначена роль создателя подписки azure в разделе счета, который создается для своего отдела в рамках соглашения Enterprise. Если ваша учетная запись не принадлежит отдел, вы получаете роль "Автор" Azure подписки на раздел с именем раздела счета по умолчанию. Роль предоставляет возможность создавать подписки Azure для в разделе счета.

### <a name="create-an-azure-subscription"></a>Создание подписки Azure.

Можно создать подписки Azure для вашего счета раздела на портале Azure. Дополнительные сведения см. в разделе [создайте дополнительную подписку Azure для клиентов соглашения Microsoft](billing-mca-create-subscription.md)

### <a name="view-charges-for-your-account"></a>Просмотр расходов для вашей учетной записи

Чтобы просмотреть расходы для подписки, принадлежащие учетной записи, перейдите к [страницу "подписки"](https://portal.azure.com/#blade/Microsoft_Azure_Billing/SubscriptionsBlade) на портале Azure. На странице подписки отображаются расходы для вашей подписки.

### <a name="view-charges-for-a-subscription"></a>Просмотр расходов для подписки

Либо можно просмотреть расходы для подписки на [страницу "подписки"](https://portal.azure.com/#blade/Microsoft_Azure_Billing/SubscriptionsBlade) или анализ затрат Azure. Дополнительные сведения по анализу затрат Azure см. в разделе [исследовать и анализировать затраты с помощью анализа затрат](../cost-management/quick-acm-cost-analysis.md).
