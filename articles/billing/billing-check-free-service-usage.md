# <a name="check-usage-of-free-services-included-with-your-azure-free-account"></a>Проверка использования бесплатных служб, включаемых в бесплатную учетную запись Azure

Плата за использование служб, включенных в вашу бесплатную учетную запись Azure, не взимается, если не превышаются ограничения для этих служб. Чтобы не превышать ограничения, вы можете использовать портал Azure для отслеживания использования бесплатных служб.

## <a name="check-usage-in-the-azure-portal"></a>Отслеживание использования на портале Azure

1.  Войдите на [портале Azure](https://portal.azure.com).

2.  Выполните поиск по запросу **Подписки**.

    ![Снимок экрана: поиск подписок на портале](./media/billing-check-usage-of-free-services/billing-search-subscriptions.png)

3.  Выберите подписку, созданную при регистрации вашей бесплатной учетной записи Azure.

4.  Прокрутите вниз, чтобы найти таблицу использования бесплатной службы.

    ![Снимок экрана со сведениями о потреблении бесплатных служб](./media/billing-check-usage-of-free-services/subscription-usage-free-services.png)

    Ниже перечислены столбцы этого списка:

* **Единица измерения.** Определяет единицу измерения для использованной службы.
* **Использование и ограничение.** Текущее месячное потребление и ограничение для единицы измерения.
* **Состояние.** Состояние использования службы. На основании данного использования можно иметь одно из этих состояний.
  * **Не используется.** Единица измерения не используется или потребление единицы не достигло системы выставления счетов.
  * **Превышено \<дата>.** Превышено ограничение для единицы измерения (\<дата>).
  * **Превышение маловероятно.** Превышение ограничения для единицы измерения маловероятно.
  * **Превышает \<дата>.** Превышение ограничения для единицы измерения \<дата> маловероятно.

> [!IMPORTANT]
>
> Бесплатные службы доступны только для подписки, которая была создана при регистрации вашей бесплатной учетной записи Azure. Если вы не видите таблицу бесплатных служб на странице обзора подписки, значит они недоступны для подписки.
