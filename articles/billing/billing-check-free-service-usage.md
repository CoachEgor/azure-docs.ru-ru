# <a name="check-free-service-usage-included-with-your-azure-free-account"></a>Проверка использования бесплатной версии службы, включенные в бесплатной учетной записи Azure

Если не превышаются ограничения служб, плата не взимается для служб, включенных в бесплатной учетной записи Azure. Оставаться в пределах, можно использовать портал Azure и CSV-файл для контроля и отслеживания использования бесплатной версии службы.

## <a name="check-usage-in-the-azure-portal"></a>Проверка использования на портале Azure

1.  Войдите на [портале Azure](https://portal.azure.com).

2.  В области навигации слева выберите **все службы**.

3.  Выберите **Подписки**.

4.  Выберите подписку, созданную при регистрации для получения бесплатной учетной записи.

    ![Снимок экрана, на котором показаны все подписки](./media/billing-check-usage-of-free-services/select-free-account-subscription.png)

5.  В разделе Обзор показывает важные сведения о подписке. Например идентификатор подписки, тип предложения и имя подписки. Можно также найти сведения по истечении срока действия кредита бесплатной учетной записи.

    ![Снимок экрана с важными сведениями о подписке](./media/billing-check-usage-of-free-services/subscription-essential-information.png)

6.  Прокрутите вниз, чтобы найти сведения о вашей текущих и прогнозируемых затратах. Стоимость включает использование службы, не входит в состав бесплатной учетной записи и использование ресурсов сверх ограничения бесплатных служб.

    ![Снимок экрана со сведениями о стоимости подписки](./media/billing-check-usage-of-free-services/subscription-cost-information.png)

7.  Последний сегмент в обзоре есть таблица, показывающая использование бесплатной версии службы.

    ![Снимок экрана со сведениями о потреблении бесплатных служб](./media/billing-check-usage-of-free-services/subscription-usage-free-services.png)

    Ниже перечислены столбцы этого списка:

* **Имя единицы измерения.** Указывает единицу измерения для использованного средства измерения. Дополнительные сведения о службе сопоставления учета см. в статье [Сопоставление единиц измерения бесплатных служб](billing-understand-free-service-meter-mapping.md).
* **Использование и ограничение.** Текущее месячное потребление и ограничение для единицы измерения. Кроме того, эти сведения можно найти в строке состояния.
* **Состояние.** Состояние использования единицы измерения. На основании данной модели использования, вы может иметь одно из следующих состояний:
  * **Не используется.** Единица измерения не используется или потребление единицы не достигло системы выставления счетов.
  * **Превышено \<дата>.** Превышено ограничение для единицы измерения (\<дата>).
  * **Превышение маловероятно.** Превышение ограничения для единицы измерения маловероятно.
  * **Превышает \<дата>.** Превышение ограничения для единицы измерения \<дата> маловероятно.

## <a name="check-usage-with-the-usage-file"></a>Проверка использования в файле использования

Файл использования предоставляет подробные сведения для своей подписки Azure. Ежедневные и ежемесячные CSV-файл можно загрузить в центре учетных записей Azure. Чтобы узнать, как скачать файл использования и получить необходимый доступ, см. статью [Скачивание или просмотр счета на оплату и данных о ежедневном использовании в Azure](billing-download-azure-invoice-daily-usage-date.md). Дополнительные сведения о столбцах в файле использования см. в статье [Значение терминов в файле с подробными данными об использовании Microsoft Azure](billing-understand-your-usage.md).

Файл использования содержит сведения об использовании бесплатных и платных служб. Единицам измерений бесплатных служб будет добавлено **Бесплатно** в конце имени. Найти бесплатные единицы измерения, открыть файл в excel и фильтр **столбец Категория счетчика** для ячеек, содержащих текст **- Free** (используйте текстовые фильтры &rarr; фильтра Contains).


![Снимок экрана со сведениями о потреблении бесплатных служб](./media/billing-check-usage-of-free-services/free-services-usage-csv.png)
