# <a name="how-to-pay-your-bill-for-microsoft-azure"></a>Способы оплаты счета за Microsoft Azure

Эта статья относится к клиенты с соглашением Microsoft клиента, которые зарегистрировались в Azure через веб-сайте Azure. [Проверьте доступ к соглашению клиента Microsoft](#check-access-to-a-microsoft-customer-agreement).

Способ оплаты по умолчанию профиля выставления счетов можно кредитной карты или проверку подключения передачи. Если у вас есть кредиты Azure, тех, автоматически применяются к счета каждый расчетный период. Чтобы получить право на оплату по проверка/банковский перевод, см. в разделе [способы оплаты по счету-фактуре](billing-how-to-pay-by-invoice.md).

Если проверка/банковский перевод способа оплаты по умолчанию, можно отправить платеж адрес, указанный в нижней части счета. Кроме того Оплата может производиться через Интернет с помощью кредитной карты на портале Azure.

Если метод оплаты по умолчанию — кредитная карта, автоматически продаются вашей кредитной карты каждый расчетный период после создания счета. Оплата может производиться также через Интернет на портале Azure, если вы хотите использовать другую кредитную карту, или при сбое автоматического плата.

## <a name="pay-now-in-the-azure-portal"></a>Платить сейчас на портале Azure

Будьте выставления счетов профиль владельца, участника или диспетчера счет для оплаты счета-фактуры на портале Azure.

1. Войдите на [портал Azure](https://portal.azure.com).
1. Выполните поиск по **Управление затратами + выставление счетов**.
1. В **Обзор** колонке щелкните *Сводка* вкладки.

    ![Снимок экрана: вкладка "Сводка"](./media/billing-how-to-pay-bill/billing-account-overview.png)

1. При наличии срока выполнения или просроченные счета, вы увидите Общая сумма оплаты.
1. Щелкните синюю **платить сейчас** кнопки.

    ![Снимок экрана, показывающий платить кнопку](./media/billing-how-to-pay-bill/paynow-button-overview.png)

1. В новой колонке щелкните **выберите способ оплаты** для выбора существующей кредитной карты или добавить новую.<!-- screenshot of select payment method link -->
1. Выбрав метод оплаты, щелкните **платить сейчас**.

Состояние счета будет обновлено на платную версию в колонке "счета".
