# <a name="understand-free-service-to-meter-mapping"></a>Сопоставление единиц измерения бесплатных служб

Использование каждой службы Azure включает показатели, которые использует система выставления счетов Azure, чтобы взимать плату с пользователей за использование служб. Чтобы лучше понять использование бесплатной версии службы, давайте взглянем на службе сопоставления учета для служб. Чтобы узнать, как создать бесплатные службы, ознакомьтесь со статьей о [создании бесплатных служб с помощью бесплатной учетной записи Azure](billing-create-free-services-included-free-account.md).
