# <a name="purchase-reservations-with-monthly-payments"></a>Покупка резервирований с ежемесячными платежами

До этого момента в резервированиях Azure требуется оплата на передний план. Теперь вы можете платить за резервирования с ежемесячными платежами. В отличие от предварительной покупки, когда вы оплачиваете полную сумму, вариант месячного платежа делит общую стоимость резервирования на каждый месяц термина. Общие затраты на резервирование до первого и ежемесячного резервирования одинаковы, и вы не оплачиваете дополнительные платы, если вы выберете ежемесячную оплату.

Сумма ежемесячного платежа может отличаться в зависимости от курса продаж на текущий месяц для вашей местной валюты.

Ежемесячные платежи доступны для:

- ВМ
- База данных SQL
- Хранилище данных SQL
- Cosmos DB
- Плата за метки службы приложений

Приобретите резервирования в [портал Azure](https://portal.azure.com/?Microsoft_Azure_Reservations_EnableMultiCart=true&amp;paymentPlan=true#blade/Microsoft_Azure_Reservations/CreateBlade).

![Пример покупки резервирования](./media/billing-monthly-payments-reservations/purchase-reservation.png)

При осуществлении покупки резервирования можно просмотреть график оплаты. Щелкните **Просмотреть полный график оплаты**.

![Пример, демонстрирующий расписание оплаты резервирования](./media/billing-monthly-payments-reservations/prepurchase-schedule.png)

Чтобы просмотреть расписание платежей после покупки, выберите резервирование, щелкните **идентификатор заказа на резервирование**, а затем перейдите на вкладку **оплаты** .

## <a name="view-payments-made"></a>Просмотр произведенных платежей

Вы можете просматривать платежи, выполненные с помощью API, данных об использовании и анализа затрат. Для резервирования по месячному тарифу значение частоты отображается как **повторяющееся** в API-интерфейсе оплаты за использование данных и резервирование. Для резервирования, выплачиваемого спереди, значение отображается как **OneTime**.

Анализ затрат отображает ежемесячные покупки в представлении по умолчанию. Примените фильтр **покупки** для **типа оплаты** и **повторяйте** для **периодичности** , чтобы увидеть все покупки. Чтобы просмотреть только резервирования, примените фильтр для **резервирования**.

![Пример, демонстрирующий стоимость приобретения резервирования в анализе затрат](./media/billing-monthly-payments-reservations/cost-analysis.png)

## <a name="switch-to-monthly-payments-at-renewal"></a>Переключиться на ежемесячные платежи при продлении

При продлении резервирования можно изменить частоту выставления счетов на месяц.

## <a name="exchange-and-refunds"></a>Обмен и возмещение

Как и в случае с другими резервированиями, вы можете платить и заставлять резервирования для обмена с ежемесячным выставлением счетов. В настоящее время вы можете отправить запрос в службу поддержки, чтобы начать обмен, или возвратить возмещение, приобретенное при ежемесячном выставлении счетов.

При обмене резервированием, которое оплачивается ежемесячно, стоимость общего времени существования новой покупки должна быть больше, чем у оставшихся платежей, отмененных для возвращенного резервирования. Нет других ограничений или плат для Exchange. Вы можете обмениваться зарезервированными данными, чтобы приобрести новое резервирование, которое оплачивается ежемесячно. Однако значение времени существования нового резервирования должно быть больше, чем пропорциональное значение резервируемого резервирования.

При отмене резервирования, которое оплачивается ежемесячно, корпорация Майкрософт может применить плату об отмене для будущих подтвержденных выплат, которые отменяются. Оставшиеся зафиксированные выплаты начисляются до предельной суммы возмещения $50 000 долларов США.

Дополнительные сведения об обмене и возврате денег см. в статье [самостоятельная служба Exchange и возмещение для резервирования Azure](billing-azure-reservations-self-service-exchange-and-refund.md).

## <a name="faq"></a>Часто задаваемые вопросы

В. Предлагает ли Azure "частичные резервирования" на передний план? "<br>
A. Нет. Поскольку затраты на резервирования, оплаченные заранее и ежемесячно, корпорация Майкрософт не поддерживает частичные платежи.

В. Доступны ли ежемесячные платежи для программы поставщика решений Microsoft Cloud (CSP)?<br>
A. Да, партнеры могут приобрести резервирование для своих клиентов CSP в портал Azure. Возможность приобретения резервирований с ежемесячным выставлением счетов недоступна в центре партнеров.

В. Я являюсь клиентом Azure для государственных организаций, могу ли я платить ежемесячно для покупок резервирования?<br>
A. В настоящее время нет.

В. Когда можно обмениваться данными с портал Azure, вместо того чтобы создавать запрос в службу поддержки?<br>
A. В настоящее время нет. Запросы на резервирование Exchange и возмещение с ежемесячными платежами обрабатываются службой поддержки Azure.
