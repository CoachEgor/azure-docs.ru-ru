# <a name="purchase-reservations-with-monthly-payments"></a>Приобретение резервирований с ежемесячными платежами

До этого момента в резервированиях Azure требовалась предварительная оплата. Теперь вы можете платить за резервирования ежемесячно. В отличие от предварительной покупки, когда вы оплачиваете полную сумму, при ежемесячном платеже общая стоимость резервирования делится на каждый месяц срока. Общие затраты на резервирование с предварительными или ежемесячными платежами одинаковы, и вы не будете переплачивать, если выберете ежемесячную оплату.

Сумма ежемесячного платежа может отличаться в зависимости от рыночного курса вашей местной валюты на текущий месяц.

Ежемесячные платежи возможны для таких ресурсов:

- Виртуальные машины
- База данных SQL
- Хранилище данных SQL
- Cosmos DB
- Гербовый сбор службы приложений Azure.

Приобретите резервирования на [портале Azure](https://portal.azure.com/?Microsoft_Azure_Reservations_EnableMultiCart=true&amp;paymentPlan=true#blade/Microsoft_Azure_Reservations/CreateBlade).

![Пример покупки резервирования](./media/billing-monthly-payments-reservations/purchase-reservation.png)

При покупке резервирования можно просмотреть график оплаты. Щелкните **Просмотр графика полной оплаты**.

![Пример, демонстрирующий график оплаты резервирования](./media/billing-monthly-payments-reservations/prepurchase-schedule.png)

Чтобы просмотреть график оплаты после покупки, выберите резервирование, щелкните **Идентификатор заказа на резервирование**, а затем перейдите на вкладку **Оплаты**.

## <a name="view-payments-made"></a>Просмотр произведенных платежей

Вы можете просматривать осуществленные платежи с помощью API, в данных об использовании и анализе затрат. Для резервирования по месячному тарифу значение частоты отображается как **Повторение** в API-интерфейсе расходов на резервирование и данных об использовании. Для резервирования с предварительными платежами значение отображается как **onetime** (Однократно).

В анализе стоимости отображаются данные о ежемесячных покупках в представлении по умолчанию. Примените фильтр **покупки** для **типа оплаты** и значение **Повторение** для **частоты**, чтобы увидеть все покупки. Чтобы просмотреть только резервирования, примените фильтр для **Резервирования**.

![Пример, демонстрирующий стоимость приобретения резервирования в анализе стоимости](./media/billing-monthly-payments-reservations/cost-analysis.png)

## <a name="switch-to-monthly-payments-at-renewal"></a>Переключение на ежемесячные платежи при продлении

При продлении резервирования можно изменить периодичность выставления счетов на ежемесячную.

## <a name="exchange-and-refunds"></a>Обмен и возврат денег

Как и в случае с другими резервированиями, вы можете обменять резервирования или возместить средства, потраченные на них, с ежемесячным выставлением счетов. В настоящее время вы можете отправить запрос в службу поддержки, чтобы начать обмен или возвратить деньги за резервирование с ежемесячным выставлением счетов.

При обмене резервирования, которое оплачивается ежемесячно, общая стоимость новой покупки должна быть больше, чем оставшиеся платежи, отмененные для возвращенного резервирования. Других ограничений или платежей за обмен нет. Вы можете обменять резервирование с предварительным платежом, чтобы приобрести новое резервирование, которое оплачивается ежемесячно. Однако общая стоимость нового резервирования должна быть больше, чем пропорциональное значение стоимости возвращенного резервирования.

При отмене резервирования, которое оплачивается ежемесячно, корпорация Майкрософт может потребовать плату за отмену будущих зафиксированных выплат. Оставшиеся зафиксированные выплаты не могут превышать предельную сумму возмещения 50 000 долларов США.

Дополнительные сведения об обмене и возврате денег см. в статье [Самостоятельное выполнение обмена и возмещения средств для Azure Reserved Virtual Machine Instances](billing-azure-reservations-self-service-exchange-and-refund.md).

## <a name="faq"></a>Часто задаваемые вопросы

В. Предлагает ли Azure резервирования с частичным предварительным платежом?<br>
О. № Так как затраты на резервирования с предварительными или ежемесячными платежами одинаковы, корпорация Майкрософт не принимает частичные предварительные платежи.

В. Возможны ли ежемесячные платежи для программы поставщиков облачных решений (Майкрософт) (CSP)?<br>
О. Да, партнеры могут приобрести резервирование для своих клиентов CSP на портале Azure. Возможность приобретения резервирований с ежемесячным выставлением счетов недоступна в Центре партнеров.

В. Я являюсь клиентом Azure для государственных организаций. Могу ли я платить ежемесячно за покупку резервирования?<br>
О. В настоящее время нет.

В. Когда можно обменять резервирование или вернуть за него деньги на портале Azure, чтобы не создавать запрос в службу поддержки?<br>
О. В настоящее время нет. Запросы на обмен и возврат денег за резервирование с ежемесячными платежами обрабатываются службой поддержки Azure.
