# <a name="troubleshoot-azure-payment-issues"></a>Устранение проблем с оплатой в Azure

При попытке обновить учетную запись платежных данных в портал Microsoft Azure или в центре управления учетной записью Azure может возникнуть ошибка. Прежде чем устранять проблему, учитывайте следующие рекомендации.

- Убедитесь, что сведения, указанные для профиля учетной записи Azure (включая адрес электронной почты, почтовый адрес и номер телефона), указаны правильно.
- Проверьте правильность информации о кредитной карте.
- Убедитесь, что у вас нет учетной записи Майкрософт с такими же сведениями.

## <a name="issues"></a>Вопросы

Чтобы устранить ошибки, выберите проблему, которая возникла при попытке зарегистрироваться в Azure.

### <a name="unable-to-remove-a-credit-card-from-a-saved-billing-payment-method"></a>Не удалось удалить кредитную карту из сохраненного метода оплаты выставления счетов

По проекту нельзя удалить кредитную карту из активной подписки.

Если необходимо удалить существующую карточку, необходимо добавить новую карточку в подписку, чтобы старое средство оплаты можно было успешно удалить, или отменить подписку. Это приведет к окончательному удалению подписки и удалению карточки.

### <a name="unable-to-delete-an-old-payment-method-after-adding-a-new-payment-method"></a>Не удается удалить старый метод оплаты после добавления нового метода оплаты

Новое платежное средство может не быть связано с подпиской. Сведения о связывании платежного средства с подпиской см. в статье [Добавление, обновление или удаление кредитной или дебетовой карты для Azure](billing-how-to-change-credit-card.md).

Сведения об устранении неполадок, связанных с отклоненной картой, см. в разделе [Устранение неполадок с отклоненной картой при регистрации в Azure](billing-troubleshoot-declined-card.md).

### <a name="unable-to-delete-a-payment-method-because-of-cannot-delete-payment-method-error"></a>Не удалось удалить метод оплаты из-за *невозможности удалить метод оплаты* .

Это происходит из-за невыполненного баланса. Очистите все необработанные балансы перед удалением метода оплаты.

### <a name="unable-to-see-subscriptions-under-my-account-to-update-the-payment-method"></a>Не удается просмотреть подписки в моей учетной записи для обновления метода оплаты

Возможно, вы используете идентификатор электронной почты, отличающийся от того, который используется для подписок.

Сведения об устранении этой проблемы см. в статье [отсутствие подписок при входе в систему для портал Azure или центра учетных записей Azure](billing-no-subscriptions-found.md).

### <a name="unable-to-make-payment-for-a-subscription"></a>Не удается выполнить оплату для подписки

Если появится сообщение об ошибке: *Платеж просрочен. Возникла проблема с методом* оплаты, и *мы не можем сохранить информацию. Закройте браузер и повторите попытку.* , на карточке имеется ожидающая оплата, так как карта была отклонена финансовым учреждением.

Убедитесь, что кредитная карта имеет достаточный баланс для выполнения платежа. Если это не так, используйте другую карту для выполнения оплаты или обратитесь к финансовому учреждению, чтобы решить проблему.

Обратитесь в свой банк за следующими вопросами:

- Международные транзакции не включены.
- У карточки есть кредитный лимит, и баланс должен быть сопоставлен.
- На карточке включена повторяющаяся оплата.

### <a name="unable-to-change-payment-method-because-of-browser-issues-browser-does-not-respond-does-not-load-and-so-on"></a>Не удалось изменить метод оплаты из-за проблем с браузером (браузер не отвечает, не загружается и т. д.)

Выйдите из всех активных сеансов Azure, а затем выполните действия, описанные в [статье Обзор InPrivate в Microsoft ребра](https://support.microsoft.com/help/4026200/microsoft-edge-browse-inprivate) , чтобы запустить сеанс InPrivate в Microsoft ребре или Internet Explorer.

В частном сеансе выполните действия, описанные в статье [Изменение кредитной карты](billing-how-to-change-credit-card.md) для обновления или изменения данных кредитной карты.

Также можно попробовать выполнить следующие действия.

- Обновление браузера
- Использовать другой браузер
- Удаление кэшированных файлов cookie

### <a name="my-subscription-is-still-disabled-after-updating-the-payment-method"></a>После обновления метода оплаты Моя подписка по-прежнему будет отключена.

Эта проблема возникает из-за непревзойденного баланса. Очистите все необработанные балансы перед удалением метода оплаты.

### <a name="unable-to-change-payment-method-because-of-an-xml-error-response-page"></a>Не удалось изменить метод оплаты из-за страницы ответа на ошибку XML

Это сообщение появляется, если вы используете [портал Azure](https://portal.azure.com/) для добавления новой кредитной карты.

Чтобы добавить сведения о карточке, войдите на портал учетных записей Azure, используя адрес электронной почты администратора учетной записи.
