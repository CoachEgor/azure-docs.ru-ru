# <a name="view-the-tax-documents-for-your-microsoft-customer-agreement"></a>Просмотр налоговых документов по Клиентскому соглашению Майкрософт

В этой статье рассматривается учетная запись выставления счетов для Клиентского соглашения Майкрософт. [Проверьте наличие доступа к Клиентскому соглашению Майкрософт](#check-access-to-a-microsoft-customer-agreement).

## <a name="view-and-download-tax-documents"></a>Просмотр и скачивание налоговых документов

Для просмотра и скачивания налоговых документов необходимо иметь роль владельца, участника, читателя или менеджера по счетам в профиле выставления счетов. Дополнительные сведения о ролях выставления счетов для Клиентского соглашения Майкрософт см. в разделе [Billing profile roles and tasks](billing-understand-mca-roles.md#billing-profile-roles-and-tasks) (Роли и задачи профиля выставления счетов).


1. Войдите на [портал Azure](https://portal.azure.com).
1. Выполните поиск по фразе *Управление затратами + выставление счетов*.
1. Выберите профиль выставления счетов.
1. Выберите **Счета**.
1. В сетке счетов найдите строку счета, соответствующую налоговому документу, который требуется скачать.
1. Нажмите кнопку с многоточием (`...`) в конце строки.
7. Выберите **Налоговый документ**. В зависимости от страны или региона профиля выставления счетов может отображаться более одного налогового документа для каждого счета.
