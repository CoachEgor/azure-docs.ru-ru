# <a name="create-an-additional-azure-subscription-for-microsoft-customer-agreement"></a>Создайте дополнительную подписку Azure для клиентов соглашения Microsoft

Создание дополнительных подписок для учетной записи выставления счетов для настройки отдельных сред для разработки и тестирования, безопасности или для изоляции данных для обеспечения соответствия.

Эта статья относится к учетной записи выставления счетов для клиента соглашение Microsoft. [Проверьте наличие доступа к соглашению клиента Microsoft](#check-access-to-a-microsoft-customer-agreement). Если вы хотите создать подписки для других выставления счетов, см. в разделе [создайте дополнительную подписку на портале Azure](billing-create-subscription.md).

Для создания подписки, необходимо быть **счета владелец разделе**, **участник раздел счета**, или **создатель подписки Azure**. Дополнительные сведения см. в разделе [подписки выставления счетов ролей и задач](billing-understand-mca-roles.md#subscription-billing-roles-and-tasks). Чтобы предоставить другим пользователям разрешение на создание подписок Azure для учетной записи выставления счетов, см. в разделе [предоставить другим пользователям разрешение на создание подписок Azure](#give-others-permission-to-create-azure-subscriptions).

## <a name="create-a-subscription-in-the-azure-portal"></a>Создание подписки на портале Azure

1. Войдите на [портале Azure](https://portal.azure.com).

2. Выполните поиск по запросу **Подписки**.

   ![Снимок экрана, показывающий поиска для подписок на портале](./media/billing-mca-create-subscription/billing-search-cost-management-billing.png)

3. Выберите **Добавить**.

4. Если у вас есть доступ к нескольким учетным записям выставления счетов, выберите учетную запись выставления счетов для соглашение с клиентом Microsoft.

   ![Снимок экрана, показывающий «создание подписки»](./media/billing-mca-create-subscription/billing-mca-create-azure-subscription.png)

5. Выберите профиль выставления счетов. Расходы для вашей подписки будут отображены в профиль выставления счетов и будет производиться с использованием его способов оплаты. Если у вас есть доступ к только один профиль выставления счетов, выделение будет неактивна.

6. Выберите раздел счета. Расходы для вашей подписки будут отображены в профиль выставления счетов в этом разделе. Если у вас есть доступ только один раздел счета, выделение будет неактивна.

7. Выберите план для подписки. Выберите **план Microsoft Azure для DevTest**, если вы планируете использовать эту подписку для разработки или тестирования рабочие нагрузки еще используют **план Azure Майкрософт**. Если у вас есть доступ к только один план, выделение будет неактивна.

8. Введите имя подписки. Это имя помогает вам легко определить подписку на портале Azure.

9. Нажмите кнопку **Создать**.

## <a name="give-others-permission-to-create-azure-subscriptions"></a>Предоставить другим пользователям разрешение на создание подписок Azure

Добавьте пользователей в качестве creators подписки Azure в разделе "счета", чтобы предоставить им разрешения на создание подписок Azure.

1. Войдите на [портале Azure](https://portal.azure.com).

2. Выполните поиск по **Управление затратами + выставление счетов**.

   ![Снимок экрана, показывающий поиска для подписок на портале](./media/billing-mca-create-subscription/billing-search-cost-management-billing.png)

3. Перейдите к разделу счета. В зависимости от вашего доступа необходимо выбрать учетную запись выставления счетов или профиль выставления счетов. Выставления счетов учетной записи или профиля выберите **выставляется счет разделы** и затем раздел счета.

4. Выберите **управления доступом (IAM)** со стороны левого верхнего.

5. В верхней части страницы выберите **Добавить**.

6. Выберите **создатель подписки Azure** для роли.

   ![Снимок экрана с предоставлением пользователю роль "Автор" подписки Azure](./media/billing-mca-create-subscription/billing-mca-add-azure-subscription-creator.png)

7. Введите адрес электронной почты пользователя, которому вы хотите предоставить доступ.

8. Щелкните **Сохранить**.
