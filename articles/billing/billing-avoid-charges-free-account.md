# <a name="avoid-getting-charged-for-your-azure-free-account"></a>Как не платить за бесплатную учетную запись Azure

Бесплатная учетная запись Azure предоставляет деньги на счете в Azure на сумму 200 долларов США, которые можно потратить в течение первых 30 дней, и 12 месяцев использования бесплатных служб в ограниченном количестве. Дополнительные сведения см. на странице [создания бесплатной учетной записи Azure](https://azure.microsoft.com/free/). В зависимости от состояния денег на счете вы можете использовать их или получить счет за использование, на который не распространяются преимущества бесплатной учетной записи Azure (бесплатные службы).

## <a name="you-still-have-active-azure-free-account-credit"></a>У вас все еще есть деньги на счете в бесплатной учетной записи Azure 
Если у вас остались деньги на счете в Azure, срок использования которых не вышел, этот остаток используется, чтобы оплатить услуги, на которые не распространяются преимущества бесплатной учетной записи Azure.

## <a name="your-credit-ran-out-or-has-expired"></a>Деньги на счете в Azure закончились, или их срок действия истек
Если через 30 дней деньги на счете в Azure закончились или истек срок их действия, Azure отключит вашу подписку. Чтобы продолжить использование служб Azure, необходимо обновить подписку до подписки с оплатой по мере использования. Дополнительные сведения см. в статье [Обновление с бесплатной пробной версии или подписки Azure Microsoft Imagine до подписки с оплатой по мере использования](billing-upgrade-azure-subscription.md). После обновления ваша подписка по-прежнему имеет доступ к бесплатным службам в течение 12 месяцев. Плата взимается только за использование платных служб и ресурсов.

Рассмотрим некоторые из причин, из-за которых может взиматься плата в бесплатной учетной записи Azure.

### <a name="your-usage-exceeds-the-limits-of-free-services-included-with-your-azure-free-account"></a>Потребление превышает ограничения бесплатных служб, включенных в бесплатную учетную запись Azure

Вы получаете ограниченное количество бесплатных служб каждый месяц в бесплатной учетной записи Azure. Количество бесплатных служб истекает в конце месяца и не переносится на следующий месяц. Например, вы получаете 5 ГБ хранилища файлов каждый месяц. Если в месяц использовалось 2 ГБ, оставшиеся 3 ГБ не переносятся на следующий месяц. Чтобы избежать выставления счетов, потребляйте ресурсы в допустимых пределах. Дополнительные сведения об ограничениях на бесплатные службы см. в статье [Вопросы и ответы о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Чтобы проверить потребление бесплатных служб, ознакомьтесь со статьей [Проверка использования бесплатных служб, включаемых в бесплатную учетную запись Azure](billing-check-free-service-usage.md).

### <a name="youre-using-services-not-included-for-free-with-your-azure-free-account"></a>Используются службы, не входящие в бесплатную учетную запись Azure

Выставление счетов за использование служб, которые не включены в бесплатную учетную запись Azure, будет осуществляться согласно тарифам оплаты по мере использования. Дополнительные сведения о службах, входящих в бесплатную учетную запись, см. в статье [Вопросы и ответы о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Использование служб можно проверить на портале Azure или через файл потребления Azure. Дополнительные сведения см. в разделе [Регулярно просматривайте на портале детализацию затрат и график расходов](billing-getting-started.md#costs) и статье [Скачивание или просмотр счета на оплату и данных о ежедневном использовании в Azure](billing-download-azure-invoice-daily-usage-date.md). 

### <a name="youve-reached-the-end-of-your-12-months-free-services"></a>Истек годичный срок действия по использованию бесплатных служб

Срок использования бесплатных служб истекает в конце года. Узнать окончание срока действия бесплатных служб можно на портале Azure.

1. Войдите на [портале Azure](https://portal.azure.com).

2. Из области навигации слева выберите **Все службы**.

3.  Выберите **Подписки**.

4.  Выберите подписку, созданную при регистрации для получения бесплатной учетной записи.

5.  Прокрутите вниз, чтобы найти сетку бесплатных служб. Щелкните подсказку, расположенную в верхней левой области сетки.

![Снимок экрана, на котором показано расположение даты окончания срока действия возможностей бесплатной учетной записи](./media/billing-avoid-charges-free-account/freeaccount-benefits-expiration-date.png)


После окончания срока действия по использованию бесплатных служб Azure будет взиматься оплата по мере использования служб. Портал Azure можно использовать для удаления ресурсов неиспользуемых служб. Если вы не собираетесь использовать какие-либо службы Azure, можно [отказаться от подписки](billing-how-to-cancel-azure-subscription.md).
