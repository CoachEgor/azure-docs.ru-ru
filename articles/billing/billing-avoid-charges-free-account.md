# <a name="avoid-charges-with-your-azure-free-account"></a>Экономия благодаря использованию бесплатной учетной записи Azure

С [бесплатной учетной записью Azure](https://azure.microsoft.com/free/) вы получаете деньги на счет в Azure на сумму 200 долларов США, которые можно потратить в течение первых 30 дней, и 12 месяцев использования бесплатных служб в ограниченном количестве. См. сведения об ограничениях бесплатных служб в [вопросах и ответах о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). При условии, что срок действия кредита не истек и вы используете бесплатные службы только в пределах лимитов, плата не будет взиматься. 

Рассмотрим некоторые из причин, из-за которых может взиматься плата в бесплатной учетной записи Azure.

## <a name="your-credit-runs-out-or-is-expired"></a>Деньги на счете закончились, или срок действия кредита истек

Ваша подписка и службы будут отключены, когда кредит будет исчерпан или истечет срок действия кредита (30 дней). Чтобы продолжить использование служб Azure, необходимо обновить учетную запись. Дополнительные сведения см. в разделе [Обновление бесплатной учетной записи Azure](billing-upgrade-azure-subscription.md). После обновления вы все еще располагаете доступом к бесплатным службам в течение 12 месяцев, а плата взимается только за использование, превышающее количество бесплатных служб.


## <a name="usage-exceeds-the-limits-of-free-services"></a>Показатели использования превышают ограничения бесплатных служб

Вы получаете ограниченное количество бесплатных служб каждый месяц в бесплатной учетной записи Azure. Количество бесплатных служб истекает в конце месяца и не переносится на следующий месяц. Например, вы получаете 5 ГБ хранилища файлов каждый месяц. Если в месяц использовалось 2 ГБ, оставшиеся 3 ГБ не переносятся на следующий месяц. Чтобы избежать выставления счетов, потребляйте ресурсы в допустимых пределах. См. сведения об ограничениях бесплатных служб в [вопросах и ответах о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Чтобы проверить потребление бесплатных служб, ознакомьтесь со статьей [Проверка использования бесплатных служб, включаемых в бесплатную учетную запись Azure](billing-check-free-service-usage.md).

## <a name="you-used-some-services-that-arent-free"></a>Вы использовали некоторые платные службы

После обновления учетной записи выставление счетов за использование служб, не входящих в бесплатную учетную запись Azure, будет осуществляться в рамках модели оплаты по мере использования. Дополнительные сведения о службах, входящих в бесплатную учетную запись, см. в разделе [Вопросы и ответы о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Показатели использования можно проверить на портале Azure. Дополнительные сведения, см. в разделе [Отслеживание распределения затрат и темп работ](billing-getting-started.md#costs).

## <a name="you-reached-the-end-of-your-free-12-months"></a>Вы достигли конца бесплатного 12-месячного периода

Срок использования бесплатных служб истекает в конце года. Узнать окончание срока действия бесплатных служб можно на портале Azure.

1. Войдите на [портале Azure](https://portal.azure.com).

2. В области навигации слева выберите **Все службы**.

3.  Выберите **Подписки**.

4.  Выберите подписку, созданную при регистрации для получения бесплатной учетной записи.

5.  Прокрутите вниз, чтобы найти сетку бесплатных служб. Щелкните подсказку, расположенную в верхней левой области сетки.

![Снимок экрана, на котором показано расположение даты окончания срока действия возможностей бесплатной учетной записи](./media/billing-avoid-charges-free-account/freeaccount-benefits-expiration-date.png)


После окончания срока действия бесплатных служб оплата будет взиматься в рамках модели оплаты по мере использования служб. Портал Azure можно использовать для удаления ресурсов неиспользуемых служб. Если вы не собираетесь использовать какие-либо службы Azure, можно [отказаться от подписки](billing-how-to-cancel-azure-subscription.md).
