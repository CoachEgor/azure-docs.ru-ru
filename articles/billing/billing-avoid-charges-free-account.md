# <a name="avoid-charges-with-your-azure-free-account"></a>Избежать расходов с помощью бесплатной учетной записи Azure

Бесплатная учетная запись Azure предоставляет деньги на счете в Azure на сумму 200 долларов США, которые можно потратить в течение первых 30 дней, и 12 месяцев использования бесплатных служб в ограниченном количестве. Дополнительные сведения см. на странице [создания бесплатной учетной записи Azure](https://azure.microsoft.com/free/). В зависимости от состояния денег на счете вы можете использовать их или получить счет за использование, на который не распространяются преимущества бесплатной учетной записи Azure (бесплатные службы).

## <a name="azure-free-account-might-use-account-credit"></a>Бесплатная учетная запись Azure может использовать кредит учетной записи
Если у вас остались деньги на счете в Azure, срок использования которых не вышел, этот остаток используется, чтобы оплатить услуги, на которые не распространяются преимущества бесплатной учетной записи Azure.

## <a name="your-credit-runs-out-or-is-expired"></a>Деньги на счете закончились или истек срок действия
Если срок действия истекает в конце 30 дней деньги на счете закончится срок действия, Azure отключит вашу подписку. Чтобы продолжить использование служб Azure, необходимо обновить подписку до отдельной подписки с оплатой. Дополнительные сведения см. в разделе [обновление бесплатной пробной версии подписки Azure](billing-upgrade-azure-subscription.md). После обновления ваша подписка по-прежнему имеет доступ к бесплатным службам в течение 12 месяцев. Плата взимается только за использование платных служб и ресурсов.

Рассмотрим некоторые из причин, из-за которых может взиматься плата в бесплатной учетной записи Azure.

### <a name="usage-exceeds-the-limits-of-free-services"></a>Уровень использования превышает ограничения бесплатных служб

Вы получаете ограниченное количество бесплатных служб каждый месяц в бесплатной учетной записи Azure. Количество бесплатных служб истекает в конце месяца и не переносится на следующий месяц. Например, вы получаете 5 ГБ хранилища файлов каждый месяц. Если в месяц использовалось 2 ГБ, оставшиеся 3 ГБ не переносятся на следующий месяц. Чтобы избежать выставления счетов, потребляйте ресурсы в допустимых пределах. Дополнительные сведения о ограничения бесплатных служб, см. в разделе [бесплатной учетной записи Azure часто задаваемые вопросы о](https://azure.microsoft.com/free/free-account-faq/). Чтобы проверить потребление бесплатных служб, ознакомьтесь со статьей [Проверка использования бесплатных служб, включаемых в бесплатную учетную запись Azure](billing-check-free-service-usage.md).

### <a name="some-services-are-not-free"></a>Некоторые службы не предоставляются бесплатно

Плата взимается оплата за использование служб, которые не включены в бесплатной учетной записи Azure. Дополнительные сведения о включенных в бесплатную учетную запись службы, см. в разделе [бесплатной учетной записи Azure часто задаваемые вопросы о](https://azure.microsoft.com/free/free-account-faq/). Вы можете проверить использование служб на портале Azure или в файле использования Azure. Дополнительные сведения см. в разделе [Регулярно просматривайте на портале детализацию затрат и график расходов](billing-getting-started.md#costs) и статье [Скачивание или просмотр счета на оплату и данных о ежедневном использовании в Azure](billing-download-azure-invoice-daily-usage-date.md).

### <a name="you-reached-the-end-of-your-free-12-months"></a>Достигнут конец вашей бесплатной 12 месяцев

Срок использования бесплатных служб истекает в конце года. Узнать окончание срока действия бесплатных служб можно на портале Azure.

1. Войдите на [портале Azure](https://portal.azure.com).

2. В области навигации слева выберите **все службы**.

3.  Выберите **Подписки**.

4.  Выберите подписку, созданную при регистрации для получения бесплатной учетной записи.

5.  Прокрутите вниз, чтобы найти сетку бесплатных служб. Щелкните подсказку, расположенную в верхней левой области сетки.

![Снимок экрана, на котором показано расположение даты окончания срока действия возможностей бесплатной учетной записи](./media/billing-avoid-charges-free-account/freeaccount-benefits-expiration-date.png)


После окончания срока действия по использованию бесплатных служб Azure будет взиматься оплата по мере использования служб. Портал Azure можно использовать для удаления ресурсов неиспользуемых служб. Если вы не собираетесь использовать какие-либо службы Azure, можно [отказаться от подписки](billing-how-to-cancel-azure-subscription.md).
