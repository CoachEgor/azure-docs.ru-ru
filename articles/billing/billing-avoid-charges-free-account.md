# <a name="avoid-charges-with-your-azure-free-account"></a>Экономия благодаря использованию бесплатной учетной записи Azure

Бесплатная учетная запись Azure предоставляет деньги на счете в Azure на сумму 200 долларов США, которые можно потратить в течение первых 30 дней, и 12 месяцев использования бесплатных служб в ограниченном количестве. Дополнительные сведения см. на странице [создания бесплатной учетной записи Azure](https://azure.microsoft.com/free/). В зависимости от состояния денег на счете вы можете использовать их или получить счет за использование, на который не распространяются преимущества бесплатной учетной записи Azure (бесплатные службы).

## <a name="azure-free-account-might-use-account-credit"></a>В бесплатной учетной записи Azure можно использовать деньги на соответствующем счете
Если у вас остались деньги на счете в Azure, срок использования которых не вышел, этот остаток используется, чтобы оплатить услуги, на которые не распространяются преимущества бесплатной учетной записи Azure.

## <a name="your-credit-runs-out-or-is-expired"></a>Деньги на счете закончились, или срок действия кредита истек
Если через 30 дней деньги на счете закончились или истек срок действия кредита, Azure отключит вашу подписку. Чтобы и дальше использовать службы Azure, обновите свою подписку до отдельной подписки с оплатой по мере использования. См. сведения об [обновлении бесплатной пробной подписки Azure](billing-upgrade-azure-subscription.md). После обновления ваша подписка по-прежнему имеет доступ к бесплатным службам в течение 12 месяцев. Плата взимается только за использование платных служб и ресурсов.

Рассмотрим некоторые из причин, из-за которых может взиматься плата в бесплатной учетной записи Azure.

### <a name="usage-exceeds-the-limits-of-free-services"></a>Показатели использования превышают ограничения бесплатных служб

Вы получаете ограниченное количество бесплатных служб каждый месяц в бесплатной учетной записи Azure. Количество бесплатных служб истекает в конце месяца и не переносится на следующий месяц. Например, вы получаете 5 ГБ хранилища файлов каждый месяц. Если в месяц использовалось 2 ГБ, оставшиеся 3 ГБ не переносятся на следующий месяц. Чтобы избежать выставления счетов, потребляйте ресурсы в допустимых пределах. См. сведения об ограничениях бесплатных служб в [вопросах и ответах о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Чтобы проверить потребление бесплатных служб, ознакомьтесь со статьей [Проверка использования бесплатных служб, включаемых в бесплатную учетную запись Azure](billing-check-free-service-usage.md).

### <a name="some-services-are-not-free"></a>Некоторые службы не предоставляются бесплатно

Выставление счетов за использование служб, которые не включены в бесплатную учетную запись Azure, будет осуществляться в рамках модели оплаты по мере использования. См. сведения о службах, входящих в бесплатную учетную запись, в [вопросах и ответах о бесплатной учетной записи Azure](https://azure.microsoft.com/free/free-account-faq/). Показатели использования можно проверить на портале Azure или в файле использования Azure. Дополнительные сведения см. в разделе [Регулярно просматривайте на портале детализацию затрат и график расходов](billing-getting-started.md#costs) и статье [Скачивание или просмотр счета на оплату и данных о ежедневном использовании в Azure](billing-download-azure-invoice-daily-usage-date.md).

### <a name="you-reached-the-end-of-your-free-12-months"></a>Вы достигли конца бесплатного 12-месячного периода

Срок использования бесплатных служб истекает в конце года. Узнать окончание срока действия бесплатных служб можно на портале Azure.

1. Войдите на [портале Azure](https://portal.azure.com).

2. В области навигации слева выберите **Все службы**.

3.  Выберите **Подписки**.

4.  Выберите подписку, созданную при регистрации для получения бесплатной учетной записи.

5.  Прокрутите вниз, чтобы найти сетку бесплатных служб. Щелкните подсказку, расположенную в верхней левой области сетки.

![Снимок экрана, на котором показано расположение даты окончания срока действия возможностей бесплатной учетной записи](./media/billing-avoid-charges-free-account/freeaccount-benefits-expiration-date.png)


После окончания срока действия по использованию бесплатных служб Azure будет взиматься оплата по мере использования служб. Портал Azure можно использовать для удаления ресурсов неиспользуемых служб. Если вы не собираетесь использовать какие-либо службы Azure, можно [отказаться от подписки](billing-how-to-cancel-azure-subscription.md).
