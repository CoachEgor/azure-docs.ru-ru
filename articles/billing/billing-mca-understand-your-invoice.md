# <a name="terms-in-your-microsoft-customer-agreement-invoice"></a>Условия в счете по соглашению для клиентов Майкрософт

Эта статья относится к учетной записи выставления счетов Azure для соглашения клиента Майкрософт. [Проверьте, есть ли у вас доступ к соглашению клиента Майкрософт](#check-access-to-a-microsoft-customer-agreement).

В счете представлены сводные данные о расходах и инструкции по оплате. Он доступен для скачивания в формате PDF на [портале Azure](https://portal.azure.com/). Также его можно получить по электронной почте. Дополнительные сведения см. [в статье Просмотр и скачивание счета Microsoft Azure](billing-download-azure-invoice.md).

## <a name="billing-period"></a>Период оплаты

Счет выставляется ежемесячно. Вы можете узнать, какой день месяца вы получили счета, проверив *дату счета* в разделе Свойства профиля выставления счетов в [портал Azure](https://portal.azure.com/). Плата, которая происходит между концом расчетного периода и датой счета, включается в счет следующего месяца, так как они находятся в следующем расчетном периоде. Даты начала и окончания расчетного периода для каждого счета перечислены в отчете PDF-файл с накладной, представленный в **сводке по выставлению счетов**.

## <a name="invoice-terms-and-descriptions"></a>Условия и описания счетов

В следующих разделах перечислены важные термины, отображаемые в счете, и приведены описания для каждого термина.

### <a name="invoice-summary"></a>Сводка по счетам

**Сводка** по счетам находится в верхней части первой страницы и отображает сведения о профиле выставления счетов и о том, как вы платите.

![Раздел "Сводка по счету"](./media/billing-understand-your-invoice-mca/invoicesummary.png)

| Термин | Описание |
| --- | --- |
| Покупатель |Адрес юридического лица, найденный в свойствах учетной записи выставления счетов|
| Получатель счета |Адрес выставления счета для профиля выставления счетов, полученный в свойствах профиля выставления счетов|
| Профиль выставления счетов |Имя профиля выставления счетов, получающего счет |
| П.О. номер |Необязательный номер заказа на покупку, присваиваемый вами для отслеживания |
| Номер счета |Уникальный номер счета, созданный корпорацией Майкрософт, используемый для отслеживания |
| Дата выставления счета |Дата создания счета, обычно через пять – 12 дней после окончания цикла выставления счетов. Вы можете проверить дату выставления счета в свойствах профиля выставления счетов.|
| Условия оплаты |Как вы платите за счет Майкрософт. *30 дней* означает, что вы платите в течение 30 дней с даты выставления счета. |

### <a name="billing-summary"></a>Сводка по выставлению счетов

В **сводке по выставлению счетов** отображаются платежи по профилю выставления счетов с момента предыдущего расчетного периода, всех кредитов, которые были применены, налоги и общая сумма оплаты.

![Раздел сводки по выставлению счетов](./media/billing-understand-your-invoice-mca/billingsummary.png)

| Термин | Описание |
| --- | --- |
| Расходы|Общее число выплат Майкрософт за этот профиль выставления счетов с момента последнего расчетного периода |
| Титры |Кредиты, полученные от возврата |
| Применены кредиты Azure | Кредиты Azure, которые автоматически применяются к платам за Azure за каждый расчетный период |
| Промежуточный итог |Сумма до уплаты налогов |
| Налог |Тип и сумма выплачиваемого налога в зависимости от страны или региона профиля выставления счетов. Если вам не нужно оплачивать налог, вы не увидите налог на ваш счет. |
| Предполагаемая общая экономия |Предполагаемая общая сумма, сохраненная из эффективных скидок. Если применимо, эффективные тарифные тарифы перечислены под разделом строки покупки в разделе сведения по счету. |

### <a name="invoice-sections"></a>Разделы счетов

Для каждого раздела счета в профиле выставления счетов вы увидите оплату, объем примененных кредитов Azure, налог и общую сумму оплаты.

`Total = Charges - Azure Credit + Tax`

### <a name="details-by-invoice-section"></a>Раздел сведений по счету

Подробные сведения показывают затраты для каждого раздела счета, разбитого по заказу на продукцию. В каждом заказе продуктов стоимость разделяется по типу службы. Ежедневные расходы на ваши продукты и услуги можно найти в портал Azure, а также в CSV-файле с данными об использовании и расходах Azure. Дополнительные сведения см. в статье сведения о [расходах на ваш счет для соглашения клиента Майкрософт](billing-mca-understand-your-bill.md).

Общая сумма для каждого семейства услуг рассчитывается путем вычитания *кредитов Azure* на основе *кредитов и расходов* , а также за добавление *налога*:


![Раздел сведений по счету](./media/billing-understand-your-invoice-mca/invoicesectiondetails.png)

| Термин |Описание |
| --- | --- |
| Цена за единицу | Эффективная цена за единицу службы (в валюте ценообразования), используемая для ставки использования. Это уникальное название продукта, семейства услуг, средства оценки и предложения. |
| Количество | Приобретенное или потребленное количество в течение расчетного периода |
| Расходы и кредиты | Чистая сумма расходов после применения кредитов и возмещения |
| Кредиты Azure | Количество кредитов Azure, примененных к расходам/кредитовым обязательствам|
| Ставка налога | Налоговые ставки в зависимости от страны или региона |
| Сумма налогов | Объем налога, применяемого к покупке на основе налоговой ставки |
| Всего | Общая сумма покупки |

### <a name="how-to-pay"></a>Как платить

В нижней части счета приведены инструкции по оплате счета. Вы можете платить по чекам, сети или сети. Если вы платите через Интернет, вы можете использовать кредитную карту или кредиты Azure, если это применимо.

### <a name="publisher-information"></a>Publisher Information (Сведения об издателе)

Если в счете есть сторонние службы, имя и адрес каждого издателя перечислены в нижней части счета.
