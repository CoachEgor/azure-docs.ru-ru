## <a name="invoice-terms-and-descriptions"></a>Условия предоставления и описания

В следующих разделах перечислены важные термины, которые встречаются в счете и описания каждого термина.

### <a name="invoice-summary"></a>Сводка по счету

**Сводка по счету** верхней части первой страницы и отображает сведения о профиле выставления счетов и метод оплаты.

![Раздел "Сводка по счету"](./media/billing-understand-your-invoice-mca/invoicesummary.png)

| Термин | ОПИСАНИЕ |
| --- | --- |
| Покупатель |Адрес вашего юридического лица, в выставления счетов свойства учетной записи|
| Получатель счета |Расчетный адрес для получения счета, найти в свойствах выставления счетов профиля в профиль выставления счетов|
| Профиль выставления счетов |Имя профиля выставления счетов, получения счета |
| АБОНЕНТСКИЙ number |Необязательный номер заказа на покупку, присваиваемый вами для отслеживания |
| Номер счета |Номер счета уникальный, созданный корпорацией Майкрософт, используется для отслеживания |
| Дата выставления счета |Дата создания счета, обычно пяти до 12 дней после конца цикла выставления счетов. Вы можете проверить Дата счета в выставления счетов свойства профиля.|
| Условия оплаты |Как вы платите за счет корпорации Майкрософт. *В течение 30 дней* означает, что плата взимается в течение 30 дней после выставления счета. |

### <a name="billing-summary"></a>Сводка по выставлению счетов

**Выставления счетов Сводка** выплаты от профиль выставления счетов с момента предыдущего расчетного периода, все кредиты, которые были применены, налог и общая сумма оплаты.

![Раздел сводки выставления счетов](./media/billing-understand-your-invoice-mca/billingsummary.png)

| Термин | ОПИСАНИЕ |
| --- | --- |
| Расходы|Общее число Майкрософт взимает плату за этот профиль выставления счетов, так как последний расчетный период |
| Баллы |Деньги на счете, полученный от возвращает |
| Кредиты Azure | Кредиты Azure, которые автоматически применяются к Azure плата за каждый расчетный период |
| Подытог |Сумма до уплаты налогов |
| Налог |Тип и сумма налога, вы платите, в зависимости от страны профиль выставления счетов. Если не нужно платить налог, вы не увидите налог на вашем счете. |
| Оценочная общая экономия |Оценочная Общая сумма, сохраненное с эффективным скидки. Если это применимо, льготные тарифы, действующие перечислены под позиции закупки в сведениях о разделе счета. |

### <a name="invoice-sections"></a>Разделы счетов

Для каждого раздела счета в профиль выставления счетов вы увидите накладные расходы, объем кредиты Azure, налог и общая сумма оплаты.

`Total = Charges - Azure Credit + Tax`

### <a name="details-by-invoice-section"></a>Сведения о разделе счета

В сведениях отображается стоимость для каждого раздела счета, разбитые по заказа. В каждый заказ продуктов стоимость распределяется по тип службы. Ежедневного за для продуктов и служб можно найти на портале Azure и Azure об использовании и затратах CSV. Дополнительные сведения см. Дополнительные [разбивка по расходам в счете клиента соглашения Microsoft](billing-mca-understand-your-bill.md).

Общая сумма к оплате для каждого семейства службы рассчитывается путем вычитания *кредиты Azure* из *кредиты/плата* и добавление *налог*:

<!-- `Total = Charges/Credits - Azure Credit + Tax` -->

![Сведения о разделе счета](./media/billing-understand-your-invoice-mca/invoicesectiondetails.png)

| Термин |ОПИСАНИЕ |
| --- | --- |
| Цена за единицу | Эффективное модульное цены службы (в цены в валюте), которая используется частоте использования. Он уникален для продукта, семейства служб, счетчиков и предложения. |
| Кол-во | Количество приобретенных или период выставления счетов |
| Плата за/кредиты | Чистая сумма расходов, после применения кредиты и возвраты |
| Кредиты Azure | Объем кредиты Azure, плата за/кредиты|
| Налоговая ставка | Налоговых ставок в зависимости от страны |
| Сумма налогов | Объем налог на покупку на основании налоговая ставка |
| Итого | Общая сумма к оплате за покупку |

### <a name="how-to-pay"></a>Способы оплаты

В нижней части счета нет указаний за оплату счета. Оплата может производиться при проверке, сети, или в оперативном режиме. Если оплата осуществляется через Интернет, можно использовать кредитной или дебетовой карты или кредиты Azure, если применимо.

### <a name="publisher-information"></a>Publisher Information (Сведения об издателе)

Если у вас есть сторонние службы в вашем счете, имя и адрес каждого издателя отображается внизу счета.
