# <a name="understand-the-charges-on-your-microsoft-customer-agreements-invoice"></a>Основная информация о расходах в соглашении клиента Microsoft счет

Вы можете отслеживать расходы на вашем счете путем анализа отдельные транзакции.

В учетной записи выставления счетов для клиента соглашение Microsoft счет создается каждый месяц для каждого профиля выставления счетов. Счет включает в себя все расходы с предыдущего месяца. Счетов можно просмотреть на портале Azure. Дополнительные сведения см. в разделе [скачивать счета клиента соглашения Microsoft](billing-download-azure-invoice-daily-usage-date.md#download-invoices-for-a-microsoft-customer-agreement).

Эта статья относится к учетной записи выставления счетов для клиента соглашение Microsoft. [Проверьте наличие доступа к соглашению клиента Microsoft](#check-access-to-a-microsoft-customer-agreement).

## <a name="view-transactions-for-an-invoice-in-the-azure-portal"></a>Просмотр транзакций для счета на портале Azure

1. Войдите на [портале Azure](https://www.azure.com).

2. Выполните поиск **Cost Management + выставление счетов**.

    ![Снимок экрана, показывающий поиска портала Azure для управления затратами + выставление счетов](./media/billing-understand-your-bill-mca/billing-search-cost-management-billing.png)

3. Выберите **все транзакции** из левой части экрана. В зависимости от вашего доступа, возможно, чтобы выбрать учетную запись выставления счетов или профиля выставления счетов, а затем выберите **все транзакции**.

4. Все страница «транзакции» отображаются следующие сведения:

    ![Снимок экрана, показывающий список выставляется счет транзакций](./media/billing-understand-your-bill-mca/mca-billed-transactions-list.png)

    |столбец  |Определение  |
    |---------|---------|
    |Дата     | Дата транзакции  |
    |ИД счета     | Идентификатор для счета, получили транзакции выставлен счет. Если отправить запрос в службу поддержки, совместно использовать код в службу поддержки с просьбой ускорить обработку запроса в службу поддержки Azure |
    |Тип транзакции     |  Тип транзакции, как плата за покупку, Отмена и использование  |
    |Семейство продуктов     | Категория продукта, например вычислений для виртуальных машин или базы данных для базы данных Azure SQL|
    |Номер sku продукта     | Уникальный код, определяющий экземпляр продукта |
    |Сумма     |  Количество транзакций      |
    |Раздел счета     | Транзакции отображаются на этой части профиля счет на оплату |
    |Профиль выставления счетов     | Транзакции отображаются на этот профиль выставления счетов |

5. Выполните поиск идентификатора счета-фактуры для фильтрации операций по счету.

### <a name="view-transactions-by-invoice-sections"></a>Просмотр транзакций, разделы счетов

Разделы счетов позволяют организовать затраты в счете профиль выставления счетов. Дополнительные сведения см. в разделе [понять раздел счета](billing-mca-overview.md#understand-invoice-sections). При создании счета, плата за все разделы в профиль выставления счетов отражать в счете.

Ниже выплаты для учета Dept счета раздела Пример счет на.

![Пример изображения с подробными сведениями в разделе сведений о счете](./media/billing-understand-your-bill-mca/invoicesection-details.png)

После определения накладные расходы в разделе счета, транзакции можно просмотреть на портале Azure, чтобы информация о расходах.

1. Перейдите к странице все транзакции, на портале Azure, чтобы просмотреть операции для счета. Дополнительные сведения см. в разделе [просматривать транзакции счета на портале Azure](#view-transactions-for-an-invoice-in-the-azure-portal).

2. Фильтровать по имени раздела счета, чтобы просматривать транзакции в разделе счета.

## <a name="understand-pending-charges-to-estimate-your-next-invoice"></a>Понять ожидающих расходов для оценки следующего счета

В учетной записи выставления счетов для клиента соглашение Microsoft пока не взимается плата, они оценки и считается ожидание. Позволяет просматривать ожидающие расходы на портале Azure, чтобы оценить ближайшую накладную. Ожидающие накладные расходы, оценки и без учета налогов, фактические расходы в ближайшую накладную будут отличаться от ожидающие накладные расходы.

### <a name="view-summary-of-pending-charges"></a>Просмотр сводки по ожидающих расходов

1. Войдите на [портале Azure](https://www.azure.com).

2. Выполните поиск **Cost Management + выставление счетов**.

   ![Снимок экрана, показывающий поиска портала Azure для управления затратами + выставление счетов](./media/billing-understand-your-bill-mca/billing-search-cost-management-billing.png)

3. Выберите профиль выставления счетов. В зависимости от вашего доступа необходимо выбрать учетную запись выставления счетов. Учетная запись выставления счетов, выберите **выставления счетов профили** выберите профиль выставления счетов.

4. Выберите **Сводка** вкладки в верхней части экрана.

5. В разделе расходов отображения месяца с начала и расходы за последний месяц.

   ![Снимок экрана, показывающий поиска портала Azure для управления затратами + выставление счетов](./media/billing-understand-your-bill-mca/mca-billing-profile-summary.png)

Расходы с начала месяца — ожидание платежи за текущий месяц и будут выставляться счета, при создании счета за месяц. Если по-прежнему не создается счета за прошлый месяц, то расходы за последний месяц также не ожидается и будет отражать в ближайшую накладную.

### <a name="view-pending-transactions"></a>Просмотр ожидающих транзакций

После того, как ожидающие расходы, можно понять расходы путем анализа отдельные транзакции, из которых составлены накладные расходы. На этом этапе ожидается использование расходы не отображаются на странице все транзакции. Плата за использование ожидающие можно просмотреть на странице подписки Azure. Дополнительные сведения см. в разделе [Просмотр ожидающих плата за использование](#view-pending-usage-charges)

1. Войдите на [портале Azure](https://www.azure.com).

2. Выполните поиск **Cost Management + выставление счетов**.

   ![Снимок экрана, показывающий поиска портала Azure для управления затратами + выставление счетов](./media/billing-understand-your-bill-mca/billing-search-cost-management-billing.png)

3. Выберите профиль выставления счетов. В зависимости от вашего доступа необходимо выбрать учетную запись выставления счетов. Учетная запись выставления счетов, выберите **выставления счетов профили** выберите профиль выставления счетов.

4. Выберите **все транзакции** из левой части экрана.

5. Поиск **ожидающие**. Используйте **Timespan** фильтр, чтобы просмотреть ожидающие расходов для текущего или последнего месяца.

   ![Снимок экрана, показывающий список ожидающих транзакций](./media/billing-understand-your-bill-mca/mca-pending-transactions-list.png)

### <a name="view-pending-usage-charges"></a>Просмотреть ожидающие плата за использование

1. Войдите на [портале Azure](https://www.azure.com).

2. Выполните поиск **Cost Management + выставление счетов**.

   ![Снимок экрана, показывающий поиска портала Azure для управления затратами + выставление счетов](./media/billing-understand-your-bill-mca/billing-search-cost-management-billing.png)

5. На странице подписок Azure отображается текущий и плата за последний месяц для каждой подписки в профиль выставления счетов. Расходы с начала месяца — ожидание платежи за текущий месяц и будут выставляться счета, при создании счета за месяц. Если по-прежнему не создается счета за прошлый месяц, то расходы за последний месяц также не ожидается.
