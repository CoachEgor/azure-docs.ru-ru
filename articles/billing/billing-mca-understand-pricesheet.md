# <a name="terms-in-your-microsoft-customer-agreement-price-sheet"></a>Условия в прейскуранте Клиентского соглашения Майкрософт

В этой статье рассматривается учетная запись выставления счетов Azure для клиентского соглашения Майкрософт. [Проверьте наличие доступа к Клиентскому соглашению Майкрософт](#check-access-to-a-microsoft-customer-agreement).

Если вы являетесь владельцем профиля выставления счетов, участником, читателем или менеджером счетов, вы можете скачать прейскурант для вашей организации из портала Azure. См. [Просмотр и скачивание сведений о ценах для вашей организации](billing-ea-pricing.md).

## <a name="terms-and-descriptions-in-your-price-sheet"></a>Условия и описания в прейскуранте

В следующем разделе описываются важные условия, приведенные в прейскуранте Клиентского соглашения Майкрософт.

| **Имя поля**   | **Описание**   |
| --- | --- |
| basePrice  | Рыночная цена на момент регистрации клиента или рыночная цена на момент запуска единицы измерения службы, если она установлена после регистрации.   |
| billingAccountId  | Уникальный идентификатор учетной записи выставления счетов.   |
| billingAccountName  | Имя учетной записи выставления счетов.  |
| billingCurrency | Валюта, в которой размещаются платежи. |
| billingProfileId  | Уникальный идентификатор профиля выставления счетов.   |
| billingProfileName  | Имя профиля выставления счетов, настроенного на получение счетов. Цены в прейскуранте связаны с этим профилем выставления счетов. |
| currency | Валюта, в которой отображаются все цены. |
| discount | Скидка за уровень "Выпуск", уровень "Бесплатный", включенный объем или договорная скидка (если применимо). Она представляется в процентах. |
| effectiveEndDate  | Дата окончания срока действия цены. |
| effectiveStartDate  | Дата вступления цены в силу. |
| includedQuantity | Объемы потребления конкретных служб без добавочной оплаты. |
| marketPrice | Текущая, приоритетная цена определенной службы на рынке. |
| meterId  | Уникальный идентификатор для средства измерения. |
| meterCategory  | Имя категории классификации для единицы измерения. Например, _Облачные службы_, _Сеть_и т. д. |
| meterName  | Имя индикатора. Единица измерения представляет собой развертываемый ресурс службы Azure. |
| meterSubCategory  | Имя категории подклассификации для единицы измерения.  |
| meterType  |  Имя типа единицы измерения. |
| meterRegion  | Имя региона, в котором доступна единица измерения для службы. В этом столбце указывается расположение центра обработки данных для определенных служб. От этого расположения может зависеть стоимость некоторых услуг.    |
| Продукт  | Название продукта, в котором начисляется плата. Например: Базовая база данных SQL и стандартная база данных SQL  |
| productId  | Уникальный идентификатор продукта, используемый средством измерения. |
| productOrderName  | Название приобретенного плана продукта. |
| serviceFamily  | Тип службы Azure. Например: Вычисление, Аналитика, Безопасность |
| tierMinimumUnits  | Определяет нижний предел диапазона уровней, для которого определяются цены. Например, если диапазон колеблется от 0 до 100, tierMinimumUnits будут равны 0.  |
| unitOfMeasure  | Определяет единицы измерения для выставления счетов для службы. Например, счета за службы вычислений выставляются на почасовой основе. |
| unitPrice  | Цена за единицу продукции на момент выставления счета (не эффективная смешанная цена) в зависимости от единицы измерения и наименования заказа продукта.  Примечание. Цена за единицу отличается от фактической цены при загрузке подробной информации об использовании в случае услуг, имеющих дифференцированную цену по уровням.  В случае услуг с многоуровневым ценообразованием, эффективная цена представляет собой смешанную ставку для всех уровней и не отражает цену единицы продукции для каждого уровня. Смешанная цена или эффективная цена — это чистая цена за потребленное количество, охватывающая несколько уровней (где каждый уровень имеет свою цену единицы продукции). |
