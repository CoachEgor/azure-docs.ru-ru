# <a name="resolve-past-due-balance-for-your-azure-subscription"></a>Устранение проблемы с задолженностью по подписке Azure

В этой статье раскрываются сведения для клиентов с учетными записями программы Microsoft Online Services.

Если ваш платеж не получен или мы не можем обработать его, вы можете получить сообщение электронной почты или оповещение в Центре управления учетной записью либо на портале Azure.
Если вы [администратор](billing-subscription-transfer.md#whoisaa) учетной записи, вы можете сопоставить неоплаченные расходы на [портале Azure](https://portal.azure.com). Если вы оплачиваете услугу по выставляемым счетам, отправьте платеж в местоположение, указанное внизу счета.

> [!IMPORTANT]
> * Если у вас несколько подписок, использующих одну и ту же кредитную карту, и все они просрочены, необходимо одновременно оплатить всю задолженность.
> * Платежное средство, которое вы используете для погашения платежей, станет новым активным способом оплаты для всех подписчиков, которые использовали неудачный способ оплаты.

## <a name="resolve-past-due-balance-in-the-azure-portal"></a>Решение проблемы просроченных платежей на портале Azure

1. Войдите на [портал Azure](https://portal.azure.com) в качестве администратора учетной записи.
1. Выполните поиск по фразе **Управление затратами + выставление счетов**.
1. На странице "Обзор" вы увидите список ваших подписок. Если состояние подписки — просрочено, щелкните ссылку **Покрытие задолженности**.
    ![Снимок экрана, показывающий ссылку на покрытие задолженности](./media/billing-azure-subscription-past-due-balance/settle-balance-entry-point.png)
1. Общая сумма непогашенной задолженности отражает непогашенные платежи по всем службам Microsoft, использующим неудачный способ оплаты.
1. Выберите способ оплаты, чтобы оплатить задолженность. Данный метод оплаты станет активным для всех подписчиков, использующих неудачный метод оплаты.
    ![Снимок экрана, на котором показана ссылка на выбор метода оплаты](./media/billing-azure-subscription-past-due-balance/settle-balance-screen.png)
1. Если выбранный метод оплаты также имеет непревзойденные расходы на службы Майкрософт, это будет отражено в общей задолженности. Вы также обязаны оплатить эти расходы.
1. Нажмите кнопку **Оплата**.

## <a name="troubleshoot-declined-credit-card"></a>Устранение проблем с отклонением кредитной карты

Если платеж по карте был отклонен вашим финансовым учреждением, обратитесь в это финансовое учреждение, чтобы решить проблему. Обратитесь в банк, чтобы убедиться в следующем:
- Международные транзакции включены на карте.
- На карте есть достаточный кредитный лимит или средства для покрытия задолженности.
- На карте разрешены регулярные платежи.

## <a name="not-getting-billing-email-notifications"></a>Платежные уведомления не приходят по электронной почте?

Если вы являетесь администратором учетной записи, [проверьте, какой адрес электронной почты используется для уведомлений](billing-how-to-change-azure-account-profile.md). Мы рекомендуем использовать адрес электронной почты, который будет регулярно проверяться. Если адрес электронной почты указан правильно, проверьте папку спама.

## <a name="if-i-forget-to-pay-what-happens"></a>Что случится, если я забуду заплатить?

Служба отменяется, а ресурсы становятся недоступными. Данные Azure будут удалены через 90 дней после прекращения услуги. Дополнительные сведения см. в разделе [What happens to your data if you leave the service](https://go.microsoft.com/fwLink/p/?LinkID=822930&clcid=0x409) (Что происходит с данными, когда вы прекращаете использовать службу).

Если вы знаете, что платеж обработан, но подписка по-прежнему отключена, обратитесь в [службу поддержки Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade).
