# <a name="resolve-past-due-balance-for-your-azure-subscription-after-getting-an-email-from-azure"></a>Решение проблемы просроченных платежей по подписке Azure после получения электронного сообщения от Azure

Если ваш платеж не получен или мы не можем обработать его, вы можете получить сообщение электронной почты или оповещение в Центре управления учетной записью либо на портале Azure. Это может быть сообщение следующего содержания:

**Мы не смогли получить оплату за вашу подписку с вашей кредитной карты. Чтобы избежать перерывов в работе служб, обновите платежную информацию.**

Чтобы устранить задолженность, следуйте указаниям, приведенным в этой статье.

## <a name="resolve-the-past-due-balance"></a>Решение проблемы просроченных платежей

* Если кредитная карта отклонена или истек ее срок действия, [измените кредитную карту, используемую для оплаты подписки](billing-how-to-change-credit-card.md), или обратитесь в банк, чтобы устранить проблему. При обновлении метода оплаты Azure немедленно обрабатывает платежи по всем непогашенным задолженностям. Кроме того, обрабатываются платежи по остальным задолженностям за использование служб Майкрософт.

* Если вы оплачиваете услугу по выставляемым счетам, отправьте платеж в местоположение, указанное внизу счета. Если вам нужна помощь, обратитесь в [службу поддержки Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade).

* Если счет не получен, потому что администратор учетной записи покинул компанию либо изменились роли, обратитесь в [службу поддержки Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) для устранения этой проблемы.

* Если вы знаете, что платеж обработан, но подписка по-прежнему отключена, обратитесь в [службу поддержки Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade).

## <a name="not-getting-billing-email-notifications"></a>Платежные уведомления не приходят по электронной почте?

Если вы являетесь администратором учетной записи, [проверьте, какой адрес электронной почты используется для уведомлений](billing-how-to-change-azure-account-profile.md). Мы рекомендуем использовать адрес электронной почты, который будет регулярно проверяться. Если адрес электронной почты указан правильно, проверьте папку спама.

## <a name="if-i-forget-to-pay-what-happens"></a>Что случится, если я забуду заплатить?

Служба отменяется, а ресурсы становятся недоступными. Данные Azure будут удалены через 90 дней после прекращения услуги. Дополнительные сведения см. в разделе [What happens to your data if you leave the service](https://go.microsoft.com/fwLink/p/?LinkID=822930&clcid=0x409) (Что происходит с данными, когда вы прекращаете использовать службу).
