# <a name="understand-microsoft-customer-agreement-administrative-roles-in-azure"></a>Понять соглашении клиента Microsoft административные роли в Azure

Для управления учетной записи выставления счетов для клиента соглашение Microsoft используйте ролей, описанных в следующих разделах. Эти роли являются дополнением к встроенных ролей Azure имеет для управления доступом к ресурсам. Дополнительные сведения см. в статье [Встроенные роли для ресурсов Azure](../role-based-access-control/built-in-roles.md).

Эта статья относится к учетной записи выставления счетов для клиента соглашение Microsoft. Проверьте наличие доступа к соглашению клиентов Microsoft.

## <a name="billing-role-definitions"></a>Определения ролей выставления счетов

В следующей таблице описаны выставления счетов ролей используется для управления учетной записью выставления счетов, выставления счетов профили и выставлять счета разделах.

|Role|Описание|
|---|---|
|Владелец учетной записи выставления счетов|Управлять всем для выставления счета|
|Участник учетной записи выставления счетов|Управлять всем, кроме разрешения на учетную запись выставления счетов|
|Читатель счетов учетной записи|Только для чтения представление все, что в платежный счет|
|Владелец профиль выставления счетов|Управлять всем для профиль выставления счетов|
|Участник профиля выставления счетов|Управлять всем, кроме разрешения на профиль выставления счетов|
|Читатель данных профиля выставления счетов|Только для чтения представление все, что в профиль выставления счетов|
|Управление счетами|Просмотр и оплачивать счета-фактуры профиль выставления счетов|
|Владелец раздел счета|Управлять всем, в разделе "счета"|
|Участник раздел счета|Управлять всем, кроме разрешения в разделе "счета"|
|Модуль чтения разделе счета|Только для чтения представление все, что в разделе счета|
|Создатель подписки Azure|Создавать подписки Azure|

## <a name="manage-billing-roles-in-the-azure-portal"></a>Управление ролями выставления счетов на портале Azure

1. Войдите на [портале Azure](https://portal.azure.com).

2. Поиск **Cost Management + выставление счетов**.

   ![Снимок экрана, показывающий поиск на портале Azure](./media/billing-understand-mca-roles/billing-search-cost-management-billing.png)

3. Выберите **управление доступом (IAM)** в области, например платежного счета, профиль выставления счетов или разделе счета, где вы хотите предоставить доступ.

4. На странице управления (IAM) доступа перечислены пользователи и группы, назначенные каждой роли для данной области.

   ![Снимок экрана, показывающий список "Администраторы" для учетной записи выставления счетов](./media/billing-understand-mca-roles/billing-list-admins.png)

5. Чтобы предоставить доступ пользователю, выберите **добавить** в верхней части страницы. В раскрывающемся списке ролей выберите роль. Введите адрес электронной почты пользователя, которому вы хотите предоставить доступ. Выберите **Сохранить** для назначения роли.

   ![Снимок экрана, показывающий Добавление администратора к учетной записи выставления счетов](./media/billing-understand-mca-roles/billing-add-admin.png)

6. Чтобы удалить доступ для пользователя, выберите пользователя, с назначением ролей, которые вы хотите удалить. Выберите Удалить.

   ![Снимок экрана, показывающий, удаление администратора из учетной записи выставления счетов](./media/billing-understand-mca-roles/billing-remove-admin.png)
