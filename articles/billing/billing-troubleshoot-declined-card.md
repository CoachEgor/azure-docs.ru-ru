# <a name="troubleshoot-a-declined-card-at-azure-sign-up"></a>Устранение неполадок с отклоненной картой при регистрации в Azure

Эта статья поможет устранить проблемы, при которых кредитная карта отклонена в службе регистрации Azure в портал Azure или в центре учетных записей Azure. Прежде чем приступить к устранению проблемы, проверьте следующие моменты.

- Убедитесь, что сведения, указанные для профиля учетной записи Azure, такие как Контактная электронная почта, адрес и номер телефона, верны.
- Проверьте правильность информации о кредитной карте.
- Убедитесь, что у вас нет учетной записи Майкрософт с такими же сведениями.
- Дебетовые карты не принимаются

## <a name="issues"></a>Вопросы

Ниже перечислены распространенные проблемы, которые могут привести к отклонению кредитной карты при регистрации в Azure.

### <a name="the-credit-card-provider-is-not-accepted-for-your-country"></a>Поставщик кредитных карт не принят в вашей стране

При выборе карточки Azure отображает параметры карты, допустимые в выбранной стране. Свяжитесь с банком или поставщиком карточки, чтобы убедиться, что ваша кредитная карта включена для международных транзакций. Дополнительные сведения о поддерживаемых странах и валютах см. в статье [вопросы и ответы по покупке Azure](https://azure.microsoft.com/pricing/faq/).

>[!Note]
>В настоящее время кредитные карты American Express не поддерживаются в качестве платежного средства в Индии. У нас нет интервала времени, когда он может быть принятой формой оплаты.

### <a name="youre-using-a-virtual-or-prepaid-card"></a>Вы используете виртуальную или предоплаченную карту 

Оплата с использованием таких карт не поддерживается для подписок Azure.

### <a name="your-credit-information-is-inaccurate-or-incomplete"></a>Ваша кредитная информация неточная или неполная 

Введенные имя, адрес и код CVV-должны точно совпадать с тем, что печатается на карточке.

### <a name="the-card-is-inactive-or-blocked"></a>Карта неактивна или заблокирована 

Обратитесь в банк, чтобы убедиться, что ваша карта активна.

Возможно, у вас возникли другие проблемы с регистрацией 

Дополнительные сведения об устранении проблем с регистрацией в Azure см. в следующей статье базы знаний: 

[Вы не можете зарегистрироваться в Azure в портал Azure или в центре учетных записей Azure.](billing-troubleshoot-azure-sign-up.md)

### <a name="you-represent-a-business-that-doesnt-want-to-pay-by-card"></a>Вы представляете компанию, которая не хочет платить по карте 

Если вы представляете предприятие, вы можете использовать способы оплаты счетов, такие как чеки, ночные проверки или проводные передачи для оплаты подписки Azure. После настройки учетной записи для оплаты по счету нельзя перейти на другой вариант оплаты, если у вас нет соглашения с клиентами Майкрософт и вы не подписались на Azure с помощью веб-сайта Azure.

Дополнительные сведения о способах оплаты по счету см. в статье [Отправка запроса на оплату подписки Azure по счету](billing-how-to-pay-by-invoice.md).

### <a name="your-credit-card-information-is-outdated"></a>Сведения о кредитной карте устарели 

Сведения о том, как управлять сведениями о карте, включая изменение или удаление карточки, см. [в статье Добавление, обновление или удаление кредита для Azure](billing-how-to-change-credit-card.md).
