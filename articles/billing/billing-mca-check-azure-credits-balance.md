# <a name="track-azure-credit-balance-for-microsoft-customer-agreement"></a>Отслеживать счета на счете в Azure для клиентов соглашения Microsoft

Можно проверить баланс на счете в Azure для клиентов соглашения Microsoft на портале Azure. Использовать кредиты для оплаты продуктов, охваченных сумму кредита.

Если вы используете продукты, которые не распространяются деньги на счете или Вы исчерпали вашего сальдо по кредиту, плата взимается. Дополнительные сведения см. в разделе [продуктов, которые не распространяются кредиты Azure.](#products-that-arent-covered-by-azure-credits)

Эта статья относится к учетной записи выставления счетов для клиента соглашение Microsoft. [Проверьте наличие доступа к соглашению клиента Microsoft](#check-access-to-a-microsoft-customer-agreement).

## <a name="check-credit-balance-in-the-azure-portal"></a>Проверьте сальдо по кредиту на портале Azure

1. Войдите на [портале Azure]( https://portal.azure.com).

2. Выполните поиск по **Управление затратами + выставление счетов**.

   ![Снимок экрана, показывающий поиска на портале для управление затратами + выставление счетов](./media/billing-mca-check-azure-credits-balance/billing-search-cost-management-billing.png)

3. Перейдите к профилю выставления счетов. В зависимости от уровня вашего доступа вам, возможно, потребуется выбрать учетную запись выставления счетов. Учетная запись выставления счетов, выберите **выставления счетов профили** и затем профиля выставления счетов.

4. Выберите **кредиты Azure**.

5. На странице кредиты Azure отображаются следующие сведения.

   ![Снимок экрана сальдо по кредиту и транзакции для профиль выставления счетов](./media/billing-mca-check-azure-credits-balance/billing-mca-credits-overview.png)

   | Термин               | Определение                           |
   |--------------------|--------------------------------------------------------|
   | Оценка баланса  | Предполагаемый размер кредитов, которые вы получаете после рассмотрения всех выставления счетов и ожидающих транзакций |
   | Текущий баланс    | Сумма кредитов, начиная с последнего выставления счета. Она не включает все ожидающие проводки |
   | Транзакции       | Все транзакции выставления счетов, которые затронуты ваш баланс на счете в Azure |

   Если ваш баланс предполагаемое снижается до 0, вы оплачиваете все данные об использовании, включая для продуктов, охваченных деньги на счете.

6. Выберите **списка кредиты** Чтобы просмотреть список кредиты за профиль выставления счетов. Кредиты представлены следующие сведения:

   ![Снимок экрана списки кредиты для профиля выставления счетов](./media/billing-mca-check-azure-credits-balance/billing-mca-credits-list.png)

   | Термин                 | Определение                           |
   |----------------------|--------------------------------------------------------|
   | Источник               | Источник получения кредита |
   | Дата начала           | Дата, когда вы приобрели кредит |
   | Срок действия      | Дата, когда истекает срок действия кредита |
   | Balance              | Баланс, начиная с последнего выставления счета |
   | Исходная сумма      | Исходное количество кредитов для оплаты |
   | Статус               | Текущее состояние кредита. Состояние может быть активным, используемый, срок действия истек или истекает срок действия |

## <a name="how-credits-are-used-in-microsoft-customer-agreement"></a>Использование деньги на счете в соглашении клиента Microsoft

В учетной записи выставления счетов для клиента соглашение Microsoft вы профили выставления счетов для управления счета-фактуры и способы оплаты. Ежемесячный счет создается для каждого профиля выставления счетов и использовать методы оплаты для оплаты счета.

Кредиты Azure являются одним из способов оплаты. Вы получаете кредит от корпорации Майкрософт как специальная кредит и кредит уровня обслуживания. Эти кредиты, назначенные профиля выставления счетов. При для профиль выставления счета, деньги на счете автоматически применяются к Общая сумма по документу для расчета суммы, вам нужно платить. Оплата Оставшаяся сумма с помощью другой метод оплаты, такие как проверка или проводной передачи.
