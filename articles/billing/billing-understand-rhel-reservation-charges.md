# <a name="understand-how-the-red-hat-linux-enterprise-software-reservation-plan-discount-is-applied-for-azure"></a>Сведения о применении скидки план программного обеспечения Red Hat Linux Enterprise для Azure

После покупки плана Red Hat Linux, скидки для развернутых Red Hat виртуальных машин (ВМ), которые соответствуют резервирование применяется автоматически. Red Hat Linux план покрывает стоимость программным обеспечением Red Hat на виртуальной Машине Azure.

Чтобы купить подходящего плана Red Hat Linux, необходимо понимать, какие Red Hat виртуальные машины, выполнив и число виртуальных ЦП на этих виртуальных машинах. Следующие разделы помогут вам определить подходящий план на основе CSV-файла об использовании.

## <a name="discount-applies-to-different-vm-sizes"></a>Скидка применяется к различных размеров виртуальных Машин

Как и зарезервированные экземпляры виртуальных Машин Red Hat плана покупки обеспечивают гибкость размер экземпляра. Это означает, что ваша скидка применяется даже в случае развертывания виртуальной машины с разным количеством виртуальных процессоров. Скидка применяется к виртуальным машинам разных размеров в рамках плана программного обеспечения.

Сумма скидки зависит от коэффициента, указанного в следующих таблицах. Коэффициент сравнивает относительный охват для каждой метрики в этой группе. Коэффициент зависит от виртуальных ЦП виртуальной машины. Используйте значение отношение, для которого требуется вычислить, сколько экземпляров виртуальных Машин компания предлагает скидку в Red Hat Linux плана.

Например если вы покупаете план для Red Hat Linux Enterprise Server для виртуальной Машины с 3 или 4 виртуальных ЦП, коэффициент для этого резервирования — 2. Скидка хранилища охватывает стоимость программного обеспечения Red Hat:

- 2 развернутых виртуальных машин с 1 или 2 виртуальными ЦП;
- 1 развернутой виртуальной машины с 3 или 4 виртуальными ЦП;
- примерно 77 % (0,77) виртуальной машины с 5 или более виртуальными ЦП.

Коэффициент для 5 или более виртуальных ЦП составляет 2,6. Так что резервирование для Red Hat с виртуальной Машиной с 5 или более виртуальных ЦП охватывает только часть затрат на программное обеспечение, который составляет около 77%.
