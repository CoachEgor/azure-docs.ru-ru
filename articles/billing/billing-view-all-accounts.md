# <a name="view-billing-accounts-in-azure-portal"></a>Просмотр учетной записи выставления счетов на портале Azure  

Учетная запись выставления счетов создается при регистрации в Azure. Вы используете свою учетную запись выставления счетов для управления счетами, платежами и отслеживания расходов. Вы можете иметь доступ к нескольким учетным записям выставления счетов. Например, вы могли зарегистрироваться в Azure для своих личных проектов. Вы также могли получить доступ через Соглашение Enterprise Agreement вашей организации или Клиентское соглашение Майкрософт. Для каждого из этих сценариев у вас будет отдельная учетная запись выставления счетов.

В настоящее время портал Azure поддерживает следующие типы учетных записей выставления счетов:

- **Программа Microsoft Online Services**: Учетная запись выставления счетов для программы Microsoft Online Services создается при регистрации в Azure с помощью веб-сайта Azure. Например, при регистрации с [бесплатной учетной записи Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), [учетной записью с оплатой по мере использования](https://azure.microsoft.com/offers/ms-azr-0003p/) или как [подписчик Visual Studio](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/).

- **Соглашение Enterprise**. Учетная запись выставления счетов для Соглашения Enterprise создается когда ваша организация подписывает [Соглашение Enterprise (EA)](https://azure.microsoft.com/pricing/enterprise-agreement/) для использования Azure.

- **Клиентское соглашение Майкрософт**: Учетная запись выставления счетов для Клиентского соглашения Майкрософт создается, когда ваша организация работает с представителем корпорации Майкрософт для подписания Клиентского соглашения Майкрософт. Некоторые клиенты в выбранных регионах, которые выполняют вход через веб-сайт Azure [для учетной записи с оплатой по мере использования](https://azure.microsoft.com/offers/ms-azr-0003p/) или обновляют [бесплатную учетную запись Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), могут иметь учетную запись выставления счетов также для Клиентского соглашения Майкрософт. Дополнительные сведения см. в статье [Get started with your billing account for Microsoft Customer Agreement](billing-mca-overview.md) (Учетные записи выставления счетов для клиентского соглашения Майкрософт).

## <a name="scopes-for-billing-accounts"></a>Области для учетных записей выставления счетов
Область — это узел в учетной записи выставления счетов, который используется пользователями для просмотра и управления выставлением счетов. Именно здесь пользователи управляют данными выставления счетов, платежами, счетами и занимаются общим управлением учетными записями. 

### <a name="microsoft-online-services-program"></a>Программа Microsoft Online Services

|Область  |Определение  |
|---------|---------|
|учетная запись выставления счетов;     | Представляет одного владельца (администратора учетной записи) для одной или нескольких подписок Azure. Администратор учетной записи имеет право выполнять различные задачи выставления счетов, такие как создание подписок, просмотр счетов или изменение счетов для подписок.  |
|Subscription     |  Представляет группирование ресурсов Azure. В этой области создается счет. Она имеет собственные методы оплаты, которые используются для оплаты счета.|


### <a name="enterprise-agreement"></a>Соглашение Enterprise

|Область  |Определение  |
|---------|---------|
|учетная запись выставления счетов;    | Представляет регистрацию на Соглашение Enterprise. В этой области создается счет. Она структурирована с помощью отделов и учетных записей регистрации.  |
|отдел;     |  Необязательное группирование учетных записей регистрации.      |
|учетная запись регистрации;     |  Представляет одного владельца учетной записи. Подписки Azure создаются в этой области.  |


### <a name="microsoft-customer-agreement"></a>Клиентское соглашение Майкрософт

|Область  |Задачи  |
|---------|---------|
|учетная запись выставления счетов;     |   Представляет клиентское соглашение для нескольких продуктов и служб корпорации Майкрософт. Она структурирована с помощью профилей выставления счетов и разделов счетов.   |
|Профиль выставления счетов     |  Представляет счет и методы его оплаты. В этой области создается счет. В ней может быть несколько разделов счета.      |
|Раздел счета     |   Представляет группу затрат в счете. С этой областью связаны все подписки и другие покупки.    |


## <a name="switch-billing-scope-in-the-azure-portal"></a>Переключение области выставления счетов на портал Azure


1. Войдите на [портале Azure](https://portal.azure.com).

2. Выполните поиск по фразе **Управление затратами + выставление счетов**.

   ![Снимок экрана, показывающий поиск на портале Azure](./media/billing-view-all-accounts/billing-search-cost-management-billing.png)

3. Выберите **Все области выставления счетов** с левой стороны.

   ![Снимок экрана, на котором показаны все области счетов](./media/billing-view-all-accounts/billing-list-of-accounts.png)

   ** Если у вас есть доступ только к одной области, пункт **Все области выставления счетов** не будет отражаться.

4. Выберите область для просмотра сведений.
