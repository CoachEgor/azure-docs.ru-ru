# <a name="view-your-billing-accounts-in-azure-portal"></a>Просмотреть учетные записи выставления счетов на портале Azure  

Учетную запись выставления счетов создается для каждого соглашения, что это входить с корпорацией Майкрософт для использования Azure. Используйте учетную запись выставления счетов для управление способами счета-фактуры и платежей и отслеживать затраты на службы Azure. Может иметь доступ к нескольким учетным записям выставления счетов. Например вы возможно уже зарегистрировались для Azure личных проектах. Также имеется доступ через соглашение Enterprise или соглашения Microsoft Customer вашей организации. Для каждого из этих сценариев имеется отдельная учетная запись выставления счетов.

Портал Azure в настоящее время поддерживает следующий тип выставления счетов:

- **Программа Microsoft Online Services**: Учетную запись выставления счетов для программы Microsoft Online Services создается при регистрации для Azure через веб-сайте Azure. Например, когда пользователь регистрирует [бесплатную учетную запись Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), [предложение Azure оплатой по мере](https://azure.microsoft.com/offers/ms-azr-0003p/) или как [подписчик Visual studio](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/).

- **Соглашение Enterprise**. Учетную запись выставления счетов для соглашения Enterprise создается при регистрации организации в соглашение Enterprise для использования Azure. Администраторы в учетной записи выставления счетов для соглашения Enterprise использовать отделов и учетных записей, выставление счетов областей для управления и организации выставление счетов для своей организации.

- **Соглашение Microsoft Customer**: Учетную запись выставления счетов для клиента соглашение Microsoft создается при регистрации организации в соглашение Microsoft клиента для использования Azure. Администраторы в учетной записи выставления счетов для клиента соглашение Microsoft профили выставления счетов и накладной разделах выставления счетов областей для управления и организации выставление счетов для своей организации. Дополнительные сведения см. в разделе [начало работы с учетной записью выставления счетов для клиентов соглашения Microsoft](billing-mca-overview.md).

## <a name="change-billing-account-view-in-the-azure-portal"></a>Изменение выставления счетов представление учетной записи на портале Azure

Область является узлом в учетную запись выставления счетов, который пользователи будут использовать для просмотра и управления выставления счетов. Например выставления счетов для программы Microsoft Online Services имеет только область - выставления счетов учетной записи, а выставления счетов, учетные записи для соглашения Enterprise платежного счета, отдел и области учетных записей. Если у вас есть доступ для нескольких областей выставления счетов, по умолчанию, одна из областей отображается в Azure Cost Management + Billing страницы. Можно изменить область, следуя приведенным ниже инструкциям:

1. Войдите на [портале Azure](https://portal.azure.com).

2. Выполните поиск по **Управление затратами + выставление счетов**.

   ![Снимок экрана, показывающий поиск на портале Azure](./media/billing-view-all-accounts/billing-search-cost-management-billing.png)

3. Выберите **все выставления счетов области** с левой стороны.

   ![Снимок экрана, показывающий все области выставления счетов](./media/billing-view-all-accounts/billing-list-of-accounts.png)

4. Выберите область для получения сведений.
