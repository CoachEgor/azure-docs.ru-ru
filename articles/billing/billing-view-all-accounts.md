# <a name="view-billing-accounts-in-azure-portal"></a>Просмотр выставления счетов на портале Azure  

Учетную запись выставления счетов создается при регистрации для использования Azure. Использование учетной записи выставления счетов для управления ваши счета, платежи и отслеживать затраты. Может иметь доступ к нескольким учетным записям выставления счетов. Например вы возможно уже зарегистрировались для Azure личных проектах. Также имеется доступ через соглашение Enterprise или соглашения Microsoft Customer вашей организации. Для каждого из этих сценариев имеется отдельная учетная запись выставления счетов.

Портал Azure в настоящее время поддерживает следующий тип выставления счетов:

- **Программа Microsoft Online Services**: Учетную запись выставления счетов для программы Microsoft Online Services создается при регистрации для Azure через веб-сайте Azure. Например, когда пользователь регистрирует [бесплатную учетную запись Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), [учетную запись с оплатой](https://azure.microsoft.com/offers/ms-azr-0003p/) или как [подписчик Visual studio](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/).

- **Соглашение Enterprise**. Учетную запись выставления счетов для соглашения Enterprise создается при регистрации организации [Enterprise Agreement (EA)](https://azure.microsoft.com/pricing/enterprise-agreement/) для использования Azure.

- **Соглашение Microsoft Customer**: Учетную запись выставления счетов для клиента соглашение Microsoft создается в том случае, если ваша организация работает с представителем Майкрософт для подписания соглашения Microsoft Customer. Некоторые клиенты в избранных регионах, зарегистрируйтесь на веб-сайте Azure для [учетную запись с оплатой](https://azure.microsoft.com/offers/ms-azr-0003p/) или обновить их [бесплатную учетную запись Azure](https://azure.microsoft.com/offers/ms-azr-0044p/) может иметь учетную запись выставления счетов для клиента Microsoft Также и соглашение. Дополнительные сведения см. в разделе [начало работы с учетной записью выставления счетов для клиентов соглашения Microsoft](billing-mca-overview.md).

## <a name="scopes-for-billing-accounts"></a>Области для выставления счетов
Область является узлом в учетную запись выставления счетов, который пользователи будут использовать для просмотра и управления выставления счетов. Это, где пользователи данные о выставлении счетов, платежи, счета-фактуры, управлять и проводить общие учетные записи. 

### <a name="microsoft-online-services-program"></a>Программа Microsoft Online Services

|`Scope`  |Определение  |
|---------|---------|
|учетная запись выставления счетов;     | Представляет один владелец (администратор учетной записи) для одной или нескольких подписок Azure. Учетная запись администратора имеет права для выполнения различных задач выставления счетов, как создавать подписки, Просмотр счетов или менять тарифный план для подписок.  |
|Подписка     |  Представляет группирование ресурсов Azure. Счет создается в этой области. Он имеет свои собственные способы оплаты, которые используются для оплаты счета, его.|


### <a name="enterprise-agreement"></a>Соглашение Enterprise

|`Scope`  |Определение  |
|---------|---------|
|учетная запись выставления счетов;    | Представляет соглашения Enterprise Agreement. Счет создается в этой области. Структурируется путем использования отделов и учетных записей регистрации.  |
|Department     |  Группирование учетных записей регистрации.      |
|учетная запись регистрации;     |  Представляет один владелец. Подписки Azure создаются в этой области.  |


### <a name="microsoft-customer-agreement"></a>Клиентское соглашение Майкрософт

|`Scope`  |Задачи  |
|---------|---------|
|учетная запись выставления счетов;     |   Представляет соглашении клиента для нескольких продуктов и служб Майкрософт. Структурируется путем использования профилей выставления счетов и разделы счетов.   |
|Профиль выставления счетов     |  Представляет счет и его методы оплаты. Счет создается в этой области. Он может находиться несколько разделов счета.      |
|Раздел счета     |   Представляет группу затрат в счет. Подписки и других покупки связаны с этой области.    |


## <a name="switch-billing-scope-in-the-azure-portal"></a>Переключение области выставления счетов на портале Azure


1. Войдите на [портале Azure](https://portal.azure.com).

2. Поиск **Cost Management + выставление счетов**.

   ![Снимок экрана, показывающий поиск на портале Azure](./media/billing-view-all-accounts/billing-search-cost-management-billing.png)

3. Выберите **все выставления счетов области** из левой части окна.

   ![Снимок экрана, показывающий все области выставления счетов](./media/billing-view-all-accounts/billing-list-of-accounts.png)

   ** Вы не увидите **все выставления счетов области** при наличии доступа только к одной области.

4. Выберите область для получения сведений.
