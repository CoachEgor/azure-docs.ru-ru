# <a name="view-and-download-your-microsoft-azure-invoice"></a>Просмотр и скачивание счета Microsoft Azure

Счет можно скачать на [портале Azure](https://portal.azure.com/) или получить по электронной почте. Если вы являетесь клиентом Azure с Соглашением Enterprise (клиент EA), вы не можете скачать счет своей организации. Вместо этого счета отправляются пользователю, который настроил получение счетов за регистрацию.

[!INCLUDE [GDPR-related guidance](../../../includes/gdpr-intro-sentence.md)]

## <a name="when-invoices-are-generated"></a>Создание счетов

Счет создается на основе типа учетной записи выставления счетов. Это могут быть учетные записи выставления счетов программы Microsoft Online Service Program (MOSP), Клиентского соглашения Майкрософт (MCA) и Соглашения с партнером Майкрософт (MPA). Счета также создаются для учетных записей выставления счетов Соглашения Enterprise (EA). Однако счета учетных записей выставления счетов не отображаются на портале Azure.

Дополнительные сведения об учетных записях выставления счетов и определении типа своей учетной записи выставления счетов см. в статье [Учетные записи и области выставления счетов на портале Azure](../manage/view-all-accounts.md).

## <a name="invoices-for-mosp-billing-accounts"></a>Счета для учетных записей выставления счетов MOSP

Учетная запись выставления счетов MOSP создается при регистрации в Azure через веб-сайт Azure. Например, при регистрации с [бесплатной учетной записи Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), [учетной записью с оплатой по мере использования](https://azure.microsoft.com/offers/ms-azr-0003p/) или как [подписчик Visual Studio](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/).

В некоторых регионах клиенты, которые зарегистрировались на веб-сайте Azure для получения [учетной записи с оплатой по мере использования](https://azure.microsoft.com/offers/ms-azr-0003p/) или [бесплатной учетной записи Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), могут использовать учетную запись выставления счетов для MCA.

Если вы не знаете точно, какой тип учетной записи выставления счетов используется, ознакомьтесь со статьей [Проверка типа учетной записи](../manage/view-all-accounts.md#check-the-type-of-your-account), прежде чем выполнять инструкции из этой статьи. 

Учетная запись выставления счетов может иметь следующие счета:

**Оплата за услуги Azure.** Счет создается для каждой подписки Azure, содержащей ресурсы Azure, используемые подпиской. Счет содержит расходы за период выставления счетов. Период выставления счетов определяется по дню месяца создания подписки.

Например, Джон создает *подписку Azure 01* 5 марта, а *подписку Azure 02* 10 марта. Счет за *подписку Azure 01* будет выставляться с пятого дня месяца по четвертый день следующего месяца. Счет за *подписку Azure 02* будет выставляться c десятого дня месяца по девятый день следующего месяца. Счета для всех подписок Azure обычно создаются в день месяца создания учетной записи. Однако может возникнуть задержка до двух дней. В этом примере, если Джон создал свою учетную запись 2 февраля, счета для подписки Azure *01* и *02*, как правило, создаются на второй день каждого месяца. Однако может возникнуть задержка до двух дней.

**Azure Marketplace, резервирования и точечные виртуальные машины.** Счет за резервирования, продукты Marketplace и точечные виртуальные машины создается с помощью подписки. Счет включает соответствующие расходы за предыдущий месяц. Например, Джон приобрел резервирование 1 марта, а второе — 30 марта. Для резервирования в апреле будет создан общий счет. Счет для Azure Marketplace, резервирований и точечных виртуальных машин всегда создается приблизительно на девятый день месяца.

Если вы платите за использование Azure с помощью кредитной карты и приобрели резервирование, служба Azure немедленно создаст счет. Однако если вы используете оплату по счетам, то плата за резервирование будет включена в счет следующего месяца.

**План поддержки Azure.** Для подписки плана поддержки счет создается каждый месяц. Первый счет создается в день покупки или на протяжении двух дней. Последующие счета за план поддержки обычно формируются в день месяца, когда была создана учетная запись. Однако может возникнуть задержка до двух дней.

## <a name="download-your-mosp-azure-subscription-invoice"></a>Скачивание счета для подписки Azure MOSP

Счет создается только для подписки, которая относится к учетной записи выставления счетов MOSP. [Проверьте доступ к учетной записи MOSP](../manage/view-all-accounts.md#check-the-type-of-your-account). 

Чтобы скачать счет подписки, вам должна быть назначена роль администратора учетной записи. Пользователи с ролями владельца, участника или читателя могут скачать свой счет, если администратор учетной записи предоставил им разрешение. Дополнительные сведения см. в разделе [Предоставление пользователям разрешений на скачивание счетов](../manage/manage-billing-access.md#opt-in).

1. Выберите свою подписку на [странице подписок](https://portal.azure.com/#blade/Microsoft_Azure_Billing/SubscriptionsBlade) портала Azure.
1. В разделе выставления счетов выберите **Счета**.  
    ![Снимок экрана, на котором показан выбор пользователем параметра "Счета" для подписки](./media/download-azure-invoice/select-subscription-invoice.png)
1. Выберите **Скачать**, чтобы скачать PDF-версию счета, а затем выберите **Скачать** в разделе счета.  
    [![Снимок экрана, на котором показаны периоды выставления счетов, действие скачивания и общие суммы за каждый расчетный период](./media/download-azure-invoice/download-invoice-subscription.png)](./media/download-azure-invoice/download-invoice-subscription-zoomed.png#lightbox)
1. Вы также можете скачать сведения об израсходованных объемах с разбивкой по дням и расходам, выбрав **Скачать** в разделе сведений об использовании. Подготовка CSV-файла может занять несколько минут.  
    ![Снимок экрана с действием "Скачать счет" и страницей потребления](./media/download-azure-invoice/usage-and-invoice-subscription.png)

Дополнительные сведения о счете см. в статье [Расшифровка счета за использование Microsoft Azure](../understand/review-individual-bill.md). Справочные сведения об управлении затратами см. в статье [Предотвращение непредвиденных расходов с помощью функции выставления счетов и управления затратами в Azure](../manage/getting-started.md).

## <a name="download-your-mosp-support-plan-invoice"></a>Скачивание счета для плана поддержки MOSP

Счет создается только для плана поддержки, который относится к учетной записи выставления счетов MOSP. [Проверьте доступ к учетной записи MOSP](../manage/view-all-accounts.md#check-the-type-of-your-account). 

Чтобы скачать счет подписки плана поддержки, вам должна быть назначена роль администратора учетной записи.

1.  Войдите на [портал Azure](https://portal.azure.com).
1.  Выполните поиск по фразе **Управление затратами + выставление счетов**.  
    ![Снимок экрана с изображением поиска на портале по фразе "Управление затратами + выставление счетов"](./media/download-azure-invoice/search-cmb.png)
1.  Выберите **Счета** в левой части.
1.  Выберите подписку плана поддержки, а затем щелкните **Скачать**.  
    [![Снимок экрана, на котором показан вариант "Профиль выставления счетов" в списке](./media/download-azure-invoice/cmb-invoices.png)](./media/download-azure-invoice/cmb-invoices-zoomed-in.png#lightbox)
1.  Выберите **Скачать**, чтобы скачать PDF-версию счета.  
    ![Снимок экрана, на котором показаны периоды выставления счетов, действие скачивания и общие суммы за каждый расчетный период](./media/download-azure-invoice/download-invoice-support-plan.png)

## <a name="allow-others-to-download-an-mosp-subscription-invoice"></a>Предоставление пользователям разрешений на скачивание счетов подписки MOSP

PDF-версия счета содержит личные сведения об администраторе учетной записи, поэтому администратор учетной записи этой подписки должен предоставить другим пользователям разрешение на скачивание счетов. После предоставления разрешений следующие пользователи с соответствующими ролями смогут скачивать счета подписки:

- Пользователи
- Группы
- субъекты-службы с ролью владельца;
- Участник
- Читатель
- администратор доступа пользователей;
- Читатель выставления счетов
- Соадминистратор
- Администратор служб

Чтобы скачать счет, выполните следующие действия:

1.  Войдите на [портал Azure](https://portal.azure.com) в качестве администратора учетной записи для подписки.
1.  Выполните поиск по фразе **Управление затратами + выставление счетов**.  
1.  Выберите **Счета** в левой части.
1.  Выберите подписку Azure и щелкните **Доступ к счету**.  
    [![Снимок экрана, показывающий выбор параметра "Доступ к счету"](./media/download-azure-invoice/cmb-select-access-to-invoice.png)](./media/download-azure-invoice/cmb-select-access-to-invoice-zoomed-in.png#lightbox)
1.  Выберите **Вкл.** , а затем — **Сохранить** в верхней части страницы.  
    ![Снимок экрана, показывающий выбор параметра "Вкл." для доступа к счету](./media/download-azure-invoice/cmb-access-to-invoice.png)

## <a name="get-mosp-subscription-invoice-in-email"></a>Получение счета подписки MOSP по электронной почте

Чтобы принимать счета по электронной почте, вам должна быть назначена роль администратора учетной записи подписки или плана поддержки. Получение счетов по электронной почте доступно только для подписок и планов поддержки. Эта возможность недоступна для резервирования или покупок в Azure Marketplace. Если вы имеете такую возможность, можно добавить дополнительных получателей, которые также будут получать счета по электронной почте.

1.  Войдите на [портал Azure](https://portal.azure.com).
1.  Выполните поиск по фразе **Управление затратами + выставление счетов**.  
1.  Выберите **Счета** в левой части.
1.  Выберите подписку Azure или план поддержки, а затем щелкните **Отправить счет по электронной почте**.  
    [![Снимок экрана, на котором показан вариант "Профиль выставления счетов" в списке](./media/download-azure-invoice/cmb-email-invoice.png)](./media/download-azure-invoice/cmb-email-invoice-zoomed-in.png#lightbox)
1. Выберите **Согласиться** и примите условия.  
    ![Снимок экрана, на котором отображается процесс предоставления согласия (шаг 2)](./media/download-azure-invoice/invoice-article-step02.png)
1. Счет будет отправлен на предпочтительный адрес электронной почты для связи. Вы можете обновить адрес электронной почты в разделе с [контактной информацией](https://portal.azure.com/#blade/Microsoft_Azure_GTM/ModernBillingMenuBlade/Properties) учетной записи выставления счетов.  
    ![Снимок экрана, на котором отображается процесс предоставления согласия (шаг 3)](./media/download-azure-invoice/invoice-article-step03-verify-email.png)

## <a name="share-your-mosp-invoices-with-others-by-email"></a>Предоставление общего доступа к счетам MOSP по электронной почте

Каждый месяц вам может потребоваться предоставлять общий доступ к счетам подписки или плана поддержки сотрудникам из отдела бухгалтерского учета. Вам также может понадобиться отправить счета на другой адрес электронной почты.

1. Выполните действия из раздела [Получение счета подписки MOSP по электронной почте](#get-mosp-subscription-invoice-in-email) и выберите **Выбор получателей**.  
    ![Снимок экрана, на котором показан выбор пользователем параметра "Выбор получателей"](./media/download-azure-invoice/invoice-article-step03.png)
1. Введите адрес электронной почты, а затем выберите **Добавить получателя**. Вы можете добавить несколько адресов электронной почты.  
    ![Снимок экрана, показывающий добавление дополнительных получателей пользователем](./media/download-azure-invoice/invoice-article-step04.png)
1. Добавив все адреса электронной почты, выберите **Готово** в нижней части экрана.

## <a name="invoices-for-mca-and-mpa-billing-accounts"></a>Счета для учетных записей выставления счетов MCA и MPA

Учетная запись выставления счетов MCA создается, когда ваша организация работает с представителем корпорации Майкрософт для подписания MCA. В некоторых регионах клиенты, которые зарегистрировались на веб-сайте Azure для получения [учетной записи с оплатой по мере использования](https://azure.microsoft.com/offers/ms-azr-0003p/) или [бесплатной учетной записи Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), могут также использовать учетную запись выставления счетов для MCA. Дополнительные сведения см. в статье [Начало работы с учетной записью выставления счетов клиентского соглашения Майкрософт](../understand/mca-overview.md).

Учетная запись выставления счетов MPA создается для того, чтобы поставщики облачных решений могли управлять выставлением счетов для своих клиентов по новой коммерческой программе. Чтобы управлять учетной записью выставления счетов на портале Azure, у партнера должен быть хотя бы один клиент, использующий [план Azure](https://docs.microsoft.com/partner-center/purchase-azure-plan). Дополнительные сведения см. в статье [Начало работы с учетной записью выставления счетов Соглашения с партнером Майкрософт](../understand/mpa-overview.md).

В начале месяца для каждого профиля выставления счетов в вашей учетной записи создается счет за месяц. Счет содержит соответствующие расходы на все подписки Azure и другие покупки за предыдущий месяц. Например, Джон создал *подписку Azure 01* 5 марта, а *подписку Azure 02* 10 марта. Он приобрел подписку на *поддержку Azure 01* 28 марта с помощью *профиля выставления счетов 01*. Джон получит общий счет в начале апреля, который будет содержать расходы для подписок и плана поддержки Azure.

##  <a name="download-an-mca-or-mpa-billing-profile-invoice"></a>Скачивание счета профиля выставления счетов MCA или MPA

Чтобы скачать счет профиля выставления счетов на портале Azure, вам должна быть назначена роль владельца, участника, читателя или менеджера счетов. Пользователи с ролью владельца, участника или читателя в учетной записи выставления счетов могут скачивать счета всех профилей выставления счетов в учетной записи.

1.  Войдите на [портал Azure](https://portal.azure.com).
1.  Выполните поиск по фразе **Управление затратами + выставление счетов**.  
1. Выберите **Счета** в левой части.
    [![Снимок экрана со страницей счетов для учетной записи выставления счетов MCA](./media/download-azure-invoice/mca-billing-profile-invoices.png)](./media/download-azure-invoice/mca-billing-profile-invoices-zoomed.png#lightbox)
1. В таблице счетов выберите счет, который необходимо скачать.
1. В верхней части страницы выберите **Скачать PDF-файл счета**.  
    ![Снимок экрана с параметром "Скачать PDF-файл счета"](./media/download-azure-invoice/mca-billing-profile-download-invoice.png)
1. Вы также можете скачать CSV-файл с данными об израсходованных объемах с разбивкой по дням и ориентировочной величиной расходов, выбрав **Скачать сведения об использовании Azure**. Подготовка CSV-файла может занять несколько минут.

## <a name="get-your-billing-profiles-invoice-in-email"></a>Получение счета для профиля выставления счетов по электронной почте

Чтобы обновить параметры выставления счетов по электронной почте, вам должна быть назначена роль владельца или участника в профиле или учетной записи выставления счетов. Все пользователи с назначенными ролями владельца, участника, читателя или менеджера счетов в профиле выставления счетов получат свой счет по электронной почте. 

1.  Войдите на [портал Azure](https://portal.azure.com).
1.  Выполните поиск по фразе **Управление затратами + выставление счетов**.  
1.  Выберите **Счета** в левой части, а затем в верхней части страницы щелкните **Отправить счет по электронной почте**.  
    [![Снимок экрана со страницей счетов для учетной записи выставления счетов MCA](./media/download-azure-invoice/mca-billing-profile-select-email-invoice.png)](./media/download-azure-invoice/mca-billing-profile-select-email-invoice-zoomed.png#lightbox)
1.  Выберите необходимый профиль выставления счетов (если у вас их несколько), а затем щелкните **Согласиться**.  
    ![Снимок экрана со страницей счетов для учетной записи выставления счетов MCA](./media/download-azure-invoice/mca-billing-profile-email-invoice.png)
1.  Нажмите кнопку **Обновить**.

## <a name="give-others-access-to-mca-or-mpa-invoices"></a>Предоставление общего доступа к счетам MCA или MPA

Вы можете предоставить другим пользователям доступ к просмотру, скачиванию и оплате счетов, назначив им роль менеджера счетов для профиля выставления счетов MCA или MPA. Если вы согласились получать счета по электронной почте, эти пользователи также получат счета по электронной почте.

1. Войдите на [портал Azure](https://portal.azure.com).
1. Выполните поиск по фразе **Управление затратами + выставление счетов**.  
1. Выберите раздел **Профили выставления счетов** слева. В списке профилей выставления счетов выберите профиль выставления счетов, которому требуется назначить роль "менеджера счетов".  
   ![Снимок экрана, на котором показан вариант "Профиль выставления счетов" в списке](./media/download-azure-invoice/mca-select-profile-zoomed-in.png)
1. Выберите **Система управления идентификацией и доступом (IAM)** с левой стороны, а затем **Добавить** в верхней части страницы.  
    ![Снимок экрана, на котором показана страница контроля доступа](./media/download-azure-invoice/mca-select-access-control-zoomed-in.png)
1. В раскрывающемся списке "Роль" выберите **Менеджер счетов**. Введите адрес электронной почты пользователя, которому необходимо предоставить доступ. Чтобы присвоить роль, щелкните **Сохранить**.  
   ![Снимок экрана, показывающий добавление пользователя в качестве менеджера счетов](./media/download-azure-invoice/mca-added-invoice-manager.png)

## <a name="why-you-might-not-see-an-invoice"></a><a name="noinvoice"></a> Почему вы можете не видеть счет

Это может происходить по нескольким причинам:

- прошло менее 30 дней со дня оформления подписки Azure; 
- Azure выставляет счет через несколько дней после окончания периода выставления счетов. Таким образом, счет может быть еще не создан.
- у вас нет разрешения на просмотр счетов.
    Если у вас есть учетная запись выставления счетов MCA или MPA, чтобы просмотреть счета, необходимо иметь роль владельца, участника, читателя или менеджера счетов для профиля выставления счетов или роль владельца, участника, читателя для учетной записи выставления счетов. Для других подписок вы можете не видеть счета, если не являетесь администратором учетной записи. Дополнительные сведения о доступе к сведениям о выставлении счетов см. в статье [Управление доступом для выставления счетов Azure с помощью ролей](../manage/manage-billing-access.md);
- Если у вас есть учетная запись выставления счетов MOSP и вы зарегистрировали бесплатную учетную запись Azure или подписку с ежемесячным кредитом, вы получите счет только при превышении ежемесячной суммы кредита. Если у вас есть учетная запись выставления счетов для MCA или MPA, вы всегда получаете счет.
