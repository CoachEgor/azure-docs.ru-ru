# <a name="how-to-pay-your-bill-for-microsoft-azure"></a>Как оплатить счет за Microsoft Azure

Эта статья относится к клиентам с Клиентским соглашением Майкрософт, которые подписались на Azure с помощью веб-сайта Azure. [Проверка доступа к Клиентскому соглашению Майкрософт](#check-access-to-a-microsoft-customer-agreement).

Метод оплаты по умолчанию для профиля выставления счетов можно выбрать из оплаты кредитной картой, либо банковским переводом. Если у вас есть кредиты Azure, они автоматически применяются к вашему счету за каждый расчетный период. Чтобы получить право на оплату с помощью чека или банковского перевода, см. статью [Оплата по счету](../manage/pay-by-invoice.md).

Если ваш метод оплаты по умолчанию — чек или банковский перевод, вы можете отправить платеж по адресу, указанному в нижней части счета. Кроме того, на портале Azure вы можете платить онлайн с помощью кредитной карты.

Если ваш метод оплаты по умолчанию — кредитная карта, при создании счета будет автоматически взиматься плата за каждый расчетный период. Вы также можете платить онлайн на портале Azure, если хотите использовать другую кредитную карту или если автоматическая оплата не удалась.

## <a name="pay-now-in-the-azure-portal"></a>Оплатите сейчас на портале Azure

Для оплаты счетов на портале Azure, необходимо быть владельцем профиля выставления счетов, участником или менеджером по счетам.

1. Войдите на [портал Azure](https://portal.azure.com).
1. Выполните поиск по **Управление затратами + выставление счетов**.
1. В меню слева в разделе "Выставление счетов" выберите **Счета**.
1. Если какие-либо из ваших счетов подлежат оплате или просрочены, вы увидите для этого счета синюю ссылку **Платить сейчас**. Щелкните **Платить сейчас**.
1. В новой колонке щелкните **Выбрать метод оплаты**, чтобы выбрать существующую кредитную карту или добавить новую.
1. Выбрав метод оплаты, нажмите кнопку **Платить сейчас**.

Состояние счета будет обновлено на "Оплачено" в колонке счета.
