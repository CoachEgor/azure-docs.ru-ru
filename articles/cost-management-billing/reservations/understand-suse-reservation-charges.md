# <a name="azure-software-plan-discount"></a>Скидка на план программного обеспечения Azure

Планы программного обеспечения Azure для SUSE и RedHat — это резервирования, которые применяются к развернутым виртуальным машинам. Скидка на план программного обеспечения применяется к использованию программного обеспечения развернутых виртуальных машин, соответствующих резервированию.

При завершении работы виртуальной машины скидка автоматически применяется к другой подходящей виртуальной машине, если она доступна. План программного обеспечения охватывает затраты на запуск программного обеспечения на виртуальной машине. Другие расходы, такие как вычисления, хранение и сетевое подключение, взимаются отдельно.

Чтобы купить подходящий план, необходимо изучить использование виртуальной машины и количество виртуальных ЦП на этих виртуальных машинах. Используйте следующие разделы, чтобы определить, какой план следует купить, на основе данных об использовании.

## <a name="how-reservation-discount-is-applied"></a>Применение скидки на резервирование

Скидка на резервирование *сгорает, если не используется*. Поэтому, если у вас не существует подходящих ресурсов в течение некоторого часа, вы потеряете количество ресурсов, зарезервированных на этот час. Вы не можете переносить неиспользуемые зарезервированные часы.

При завершении работы ресурса скидка на резервирование автоматически переносится на другой подходящий ресурс в указанной области резервирования. Если в указанной области не найдены подходящие ресурсы, зарезервированные часы *теряются*.

## <a name="review-redhat-vm-usage-before-you-buy"></a>Общие сведения об потреблении виртуальной машины RedHat перед покупкой

Получите название продукта из данных об использовании и купите план RedHat с тем же типом и размером.

Например, если вы используете продукт **Red Hat Enterprise Linux — лицензию на виртуальную машину с 1–4 виртуальными ЦП**, вы должны приобрести **Red Hat Enterprise Linux** для **виртуальной машины с 1–4 виртуальными ЦП**.

## <a name="review-suse-vm-usage-before-you-buy"></a>Общие сведения о потреблении виртуальной машины SUSE перед покупкой

Получите название продукта из данных об использовании и купите план SUSE с тем же типом и размером.

Например, если вы используете продукт **SUSE Linux Enterprise Server (цен. категория "Приоритетный") — поддержку виртуальной машины с 2–4 виртуальными ЦП**, вы должны приобрести **SUSE Linux Enterprise Server (цен. категория "Приоритетный")** для **2–4 виртуальных ЦП**.

## <a name="discount-applies-to-different-vm-sizes-for-suse-plans"></a>Скидка применяется к различным размерам виртуальных машин для планов SUSE

Как и зарезервированные экземпляры виртуальных машин, план SUSE предлагает гибкость размера экземпляра. Это означает, что ваша скидка применяется даже в случае развертывания виртуальной машины с разным количеством виртуальных процессоров. Скидка применяется к виртуальным машинам разных размеров в рамках плана программного обеспечения.

Сумма скидки зависит от коэффициента, указанного в следующих таблицах. Коэффициент сравнивает относительный охват для каждой метрики в этой группе. Коэффициент зависит от виртуальных ЦП виртуальной машины. Используйте значение коэффициента для вычисления того, на сколько экземпляров виртуальных машин можно получить скидку по плану SUSE Linux.

Например, если вы покупаете план для SUSE Linux Enterprise Server для HPC (цен. категория "Приоритетный") для виртуальной машины с 3 или 4 виртуальными ЦП, то коэффициент для этого резервирования — 2. Скидка покрывает стоимость программного обеспечения SUSE для:

- 2 развернутых виртуальных машин с 1 или 2 виртуальными ЦП;
- 1 развернутой виртуальной машины с 3 или 4 виртуальными ЦП;
- примерно 77 % (0,77) виртуальной машины с 5 или более виртуальными ЦП.

Коэффициент для 5 или более виртуальных ЦП составляет 2,6. Так что резервирование для SUSE с виртуальной машиной с 5 или более виртуальными ЦП покрывает только часть затрат на программное обеспечение и составляет около 77 %.
