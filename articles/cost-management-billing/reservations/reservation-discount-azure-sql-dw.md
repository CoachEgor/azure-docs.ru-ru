# <a name="how-reservation-discounts-apply-to-azure-synapse-analytics"></a>Применение скидок на резервирование к Azure Synapse Analytics

После приобретения зарезервированной емкости Azure Synapse Analytics скидка на резервирование автоматически применяется к подготовленным к работе экземплярам, которые существуют в этом регионе. Скидка на резервирование применяется к потреблению, измеряемому в единицах cDWU Azure Synapse Analytics. Служба хранилища и работа в сетевые подключения оплачиваются по тарифу «по мере использования».

## <a name="reservation-discount-application"></a>Применение скидки на резервирование

Скидка на зарезервированную емкость Azure Synapse Analytics применяется к выполняющимся хранилищам на почасовой основе. Если вы не развернули хранилище в течение часа, то резервная мощность будет потрачена впустую на этот час. Она не переносится.

После покупки приобретенное резервирование сопоставляется с показателями потребления Azure Synapse Analytics в соответствии с выполняющимися хранилищами в любой момент времени. Если для нескольких хранилищ работа завершена, то скидки на резервирование автоматически распространяются на любые другие подходящие хранилища.

Для хранилищ, которые не работают в течение полного часа, резервирование автоматически применяется к другим соответствующим экземплярам в этот час.

## <a name="discount-examples"></a>Примеры скидок

В следующих примерах показано, как применяется скидка на зарезервированную емкость Azure Synapse Analytics в зависимости от развертываний.

- **Пример 1**: Ви покупаете 5 единиц с резервной мощностью 100 cDWU. Вы запускаете экземпляр DW1500c Azure Synapse Analytics на час. В этом случае потребление выдано для 15 единиц с потреблением по 100 cDWU. Скидка на резервирование распространяется на 5 использованных единиц. Плата взимается по фактической ставке за оставшиеся 10 использованных единиц с потреблением по 100 cDWU. Другими словами, частичное покрытие может охватывать несколько резервирований.

- **Пример 2**. Ви покупаете 5 единиц с резервной мощностью 100 cDWU. Вы запускаете два экземпляра DW100c Azure Synapse Analytics на час. В этом случае два события потребления выдано для 1 единицы с потреблением 100 cDWU. На оба события потребления распространяется скидка на резервирование. Оставшиеся 3 блока по 100 cDWU резервной мощности тратятся впустую и не переносятся для использования в будущем. Другими словами, одно резервирование можно сопоставить с несколькими экземплярами Azure Synapse Analytics.

- **Пример 3**: Ви покупаете 1 единицу с резервной мощностью 100 cDWU. Вы запускаете два экземпляра DW100c Azure Synapse Analytics. Каждый работает по 30 минут. В этом случае, скидка на резервирование распространяется на оба события потребления. Оплата за потребление не взимается при использовании тарифов «оплаты по мере использования».
