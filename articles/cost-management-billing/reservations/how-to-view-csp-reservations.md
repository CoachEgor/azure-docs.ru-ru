# <a name="view-azure-reservations-as-a-cloud-solution-provider-csp"></a>Просмотр резервирований Azure в рамках программы "Поставщик облачных решений" (CSP)

У участников программы "Поставщик облачных решений" есть доступ к резервированиям, приобретенным для клиентов. Используйте приведенные ниже сведения для просмотра резервирований на портале Azure.

1. Обратитесь к глобальному администратору, чтобы вас добавили как **агента по администрированию** в вашем арендаторе.
    Этот параметр доступен для глобальных администраторов в Центре партнеров. Параметр можно найти в разделе **Параметры** (значок шестеренки в правом верхнем углу страницы) > **Управление пользователями**.  
1. Получив права агента по администрированию, перейдите на портал Azure по ссылке **Admin on Behalf Of** (Администратор от имени).
1. Перейдите к разделу**Клиенты** в Центре партнеров, разверните раздел сведений о клиенте и выберите элемент **Портал управления Microsoft Azure**.
1. На портале Azure перейдите к разделу **Резервирование**.

> [!NOTE]
> Гостевая учетная запись в арендаторе клиента препятствует просмотру резервирований. Если у вас есть гостевой доступ, его необходимо удалить из арендатора. Разрешение агента по администрированию не переопределяет гостевой доступ.

- Чтобы удалить гостевой доступ в Центре партнеров, перейдите к разделу **Моя учетная запись** >  **[Организации](https://myaccount.microsoft.com/organizations)** и выберите **Покинуть организацию**.

Кроме того, можно попросить другого пользователя, у которого есть доступ к резервированию, добавить вашу гостевую учетную запись в заказ на резервирование.
