# <a name="why-is-my-azure-for-students-subscription-disabled-and-how-do-i-reactivate-it"></a>Почему подписка "Azure для учащихся" отключена и как активировать ее повторно?

Подписка "Azure для учащихся" может быть отключена, поскольку на счете закончились все деньги, истек срок действия подписки или подписка была случайно отменена. Выберите подходящую ситуацию и узнайте, как активировать подписку повторно.

## <a name="youve-used-all-of-your-credit"></a>Были использованы все деньги на счете

Учетная запись "Azure для учащихся" предоставляет деньги на счете в Azure на сумму 100 долларов США и 12 месяцев использования бесплатных служб в ограниченном количестве. За использование любых служб, кроме бесплатных, из счета будут вычитаться деньги. Azure отключит службы и подписку, как только закончатся деньги на счете. Чтобы продолжить использование служб Azure, необходимо обновить подписку до подписки с оплатой по мере использования, обратившись в службу [поддержки Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade). После обновления ваша подписка по-прежнему имеет доступ к бесплатным службам в течение 12 месяцев с даты регистрации. Плата взимается только за использование платных служб и ресурсов.

Вы можете проверить оставшиеся средства на счете в Azure на [портале спонсорского предложения Microsoft Azure](https://www.microsoftazuresponsorships.com/balance).

1. Войдите в учетную запись, используя учетные данные "Azure для учащихся".
2. На странице баланса приводятся сведения о деньгах, которые были использованы и которые остались на счете. Дату истечения срока действия средств на счете можно найти под диаграммой "кредит".  

    ![На снимке экрана показана информация о средствах на счете для подписки "Azure для учащихся"](./media/azurestudents-subscription-disabled/azurestudents-credit-balance.png)

3. Чтобы проверить использование служб, которые потребляют средства со счета Azure, в панели навигации слева выберите **Использование**, а затем выберите подписку "Azure для учащихся".

    ![На снимке экрана показана информация об использовании средств со счета для подписки "Azure для учащихся"](./media/azurestudents-subscription-disabled/azurestudents-credit-usage.png)

Таблица содержит следующие столбцы:

* **Service Name** (Имя службы): служба, которая использует средства со счета Azure.
* **Service Resource** (Ресурс службы): единицы измерения, используемые службой.
* **Spend** (Расходы): объем кредита (долл. США), затраченный на службу.

## <a name="your-credit-has-expired"></a>Истек срок действия кредита

Кредит для подписки "Azure для учащихся" истечет по окончании 12 месяцев. После того, как все средства кредита будут израсходованы, Azure отключит подписку. Чтобы продолжить использование служб Azure, необходимо обновить подписку до подписки с оплатой по мере использования, обратившись в [службу поддержки Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade). После обновления Azure выставляет счета за использование любых служб согласно тарифам оплаты по мере использования.

## <a name="youve-accidentally-canceled-your-subscription"></a>Случайная отмена подписки

Если вы случайно отменили подписку "Azure для учащихся", ее можно повторно активировать, обратившись в службу [поддержки Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade). После повторной активации у вас останется доступ к оставшимся средствам на счете и бесплатным службам на протяжении 12 месяцев с даты регистрации.
