# <a name="billing-accounts-and-scopes-in-the-azure-portal"></a>Счета и области выставления счетов в портал Azure

Учетная запись выставления счетов создается при регистрации в Azure. Вы используете свою учетную запись выставления счетов для управления счетами, платежами и отслеживания расходов. Вы можете иметь доступ к нескольким учетным записям выставления счетов. Например, вы могли зарегистрироваться в Azure для своих личных проектов. Вы также могли получить доступ через Соглашение Enterprise Agreement вашей организации или Клиентское соглашение Майкрософт. Для каждого из этих сценариев у вас будет отдельная учетная запись выставления счетов.

Портал Azure поддерживает следующие типы учетных записей выставления счетов:

- **Программа Microsoft Online Services**. учетная запись выставления счетов для программы Microsoft Online Services создается при регистрации в Azure с помощью веб-сайта Azure. Например, при регистрации с [бесплатной учетной записи Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), [учетной записью с оплатой по мере использования](https://azure.microsoft.com/offers/ms-azr-0003p/) или как [подписчик Visual Studio](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/).

- **Соглашение Enterprise**: счет выставления счетов для Соглашение Enterprise создается, когда ваша организация подписывает [Соглашение Enterprise (EA)](https://azure.microsoft.com/pricing/enterprise-agreement/) для использования Azure.

- **Соглашение клиента Майкрософт**. счет выставления счетов для соглашения клиента Майкрософт создается, когда ваша организация работает с представителями корпорации Майкрософт для подписания соглашения клиента Майкрософт. Некоторые клиенты в выбранных регионах, которые подписываются через веб-сайт Azure для [учетной записи с тарифами с оплатой по мере](https://azure.microsoft.com/offers/ms-azr-0003p/) использования или [бесплатной учетной записью Azure](https://azure.microsoft.com/offers/ms-azr-0044p/) , могут также иметь счет на выставление счетов для соглашения клиента Майкрософт. Дополнительные сведения см. в статье [Get started with your billing account for Microsoft Customer Agreement](../understand/mca-overview.md) (Учетные записи выставления счетов для клиентского соглашения Майкрософт).

- **Соглашение для партнеров Майкрософт**. счет выставления счетов для партнерского соглашения Майкрософт создается для партнеров поставщика облачных решений (CSP) для управления клиентами в новом коммерческом опыте. Партнерам необходимо иметь по крайней мере одного клиента с [планом Azure](https://docs.microsoft.com/partner-center/purchase-azure-plan) для управления учетной записью выставления счетов в портал Azure. Дополнительные сведения см. в [статье Начало работы с учетной записью выставления счетов для партнерского соглашения Майкрософт](../understand/mpa-overview.md).

Чтобы определить тип учетной записи выставления счетов, см. раздел [Проверка типа учетной записи для выставления счетов](#check-the-type-of-your-account).

## <a name="scopes-for-billing-accounts"></a>Области для учетных записей выставления счетов
Область — это узел в учетной записи выставления счетов, который используется для просмотра и управления выставлением счетов. Здесь вы управляете данными о выставлении счетов, платежами, счетами и проводите общее управление учетными записями.

### <a name="microsoft-online-services-program"></a>Программа Microsoft Online Services

 ![Снимок экрана, на котором показана иерархия MOSP](./media/view-all-accounts/mosp-hierarchy.png)

|Область действия  |Определение  |
|---------|---------|
|учетная запись выставления счетов;     | Представляет соглашение, которое клиент принимает для использования Azure. Он содержит одну или несколько подписок.  |
|Subscription     |  Представляет группирование ресурсов Azure. В этой области создается счет. Другие сведения о выставлении счетов, такие как способы оплаты и адрес использования, связаны с этой областью.|

### <a name="enterprise-agreement"></a>Соглашение Enterprise

![Снимок экрана, на котором показана иерархия EA](./media/view-all-accounts/ea-hierarchy.png)

|Область действия  |Определение  |
|---------|---------|
|учетная запись выставления счетов;    | Представляет регистрацию на Соглашение Enterprise. Он содержит один или несколько отделов и учетных записей. В этой области создается счет. |
|отдел;     |  Необязательное группирование учетных записей для сегментирования затрат на логические группы и установка бюджета.     |
|Учетная запись     |  Представляет одного владельца учетной записи. Владельцы учетных записей имеют разрешения на создание подписок Azure и управление ими, которые выставляются при регистрации. |

### <a name="microsoft-customer-agreement"></a>Клиентское соглашение Майкрософт

![Снимок экрана, на котором показана иерархия MCA](./media/view-all-accounts/mca-hierarchy.png)

|Область действия  |Задачи  |
|---------|---------|
|учетная запись выставления счетов;     |   Представляет соглашение, которое клиент принимает на использование продуктов и служб Майкрософт. Он содержит один или несколько профилей выставления счетов. |
|Профиль выставления счетов     |   Представляет счет и связанные с ним сведения о выставлении счетов, такие как способы оплаты и адрес для выставления счетов. Он содержит один или несколько разделов счета. |
|Раздел счета     |   Представляет группирование затрат в счете. Подписки Azure и другие покупки, такие как Azure Marketplace и продукты источника приложений, связаны с этой областью.    |

### <a name="microsoft-partner-agreement"></a>Соглашение для партнеров Майкрософт

![Снимок экрана, на котором показана иерархия MPA](./media/view-all-accounts/mpa-hierarchy.png)

|Область действия  |Задачи  |
|---------|---------|
|учетная запись выставления счетов;     |   Представляет соглашение партнера для управления продуктами и службами Майкрософт клиентов в новом коммерческом опыте. Он содержит один или несколько профилей выставления счетов и клиентов.   |
|Профиль выставления счетов     |   Представляет счет для валюты.     |
|Customer    |   Представляет клиента для партнера поставщика облачных решений (CSP).  Подписки Azure и другие покупки, такие как Azure Marketplace и продукты источника приложений, связаны с этой областью.  |
|Reseller    |   Торговый посредник, предоставляющий услуги клиенту. Это необязательное поле для подписки. оно применимо только к косвенным поставщикам в двухуровневой модели CSP.     |

## <a name="switch-billing-scope-in-the-azure-portal"></a>Переключение области выставления счетов на портал Azure

1. Войдите на [портал Azure](https://portal.azure.com).

2. Выполните поиск по фразе **Управление затратами + выставление счетов**.

   ![Снимок экрана, показывающий поиск на портале Azure](./media/view-all-accounts/billing-search-cost-management-billing.png)

3. На странице Обзор выберите **параметр область**.

   ![Снимок экрана, показывающий области выставления счетов](./media/view-all-accounts/overview-select-scopes.png)

   > [!Note]
    >
    > Вы не увидите область коммутатора, если у вас есть доступ только к одной области.

4. Выберите область для просмотра сведений.

   ![Снимок экрана, показывающий области выставления счетов](./media/view-all-accounts/list-of-scopes.png)
