# <a name="resolve-past-due-balance-for-your-azure-subscription"></a>Устранение проблемы с задолженностью по подписке Azure

Эта статья предназначена для клиентов, которые зарегистрировались в Azure через Интернет, используя кредитную карту и имеют учетную запись выставления счетов в программе Microsoft Online Services. Узнайте, как [проверить тип учетной записи выставления счетов](#check-the-type-of-your-account). Если у вас есть учетная запись выставления счетов по клиентскому соглашению Майкрософт, см. статью об [оплате счета за Microsoft Azure](../understand/pay-bill.md).

Если ваш платеж не получен или мы не можем обработать этот платеж, вы получите сообщение электронной почты и увидите оповещение на портале Azure о том, что подписка просрочена. Если ваш метод оплаты по умолчанию — кредитная карта, [администратор учетной записи](billing-subscription-transfer.md#whoisaa) может покрыть неоплаченные платежи на портале Azure. Если вы оплачиваете по выставляемому счету (чеку / банковскому переводу), то отправьте платеж по тем реквизитам, которые указаны внизу счета.

> [!IMPORTANT]
> * Если у вас несколько подписок, использующих одну и ту же кредитную карту, и все они просрочены, необходимо одновременно оплатить всю задолженность.
> * Кредитная карта, которую вы используете для погашения платежей, станет новым способом оплаты по умолчанию для всех подписчиков, которые использовали неудачный способ оплаты.

## <a name="resolve-past-due-balance-in-the-azure-portal"></a>Решение проблемы просроченных платежей на портале Azure

1. Войдите на [портал Azure](https://portal.azure.com) в качестве администратора учетной записи.
1. Выполните поиск по фразе **Управление затратами + выставление счетов**.
1. Выберите просроченную подписку на странице **Обзор**.
1. На странице **Обзор подписки** щелкните баннер с красной датой, чтобы покрыть задолженность.
    > [!NOTE]
    > Если вы не являетесь администратором учетной записи, вы не сможете покрыть задолженность.
1. На странице **Покрыть задолженность** щелкните **Выберите метод оплаты**.
    ![Снимок экрана, на котором показана ссылка на выбор метода оплаты](./media/resolve-past-due-balance/settle-balance-screen.png)

1. Справа откроется новая колонка, где нужно выбрать кредитную карту в раскрывающемся списке или добавить новую с помощью синей ссылки **Добавить новый метод оплаты**. Эта кредитная карта станет активным методом оплаты для всех подписчиков, использующих неудачный метод оплаты.
     > [!NOTE]
     > * Общая сумма непогашенной задолженности отражает непогашенные платежи по всем службам Microsoft, использующим неудачный способ оплаты.
     > * Если выбранный метод оплаты также имеет непревзойденные расходы на службы Майкрософт, это будет отражено в общей задолженности. Вы также обязаны оплатить эти расходы.
1. Нажмите кнопку **Оплата**.

## <a name="troubleshoot-declined-credit-card"></a>Устранение проблем с отклонением кредитной карты

Если платеж по карте был отклонен вашим финансовым учреждением, обратитесь в это финансовое учреждение, чтобы решить проблему. Обратитесь в банк, чтобы убедиться в следующем:
- Международные транзакции включены на карте.
- На карте есть достаточный кредитный лимит или средства для покрытия задолженности.
- На карте разрешены регулярные платежи.

## <a name="not-getting-billing-email-notifications"></a>Платежные уведомления не приходят по электронной почте?

Если вы являетесь администратором учетной записи, [проверьте, какой адрес электронной почты используется для уведомлений](change-azure-account-profile.md). Мы рекомендуем использовать адрес электронной почты, который будет регулярно проверяться. Если адрес электронной почты указан правильно, проверьте папку спама.

## <a name="if-i-forget-to-pay-what-happens"></a>Что случится, если я забуду заплатить?

Служба отменяется, а ресурсы становятся недоступными. Данные Azure будут удалены через 90 дней после прекращения услуги. Дополнительные сведения см. в разделе [What happens to your data if you leave the service](https://go.microsoft.com/fwLink/p/?LinkID=822930&clcid=0x409) (Что происходит с данными, когда вы прекращаете использовать службу).

Если вы знаете, что платеж обработан, но подписка по-прежнему отключена, обратитесь в [службу поддержки Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade).
