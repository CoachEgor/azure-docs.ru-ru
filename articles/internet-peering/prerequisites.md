# <a name="prerequisites-to-set-up-peering-with-microsoft"></a>Предварительные требования для настройки пиринга с Майкрософт

Убедитесь, что предпосылки, приведенные ниже, будут выполнены до запроса на новый пиринг или преобразуйте устаревшее пиринг в ресурс Azure.

## <a name="azure-related-prerequisites"></a>Предпосылки, связанные с Azure
* **Учетная запись Microsoft Azure:** Если у вас нет учетной записи Microsoft Azure, создайте [учетную запись Microsoft Azure.](https://azure.microsoft.com/free) Для настройки пиринговой подписки Microsoft Azure требуется действительная и активная подписка Microsoft Azure, поскольку пиринги моделируются в качестве ресурсов в подписях Azure. Важно отметить следующее.
    * Типы ресурсов Azure, используемые для настройки пиринга, всегда свободны для продуктов Azure, т.е. с вас не взимается плата за создание учетной записи Azure или создание подписки или доступ к ресурсам Azure **PeerAsn** и **Peering** для настройки пиринга. Это не следует путать с пиринговым соглашением для прямого пиринга между вами и корпорацией Майкрософт, условия которого явно обсуждаются с нашей командой пиринга. Обратитесь в [корпорацию Майкрософт,](mailto:peering@microsoft.com) если возникают какие-либо вопросы в этой связи.
    * Вы можете использовать ту же подписку Azure для доступа к другим продуктам Azure или облачным службам, которые могут быть бесплатными или платными. При доступе к платному продукту вы будете нести расходы.
    * При создании новой учетной записи Azure и/или подписке вы можете иметь право на бесплатный кредит Azure в течение пробного периода, который вы можете использовать для пробных служб Azure Cloud. Если вы заинтересованы, посетите [учетную запись Microsoft Azure](https://azure.microsoft.com/free) для получения дополнительной информации.

* **Ассоциированный Peer ASN:** Прежде чем запросить для пиринга, сначала свяжите свой ASN и свяжитесь с вашей подпиской. Следуйте инструкциям в [подписке Associate Peer ASN к Azure.](howto-subscription-association-powershell.md)

## <a name="other-prerequisites"></a>Другие предпосылки
* **Профиль PeeringDB:** Ожидается, что у сверстников будет полный и актуальный профиль на [PeeringDB.](https://www.peeringdb.com) Мы используем эту информацию в нашей системе регистрации для проверки данных узла, таких как информация NOC, техническая контактная информация, их присутствие в пиринговых объектах и т.д.
