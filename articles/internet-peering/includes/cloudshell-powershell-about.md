Для запуска командлетов можно использовать бесплатную интерактивную оболочку Azure Cloud Shell. Она включает предварительно установленные общие инструменты Azure и настроена для использования с вашей учетной записью. Нажмите кнопку **Копировать**, чтобы скопировать код. Вставьте его в Cloud Shell и нажмите клавишу ВВОД, чтобы запустить код. Cloud Shell можно запустить разными способами:


|  |   |
|-----------------------------------------------|---|
| Откройте Cloud Shell в браузере. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Нажмите кнопку **Cloud Shell** в меню в правом верхнем углу портала. | [![Cloud Shell на портале](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Если вы не хотите использовать Azure CloudShell, вместо этого можно установить PowerShell локально. Если вы решили установить и использовать PowerShell локально, обязательно установите последнюю версию командлетов PowerShell диспетчер ресурсов. Командлеты PowerShell часто обновляются, и вам, как правило, необходимо обновить командлеты PowerShell, чтобы получить последние функциональные возможности функций, которые могут возникать при сбое.

Чтобы найти версию PowerShell, которая выполняется локально, используйте командлет Get-Module-ListAvailable AZ. Если необходимо выполнить обновление, см. статью [об установке модуля Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps). Подробнее: [Установка и настройка Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/overview).

Если вы используете PowerShell для macOS, выполните действия, описанные в статье [Установка PowerShell Core на macOS.](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)
