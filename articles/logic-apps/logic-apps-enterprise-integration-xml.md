# <a name="xml-messages-and-flat-files-in-azure-logic-apps-with-enterprise-integration-pack"></a>XML-сообщения и неструктурированные файлы в Azure Logic Apps с Пакетом интеграции Enterprise

В [приложениях Логики Azure](logic-apps-overview.md)вы можете обрабатывать XML-сообщения, которые отправляете и получаете, используя пакет интеграции enterprise Pack. Если вы использовали BizTalk Server, пакет корпоративной интеграции предоставляет аналогичные возможности для преобразования и проверки сообщений, работы с плоскими файлами и даже использования XPath для обогащения или извлечения определенных свойств из сообщения. Если вы новичок в этом пространстве, эти функции расширяют процесс обработки сообщений в рабочем процессе приложения логики. Например, если у вас есть сценарий «бизнес-бизнес» (B2B) и работаете с определенными схемами XML, можно использовать пакет интеграции предприятий для улучшения процесса обработки этих сообщений компанией.

Например, пакет интеграции предприятий включает в себя следующие возможности:

* [Проверка XML:](logic-apps-enterprise-integration-xml-validation.md)Проверка входящего или исходящего XML сообщения на определенную схему.

* [XML преобразует](logic-apps-enterprise-integration-transform.md): Конвертировать или настроить сообщение XML на основе ваших требований или требований партнера с помощью карт.

* [Кодирование плоского файла и декодирование плоского файла:](logic-apps-enterprise-integration-flatfile.md)кодирование или декодирование плоского файла.

  Например, SAP принимает и отправляет IDOC-файлы в виде неструктурированных файлов. Многие платформы интеграции создают сообщения XML, включая Logic Apps. Поэтому можно создать приложение логики, которое использует кодировщик неструктурированных файлов для преобразования XML-файлов в неструктурированные файлы.

* [XPath](workflow-definition-language-functions-reference.md#xpath): Обогащайте сообщение и извлекайте из сообщения определенные свойства. Эти извлеченные свойства можно затем использовать для перенаправления сообщения в целевую или промежуточную конечную точку.
