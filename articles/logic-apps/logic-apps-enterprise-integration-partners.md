# <a name="add-trading-partners-to-integration-accounts-for-azure-logic-apps"></a>Добавление учетных записей интеграции торговых партнеров для Azure Logic Apps

В [Azure Logic Apps](../logic-apps/logic-apps-overview.md), можно создать автоматизированные рабочие процессы интеграции бизнес бизнес (B2B) с помощью [учетной записи интеграции](../logic-apps/logic-apps-enterprise-integration-create-integration-account.md) с приложениями логики. Для представления вашей организации и другим пользователям, создавать и добавлять торговых партнеров как артефакты в учетную запись интеграции. Партнеры, сущности, которые участвуют в транзакциях B2B и обмен сообщениями друг с другом.

Перед созданием этих партнеров, убедитесь, что, обменивающихся информацией со своими партнерами, о том, как идентифицировать и проверять сообщения, отправляемые в другой. После вы согласны на эти сведения, все готово для создания партнеров в учетной записи интеграции.

## <a name="partner-roles-in-integration-accounts"></a>Роли в роли партнера в учетных записях интеграции

Задать сведения о сообщениях, обмен которыми осуществляется с партнерами, создания и добавления [соглашения](../logic-apps/logic-apps-enterprise-integration-agreements.md) как артефакты в учетную запись интеграции. Соглашения требуется по крайней мере двух партнеров в учетной записи интеграции. Ваша организация — всегда *главный партнер* в соглашении. Организация, которая обменивается сообщениями с вашей организации является *гостевой партнер*. Гостевой партнер может быть другая компания или даже подразделение вашей организации. Добавив этих партнеров, можно приступать к созданию соглашения.

В соглашении необходимо задать сведения для обработки входящих и исходящих сообщений с точки зрения главного партнера. Для входящих сообщений **параметры получения** укажите, как главный партнер получает сообщения от гостевого партнера в соглашении. Для исходящих сообщений **параметры отправки** укажите, как главный партнер отправляет сообщения партнеру-гостю.

## <a name="prerequisites"></a>Технические условия

* Подписка Azure. Если у вас еще нет подписки Azure, [получите бесплатную учетную запись Azure](https://azure.microsoft.com/free/).

* [Учетной записи интеграции](../logic-apps/logic-apps-enterprise-integration-create-integration-account.md) для хранения ваших партнеров, соглашений и другие артефакты B2B. Эта учетная запись интеграции, необходимо связать с вашей подпиской Azure.

## <a name="create-partner"></a>Создание партнера

1. Войдите на [портале Azure](https://portal.azure.com).

1. В главном меню Azure выберите **Все службы**. В поле поиска введите «интеграция» и выберите **учетные записи интеграции**.

   ![Выберите «Учетные записи интеграции»](./media/logic-apps-enterprise-integration-partners/find-integration-accounts.png)

1. В разделе **Учетные записи интеграции** выберите учетную запись интеграции, в которую нужно добавить партнеров.

   ![Выбор учетной записи интеграции](./media/logic-apps-enterprise-integration-partners/select-integration-account.png)

1. Выберите плитку **Партнеры**.

   ![Выбор плитки "Партнеры"](./media/logic-apps-enterprise-integration-partners/choose-partners.png)

1. В разделе **Партнеры** выберите **Добавить**. В разделе **добавить партнера**, укажите сведения для партнера, как описано в таблице ниже.

   ![Выберите «Добавить» и предоставьте сведения о партнере](./media/logic-apps-enterprise-integration-partners/add-partners.png)

   | Свойство | Обязательно для заполнения | Описание |
   |----------|----------|-------------|
   | **Имя** | Да | Имя партнера |
   | **Квалификатор** | Да | Проверка подлинности текст, который предоставляет уникальные бизнес-идентификаторы для организаций, например, **D-U-N-S (Dun & Bradstreet)** . <p>Партнеры могут использовать взаимно определенные бизнес-удостоверение. В таких случаях выберите **взаимно определено** для EDIFACT или **взаимно определено (X12)** для X12. <p>Для RosettaNet, выберите только **DUNS**, который является стандартной. |
   | **Значение** | Да | Значение, определяющее документы, которые получают приложения логики. <p>Для RosettaNet это значение должно быть числом девятизначные, соответствующее номеру DUNS. |
   ||||

   > [!NOTE]
   > Для партнеров, использующих RosettaNet, можно указать дополнительные сведения, сначала создавая эти партнеры и затем [их после редактирования](#edit-partner).

1. Когда все будет готово, нажмите кнопку **ОК**.

   Новый партнер теперь отображается на **партнеров** списка. Кроме того **партнеров** плитки обновляет текущее количество партнеров.

   ![Новый партнер](./media/logic-apps-enterprise-integration-partners/new-partner.png)

## <a name="edit-partner"></a>Изменение данных партнера

1. На [портале Azure](https://portal.azure.com) найдите и выберите свою учетную запись интеграции.
Выберите плитку **Партнеры**.

   ![Выбор плитки "Партнеры"](./media/logic-apps-enterprise-integration-partners/edit.png)

1. В разделе **партнеров**, выберите партнера, которого вы хотите изменить и выберите **изменить**. В разделе **изменить**, внесите необходимые изменения.

   ![Внесение и сохранение изменений](./media/logic-apps-enterprise-integration-partners/edit-partner.png)

   Для RosettaNet в разделе **свойства партнера RosettaNet**, можно указать эти дополнительные сведения:

   | Свойство | Обязательно для заполнения | Описание |
   |----------|----------|-------------|
   | **Классификации партнера** | Нет | Тип организации партнера |
   | **Код цепочки поставок** | Нет | Предоставить цепочки партнера код, например, «Информационные технологии» или «Электронных компонентов» |
   | **Имя контакта** | Нет | Имя контактного лица партнера |
   | **Электронная почта** | Нет | Адрес электронной почты партнера |
   | **Fax** | Нет | Номер факса партнера |
   | **Телефон** | Нет | Номер телефона партнера |
   ||||

1. Когда все будет готово, выберите **ОК** для сохранения изменений.

## <a name="delete-partner"></a>Удаление партнера

1. На [портале Azure](https://portal.azure.com) найдите и выберите свою учетную запись интеграции. Выберите плитку **Партнеры**.

   ![Выбор плитки "Партнеры"](./media/logic-apps-enterprise-integration-partners/choose-partners-to-delete.png)

1. В разделе **Партнеры** выберите партнера, которого нужно удалить. Щелкните **Удалить**.

   ![Удаление партнера](./media/logic-apps-enterprise-integration-partners/delete-partner.png)
