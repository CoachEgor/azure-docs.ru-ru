# <a name="open-a-support-ticket-for-azure-hpc-cache"></a>Откройте вспомогательный билет для кэша Azure HPC

Воспользуйтесь порталом Azure, чтобы открыть билет поддержки. Перейдите к экземпляру кэша, а затем нажмите на новую ссылку **запроса поддержки,** которая отображается в нижней части боковой панели.

Чтобы открыть билет, если у вас нет активного кэша, используйте основную страницу поддержки Справки и с портала Azure. Откройте меню портала из элемента управления в левом верхнем углу экрана, затем прокрутите вниз и нажмите **Кнопка Справка и поддержка.**

Выберите **новый запрос поддержки** и выберите **Технический** для справки, специфичный для кэша HPC Azure.

Выберите подписку из списка.

Чтобы найти службу Cache Azure HPC, нажмите кнопку **Все службы** и ищите HPC.

![Скриншот запроса на поддержку - Вкладка Основы, частично заполненная как описано](media/hpc-cache-support-request.png)

Заполните остальные поля с вашей информацией и предпочтениями, а затем представить билет, когда вы будете готовы.

После отправки запроса вы получите подтверждение электронной почты с номером билета. Сотрудник службы поддержки свяжется с вами по поводу запроса.
