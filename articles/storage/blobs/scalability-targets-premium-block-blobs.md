# <a name="scalability-targets-for-premium-block-blob-storage-accounts"></a>Цели масштабируемости для учетных записей хранения премиум-блоков

Премиум-производительность блок blob хранения счета оптимизирован для приложений, которые используют меньше, килобайт диапазона объектов. Он идеально подходит для приложений, которые требуют высоких транзакционных ставок или последовательного хранения с низкой задержкой. Премиум-хранилище блоков блоков предназначено для масштабирования с помощью приложений. Если для того, чтобы сценарий размещал приложения (ы), требующие сотни тысяч запросов в секунду или петабайт емкости хранилища, свяжитесь с корпорацией Майкрософт, отправив запрос на поддержку на [портале Azure.](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)
