# <a name="security-controls-for-azure-storage"></a>Средства управления безопасностью для службы хранилища Azure

В этой статье описываются элементы управления безопасностью, встроенные в службу хранилища Azure. 

[!INCLUDE [Security controls Header](../../../includes/security-controls-header.md)]

## <a name="data-protection"></a>Защита данных

| Управление безопасностью | Да/нет | Примечания |
|---|---|--|
| Шифрование неактивных на стороне сервера: ключи, управляемые корпорацией Майкрософт | Да |  |
| Шифрование неактивных на стороне сервера: ключи, управляемые клиентом (BYOK) | Да | См. раздел [Шифрование службы хранилища использование управляемых клиентом ключей в Azure Key Vault](storage-service-encryption-customer-managed-keys.md?toc=%2fazure%2fstorage%2fblobs%2ftoc.json).|
| Шифрование на уровне столбцов (службы данных Azure)| Н/Д |  |
| Шифрование при передаче (например, шифрование ExpressRoute, Шифрование виртуальной сети и шифрование виртуальной сети)| Да | Поддержка стандартных механизмов HTTPS/TLS.  Пользователи также могут шифровать данные перед их передачей в службу. |
| Вызовы API в зашифрованном виде| Да |  |

## <a name="network"></a>Сеть

| Управление безопасностью | Да/нет | Примечания |
|---|---|--|
| Поддержка конечных точек службы| Да |  |
| Поддержка внедрения виртуальной сети| Н/Д |  |
| Поддержка сетевой изоляции и брандмауэров| Да | |
| Поддержка принудительного туннелирования| Н/Д |  |

## <a name="monitoring--logging"></a>Мониторинг & ведения журнала

| Управление безопасностью | Да/нет | Примечания|
|---|---|--|
| Поддержка мониторинга Azure (log Analytics, App Insights и т. д.)| Да | Метрики Azure Monitor|
| Ведение журнала и аудит в плоскости управления и управления | Да | Журнал действий Azure Resource Manager |
| Ведение журнала и аудит в плоскости данных| Да | Журналы диагностики службы.|

## <a name="identity"></a>Тождество

| Управление безопасностью | Да/нет | Примечания|
|---|---|--|
| Аутентификация| Да | Azure Active Directory, общий ключ, маркер общего доступа. |
| Авторизация| Да | Поддержка авторизации через RBAC, списки управления доступом POSIX и токены SAS |

## <a name="configuration-management"></a>Управление конфигурацией

| Управление безопасностью | Да/нет | Примечания|
|---|---|--|
| Поддержка управления конфигурацией (управление версиями конфигураций и т. д.)| Да | Поддержка управления версиями поставщика ресурсов с помощью Azure Resource Manager API |
