# <a name="plan-and-manage-costs-for-azure-storage"></a>Планирование и управление затратами на хранение Azure

В этой статье описывается, как вы планируете и управляете затратами на хранение Azure. Во-первых, вы используете калькулятор цен Azure, чтобы помочь спланировать затраты на хранение перед добавлением ресурсов. После начала использования ресурсов хранения Azure используйте функции управления затратами для настройки бюджетов и мониторинга затрат. Можно также просмотреть прогнозируемые затраты и отслеживать тенденции расходов, чтобы определить области, в которых вы можете действовать.

Имейте в виду, что затраты на хранение Azure — это лишь часть ежемесячных затрат в счете Azure. Хотя в этой статье объясняется, как планировать и управлять затратами для хранения Azure, вы должны выставить счет за все службы Azure и ресурсы, используемые для подписки Azure, включая сторонние службы. После того, как вы знакомы с управлением затратами для хранения Azure, можно применить аналогичные методы для управления затратами для всех служб Azure, используемых в подписке.

## <a name="prerequisites"></a>Предварительные требования

Анализ стоимости поддерживает различные типы учетных записей Azure. Полный список поддерживаемых типов учетных записей см. в статье [Understand Cost Management data](../../cost-management-billing/costs/understand-cost-mgt-data.md) (Интерпретация данных службы "Управление затратами"). Чтобы просмотреть данные о затратах, нужен как минимум доступ на чтение для учетной записи Azure. Дополнительные сведения о назначении доступа к данным службы "Управление затратами" Azure см. в [этой статье](../../cost-management-billing/costs/assign-access-acm-data.md).

## <a name="estimate-costs-before-creating-an-azure-storage-account"></a>Оценка затрат перед созданием учетной записи хранения Azure

Используйте [калькулятор ценообразования Azure](https://azure.microsoft.com/pricing/calculator/) для оценки затрат перед созданием и началом передачи данных на учетную запись Azure Storage.

1. На странице [калькулятора цен Azure](https://azure.microsoft.com/pricing/calculator/) выберите плитку **Учетных записей.**

2. Прокрутите страницу и найдите раздел **учетных записей хранилища** вашей оценки.

3. Выберите варианты из списков выпадающих. 

   При изменении значения этих списков выпадающих данных изменяется смета затрат. Эта оценка отображается в верхнем углу, а также в нижней части сметы. 
    
   ![Мониторинг затрат с помощью панели анализа затрат](media/storage-plan-manage-costs/price-calculator-storage-type.png)

   При изменении значения списка выпадения **типов** изменяются и другие параметры, которые отображаются в этом листе. Используйте ссылки в разделе **Подробнее Информация,** чтобы узнать больше о том, что каждый вариант означает и как эти параметры влияют на цену операций, связанных с хранением. 

4. Измените оставшиеся параметры, чтобы увидеть их влияние на смету.

## <a name="use-budgets-and-cost-alerts"></a>Использование оповещений о бюджете и затратах

Вы можете создать [бюджеты](../../cost-management-billing/costs/tutorial-acm-create-budgets.md), чтобы управлять затратами и создавать оповещения, которые автоматически уведомляют заинтересованные лица о нетипичных и чрезмерных затратах. Оповещения основываются на расходах по сравнению с пороговыми значениями бюджета и затрат. Бюджеты и оповещения создаются для подписок Azure и групп ресурсов, поэтому они полезны как часть общей стратегии мониторинга затрат. Однако они могут иметь ограниченную функциональность для управления отдельными затратами на обслуживание Azure, такими как стоимость хранения Azure, поскольку они предназначены для отслеживания затрат на более высоком уровне.

## <a name="monitor-costs"></a>Мониторинг затрат

При использовании ресурсов Azure с azure Storage вы несете расходы. Затраты на использование ресурсов варьируются в зависимости от временных интервалов (секунды, минуты, часы и дни) или использования единицы (байты, мегабайты и так далее). Затраты понесены сразу же после начала использования хранилища Azure. Вы можете увидеть затраты в панели [анализа затрат](../../cost-management-billing/costs/quick-acm-cost-analysis.md) на портале Azure.

При использовании анализа затрат можно просматривать затраты на хранение Azure в графиках и таблицах с разными временными интервалами. Некоторые примеры по дням, текущий и предыдущий месяц, и год. Вы также можете просмотреть затраты в отношении бюджетов и прогнозируемых затрат. Переход на более длинные представления с течением времени может помочь вам определить тенденции расходов и увидеть, где может произойти перерасход средств. Если вы создали бюджеты, вы также можете легко увидеть, где они превышены.

Для просмотра затрат на хранение Azure при анализе затрат:

1. Войти на [портал Azure](https://portal.azure.com).

2. Откройте окно **управления затратами и выставлением счетов,** выберите **управление затратами** из меню, а затем выберите **анализ затрат.** Затем можно изменить область действия определенной подписки из момента выпадения **области.**

   ![Мониторинг затрат с помощью панели анализа затрат](./media/storage-plan-manage-costs/cost-analysis-pane.png)

4. Чтобы просмотреть только затраты для хранилища Azure, выберите **Добавить фильтр,** а затем выберите **имя службы.** Затем выберите **хранилище** из списка. 

   Вот пример, показывающий затраты только для хранения Azure:

   ![Мониторинг затрат на хранение с помощью панели анализа затрат](./media/storage-plan-manage-costs/cost-analysis-pane-storage.png)

В предыдущем примере можно увидеть текущую стоимость службы. Также отображаются затраты по регионам (локациям) Azure и группе ресурсов.
