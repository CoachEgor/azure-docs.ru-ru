# <a name="recover-a-deleted-dashboard-in-the-azure-portal"></a>Восстановление удаленной панели мониторинга на портале Azure

Если вы находитесь в общедоступном облаке Azure и удаляете _опубликованную_ панель мониторинга на портале Azure, вы можете восстановить эту панель мониторинга в течение 14 дней после удаления. Если вы находитесь в облаке управления Azure или панель мониторинга не опубликована, вы не можете восстановить его, и вы должны восстановить его. Для получения дополнительной информации о публикации панели мониторинга [см.](azure-portal-dashboard-share-access.md#publish-dashboard) Выполните следующие действия, чтобы восстановить опубликованную панель мониторинга:

1. Из меню портала Azure выберите **группы ресурсов,** затем выберите группу ресурсов, в которой была опубликована панель мониторинга (по умолчанию она называется **панели мониторинга).**

1. Под **журналом активности**расширьте работу **панели удалений.** Выберите вкладку **«История изменений»,** а затем выберите ** \<\>удаленный ресурс.**

    ![Скриншот вкладки истории изменений](media/recover-shared-deleted-dashboard/change-history-tab.png)

1. Выберите и скопируйте содержимое левой панели, а затем сохраните в текстовом файле с расширением файла _.json._ Портал использует файл JSON для воссоздания панели мониторинга.

    ![Скриншот дисфа истории изменений](media/recover-shared-deleted-dashboard/change-history-diff.png)

1. Из меню портала Azure выберите **панели мониторинга,** а затем выберите **Upload**.

    ![Скриншот загрузки приборной панели](media/recover-shared-deleted-dashboard/dashboard-upload.png)

1. Выберите сохраненный файл JSON. Портал воссоздает панель мониторинга с тем же именем и элементами, что и удаленная панель мониторинга.

1. Выберите **Share,** чтобы опубликовать панель мониторинга и восстановить соответствующий элемент управления доступом.

    ![Скриншот доли панели мониторинга](media/recover-shared-deleted-dashboard/dashboard-share.png)
