# <a name="get-started-with-the-azure-quickstart-center"></a>Начните с Центра быстрого запуска Azure

Azure quickstart Center — это управляемый опыт на портале Azure, доступный всем, кто хочет улучшить свои знания о Azure. Для организаций, не передованных для Azure, это самый быстрый способ посадки на борт и настройки облачной среды.

## <a name="overview"></a>Обзор

Центр быстрого запуска Azure имеет три варианта для начала работы:

* **Руководство по настройке:** разработанные для ИТ-адиона и облачного архитектора, наши гиды вводят ключевые концепции для внедрения Azure. Структурированные шаги помогут вам принять меры по мере извенания, применяя рекомендуемые методы Майкрософт. Руководство по миграции помогает вам оценивать готовность и планировать, когда вы готовитесь к переходу на Azure.

* **Запустите проект:** Если вы готовы создать ресурс, этот раздел позволит вам узнать больше о вашем выборе, прежде чем взять на себя обязательство по опции обслуживания. Вы узнаете больше об услуге и о том, почему вы должны использовать ее, изучить затраты и определить предпосылки. Сделав свой выбор, вы можете перейти непосредственно к созданию.

* **Онлайн-обучение**: В этом разделе Центра быстрого запуска Azure освещаются бесплатные вводные модули от Microsoft Learn. Выберите плитку для запуска курса и узнайте больше об облачных концепциях и управлении ресурсами в Azure.

## <a name="how-to-use-azure-quickstart-center"></a>Как использовать центр быстрого запуска Azure

1. Войдите на [портал Azure](https://portal.azure.com).

1. Выберите **все службы** из меню портала Azure.

1. Выберите **Общий** > **центр быстрого запуска**.
