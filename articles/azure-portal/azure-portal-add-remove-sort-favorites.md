# <a name="add-remove-and-sort-favorites"></a>Добавление, удаление и сортировка избранного

Добавлять или удалять элементы из вашей **"Избранное"** списке таким образом, можно быстро перейти к службам, которые наиболее часто. Мы уже добавили некоторые общие службы для вашего **"Избранное"** списка, но вы скорее всего захотите настроить его. Вы являетесь единственным, кто видит изменения, внесенные в **"Избранное"**.

## <a name="add-or-remove-a-favorite"></a>Добавление или удаление из избранного

Элементы, которые отображаются в категории **"Избранное"** выбираются из **все службы**. Значок звездочки ![желтый значок звездочки](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-star.png) рядом с каждой службы указывает, отображается ли элемент в списке. Выберите значок звездочки, чтобы добавить или удалить службу из **"Избранное"** списка.

### <a name="add-cost-management--billing-to-favorites"></a>Добавить в "Избранное" Управление затратами + выставление счетов

1. Выберите **все службы** из верхней части боковой панели.

    ![Снимок экрана, показывающий все выбранные службы](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-all-services.png)

1. Введите слово «расходы» в поле поиска. Только те службы, «стоимость» в заголовке или, «cost» как будет показано ключевое слово.

   ![Снимок экрана: отображение поиска во всех службах](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-search.png)

1. Выберите значок звездочки.

   ![Снимок экрана выбранные отображение типа "звезда", рядом с управление затратами + выставление счетов](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-add.png)

1. **Cost Management + выставление счетов** теперь добавлена как последний элемент в вашей **"Избранное"** списка.

## <a name="sort-favorites"></a>Сортировка "Избранное"

Можно изменить порядок перечисления любимыми службами. Просто перетащить элемент меню в другое расположение в разделе **"Избранное"**.

### <a name="move-cost-management--billing-to-the-top-of-favorites"></a>Переместить в верхнюю часть "Избранное" Управление затратами + выставление счетов

1. Выберите и удерживайте **управление затратами + выставление счетов** запись на **"Избранное"** списка.

   ![Снимок экрана, показывающий управление затратами + выставление счетов выбранные](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-sort.png)

1. Продолжая для хранения, перетащите элемент в верхнюю часть **"Избранное"** и отпустите.
