# <a name="create-and-share-dashboards-in-the-azure-portal"></a>Создание панелей мониторинга на портале Azure и предоставление общего доступа к ним

Панели мониторинга представляют собой целенаправленное и организованное представление обоблачных ресурсов на портале Azure. Используйте панели мониторинга в качестве рабочей области, где можно быстро запускать задачи для повседневных операций и мониторинга ресурсов. Например, создавайте настраиваемые панели мониторинга на основе проектов, задач или ролей пользователей.

Портал Azure предоставляет панель мониторинга по умолчанию в качестве отправной точки. Можно отсеять панель мониторинга по умолчанию. Создавайте и настраивайте дополнительные панели мониторинга, а также публикуйте и делитесь панелами мониторинга, чтобы сделать их доступными для других пользователей. В этой статье описывается, как создать новую панель мониторинга, настроить интерфейс, а также опубликовать и поделиться панелей мониторинга.

## <a name="create-a-new-dashboard"></a>Создание панели мониторинга

В этом примере мы создаем новую частную панель мониторинга и назначаем имя. Чтобы приступить к работе, выполните эти действия:

1. Войдите на [портал Azure](https://portal.azure.com).

1. Из меню портала Azure выберите **панель мониторинга.** Представление по умолчанию уже может быть настроено на панель мониторинга.

    ![Откройте приборную панель](./media/azure-portal-dashboards/portal-menu-dashboard.png)

1. Выберите **новую панель мониторинга.**

    ![Скриншот новой приборной панели](./media/azure-portal-dashboards/create-new-dashboard.png)

    Это действие открывает **плитку Галерея**, из которой вы будете выбирать плитки, и пустая сетка, где вы будете расставлять плитки.

    ![Скриншот галереи плитки и пустой сетки](./media/azure-portal-dashboards/dashboard-name.png)

1. Выберите текст **My Dashboard** на метке панели мониторинга и введите имя, которое поможет вам легко определить пользовательскую панель мониторинга.

1. Выберите **настройку Готово** в заголовке страницы для выхода из режима отображаемого.

Представление панели мониторинга теперь показывает новую панель мониторинга. Выберите стрелку рядом с именем панели мониторинга, чтобы увидеть доступные для вас панели мониторинга. Список может включать панели мониторинга, созданные и общие для других пользователей.

## <a name="edit-a-dashboard"></a>Изменение панели мониторинга

Теперь давайте отсеивать панель мониторинга, чтобы добавить, изменить размер и расположить плитки, представляющие ресурсы Azure.

### <a name="add-tiles-from-the-dashboard"></a>Добавление плиток с панели мониторинга

Чтобы добавить плитки в панель мониторинга, выполните следующие действия:

1. Выберите ![](./media/azure-portal-dashboards/dashboard-edit-icon.png) **Edit** отсеивание значка от заголовка страницы.

    ![Скриншот подсветки приборной панели](./media/azure-portal-dashboards/dashboard-edit.png)

1. Просмотрите **галерею плитки** или используйте поле поиска, чтобы найти плитку, которая вам нужна.

1. Выберите **Добавить,** чтобы добавить плитку на панель мониторинга с размером и местоположением по умолчанию. Или перетащите плитку в сетку и поместите ее там, где вы хотите.

> [!TIP]
> Если вы работаете с более чем одной организацией, добавьте в панель мониторинга итоговую панель **идентификационных данных** и включаем в панель мониторинга, чтобы четко показать, к какой организации принадлежат ресурсы.

### <a name="add-tiles-from-a-resource-page"></a>Добавление плиток со страницы ресурса

Существует альтернативный способ добавления плитки в панель мониторинга. Многие страницы ресурсов включают значок pushpin в панели команд. При выборе значка к панели мониторинга, которая в настоящее время активна, крепится плитка, представляющая исходную страницу. 

![Скриншот панели команд страницы с значком штифта](./media/azure-portal-dashboards/dashboard-pin-blade.png)

### <a name="resize-or-rearrange-tiles"></a>Изменить размер или переставить плитки

Чтобы изменить размер плитки или переставить плитки на приборной панели, выполните следующие действия:

1. Выберите ![](./media/azure-portal-dashboards/dashboard-edit-icon.png) **Edit** отсеивание значка от заголовка страницы.

1. Выберите контекстное меню в правом верхнем углу плитки. Затем выберите размер плитки. Плитки, которые поддерживают любой размер также включают в себя "ручку" в правом нижнем углу, что позволяет перетащить плитку до размера, который вы хотите.

    ![Скриншот приборной панели с открытым меню размера плитки](./media/azure-portal-dashboards/dashboard-tile-resize.png)

1. Выберите плитку и перетащите ее в новое место на сетке, чтобы организовать панель мониторинга.

### <a name="additional-tile-configuration"></a>Дополнительная конфигурация плитки

Некоторые плитки могут потребовать больше конфигурации, чтобы показать информацию, которая вам нужна. Например, плитка **диаграммы метрик идиаграммы** должна быть настроена для отображения метрики из **Azure Monitor.** Можно также настроить данные плитки для переопределения параметров времени выполнения панели мониторинга по умолчанию.

Любая плитка, которая должна быть настроена отображает баннер **плитки настройки,** пока вы не настроить плитку. Чтобы настроить плитку:

1. Выберите **настройку Готово** в заголовке страницы для выхода из режима отображаемого.

1. Выберите баннер, а затем сделайте необходимую настройку.

    ![Скриншот плитки, требующей конфигурации](./media/azure-portal-dashboards/dashboard-configure-tile.png)

> [!NOTE]
> Плитка разметки позволяет отображать пользовательское статическое содержимое на панели мониторинга. Это могут быть основные инструкции, изображение, набор гиперссылок или даже контактная информация. Для получения дополнительной информации об использовании плитки разметки [см.](azure-portal-markdown-tile.md)

### <a name="customize-tile-data"></a>Настройка данных плитки

Данные на панели мониторинга автоматически отображали активность за последние 24 часа. Чтобы показать другой промежуток времени только для этой плитки, выполните следующие действия:

1. Выберите **настраиваемые данные плитки** из контекстного меню или фильтра значка ![фильтра](./media/azure-portal-dashboards/dashboard-filter.png) из верхнего левого угла плитки.

    ![Скриншот контекстного меню плитки](./media/azure-portal-dashboards/dashboard-customize-tile-data.png)

1. Выберите флажок, чтобы **переопределить настройки времени панели мониторинга на уровне плитки.**

    ![Скриншот диалога для настройки настроек времени плитки](./media/azure-portal-dashboards/dashboard-override-time-settings.png)

1. Выберите временной промежуток для этой плитки. Вы можете выбрать из последних 30 минут до последних 30 дней или определить пользовательский диапазон.

1. Выберите детализацию времени для отображения. Вы можете показывать от одной минуты приращений до одного месяца.

1. Нажмите кнопку **Применить**.

## <a name="delete-a-tile"></a>Удаление плитки

Чтобы удалить плитку с панели мониторинга, выполните следующие действия:

* Выберите контекстное меню в правом верхнем углу плитки, а затем выберите **Удалить из приборной панели.** или

* Выберите ![](./media/azure-portal-dashboards/dashboard-edit-icon.png) **значок** edit, чтобы войти в режим настройки. Нависните в правом верхнем ![углу](./media/azure-portal-dashboards/dashboard-delete-icon.png) плитки, затем выберите значок удаления, чтобы удалить плитку с панели мониторинга.

   ![Скриншот, показывающий, как удалить плитку с приборной панели](./media/azure-portal-dashboards/dashboard-delete-tile.png)

## <a name="clone-a-dashboard"></a>Клонирование панели мониторинга

Чтобы использовать существующую панель мониторинга в качестве шаблона для новой панели мониторинга, выполните следующие действия:

1. Убедитесь, что представление панели мониторинга показывает панель мониторинга, которую вы хотите скопировать.

1. В заголовке страницы ![выберите](./media/azure-portal-dashboards/dashboard-clone.png) **значок**клона клона.

1. Копия панели мониторинга, **названная Клоном** *имени панели мониторинга,* открывается в режиме отодвилость. Используйте предыдущие шаги в этой статье, чтобы переименовать и настроить панель мониторинга.

## <a name="publish-and-share-a-dashboard"></a>Опубликовать и поделиться панелью мониторинга

Когда вы создаете панель мониторинга, она является частной по умолчанию, что означает, что вы единственный, кто может видеть его. Чтобы сделать панели мониторинга доступными для других, вы можете публиковать и делиться ими. Для получения дополнительной информации [см. Панели мониторинга Share Azure с помощью управления доступом на основе ролей.](azure-portal-dashboard-share-access.md)

### <a name="open-a-shared-dashboard"></a>Откройте общую панель мониторинга

Чтобы найти и открыть общую панель мониторинга, выполните следующие действия:

1. Выберите стрелку рядом с именем панели мониторинга.

1. Выберите из отображаемого списка панелей мониторинга. Если панель мониторинга, которая требуется открыть, не указана:

    1. выберите **все панели мониторинга.**

        ![Скриншот меню выбора приборной панели](./media/azure-portal-dashboards/dashboard-browse.png)

    1. В поле **Type** выберите **общие панели мониторинга.**

        ![Скриншот всех меню выбора панелей мониторинга](./media/azure-portal-dashboards/dashboard-browse-all.png)

    1. Выберите одну или несколько подписок. Вы также можете ввести текст для фильтрации панелей мониторинга по имени.

    1. Выберите панель мониторинга из списка общих панелей мониторинга.

## <a name="delete-a-dashboard"></a>Удаление панели мониторинга

Чтобы навсегда удалить приватную или общую панель мониторинга, выполните следующие действия:

1. Выберите панель мониторинга, которая требуется удалить из списка рядом с именем панели мониторинга.

1. Выберите ![](./media/azure-portal-dashboards/dashboard-delete-icon.png) значок удалить значок **Удалить** из заголовка страницы.

1. Для частной панели мониторинга выберите **OK** в диалоге подтверждения, чтобы удалить панель мониторинга. Для общей панели мониторинга в диалоге подтверждения выберите флажок, чтобы подтвердить, что опубликованная панель мониторинга больше не будет досматриваться другими пользователями. Затем выберите **OK**.

    ![Скриншот подтверждения удаления](./media/azure-portal-dashboards/dashboard-delete-dash.png)
