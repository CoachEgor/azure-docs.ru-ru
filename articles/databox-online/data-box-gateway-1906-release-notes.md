# <a name="azure-data-box-edge-and-azure-data-box-gateway-1906-release-notes"></a>Заметки о выпуске Azure Data Box Edge и Azure Data Box Gateway 1906

Следующие примечания к выпуску определяют критические открытые проблемы и решенные проблемы для выпуска 1906 года для Azure Data Box Edge и Azure Data Box Gateway.

Заметки о выпуске постоянно обновляются: обнаруживаются и добавляются критические проблемы, требующие временного решения. Перед развертыванием шлюза Data Box Edge/Data Box, внимательно просмотрите информацию, содержащуюся в примечаниях к выпуску.

Этот релиз соответствует версиям программного обеспечения:

- **Шлюз Box 1906 (1.6.978.743)**
- **Дата Box Edge 1906 (1.6.978.743)**

> [!NOTE]
> Обновление 1906 может быть применено только к устройствам Data Box Edge, которые работают с общей доступностью (GA) или 1905 версии программного обеспечения.

## <a name="whats-new"></a>Новые возможности

- **Исправление ошибки в рабочем процессе управления ключом восстановления** - В предыдущем выпуске была ошибка, из-за которой ключ восстановления не был применен. В этом обновлении эта ошибка устранена. Мы настоятельно рекомендуем применить это обновление, поскольку ключ восстановления позволяет восстановить данные на устройстве, в случае, если устройство не загружается. Для получения дополнительной информации узнайте, как [сохранить ключ восстановления при развертывании Data Box Edge или шлюза Box Data.](data-box-edge-deploy-connect-setup-activate.md#set-up-and-activate-the-physical-device)
- **Полевые программируемые ворота Array (FPGA) лесозаготовки улучшений** - Начиная 1905 релиз, журнал и оповещения усовершенствования, связанные с FPGA. Это по-прежнему является необходимым обновлением для Data Box Edge, если вы используете функцию вычислений Edge с FPGA. Для получения дополнительной информации узнайте, как [преобразовать данные с помощью Edge compute на вашем краю коробки данных.](data-box-edge-deploy-configure-compute-advanced.md)

## <a name="known-issues-in-ga-release"></a>Известные проблемы в релизе GA

Нет новых выпусков отмечены для этого релиза. Все выпуск отметил вопросы перенесены из предыдущих релизов. Чтобы увидеть список известных проблем, перейдите на [известные вопросы в релизе GA](data-box-gateway-release-notes.md#known-issues-in-ga-release).
