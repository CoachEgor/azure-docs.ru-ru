# <a name="return-or-replace-your-azure-data-box-edge-device"></a>Возврат или замена устройства Azure Data Box Edge

В этой статье описывается, как уничтожить данные, а затем вернуть устройство Azure Data Box Edge. После возврата устройства можно также удалить ресурс, связанный с устройством, или заказать заменяющее устройство.

Вы узнаете, как выполнять следующие задачи:

> [!div class="checklist"]
> * Стереть данные с дисков данных на устройстве
> * Откройте билет поддержки, чтобы вернуть устройство
> * Упакуйте устройство и запланируйте пикап
> * Удаление ресурса на портале Azure
> * Получить заменяющее устройство

## <a name="erase-data-from-the-device"></a>Удаление данных с устройства

Чтобы стереть данные с дисков данных устройства, необходимо сбросить устройство. Вы можете сбросить устройство с помощью локального веб-интерфейса или интерфейса PowerShell.

Перед сбросить, создать копию локальных данных на устройстве, если это необходимо. Данные можно скопировать с устройства в контейнер хранения Azure.

Чтобы сбросить устройство с помощью локального веб-мина, примите следующие меры.

1. В локальном веб-файле можно перейти на **сбросить > устройства обслуживания.**
2. Выберите **устройство спрозана.**

    ![Сброс настроек устройства](media/data-box-edge-return-device/device-reset-1.png)

3. При запросе для подтверждения, просмотрите предупреждение и выберите **Да** для продолжения.

    ![Подтвердить сбросить](media/data-box-edge-return-device/device-reset-2.png)  

Сбросить стирает данные с дисков данных устройства. В зависимости от объема данных на устройстве, этот процесс занимает около 30-40 минут.

Кроме того, подключитесь к интерфейсу PowerShell устройства и используйте `Reset-HcsAppliance` cmdlet для удаления данных с дисков данных. Для получения дополнительной информации [см.](data-box-edge-connect-powershell-interface.md#reset-your-device)

> [!NOTE]
> - При обмене или обновлении на новое устройство мы рекомендуем сбросить устройство только после того, как вы получили новое устройство.
> - Сбросить устройство удаляет только все локальные данные с устройства. Данные, наиболее наданные в облаке, не удаляются и собирают [сборы.](https://azure.microsoft.com/pricing/details/storage/) Эти данные должны быть удалены отдельно с помощью инструмента управления облачными хранилищами, такого как [Azure Storage Explorer.](https://azure.microsoft.com/features/storage-explorer/)

## <a name="open-a-support-ticket"></a>Открыть билет поддержки

Чтобы начать процесс возврата, примите следующие шаги.

1. Откройте билет поддержки с поддержкой Майкрософт, указывающий на то, что вы хотите вернуть устройство. Выберите тип проблемы в качестве **оборудования Data Box Edge.**

    ![Открыть запрос в службу поддержки](media/data-box-edge-return-device/open-support-ticket-1.png)  

2. С вами свяжется инженер службы поддержки Майкрософт. Предоставьте сведения о доставке.
3. Если вам нужна коробка доставки возвращения, то вы можете спросить его. Ответ **Да** на вопрос **Нужна пустая коробка, чтобы вернуться**.


## <a name="schedule-a-pickup"></a>Расписание пикапа

1. Завершите работу устройства. В локальном пользовательском веб-интерфейсе последовательно выберите **Обслуживание > Настройки питания**.
2. Выберите **Выключить**. Когда запрос для подтверждения, нажмите **Да,** чтобы продолжить. Для получения дополнительной информации [см.](data-box-gateway-manage-access-power-connectivity-mode.md#manage-power)
3. Отключите силовые кабели и удалите все сетевые кабели с устройства.
4. Подготовьте посылку, используя собственную коробку или пустую коробку, полученную от Azure. Поместите устройство и шнуры питания, которые были отправлены с устройством в коробку.
5. Прикрепите этикетку доставки, полученную от Azure на упаковке.
6. Запланируйте отправку, обратившись к вашему региональному поставщику. При возврате устройства в США вашим перевозчиком может стать UPS или FedEx. Чтобы запланировать пикап с UPS:

    1. Позвоните в местное отделение UPS (бесплатный номер конкретной страны).
    2. В своем звонке процитируйте обратный номер отслеживания груза, указанный на печатной этикетке.
    3. Если номер отслеживания не указан, UPS потребует от вас оплаты дополнительной платы во время пикапа.

    Вместо планирования пикапа, вы также можете высадить Data Box Edge в ближайшем месте высадки.

## <a name="delete-the-resource"></a>Удалить ресурс

После получения устройства в центре обработки данных Azure устройство проверяется на наличие повреждений или признаков фальсификации.

- Если устройство поступает нетронутым и в хорошей форме, счетчик выставления счетов останавливается для этого ресурса. Служба поддержки Майкрософт свяжется с вами, чтобы подтвердить, что устройство было возвращено. Затем можно удалить ресурс, связанный с устройством, на портале Azure.
- Если устройство поступает значительно повреждены, штрафы могут применяться. Для получения [подробной](https://www.microsoft.com/licensing/product-licensing/products)информации см. [FAQ on lost or damaged device](https://azure.microsoft.com/pricing/details/databox/edge/)  


Удалять устройство можно на портале Azure:
-   После того, как вы разместили заказ и до того, как устройство будет подготовлено корпорацией Майкрософт.
-   После того, как устройство вернулось в корпорацию Майкрософт, оно проходит физический осмотр в центре обработки данных Azure, и служба поддержки Майкрософт требует подтверждения возврата устройства.

Если вы активировали устройство против другой подписки или местоположения, корпорация Майкрософт переместит ваш заказ в новую подписку или местоположение в течение одного рабочих дней. После перемещения заказа можно удалить этот ресурс.


следующие шаги, чтобы удалить устройство и ресурс на портале Azure.

1. На портале Azure перейдите на свой ресурс, а затем в **Обзор**. Из панели команд выберите **Удалить**.

    ![Выбор команды удаления](media/data-box-edge-return-device/delete-resource-1.png)

2. В лезвии **устройства Delete** введите имя устройства, которое вы хотите удалить, и выберите **Delete.**

    ![Подтверждение удаления](media/data-box-edge-return-device/delete-resource-2.png)

Вы будете уведомлены после успешного удаления устройства и связанного с ним ресурса.

## <a name="get-a-replacement-device"></a>Получить заменяющее устройство

Замена устройства необходима, когда существующее устройство имеет аппаратный сбой или нуждается в обновлении. При следующем шаге, когда устройство имеет аппаратную проблему:

1. [Откройте билет поддержки для аппаратной проблемы](#open-a-support-ticket). Поддержка Майкрософт определит, что подразделение позамены полей (FRU) недоступно для данного экземпляра или устройство нуждается в обновлении оборудования. В любом случае, поддержка закажет замену устройства.
2. [Создайте новый ресурс](data-box-edge-deploy-prep.md#create-a-new-resource) для замены устройства. Убедитесь в том, чтобы выбрать флажок против **у меня есть устройство Data Box Edge**. 
3. После получения замены устройства [установите](data-box-edge-deploy-install.md) и [активируйте](data-box-edge-deploy-connect-setup-activate.md) заменяющее устройство на новом ресурсе.
4. Выполните все действия по возвращению исходного устройства:
    1. Откройте другой билет, чтобы вернуть исходное устройство.
    2. [Стереть данные на устройстве](#erase-data-from-the-device).
    3. [Расписание пикапа](#schedule-a-pickup).
    5. [Удалите ресурс, связанный](#delete-the-resource) с возвращенным устройством.
