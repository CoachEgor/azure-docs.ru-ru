# <a name="azure-data-box-edge-and-azure-data-box-gateway-1905-release-notes"></a>Заметки о выпуске Azure Data Box Edge и Azure Data Box Gateway 1905

## <a name="overview"></a>Обзор

Следующие примечания к выпуску определяют критические открытые проблемы и решенные проблемы для выпуска 1905 года для Azure Data Box Edge и Azure Data Box Gateway.

Заметки о выпуске постоянно обновляются: обнаруживаются и добавляются критические проблемы, требующие временного решения. Перед развертыванием шлюза Data Box Edge/Data Box, внимательно просмотрите информацию, содержащуюся в примечаниях к выпуску.

Этот релиз соответствует версиям программного обеспечения:

- **Шлюз Box 1905 (1.6.887.626)**
- **Дата Box Edge 1905 (1.6.887.626)**

> [!NOTE]
> Обновление 1905 может быть применено только к устройствам Data Box Edge, которые работают с GA-версией программного обеспечения.

## <a name="whats-new"></a>Новые возможности

- **Полевые программные ворота Array (FPGA) улучшения регистрации** - В этом выпуске, мы сделали лесозаготовки и оповещения усовершенствования, связанные с FPGA. Это необходимое обновление для Data Box Edge, если вы используете функцию вычислений Edge с FPGA. Для получения дополнительной информации узнайте, как [преобразовать данные с помощью Edge compute на вашем краю коробки данных.](data-box-edge-deploy-configure-compute-advanced.md)

## <a name="known-issues-in-ga-release"></a>Известные проблемы в релизе GA

Нет новых выпусков отмечены для этого релиза. Все выпуск отметил вопросы перенесены из предыдущих релизов. Чтобы увидеть список известных проблем, перейдите на [известные вопросы в релизе GA](data-box-gateway-release-notes.md#known-issues-in-ga-release).
