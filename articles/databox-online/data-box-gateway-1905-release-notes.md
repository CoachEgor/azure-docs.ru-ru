# <a name="azure-data-box-edge-and-azure-data-box-gateway-1905-release-notes"></a>Заметки о выпуске Azure Edge поле данных и Azure Data Box шлюза 1905

## <a name="overview"></a>Обзор

В следующих заметках важные нерешенные и устраненные проблемы для выпуска 1905 для Edge поле данных Azure и Azure Data Box шлюза.

Заметки о выпуске постоянно обновляются: обнаруживаются и добавляются критические проблемы, требующие временного решения. Перед развертыванием ваш Data Gateway поле Edge/Data Box, внимательно просмотрите сведения, содержащиеся в заметках о выпуске.

Этот выпуск соответствует версии программного обеспечения:

- **Шлюз поля данных 1905 (1.6.887.626)**
- **Край поля данных 1905 (1.6.887.626)**

> [!NOTE]
> Обновление 1905 можно применить только к устройствам Edge поле данных, работающих под управлением Общедоступные версии программного обеспечения.

## <a name="whats-new"></a>Новые возможности

- **Улучшения ведения журнала поле программируемых шлюза массива (FPGA)** — в этом выпуске мы внесли ведения журналов и оповещений усовершенствования, связанные с FPGA. Это обязательное обновление для Edge поле данных, если вы используете функцию Edge вычислений с FPGA. Дополнительные сведения см. в разделе Практическое [преобразование данных с помощью граничные вычисления на вашей стороне поле данных](data-box-edge-deploy-configure-compute-advanced.md).

## <a name="known-issues-in-ga-release"></a>Известные проблемы в выпуск общедоступной версии

Нет новых проблем освобождаются указаны для этого выпуска. Все версии отметить, что проблемы были перенесены из предыдущих версий. Чтобы просмотреть список известных проблем, перейдите к [известные проблемы в работе выпуска общедоступной версии](data-box-gateway-release-notes.md#known-issues-in-ga-release).
