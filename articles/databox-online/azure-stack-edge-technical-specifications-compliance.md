# <a name="azure-stack-edge-technical-specifications"></a>Технические характеристики Azure Stack Edge

Компоненты оборудования устройства Microsoft Azure Stack Edge соответствуют техническим характеристикам и нормативным стандартам, описанным в этой статье. В технических спецификациях описываются источники питания, габаритные размеры, емкость и стандарты охраны окружающей среды.

## <a name="compute-memory-specifications"></a>Спецификации вычислительных компонентов и запоминающих устройств

Устройство Azure Stack Edge соответствует следующим спецификациям для вычислительных компонентов и запоминающих устройств:

| Спецификация           | Значение                  |
|-------------------------|----------------------------|
| ЦП    | 2 х 10-ядерных ЦП                     |
| Память              | ОЗУ 128 ГБ.                  |

## <a name="fpga-specifications"></a>Спецификации ППВМ

Программируемая пользователем вентильная матрица (ППВМ) предусмотрена на каждом устройстве Azure Stack Edge, обеспечивая сценарии Машинного обучения.

| Спецификация           | Значение                  |
|-------------------------|----------------------------|
| ППВМ   | Intel Arria 10 <br> Доступные модели с поддержкой глубокой нейронной сети (DNN) те же, что и [экземпляры ППВМ с поддержкой облака](https://docs.microsoft.com/azure/machine-learning/how-to-deploy-fpga-web-service#whats-supported-on-azure).|

## <a name="power-supply-unit-specifications"></a>Спецификации источника питания

Устройство Azure Stack Edge имеет два источника питания на 100–240 В, оснащенных высокопроизводительными вентиляторами. Два таких источника питания обеспечивают конфигурацию с избыточным питанием. В случае сбоя источника питания устройство продолжает работать в обычном режиме на другом источнике питания до замены отказавшего модуля. В таблице ниже приведены технические спецификации модулей питания.

| Спецификация           | БП 750 ВТ                  |
|-------------------------|----------------------------|
| Максимальная выходная мощность    | 750 Вт                     |
| Частота               | 50/60 Гц                   |
| Выбор диапазона напряжения | Автоматический выбор напряжения: 100–240 В переменного тока |
| «Горячая» замена           | Да                        |

### <a name="azure-stack-edge-power-cord-specifications-by-region"></a>Характеристики кабелей питания для Azure Stack Edge в разных регионах

Требования к кабелю питания, необходимому для устройства Azure Stack Edge, зависят от региона Azure.
Технические спецификации всех поддерживаемых кабелей питания см. в разделе [Спецификации кабелей питания Azure Stack Edge по регионам](azure-stack-edge-technical-specifications-power-cords-regional.md).

## <a name="storage-specifications"></a>Спецификации хранилища

Устройство Azure Stack Edge имеет 9 х 2,5" твердотельных накопителей с интерфейсом NVMe, каждый емкостью 1,6 ТБ. Из этих твердотельных накопителей 1 — это диск операционной системы, а оставшиеся 8 — это диски данных. Общая полезная емкость устройства составляет примерно 12,5 ТБ. В таблице ниже приведены подробные сведения о емкости хранилища устройства.

|     Спецификация                          |     Значение             |
|--------------------------------------------|-----------------------|
|    Количество твердотельных накопителей (SSD)     |    8                  |
|    Емкость одного твердотельного накопителя (SSD)                     |    1,6 TБ             |
|    Общая емкость                          |    12,8 ТБ            |
|    Общая доступная емкость*                  |    ~ 12,5 ТБ            |

**Определенное пространство зарезервировано для внутреннего использования.*
