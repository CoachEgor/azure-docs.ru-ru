# <a name="enable-compute-network-on-your-azure-data-box-edge"></a>Включение сети вычислений в Edge поле данных в Azure

В этой статье описывается, как доступ к сети вычислений, на устройстве для модулей, запущенных в Edge поле данных в Azure.

Для настройки сети, вы будете сделайте следующее:

- Включение сетевого интерфейса на устройстве Edge поле данных для вычислений
- Добавление модуля доступа к сети вычислений на вашей стороне поле данных
- Убедитесь, что модуль можно получить доступ к включенного сетевого интерфейса

В этом руководстве вы используете модуль приложения веб-сервера для демонстрации сценария.

## <a name="prerequisites"></a>Технические условия

Перед началом работы вам потребуется:

- Устройство Edge поле данных с завершения настройки устройства.
- Вы завершили **Настройка вычислений** шаг согласно [руководства: Преобразование данных с помощью Azure Data Box Edge](data-box-edge-deploy-configure-compute-advanced.md#configure-compute) на вашем устройстве. На устройстве будет использоваться связанный ресурс центра Интернета вещей устройство Интернета вещей и устройства IoT Edge.

## <a name="enable-network-interface-for-compute"></a>Включение сетевого интерфейса для вычислений

Для доступа к модулям, установленной на устройстве с помощью внешней сети, необходимо назначить IP-адрес для сетевого интерфейса на устройстве. Вы можете управлять этими параметрами вычислений из локального пользовательского веб-интерфейса.

Выполните следующие действия в локальном веб-интерфейсе, чтобы настроить параметры вычислений.

1. Выберите **Конфигурация > Параметры вычисления**.  

2. **Включить** сетевого интерфейса, который вы хотите использовать для подключения к модулю вычислений, вы запустите на устройстве.

    - При использовании статических IP-адресов укажите IP-адрес сетевого интерфейса.
    - Если используется DHCP, автоматически назначаются IP-адреса. В этом примере используется DHCP.

    ![Включить параметры вычислений 1](media/data-box-edge-extend-compute-access-modules/enable-compute-setting-1.png)

3. Выберите **Применить** для применения параметров. Запомните или запишите IP-адрес, назначенный сетевому интерфейсу, если используется DHCP.

    ![Включение параметров вычислений](media/data-box-edge-extend-compute-access-modules/enable-compute-setting-2.png)

## <a name="add-webserver-app-module"></a>Модуль приложения веб-сервера

Выполните следующие действия, чтобы добавить модуль веб-сервера приложения на устройстве Edge поле данных.

1. Перейдите в центр Интернета вещей ресурс, связанный с вашим устройством Edge поле данных, а затем выберите **устройство IoT Edge**.
2. Выберите устройство IoT Edge, связанное с устройство Edge поле данных. На **сведений об устройстве**выберите **задание модулей**. На **добавления модулей**выберите **+ добавить** , а затем выберите **модуля IoT Edge**.
3. В **пользовательские модули Edge Интернета вещей** колонки:

    1. Укажите **имя** для вашего модуля приложения веб-сервер, который вы хотите развернуть.
    2. Укажите **URI образа** для образа модуля. Модуль сопоставления, предоставленные имя и теги извлекается. В этом случае `nginx:stable` будет извлечь образ nginx стабильная (с тегом как стабильные) из общедоступной [репозиторий Docker](https://hub.docker.com/_/nginx/).
    3. В **параметры создания контейнера**, вставьте следующий образец кода:  

        ```
        {
            "HostConfig": {
                "PortBindings": {
                    "80/tcp": [
                        {
                            "HostPort": "8080"
                        }
                    ]
                }
            }
        }
        ```

        Эта конфигурация позволяет получить доступ к модуль, используя IP-адрес сети вычислений над *http* через TCP-порт 8080 (с веб-сервера по умолчанию порт 80,).

        ![Укажите сведения о порте в колонке пользовательский модуль Edge Интернета вещей](media/data-box-edge-extend-compute-access-modules/module-information.png)

    4. Щелкните **Сохранить**.

## <a name="verify-module-access"></a>Проверка доступа модуля

1. Проверьте модуль успешно развернута и запущена. На **сведений об устройстве** страницы на **модули** вкладке состояние среды выполнения модуля должно быть **под управлением**.  
2. Подключитесь к модуль веб-сервера приложения. Откройте окно браузера и введите:

    `http://<compute-network-IP-address>:8080`

    Вы увидите, что приложение веб-сервера запущена.

    ![Проверка подключения к модулю через указанный порт](media/data-box-edge-extend-compute-access-modules/verify-connect-module-1.png)
