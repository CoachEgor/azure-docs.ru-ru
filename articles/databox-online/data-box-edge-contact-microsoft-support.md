# <a name="open-a-support-ticket-for-azure-data-box-edge-and-azure-data-box-gateway"></a>Откройте вспомогательный билет для Azure Data Box Edge и шлюза Ящика данных Azure

Эта статья относится к Azure Data Box Edge и шлюзу коробки данных Azure, которые управляются службой Azure Data Box Edge/Azure Data Box Gateway. Если у вас возникли какие-либо проблемы с службой, можно создать запрос на техническую поддержку. В этой статье описаны следующие операции.

* Создание запроса на техническую поддержку.
* Управление жизненным циклом запроса на техническую поддержку на портале.

## <a name="create-a-support-request"></a>Создание запроса на обслуживание

Для создания запроса на обслуживание выполните следующие действия.

1. Перейдите к порядку Data Box Edge или Data Box Gateway. Перейдите в **раздел Поддержки и устранения неполадок,** а затем выберите **новый запрос поддержки.**
   
2. В **новом запросе поддержки,** на вкладке **Основы,** принять следующие меры:
    
    1. В раскрывающемся списке **Тип проблемы** выберите **Техническая**.
    2. Выберите **подписку**.
    3. В разделе **Служба** установите флажок **Мои службы**. Из списка выпадающих данных выберите **Data Box Edge и шлюз Box Data.**
    4. Выберите **ресурс.** Это соответствует названию вашего заказа.
    5. Краткое **резюме** проблемы, с которую вы сталкиваетесь. 
    6. Выберите **тип проблемы**.
    7. На основе выбранного типа проблемы выберите соответствующий **подтип проблемы.**
    8. Выберите **далее: Решения >>**.

        ![Основы](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-1.png)

3. На вкладке **«Подробности»** следующие шаги:
    
    1. Укажите дату и время начала для проблемы.
    2. Предоставьте **описание** для вашей проблемы.
    3. В **загрузке файла**выберите значок папки для просмотра любых других файлов, которые вы хотите загрузить.
    4. Установите флажок **Предоставить диагностическую информацию**.
    5. На основании подписки **план поддержки** автоматически заполняется.
    6. Из списка выпадающих данных выберите **severity.**
    7. Укажите **предпочтительный метод контакта**.
    8. **Часы отклика** автоматически выбираются на основе плана подписки.
    9. Предоставьте язык, который предпочитается для поддержки.
    10. В **Контактной информации**укажите свое имя, электронную почту, телефон, дополнительный контакт, страну/регион. Служба технической поддержки Майкрософт будет использовать эту информацию, чтобы связаться с вами для получения дополнительных сведений, диагностики и устранения проблемы. 
    11. Выберите **далее: Обзор и создайте >>**.

        ![Проблема](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-2.png)

4. На вкладке **«Обзор » Создайте** информацию, относящуюся к билету поддержки. Выберите **Создать**. 

    ![Проблема](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-3.png)

    После того, как вы создали билет поддержки, инженер службы поддержки свяжется с вами как можно скорее, чтобы продолжить ваш запрос.

## <a name="get-hardware-support"></a>Получите аппаратную поддержку

Эта информация относится только к устройству Data Box Edge. Процесс сообщения о проблемах с оборудованием заключается в следующем:

1. Откройте билет поддержки с портала Azure для проблемы с оборудованием. Под **типом проблемы**выберите **оборудование Data Box Edge.** Выберите **подтип проблемы** как **аппаратный сбой.** 

    ![Проблема оборудования](./media/data-box-edge-contact-microsoft-support/data-box-edge-hardware-issue-1.png)

    После того, как вы создали билет поддержки, инженер службы поддержки свяжется с вами как можно скорее, чтобы продолжить ваш запрос. 

2. Если microsoft Support определяет, что это проблема с оборудованием, то возникает одно из следующих действий: 

    - Направляется группа позамену поля (FRU) для неисправной аппаратной части. В настоящее время блок питания является единственным поддерживаемым FRU. 
    - Для любой другой части сбоя, Microsoft делает полную замену системы (FSR) или устройство своп.

3. Если билет поддержки поднят до 16:30 по местному времени (с понедельника по пятницу), на следующий рабочий день на ваше место происшествия отправляется специалист по месту службы, чтобы выполнить FRU или полную замену устройства.

## <a name="manage-a-support-request"></a>Управление запросом в службу поддержки

После создания запроса вы можете управлять его жизненным циклом, не покидая портал.

#### <a name="to-manage-your-support-requests"></a>Управление запросами в службу поддержки

1. Чтобы открыть страницу справки и поддержки, последовательно щелкните **Обзор > Справка и поддержка**.

    ![Управление запросами на поддержку](./media/data-box-edge-contact-microsoft-support/data-box-edge-manage-support-request-1.png)   

2. Список **недавних запросов в службу поддержки** в виде таблицы отображается в разделе **Справка и поддержка**.

    <!--[Manage support requests](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-1.png)--> 

3. Выберите и щелкните запрос на поддержку. Для запроса можно просмотреть состояние и подробные сведения. Щелкните **+ Новое сообщение**, если нужно следить за этим запросом.
