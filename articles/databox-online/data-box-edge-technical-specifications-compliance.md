# <a name="azure-data-box-edge-technical-specifications"></a>Технические характеристики Azure Data Box Edge

Аппаратные компоненты устройства Microsoft Azure Data Box Edge соответствуют техническим спецификациям и нормативным стандартам, изложенным в этой статье. Технические характеристики описывают блоки питания (ПСУ), емкость хранилища, корпуса и экологические стандарты. 

## <a name="compute-memory-specifications"></a>Вычисления, спецификации памяти

Устройство Data Box Edge имеет следующие спецификации для вычислений и памяти:

| Спецификация           | Значение                  |
|-------------------------|----------------------------|
| ЦП    | 2 X 10 процессора ядра                     |
| Память              | ОЗУ 128 ГБ.                  |


## <a name="fpga-specifications"></a>Спецификации FPGA

Полевые программируемые ворота Array (FPGA) включены в каждое устройство Data Box Edge, которое позволяет машинное обучение (ML) сценарии. 

| Спецификация           | Значение                  |
|-------------------------|----------------------------|
| Fpga   | Intel Arria 10 <br> Доступные модели Deep Neural Network (DNN) такие же, как и [модели, поддерживаемые облачными экземплярами FPGA.](https://docs.microsoft.com/azure/machine-learning/how-to-deploy-fpga-web-service#whats-supported-on-azure)| 


## <a name="power-supply-unit-specifications"></a>Спецификации блока питания

Устройство Data Box Edge имеет два 100-240 V блоки питания (PSUs) с высокопроизводительными вентиляторами. Два PsUs обеспечивают избыток конфигурации питания. Если PSU выходит из строя, устройство продолжает нормально работать на другом PSU до тех пор, пока не будет заменен неудавшийся модуль. В следующей таблице перечислены технические характеристики ПСУ.

| Спецификация           | 750 W PSU                  |
|-------------------------|----------------------------|
| Максимальная выходная мощность    | 750 Вт                     |
| Частота               | 50/60 Гц                   |
| Выбор диапазона напряжения | Автодиапазон: 100-240 V AC |
| «Горячая» замена           | Да                        |

## <a name="storage-specifications"></a>Спецификации хранилища

Устройства Data Box Edge оснащены 9 X 2.5"NVMe SSD, каждый из которых способен 1,6 ТБ. Из этих SSD-нада, 1 является диском операционной системы, а остальные 8 дисков данных. Общая емкость устройства составляет примерно 12,5 ТБ. В следующей таблице приведены сведения о емкости устройства.

|     Спецификация                          |     Значение             |
|--------------------------------------------|-----------------------|
|    Количество твердотельных накопителей (SSD)     |    8                  |
|    Емкость одного твердотельного накопителя (SSD)                     |    1,6 TБ             |
|    Общая емкость                          |    12,8 ТБ            |
|    Общая доступная емкость*                  |    12,5 ТБ            |

**Некоторое пространство зарезервировано для внутреннего использования.*
