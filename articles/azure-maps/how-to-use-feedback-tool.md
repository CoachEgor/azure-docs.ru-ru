# <a name="provide-data-feedback-to-azure-maps"></a>Предоставление обратной связи с данными на Azure Maps

Карты Azure доступны с мая 2018 года. Azure Maps предоставляет свежие картографические данные, простые в использовании АДИ REST и мощные SDK для поддержки наших корпоративных клиентов различными типами бизнес-использования. Реальный мир меняется каждую секунду, и для нас очень важно обеспечить фактическое цифровое представление для наших клиентов. Нашим клиентам, которые планируют открыть или закрыть объекты, необходимо оперативно обновить наши карты. Таким образом, они могут эффективно планировать доставку, техническое обслуживание или обслуживание клиентов на нужных объектах. Мы создали сайт обратной связи с данными Azure Maps, чтобы дать нашим клиентам возможность предоставлять прямую обратную связь с данными. Отзывы клиентов напрямую идут к нашим поставщикам данных и их редакторам карт. Они могут быстро оценить и включить обратную связь в наши картографические продукты.  

[Сайт обратной связи Azure Maps Data](https://feedback.azuremaps.com) предоставляет нашим клиентам простой способ предоставления обратной связи с картами данных, особенно по бизнес-вопросам, представляющим интерес, и адресам проживания. В этой статье вы можете предоставить различные виды обратной связи с помощью сайта обратной связи Azure Maps.

## <a name="add-a-business-place-or-a-residential-address"></a>Добавить деловое или жилой адрес 

Вы можете предоставить обратную связь о недостающей точке интереса или жилой адрес. Это можно сделать двумя способами. Откройте сайт обратной связи с данными Azure Map, ищите координаты недостающих местоположений, а затем нажмите кнопку "Добавить место"

  ![поиск недостающих местоположений](./media/how-to-use-feedback-tool/search-poi.png)

Или вы можете взаимодействовать с картой. Нажмите на место, чтобы уронить булавку в координате и нажмите кнопку "Добавить место".

  ![добавить булавку](./media/how-to-use-feedback-tool/add-poi.png)

При нажатии, вы будете направлены на форму, чтобы предоставить соответствующие детали для места.

  ![добавить место](./media/how-to-use-feedback-tool/add-a-place.png)

## <a name="fix-a-business-place-or-a-residential-address"></a>Исправить деловое место или адрес проживания 

Сайт обратной связи также позволяет искать и находить бизнес-место или адрес. Вы можете предоставить обратную связь, чтобы исправить адрес или место контакта, если они не являются правильными. Чтобы обеспечить обратную связь для исправления адреса, используйте панель поиска для поиска бизнес-места или адреса проживания. Нажмите на местоположение вашего интереса из списка результатов. Нажмите на кнопку "Исправить это место".

  ![поиск место, чтобы исправить](./media/how-to-use-feedback-tool/fix-place.png)

Чтобы обеспечить обратную связь, чтобы исправить адрес, заполните форму "Исправить место", а затем нажмите на кнопку "Представить".

  ![исправить форму](./media/how-to-use-feedback-tool/fix-form.png)

Если место контакта для места неправильно, проверьте флажок на форме "Fix a place", в форме которого написано " Расположение штыря неправильно". Переместите штифт в нужное место, а затем нажмите кнопку "отправить".

  ![переместить месторасположен иного контакта](./media/how-to-use-feedback-tool/move-pin.png)

## <a name="add-a-comment"></a>Добавить комментарий 

В дополнение к позволяя вам искать местоположение, инструмент обратной связи также позволяет добавлять бесплатный комментарий текста формы для деталей, связанных с местоположением. Чтобы добавить комментарий, ищите местоположение или нажмите на местоположение. Нажмите кнопку "Добавить комментарий", напишите комментарий, а затем нажмите кнопку "Отправить".

  ![добавить комментарий](./media/how-to-use-feedback-tool/add-comment.png)

## <a name="track-status"></a>Отслеживание состояния 

Вы также можете отслеживать состояние вашего запроса, проверяя поле "Я хочу отслеживать статус" и предоставляя вашу электронную почту при этом запросе. Вы получите ссылку на отслеживание в письме, которая обеспечивает актуальный статус вашего запроса. 

  ![статус обратной связи](./media/how-to-use-feedback-tool/feedback-status.png)
