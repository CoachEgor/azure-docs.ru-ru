# <a name="add-a-polygon-layer-to-the-map"></a>Добавление слоя многоугольников на карту

В этой статье показано, как визуализировать области `Polygon` и `MultiPolygon` геометрии компонентов на карте с помощью слоя многоугольников. Azure Maps веб-пакет SDK также поддерживает создание круговых геометрических объектов, как определено в [расширенной схеме геоjson](extend-geojson.md#circle). Эти круги преобразуются в многоугольники при отображении на карте. Все геометрические функции также можно легко обновлять, если они заключены в [Atlas. Класс Shape](https://docs.microsoft.com/javascript/api/azure-maps-control/atlas.shape?view=azure-iot-typescript-latest) .

## <a name="use-a-polygon-layer"></a>Использование слоя многоугольников 

Если слой многоугольников подключен к источнику данных и загружен на карту, он отображает область `Polygon` и `MultiPolygon` компонентов. В следующем коде показано, как создать многоугольник, добавить его в источник данных и визуализировать его с помощью слоя многоугольников, используя класс [полигонлайер](https://docs.microsoft.com/javascript/api/azure-maps-control/atlas.layer.polygonlayer?view=azure-iot-typescript-latest) .

```javascript
//Create a data source and add it to the map.
var dataSource = new atlas.source.DataSource();
map.sources.add(dataSource);

//Create a rectangular polygon.
dataSource.add(new atlas.data.Feature(
    new atlas.data.Polygon([[
        [-73.98235, 40.76799],
        [-73.95785, 40.80044],
        [-73.94928, 40.7968],
        [-73.97317, 40.76437],
        [-73.98235, 40.76799]
    ]])
));

//Create and add a polygon layer to render the polygon to the map.
map.layers.add(new atlas.layer.PolygonLayer(dataSource, null,{
    fillColor: 'red',
    opacaty: 0.5
}));
```

## <a name="add-a-circle-to-the-map"></a>Добавление окружности к карте

Azure Maps использует расширенную версию схемы геоjson, которая предоставляет определение для кругов, как указано [здесь](extend-geojson.md#circle). Окружность можно визуализировать на карте путем создания `Point` функции со свойством `subType` со значением `"Circle"` и свойством `radius` с числом, представляющим радиус в метрах. Пример.

```javascript
{
    "type": "Feature",
    "geometry": {
        "type": "Point",
        "coordinates": [-122.126986, 47.639754]
    },
    "properties": {
        "subType": "Circle",
        "radius": 100
    }
}  
```

Azure Maps веб-пакет SDK преобразует эти `Point` функции в функции `Polygon`, которые можно будет визуализировать на карте с помощью слоев Polygon и Line, как показано в следующем примере кода.

## <a name="make-a-geometry-easy-to-update"></a>Упрощение обновления геометрии

Класс `Shape` инкапсулирует [геометрию](https://docs.microsoft.com/javascript/api/azure-maps-control/atlas.data.geometry?view=azure-iot-typescript-latest) или [функцию](https://docs.microsoft.com/javascript/api/azure-maps-control/atlas.data.feature?view=azure-iot-typescript-latest) и упрощает их обновление и поддержку. Фигуру можно создать, передав геометрию и набор свойств либо передав функцию, как показано в следующем коде.

```javascript
//Creating a shape by passing in a geometry and a object containing properties.
var shape1 = new atlas.Shape(new atlas.data.Point[0,0], { myProperty: 1 });

//Creating a shape using a feature.
var shape2 = new atlas.Shape(new atlas.data.Feature(new atlas.data.Point[0,0], { myProperty: 1 });
```

В следующем образце кода показано, как заключить объект окружности в круге на класс Shape и легко обновлять его свойство RADIUS с помощью ползунка. Когда значение радиуса изменяется в фигуре, отрисовка окружности автоматически обновляется на карте.
