# <a name="azure-maps-weather-coverage"></a>Покрытие погодой Azure Maps

В этой статье содержится информация о [поперечном](https://aka.ms/AzureMapsWeatherService)сообщении для служб погоды Azure Maps. Службы данных по погоды Azure Maps возвращают такие сведения, как радиолокационные плитки, текущие погодные условия, прогнозы погоды и погода вдоль маршрута.

Карты Azure не имеют одинакового уровня информации и точности для всех стран и регионов.

В следующей таблице содержится информация о том, какую информацию о погоде вы можете запросить у каждой страны/региона.

| Символ | Значение |
|--------|---------|
|*       |Покрывает погоду вдоль маршрута, текущие условия, почасовой прогноз, прогноз квартала и ежедневные AA прогноза |
