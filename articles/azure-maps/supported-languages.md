## <a name="azure-maps-supported-views"></a>Карты Azure поддерживается представления

Azure сопоставляет представление (также называется «параметр области пользователя») равен двухбуквенный код страны ISO 3166, который покажет, что правильный карты для этой страны или региона, указав, какой набор геополитически спорные границы и метки отображаются на карте.  По умолчанию является значение параметра View **«Единой»** .  Странах и регионах, которые не входят в список представление по умолчанию в представлении «Единой». Это необходимо определить местонахождение пользователей, а затем установите параметр View правильно к этому расположению. Параметр View в Azure сопоставляет должен использоваться в соответствии с применимым законам, включая те, о сопоставление страны, где карты, изображения и другое содержимое данных и сторонних поставщиков, права на доступ к через Azure сопоставляет становится доступным.
