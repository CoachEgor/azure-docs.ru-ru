# <a name="azure-maps-routing-coverage"></a>Покрытие для маршрутизации в службе "Карты Azure"

В этой статье приведены сведения о покрытии для маршрутизации в службе "Карты Azure". При поисковом запросе Azure Maps возвращает оптимальный маршрут из расположения А в расположение B. Вам предоставляются точное время в пути, живое обновление информации о путешествии и инструкции по маршруту. Вы также можете добавить дополнительные параметры поиска, такие как текущий трафик, тип транспортного средства и условия, чтобы избежать. Оптимизация маршрута зависит от региона. Это потому, что Azure Maps имеет различные уровни информации и точности для различных регионов. В следующей таблице перечислены регионы и какая информация вы можете запросить для них.
 
Проверьте покрытие для [**геокодирования**](geocoding-coverage.md).       
Проверьте покрытие для [**трафика**](traffic-coverage.md).  
Проверьте покрытие для [**отображения**](render-coverage.md).
<br>  
**Легенда**

| Символ             | Значение                                |
|--------------------|----------------------------------------|
| ✓                  | Страна/регион обеспечены подробными данными.   |
| Ø                  | Страна/регион обеспечены упрощенными данными. |
| Страна отсутствует. | Данные по странам/регионам не предоставляются.          |
