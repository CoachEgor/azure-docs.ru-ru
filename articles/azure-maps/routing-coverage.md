# <a name="azure-maps-routing-coverage"></a>Покрытие для маршрутизации в службе "Карты Azure"

В этой статье приведены сведения о покрытии для маршрутизации в службе "Карты Azure". При поисковых запросах Azure Maps Возвращает оптимальный маршрут из расположения а в расположение B. Вы предоставляете точные сведения о времени поездки, динамическом обновлении информации о командировках и инструкциях по маршрутизации. Чтобы избежать этого, можно также добавить дополнительные параметры поиска, такие как текущий трафик, тип автомобиля и условия. Оптимизация маршрута зависит от региона. Это обусловлено тем, что Azure Maps имеет различные уровни информации и точность для разных регионов. В следующей таблице перечислены регионы и сведения, которые можно запросить.
 
Проверьте покрытие для [**геокодирования**](geocoding-coverage.md).       
Проверьте покрытие для [**трафика**](traffic-coverage.md).  
Проверьте покрытие для [**отображения**](render-coverage.md).
<br>  
**Условные обозначения**

| Символ             | Значение                                |
|--------------------|----------------------------------------|
| ✓                  | Страна или регион, предоставленные с подробными данными.   |
| Ø                  | Страна или регион, предоставленные с упрощенными данными. |
| Страна отсутствует. | Данные о стране или регионе не указаны.          |
