# <a name="azure-maps-mobility-coverage"></a>Покрытие Azure Maps мобильности

Azure Maps предлагает [службу Mobility Service](https://aka.ms/AzureMapsMobilityService). Служба Mobility Service улучшает время разработки для приложений с общедоступными транзитными, велосипедными и мотороллере общими функциями. Функции для общедоступного транзитного пути включают поиск ближайших станций, Поиск остановок, Поиск транзитных строк, получение сведений о расписании, получение общедоступных транзитных прибытия, список прекращений и список руководств маршрутов. Служба возвращает сведения о доступности в режиме реального времени и статическую доступность для общих транспортных средств, таких как, автомобили, велосипеды и скутерс. Кроме того, служба Mobility Service предоставляет возможности маршрутизации с вариантами планирования мультимодальный поездку. Мультимодальный поездку включает в себя велотуров, Хаилинг и транзитные варианты в одном дороге. Пользователи имеют доступ к подробным сведениям и мультимодальный пошаговые расписания и оповещения службы.

Azure Maps не имеет одинакового уровня информации и точности для всех городов и стран и регионов. Возможность вызова транзитных данных зависит от города. Данные могут не включать в себя все общие транзитные варианты и агентства, обслуживающие город.

В следующей таблице приведены сведения о покрытии для службы Azure Maps Mobility Service.
