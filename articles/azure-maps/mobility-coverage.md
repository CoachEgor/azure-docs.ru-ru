# <a name="azure-maps-mobility-coverage"></a>Покрытие Azure Maps мобильности

Служба Azure Maps [Mobility Service](/rest/api/maps/mobility) повышает время разработки приложений с помощью общедоступных транзитных функций, таких как транзитная Маршрутизация и поиск ближайших общественных транзитных точек. Пользователи могут получить подробные сведения об остановках, маршрутах и расписаниях общественного транспорта. Также служба "Мобильность" позволяет пользователям получать геометрические изображения остановок и маршрутов, оповещения по остановкам, маршрутам и зонам обслуживания, а также оповещения о прибытии транспорта и работе служб в реальном времени. Кроме того, служба "Мобильность" предоставляет возможность построения маршрутов с несколькими видами перемещения. Построение маршрута с несколькими видами перемещения объединяет в одной поездке ходьбу, движение на велосипеде и (или) общественном транспорте. Также пользователь может получить подробное пошаговое описание маршрута с несколькими видами перемещения. 

Azure Maps не предоставляет одинаковый уровень информации и точности для всех городов и стран и регионов. Возможность вызова общедоступных транзитных данных зависит от области Metro. Кроме того, данные карт могут не включать все общие транзитные варианты и агентства, обслуживающие область Metro.

В следующей таблице приведены сведения о покрытии для службы Azure Maps Mobility Service.

| Символ | Значение |
|--------|---------|
| *      |Почти полный охват страны или региона.|
