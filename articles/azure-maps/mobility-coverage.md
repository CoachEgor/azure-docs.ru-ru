# <a name="azure-maps-mobility-coverage"></a>Охват мобильности Azure Maps

Azure Maps предлагает [услугу мобильности.](https://aka.ms/AzureMapsMobilityService) Служба мобильности улучшает время разработки приложений с общественным транспортом, долей велосипедов и функций обмена скутерами. Особенности общественного транспорта включают поиск близлежащих станций, поиск остановок, поиск транзитных линий, получение информации о расписании, получение в режиме реального времени общественного транспорта прибытия, список остановок, и список маршрутов руководства. Услуга возвращает информацию о доступности в режиме реального времени и статичной доступности для общих транспортных средств, таких как автомобили, велосипеды и скутеры. Кроме того, услуга Mobility предоставляет возможности разгрома с мультимодальными вариантами планирования поездок. Мультимодальное планирование поездки включает в себя велосипед, езда град и транзитные варианты, все в одну поездку. Пользователи кулачок доступ подробные и мультимодальные пошаговые маршруты и услуги оповещения.

Карты Azure не имеют одинакового уровня информации и точности для всех городов и стран/регионов. Возможность вызова транзитных данных зависит от города. И, данные не могут включать все варианты общественного транспорта и учреждений, которые обслуживают город.

В следующей таблице приводится информация об охвате службы Azure Maps Mobility.
