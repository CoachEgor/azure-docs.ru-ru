# <a name="azure-maps-mobility-coverage"></a>Покрытие Azure Maps мобильности

Служба Azure Maps [Mobility Service](https://aka.ms/AzureMapsMobilityService) повышает время разработки приложений с помощью общедоступных транзитных функций, таких как транзитная Маршрутизация и поиск ближайших общественных транзитных точек. Пользователи могут получать подробные сведения о прекращении передачи, строках и расписаниях. Служба Mobility Service также позволяет пользователям получать остановки и штриховых геометрических объектов, оповещений для остановок, линий и областей обслуживания, а также для общедоступных транзитных уведомлений в режиме реального времени и оповещений службы. Кроме того, служба Mobility Service предоставляет возможности маршрутизации с вариантами планирования мультимодальный поездку. Мультимодальный поездку включает в себя путешествие, велотуров и общие варианты транзита. Пользователи также могут получить подробные пошаговые мультимодальныйные маршруты. 

Azure Maps не предоставляет одинаковый уровень информации и точности для всех городов и стран и регионов. Возможность вызова общедоступных транзитных данных зависит от области Metro. Кроме того, данные карт могут не включать все общие транзитные варианты и агентства, обслуживающие область Metro.

В следующей таблице приведены сведения о покрытии для службы Azure Maps Mobility Service.

|--------|---------|
