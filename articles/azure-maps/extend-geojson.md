# <a name="extended-geojson-geometries"></a>Расширенные геометрические объекты геоjson

Azure Maps предоставляет список эффективных API-интерфейсов для поиска внутри географических функций или с их использованием.
Эти API стандартизации для географической [спецификации][1] , представляющей географические функции (например, границы состояний, маршруты).  

[Спецификация геоjson][1] поддерживает только следующие геометрические объекты:

* GeometryCollection;
* LineString;
* MultiLineString;
* MultiPoint;
* MultiPolygon;
* Point;
* Polygon.

Некоторые Azure Maps API (например: [Поиск внутри геометрии](https://docs.microsoft.com/rest/api/maps/search/postsearchinsidegeometry)) принимает геометрические объекты, такие как Circle, которые не являются частью [спецификации геоjson][1].

Эта статья содержит подробное описание того, как Azure Maps расширяет [спецификацию геоjson][1] для представления определенных геометрических объектов.

## <a name="circle"></a>Круг

Эта `Circle` геометрия не поддерживается в [спецификации геоjson][1]. Для представления окружности используется `GeoJSON Point Feature` объект.

Геометрический объект `Circle`, представленный с помощью объекта `GeoJSON Feature`, __должен__ содержать следующие параметры:

- По центру

    Центр окружности представлен с помощью `GeoJSON Point` объекта.

- Радиус

    Значение `radius` круга представлено с помощью свойств `GeoJSON Feature`. Значение радиуса отображается в _метрах_ и должно быть типа `double`.

- SubType

    Геометрический объект "круг" также должен содержать свойство `subType`. Это свойство должно быть частью свойств, `GeoJSON Feature`а его значение должно быть _кругом_

#### <a name="example"></a>Пример

Вот как вы будете представлять круг в центре (Широта: 47,639754, Долгота:-122,126986) с радиусом, равным 100 м, `GeoJSON Feature` с использованием объекта:

```json            
{
    "type": "Feature",
    "geometry": {
        "type": "Point",
        "coordinates": [-122.126986, 47.639754]
    },
    "properties": {
        "subType": "Circle",
        "radius": 100
    }
}          
```

## <a name="rectangle"></a>Прямоугольник

Эта `Rectangle` геометрия не поддерживается в [спецификации геоjson][1]. Для представления прямоугольника используется `GeoJSON Polygon Feature` объект. Расширение Rectangle в основном используется модулем средств рисования веб-пакета SDK.

Геометрический объект `Rectangle`, представленный с помощью объекта `GeoJSON Polygon Feature`, __должен__ содержать следующие параметры:

- Угол

    Углы прямоугольника представлены с помощью координат `GeoJSON Polygon` объекта. Должно быть пять координат, по одному для каждого угла и пятая координата, совпадающая с первой, чтобы закрыть кольцо многоугольника. Предполагается, что эти координаты будут выравниваются и повернуты по мере необходимости разработчику.

- SubType

    Фигура Rectangle также должна содержать `subType` свойство. Это свойство должно быть частью свойств, `GeoJSON Feature`а его значение должно быть _прямоугольником_

### <a name="example"></a>Пример

```json
{
    "type": "Feature",
    "geometry": {
        "type": "Polygon",
        "coordinates": [[[5,25],[14,25],[14,29],[5,29],[5,25]]]
    },
    "properties": {
        "subType": "Rectangle"
    }
}

```
