# <a name="azure-maps-render-coverage"></a>Покрытие для отображения в службе "Карты Azure"

Для создания карт в службе "Карты Azure" используются растровые и векторные плитки. При самом низком разрешении весь мир помещается в одну плитку. На самом высоком разрешении, одна плитка составляет 38 квадратных метров. Вы увидите более подробную информацию о континентах, регионах, городах и отдельных улицах при увеличении карты. Для получения дополнительной информации о плитках, [см. Увеличить уровни и плитки сетки](zoom-levels-and-tile-grid.md).

Тем не менее, Карты не имеют одинаковый уровень информации и точности для всех регионов. В следующих таблицах подробно описывается уровень информации, который можно сделать для каждого региона.

## <a name="legend"></a>Условные обозначения

| Символ | Значение |
|--------|---------|
| ✓ | Регион с подробными сведениями.   |
| Ø | Регион с общими сведениями. |
