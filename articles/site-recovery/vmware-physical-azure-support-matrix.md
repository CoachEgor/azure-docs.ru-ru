# <a name="support-matrix-for-disaster-recovery--of-vmware-vms-and-physical-servers-to-azure"></a>Таблица поддержки аварийного восстановления виртуальных машин VMware и физических серверов в Azure

В этой статье перечислены поддерживаемые компоненты и параметры для аварийного восстановления виртуальных машин VMware и физических серверов в Azure с помощью [Azure Site Recovery](site-recovery-overview.md).

- [Дополнительные сведения](vmware-azure-architecture.md) о аварийного восстановления виртуальных Машин VMware или физических сервера архитектуры.
- Выполните наши [учебники](tutorial-prepare-azure.md) опробовать аварийного восстановления.

## <a name="deployment-scenarios"></a>Сценарии развертывания

**Сценарий** | **Дополнительные сведения**
--- | ---
Аварийное восстановление виртуальных машин VMware | Репликация локальных виртуальных машин VMware в Azure. Этот сценарий можно развернуть с помощью портала Azure или [PowerShell](vmware-azure-disaster-recovery-powershell.md).
Аварийное восстановление физических серверов | Репликация локальных физических серверов Windows или Linux в Azure. Этот сценарий можно развернуть с помощью портала Azure.

- Для получения наилучших результатов рекомендуется запустить [инструмент Deployment Planner](site-recovery-deployment-planner.md)и выполнять расширенное тестирование приложения с помощью тестовые отработки отказа, чтобы получить точные сведения о производительности для вашего приложения.
