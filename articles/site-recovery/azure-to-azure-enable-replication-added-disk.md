# <a name="enable-replication-for-a-disk-added-to-an-azure-vm"></a>Включить репликацию для диска, добавленного в Azure VM


В этой статье описывается, как включить репликацию дисков данных, которые добавляются в Azure VM, который уже включен для восстановления аварийного использования в другой области Azure, используя [восстановление сайта Azure.](site-recovery-overview.md)

Включение репликации диска, добавленного в VM, поддерживается для VMs Azure с управляемыми дисками.

При добавлении нового диска в Azure VM, реплицируемый в другой регион Azure, возникает следующее:

-   Здоровье репликации для VM показывает предупреждение, и примечание на портале сообщает вам, что один или несколько дисков доступны для защиты.
-   Если вы включите защиту добавленных дисков, предупреждение исчезнет после первоначальной репликации диска.
-   Если вы решили не включать репликацию для диска, вы можете отклонить предупреждение.

![Добавлен новый диск](./media/azure-to-azure-enable-replication-added-disk/newdisk.png)



## <a name="before-you-start"></a>Перед началом работы

Эта статья предполагает, что вы уже создали аварийное восстановление для VM, к которому вы добавляете диск. Если вы этого не сможете, следуйте [инструкциям по восстановлению аварийного возмещения Azure в Azure.](azure-to-azure-tutorial-enable-replication.md)

## <a name="enable-replication-for-an-added-disk"></a>Включение репликации для добавленного диска

Для репликации добавленного диска сделайте следующее:

1. В хранилище > **Реплицированные элементы,** нажмите на VM, к которому вы добавили диск.
2. Нажмите **на диски,** а затем выберите диск данных, для которого вы хотите включить репликацию (эти диски имеют **незащищенный** статус).
3.  В **деталях диска**нажмите **Включить репликации**.

    ![Включить репликацию для добавленного диска](./media/azure-to-azure-enable-replication-added-disk/enabled-added.png)

После выполнения задания репликации включить и завершения первоначальной репликации предупреждение о работе с воспроизведением для проблемы диска удаляется.
