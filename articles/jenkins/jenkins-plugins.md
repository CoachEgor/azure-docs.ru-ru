# <a name="jenkins-plugins-for-azure"></a>Плагины Jenkins для Azure

Следующие плагины Jenkins поддерживают различные функции для использования в Azure.
  
| Линк-дин | Описание                                   |
|------------------------------------------------------------------------------|
| [Подключаемый модуль службы приложений Azure](https://plugins.jenkins.io/azure-app-service)     | Плагин Jenkins для развертывания службы приложений Azure (в настоящее время поддерживает только Web App). | 
| [Подключаемый модуль AAD](https://plugins.jenkins.io/azure-ad)                       | Плагин Jenkins, поддерживающий авторизацию аутентификации & через Azure Active Directory. | 
| [Плагин Azure Artifact Manager](https://plugins.jenkins.io/azure-artifact-manager) | Плагин Azure Artifact Manager — это artifact Manager, который позволяет хранить артефакты в хранилище Azure Blob. Плагин Azure Artifact Manager работает прозрачно для Дженкинса и ваших рабочих мест, это как менеджер артефактов по умолчанию. | 
| [Подключаемый модуль агентов контейнеров Azure](https://plugins.jenkins.io/azure-container-agents) | Плагин Azure Container Agents может помочь вам запустить контейнер в качестве агента в Дженкинсе | 
| [Плагин Azure Container Registry Task](https://plugins.jenkins.io/azure-container-registry-tasks)       | Плагин Jenkins для отправки запроса на сборку докеров в [реестр контейнеров Azure.](/azure/container-registry/container-registry-tasks-overview) |
| [Подключаемый модуль Службы контейнеров Azure](https://plugins.jenkins.io/azure-acs)       | Плагин Jenkins для развертывания конфигураций в службе контейнеров Azure (AKS). | 
| [Подключаемый модуль учетных данных Azure](https://plugins.jenkins.io/azure-credentials)      | Плагин Jenkins для управления учетными данными Azure. | 
| [Подключаемый модуль функций Azure](https://plugins.jenkins.io/azure-function)           | Плагин Jenkins для развертывания функции Azure. | 
| [Плагин Azure IoT Edge](https://plugins.jenkins.io/azure-iot-edge)           | Плагин Azure IoT Edge упрощает настройку конвейера CI/CD для разработки решения IoT Edge на Jenkins. | 
| [Подключаемый модуль Microsoft Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Плагин Jenkins для проектов Linux Azure Service Fabric. |
| [Подключаемый модуль для службы хранилища Azure](https://plugins.jenkins.io/windows-azure-storage)     | плагин для загрузки артефактов сборки или загрузки зависимостей сборки из хранилища Microsoft Azure Blob. | 
| [Плагин агентов Azure VM](https://plugins.jenkins.io/azure-vm-agents)         | Плагин Jenkins для создания агентов Jenkins в виртуальных машинах Azure (через шаблон Azure Resource Manager). | 
| [Плагин виртуального масштаба машины Azure](https://plugins.jenkins.io/azure-vmss)           | Плагин Jenkins для развертывания VM-изображений в наборах виртуальных машин Azure. |
