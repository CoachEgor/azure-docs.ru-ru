# <a name="linux-support"></a>Поддержка Linux

Вы можете использовать Linux SDK для Windows Virtual Desktop для создания автономного клиента Windows Virtual Desktop. Вы также можете использовать его для включения поддержки Windows Virtual Desktop в клиентском приложении. Это краткое руководство объяснит, что такое Linux SDK и как начать его использовать.

## <a name="what-is-the-linux-sdk"></a>Что такое Linux SDK?

Вы можете использовать AIS SDK для извлечения каналов ресурсов, подключения к настольным или удаленным службам приложений и использования многих перенаправлений, поддерживаемых нашими клиентами-первыми лицами.

> [!NOTE]
> В настоящее время Разрабатывается СДК. Мы обновим этот документ инструкциями для доступа к SDK, когда он будет доступен.

### <a name="supported-linux-distributions"></a>Поддерживаемые дистрибутивы Linux

SDK совместим с большинством операционных систем на основе Ubuntu 18.04 или позже. Если у вас другой дистрибутив Linux, мы можем работать с вами, чтобы выяснить, как наилучшим образом поддержать ваши потребности.

### <a name="feature-support"></a>Поддержка компонентов и функций

SDK поддерживает несколько подключений к настольным и удаленным сеансам приложений. Поддерживаются следующие перенаправления:

| Перенаправление       | Поддерживается |
| :---------------- | :-------: |
| Клавиатура          | &#10004;  |
| Мышь             | &#10004;  |
| Аудио в          | &#10004;  |
| Аудио выход         | &#10004;  |
| Клипборд (текст)  | &#10004;  |
| Клипборд (изображение) | &#10004;  |
| Клипборд (файл)  | &#10004;  |
| Смарт-карта         | &#10004;  |
| Привод/папка      | &#10004;  |

SDK также поддерживает несколько конфигураций дисплея монитора, до тех пор, пока мониторы, которые вы выбираете для сеанса, являются смежными.

Мы будем обновлять этот документ по мере добавления поддержки новых функций и перенаправлений. Если вы хотите предложить новые функции и другие улучшения, посетите нашу [страницу UserVoice](https://go.microsoft.com/fwlink/?linkid=2116523).

## <a name="get-started-with-the-linux-sdk"></a>Начать работу с Linux SDK

Прежде чем вы сможете разработать клиент Linux для Windows Virtual Desktop, вам нужно сделать следующие вещи:

1. Создание и развертывание среды виртуального рабочего стола Windows для тестирования или использования продукции.
2. Проверьте доступных клиентов от первого участника, чтобы ознакомиться с пользовательским интерфейсом Windows Virtual Desktop.
