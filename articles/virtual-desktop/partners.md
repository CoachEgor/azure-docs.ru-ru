## <a name="citrix"></a>Citrix

![Логотип Citrix](./media/partners/citrix.png)

Citrix является утвержденным поставщиком, который предлагает предприятиям централизованное гибридное управление виртуальными приложениями и рабочими нагрузками на настольных компьютерах в Azure, бок о бок с развертыванием на месте. Citrix Workspace с помощью службы виртуальных приложений и настольных компьютеров позволяет пользователям получать доступ к приложениям и настольным компьютерам с любого устройства, используя расширенный протокол Citrix HDX, чтобы обеспечить высокое разрешение из любого места.

Citrix расширяет значение Windows Virtual Desktop надежными корпоративными инструментами для повышения плотности и производительности пользователей, предоставления рабочих нагрузок по требованию и упрощения управления изображением и приложениями. ИТ-интерфейсы могут оптимизировать затраты с помощью интеллектуальных инструментов масштабирования, обеспечивая при этом невероятный пользовательский опыт, который тестируется на местах в сложных приложениях в различных отраслях. Кроме того, Citrix Managed Desktops — это программа Windows Virtual Desktop с поддержкой настольных компьютеров, которая обеспечивает простое облачное решение для управления для доставки виртуальных приложений и настольных компьютеров на любое устройство.

## <a name="vmware"></a>VMware

![Логотип VMware](./media/partners/vmware.png)

Облако VMware Horizon на Microsoft Azure дает организациям возможность подключить свой собственный экземпляр Microsoft Azure к простой и интуитивно понятной плоскости управления Horizon Cloud, создавая безопасное, всеобъемлющее, облачное решение для доставки виртуализированного Windows-приложения и настольные компьютеры. Выпуская Windows Virtual Desktop, VMware сотрудничает с корпорацией Майкрософт, чтобы расширить функциональность Windows Virtual Desktop для клиентов, использующих Horizon Cloud в Microsoft Azure. Эта функциональность будет доступна в Tech Preview к концу 2019 года.
 
В качестве утвержденного поставщика преимущества Windows Virtual Desktop будут распространены на VMware Horizon Cloud для клиентов Microsoft Azure, включая Windows 10 Enterprise для многосессионных, Windows 7 с бесплатными расширенными обновлениями безопасности на срок до трех лет, а также FSLogix. Кроме того, клиенты получат функции корпоративного класса Horizon Cloud, такие как широкая поддержка конечных точек; глобальное, облачно-оптимизированное брокерство; гибкие параметры и конфигурации рабочего стола; управление пользовательской средой; и поддержка гибридных сред.

- [Зарегистрируйтесь для технического просмотра облака Horizon в Microsoft Azure.](https://www.vmware.com/learn/323526_TECH_PREVIEW_REG.html)
