# <a name="connect-from-a-web-browser"></a>Подключение из веб-браузера

Веб-клиента позволяет получить доступ к ресурсам Апробация виртуального рабочего стола Windows в веб-браузере без длительных установку.

>[!NOTE]
>Веб-клиент не имеет в настоящее время поддерживает мобильного операционной системы.

## <a name="supported-operating-systems-and-browsers"></a>Поддерживаемые операционные системы и браузеры

Хотя любой HTML5 с поддержкой браузера должны работать, мы официально поддерживает следующие операционные системы и браузеры.

| Browser           | Поддерживаемая ОС                     | Примечания               |
|-------------------|----------------------------------|---------------------|
| Microsoft Edge    | Windows                          |                     |
| Internet Explorer | Windows                          |                     |
| Apple Safari;      | macOS                            |                     |
| Mozilla Firefox   | Windows, macOS, Linux            | 55 или более поздней версии |
| Google Chrome     | Windows, macOS, Linux, Chrome OS |                     |

## <a name="access-remote-resources-feed"></a>Доступа к удаленным ресурсам веб-канала

В браузере перейдите к [виртуальный рабочий стол Windows веб-клиент](https://rdweb.wvd.microsoft.com/webclient) и войдите с помощью учетной записи пользователя.

>[!NOTE]
>Если вы уже вошли с использованием Azure Active Directory учетную запись от того, что вы хотите использовать для виртуального рабочего стола Windows, следует выйти из системы или использовать окно браузера в конфиденциальном режиме.

После входа вы увидите список ресурсов. Ресурсы можно запустить, выбрав их так же, как обычное приложение в **все ресурсы** вкладки.
