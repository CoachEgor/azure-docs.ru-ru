# <a name="storage-options-for-fslogix-profile-containers-in-windows-virtual-desktop"></a>Варианты хранения для контейнеров профилей Фслогикс в виртуальном рабочем столе Windows

Azure предлагает несколько решений для хранения данных, которые можно использовать для хранения контейнера профиля Фслогикс. В этой статье сравниваются решения хранилища, предоставляемые Azure для контейнеров профилей пользователей Фслогикс виртуальных рабочих столов Windows.

Виртуальный рабочий стол Windows предлагает контейнеры профилей Фслогикс в качестве рекомендуемого решения для профилей пользователей. Фслогикс предназначен для перемещения профилей в удаленных вычислительных средах, таких как виртуальные рабочие столы Windows. При входе в систему этот контейнер динамически вкладывается в вычислительную среду с помощью встроенного виртуального жесткого диска (VHD) и виртуального жесткого диска Hyper-V (VHDX). Профиль пользователя доступен сразу и отображается в системе точно так же, как профиль собственного пользователя.

В следующих таблицах сравниваются решения хранилища Azure для хранения профилей пользователей контейнера профилей Фслогикс для виртуальных рабочих столов Windows.
