# <a name="connect-with-the-ios-client"></a>Подключение с помощью клиента iOS

> Применяется к: iOS 13.0 или позже. Совместим с iPhone, iPad и iPod touch.

Вы можете получить доступ к ресурсам Windows Virtual Desktop с вашего устройства iOS с нашим загружаемым клиентом. Это руководство расскажет вам, как настроить клиент iOS.

## <a name="install-the-ios-client"></a>Установка клиента iOS

Чтобы начать работу, [загрузите](https://aka.ms/rdios) и установите клиент на устройстве iOS.

## <a name="subscribe-to-a-feed"></a>Подписка на веб-канал

Подпишитесь на канал, предоставленный вашим админом, чтобы получить список управляемых ресурсов, к которым можно получить доступ на устройстве iOS.

Чтобы подписаться на канал:

1. В центре подключения, нажмите **+**, а затем нажмите **Добавить рабочее пространство**.
2. Введите URL-адрес ленты в поле **URL-адреса Feed.** URL-адрес омыла может быть URL-адресом или адресом электронной почты.
   - Если вы используете URL-адрес, используйте тот, который вам дал ваш админ. Как правило, <https://rdweb.wvd.microsoft.com>URL-адрес .
   - Чтобы использовать электронную почту, введите свой адрес электронной почты. Это говорит клиенту искать URL,связанный с вашим адресом электронной почты, если ваш администратор настроил сервер таким образом.
3. Нажмите **далее**.
4. При появлении запроса предоставьте свои учетные данные.
   - Для **имени пользователя,** дать имя пользователя с разрешением на доступ к ресурсам.
   - Для **Password,** дать пароль, связанный с именем пользователя.
   - Вам также может быть предложено предоставить дополнительные факторы, если ваш админ настроен аутентификации таким образом.
5. Коснитесь элемента **Сохранить**.

После этого Центр подключения должен отображать удаленные ресурсы.

После того, как он подписался на канал, содержимое канала будет обновляться автоматически на регулярной основе. Ресурсы могут быть добавлены, изменены или удалены на основе изменений, внесенных администратором.
