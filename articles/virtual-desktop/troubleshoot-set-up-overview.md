# <a name="troubleshooting-overview-feedback-and-support"></a>Общие сведения об устранении неисправностей, отзывы и поддержка

Статья содержит обзор проблем, которые могут возникнуть при настройке среды клиента виртуального рабочего стола Windows и предоставляет способы устранения проблем.

## <a name="provide-feedback"></a>Отзывы

В период предоставления предварительной версии Виртуального рабочего стола Windows мы не принимаем запросы в службу поддержки. Посетите [техническое сообщество Виртуального рабочего стола Windows](https://techcommunity.microsoft.com/t5/Windows-Virtual-Desktop/bd-p/WindowsVirtualDesktop), чтобы обсудить службу "Виртуальный рабочий стол Windows" с группой разработчиков и активными членами сообщества.

## <a name="escalation-tracks"></a>Отслеживает эскалации

Используйте следующую таблицу для определения и устранения проблем, которые могут возникнуть при настройке среды клиента, с помощью клиента удаленного рабочего стола. После настройки вашего клиента и регистрации, вы можете использовать наш новый [службу диагностики](https://docs.microsoft.com/azure/virtual-desktop/diagnostics-role-service) для выявления проблем для распространенных сценариев.

>[!NOTE]
>В период предоставления предварительной версии Виртуального рабочего стола Windows мы не принимаем запросы в службу поддержки. Каждый раз, когда мы называем поддержки виртуального рабочего стола Windows, перейдите в нашем форуме сообщество Tech Community теперь. Посетите [Tech Community виртуального рабочего стола Windows](https://techcommunity.microsoft.com/t5/Windows-Virtual-Desktop/bd-p/WindowsVirtualDesktop) обсудить проблемы, связанные с группой разработчиков и членами сообщества active. Если вам требуется устранить проблему в службе поддержки, включают идентификатор действия и приблизительный интервал времени для возникновения проблемы.

|----------------------------------------------------------------------|-------------------------------------------------|

Если проблему разрешить не удается, обратитесь в службу [GitHub поддержки](https://github.com/contact). <br> <br> Если ошибка возникает после доступа к шаблона в GitHub, обратитесь в службу [поддержки Azure](https://azure.microsoft.com/support/options/).|

| Управление средой узла сеанса виртуального рабочего стола Windows на портале Azure    | Обратитесь к **поддержки Azure**. <br> <br> По вопросам управления при использовании PowerShell виртуального рабочего стола удаленного рабочего стола служб/Windows, см. в разделе [виртуального рабочего стола PowerShell Windows](troubleshoot-powershell.md) или обратитесь в службу **поддержки удаленного рабочего стола служб/Windows виртуального рабочего стола** . |

| Проблемы при обнаружении веб-канала из-за сети                                            | Пользователям нужно обратиться к администратору сети. |

| Скорость реагирования удаленные приложения или рабочего стола                                      | Если проблемы связаны с конкретным приложением или продукта, обратитесь в службу отвечает на этот продукт. |
| Лицензирования сообщения или ошибки                                                          | Если проблемы связаны с конкретным приложением или продукта, обратитесь в службу отвечает на этот продукт. |
