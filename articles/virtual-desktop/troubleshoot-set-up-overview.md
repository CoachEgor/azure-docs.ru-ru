# <a name="troubleshooting-overview-feedback-and-support"></a>Общие сведения об устранении неисправностей, отзывы и поддержка

>[!IMPORTANT]
>Это содержимое применимо к обновлению за весну 2020 года с объектами Azure Resource Manager для Виртуального рабочего стола Windows. Если вы используете выпуск Виртуального рабочего стола Windows за осень 2019 года без объектов Azure Resource Manager, см. [эту статью](./virtual-desktop-fall-2019/troubleshoot-set-up-overview-2019.md).
>
> Обновление Виртуального рабочего стола Windows за весну 2020 года пока предоставляется как общедоступная предварительная версия. без соглашений об уровне обслуживания. Мы не рекомендуем использовать ее для выполнения производственных рабочих нагрузок. Некоторые функции могут не поддерживаться или их возможности могут быть ограничены. 
> Дополнительные сведения см. в статье [Дополнительные условия использования предварительных выпусков Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

В этой статье приводятся общие сведения о проблемах, которые могут возникнуть при настройке среды виртуальных рабочих столов Windows, а также способы их устранения.

## <a name="report-issues-during-public-preview"></a>Сообщать о проблемах в общедоступной предварительной версии

Чтобы сообщить о проблемах и предложить их в общедоступной предварительной версии для выпусков пружины 2020, посетите [сообщество виртуальных рабочих столов Windows](https://techcommunity.microsoft.com/t5/Windows-Virtual-Desktop/bd-p/WindowsVirtualDesktop). Вы можете использовать сообщество сообщества, чтобы обсудить рекомендации или предложить и проголосовать за новые функции. При отдаче сообщения о проблеме, связанной с общедоступным выпуском, убедитесь, что тип проблемы обозначен как **Обновление пружины 2020 (Предварительная версия)**.

Когда вы отправляете запрос на получение справки или предлагаете новую функцию, обязательно опишите этот раздел как можно более подробный. Подробные сведения могут помочь другим пользователям ответить на ваш вопрос или разобраться в функции, для которой вы предлагаете голосование.

## <a name="escalation-tracks"></a>Дорожки эскалации

Прежде чем выполнять какие-либо другие действия, проверьте [страницу состояния Azure](https://status.azure.com/status) и [работоспособность службы Azure](https://azure.microsoft.com/features/service-health/) , чтобы убедиться, что служба Azure работает правильно.

Используйте следующую таблицу для определения и устранения проблем, с которыми вы можете столкнуться при настройке среды с помощью клиента удаленный рабочий стол. После настройки среды можно использовать нашу новую [службу диагностики](diagnostics-role-service.md) для определения проблем в распространенных сценариях.
