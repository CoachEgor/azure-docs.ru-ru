# <a name="connect-with-the-macos-client"></a>Подключение с помощью клиента macOS

> Применяется к: macOS 10.12 или позже

Вы можете получить доступ к ресурсам Windows Virtual Desktop с устройств macOS с нашим загружаемым клиентом. Это руководство расскажет вам, как настроить клиента.

## <a name="install-the-client"></a>Установка клиента

Чтобы начать работу, [загрузите](https://apps.apple.com/app/microsoft-remote-desktop/id1295203466?mt=12) и установите клиент на вашем macOS устройстве.

## <a name="subscribe-to-a-feed"></a>Подписка на веб-канал

Подпишитесь на канал, который дал вам ваш админ, чтобы получить список управляемых ресурсов, доступных вам на вашем macOS устройстве.

Чтобы подписаться на канал:

1. Выберите **Добавить канал** на главной странице, чтобы подключиться к службе и получить ресурсы.
2. Введите значение параметра Feed URL (URL-адрес веб-канала). Это может быть URL-адрес или адрес электронной почты.
   - Если вы используете URL-адрес, используйте тот, который вам дал ваш админ. Как правило, <https://rdweb.wvd.microsoft.com>URL-адрес .
   - Чтобы использовать электронную почту, введите свой адрес электронной почты. Это говорит клиенту искать URL,связанный с вашим адресом электронной почты, если ваш администратор настроил сервер таким образом.
3. Выберите **Подписку**.
4. При появлении запроса войдите в систему со своей учетной записью.

После вхува в список доступных ресурсов.

После того как вы подписались на канал, содержимое канала будет обновляться автоматически на регулярной основе. Ресурсы могут быть добавлены, изменены или удалены на основе изменений, внесенных администратором.
