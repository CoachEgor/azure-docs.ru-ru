# <a name="how-to-use-tiles"></a>Как использовать плитки
Плитки можно использовать для настройки панели мониторинга приложений, панели мониторинга устройства, и устройство устанавливает панелей мониторинга. Можно добавить несколько плиток на панели мониторинга и настраивать эти плитки для отображения сведений, необходимых приложению. Можно также изменить размер плиток и настроить макет на любую панель мониторинга. Приведенном ниже снимке экрана показана панель мониторинга приложения с помощью различных плиток.

![Панель мониторинга приложений](media/howto-use-tiles/image1a.png)


В следующей таблице перечислены использование плиток в Azure IoT Central:

 
| Плитка | панель мониторинга | Описание
| ----------- | ------- | ------- |
| Ссылка | Приложения и устройства набор панелей мониторинга |Ссылка плиток: щелчка плитки, которые отображают текст, заголовок и описание. Используйте плитку ссылку, чтобы включить пользователю перейти на URL-адрес, который связан с вашего приложения. |
| Image | Приложения и устройства набор панелей мониторинга |Плитки изображения; отображать пользовательский образ и может быть активным. Добавление графических элементов на панели мониторинга и при необходимости пользователи могут перейти на URL-адрес, относящийся к приложению, используйте плитке с изображением.|
| Метка | Панели мониторинга приложения |Метка плитки содержат текст на панели мониторинга. Можно выбрать размер текста. Использовать плитку метку для добавлять соответствующие сведения на панель мониторинга такого описания, контактные сведения или справки|
| Сопоставление | Приложения и устройства набор панелей мониторинга |Мозаичных элементов отображения расположение и состояние устройства на карте. Например можно отобразить где устройства и ли включается вентилятора.|
| График | Панели мониторинга приложения и устройства |Плитках с графиками отобразить диаграмму агрегатные измерения для устройства в течение периода времени. Например можно отобразить график, показывающий среднее значение температуры и давления на устройстве за последний час|
| Линейчатая диаграмма | Панели мониторинга приложения и устройства |Линейчатая диаграмма плитки отображения диаграммы статистических показателей для устройства в течение периода времени. Например можно отобразить линейчатую диаграмму, показывающая среднее значение температуры и давления на устройстве за последний час |
| Журнал событий | Панели мониторинга приложения и устройства |Плитки событий журнала отображает события для устройства за определенный период времени. Например его можно использовать чтобы показать, что все изменения температуры произошла для устройства в течение последнего часа. |
| State History ("Журнал состояний"). | Панели мониторинга приложения и устройства |Журнал состояний плитки отображаются значения измерения в течение периода времени. Например его можно использовать для отображения значения температуры для устройства в течение последнего часа.|
| Ключевой показатель эффективности | Панели мониторинга приложения и устройства | На плитке ключевого показателя Эффективности отображается статистические данные телеметрии или событие измерения в течение периода времени. Например его можно использовать для отображения максимальной температуры, связаться для устройства во время последний час.|
| Последнее известное значение | Панели мониторинга приложения и устройства |Последнее известное значение плитке отображается последнее значение для измерения телеметрии или состояния. Например этот элемент можно использовать для отображения самых последних измерениях, температуры, давления и влажности для устройства.|
| Сетка набора устройств | Приложения и устройства набор панелей мониторинга | Устройство устанавливает Сетка отображает сведения о наборе устройств. Используйте плитку сетки устройство, чтобы показать сведения, такие как имя, расположение и модель всех устройств в наборе устройств.|


Дополнительные сведения о том, как настроить панель мониторинга в приложении центра Интернета вещей Azure, см. в разделе [настроить панель мониторинга приложений](howto-configure-homepage.md).
