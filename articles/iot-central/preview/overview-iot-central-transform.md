# <a name="transform-your-iot-data-preview-features"></a>Преобразование данных Интернета вещей (предварительные версии функций)

[!INCLUDE [iot-central-pnp-original](../../../includes/iot-central-pnp-original-note.md)]

В качестве платформы приложений IoT Central предоставляет несколько ключевых аспектов, которые помогут вам преобразовать данные Интернета вещей в бизнес-аналитику, которая помогает выполнять действия. IoT Central предоставляет способы расширения данных Интернета вещей на внешние системы для создания интеграции с бизнес-приложениями и пользовательскими приложениями. Вы можете отслеживать работоспособность и операции устройств, создавая правила, которые уведомляют специалистов через мобильную службу. Вы можете создать конкретные бизнес-аналитики с помощью пользовательской аналитики и машинного обучения, экспортировав необработанные данные телеметрии в службы в Azure. Вы можете создавать службы и средства для отслеживания устройств и управления ими, а также управлять IoT Central приложением с помощью общедоступных API. 

![Обзор преобразования в IoT Central](media/overview-iot-central-transform/transform.png)

## <a name="monitor-device-health-and-operations-using-rules"></a>Мониторинг работоспособности и операций устройств с помощью правил
После подключения компьютеров и отправки данных можно указать, на каких компьютерах возникают проблемы, или отправить сообщения об ошибках, чтобы их можно было исправить для нормального запуска с минимальным временем простоя. Вы можете создавать правила в приложении IoT Central, чтобы отслеживать данные телеметрии, отправляемые устройствами, и оповещать о превышении порога или отправке определенного сообщения о событии. Вы можете настроить такие действия, как действия электронной почты и веб-перехватчики для правил, чтобы уведомить нужных людей и правые подчиненные системы. Дополнительные сведения о правилах см. здесь.

## <a name="run-custom-analytics-and-processing-on-your-exported-data"></a>Выполнение пользовательской аналитики и обработка экспортированных данных
Вы можете создавать очень настраиваемые бизнес-аналитики, такие как определение тенденций эффективности машин или прогнозирование будущего потребления энергии в цехе, путем создания настраиваемых конвейеров аналитики для обработки необработанных данных телеметрии IoT и сохранения конечного результата. Вы можете создавать экспорты данных в приложении IoT Central для экспорта телеметрии, свойств устройств и жизненного цикла, а также изменять шаблоны устройств в другие службы Azure, чтобы можно было анализировать, хранить и визуализировать данные в наиболее ценных средствах. Дополнительные сведения об экспорте данных см. здесь.

## <a name="build-custom-iot-solutions-and-integrations-with-apis"></a>Создание пользовательских решений IoT и интеграции с помощью API
Вы можете создавать решения IoT, такие как мобильные приложения, которые могут удаленно управлять устройствами и настраивать новые устройства, а также пользовательские интеграции с существующими бизнес-приложениями для взаимодействия с устройствами Интернета вещей и данными, адаптированными для вашего бизнеса. IoT Central можно использовать в качестве магистрали для моделирования устройств, адаптации, управления и доступа к данным. Дополнительные сведения об общедоступном API в этом модуле обучения.
