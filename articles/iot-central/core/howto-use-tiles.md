# <a name="how-to-use-tiles"></a>Использование плиток
Плитки можно использовать для настройки панелей мониторинга приложений, панелей мониторинга устройств и наборов панелей устройств. На панель мониторинга можно добавить несколько плиток и настроить эти плитки, чтобы отобразить сведения, относящиеся к приложению. Можно также изменить размеры плиток и настроить макет на любой панели мониторинга. На приведенном ниже снимке экрана показана панель мониторинга приложения с различными плитками.

![Панель мониторинга приложений](media/howto-use-tiles/image1a.png)


В следующей таблице перечислены сведения об использовании плиток в Azure IoT Central.

 
| Плитка | панель мониторинга | Описание
| ----------- | ------- | ------- |
| Ссылка | Панели мониторинга наборов приложений и устройств |Плитки ссылок — это щелкающие плитки, которые отображают текст заголовка и описания. Используйте плитку ссылки, чтобы предоставить пользователю возможность переходить по URL-адресу, связанному с вашим приложением. |
| Образ — | Панели мониторинга наборов приложений и устройств |Плитки изображений отображают пользовательский образ и могут быть доступны для щелчка. С помощью плитки изображения можно добавить графику на панель мониторинга и при необходимости разрешить пользователю переходить по URL-адресу, связанному с вашим приложением.|
| Метка | Панели мониторинга приложений |Плитки меток отображают пользовательский текст на панели мониторинга. Можно выбрать размер текста. Используйте плитку метки, чтобы добавить нужные сведения на панель мониторинга, такие как описания, контактные данные или справку.|
| Сопоставление | Панели мониторинга наборов приложений и устройств |Плитки карт отображают расположение и состояние устройства на карте. Например, можно отобразить место, где находится устройство, а также включить ли вентилятор.|
| График | Панели мониторинга приложений и устройств |Плитки на графике отображают диаграмму совокупного измерения для устройства за период времени. Например, можно отобразить график, показывающий среднюю температуру и давление на устройстве за последний час.|
| Линейчатая диаграмма | Панели мониторинга приложений и устройств |На плитках линейчатой диаграммы отображается диаграмма агрегатных измерений для устройства за период времени. Например, можно отобразить линейчатую диаграмму, показывающую среднюю температуру и давление на устройстве за последний час. |
| Журнал событий | Панели мониторинга приложений и устройств |Плитка журнала событий отображает события для устройства за определенный период времени. Например, его можно использовать для отображения всех изменений температуры, произошедших для устройства за последний час. |
| State History ("Журнал состояний"). | Панели мониторинга приложений и устройств |Плитка журнала состояний отображает значения измерения за период времени. Например, его можно использовать для отображения значений температуры устройства за последний час.|
| Ключевой показатель эффективности | Панели мониторинга приложений и устройств | Плитка ключевого показателя эффективности отображает статистическую функцию телеметрии или измерение событий за период времени. Например, его можно использовать для отображения максимальной температуры для устройства за последний час.|
| Последнее известное значение | Панели мониторинга приложений и устройств |Плитка последней известного значения отображает Последнее значение для измерения телеметрии или состояния. Например, эту плитку можно использовать для отображения самых последних измерений температуры, давления и влажности для устройства.|
| Сетка набора устройств | Панели мониторинга наборов приложений и устройств | Сетка набор устройств отображает сведения о наборе устройств. Используйте плитку "Сетка набора устройств" для отображения таких сведений, как имя, расположение и модель всех устройств в наборе устройств.|


Дополнительные сведения о настройке панели мониторинга в приложении IoT Central Azure см. в статье [Добавление плиток на панель мониторинга](./howto-add-tiles-to-your-dashboard.md).
