# <a name="connect-a-rigado-cascade-500-gateway-device-to-your-azure-iot-central-application"></a>Подключите шлюз устройства Rigado Cascade 500 к приложению Azure IoT Central


В этой статье описывается, как, будучи разработчиком решений, можно подключить шлюз-устройство Rigado Cascade 500 к приложению Microsoft Azure IoT Central. 

## <a name="what-is-cascade-500"></a>Что такое Каскад 500?

Cascade 500 IoT шлюз является аппаратное предложение от Ригадо, который включен в качестве части их Каскад Edge-как-сервис решение. Он предоставляет коммерческие ioT-проектные и продуктовые команды с гибкой вычислительной мощностью края, надежной контейнерной средой приложений и широким спектром вариантов подключения к беспроводным устройствам, включая Bluetooth 5, LTE, & Wi-Fi.

Cascade 500 предварительно сертифицирован для Azure IoT Plug and Play (предварительный просмотр), что позволяет нашим разработчикам решений легко сесть на борт устройства в свои до конечные решения. Шлюз Cascade позволяет беспроводное подключение к различным датчикам мониторинга состояния, которые находятся в непосредственной близости от устройства шлюза. Эти датчики могут быть на борту в IoT Central с помощью шлюза устройства.

## <a name="prerequisites"></a>Предварительные требования
Чтобы пройти через это руководство, вам нужны следующие ресурсы:

* Устройство Rigado Cascade 500. Для получения дополнительной информации, пожалуйста, посетите [Ригадо](https://www.rigado.com/).
* Приложение Azure IoT Central. Для получения дополнительной информации смотрите [создание нового приложения](./quick-deploy-iot-central.md).

## <a name="add-a-device-template"></a>Добавление шаблона устройства

Для того, чтобы на борт шлюза Cascade 500 войти в экземпляр приложения Azure IoT Central, необходимо настроить соответствующий шаблон устройства в приложении.

Чтобы добавить шаблон устройства Cascade 500: 

1. Перейдите на вкладку ***шаблонов устройств*** в левом ![стеле, выберите **новый:** Создайте новый шаблон устройства](./media/howto-connect-rigado-cascade-500/device-template-new.png)
1. Страница дает возможность ***создать пользовательский шаблон*** или использовать ***заранее настроенный шаблон устройства***
1. Выберите шаблон устройства C500 из списка предварительно наверстных ![шаблонов устройств, как показано ниже: Выберите шаблон устройства C500](./media/howto-connect-rigado-cascade-500/device-template-preconfigured.png)
1. Выберите ***следующий: Настроить,*** чтобы продолжить следующий шаг. 
1. На следующем экране выберите ***Создать*** на борту шаблона устройства C500 в вашем приложении IoT Central.

## <a name="retrieve-application-connection-details"></a>Извлечение сведений о подключении приложения

Теперь для подключения устройства Каскад 500 необходимо получить **идентификатор scope ID** и **основной ключ** для приложения Azure IoT Central. 

1. Перейдите к **администрации** в левом стеле и нажмите на **соединение устройства.** 
2. Запишите **идентификатор Scope ID** для вашего приложения IoT Central.
![Идентификатор области приложения](./media/howto-connect-rigado-cascade-500/app-scope-id.png)
3. Теперь нажмите на **View Keys** и сделайте примечание **ключевого основного**
![ключа Первоначального общества](./media/howto-connect-rigado-cascade-500/primary-key-sas.png)  

## <a name="contact-rigado-to-connect-the-gateway"></a>Свяжитесь с Ригадо для подключения шлюза 

Для подключения устройства Cascade 500 к приложению IoT Central необходимо связаться с Ригадо и предоставить им сведения о подключении к приложению из вышеуказанных шагов. 

Как только устройство подключено к Интернету, Rigado сможет нажать вниз обновление конфигурации вниз к устройству шлюза Каскад 500 через безопасный канал. 

Это обновление будет применяться IoT Центральной детали соединения на устройстве Каскад 500 и он появится в списке устройств. 

![Первичный ключ](./media/howto-connect-rigado-cascade-500/devices-list-c500.png)  

Теперь вы готовы использовать устройство C500 в приложении IoT Central!
