# <a name="use-device-groups-in-your-azure-iot-central-application-preview-features"></a>Использование групп устройств в приложении IoT Central Azure (Предварительная версия компонентов)

[!INCLUDE [iot-central-pnp-original](../../../includes/iot-central-pnp-original-note.md)]

В этой статье описывается, как оператор, чтобы использовать группы устройств в приложении IoT Central Azure.

Группа устройств — это список устройств, сгруппированных вместе, так как они соответствуют некоторым указанным критериям. Группы устройств помогают управлять, визуализировать и анализировать устройства в масштабе, объединяя устройства в логические группы. Например, можно создать группу устройств, чтобы получить список всех устройств кондиционирования воздуха в Сиэтле, чтобы дать специалисту возможность найти устройства, на которых они отвечают. В этой статье показано, как создать и настроить группы устройств.

>  [!NOTE] 
> Для Azure IoT Edge устройств необходимо выбрать Azure IoT Edge шаблоны для создания группы устройств.

## <a name="create-a-device-group"></a>Создание группы устройств

Чтобы создать группу устройств, выполните следующие действия.

1. В области слева выберите **группы устройств** .

1. Выберите **+ создать**.

    ![Создать группу устройств](media/howto-use-device-groups-pnp/image1.png)

1. Присвойте группе устройств имя, уникальное для всего приложения. Вы также можете добавить описание. Группа устройств может содержать только устройства из одного шаблона устройства. Выберите шаблон устройства, который будет использоваться для этой группы.

1. Создайте запрос для обнаружения устройств для группы устройств, выбрав свойство, оператор сравнения и значение. Вы можете добавить несколько запросов и устройств, соответствующих **всем** критериям, в группу устройств. Созданная группа устройств доступна всем, кто имеет доступ к приложению, чтобы любой пользователь мог просматривать, изменять или удалять группу устройств.

    ![Запрос группы устройств](media/howto-use-device-groups-pnp/image2.png)

    > [!NOTE]
    > Группа устройств является динамическим запросом. Каждый раз, когда вы просматриваете список устройств, в нем могут содержаться разные устройства. Список зависит от того, какие устройства в настоящее время соответствуют критериям запроса.

1. Нажмите **Сохранить**.
