# <a name="manage-your-personal-application-preferences"></a>Управление личными настройками приложений

*Эта статья предназначена для операторов, разработчиков и администраторов.*

IoT Central позволяет настраивать приложения в соответствии со своими требованиями. Мы также обеспечиваем определенную гибкость, позволяя каждому пользователю настраивать собственное представление. В этой статье описываются различные параметры настройки, которые пользователь может применить в своем профиле.

## <a name="changing-language"></a>Изменение языка

IoT Central поддерживается на нескольких языках. Вы можете переключить предпочитаемый язык с помощью **средства выбора языка** на значке параметров на верхней панели навигации. После изменения языка IoT Central запоминает выбор и применяет его для всех приложений. Настройка в приложениях, таких как изображения информационной панели, не локализована.

## <a name="changing-theme"></a>Изменение темы

Мы поддерживаем темную и светлую темы. Хотя светлой темой является значение по умолчанию, можно изменить тему, щелкнув значок Параметры на верхней панели навигации.

![Средство выбора темы в IoT Central](media/howto-manage-preferences/settings.png)

> [!NOTE]
> Возможность выбора между светлой и темной темами будет недоступен, если администратор настроил пользовательскую тему для приложения.
