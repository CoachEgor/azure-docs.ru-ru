# <a name="configure-the-application-dashboard"></a>Настройка панели мониторинга приложения

**Панель мониторинга** — это страница, которая загружается, когда пользователи, имеющие доступ к приложению, перемещаются по URL-адресу приложения. Если вы создали приложение из одного из **шаблонов приложений,** ваше приложение будет иметь заранее определенную панель мониторинга для запуска. Если вы создали приложение из шаблона **приложения Legacy,** панель мониторинга будет застойной.

> [!NOTE]
> Пользователи могут [создавать несколько панелей мониторинга](howto-create-personal-dashboards.md) в дополнение к панели мониторинга приложения по умолчанию. Эти панели мониторинга могут быть личными только для пользователя или совместно со всеми пользователями приложения. 

## <a name="add-tiles"></a>Добавление плиток

На следующем скриншоте показана панель мониторинга в приложении, созданном из шаблона **пользовательского приложения.** Чтобы настроить панель мониторинга по умолчанию для приложения, выберите **«Отображьте»** в левом верхнем виде страницы.

> [!div class="mx-imgBorder"]
> ![Панель мониторинга для приложений на основе шаблона "Образец Контосо"](media/howto-add-tiles-to-your-dashboard/dashboard-sample-contoso.png)

Selecting **Edit** opens the dashboard library panel. Библиотека содержит плитки и примитивы приборной панели, которые можно использовать для настройки панели мониторинга.

> [!div class="mx-imgBorder"]
> ![Библиотека панелей мониторинга](media/howto-add-tiles-to-your-dashboard/dashboard-library.png)

Например, можно добавить плитку **телеметрии** для текущей температуры устройства. Для этого:
1. Выберите **шаблон устройства**
1. Выберите **экземпляр устройства** для устройства, которое вы хотите увидеть на плитке панели мониторинга. Затем вы увидите список свойств устройства, которые могут быть использованы на плитке.
1. Чтобы создать плитку на приборной панели, нажмите на **температуру** и перетащите ее в область приборной панели. Вы также можете нажать на флажок рядом с **температурой** и нажмите **Комбинат**. На следующем скриншоте показан выбор шаблона устройства и экземпляра устройств, а затем создается плитка телеметрии температуры на приборной панели.
1. Выберите **Сохранить** в левом верхнем, чтобы сохранить плитку на приборной панели.

> [!div class="mx-imgBorder"]
> ![Форма настройки сведений об устройстве со сведениями о параметрах и свойствах](media/howto-add-tiles-to-your-dashboard/device-details.png)

Теперь, когда оператор просматривает панель мониторинга приложения по умолчанию, он видит новую плитку с **температурой** для устройства. Каждая плитка имеет предварительно выбранный график, диаграмму и т.д., которые будут отображаться при создании плитки. Тем не менее, пользователи могут выбрать для отодевать и изменить эту визуализацию. 

> [!div class="mx-imgBorder"]
> ![Вкладка "Панель мониторинга", на которой отображаются параметры и свойства плитки](media/howto-add-tiles-to-your-dashboard/settings-and-properties.png)

## <a name="edit-tiles"></a>Плитка для отсечения

Чтобы отсеять плитку на приборной панели, сначала щелкните **отодёт** в левом верхнем верхнем листе, который откроет режим отсечения для приборной панели и всех ее плиток. 

> [!div class="mx-imgBorder"]
> ![Экран панели мониторинга с режимом отправления, активированным для выбранной плитки](media/howto-add-tiles-to-your-dashboard/edit-mode.png)

Затем нажмите значок **Gear** в правом верхнем углу плитки, который вы хотите отредовать. Здесь вы можете отсеить аспекты плитки, включая ее название, ее визуализацию, агрегацию и т.д.

> [!div class="mx-imgBorder"]
> ![Выпадение для настроек агрегации плитки](media/howto-add-tiles-to-your-dashboard/aggregation-settings.png)

Вы также можете изменить визуализацию диаграммы, нажав значок **Правителя** на плитке.

> [!div class="mx-imgBorder"]
> ![Выпадение для настроек визуализации плитки](media/howto-add-tiles-to-your-dashboard/visualization-settings.png)

## <a name="tile-types"></a>Типы плитки

В следующей таблице кратко излагается использование плиток в Azure IoT Central:
 
| Плитка | Панель мониторинга | Описание
| ----------- | ------- | ------- |
| Содержимое | Панели мониторинга приложений и устройств |Поддерживаемые markdown плитки являются интерактивными плитками, которые отображают заголовок и текст описания. Вы также можете использовать эту плитку в качестве ссылки плитки, чтобы позволить пользователю перейти к URL, связанных с вашим приложением.|
| Образ — | Панели мониторинга приложений и устройств |Плитки изображения отображают пользовательское изображение и могут быть интерактивными. Используйте плитку изображения, чтобы добавить графику в панель мониторинга и дополнительно дать пользователю возможность перейти к URL-адресу, относящемуся к вашему приложению.|
| Метка | Панели мониторинга приложений |Плитки этикетки отображают пользовательский текст на приборной панели. Вы можете выбрать размер текста. Используйте плитку метки, чтобы добавить соответствующую информацию в панель мониторинга, такие описания, контактные данные или справки.|
| Схема | Панели приложений и приборов устройств |Картографические плитки отображают местоположение устройства на карте. Вы также можете отображать до 100 точек истории местоположения устройства. Например, можно отобразить пробный маршрут того, где устройство находилось на прошлой неделе.|
| График | Панели приложений и приборов устройств |Плитки линейной диаграммы отображают диаграмму совокупного измерения для устройства за период времени. Например, можно отобразить линейную диаграмму, показываюую среднюю температуру и давление устройства за последний час.|
| Линейчатая диаграмма | Панели приложений и приборов устройств |Плитки диаграммы баров отображают диаграмму агрегированных измерений для устройства за период времени. Например, можно отобразить диаграмму баров, которая показывает среднюю температуру и давление устройства за последний час.|
| Круговая диаграмма | Панели мониторинга приложений и устройств |Плитки диаграммы пирога отображают диаграмму агрегированных измерений для устройства за период времени.|
| Тепловая карта | Панели мониторинга приложений и устройств |Плитки Heat Map отображают информацию о наборе устройств, представленную в виде цветов.|
| История событий | Панели приложений и приборов устройств |Плитки истории событий отображают события для устройства в течение периода времени. Например, его можно использовать для отображижижижив все изменения температуры для устройства в течение последнего часа.|
| State History ("Журнал состояний"). | Панели приложений и приборов устройств |Плитки истории состояния отображают значения измерений за период времени. Например, его можно использовать для отображижима значения температуры для устройства в течение последнего часа.|
| Ключевой показатель эффективности | Панели приложений и приборов устройств | Плитки KPI отображают агрегированную телеметрию или измерение событий за период времени. Например, его можно использовать для отображания максимальной температуры, достигнутой для устройства в течение последнего часа.|
| Последнее известное значение | Панели приложений и приборов устройств |Последние известные значения плитки отображают последнее значение для телеметрии или измерения состояния. Например, эту плитку можно использовать для отображения последних измерений температуры, давления и влажности устройства.|
