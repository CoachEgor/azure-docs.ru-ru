# <a name="configure-the-application-dashboard"></a>Настройка панели мониторинга приложения

**Панель мониторинга** — это страница, которая загружается по URL-адресу приложения для пользователя, имеющего доступ к этому приложению. Если вы создали приложение на основе одного из **шаблонов приложений**, то у него сразу будет стандартная панель мониторинга. Если приложение создано на основе шаблона **пользовательского приложения**, то на панели мониторинга будут показаны советы по началу работы.

> [!NOTE]
> Пользователи могут [создать несколько панелей мониторинга](howto-create-personal-dashboards.md) в дополнение к панели мониторинга по умолчанию приложения. Это могут быть личные панели мониторинга данного пользователя или общие панели мониторинга всех пользователей приложения.  

## <a name="add-tiles"></a>Добавление плиток

На следующем снимке экрана показана панель мониторинга в приложении, созданном из шаблона **пользовательского приложения**. Чтобы настроить панель мониторинга по умолчанию для приложения, выберите **Изменить** в левом верхнем углу страницы.

> [!div class="mx-imgBorder"]
> ![Панель мониторинга для приложений, основанных на шаблоне Sample Contoso](media/howto-add-tiles-to-your-dashboard/dashboard-sample-contoso.png)

Если выбрать **Изменить**, откроется панель "Библиотека панелей мониторинга". Эта библиотека содержит плитки и примитивы панели мониторинга, которые можно использовать для настройки панели мониторинга.

> [!div class="mx-imgBorder"]
> ![Библиотека панелей мониторинга](media/howto-add-tiles-to-your-dashboard/dashboard-library.png)

Например, можно добавить плитку **Телеметрия** для отображения текущей температуры устройства. Для этого:

1. Выберите **Шаблон устройства**.
1. Из списка **Устройства** выберите устройство, которое вы хотите отобразить на плитке панели мониторинга. Затем вы увидите список свойств устройства, которые можно использовать на плитке.
1. Чтобы создать плитку на панели мониторинга, щелкните плитку **Температура** и перетащите ее в область панель мониторинга. Можно также установить флажок рядом с плиткой **Температура** и нажать кнопку **Добавить плитку**. На следующем снимке экрана показан выбор шаблона устройства и экземпляра устройства, а также создание плитки телеметрии температуры на панели мониторинга.
1. Выберите **Сохранить** в верхнем левом углу, чтобы сохранить плитку на панели мониторинга.

> [!div class="mx-imgBorder"]
> ![Форма настройки сведений об устройстве со сведениями о параметрах и свойствах](media/howto-add-tiles-to-your-dashboard/device-details.png)

Теперь, когда оператор посмотрит на панель мониторинга по умолчанию приложения, он увидит новую плитку **Температура** для устройства. У каждой плитки есть предварительно выбранный граф, диаграмма и т. д., которые будут отображаться после создания этой плитки. Однако пользователи могут изменить эту визуализацию.  

> [!div class="mx-imgBorder"]
> ![Вкладка "Панель мониторинга", на которой отображаются параметры и свойства плитки](media/howto-add-tiles-to-your-dashboard/settings-and-properties.png)

## <a name="edit-tiles"></a>Изменение плиток

Чтобы изменить плитку на панели мониторинга, сначала щелкните **Изменить** в верхнем левом углу страницы. Включится режим редактирования панели мониторинга и всех ее плиток.  

> [!div class="mx-imgBorder"]
> ![Экран панели мониторинга в режиме редактирования выбранной плитки](media/howto-add-tiles-to-your-dashboard/edit-mode.png)

Щелкните значок **шестеренки** в правом верхнем углу плитки, которую вы хотите изменить. Можно изменить различные аспекты плитки, включая ее заголовок, визуализацию, агрегирование и т. д.

> [!div class="mx-imgBorder"]
> ![Раскрывающийся список параметров агрегирования плитки](media/howto-add-tiles-to-your-dashboard/aggregation-settings.png)

Можно также изменить визуализацию диаграммы, щелкнув значок **линейки** на плитке.

> [!div class="mx-imgBorder"]
> ![Раскрывающийся список параметров визуализации плитки](media/howto-add-tiles-to-your-dashboard/visualization-settings.png)

### <a name="customizing-visualizations"></a>Настройка визуализаций

Для графиков, линейчатых диаграмм и круговых диаграмм можно настроить цвета отображаемых данных телеметрии. Для этого щелкните значок палитры рядом с данными телеметрии, которые нужно настроить, и выберите цвет.

> [!div class="mx-imgBorder"]
> ![Раскрывающийся список параметров отображения цвета](media/howto-add-tiles-to-your-dashboard/color-customization.png)

Для данных телеметрии или свойств строкового типа можно выбрать способ визуализации текста. Например, если устройство отправляет URL-адрес как строковую телеметрию, можно отобразить этот URL-адрес как доступную для щелчка ссылку. Если URL-адрес ссылается на изображение, то это изображение можно отобразить на плитке последнего известного значения или плитке свойств. Чтобы изменить способ отображения строковой телеметрии, щелкните значок шестеренки рядом с именем телеметрии. Так можно отобразить текст как текст, ссылку или изображение.

> [!div class="mx-imgBorder"]
> ![Раскрывающийся список параметров визуализации строки](media/howto-add-tiles-to-your-dashboard/string-viz-customization.png)
