# <a name="customize-the-azure-iot-central-ui"></a>Настройка центрального uI Azure IoT

В этой статье описывается, как, будучи администратором, вы можете настроить пользовательский пользовательский доступ к приложению, применяя пользовательские темы и изменяя ссылки справки, чтобы указать на ваши собственные ресурсы справки. 



На следующем скриншоте показана страница, используюая стандартную тему:

![Стандартная тема IoT Центральной](./media/howto-customize-ui/standard-ui.png)

На следующем скриншоте показана страница с помощью пользовательского скриншота с выделенными на нет элементами пользовательского пуля:

![Пользовательская тема IoT Центральной](./media/howto-customize-ui/themed-ui.png)

## <a name="create-theme"></a>Создание темы

Чтобы создать пользовательскую тему, перейдите на страницу **настройки приложения** в разделе **Администрация:**

![Центральные темы IoT](./media/howto-customize-ui/themes.png)

На этой странице вы можете настроить следующие аспекты приложения:

### <a name="application-logo"></a>Логотип приложения

Изображение PNG, не больше 1 МБ, с прозрачным фоном. Этот логотип отображается слева на панели заголовка приложения IoT Central.

Если изображение логотипа включает в себя название приложения, вы можете скрыть текст названия приложения. Для получения дополнительной информации [см.](howto-administer.md#change-application-name-and-url)

### <a name="browser-icon-favicon"></a>Значок браузера (значок)

Изображение PNG, не больше 32 х 32 пикселей, с прозрачным фоном. Веб-браузер может использовать это изображение в адресной строке, истории, закладках и вкладке браузера.

### <a name="browser-colors"></a>Цвета браузера

Вы можете изменить цвет заголовка страницы и цвет, используемый для акцентирования кнопки и другие моменты. Используйте шестисимвольное цветовое значение в формате. `##ff6347` Для получения дополнительной информации о цветной нотации **ЗНАЧЕНИЯ HEX** см. [HTML Colors](https://www.w3schools.com/html/html_colors.asp)

> [!NOTE]
> Вы всегда можете вернуться к опциям по умолчанию на странице **настройки приложения.**

### <a name="changes-for-operators"></a>Изменения для операторов

Если администратор создает пользовательскую тему, операторы и другие пользователи приложения больше не могут выбирать тему в **настройках.**

## <a name="replace-help-links"></a>Замена ссылок на справки

Чтобы предоставить пользовательскую информацию о справках операторам и другим пользователям, можно изменить ссылки в меню **справки** приложения.

Чтобы изменить ссылки справки, перейдите на страницу **справки Customize** в разделе **Администрация:**

![Настройка ссылок на справку IoT Central](./media/howto-customize-ui/help-links.png)

Вы также можете добавить новые записи в меню справки и удалить записи по умолчанию:

![Индивидуальная помощь IoT Central](./media/howto-customize-ui/custom-help.png)

> [!NOTE]
> Вы всегда можете вернуться к ссылкам справки по умолчанию на странице **справки Customize.**
