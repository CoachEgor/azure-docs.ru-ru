# <a name="what-are-application-templates"></a>Что представляют собой шаблоны приложений?

Шаблоны приложений в Azure IoT Central являются инструментом, помогая разработчикам решений для решения. Вы можете использовать шаблоны приложений для всего, от получения почувствовать, что это возможно, чтобы полностью настроить и ваше приложение для перепродажи для ваших клиентов.

Шаблоны приложений состоят из:

- примеров панелей мониторинга для операторов;
- Образцы шаблонов устройств
- Имитированные устройства, производящие данные в режиме реального времени
- предварительно настроенных правил и заданий;
- Богатая документация, включая учебники и как-tos

При создании приложения вы выбираете шаблон приложения. Вы не можете изменить шаблон после создания приложения.

## <a name="custom-templates"></a>Пользовательские шаблоны

Если вы хотите создать приложение с нуля, выберите один из двух шаблонов пользовательских приложений:

- Custom application (Пользовательское приложение)
- Пользовательское приложение (наследие)

Выберите шаблон **пользовательского приложения,** если у вас нет конкретной причины для использования устаревшего шаблона.

## <a name="industry-focused-templates"></a>Отраслевые шаблоны

Azure IoT Central — это отраслевая агностичная прикладная платформа. Шаблоны приложений являются отраслевыми примерами, доступными для этих отраслей сегодня, и в будущем их ждет еще больше:

- [Розничная версия](../retail/overview-iot-central-retail.md)
  - Интеллектуальная логистика
  - Цифровой центр дистрибуции
  - Мониторинг условий для аналитики магазина
  - Оформление заказов для аналитики магазина
  - Интеллектуальное управление запасами
- [Энергетика](../energy/overview-iot-central-energy.md)
  - Мониторинг интеллектуальных счетчиков
  - Мониторинг солнечных панелей
- [Государственные организации](../government/overview-iot-central-government.md)
  - Управление связанными отходами
  - Мониторинг потребления воды
  - Мониторинг качества воды
- [Здравоохранение](../healthcare/overview-iot-central-healthcare.md).
  - Непрерывный мониторинг пациентов
