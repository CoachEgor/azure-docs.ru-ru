# <a name="configure-the-application-dashboard"></a>Настроить панель мониторинга приложений

**Панели мониторинга** — страница, которая загружает, когда пользователи, имеющие доступ к приложению переходят к URL-адрес приложения. Если вы выбрали либо **пример Contoso** или **Devkits пример** шаблон приложения для создания приложения, приложение имеет предустановленной панели мониторинга. Если вы выбрали **пользовательское приложение** шаблон приложения панели мониторинга пуст.

> [!NOTE]
> Пользователи также могут [создавать свои собственные личные панели мониторинга](howto-personalize-dashboard.md) для использования вместо панели мониторинга приложения по умолчанию.

## <a name="add-tiles"></a>Добавление плиток

На следующем рисунке показаны панели мониторинга в приложения, созданного из **пример Contoso** шаблона. Чтобы настроить панель мониторинга по умолчанию для вашего приложения, выберите **изменить** в верхней правой части страницы.

![Панель мониторинга для приложений на основе шаблона «Пример Contoso»](media/howto-configure-homepage/image1a.png)

Выбрав **изменить**, открывает библиотеки панели мониторинга. Библиотека содержит плитки и панели мониторинга примитивы, которые можно использовать для настройки панели мониторинга.

![Библиотека панелей мониторинга](media/howto-configure-homepage/image2a.png)

Например, можно добавить **параметров устройств и свойства** плитку, чтобы показать выбор текущих значений параметров и свойств для устройства. Для этого выберите **Шаблон устройства**, а затем **Экземпляр устройства**. Затем присвойте плитке заголовок и выберите отображаемый **Параметр** или **Свойство**. На следующем рисунке показаны параметры и свойства, выбранные для добавления к плитке. Выберите **сделать** для сохранения изменений в панель мониторинга.

![Форма настройки сведений об устройстве со сведениями о параметрах и свойствах](media/howto-configure-homepage/image3a.png)

Теперь, когда оператор просматривает панель мониторинга приложения по умолчанию, они видят новую плитку с **значение температуры** для устройства:

![Вкладка "Панель мониторинга", на которой отображаются параметры и свойства плитки](media/howto-configure-homepage/image4a.png)

Вы можете изучить другие плитки типы в библиотеке для обнаружения для дальнейшей настройки панели мониторинга приложения по умолчанию.

Дополнительные сведения об использовании плиток в Azure IoT Central, см. в разделе [использовать панели мониторинга плитки](howto-use-tiles.md).
