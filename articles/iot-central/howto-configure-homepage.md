# <a name="configure-the-application-dashboard"></a>Настройка панели мониторинга приложения

**Панель мониторинга** — это страница, которая загружается, когда пользователи, имеющие доступ к приложению, обращаются к URL-адресу приложения. Если вы выбрали шаблон приложения **Пример Contoso** или **Sample девкитс** для создания приложения, приложение будет иметь предварительно определенную панель мониторинга. Если вы выбрали шаблон **пользовательского** приложения, панель мониторинга будет пустой.

> [!NOTE]
> Пользователи также могут [создавать собственные персональные панели мониторинга](howto-personalize-dashboard.md) для использования вместо панели мониторинга приложения по умолчанию.

## <a name="add-tiles"></a>Добавление плиток

На следующем снимке экрана показана панель мониторинга в приложении, созданном из **примера шаблона Contoso** . Чтобы настроить панель мониторинга по умолчанию для приложения, выберите **изменить** в верхней правой части страницы.

![Панель мониторинга для приложений на основе шаблона "пример Contoso"](media/howto-configure-homepage/image1a.png)

При нажатии кнопки **изменить**открывается панель Библиотека панели мониторинга. Библиотека содержит плитки и примитивы панели мониторинга, которые можно использовать для настройки панели мониторинга.

![Библиотека панелей мониторинга](media/howto-configure-homepage/image2a.png)

Например, можно добавить плитку " **Параметры устройства" и "свойства** ", чтобы отобразить Выбор текущих параметров и значений свойств для устройства. Для этого выберите **Шаблон устройства**, а затем **Экземпляр устройства**. Затем присвойте плитке заголовок и выберите отображаемый **Параметр** или **Свойство**. На следующем снимке экрана показаны параметры и свойства, выбранные для добавления к плитке. Нажмите кнопку **Готово** , чтобы сохранить изменения на панели мониторинга.

![Форма настройки сведений об устройстве со сведениями о параметрах и свойствах](media/howto-configure-homepage/image3a.png)

Теперь, когда оператор просматривает панель мониторинга приложения по умолчанию, он увидит новую плитку с параметром **установить температуру** для устройства:

![Вкладка "Панель мониторинга", на которой отображаются параметры и свойства плитки](media/howto-configure-homepage/image4a.png)

Другие типы плиток в библиотеке можно изучить, чтобы узнать, как дополнительно настроить панель мониторинга приложения по умолчанию.

Дополнительные сведения об использовании плиток в Azure IoT Central см. в статье [использование плиток панели мониторинга](howto-use-tiles.md).
