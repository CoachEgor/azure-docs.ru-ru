# <a name="customize-the-azure-iot-central-ui"></a>Настройка Azure IoT Central пользовательского интерфейса 

*Эта статья относится к администраторам.*

IoT Central позволяет настраивать пользовательский Интерфейс приложения, применяя пользовательские темы и изменение ссылки на справку указывал на ваших собственных ресурсов специальную справку. На следующем рисунке показаны страницы с использованием стандартной темы:

![Стандартную тему IoT Central](./media/howto-customize-ui/standard-ui.png)

На следующем рисунке показан на странице с помощью пользовательских снимок экрана с выделенным настраиваемые элементы пользовательского интерфейса:

![Пользовательские темы центра Интернета вещей](./media/howto-customize-ui/themed-ui.png)

## <a name="create-theme"></a>Создать тему

Чтобы создать пользовательские темы, перейдите к **настройки приложения** странице в **администрирования** раздел:

![Темы IoT Central](./media/howto-customize-ui/themes.png)

На этой странице можно настроить следующие аспекты приложения.

### <a name="application-logo"></a>Логотип приложения

Изображения в формате PNG, размером не более 1 МБ с прозрачным фоном. Этот логотип отображается слева в строке заголовка IoT Central.

Если ваш логотип содержит имя приложения, можно скрыть текстовое имя приложения. Дополнительные сведения см. в разделе [Управление параметрами приложения](./howto-administer.md#manage-application-settings).

### <a name="browser-icon-favicon"></a>Значок обозревателя (значка)

Изображения в формате PNG, размером не более 32 x 32 пикселя с прозрачным фоном. Веб-браузере можно использовать этот образ в адресной строке, журнал, закладки и вкладка "браузер".

### <a name="browser-colors"></a>Цвета браузера

Можно изменить цвет заголовка страницы и цвет, используемый для ударению кнопки и другие документы. Использовать цвет с шестнадцатеричным значением шесть символьное значение в формате `##ff6347`. Дополнительные сведения о **ШЕСТНАДЦАТЕРИЧНОЕ значение** цвета нотации, см. в разделе [цвета HTML](https://www.w3schools.com/html/html_colors.asp).

> [!NOTE]
> Вы всегда можете вернуться обратно к параметрам по умолчанию на **настроить приложение** страницы.

### <a name="changes-for-operators"></a>Изменения для операторов

Если администратор создает пользовательскую тему, а затем операторов и других пользователей, приложения больше не может выбрать тему в **параметры**.

## <a name="replace-help-links"></a>Замените ссылки на справку

Чтобы предоставить сведения специальную справку для вашей операторов и других пользователей, можно изменить ссылки на приложение **помочь** меню.

Чтобы изменить ссылки на справку, перейдите к **настройки справки** странице в **администрирования** раздел:

![Настройка ссылки на справку IoT Central](./media/howto-customize-ui/help-links.png)

Можно также добавить новые записи в меню "Справка" и удалять записи по умолчанию:

![Настройки справки IoT Central](./media/howto-customize-ui/custom-help.png)

> [!NOTE]
> Можно всегда вернуться обратно к ссылки на справку по умолчанию на **настройки справки** страницы.
