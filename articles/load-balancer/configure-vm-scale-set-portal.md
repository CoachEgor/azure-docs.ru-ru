# <a name="configure-a-virtual-machine-scale-set-with-an-existing-azure-load-balancer-using-the-azure-portal"></a>Настроили виртуальный набор масштабов машины с существующим балансом загрузки Azure с помощью портала Azure

В этой статье вы узнаете, как настроить виртуальный набор масштабов машины с существующим балансом загрузки Azure. 

## <a name="prerequisites"></a>Предварительные требования

- Подписка Azure.
- Существующий стандартный балансер загрузки sku в подписке, где будет развернут виртуальный набор масштабов машины.
- Виртуальная сеть Azure для набора виртуальной машины.

## <a name="sign-in-to-the-azure-portal"></a>Вход на портал Azure

Войти на портал Azure по адресу [https://portal.azure.com](https://portal.azure.com).



## <a name="deploy-virtual-machine-scale-set-with-existing-load-balancer"></a>Развертывание виртуального набора масштабов машин с существующим балансером нагрузки

В этом разделе вы создадите виртуальную шкалу машин, установленную на портале Azure, с существующим балансисатором нагрузки Azure.

> [!NOTE]
> Следующие шаги предполагают, что ранее была развернута виртуальная сеть под названием **myVNet** и балансилер нагрузки Azure под названием **myLoadBalancer.**

1. В верхней левой части экрана нажмите **«Создайте ресурс** > **Compute** > **Virtual machine scale set»** или поищите **масштаб виртуальной машины, установленный** в поиске на рынке.

2. Выберите **Создать**.

3. При **создании виртуального набора масштабов машины**введите или выберите эту информацию во вкладке **Основы:**

    | Параметр                        | Значение                                                                                                 |
    |--------------------------------|-------------------------------------------------------------------------------------------------------|
    | **Детали проекта**            |                                                                                                       |
    | Подписка                   | Выбор подписки Azure                                                                        |
    | Группа ресурсов                 | Выберите Создать новый, введите **myResourceGroup**, затем выберите OK, или выберите существующую группу ресурсов. |
    | **Детали набора масштаба**          |                                                                                                       |
    | Имя масштабируемого набора виртуальных машин | Введите **myVMSS**                                                                                      |
    | Регион                         | Выберите регион **Восточная часть США 2**                                                                                    |
    | Зона доступности              | Выберите **Нет**                                                                                       |
    | **Детали инстанции**           |                                                                                                       |
    | Изображение                          | Выберите **Ubuntu Server 18.04 LTS**                                                                    |
    | Экземпляр Azure Spot            | Выберите **Нет**.                                                                                         |
    | Размер                           | Оставить по умолчанию                                                                                      |
    | **Учетная запись администратора**      |                                                                                                       |
    | Authentication type (Тип проверки подлинности)            | Выберите **пароль**                                                                                   |
    | Имя пользователя                       | Введите имя пользователя admin        |
    | Пароль                       | Введите пароль для админ-адиктора    |
    | Подтверждение пароля               | Повторное ввоза пароля админ |


    :::image type="content" source="./media/vm-scale-sets/create-vm-scale-set-01.png" alt-text="Создание виртуального набора масштабов машин." border="true":::

4. Выберите вкладку **Сеть.**

5. Введите или выберите эту информацию во вкладке **Сеть:**

     Параметр                           | Значение                                                    |
    |-----------------------------------|----------------------------------------------------------|
    | **Настройка виртуальной сети** |                                                          |
    | Виртуальная сеть                   | Выберите **myVNet** или существующую виртуальную сеть.      |
    | **Балансировка нагрузки**                |                                                          |
    | Использование балансивизатора нагрузки               | выберите **Да**.                                           |
    | **Параметры балансировки нагрузки**       |                                                          |
    | Параметры балансировки нагрузки            | Выберите **балансиватор нагрузки Azure**                           |
    | Выберите балансизатор нагрузки            | Выберите **myLoadBalancer** или существующий балансер нагрузки |
    | Выберите пул бэкэнда             | Выберите **myBackendPool** или существующий пул бэкэнда.  |

    :::image type="content" source="./media/vm-scale-sets/create-vm-scale-set-02.png" alt-text="Создание виртуального набора масштабов машин." border="true":::

6. Выберите вкладку **«Управление».**

7. Во вкладке **"Управление"** **Off**установите **диагностику boot.**

8. Выберите синюю кнопку **«Обзор» и создайте** кнопку.

9. Просмотрите настройки и выберите кнопку **«Создание».**
