# <a name="security-controls-for-azure-load-balancer"></a>Элементы управления безопасностью для баланса загрузки Azure

В этой статье документируются элементы управления безопасностью, встроенные в балансер загрузки Azure.

[!INCLUDE [Security controls Header](../../includes/security-controls-header.md)]

## <a name="network"></a>Сеть

| Контроль безопасности | Да/нет | Примечания |
|---|---|--|
| Поддержка конечных точек обслуживания| Недоступно | |
| Поддержка инъекций VNet| Недоступно | |
| Поддержка сетевой изоляции и брандмауэра| Недоступно |  |
| Принудительная поддержка туннелирования| Недоступно | |

## <a name="monitoring--logging"></a>Мониторинг & лесозаготовки

| Контроль безопасности | Да/нет | Примечания|
|---|---|--|
| Поддержка мониторинга Azure (аналитика журналов, информация о приложениях и т.д.)| Да | Смотрите [журналы Azure Monitor для общедоступных балансов основных нагрузок.](load-balancer-monitor-log.md) |
| Управление и управление самолетом лесозаготовки и аудит| Да | Смотрите [журналы Azure Monitor для общедоступных балансов основных нагрузок.](load-balancer-monitor-log.md) |
| Регистрация и аудит плоскости данных | Недоступно |  |

## <a name="identity"></a>идентификации

| Контроль безопасности | Да/нет | Примечания|
|---|---|--|
| Проверка подлинности| Недоступно |  |
| Авторизация| Недоступно |  |

## <a name="data-protection"></a>Защита данных

| Контроль безопасности | Да/нет | Примечания |
|---|---|--|
| Шифрование на стороне сервера в состоянии покоя: клавиши, управляемые Корпорацией Майкрософт | Недоступно | |
| Шифрование в пути (например, шифрование ExpressRoute, шифрование VNet и шифрование VNet-VNet)| Недоступно | |
| Шифрование на стороне сервера в состоянии покоя: ключи, управляемые клиентом (BYOK) | Недоступно | |
| Шифрование уровня столбцов (Службы данных Azure)| Недоступно | |
| Вызовы API в зашифрованном виде| Да | Через [менеджер ресурсов Azure](../azure-resource-manager/index.yml). |

## <a name="configuration-management"></a>Управление конфигурацией

| Контроль безопасности | Да/нет | Примечания|
|---|---|--|
| Поддержка управления конфигурацией (версия конфигурации и т.д.)| Недоступно |  |
