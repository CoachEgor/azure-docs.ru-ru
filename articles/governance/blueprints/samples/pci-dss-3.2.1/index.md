# <a name="overview-of-the-pci-dss-v321-blueprint-sample"></a>Общие сведения о примере схемы PCI-DSS версии 3.2.1

Пример схемы PCI-DSS версии 3.2.1 — это набор политик, которые отвечают соответствующим требованиям. Эта схема предназначена для клиентов, которые управляют облачными средами с рабочими нагрузками PCI-DSS.
Схема PCI-DSS позволяет развернуть базовый набор политик для любой архитектуры на основе Azure, требующей соответствующей аккредитации.

## <a name="control-mapping"></a>Сопоставление элементов управления

В разделе о сопоставлении средств управления описаны политики, используемые в рамках этой инициативы, и методы, с помощью которых эти политики помогают обеспечить соответствие с разными средствами управления, определенными в PCI-DSS версии 3.2.1. Назначаемые архитектуре ресурсы оцениваются Политикой Azure на предмет соответствия назначенным политикам.

Назначив эту схему, просмотрите уровень соответствия среды Azure на панели мониторинга Политики Azure.
