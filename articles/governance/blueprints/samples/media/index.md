# <a name="overview-of-the-media-blueprint-sample"></a>Общие сведения о примере схемы Media

Пример схемы Media предоставляет средства управления, которые с помощью [Политики Azure](../../../policy/overview.md) обеспечивают соответствие требованиям для прохождения аттестации [Media](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html). 

## <a name="blueprint-sample"></a>Пример схемы

Этот пример схемы позволяет развернуть базовый набор политик для любой архитектуры на основе Azure, требующей аккредитации или соответствия нормам, установленным на платформе Media. В разделе о [сопоставлении средств управления](./control-mapping.md) описаны политики, используемые в рамках этой инициативы, и методы, с помощью которых эти политики помогают обеспечить соответствие с разными средствами управления, определенными на платформе Media. Назначаемые архитектуре ресурсы оцениваются Политикой Azure на предмет соответствия назначенным политикам.
