# <a name="azure-policy-built-in-policy-definitions"></a>Определения встроенных политик в Политике Azure

Эта страница представляет собой индекс определений встроенных политик в Политике Azure.

Имя каждой из встроенных политик связано с определением политики на портале Azure. Чтобы просмотреть исходный репозиторий **GitHub для службы "Политика Azure"** , перейдите по ссылке в столбце [Источник](https://github.com/Azure/azure-policy). Встроенные инициативы группируются по свойству **category** (категория) в **метаданных**. Перейти к определенной **категории** можно с помощью меню в правой части страницы. Или нажмите <kbd>CTRL</kbd>-<kbd>F</kbd>, чтобы использовать функцию поиска в браузере.
