# <a name="azure-event-hubs---resource-manager-exceptions"></a>Концентраторы событий Azure - исключения менеджера ресурсов
В этой статье перечислены исключения, создаваемые при взаимодействии с концентраторами событий Azure с помощью менеджера ресурсов Azure - с помощью шаблонов или прямых вызовов.

> [!IMPORTANT]
> Этот документ часто обновляется. Пожалуйста, проверьте обновления.

Следующие разделы предоставляют различные исключения/ошибки, которые всплывают через менеджер ресурсов Azure.

## <a name="error-code-conflict"></a>Код ошибки: Конфликт

| Код ошибки | Подкод ошибки | Сообщение об ошибке | Описание | Рекомендация |
| ---------- | ------------- | ------------- | ----------- | -------------- |
| Конфликт | 40300 | Максимальное количество ресурсов типа EventHub было достигнуто или превышено. Фактически: К, Макс разрешено: # | Пространство имен достигло своей [квоты](event-hubs-quotas.md) на количество концентраторов событий, которые оно может содержать. | Удалите все неиспользованные или посторонние концентраторы событий из пространства имен или рассмотрите возможность обновления до [выделенного кластера.](event-hubs-dedicated-overview.md) |
| Конфликт | none | Конфигурация аварийного восстановления (DR) не может быть удалена, поскольку репликация продолжается. Fail over or break pairing before attempting to delete the DR Config. | [Репликация GeoDR](event-hubs-geo-dr.md) продолжается, поэтому конфигурация не может быть удалена в данный момент. | Чтобы разблокировать удаление конфигурации, либо подождите, пока репликация завершится, вызвать сбой, либо разорвать geoDR сопряжения. |
| Конфликт | none | Обновление Namespace не удалось с конфликтом в бэкэнде. | Еще одна операция в настоящее время делается на этом пространстве имен. | Подождите, пока текущая операция не завершится, а затем повторить попытку. |

## <a name="error-code-429"></a>Код ошибки: 429

| Код ошибки | Подкод ошибки | Сообщение об ошибке | Описание | Рекомендация |
| ---------- | ------------- | ------------- | ----------- | -------------- |
| 429 | none | Подготовка пространства имен в переходный период | Еще одна операция в настоящее время делается на этом пространстве имен. | Подождите, пока текущая операция не завершится, а затем повторить попытку. |
| 429 | none | Операция аварийного восстановления продолжается. | В настоящее время на этом пространстве имен или сопряжении выполняется операция [GeoDR.](event-hubs-geo-dr.md) | Подождите, пока текущая операция GeoDR не завершится, а затем повторная попытка. |

## <a name="error-code-badrequest"></a>Код ошибки: BadRequest

| Код ошибки | Подкод ошибки | Сообщение об ошибке | Описание | Рекомендация |
| ---------- | ------------- | ------------- | ----------- | -------------- |
| BadRequest | 40 000 | PartitionCount не может быть изменен для концентратора событий. | Базовый или стандартный уровень концентраторов Azure Event не поддерживает изменение разделов. | Создайте новый концентратор событий с требуемым числом разделов в базовом или стандартном пространстве имен уровня. Масштаб ирование разделений поддерживается для [выделенных кластеров.](event-hubs-dedicated-overview.md) |
| BadRequest | 40 000 | Значение '' для MessageRetentionInDays не является действительным для базового уровня. значение не может превышать '1' день (ы). | Основные области именных пространств событий поддерживают только сохранение сообщений до 1 дня. | Если требуется сохранение более одного дня, [создайте стандартное пространство имен событий Hubs.](event-hubs-create.md) | 
| BadRequest | none | Указанное имя недоступно. | Имена имен должны быть уникальными, и указанное имя уже принято. | Если вы являетесь владельцем существующего пространства имен с указанным именем, его можно удалить, что приведет к потере данных. Затем повторите попытку с тем же именем. Если пространство имен небезопасно для удаления (или вы не являетесь владельцем), выберите другое имя. |
| BadRequest | none | Указанная подписка достигла своей квоты именных пространств. | Ваша подписка достигла [квоты](event-hubs-quotas.md) на количество именных пространств, которые она может хранить. | Рассмотрите возможность утилизировать неиспользованные пространства имен в этой подписке, создать другую подписку или обновить [сяокопию до выделенного кластера.](event-hubs-dedicated-overview.md) |
| BadRequest | none | Не устоичяное пространство имен | Пространство имен не может быть обновлено, потому что это вторичное пространство имен в [сопряжении GeoDR.](event-hubs-geo-dr.md) | При необходимости внося изменения в основное пространство имен в этом сопряжении. В противном случае перерыв GeoDR сопряжения, чтобы внести изменения. |
| BadRequest | none | Не успомяно автонадум в базовом SKU | Автоматическое завышение не может быть включено в основных пространствах имен событий уровня. | Чтобы [включить автоматическое надувание](event-hubs-auto-inflate.md) в пространстве имен, убедитесь, что он стандартного уровня. |
| BadRequest | none | Не хватает емкости для создания пространства имен. Свяжитесь с администратором концентраторов событий. | Выбранный регион находится на емкости, и больше пространства имен не могут быть созданы. | Выберите другой регион для размещения вашего пространства имен. |
| BadRequest | none | Операция не может быть выполнена на типе сущности 'ConsumerGroup', потому что в названии «имя пространства имен» используется уровень «Основной».  | Основные области именных пространств событий имеют «квоту» (event-hubs-квоты.md-event-hubs-basic-and-standard----квоты-и-ограничения) одной группы потребителей (по умолчанию). Создание большего числа групп потребителей не поддерживается. | Продолжить использование группы потребителей по умолчанию ($Default), или если требуется больше, рассмотрите возможность использования пространства имен стандартных центров событий. | 
| BadRequest | none | «Имя» пространства имен не существует. | При условии, что пространство имен не может быть найдено. | Дважды убедитесь, что имя имени правильное и может быть найдено в вашей подписке. Если это не так, [создайте пространство имен Event Hubs.](event-hubs-create.md) | 
| BadRequest | none | Свойство местоположения ресурса не соответствует содержащемуся в его пространстве имен. | Создание концентратора событий в определенном регионе не удалось, поскольку оно не соответствовало области пространства имен. | Попробуйте создать концентратор событий в том же регионе, что и пространство имен. |
