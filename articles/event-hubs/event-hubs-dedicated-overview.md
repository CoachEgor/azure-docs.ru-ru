# <a name="overview-of-event-hubs-dedicated"></a>Обзор Центров событий ценовой категории "Выделенный"

Кластеры концентраторов событий Azure предлагают развертывания с одним клиентом для клиентов с самых высоких требований потоковой передачи. Это предложение одним клиентом имеет гарантированной соглашением об уровне ОБСЛУЖИВАНИЯ 99,99%. Она доступна только на архитектуру Dedicated ценового уровня.

Кластер концентраторов событий может принимать миллионы событий в секунду с гарантированная емкость и задержкой в долю секунды. Пространства имен и концентраторы событий создается в выделенный кластер включают все возможности, предложение поддержки Standard и многое другое, но без ограничения входящих данных. Он также включает [записи концентраторов событий](event-hubs-capture-overview.md) компонента без дополнительных затрат. Его можно использовать для автоматически пакетной службы и журнала потоков данных в хранилище Azure или Azure Data Lake.

Выделенных кластерах подготавливаются и счет за единицы емкости (Cu). Единицы мощности будут предварительно объем ресурсов ЦП и памяти. Вы можете приобрести 1, 2, 4, 8, 12, 16 или более 20 единиц мощности, для каждого кластера. Сколько можно принимать и поток на единицу CU зависит от факторов, таких как количество производители и потребители, полезные данные фигуры и коэффициент исходящих.

Дополнительные сведения см. в таблице с результатами теста производительности.

## <a name="why-use-event-hubs-dedicated"></a>Зачем использовать концентраторы событий уровня Dedicated?

Концентраторы событий уровня Dedicated предлагает три преимущества для клиентов, которым требуется емкость уровня предприятия.

#### <a name="single-tenancy-guarantees-capacity-for-better-performance"></a>Одним аренды гарантирует емкости для повышения производительности

Выделенный кластер гарантирует емкость полномасштабные. Она может принимать до гигабайта данных потоковой передачи, обеспечивая абсолютно надежное хранение и долю секунды задержки в соответствии с всплесков трафика.

#### <a name="inclusive-and-exclusive-access-to-features"></a>Инклюзивные и эксклюзивные доступ к функциям 
Специальное предложение включает в себя функции, такие как записи без дополнительной платы. Он также предлагает монопольный доступ к предстоящих функции, такие как BYOK. Эта служба также управляет балансировки нагрузки, ОС обновлений, исправлений системы безопасности и секционированием. С этими возможностями можно тратить меньше времени на обслуживание инфраструктуры и больше времени на создании функций на стороне клиента.

#### <a name="cost-savings"></a>Экономия денежных средств
Во тома высокий входящий трафик кластера затраты значительно меньше в час, чем в случае покупки сопоставимых количество единиц пропускной способности (единицы пропускной способности) в предложении Standard. Слишком большое количество — > 100 единиц пропускной способности.

## <a name="use-event-hubs-dedicated"></a>Использование концентраторов событий уровня Dedicated

Для использования концентраторов событий уровня Dedicated, [обратитесь в службу поддержки по выставлению счетов](https://ms.portal.azure.com/#create/Microsoft.Support) или к представителю корпорации Майкрософт. Можно масштабировать емкость на протяжении месяца в соответствии с потребностями, добавляя или удаляя единицы емкости. Группа разработчиков концентраторов событий позволяет обеспечат гибкое развертывание, которое подходит для вас.
