# <a name="sfctl-events"></a>события sfctl
Получение событий из хранилища событий (если служба Евентсторе уже установлена).

Системную службу Евентсторе можно добавить с помощью обновления конфигурации в любой кластер SFRP, где работает > = 6.4. Проверьте следующий URL-адрес\: HTTPS\://docs.microsoft.com/azure/service-fabric/service-fabric-diagnostics-eventstore.

## <a name="commands"></a>Команды

|Команда|Описание|
| --- | --- |
| Список ALL-Applications-List | Возвращает все события, связанные с приложениями. |
| Список ALL-nodes- | Возвращает все события, связанные с узлами. |
| Список ALL-partitions-List | Возвращает все связанные с секциями события. |
| все службы-список | Возвращает все события, связанные с службами. |
| Список приложений | Возвращает события, связанные с приложением. |
| Cluster-List | Возвращает все события, связанные с кластером. |
| список узлов | Возвращает события, связанные с узлом. |
| раздел-All-реплики — список | Возвращает все связанные с репликой события для секции. |
| Список секций | Возвращает события, связанные с секцией. |
| раздел-реплика-список | Возвращает события, связанные с репликой секции. |
| список служб | Возвращает события, связанные со службой. |

## <a name="sfctl-events-all-applications-list"></a>все события sfctl — список приложений
Возвращает все события, связанные с приложениями.

Ответ представляет собой список объектов Аппликатионевент.

### <a name="arguments"></a>Аргументы

|Аргумент|Описание|
| --- | --- |
| --конец-время в формате UTC [обязательный] | Время окончания уточняющего запроса в формате UTC гггг-мм-ddTHH\:mm\:ссZ. |
| --Start-Time-UTC [обязательный] | Время начала поискового запроса в формате UTC гггг-мм-ddTHH\:mm\:ссZ. |
| --Events-Types-Filter | Это строка с разделителями-запятыми, указывающая типы Фабрицевентс, которые должны включаться в ответ. |
| --Exclude-Analysis-события | Этот параметр отключает получение Аналисисевентс, если передается значение true. |
| --Skip-корреляция-Уточняющий запрос | Этот параметр отключает поиск сведений о Коррелатедевентс, если передается значение true. в противном случае поле Коррелатионевентс Get обработано и Хаскоррелатедевентс в каждом Фабрицевент заполняется. |
| --timeout -t | Время ожидания сервера для выполнения операции в секундах. Это время ожидания указывает период времени, в течение которого клиент желает ожидать завершения запрошенной операции. Значение по умолчанию для этого параметра — 60 секунд.  Значение по умолчанию\: 60. |

### <a name="global-arguments"></a>Глобальные аргументы

|Аргумент|Описание|
| --- | --- |
| --debug | Увеличение уровня детализации ведения журнала для отображения всех журналов отладки. |
| --help -h | Отображение этого справочного сообщения и выход. |
| --output -o | Формат выходных данных.  Допустимые значения\: json, jsonc, table, tsv.  Значение по умолчанию\: json. |
| --query | Строка запроса JMESPath. Дополнительные сведения см. на сайте http\://jmespath.org. |
| --verbose | Повышение уровня детализации ведения журнала. Используйте параметр --debug, чтобы получить полные журналы отладки. |
