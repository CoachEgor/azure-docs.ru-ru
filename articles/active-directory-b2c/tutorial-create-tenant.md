# <a name="tutorial-create-an-azure-active-directory-b2c-tenant"></a>Руководство по созданию клиента Azure Active Directory B2C

Прежде чем приложения смогут взаимодействовать с Azure Active Directory B2C (Azure AD B2C), они должны быть зарегистрированы в управляющем клиентом.

Вы узнаете, как выполнять следующие задачи:

> [!div class="checklist"]
> * Создание клиента Azure AD B2C
> * Привязка клиента к подписке
> * Переключиться на каталог, содержащий арендатора Azure AD B2C
> * Добавление ресурса Azure AD B2C в качестве **избранного** на портале Azure

Сведения о регистрации приложения приводятся в следующем руководстве.

Если у вас еще нет подписки Azure, [создайте бесплатную учетную запись](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), прежде чем начинать работу.

## <a name="create-an-azure-ad-b2c-tenant"></a>Создание клиента Azure AD B2C

1. Войдите на [портал Azure](https://portal.azure.com/). Восвай с учетной записью Azure, которая была назначена по крайней мере роль [вкладчика](../role-based-access-control/built-in-roles.md) в подписке или группы ресурсов в подписке.

1. Выберите каталог, содержащий подписку.

    В панели инструментов портала Azure выберите значок **каталога и подписки,** а затем выберите каталог, содержащий подписку. Этот каталог отличается от каталога, который будет содержать ваш арендатор Azure AD B2C.

    ![Арендатор подписки, каталог и фильтр подписки с выбранным клиентом подписки](media/tutorial-create-tenant/portal-01-pick-directory.png)

1. На **домашней странице** или в меню портала Azure выберите **Создать ресурс**.
1. Поиск **для Azure Active Directory B2C,** а затем выберите **Создать**.
1. Выберите **Создать B2C-клиент Azure AD**.

    ![Создание нового арендатора Azure AD B2C, выбранного на портале Azure](media/tutorial-create-tenant/portal-02-create-tenant.png)

1. Введите **имя организации** и **первоначальное доменное имя.** Выберите **страну или регион** (он не может быть изменен позже), а затем выберите **Создать.**

    Доменное имя используется как часть вашего полного доменного имени арендатора. В этом примере имя арендатора *contosob2c.onmicrosoft.com:*

    ![Создание формы клиента в примере значений на портале Azure](media/tutorial-create-tenant/portal-03-tenant-naming.png)

1. После завершения создания арендатора выберите **Создать новый арендатор B2C или ссылку на существующую ссылку Нанимателя** в верхней части страницы создания арендатора.

    ![Ссылка арендатора панировочные сухари ссылка выделена в Azure портал](media/tutorial-create-tenant/portal-04-select-link-sub-link.png)

1. Выберите **ссылку на существующую подписку Azure AD B2C на мою подписку Azure.**

   ![Ссылка на существующий выбор подписки на портале Azure](media/tutorial-create-tenant/portal-05-link-subscription.png)

1. Выберите созданного **вами арендатора Azure AD B2C,** а затем выберите **подписку.**

    Для параметра **Группа ресурсов** выберите **Создать**. Введите **имя** для группы ресурсов, которая будет содержать арендатора, выберите **местоположение группы ресурсов,** а затем выберите **Создать.**

    ![Форма настройки подписки на портале Azure](media/tutorial-create-tenant/portal-06-link-subscription-settings.png)

    Можно связать несколько арендаторов Azure AD B2C с одной подпиской Azure для выставления счетов.

## <a name="select-your-b2c-tenant-directory"></a>Выберите каталог арендаторов B2C

Чтобы начать использовать новый арендатор Azure AD B2C, необходимо переключиться на каталог, содержащий арендатора.

Выберите фильтр **подписки каталога в** верхнем меню портала Azure, а затем выберите каталог, содержащий ваш клиент Azure AD B2C.

Если сначала вы не увидите нового арендатора Azure B2C в списке, обновите окно браузера, выберите фильтр **подписки Каталог снова** в верхнем меню.

![На портале Azure выбран каталог, содержащий арендаторов B2C](media/tutorial-create-tenant/portal-07-select-tenant-directory.png)

## <a name="add-azure-ad-b2c-as-a-favorite-optional"></a>Добавить Azure AD B2C в качестве избранного (необязательно)

Этот дополнительный шаг упрощает выбор клиента Azure AD B2C в следующих и всех последующих учебниках.

Вместо того, чтобы искать *Azure AD B2C* во **всех службах** каждый раз, когда вы хотите работать с вашим арендатором, вы можете вместо этого понравиться ресурсу. Затем можно выбрать его из раздела **«Избранное** меню портала», чтобы быстро просмотреть его с арендатором Azure AD B2C.

Вам нужно выполнить эту операцию только один раз. Прежде чем выполнять эти действия, убедитесь, что вы перешли на каталог, содержащий ваш арендатор Azure AD B2C, как описано в предыдущем разделе, [Выберите каталог клиента B2C.](#select-your-b2c-tenant-directory)

1. Войдите на [портал Azure](https://portal.azure.com).
1. В меню портала Azure выберите **все сервисы.**
1. В поле поиска **All services** ищите **Azure AD B2C,** нависните над результатом поиска, а затем выберите значок звезды в наборе инструментов. **Azure AD B2C** теперь появляется на портале Azure под **избранное**.
1. Если вы хотите изменить положение нового избранного, перейдите в меню портала Azure, выберите **Azure AD B2C,** а затем перетащите его вверх или вниз в нужное положение.

    ![Azure AD B2C, Меню избранных, портал Microsoft Azure](media/tutorial-create-tenant/portal-08-b2c-favorite.png)
