# <a name="azure-active-directory-b2c-file-support-requests"></a>Azure Active Directory B2C Регистрация запросов в службу поддержки
Вы можете регистрировать запросы в службу поддержки для Azure Active Directory (AD) B2C на портале Azure следующим образом.

1. Переключитесь с вашего клиента B2C на клиент, с которым связана подписка Azure. Это может быть клиент вашего сотрудника или клиент по умолчанию, созданный во время регистрации для получения подписки Azure. Дополнительные сведения см. в разделе [Как подписка Azure связана с Azure AD](../active-directory/fundamentals/active-directory-how-subscriptions-associated-directory.md).

    ![Портал Azure с выделенным выбора клиента](./media/active-directory-b2c-support/support-switch-dir.png)

3. После переключения клиентов щелкните **Справка и поддержка**.

    ![Справка и поддержка плитки, выделенным на портале Azure](./media/active-directory-b2c-support/support-support.png)

4. Щелкните **Новый запрос в службу поддержки**.

    ![Новая Плитка запрос поддержки, выделенным на портале Azure](./media/active-directory-b2c-support/support-new.png)
5. В колонке **Основы** укажите эти сведения и нажмите кнопку **Далее**.

   * **Тип проблемы** — **Техническая проблема**.
   * Выберите соответствующую **подписку**.
   * **Служба** — **Active Directory**.
   * Выберите соответствующий **план поддержки**. Если у вас его нет, его можно зарегистрировать [здесь](https://azure.microsoft.com/support/plans/).

     ![Основные сведения о странице с кнопку "Далее", выделенным на портале Azure](./media/active-directory-b2c-support/support-basics.png)
6. В колонке **Проблема** укажите эти сведения и нажмите кнопку **Далее**.

   * Выберите соответствующий уровень **серьезности** .
   * **Тип проблемы** — **B2C**.
   * Выберите соответствующую **категорию**.
   * Опишите свою проблему в поле **Сведения**. Укажите подробные сведения о проблеме, такие как имя клиента B2C, описание проблемы, сообщения об ошибках, идентификаторы корреляции (при наличии) и т. д.
   * В поле **Интервал времени** укажите дату и время (включая часовой пояс) возникновения проблемы.
   * В разделе **Отправка файлов**отправьте все снимки экрана и файлы, которые, по вашему мнению, помогут в устранении этой проблемы.

     ![Странице "проблема" с кнопкой "Далее", выделенным на портале Azure](./media/active-directory-b2c-support/support-problem.png)
7. В колонке **Контактные данные** добавьте контактные данные. Нажмите кнопку **Создать**.

    ![Контактные данные страницы с выделенным на портале Azure кнопка "Создать"](./media/active-directory-b2c-support/support-contact.png)
8. Отправленный в службу поддержки запрос можно отслеживать, щелкнув **Справка и поддержка** на начальной панели и выбрав **Управление запросами в службу поддержки**.

## <a name="known-issue-filing-a-support-request-in-the-context-of-a-b2c-tenant"></a>Известная проблема: регистрация запроса в службу поддержки в контексте клиента B2C
Если вы пропустили описанный выше шаг 2 и пытаетесь создать запрос на поддержку в контексте вашего клиента B2C, появится указанная ниже ошибка.

> [!IMPORTANT]
> Не пытайтесь зарегистрироваться для получения новой подписки Azure в вашем клиенте B2C.
>
>
