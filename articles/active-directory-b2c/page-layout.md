# <a name="page-layout-versions"></a>Версии макета страницы

Пакеты макетов страниц периодически обновляются, включив в них исправления и улучшения элементов страницы. В следующем журнале изменений указаны изменения, внесенные в каждую версию.

[!INCLUDE [active-directory-b2c-public-preview](../../includes/active-directory-b2c-public-preview.md)]

## <a name="200"></a>2.0.0

- Самоутверждаемый`selfasserted`сайт ()
  - Добавлена поддержка [элементов управления дисплеями](display-controls.md) в пользовательских политиках.

## <a name="120"></a>1.2.0

- Все страницы
  -  Исправления специальных возможностей
  - Теперь можно `data-preload="true"` добавить атрибут [в HTML-теги](custom-policy-ui-customization.md#guidelines-for-using-custom-page-content) для управления заказом на нагрузку для CSS и JavaScript.
    - Загрузите связанные файлы CSS одновременно с html-шаблоном, чтобы он не «мерцал» между загрузкой файлов.
    - Контролируйте порядок, `script` в котором ваши теги извлечены и выполняются до загрузки страницы.
  - Поле электронной `type=email` почты теперь и мобильные клавиатуры предоставят правильные предложения
  - Поддержка перевода Chrome
- Единые и самоутверждаемые страницы
  - Поля имени пользователя/электронной почты `form` и паролей теперь используют html-элемент, чтобы позволить Edge и Internet Explorer (IE) правильно сохранить эту информацию.

## <a name="110"></a>1.1.0

- Страница исключений (глобальное исключение)
  - Исправление доступности
  - Удалено сообщение по умолчанию, когда нет контакта с политикой
  - Удален CSS по умолчанию
- Страница МИД (многофакторная)
  - Кнопка «Подтвердить код» удалена
  - Поле ввода кода теперь принимает только ввод до шести (6) символов
  - Страница автоматически пытается проверить код, введенный при входе в 6-значный код, без нажатия кнопки
  - Если код неправильный, поле ввода автоматически очищается
  - После трех (3) попыток с неправильным кодом, B2C отправляет ошибку обратно в relying party
  -  Исправления специальных возможностей
  - Удален CSS по умолчанию
- Самоутвержденная страница (самоутвержденная)
  - Удалено предупреждение об отмене
  - Класс CSS для элементов ошибок
  - Улучшена логика ошибки отображаются
  - Удален CSS по умолчанию
- Единый SSP (унифицированный)
  - Добавлено держать меня в "вписанный в ( KMSI) контроль
