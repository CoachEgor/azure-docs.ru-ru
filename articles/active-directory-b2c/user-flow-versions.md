# <a name="user-flow-versions-in-azure-active-directory-b2c"></a>Версии потоков пользователей в Azure Active Directory B2C

Потоки пользователей в Azure Active Directory B2C (Azure AD B2C) помогают настроить общие [политики](active-directory-b2c-reference-policies.md) , которые полностью описывают возможности идентификации клиентов. Эти возможности взаимодействия включают в себя вход в систему, регистрацию, сброс пароля и изменение профиля. В Azure AD B2C можно выбирать из коллекции как рекомендуемых потоков пользователей, так и потоков пользователей предварительной версии.

Новые потоки пользователей добавляются как новые версии. Когда потоки пользователей стабилизируются, они становятся рекомендуемыми к использованию. Потоки пользователей помечаются как **рекомендуемые**, если они были тщательно протестированы. До того времени они будут считаться потоками в предварительной версии. Используйте рекомендуемые потоки пользователей для любого рабочего приложения и выбирайте другие версии при тестировании новой доступной возможности. Не следует использовать более старые версии рекомендуемых потоков пользователей.

>[!IMPORTANT]
> Если пользовательский поток не определен как **рекомендуемый**, он считается в *предварительной версии*. Для рабочих приложений следует использовать только рекомендованные потоки пользователей.

| Вход с помощью РОПК | Нет | Позволяет пользователю с локальной учетной записью напрямую входить в собственные приложения (браузер не требуется). | Войти | Нет | Позволяет пользователю входить в свою учетную запись. Используя этот поток пользователя, можно настроить следующее: <ul><li>[Многофакторная проверка подлинности](active-directory-b2c-reference-mfa.md)</li><li>[Время существования маркера](active-directory-b2c-reference-tokens.md)</li><li>Параметры совместимости токена</li><li>Поведение сеанса</li><li>Блокирование входа</li><li>Принудительный сброс пароля</li><li>Возможность оставаться в системе</ul><br>С помощью этого потока пользователя нельзя настроить пользовательский интерфейс. |

| Регистрация и вход | Да | Позволяет пользователю создать учетную запись или войти в свою учетную запись. Используя этот поток пользователя, можно настроить следующее: <ul><li>[Многофакторная проверка подлинности](active-directory-b2c-reference-mfa.md)</li><li>[Время существования маркера](active-directory-b2c-reference-tokens.md)</li><li>Параметры совместимости токена</li><li>Поведение сеанса</li><li>[Требования к сложности пароля](active-directory-b2c-reference-password-complexity.md)</li></ul>|
