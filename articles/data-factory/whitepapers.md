# <a name="azure-data-factory-whitepapers"></a>Белые бумаги Фабрики данных Azure

Whitepapers позволяют исследовать фабрику данных Azure на более глубоком уровне. В этой статье представлен список доступных информационных бумаг для фабрики данных Azure.

| **Технический документ** | **Описание** |
| --- | --- |
|[Фабрика данных Azure—Интеграция данных в облаке](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | В этом документе описывается, как Azure Data Factory может позволить вам создать современный склад данных, позволить расширенной аналитике управлять интеллектуальными приложениями SaaS и поднимать пакеты служб интеграции серверов в Azure.|
|[Миграция данных из локтевого реляционного хранилища данных в Azure с помощью Azure Data Factory](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | В настоящем документе рассматривается сложность миграции десятков данных о ТБ из существующего склада реляционных данных (например, Netezza, Oracle, Teradata, сервера S'L) в Azure (например, Blob Storage или Azure Data Lake Storage) с помощью Azure Data Factory. Проблемы и рекомендации иллюстрируются устойчивостью, производительности, масштабируемости, управлению и безопасностью для перемещения больших данных в Azure по Базе данных Azure. |
|[Интеграция данных SAP с помощью фабрики данных Azure](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | В этом документе кратко излагается текущая поддержка Azure Data Factory по интеграции данных SAP, включая целевой сценарий, параметры разъема SAP и сравнение различных требований, а также введение на каждом разъеме SAP в Data Factory. |

Затем мы распойдем с техническими деталями создания ИК Azure-SSIS, а затем покажем, как загружать, выполнять и контролировать ваши пакеты через Фабрику данных Azure, используя инструменты, с помощью которые вы, вероятно, знакомы с такими инструментами, которые вы, вероятно, знакомы с такими, как S'L Server Management Studio (SSMS). |
|[Фабрика данных Azure - Прохождение параметров](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| Эта статья рассказывает о том, как передавать параметры между конвейером и действием, а также между действиями. |
