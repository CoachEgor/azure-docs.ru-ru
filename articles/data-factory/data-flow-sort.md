# <a name="sort-transformation-in-mapping-data-flow"></a>Сортировка преобразования в картографии потока данных

[!INCLUDE[appliesto-adf-xxx-md](includes/appliesto-adf-xxx-md.md)]

Преобразование сортировки позволяет сортировать входящие строки в текущем потоке данных. Вы можете выбрать отдельные столбцы и сортировать их в порядке восходящего или убывания.

> [!NOTE]
> Потоки карт данных выполняются на кластерах искр, которые распределяют данные по нескольким узлам и разделам. Если вы решите переоместить данные в последующее преобразование, вы можете потерять сортировку из-за перетасовки данных.

## <a name="configuration"></a>Конфигурация

![Настройки сортировки](media/data-flow/sort.png "Сортировка")

**Дело бесчувственно:** Хотите ли вы игнорировать случай при сортировке строки или текстовых полей

**Сортировать только в пределах разделов:** Поскольку потоки данных работают на искре, каждый поток данных делится на разделы. Этот параметр сортирует данные только в входящих разделах, а не сортировку всего потока данных. 

**Условия сортировки:** Выберите, какие столбцы вы сортируете и в каком порядке происходит сортировка. Заказ определяет приоритет сортировки. Выберите, будут ли нулевые отображаться в начале или конце потока данных.

### <a name="computed-columns"></a>Вычисляемые столбцы

Чтобы изменить или извлечь значение столбца перед нанесением сорта, нависните над столбцом и выберите "вычисленный столбец". Это откроет значение выражения для создания выражения для операции сортировки вместо использования значения столбца.

## <a name="data-flow-script"></a>Скрипт потока данных

### <a name="syntax"></a>Синтаксис

```
<incomingStream>
    sort(
        desc(<sortColumn1>, { true | false }),
        asc(<sortColumn2>, { true | false }),
        ...
    ) ~> <sortTransformationName<>
```

### <a name="example"></a>Пример

![Настройки сортировки](media/data-flow/sort.png "Сортировка")

Скрипт потока данных для вышеуказанной сортированной конфигурации находится в фрагменте кода ниже.

```
BasketballStats sort(desc(PTS, true),
    asc(Age, true)) ~> Sort1
```
