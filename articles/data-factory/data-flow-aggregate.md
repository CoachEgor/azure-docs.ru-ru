# <a name="aggregate-transformation-in-mapping-data-flow"></a>Преобразование Статистическая обработка в сопоставление потока данных 

[!INCLUDE [notes](../../includes/data-factory-data-flow-preview.md)]

Статистическое преобразование — это место, где вы будете определять объединения столбцов в ваших потоках данных. С помощью построителя выражений, можно определить различные типы агрегатов, таких как SUM, MIN, MAX и COUNT, который можно группировать по существующей или вычисляемых столбцов.

## <a name="group-by"></a>Группировать по
Выберите существующий столбец или создайте новый вычисляемый столбец для использования в качестве группы с предложения для вашей статистической обработки. Чтобы использовать существующий столбец, выберите нужный столбец из раскрывающегося списка. Чтобы создать новый вычисляемый столбец, наведите указатель мыши на предложение и щелкните «Вычисляемый столбец». Откроется [построитель выражений потока данных](concepts-data-flow-expression-builder.md). После создания вычисляемого столбца, введите имя выходного столбца, в поле «Имя как». Если вы хотите добавить дополнительную группу предложением, наведите указатель мыши существующего предложения и нажмите кнопку «+».

![Параметры группирования преобразование Статистическая обработка](media/data-flow/agg.png "параметры группирования Aggregate transformation")

> [!NOTE]
> Предложение group by является необязательным при преобразовании Aggregate.

## <a name="aggregate-column"></a>Столбец статистической обработки 
Перейдите на вкладку «Агрегаты» для создания статистических выражений. Можно выбрать существующий столбец и перезаписать статистической обработки или создать новое поле с новым именем. В правом поле рядом с именем средства выбора столбцов, вводится статистического выражения. Чтобы изменить выражение, щелкните в текстовом поле, чтобы открыть построитель выражений. Для добавления дополнительных статистических данных, наведите указатель мыши существующее выражение и нажмите кнопку «+», чтобы создать новый столбец статистической обработки или [шаблон столбца](concepts-data-flow-column-pattern.md).

![Статистическая обработка статистические параметры преобразования](media/data-flow/agg2.png "статистической обработки статистические параметры преобразования")

> [!NOTE]
> Каждого статистического выражения должен содержать по крайней мере одну агрегатную функцию.

> [!NOTE]
> В режиме отладки в построитель выражений не могут производить предварительного просмотра данных с агрегатными функциями. Чтобы просмотреть предварительного просмотра данных для агрегатной преобразований, закройте построитель выражений и просмотра данных с помощью вкладки «Предварительный просмотр данных».
