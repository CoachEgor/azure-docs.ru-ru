# <a name="what-are-mapping-data-flows"></a>Что такое потоки данных сопоставления?

[!INCLUDE [notes](../../includes/data-factory-data-flow-preview.md)]

Сопоставление потоков данных — это визуально спроектированное преобразование данных в фабрике данных Azure. Потоки данных позволяют инженерам данных разрабатывать логику преобразования графических данных без написания кода. Итоговые потоки данных выполняются как действия в Pipelines Фабрики данных Azure с помощью масштабированных кластеров Azure Databricks.

Намерением Потока данных Фабрики данных Azure является предоставление полностью визуальной среды без необходимости кодирования. Потоки данных будут выполняться в вашем собственном кластере выполнения для масштабированной обработки данных. Фабрика данных Azure обрабатывает все преобразование кода, оптимизацию пути и выполнение заданий потока данных.

Начните с создания потоков данных в режиме "Отладка", чтобы вы могли интерактивно проверять логику преобразования. Затем добавьте действие "Поток данных" в конвейер, чтобы выполнить и проверить поток данных при отладке конвейера, или используйте "Trigger Now" (Запустить сейчас) в конвейере, чтобы проверить Поток данных из конвейера "Действие".

Затем вы будете планировать и отслеживать действия потока данных с помощью конвейеров Фабрики данных Azure, которые выполняют действие Потока данных.

Переключатель "Режим отладки" в области конструктора Потока данных позволяет интерактивно компилировать преобразования данных. Режим отладки предоставляет среду подготовки данных и предварительный просмотр данных для построения потока данных.

## <a name="begin-building-your-data-flow-logical-graph"></a>Начало создания логической диаграммы потока данных

Начните создавать потоки данных, используя знак + в разделе ресурсы фабрики для создания нового потока данных.

![новый поток данных](media/data-flow/newdataflow2.png "новый поток данных")

Начните с настройки преобразования источника, а затем добавьте преобразование данных к каждому последующему шагу с помощью знака +. При создании логического графа можно переключаться между режимами графика и конфигурации с помощью кнопки "Показать граф" и "скрыть граф".

![Отобразить график](media/data-flow/showg.png "Отобразить график")
