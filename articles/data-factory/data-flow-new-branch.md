# <a name="creating-a-new-branch-in-mapping-data-flow"></a>Создание новой ветви в картографии потока данных

Добавьте новую ветку для проведения нескольких наборов операций и преобразований в отношении одного и того же потока данных. Добавление новой ветви полезно, если вы хотите использовать один и тот же источник для нескольких приемников или для самостоятельного объединения данных вместе.

Из списка преобразований можно добавить новую ветвь, похожую на другие преобразования. **Новая ветвь** будет доступна только в случае существующей трансформации после преобразования, вновь завершающего сяпотку.

![Добавление новой ветви](media/data-flow/new-branch2.png "Добавление новой ветви")

В приведенном ниже примере поток данных считывает данные о поездках на такси. Требуется выход, агрегированный как по дням, так и по поставщику. Вместо создания двух отдельных потоков данных, которые считываются из одного источника, может быть добавлена новая ветвь. Таким образом, обе агрегации могут быть выполнены как часть одного и того же потока данных. 

![Добавление новой ветви](media/data-flow/new-branch.png "Добавление новой ветви")
