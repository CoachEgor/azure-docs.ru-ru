# <a name="derived-column-transformation-in-mapping-data-flow"></a>Производный столбец преобразования в сопоставление потока данных

[!INCLUDE [notes](../../includes/data-factory-data-flow-preview.md)]

Для создания новых столбцов в потоке данных или для изменения существующих полей используйте преобразование "Производный столбец".

## <a name="derived-column-settings"></a>Производные параметры столбца

Чтобы переопределить существующий столбец, выберите его в специальном раскрывающемся списке столбец. В противном случае используйте поле выбора столбца как текстовое поле и введите имя нового столбца. Чтобы построить выражение производного столбца, щелкните поле «Введите выражение», чтобы открыть [построитель выражений потока данных](concepts-data-flow-expression-builder.md).

![Производные параметры столбца](media/data-flow/dc1.png "производные параметры столбца")

Чтобы добавить дополнительных производных столбцов, при наведении поверх существующего производный столбец и нажмите кнопку «+». Затем выберите «Добавить столбец» или «Добавить столбец шаблон». Шаблоны столбцов может быть полезным при имена таких столбцов переменной из источников. Дополнительные сведения см. в разделе [шаблоны столбцов](concepts-data-flow-column-pattern.md).

![Создать производный выделенный фрагмент столбца](media/data-flow/columnpattern.png "создать производный Выбор столбцов")
