# <a name="azure-data-factory-connector-overview"></a>Обзор соединителя фабрики данных Azure

Фабрика данных Azure поддерживает следующие хранилища данных: действие копирования, поток данных, действие поиска, действие получения метаданных и действие удаления. Щелкните каждое хранилище данных, чтобы узнать о поддерживаемых возможностях в подробных сведениях и соответствующих конфигурациях.
