# <a name="tutorial-manage-feature-flags-in-azure-app-configuration"></a>Руководство по Управление флагами компонентов в конфигурации приложений Azure

Можно хранить все флаги компонентов в конфигурации приложений Azure и управлять ими в одном месте. Для этого предусмотрен портал с пользовательским интерфейсом **Диспетчер функций**, который разработан специально для флагов компонентов. Кроме того, конфигурация приложения поддерживает схему данных флага компонентов .NET Core в собственном коде.

Из этого руководства вы узнаете, как выполнять следующие задачи:

> [!div class="checklist"]
> * Определяйте и управляйте флагами компонентов в конфигурации приложений.
> * Осуществляйте доступ к флагам компонентов из приложения.

## <a name="create-feature-flags"></a>Создание флагов компонентов

**Диспетчер функций** на портале Azure для конфигурации приложения предоставляет пользовательский интерфейс для создания и управления флагами компонентов, который используется в приложении. Выполните следующие действия, чтобы добавить новый флаг компонента.

1. Выберите **Диспетчер функций** > **+ Создать**, чтобы добавить флаг компонентов.

    ![Список флагов компонентов](./media/azure-app-configuration-feature-flags.png)

2. Введите уникальное имя ключа для флага компонентов. Это имя необходимо для ссылки на флаг в коде.

3. При необходимости укажите более понятное описание для флага компонента.

4. Установите начальное состояния флага компонентов. Обычно, это просто *On* (Вкл.) или *Off* (Выкл.).

    ![Создать флаг компонента](./media/azure-app-configuration-feature-flag-create.png)

5. Если указано состояние *On* (Вкл.), при необходимости, укажите любое дополнительное условие для его подтверждения с помощью команды **Добавить фильтр**. Введите встроенный или пользовательский ключ фильтрации и свяжите параметр(ы). Встроенные фильтры включают в себя:

    | Ключ | параметры JSON |
    |---|---|
    | Microsoft.Percentage | {"Value": 0–100 процентов} |
    | Microsoft.TimeWindow | {"Start": время в формате UTC, "End": время в формате UTC} |

    ![Фильтр флагов компонентов](./media/azure-app-configuration-feature-flag-filter.png)

## <a name="update-feature-flag-states"></a>Обновление состояния флага компонентов

Выполните следующие действия, чтобы изменить значение состояния флага компонентов.

1. Выберите **Диспетчер функций**.

2. Щелкните **...** > **Изменить** справа от флага компонента, который вы хотите изменить.

3. Задайте новое состояние флага компонента.

## <a name="access-feature-flags"></a>Доступ к флагам компонента

Флаги компонентов, созданные **Диспетчером функций** хранятся и извлекается в виде регулярных значений ключей. Они хранятся в специальном пространстве имен с префиксом *. appconfig.featureflag*. Можно просмотреть основные значения ключей с помощью **Обозревателя конфигурации**. Чтобы приложение могло получить их, используя поставщики конфигураций конфигурации приложений, пакеты SDK, расширения командной строки и приложения REST API.
