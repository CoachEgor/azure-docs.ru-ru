# <a name="how-to-safely-power-off-azure-fxt-edge-filer-hardware"></a>Как безопасно отключить оборудование Azure FXT Edge Filer

Хотя вы можете использовать кнопку физической мощности для включения отдельного узла, вы не должны использовать его, чтобы выключить устройство при нормальных обстоятельствах.

После того, как узел Azure FXT Edge Filer используется как часть кластера, следует использовать программное обеспечение панели управления кластера для выключения аппаратного обеспечения. 

> [!NOTE] 
> Чтобы избежать возможной потери данных или повреждения, всегда используйте программное обеспечение панели управления, чтобы выключить Файлер Azure FXT Edge. Не используйте кнопку физической мощности для выключения, если вы не проинструктированы сделать это службой поддержки и поддержки Microsoft.
> 
> В случае чрезвычайной ситуации с электричеством отключайте шнуры питания или используйте механизм отключения электроэнергии вашего центра обработки данных.

## <a name="shut-down-a-node-from-the-control-panel"></a>Выключите узла из панели управления

Следуйте этим инструкциям, чтобы безопасно отключить узел Azure FXT Edge Filer:

1. Войти на панель управления кластером. (Направления в [открытии страниц настройки](fxt-cluster-create.md#open-the-settings-pages))
1. Нажмите на вкладку **«Настройки»,** а затем загрузите страницу узлов **кластера** > **FXT.**
1. В списке кластерных узлов найдите тот, который вы хотите закрыть. Нажмите кнопку **Power Down** в столбце **«Действия».** 
1. Подождите несколько минут. Узла отключат и отключат.
