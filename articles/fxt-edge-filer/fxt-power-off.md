# <a name="how-to-safely-power-off-azure-fxt-edge-filer-hardware"></a>Безопасно выключение Azure FXT Edge Filer оборудования

Несмотря на то, что можно использовать кнопку питания, физического для переключения на отдельном узле, вам не следует использовать для завершения работы единицы в обычных условиях.

После узлом Azure FXT Edge Filer уже используется как часть кластера, следует использовать панели управления программным обеспечением кластера для завершения работы оборудования. 

> [!NOTE] 
> Чтобы избежать возможной потери данных или повреждения, всегда используете программное обеспечение панели управления для завершения работы Filer Edge FXT Azure. Не используйте кнопку питания, физического действия для завершения работы без будет предложено сделать это, в службу поддержки Майкрософт и поддержки.
> 
> В чрезвычайной электрическим отключите кабели питания или использовать данные механизм отключения электроэнергии центра.

## <a name="shut-down-a-node-from-the-control-panel"></a>Завершить работу узла с помощью панели управления

Выполните эти инструкции, чтобы безопасно питание узлом Azure FXT Edge Filer.

1. Войдите в кластер на панели управления. (Указаниям, приведенным в [откройте параметры страницы](fxt-cluster-create.md#open-the-settings-pages))
1. Нажмите кнопку **параметры** , а затем загрузить **кластера** > **FXT узлы** страницы.
1. В списке узлов кластера найдите расположение, которое вы хотите завершить работу. Нажмите кнопку **выключить питание** кнопку в его **действия** столбца. 
1. Подождите несколько минут. Узел отключается и выключение питания сам.
