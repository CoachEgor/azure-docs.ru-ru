# <a name="additional-documentation"></a>Дополнительная документация 

Некоторые ресурсы за пределами этого веб-сайта docs.microsoft.com могут помочь вам понять и работать с кэшем гибридного Edge FXT Filer для Microsoft Azure.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>VFXT Avere для документации по кэшу Azure

[VFXT Avere для Azure](https://docs.microsoft.com/azure/avere-vfxt/) — это аналогичный продукт кластера кэша, который использует виртуальные узлы в Microsoft Azure. Он создается с помощью портала Azure, но использует то же программное обеспечение конфигурации элемента управления панели в качестве кэша Azure FXT Edge Filer оборудования.

Эта технология ранее использовался в продукты Avere систем. Пользовательской документации для более старых продуктов можно найти в [веб-сайте документации для прежних версий Avere](https://azure.github.io/Avere/). 

В частности эти документы могут быть полезные сведения:

* [Руководство по настройке кластера](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) — содержит общие сведения о распространенных задачах настройки и управления кластера и содержит подробные объяснения страниц параметры кластера 

* [Руководство по панелям мониторинга](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) -описание функций панели управления **панели мониторинга** вкладку

* [Рекомендации по созданию кластера FXT](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) -рекомендации по созданию кластера из предыдущих продуктов
