# <a name="additional-documentation"></a>Дополнительная документация 

Некоторые ресурсы за пределами этого веб-сайта docs.microsoft.com могут помочь вам понять и работать с гибридным кэшем Microsoft Azure FXT Edge Filer.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Avere vFXT для документации кэша Azure

[Avere vFXT для Azure](https://docs.microsoft.com/azure/avere-vfxt/) — аналогичный кластерный кэш-продукт, в использовании виртуальных узлов в Microsoft Azure. Он создан на портале Azure, но использует то же программное обеспечение конфигурации панели управления, что и аппаратный кэш Azure FXT Edge Filer. 

## <a name="legacy-hardware-documentation"></a>Устаревшая документация по оборудованию 

Эта технология ранее использовалась в продуктах Avere Systems. Пользовательская документация для старых продуктов доступна на [веб-сайте Avere Legacy Documentation.](https://azure.github.io/Avere/) 

В частности, эти документы могут иметь полезные детали:

* [Руководство по конфигурации кластеров](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) - включает обзорную информацию об общих задачах настройки кластера и управления, а также содержит подробные объяснения страниц параметров кластера 

* [Dashboard Guide](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) - объясняет особенности вкладки панели управления **Dashboard**

* [FXT Руководство по созданию кластеров](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) - Руководство по созданию кластеров из предыдущих продуктов
