# <a name="module-assign-data-to-clusters"></a>Модуль: Присваивая данные кластерам

В этой статье описывается, как использовать модуль *Присваиваем данных для кластеров* в конструкторе машинного обучения Azure (предварительный просмотр). Модуль генерирует прогнозы с помощью модели кластеризации, которая была обучена алгоритму *кластеризации K-средств.*

Модуль Присваиваем данные кластерам возвращает набор данных, содержащий вероятные назначения для каждой новой точки данных. 

## <a name="how-to-use-assign-data-to-clusters"></a>Как использовать присваивание данных кластерам
  
1. В дизайнере машинного обучения Azure найдите ранее обученную модель кластеризации. Вы можете создать и обучить модель кластеризации, используя любой из следующих методов:  
  
    - Настроили алгоритм кластеризации K-означает с помощью модуля [кластеризации K-Means](k-means-clustering.md) и обучите модель с помощью набора данных и модуля модели кластеризации поездов (эта статья).  
  
    - Можно также добавить существующую обученную модель кластеризации из группы **Сохраненных Моделей** в рабочее пространство.

2. Прикрепите обученную модель к левому вхотвому порту **Присваиваемданные данные к кластерам.**  

3. Прикрепите новый набор данных в качестве ввода. 

   В этом наборе данных метки не являются обязательными. Как правило, кластеризация является неконтролируемым методом обучения. Вы не должны знать категории заранее. Однако столбцы ввода должны быть такими же, как и столбцы, которые использовались при обучении модели кластеризации, иначе произошла ошибка.

    > [!TIP]
    > Чтобы уменьшить количество столбцов, написанных конструктору из прогнозов кластера, используйте [столбцы Select в наборе данных](select-columns-in-dataset.md)и выберите подмножество столбцов. 
    
4. Оставьте **Check для приложения или отоверьте сьюзан для выбранного окна проверки результата только** в том случае, если вы хотите, чтобы результаты содержали полный набор входных данных, включая столбец, отображающие результаты (назначения кластеров).
  
    Если вы очистите этот флажок, будут возвращены только результаты. Эта опция может быть полезна при создании прогнозов в рамках веб-службы.
  
5.  Отправить конвейер.  
  
### <a name="results"></a>Результаты

+  Чтобы просмотреть значения в наборе данных, нажмите вправо на модуль, а затем выберите **Визуализацию.** Или выберите модуль и переключитесь на вкладку **Выходы** в правой панели, нажмите на значок гистограммы в **выводах порта,** чтобы визуализировать результат.
