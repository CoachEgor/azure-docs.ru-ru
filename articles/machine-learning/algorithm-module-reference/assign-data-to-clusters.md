# <a name="assign-data-to-clusters"></a>Назначение данных кластерам

В этой статье описывается использование [назначение данных кластерам](assign-data-to-clusters.md) модуля в визуальный интерфейс, чтобы делать прогнозы с использованием модели кластеризации, была обучена с помощью алгоритма кластеризации К-средних.

Модуль возвращает набор данных, содержащий возможные назначения для каждой новой точки данных. 


## <a name="how-to-use-assign-data-to-clusters"></a>Как использовать назначение данных кластерам
  
1.  В визуальный интерфейс найдите предварительно обученной модели кластеризации. Можно создать и обучить модель кластеризации с помощью любого из этих методов:  
  
    - Настройка алгоритма K-средних с помощью [кластеризация K-средних](k-means-clustering.md) модуля, а затем Обучение модели с помощью набора данных и [Обучение модели кластеризации](train-clustering-model.md) модуля.  
  
  
    Можно также добавить существующей обученной модели кластеризации из **сохранения моделей** в рабочей группе.

2. Присоединение обученной модели к левому порту ввода [назначение данных кластерам](assign-data-to-clusters.md).  

3. Подключите новый набор данных в качестве входных данных. В этом наборе данных метки являются необязательными. Как правило кластеризации является методом неконтролируемого обучения, поэтому не ожидается, что вы будете знать заранее категорий.

    Тем не менее входные столбцы должны быть так же, как столбцы, которые использовались при обучении, модели кластеризации, иначе возникает ошибка.

    > [!TIP]
    > Чтобы уменьшить количество выходные столбцы из кластера прогнозы, используйте [Выбор столбцов в наборе данных](select-columns-in-dataset.md)и выбрать подмножество столбцов. 
    
4. Оставьте для параметра **проверки Append или снимите флажок для результата только** выбран, если результаты должны содержать полное входного набора данных, вместе со столбцом, показывающее результат (назначений кластерам).
  
    Если вы отключите этот параметр, вы получите обратно только результаты. Это может быть полезно при создании прогнозов как часть веб-службы.
  
5.  Запустите эксперимент.  
  
### <a name="results"></a>Результаты

 
+  Для просмотра значений в наборе данных, щелкните модуль правой кнопкой мыши **привести наборы данных**и нажмите кнопку **визуализировать**.
