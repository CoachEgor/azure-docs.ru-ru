# <a name="apply-transformation-module"></a>Модуль применения преобразования

В этой статье описывается модуль визуального интерфейса (Предварительная версия) для службы машинного обучения Azure.

Этот модуль используется для изменения входного набора данных, в зависимости от ранее вычисленное преобразование.  
  
Например, если вы использовали стандартизированной величины для нормализации обучающих данных с помощью **Нормализация данных** модуля, вам следует использовать значение z показателя, вычисленная для обучения на этапе оценки также. В машинном обучении Azure можно сохранить как преобразования, а затем использовав метод нормализации **применение преобразования** для применения z показателя для входных данных до оценки.
  
Машинное обучение Azure предоставляет поддержку для создания и применения много различных типов пользовательских преобразований. Например может потребоваться сохранить, а затем повторно использовать преобразования для:  
  
- Удалите или замените отсутствующие значения, с помощью **Очистка недостающих данных**
- Нормализация данных с помощью **Нормализация данных**
  

## <a name="how-to-use-apply-transformation"></a>Как использовать применение преобразования  
  
1. Добавить **применение преобразования** эксперимент модуль. Этот модуль, в разделе можно найти **машинного обучения**в **Оценка** категории. 
  
2. Найдите существующее преобразование для использования в качестве входных данных.  Ранее сохраненные преобразования можно найти в **преобразует** группы в области навигации слева.  
  
   
  
3. Подключите набор данных, который требуется преобразовать. Набор данных должен иметь ровно ту же схему (количество столбцов, имена столбцов, типы данных) как набор данных, для которого сначала был разработан преобразование.  
  
4. Другие параметры не должны быть заданы, так как все настройки выполняются при определении преобразования.  
  
5. Чтобы применить преобразование к новый набор данных, запустите эксперимент.
