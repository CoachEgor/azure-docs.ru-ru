# <a name="apply-transformation-module"></a>Модуль применения преобразования

В этой статье описывается модуль в дизайнере машинного обучения Azure (предварительный просмотр).

Используйте этот модуль для изменения набора входных данных на основе ранее вычисленных преобразований.

Например, если вы использовали z-оценки для нормализации учебных данных с помощью модуля **Normalize Data,** вы также хотели бы использовать значение z-оценки, которое было вычислено для обучения на этапе подсчета очков. В Azure Machine Learning можно сохранить метод нормализации в качестве преобразования, а затем использовать **Apply Transformation** для применения z-оценки к входным данным перед подсчетом.

## <a name="how-to-save-transformations"></a>Как сохранить преобразования

Конструктор позволяет сохранять преобразования данных в виде **наборов данных,** чтобы можно было использовать их в других конвейерах.

1. Выберите успешно запущенный модуль преобразования данных.

1. Выберите вкладку **«Выходы и журналы».**

1. Выберите **значок Сохранить,** чтобы сохранить **преобразование результата.**

## <a name="how-to-use-apply-transformation"></a>Как использовать Apply Transformation  
  
1. Добавьте модуль **«Применить трансформацию»** в конвейер. Этот модуль можно найти в разделе **«Оценка & модели»** в палитре модуля. 
  
1. Найдите сохраненную трансформацию, которая требуется использовать в **наборах** > **данных My Datasets** в палитре модуля.

1. Соедините выход сохраненной трансформации в левый вхотливый порт модуля **Apply Transformation.**

    Набор данных должен иметь точно такую же схему (количество столбцов, имена столбцов, типы данных), что и набор данных, для которого преобразование было впервые разработано.  
  
1. Подключите выход набора данных желаемого модуля к правому вхотвому порту модуля **Apply Transformation.**
  
1. Чтобы применить преобразование в новый набор данных, запустите конвейер.
