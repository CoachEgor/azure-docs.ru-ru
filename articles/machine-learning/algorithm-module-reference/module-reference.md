# <a name="algorithm--module-reference-overview"></a>Справочные темы по алгоритмам и модулям

Ссылка в ней представлены техническими знаниями на каждый из алгоритмов машинного обучения и модули, доступные в визуальный интерфейс службы машинного обучения Azure (Предварительная версия). 

Каждый модуль представляет набор код, который можно запускать независимо друг от друга и выполнять задача машинного обучения, используя необходимые входные данные. Модуль может содержать определенный алгоритм или выполнить задачу, которая важна для машинного обучения, таких как отсутствующие замены значения или статистического анализа. 

> [!TIP]
> В любых экспериментах в визуальный интерфейс можно получить сведения о конкретного модуля. Выберите модуль, а затем выберите **помощь** ссылку в **экспресс-Справка** области.
