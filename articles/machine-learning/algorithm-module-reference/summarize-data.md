# <a name="summarize-data"></a>Сведение данных

В этой статье описывается модуль дизайнера машинного обучения Azure (предварительный просмотр).

Используйте модуль Summarize Data для создания набора стандартных статистических показателей, описывающие каждый столбец в таблице ввода.

Сводные статистические данные полезны, если вы хотите понять характеристики полного набора данных. Например, вам может понадобиться узнать:

- Сколько недостающих значений есть в каждом столбце?
- Сколько уникальных значений есть в колонке функций?
- Каково среднее и стандартное отклонение для каждой колонки?

Модуль вычисляет важные баллы для каждого столбца и возвращает ряд сводных статистических данных для каждой переменной (столбец данных), представленной в качестве ввода.

## <a name="how-to-configure-summarize-data"></a>Как настроить сумму данных  

1. Добавьте модуль **суммировать данные** в конвейер. Вы можете найти этот модуль в категории **Статистические функции** в проектировщике.

1. Подключите набор данных, для которого требуется создать отчет.

    Если вы хотите сообщить только о некоторых столбцах, используйте модуль [Select Columns в](select-columns-in-dataset.md) модуле Dataset для проецировать подмножество столбцов для работы.

1. Дополнительные параметры не требуются. По умолчанию модуль анализирует все столбцы, которые предоставляются в качестве входных данных, и в зависимости от типа значений в столбцах выводит соответствующий набор статистических данных, описанный в разделе [Результаты.](#results)

1. Отправить конвейер.

## <a name="technical-notes"></a>Технические примечания

- Для нечисловых столбцов вычисляются только значения для Число, Число уникальных значений и Число отсутствующих значений. Для других статистических показателей возвращается значение NULL.

- Столбцы, содержащие значения Boolean, обрабатываются с использованием следующих правил:

    - При вычислении Min применяется логический оператор "И".
    
    - При расчете Max применяется логическая ИЛИ
    
    - При вычислении Range модуль сначала проверяет, равно ли двум количество уникальных значений в столбце.
    
    - При вычислении любого статистического показателя, который требует вычислений с плавающей запятой, значения True принимаются равными 1,0, а значения False — 0,0.
