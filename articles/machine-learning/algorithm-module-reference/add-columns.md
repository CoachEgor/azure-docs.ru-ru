# <a name="add-columns-module"></a>Добавление модуля столбцов

В этой статье описывается модуль в дизайнере машинного обучения Azure (предварительный просмотр).

Используйте этот модуль для совмещении двух наборов данных. Для создания единого набора данных вы объединяете все столбцы из двух наборов данных, которые вы указываете в качестве входных данных. Если вам нужно совмещеть более двух наборов данных, используйте несколько **экземпляров столбцов Add.**



## <a name="how-to-configure-add-columns"></a>Как настроить добавленные столбцы
1. Добавьте модуль **«Добавленные столбцы»** в конвейер.

2. Соедините два набора данных, которые необходимо совможно. Если вы хотите объединить более двух наборов данных, можно свести воедино несколько комбинаций **столбцов Добавления.**

    - Можно объединить две колонки, которые имеют разное количество строк. Набор данных вывода набивка недостающим значениями для каждой строки в столбце меньшего источника.

    - Вы не можете выбрать отдельные столбцы для добавления. Все столбцы из каждого набора данных скатываются при использовании **столбцов Add.** Поэтому, если требуется добавить только подмножество столбцов, используйте избранные столбцы в наборе данных для создания набора данных с нужными столбцов.

3. Отправить конвейер.

### <a name="results"></a>Результаты
После запуска трубопровода:

- Чтобы увидеть первые строки нового набора данных, нажмите правой кнопкой мыши модуль **Добавления столбцов** и выберите Визуализацию. Или выберите модуль и переключитесь на вкладку **Выходы** в правой панели, нажмите на значок гистограммы в **выводах порта,** чтобы визуализировать результат.

Число столбцов в новом наборе данных равно сумме столбцов обоих входных наборов данных.

Если в наборах входных данных есть два столбца с одинаковым именем, к имени столбца добавляется числовой суффикс. Например, если есть два экземпляра столбца под названием TargetOutcome, левая колонка будет переименована TargetOutcome_1, а правая колонка будет переименована TargetOutcome_2.
