# <a name="add-columns-module"></a>Добавление столбцов модуля

В этой статье описывается модуль визуального интерфейса (Предварительная версия) для службы машинного обучения Azure.

Этот модуль используется для объединения двух наборов данных. Объединяются все столбцы из двух наборов данных, которые указаны в качестве входных данных для создания одного набора данных. Если вам нужно соединить более чем два набора данных, используйте несколько экземпляров **Добавление столбцов**.



## <a name="how-to-configure-add-columns"></a>Как настроить Добавление столбцов
1. Добавить **Добавление столбцов** эксперимент модуль.

2. Подключение двух наборов данных, которые требуется сцепить. Если вы хотите объединить более двух наборов данных, вы можете объединить в цепочку несколько комбинаций **Добавление столбцов**.

    - Это можно объединить два столбца, которые имеют разное количество строк. Выходной набор данных заполняется с отсутствующими значениями для каждой строки в исходном столбце меньшего размера.

    - Невозможно выбрать отдельные столбцы для добавления. Объединяются все столбцы из каждого набора данных, при использовании **Добавление столбцов**. Поэтому если вы хотите добавить только подмножество столбцов, используйте Выбор столбцов в наборе данных для создания набора данных со столбцами, которые нужно.

3. Запустите эксперимент.

### <a name="results"></a>Результаты
После выполнения эксперимента:

- Чтобы просмотреть первые строки новый набор данных, щелкните правой кнопкой мыши выходные данные **Добавление столбцов** и выберите визуализировать.

Число столбцов в новом наборе данных равняется сумме столбцов обоих входных наборов данных.

Если имеются два столбца с тем же именем во входных наборах данных, числовой суффикс добавляется к имени столбца. Например, если два экземпляра столбца с именем TargetOutcome, левый столбец будет переименован TargetOutcome_1 и столбцом справа будет переименован TargetOutcome_2.
