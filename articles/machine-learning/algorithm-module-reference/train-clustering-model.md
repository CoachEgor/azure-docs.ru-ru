# <a name="train-clustering-model"></a>Обучение модели кластеризации

В этой статье описывается модуль в дизайнере машинного обучения Azure (предварительный просмотр).

Используйте этот модуль для обучения модели кластеризации.

Модуль использует неподготовленную модель кластеризации, которую вы уже настроили с помощью модуля [кластеризации K-Means,](k-means-clustering.md) и обучает модель с помощью помеченного или немаркированного набора данных. Модуль создает как обученную модель, которую можно использовать для прогнозирования, так и набор кластерных заданий для каждого случая в обучающие данные.

> [!NOTE]
> Модель кластеризации не может быть обучена с помощью модуля [Train Model,](train-model.md) который является общим модулем для обучающих моделей машинного обучения. Это потому, что [Train Model](train-model.md) работает только с контролируемыми алгоритмами обучения. K-средства и другие алгоритмы кластеризации позволяют неконтролируемое обучение, что означает, что алгоритм может учиться на немаркированных данных.  
  
## <a name="how-to-use-train-clustering-model"></a>Как использовать модель кластеризации поездов  

1.  Добавьте модуль **модели кластеризации поездов** в конвейер в проектировщике. Вы можете найти модуль под **модулями машинного обучения,** в категории **Train.**  
  
2. Добавьте модуль [кластеризации K-Means](k-means-clustering.md) или другой пользовательский модуль, создающий совместимую модель кластеризации, и установите параметры модели кластеризации.  
    
3.  Прикрепите набор обучающих данных к правому вхотвому **модели кластеризации поездов.**
  
5.  В **наборе столбцов**выберите столбцы из набора данных для использования в кластерах зданий. Не забудьте выбрать столбцы, которые делают хорошие функции: например, избегать использования интра или других столбцов, которые имеют уникальные значения, или столбцов, которые имеют все те же значения.

    Если метка доступна, вы можете использовать ее в качестве функции, либо оставить ее.  
  
6. Выберите опцию, **Проверьте на наличие приложения или отменить только результат,** если вы хотите вывести обучающие данные вместе с новой меткой кластера.

    При выборе этого параметра выводится только назначения кластера. 

7. Отправить конвейер или щелкните модуль **модели кластеризации поездов** и выберите **выбранный run.**  
  
### <a name="results"></a>Результаты

После завершения обучения:

+ Чтобы сохранить моментальный снимок обученной модели, выберите вкладку **Выводы** в правой панели **модуля модели Train.** Выберите значок **набора данных Регистра,** чтобы сохранить модель в качестве многоразового модуля.

+ Для генерации баллов из модели используйте [присваивать данные кластерам.](assign-data-to-clusters.md)
