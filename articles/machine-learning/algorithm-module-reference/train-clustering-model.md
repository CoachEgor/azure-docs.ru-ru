# <a name="train-clustering-model"></a>Обучение модели кластеризации

В этой статье описывается модуль визуального интерфейса (Предварительная версия) для службы машинного обучения Azure.

Этот модуль используется для обучения модели кластеризации.

Модуль принимает необученную модель кластеризации, который уже настроен с помощью [кластеризация K-средних](k-means-clustering.md) модуля и обучает модели, используя набор данных с меткой или без метки. Модуль создает обе обученную модель, которую можно использовать для прогнозирования и набор назначения кластерам для каждого варианта в обучающих данных.

> [!NOTE]
> Быть кластеризации функционально модель обучена с использованием [Обучение модели](train-model.md) модуль, который является универсальным модулем для обучения моделей машинного обучения. Это потому, что [Обучение модели](train-model.md) работает только с контролируемыми алгоритмами обучения. K-средние и другие алгоритмы кластеризации позволяют неконтролируемого обучения, это означает, что алгоритм чему поучиться данные без метки.  
  
## <a name="how-to-use-train-clustering-model"></a>Как использовать Обучение модели кластеризации  
  
1.  Добавить **Обучение модели кластеризации** эксперимент студии модуль. Модуль в разделе можно найти **модулей машинного обучения**в **Train** категории.  
  
2. Добавить [кластеризация K-средних](k-means-clustering.md) модуля или другой пользовательский модуль, который создает совместимый кластеризации модели и задайте параметры модели кластеризации.  
    
3.  Присоединение к правой входные данные из набора данных для обучения **Обучение модели кластеризации**.
  
5.  В **набор столбцов**, выбрать столбцы из набора данных для использования при создании кластеров. Не забудьте выбрать столбцы, которые содержат хороший функции: например, старайтесь не использовать идентификаторы или других столбцов, которые имеют уникальные значения и столбцы, которые имеют те же значения.

    Если метка, можно использовать его как функцию, или оставить его.  
  
6. Выберите параметр **проверки Append или снимите флажок для результата только**, если требуется направить вывод обучающих данных, а также Новая метка для кластера.

    Если вы отключите этот параметр, только назначения кластерам выводятся. 

7. Запустите эксперимент, или нажмите кнопку **Обучение модели кластеризации** модуля и выберите **выполнить выбранные**.  
  
### <a name="results"></a>Результаты

После завершения обучения:


+  Для просмотра значений в наборе данных, щелкните модуль правой кнопкой мыши **привести наборы данных**и нажмите кнопку **визуализировать**.

+ Чтобы сохранить обученную модель для повторного использования более поздней версии, щелкните модуль правой кнопкой мыши **модели Trained**и нажмите кнопку **Сохранить как Обученную модель**.

+ Чтобы формировать оценки на основе модели, используйте [назначение данных кластерам](assign-data-to-clusters.md).
