# <a name="enter-data-manually-module"></a>Введите модуль Ручного руководства данных

В этой статье описывается модуль в дизайнере машинного обучения Azure (предварительный просмотр).

Используйте модуль **Enter Data Manually** для создания небольшого набора данных путем ввода значений. Набор данных может иметь несколько столбцов.
  
Этот модуль может быть полезен в таких сценариях, как:  
  
- Создание небольшого набора значений для тестирования.  
- Создание короткого списка меток.  
- Ввод списка имен столбцов для вставки в набор данных.

## <a name="create-a-dataset"></a>Создание набора данных 
  
1. Добавьте модуль [Enter Data Manually](./enter-data-manually.md) в конвейер. Этот модуль можно найти в категории **ввода данных и вывода данных** в разделе Машинное обучение Azure. 
  
1. Для **DataFormat**выберите один из следующих вариантов. Эти параметры определяют, каким образом следует разбирать данные, которые вы предоставляете. Требования к каждому формату сильно различаются, поэтому не забудьте прочитать соответствующие темы.  
  
   - **ARFF**: Формат файла атрибутики, используемый Weka.   
   - **CSV**: Формат значений, разделенных запятой. Для получения дополнительной информации [см.](./convert-to-csv.md)    
   - **SVMLight**: Формат, используемый Vowpal Wabbit и другими рамками машинного обучения.    
   - **TSV**: Формат разделенных значений, разделенных вкладками.

   Если вы выбираете формат и не предоставляете данные, отвечающие спецификациям формата, происходит ошибка выполнения.
  
1. Нажмите внутри текстового окна **данных,** чтобы начать ввод данных. Особого внимания требуют следующие форматы:  
  
   - **CSV**: Для создания нескольких столбцов, вставки в запятой разделенного текста, или введите несколько столбцов с помощью запятых между полями.
  
     При выборе опции **HasHeader** можно использовать первый ряд значений в качестве заголовка столбца.  
  
     При выборе этой опции используются имена столбцов (Col1, Col2 и т.д.). Имена столбцов можно добавить или изменить позже с помощью [Edit Metadata.](./edit-metadata.md)  
  
   - **TSV**: Для создания нескольких столбцов, вставки в разделите вкладку текста, или введите несколько столбцов с помощью вкладок между полями.  
  
     При выборе опции **HasHeader** можно использовать первый ряд значений в качестве заголовка столбца.  
  
     При выборе этой опции используются имена столбцов (Col1, Col2 и т.д.). Имена столбцов можно добавить или изменить позже с помощью [Edit Metadata.](./edit-metadata.md)  
  
   - **ARFF**: Вставить в существующий файл формата ARFF. Если вы накладываете значения напрямую, обязательно добавьте дополнительный заголовок и требуемые поля атрибутов в начале данных. 

     Например, следующие строки заголовка и атрибута могут быть добавлены в простой список. Заголовок колонки `SampleText`будет . Обратите внимание, что тип строки не поддерживается.
    
     ```text
     % Title: SampleText.ARFF  
     % Source: Enter Data module  
     @ATTRIBUTE SampleText NUMERIC  
     @DATA  
     \<type first data row here>  
     ```

   - **SVMLight**: Введите или вставьте значения с помощью формата SVMLight.  
  
     Например, следующая выборка представляет первые несколько строк набора данных донорства крови в формате SVMLight:  
  
     ```text  
     # features are [Recency], [Frequency], [Monetary], [Time]  
     1 1:2 2:50 3:12500 4:98   
     1 1:0 2:13 3:3250 4:28   
     ```  
  
     При запуске модуля [Enter Data Manually](./enter-data-manually.md) эти строки преобразуются в набор данных столбцов и значения индексов следующим образом:  
  
     |Col1|Col2|Col3|Col4|Метки|  
     |-|-|-|-|-|  
     |0.00016|0.004|0.999961|0.00784|1|  
     |0|0.004|0.999955|0.008615|1|  
  
1. Выберите ключ Enter после каждой строки, чтобы запустить новую строку.      
     
   При выборе Enter несколько раз добавить несколько пустых строк задней, пустые строки будут удалены или обрезаны.  
  
   Если вы создаете строки с отсутствующими значениями, вы всегда можете отфильтровать их позже.  
  
1. Подключите выходной порт к другим модулям и запустите конвейер.  
  
   Чтобы просмотреть набор данных, нажмите на модуль и выберите **Визуализацию.**
