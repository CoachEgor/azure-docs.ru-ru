# <a name="preprocess-text"></a>Preprocess Text

В этой статье описывается модуль в дизайнере машинного обучения Azure (предварительный просмотр).

Используйте модуль **Preprocess Text** для очистки и упрощения текста. Он поддерживает эти общие операции обработки текста:

* Удаление стоп-слов
* Использование регулярных выражений для поиска и замены определенных целевых строк
* Лемматизация, которая преобразует несколько связанных слов в единую каноническую форму
* Нормализация дела
* Удаление определенных классов символов, таких как числа, специальные символы и последовательности повторяющихся символов, таких как "aaaa"
* Идентификация и удаление электронных писем и URL-адресов

Модуль **Preprocess Text** в настоящее время поддерживает только английский язык.

## <a name="configure-text-preprocessing"></a>Настройка текстовая предварительная обработка  

1.  Добавьте модуль **Preprocess Text** в конвейер в Azure Machine Learning. Вы можете найти этот модуль под **text Analytics**.

1. Подключите набор данных, содержащий по крайней мере одну колонку, содержащую текст.

1. Выберите язык из списка выпадающих **языков.**

1. **Столбец текста для очистки:** Выберите столбец, который вы хотите предварительно обработать.

1. **Удалите стоп-слова**: Выберите эту опцию, если вы хотите применить заранее определенный список стоп-слов к столбце текста. 

    Списки стоп-слов зависят от языка и настраиваются.

1. **Лемматизация**: Выберите этот вариант, если вы хотите, чтобы слова были представлены в их канонической форме. Этот параметр полезен для уменьшения числа уникальных случаев аналогичных токенов текста.

    Процесс лемматизации сильно зависит от языка.

1. **Обнаружить предложения**: Выберите этот вариант, если вы хотите, чтобы модуль вставил отметку границы предложения при выполнении анализа.

    Этот модуль использует серию из `|||` трех символов трубы, чтобы представить терминатор предложения.

1. Выполняйте дополнительные операции поиска и замены с помощью регулярных выражений.

    * **Пользовательское регулярное выражение**: Определите текст, который вы ищете.
    * **Пользовательская строка замены:** Определите значение одной замены.

1. **Нормализуйте случай для нижнего регистра:** Выберите этот вариант, если вы хотите преобразовать символы верхнего регистра ASCII в их нижние формы.

    Если символы не нормализуетесь, одно и то же слово в верхнем и нижнем футляре буквы считается двумя разными словами.

1. Вы также можете удалить следующие типы символов или последовательности символов из обработанного текста вывода:

    * **Удалите номера**: Выберите эту опцию, чтобы удалить все числовые символы для указанного языка. Идентификационные номера зависят от домена и зависят от языка. Если числовые символы являются неотъемлемой частью известного слова, число может не быть удалено.
    
    * **Удалите специальные символы:** Используйте эту опцию, чтобы удалить любые не-альфанумерные специальные символы.
    
    * **Удалите дубликаты символов:** Выберите эту опцию, чтобы удалить дополнительные символы в любых последовательностях, которые повторяются более двух раз. Например, последовательность, как "aaaaa" будет сведена к "aa".
    
    * **Удалите адреса электронной почты**: `<string>@<string>`Выберите эту опцию, чтобы удалить любую последовательность формата.  
    * **Удалите URL-адреса**: Выберите эту опцию, чтобы `http` `https`удалить `ftp`любую последовательность, которая включает следующие префиксы URL: ,`www`
    
1. **Расширьте глаголы:** Эта опция применяется только к языкам, которые используют глаголы сокращений; в настоящее время, только на английском языке. 

    Например, выбрав эту опцию, можно заменить фразу *"не остаться там"* на *"не будет оставаться там"*.

1. **Нормализуйте обратные слэши на слэши:** Выберите эту опцию для картирования всех `\\` экземпляров. `/`

1. **Сплит жетоны на специальных символов**: Выберите этот вариант, если вы хотите, чтобы разорвать слова на символы, такие как `&`, `-`и так далее. Эта опция также может уменьшить специальные символы, когда он повторяется более чем в два раза. 

    Например, `MS---WORD` строка будет разделена на `MS`три `-`токена, и `WORD`.

1. Отправить конвейер.
