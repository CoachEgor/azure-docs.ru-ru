# <a name="add-rows-module"></a>Добавление модуля строк

В этой статье описывается модуль в дизайнере машинного обучения Azure (предварительный просмотр).

Используйте этот модуль для совмещении двух наборов данных. При конкатации строки второго набора данных добавляются к концу первого набора данных.  
  
Соединение строк применяется в следующих сценариях.  
  
+ Объединение в одной таблице серии статистических оценок для упрощения создания отчета.  
  
+ Объединение разных используемых наборов данных для создания окончательного набора.  

## <a name="how-to-use-add-rows"></a>Как использовать добавленные строки  

Для того чтобы конкатировать строки из двух наборов данных, строки должны иметь точно такую же схему. Это означает одинаковое количество столбцов и один и тот же тип данных в столбцах.

1.  Перетащите модуль **Add Rows** в конвейер, вы можете найти его под **преобразованием данных.**

2. Подключите наборы данных к двум входным портам. Набор данных, который необходимо добавить, должен быть подключен ко второму (правому) порту. 
  
3.  Отправить конвейер. Количество строк в наборе данных вывода должно равняться сумме строк обоих наборов входных данных.

    При добавлении одного и того же набора данных в оба ввода модуля **Add Rows** набор данных дублируется.
