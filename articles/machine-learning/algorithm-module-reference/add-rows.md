# <a name="add-rows-module"></a>Добавление модуля строк

В этой статье описывается модуль визуального интерфейса (Предварительная версия) для службы машинного обучения Azure.

Этот модуль используется для объединения двух наборов данных. В объединения строк из второго набора данных добавляются в конец первого набора данных.  
  
Объединение строк можно использовать в следующих сценариях:  
  
+ При создании серии статистических оценок и нужно объединить их в одну таблицу для упрощения создания отчета.  
  
+ Работа с разных наборов данных, а для объединения наборов данных для создания окончательного набора данных.  

## <a name="how-to-use-add-rows"></a>Как использовать Добавление строк  

Для объединения строк из двух наборов данных, строки должны иметь ровно ту же схему. Это означает, одинаковое количество столбцов и типом данных в столбцах.

1.  Перетащите **Добавление строк** модуля в эксперименте, его можно найти в разделе **преобразования данных**в **управление** категории.

2. Подключите наборы данных к двум входным портам. Набор данных, который необходимо добавить должен быть подключен к второй порт (справа). 
  
3.  Запустите эксперимент. Количество строк в выходной набор данных должно быть равно сумме строк обоих входных наборов данных.

    При добавлении одного набора данных в оба входа **Добавление строк** модуля, набор данных дублируется.
