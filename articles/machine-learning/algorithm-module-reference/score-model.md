# <a name="score-model-module"></a>Модуль "Оценка модели"

В этой статье описывается модуль в дизайнере машинного обучения Azure (предварительный просмотр).

Используйте этот модуль для создания прогнозов с помощью обученной модели классификации или регрессии.

## <a name="how-to-use"></a>Использование

1. Добавьте модуль **модели оценки** в конвейер.

2. Прикрепите обученную модель и набор данных, содержащий новые входные данные. 

    Данные должны быть в формате, совместимом с типом обучаемых моделей, которые вы используете. Схема набора входных данных должна также обычно соответствовать схеме данных, используемых для обучения модели.

3. Отправить конвейер.

## <a name="results"></a>Результаты

После того как вы создали набор баллов с помощью [модели оценки:](./score-model.md)

+ Для создания набора метрик, используемых для оценки точности (производительности модели), можно подключить набор данных для [оценки модели,](./evaluate-model.md) 
+ Нажмите на модуль и выберите **Визуализацию,** чтобы увидеть образец результатов.
<!-- + To Save the results to a dataset. -->

Оценка, или прогнозируемое значение, может быть в разных форматах, в зависимости от модели и входных данных:

- Для моделей классификации [модель оценки](./score-model.md) выводит прогнозируемое значение для класса, а также вероятность прогнозируемого значения.
- Для моделей регрессии [модель Score Model](./score-model.md) генерирует только прогнозируемое числовое значение.
