# <a name="score-model-module"></a>Модуль "Оценка модели"

В этой статье описывается модуль визуального интерфейса (Предварительная версия) для службы машинного обучения Azure.

Этот модуль используется для создания прогнозов с использованием обученной модели классификации или регрессии.

## <a name="how-to-use"></a>Использование

1. Добавить **Score Model** эксперимент модуль.

2. Подключите обученную модель и набор данных, содержащий новые входные данные. 

    Данные должны быть в формате, совместимом с типом обученной модели, которую вы используете. Схема входного набора данных также обычно должна соответствовать схеме данных, используемых для обучения модели.

3. Запустите эксперимент.

## <a name="results"></a>Результаты

После создания набора оценок с помощью [Score Model](./score-model.md):

+ Чтобы создать набор показателей, используемых для оценки точности модели (производительность).  можно подключить Оцененный набор данных для [Evaluate Model](./evaluate-model.md), 
+ Щелкните модуль правой кнопкой мыши и выберите **визуализировать** образец результатов.
+ Сохраните результаты в набор данных.

Оценка, или прогнозируемое значение может находиться в разных форматах в зависимости от модели и входных данных:

- Для моделей классификации [Score Model](./score-model.md) выводит прогнозируемое значение для класса, а также вероятность прогнозируемого значения.
- Для моделей регрессии [Score Model](./score-model.md) создает только прогнозируемое числовое значение.
- Для модели классификации изображений оценка может быть класс объекта в образе или логическое значение, указывающее, найден ли определенный компонент.
