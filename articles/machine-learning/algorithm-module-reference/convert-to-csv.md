# <a name="convert-to-csv-module"></a>Преобразование в модуль CSV

В этой статье описывается модуль в дизайнере машинного обучения Azure (предварительный просмотр).

Используйте этот модуль для преобразования набора данных в формат CSV, который можно загрузить, экспортировать или совместно использовать с модулями сценариев R или Python.

### <a name="more-about-the-csv-format"></a>Подробнее о формате CSV 

Формат CSV, который означает "связанные значения", представляет собой формат файла, используемый многими внешними инструментами машинного обучения. CSV является общим форматом обмена при работе с языками с открытым исходным кодом, такими как R или Python.

Даже если вы делаете большую часть своей работы в Azure Machine Learning, иногда может быть удобно преобразовывать набор данных в CSV для использования во внешних инструментах. Пример:

+ Загрузите файл CSV, чтобы открыть его с Excel, или импортируйте его в реляционную базу данных.  
+ Сохранить файл CSV в облачном хранилище и подключиться к нему из Power BI для создания визуализаций.  
+ Используйте формат CSV для подготовки данных для использования в R и Python. 

При преобразовании набора данных в CSV csv сохраняется в рабочем пространстве Azure ML. Для открытия и непосредственного использования файла можно использовать утилиту хранения Azure. Вы также можете получить доступ к CSV в дизайнере, выбрав **модуль Convert to CSV,** а затем выберите значок гистограммы под вкладкой **Выходов** в правой панели для просмотра вывода. Вы можете загрузить CSV из папки Результаты в местный каталог.  

## <a name="how-to-configure-convert-to-csv"></a>Как настроить Преобразование в CSV


1.  Добавьте модуль Convert в модуль CSV в конвейер. Этот модуль можно найти в группе **преобразования данных** в проектировщике. 

2. Подключите его к любому модулю, который выводит набор данных.   
  
3.  Отправить конвейер.

### <a name="results"></a>Результаты
  

Выберите вкладку **Выводы** в правой панели **Convert to CSV**и выберите на одном из этих значков под **выводами Порта.**  

+ **Зарегистрируйте набор данных:** Выберите значок и сохраните файл CSV обратно в рабочее пространство Azure ML в качестве отдельного набора данных. Вы можете найти набор данных в качестве модуля в дереве модуля в разделе **My Datasets.**

 + **Посмотреть вывод:** Выберите значок глаз и следуйте инструкциям, чтобы просмотреть **папку Results_dataset** и загрузить файл data.csv.
