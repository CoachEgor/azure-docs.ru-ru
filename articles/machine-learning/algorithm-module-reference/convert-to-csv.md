# <a name="convert-to-csv-module"></a>Преобразовать в CSV-ФАЙЛ модуля

В этой статье описывается модуль визуального интерфейса (Предварительная версия) для службы машинного обучения Azure.

Этот модуль используется для преобразования набора данных в формате CSV, который можно скачать, экспортировать или совместно с модулей сценариев R или Python.

### <a name="more-about-the-csv-format"></a>Дополнительные сведения о формате CSV 

Формат CSV, предназначенную для «значения с разделителями запятыми», — это формат файла, используемые многие внешние средства машинного обучения. CSV-ФАЙЛ является общепринятым форматом, при работе с открытым исходным кодом языками, например R или Python.

Даже в этом случае основная часть работы в машинном обучении Azure, бывают случаи, когда может быть удобно для преобразования набора данных в CSV-ФАЙЛ для использования в внешних средств. Например: 

+ Загрузить CSV-файл, чтобы открыть его с помощью Excel или импортировать его в реляционной базе данных.  
+ Сохраните файл CSV в Облачное хранилище и подключиться к нему из Power BI для создания визуализаций.  
+ Используйте формат CSV-ФАЙЛ для подготовки данных для использования в R и Python. Просто щелкните правой кнопкой мыши выходные данные модуля, для которого создается код, необходимые для доступа к данным непосредственно из Python или записной книжки Jupyter. 

При преобразовании набора данных в CSV-ФАЙЛ, файл сохраняется в рабочей области машинного Обучения Azure. Это служебная программа хранилища Azure можно использовать для открытия и использования файла напрямую, или щелкните правой кнопкой мыши выходные данные модуля и загрузить CSV-файл на компьютер или использовать его в коде R или Python.  

## <a name="how-to-configure-convert-to-csv"></a>Как настроить преобразование в CSV-ФАЙЛ

1.  Добавить [преобразовать в CSV-ФАЙЛ](./convert-to-csv.md) эксперимент модуль. Можно найти этот модуль в **преобразования формата данных** группу в интерфейсе. 

2. Подключите его к любой модуль, который выводит набор данных.   
  
3.  Запустите эксперимент.

### <a name="results"></a>Результаты
  

Дважды щелкните выходные данные [преобразовать в CSV-ФАЙЛ](./convert-to-csv.md)и выберите один из следующих вариантов.  

 + **Результирующий набор данных "->" загрузки**: Немедленно открывает копию данных в формате CSV, который можно сохранить в локальную папку. Если папка не указана, применяется значение по умолчанию имя файла и CSV-файл сохраняется в локальной **загружает** библиотеки.


 + **Результирующий набор данных "->" Сохранить как набор данных**: Сохраняет CSV-файл обратно в рабочую область машинного Обучения Azure как отдельный набор данных.

 + **Создать код доступа к данным**: Машинное Обучение Azure создает два набора кода для доступа к данным, либо с помощью Python, либо с помощью языка R. Доступ к данным, скопируйте фрагмент кода в приложении. (*Создать код доступа к данным будет обеспечена в ближайшее время.* )
