# <a name="sample-datasets-in-azure-machine-learning-designer-preview"></a>Примеры наборов данных в конструкторе Машинного обучения Azure (предварительная версия)

При создании конвейера в конструкторе Машинного обучения Azure (предварительная версия) по умолчанию в нее добавляется ряд примеров наборов данных. Эти примеры наборов данных используются в примерах конвейеров на домашней странице конструктора. 

Примеры наборов данных доступны в категории **Наборы данных**-**Примеры**. Вы можете найти ее в палитре модулей слева от холста в конструкторе. Любой из этих наборов данных можно использовать в собственном конвейере, перетащив его на холст.

Каждый пример представляет собой физическое лицо. Имеется 20 показателей, как числовых, так и категорийных, и двоичная метка (уровень кредитного риска). Записи с высоким уровнем риска имеют метку со значением 2, записи с низким уровнем риска имеют метку со значением 1. Стоимость ошибочной классификации низкого уровня риска как высокого — 1, а стоимость ошибочной классификации высокого уровня риска как низкого — 5.|

Каждый экземпляр представляет собой твит и является кортежем: идентификатор пользователя, идентификатор фильма IMDB, оценка, метка времени, число добавлений в избранное для твита и число ретвитов. Набор данных предоставлен A. Said, S. Dooms, B. Loni и D. Tikk для Recommender Systems Challenge 2014.|

Перед отправкой в конструктор набор данных был обработан следующим образом:    <br/> — идентификаторы метеорологических станций сопоставлены с соответствующими идентификаторами аэропортов;    <br/> — метеорологические станции, не связанные с 70 самыми загруженными аэропортами, удалены;    <br/> — столбец «Date» (Дата) разделен на отдельные столбцы: «Year» (Год), «Month» (Месяц) и «Day» (День);    <br/> — выбраны следующие столбцы: AirportID, Year, Month, Day, Time, TimeZone, SkyCondition, Visibility, WeatherType, DryBulbFarenheit, DryBulbCelsius, WetBulbFarenheit, WetBulbCelsius, DewPointFarenheit, DewPointCelsius, RelativeHumidity, WindSpeed, WindDirection, ValueForWindCharacter, StationPressure, PressureTendency, PressureChange, SeaLevelPressure, RecordType, HourlyPrecip, Altimeter.|

Они сохранены в формате XML.    <br/>Перед отправкой в конструктор набор данных был обработан следующим образом:    <br/> — извлечен текст по каждой конкретной компании;    <br/> — удалено форматирование Википедии;    <br/> — удалены символы, не являющиеся буквами или цифрами;    <br/> — весь текст преобразован в нижний регистр;    <br/> — добавлены известные категории компаний.    <br/>Обратите внимание, что удалось найти статьи не для всех компаний, поэтому число записей меньше, чем 500.|
