# <a name="data-science-virtual-machine-based-team-analytics-and-ai-environment"></a>Командная среда аналитики и искусственного интеллекта на базе виртуальной машины для обработки и анализа данных 
Виртуальная машина для обработки и [анализа данных](overview.md) (DSVM) предоставляет обширную среду на платформе Azure с предварительно созданным программным обеспечением для искусственного интеллекта (ии) и аналитики данных.

Традиционно DSVM использовалась в качестве индивидуального компьютера для аналитики. Отдельные специалисты по обработке и анализу данных получают производительность благодаря этой общей предварительно созданной среде аналитики. Так как крупные аналитические команды создают среды для специалистов по обработке и анализу данных, одна из повторяющихся тем — это общая инфраструктура аналитики для разработки и экспериментирования. Эта инфраструктура управляется в соответствии с корпоративными ИТ-политиками, которые также упрощают совместную работу и согласованность в группах обработки и анализа данных.

Общая инфраструктура обеспечивает более эффективное использование ИТ-среды аналитики. Некоторые организации вызывают инфраструктуру обработки и анализа данных на основе группы, которая является *песочницей аналитики*. Это позволяет специалистам по обработке и анализу данных получать доступ к различным ресурсам данных для быстрого понимания данных. Эта изолированная среда также помогает специалистам по обработке и анализу данных выполнять эксперименты, проверять достоверности и создавать прогнозные модели, не затрагивая рабочую среду.

Так как виртуальная машина для обработки и анализа данных функционирует на уровне инфраструктуры Azure, ИТ-администраторы могут легко настроить ее для работы в соответствии с ИТ-политиками предприятия. DSVM обеспечивает полную гибкость в реализации различных архитектур общего доступа, одновременно предлагая доступ к корпоративным ресурсам данных в контролируемом виде.

В этом разделе рассматриваются некоторые шаблоны и рекомендации, которые можно использовать для развертывания DSVM как командной инфраструктуры обработки и анализа данных. Поскольку стандартные блоки для этих шаблонов поступают из инфраструктуры Azure как услуги (IaaS), они применяются к любым виртуальным машинам Azure. Эта серия статей посвящена применению этих стандартных возможностей инфраструктуры Azure к DSVM.

Основными строительными блоками корпоративной среды Team Analytics являются:

* [Автоматически масштабируемый пул Dsvm](dsvm-pools.md)
