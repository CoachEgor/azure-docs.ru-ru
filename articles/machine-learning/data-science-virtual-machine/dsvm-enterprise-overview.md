# <a name="data-science-virtual-machine-based-team-analytics-and-ai-environment"></a>Командная среда аналитики и искусственного интеллекта на базе виртуальной машины для обработки и анализа данных 
[Виртуальная машина для обработки и анализа данных](overview.md) (DSVM) предоставляет среду с широкими возможностями на платформе Azure с предварительно созданным программным обеспечением для задач искусственного интеллекта и аналитики данных.

Традиционно DSVM использовалась в качестве индивидуального компьютера для аналитики. Специалисты по обработке и анализу данных повышали производительность за счет этой общей предварительно созданной аналитической среды. Когда большие аналитические команды планируют среды для специалистов по обработке и анализу данных и разработчиков искусственного интеллекта, одной из повторяющихся тем является общая аналитическая инфраструктура разработки и экспериментирования. Она управляется в соответствии с корпоративными ИТ-политиками, что также способствует упрощению совместной работы и согласованности команд по обработке и анализу данных и команд аналитики.

Общая инфраструктура позволяет ИТ-отделу лучше использовать среду аналитики. Некоторые организации также называют командную инфраструктуру аналитики, а также обработки и анализа данных *аналитической песочницей*. Она позволяет специалистам по обработке и анализу данных получать доступ к различным ресурсам данных, чтобы можно было быстро составить о них представление. Кроме того, эта среда песочницы дает возможность выполнять эксперименты, проверять гипотезы и строить прогнозные модели без влияния на рабочую среду.

Так как виртуальная машина для обработки и анализа данных функционирует на уровне инфраструктуры Azure, ИТ-администраторы могут легко настроить ее для работы в соответствии с ИТ-политиками предприятия. DSVM также предоставляет полную гибкость в реализации различных общих архитектур, обеспечивая контролируемый доступ к корпоративным данным.

В этом разделе рассматриваются некоторые шаблоны и рекомендации, которые можно использовать для развертывания DSVM как командной инфраструктуры обработки и анализа данных. Так как стандартные блоки для этих шаблонов берутся из Azure IaaS (инфраструктура как услуга), они применимы к любым виртуальным машинам Azure. В этой серии статей особое внимание уделяется применению таких стандартных возможностей инфраструктуры Azure к DSVM.
