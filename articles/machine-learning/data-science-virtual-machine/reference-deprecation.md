# <a name="reference-retirements-of-dsvm-images"></a>Справочник: прекращение использования DSVMных образов

Ниже мы расскажем о прекращении использования образов и предложениях по работе с предстоящими выпусками на виртуальной машине Azure для обработки и анализа данных.

## <a name="why-we-retire-dsvm-images"></a>Почему мы отменяем образы DSVM

В настоящее время виртуальная машина для обработки и анализа данных имеет два выпуска:

* Ubuntu
* Windows Server

Образ DSVM для этих выпусков основан на определенной версии операционной системы, например Ubuntu 18,04 LTS. Со временем период обслуживания для _версии_ операционной системы будет завершен, а средства обработки и анализа данных больше не будут поддерживать старые версии операционной системы. Чтобы DSVM образ с новейшими операционными системами и инструментами для обработки и анализа данных, мы выпустим новый образ DSVM на основе новых версий операционной системы.

## <a name="how-we-retire-dsvm-images"></a>Снятие образов DSVM

Мы выпустим _объявление о прекращении_ использования, когда существующие пользователи DSVM получают уведомление (по электронной почте) о предстоящем выпуске образа DSVM. Объявление о прекращении использования будет подробно превышать _дату выбытия_ (после этой даты, если образ недоступен) и рекомендации по устранению рисков (например, обновление до более нового образа DSVM).

На дату _объявления_ мы будем скрывать изображение в Marketplace, чтобы:

1. новые пользователи не могут случайно подготовить отмененный образ DSVM.
2. существующие пользователи могут использовать развертывания ARM до даты выбытия.

Скрытое изображение получит исправления операционной системы до _даты выбытия_ , но __не__ будет обновлять средства и платформы обработки и анализа данных. На _дату выбытия_образ будет недоступен для развертываний ARM.

Любой подготовленный образ DSVM в подписке продолжит работать после даты выбытия. Однако рекомендуется перейти на последнюю версию образа DSVM, чтобы иметь новейшие операционные системы и средства обработки и анализа данных.

## <a name="impact"></a>Влияние

Существующие подготовленные образы DSVM в подписке продолжат работать после даты прекращения использования. Однако мы рекомендуем пользователям обновить свой образ DSVM до более новой версии с помощью шаблона портал Azure или ARM.

> [!WARNING]
> Снятые с учета образы DSVM, подготовленные с помощью масштабируемых наборов виртуальных машин, не будут масштабироваться после даты выбытия.
>
> Шаблоны ARM, которые не были обновлены с помощью новых сведений об образе DSVM, не будут развернуты после даты выбытия.
