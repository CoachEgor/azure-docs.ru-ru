# <a name="data-science-virtual-machine-data-ingestion-tools"></a>Средства приема данных виртуальных машин для обработки и анализа данных

В качестве одного из первых технических этапов в проекте data science или AI необходимо определить наборы данных, которые будут использоваться, и привести их в среду аналитики. Виртуальная машина Data Science (DSVM) предоставляет инструменты и библиотеки для передачи данных из разных источников в аналитические данные локально на DSVM или в платформу данных либо в облаке, либо в локальном помещении.

Вот некоторые инструменты движения данных, которые доступны в DSVM.

## <a name="adlcopy"></a>AdlCopy

|    |           |
| ------------- | ------------- |
| Что это такое?   | Инструмент для копирования данных из хранилища Azure Blob в магазине Azure Data Lake Store. Оно также может копировать данные между двумя учетными записями Azure Data Lake Store.      |
| Поддерживаемые версии DSVM      | Windows      |
| Распространенные способы применения      | Импорт нескольких капли из хранилища Azure Blob в магазин Azure Data Lake Store.      |
|  Запуск и использование    |   Откройте запрос команды `adlcopy` и введите, чтобы получить помощь.    |
| Ссылки на примеры      | [С помощью AdlCopy](https://docs.microsoft.com/azure/data-lake-store/data-lake-store-copy-data-azure-storage-blob)      |
| Похожие инструменты на DSVM      | AzCopy, Azure CLI     |

## <a name="azure-cli"></a>Azure CLI

|    |           |
| ------------- | ------------- |
| Что это такое?   | Средство управления для Azure. Он также содержит глаголы команд для перемещения данных с платформ данных Azure, таких как хранилище Azure Blob и Azure Data Lake Store.     |
| Поддерживаемые версии DSVM      | Windows, Linux     |
| Распространенные способы применения      | Импорт и экспорт данных в магазин хранения данных Azure и Azure Data Lake Store и из нее.      |
|  Запуск и использование    |   Откройте запрос команды `az` и введите, чтобы получить помощь.    |
| Ссылки на примеры      | [Использование Azure CLI](https://docs.microsoft.com/cli/azure)     |
| Похожие инструменты на DSVM      | AzCopy, AdlCopy      |


## <a name="azcopy"></a>AzCopy

|    |           |
| ------------- | ------------- |
| Что это такое?   | Инструмент для копирования данных в локальные файлы и из них, хранилище, файлы и таблицы Azure Blob.      |
| Поддерживаемые версии DSVM      | Windows      |
| Распространенные способы применения      | Копирование файлов для хранения Azure Blob и копирование капли между учетными записями.      |
|  Запуск и использование    |   Откройте запрос команды `azcopy` и введите, чтобы получить помощь.    |
| Ссылки на примеры      | [AzCopy для Windows](https://docs.microsoft.com/azure/storage/common/storage-use-azcopy)      |
| Похожие инструменты на DSVM      | AdlCopy     |


## <a name="azure-cosmos-db-data-migration-tool"></a>Средство переноса данных Azure Cosmos DB

|    |           |
| ------------- | ------------- |
| Что это такое?   | Инструмент для импорта данных из различных источников в Azure Cosmos DB, базу данных NoS'L в облаке. Эти источники включают в себя файлы JSON, CSV-файлы, S'L, MongoDB, хранилище таблицы Azure, Amazon DynamoDB и коллекции API Azure Cosmos DB S'L API.      |
| Поддерживаемые версии DSVM      | Windows      |
| Распространенные способы применения      | Импорт файлов из VM в CosmosDB, импорт данных из таблицы Azure в CosmosDB и импорт данных из базы данных Microsoft S'L Server в CosmosDB.     |
|  Запуск и использование    |   Чтобы использовать версию командной строки, `dt`откройте запрос и введите команду. Чтобы использовать инструмент GUI, откройте `dtui`запрос команды и введите.    |
| Ссылки на примеры      | [Данные по импорту CosmosDB](https://docs.microsoft.com/azure/cosmos-db/import-data)      |
| Похожие инструменты на DSVM      | AzCopy, AdlCopy      |
