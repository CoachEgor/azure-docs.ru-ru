# <a name="using-the-deep-learning-virtual-machine"></a>Использование виртуальной машины для глубокого обучения

После подготовки виртуальной машины для глубокого обучения (DLVM) можно приступать к построению моделей глубокой нейронной сети для создания приложений с использованием функций искусственного интеллекта в таких областях, как компьютерное зрение и распознавание языка. 

В DLVM предоставляется ряд средств для искусственного интеллекта. На странице [Платформы глубокого обучения и искусственного интеллекта](dsvm-deep-learning-ai-frameworks.md) содержатся сведения об использовании этих средств. 

## <a name="deep-learning-tutorials-and-walkthroughs"></a>Руководства и пошаговые инструкции по глубокому обучению

В дополнение к примерам на основе платформы также предоставляется набор исчерпывающих пошаговых руководств, проверенных на DLVM. Эти пошаговые руководства помогут вам быстро приступить к разработке приложений для глубокого обучения в таких областях, как распознавание изображений, текста и языка. Мы продолжаем добавлять дополнительные комплексные руководства по разным областям и технологиям.   


- [Запуск нейронных сетей на разных платформах](https://github.com/ilkarman/DeepLearningFrameworks). Это комплексное руководство показывает, как переносить код с одной платформы на другую. Оно также показывает, как сравнивать эффективность модели и среды выполнения на различных платформах. 

- [Руководство. Создание комплексного решения по обнаружению продуктов на изображениях](https://github.com/Azure/cortana-intelligence-product-detection-from-images). Обнаружение изображений — это метод, который позволяет находить и классифицировать объекты на изображениях. Эта технология может быть очень полезна во многих сферах бизнеса в реальной жизни. Например, розничные продавцы могут использовать этот метод для определения того, какой продукт клиент взял с полки. Эти сведения, в свою очередь, помогают управлять запасами продукции. 

- [Hearing AI: Getting Started with Deep Learning for Audio on Azure](https://blogs.technet.microsoft.com/machinelearning/2018/01/30/hearing-ai-getting-started-with-deep-learning-for-audio-on-azure/) (ИИ для аудиообучения: начало работы с глубоким обучением для аудиоданных в Azure). В этом руководстве показано, как обучить модель глубокого обучения для обнаружения аудиособытий на основе [набора данных городских звуков](https://serv.cusp.nyu.edu/projects/urbansounddataset/urbansound8k.html), и предоставлены общие сведения о работе с аудиоданными.

- [Классификация текстовых документов](https://github.com/anargyri/lstm_han). В этом пошаговом руководстве показано, как создать и обучить две разные архитектуры нейронных сетей: сеть, ориентированную на иерархию, и сеть долгой краткосрочной памяти (LSTM). Эти нейронные сети выполняют классификацию текстовых документов с помощью API Keras для глубокого обучения. Keras — это интерфейс для трех самых популярных платформ машинного обучения: Microsoft Cognitive Toolkit, TensorFlow и Theano.
