# <a name="machine-learning-and-data-science-tools-on-azure-data-science-virtual-machines"></a>Инструменты машинного обучения и науки о данных на виртуальных машинах Azure Data Science
Виртуальные машины Azure Data Science (DSVMs) имеют богатый набор инструментов и библиотек для машинного обучения, доступных на популярных языках, таких как Python, R и Julia.

Вот некоторые из инструментов машинного обучения и библиотек на DSVMs.

## <a name="azure-machine-learning-sdk-for-python"></a>Пакет Python SDK для службы "Машинное обучение Azure"

Смотрите полную ссылку на [SDK для python для машинного обучения Azure](https://docs.microsoft.com/azure/machine-learning/overview-what-is-azure-ml)Machine Learning .

|    |           |
| ------------- | ------------- |
| Что это такое?   |   Azure Machine Learning — это облачный сервис, который можно использовать для разработки и развертывания моделей машинного обучения. Модели можно отслеживать в процессе их создания, обучения, масштабирования, а также управления ими с помощью пакета SDK для Python. Разверните модели как контейнеры и запустите их в облаке, локально или в Azure IoT Edge.   |
| Поддерживаемые выпуски     | Windows (среда Conda: AzureML), Linux (среда Conda: py36)    |
| Распространенные способы применения      | Общая платформа машинного обучения      |
| Настройка и установка      |  Установлена с поддержкой GPU.   |
| Запуск и использование      | Как Python SDK и в Azure CLI. Активируйте в среде Conda `AzureML` для Windows *или*`py36` для Linux.      |
| Ссылка на примеры      | Примеры записных книжек Jupyter приведены в каталоге `AzureML`.  |
| Связанные инструменты      | Visual Studio Code, Jupyter   |

## <a name="h2o"></a>H2O

|    |           |
| ------------- | ------------- |
| Что это такое?   | Платформа ИИ с открытым исходным кодом, которая поддерживает в памяти, распределении, быстром и масштабируемом машинном обучении.  |
| Поддерживаемые версии      | Linux   |
| Распространенные способы применения      | Распределенное, масштабируемое машинное обучение общего назначения   |
| Настройка и установка      | H2O устанавливается в `/dsvm/tools/h2o`.      |
| Запуск и использование      | Подключитесь к виртуальной машине с помощью X2Go. Запустите новый терминал и выполните `java -jar /dsvm/tools/h2o/current/h2o.jar`. Затем запустите веб-браузер и подключитесь к `http://localhost:54321`.      |
| Ссылка на примеры      | Примеры доступны на виртуальной машине в Jupyter в каталоге `h2o`.      |
| Связанные инструменты      | Apache Spark, MXNet, XGBoost, Sparkling Water, Deep Water    |

Есть несколько других библиотек машинного обучения на DSVMs, таких как популярный `scikit-learn` пакет, который является частью распространения Anaconda Python для DSVMs. Чтобы извлечь список пакетов, доступных в Python, R и Julia, запустите соответствующие диспетчеры пакетов.

## <a name="lightgbm"></a>LightGBM

|    |           |
| ------------- | ------------- |
| Что это такое?   | Быстрая, распределенная, высокопроизводительная градиент-повышающая (GBDT, GBRT, GBM, или MART) платформа, основанная на алгоритмах дерева решений. Он используется для ранжирования, классификации и многих других задач машинного обучения.    |

| Ссылка на примеры      | [Руководство LightGBM](https://github.com/Microsoft/LightGBM/tree/master/examples/python-guide)   |

## <a name="apache-drill"></a>Apache Drill
|    |           |
| ------------- | ------------- |
| Что это такое?   | Движок запроса с открытым исходным кодом S'L на больших данных    |
| Поддерживаемые версии DSVM      | Windows 2019, Linux  |
| Как он настроен и устанавливается на DSVM?      |  Установлено в `/dsvm/tools/drill*` лишь во встроенном режиме   |
| Распространенные способы применения      |  Для исследования данных на месте, не требуя экстракта, преобразования, нагрузки (ETL). Запрос различных источников данных и форматов, в том числе CSV, JSON, реляционные таблицы, и Hadoop.     |
| Как использовать и запустить его      | Кратчайший сотк рабочего стола  <br/> [Drill in 10 Minutes](https://drill.apache.org/docs/drill-in-10-minutes/) (Начало работы с Drill за 10 минут)  |
| Похожие инструменты на DSVM      |   Rattle, Weka, SQL Server Management Studio      |
