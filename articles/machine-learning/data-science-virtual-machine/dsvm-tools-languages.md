# <a name="languages-supported-on-the-data-science-virtual-machine"></a>Языки, поддерживаемые на виртуальных машинах для обработки и анализа данных 

Виртуальная машина Data Science (DSVM) поставляется с несколькими заранее построенными языками и инструментами разработки для создания приложений искусственного интеллекта (ИИ). Вот некоторые из примечательных из них.

## <a name="python-windows-server-2016-edition"></a>Python (Издание Windows Server 2016)

|    |           |
| ------------- | ------------- |
| Поддерживаемые языковые версии | Питон 2.7 и 3.7 |
| Поддерживаемые издания DSVM      | Windows Server 2016     |
| Настройка и установка на DSVM  | Создаются `conda` две глобальные среды: <br /> Среда, `root` расположенная `/anaconda/` по адресу Python 3.7. <br/> Среда, `python2` расположенная `/anaconda/envs/python2` по адресу Python 2.7.       |
| Ссылки на примеры      | Пример ноутбуков Jupyter для Python включены.     |
| Похожие инструменты на DSVM      | Пайспарк, Р, Джулия.      |

> [!NOTE]
> Сборки Windows Server 2016, созданные до марта 2018 года, содержат Python 3.5 и Python 2.7. Python 2.7 — это **среда корневой** конды, а **py37** — среда Python 3.7.

### <a name="how-to-use-and-run-it"></a>Как использовать и запустить его    

* Запуск в команде запрос:

  Откройте запрос команды и используйте один из следующих методов, в зависимости от версии Python, которая требуется запустить:

    ```
    # To run Python 2.7
    activate python2
    python --version
    
    # To run Python 3.7
    activate 
    python --version 
    ```
    
* Используйте в IDE:

  Используйте python Tools для визуальной студии (PTVS), установленные в издании Visual Studio Community. По умолчанию единственной средой, которая автоматически настраивается в PTVS, является Python 3.6. 

    > [!NOTE]
    > Чтобы указать PTVS на Python 2.7, необходимо создать пользовательскую среду в PTVS. Чтобы установить этот путь среды в Visual Studio Community Edition, перейдите на **Инструменты** -> **Python Tools** -> **Python среды** и выберите **пользовательский**. Затем установите местоположение на **c: «анаконда»--python2** и выберите **Auto Detect**.

* Использование в Jupyter:

  Откройте Jupyter и выберите **Новый** для создания нового ноутбука. Вы можете установить тип ядра как _Python «Корень Конды»_ для Python 3.7 и _Python «Conda env:python2»_ для Python 2.7.

* Установка пакетов Python:

  Среды Python по умолчанию в DSVM являются глобальными средами, которые читаемы всеми пользователями. Но только администраторы могут писать и устанавливать глобальные пакеты. Чтобы установить пакеты в глобальную среду, активируйте среду `activate` root или python2 с помощью команды в качестве администратора. Затем вы можете использовать такой `conda` `pip` набор, как менеджер пакетов, или для установки или обновления пакетов.

### <a name="how-to-use-and-run-it"></a>Как использовать и запустить его

  Откройте терминал и сделайте один из следующих, в зависимости от версии Python, которая вы хотите запустить:

    ```
    # To run Python 2.7
    source activate 
    python --version
    
    # To run Python 3.5
    source activate py35
    python --version
    
    ```
* Используйте в IDE:

  Используйте PyCharm, установленный в издании Visual Studio Community. 

* Использование в Jupyter:

  Откройте Jupyter и выберите **Новый** для создания нового ноутбука. Вы можете установить тип ядра как **Python «Корень Конды»** для Python 2.7 и **Python «Conda env:py35»** для среды Python 3.5. 

* Установка пакетов Python:

  Среды Python по умолчанию на DSVM — это глобальные среды, которые могут читать все пользователи. Но только администраторы могут писать и устанавливать глобальные пакеты. Чтобы установить пакеты в глобальную среду, активируйте среду `source activate` root или py35, используя команду в качестве администратора или пользователя с разрешениями sudo. Затем вы можете использовать такой `conda` `pip` набор, как менеджер пакетов, или для установки или обновления пакетов.

  Используйте Инструменты R для Visual Studio (RTVS), установленные в выпуске Visual Studio Community или RStudio. Они доступны в меню «Пуск» или в виде значка рабочего стола.

* Использование в Jupyter:

  Откройте Jupyter и выберите **Новый** для создания нового ноутбука. Вы можете установить тип ядра как **R** для использования ядра Jupyter R (IRKernel).

## <a name="other-languages"></a>Другие языки

**C q**: Доступно на Windows и доступно через `Developer Command Prompt for Visual Studio`издание сообщества Visual `csc` Studio или в , где вы можете запустить команду.

**Java**: OpenJDK доступен как на Linux, так и в Windows изданий DSVM и установлен на пути. Чтобы использовать Java, `javac` `java` введите запрос команды в Windows или на оболочке Bash в Linux.

**Node.js**: Node.js доступен как на Linux, так и в Windows изданий DSVM и установлен на пути. Чтобы получить доступ к Node.js, введите запрос команды `node` в Windows или `npm` на оболочке Bash в Linux. На Windows установлено расширение Visual Studio для инструментов Node.js, чтобы обеспечить графический IDE для разработки приложения Node.js.
