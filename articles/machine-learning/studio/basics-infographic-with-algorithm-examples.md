# <a name="downloadable-infographic-machine-learning-basics-with-algorithm-examples"></a>Загружаемая инфографика по основам машинного обучения с примерами алгоритмов

Скачайте понятную инфографику по основам машинного обучения и ознакомьтесь с популярными алгоритмами, которые используются для ответа на распространенные вопросы о машинном обучении. Примеры алгоритмов помогут начинающему пользователю машинного обучения понять, какие алгоритмы используются и с какой целью.

## <a name="popular-algorithms-in-machine-learning-studio-classic"></a>Популярные алгоритмы в Машинное обучение Studio (классическая модель)

[Машинное обучение Azure Studio (классическая модель)](https://studio.azureml.net/) поставляется с большой библиотекой алгоритмов для прогнозной аналитики. Этот Инфографика определяет четыре популярных семейства алгоритмов: регрессия, обнаружения аномалий, кластеризации и классификации, а также ссылки на рабочие примеры в [Коллекция решений ии Azure](https://gallery.azure.ai/). Коллекция содержит примеры экспериментов и руководства, в которых показано, как можно применять эти алгоритмы во многих реальных решениях.

* Полный алфавитный список алгоритмов и модулей в Машинное обучение Studio (классическая модель) см. в разделе [a-Z List (классическая модель) модулей машинное обучение Studio][a-z-list] в машинное обучение Studio (классическая модель) и в справке по модулям.
