# <a name="download-notebookspreview-data"></a>Скачать записные книжки (предварительный просмотр) данных

Функция Ноутбуки (предварительный просмотр) была удалена *13 апреля 2020*года. Удаление вкладок ноутбуков (предварительный просмотр) и файлов пользовательских блокнотов в настоящее время развертывается в регионах Azure по всему миру.

В этой статье содержатся пошаговые инструкции о том, как загружать данные ноутбуков (предварительный просмотр).

1. Перейдите на вкладку **Ноутбуки** в студии машинного обучения Azure (классический).
    ![Перейдите на вкладку Ноутбуки!](./media/download-notebooks/notebooks-list.png)

1. Выберите ноутбук, который вы хотите загрузить. Он откроется в Джупитере.
    ![Выберите](./media/download-notebooks/select-notebook.png) ноутбук Загрузка нескольких ноутбуков одновременно не поддерживается.

1. Перейти к -> **файлам Скачать как**, и выберите вариант формата. **Files**
    ![Параметры загрузки ноутбуков](./media/download-notebooks/download-options.PNG)

1. После выбора опции, файл ноутбука начнет сяграть. Игнорировать любые 500 или 502 ошибки, которые появляются, они не повлияют на вашу загрузку.
