# <a name="download-notebookspreview-data"></a>Скачивание данных записных книжек (Предварительная версия)

Функция записных книжек (Предварительная версия) будет удалена в *апреле 13 2020*. После этой даты вкладка записные книжки (Предварительная версия) исчезнет, и данные записных книжек нельзя будет восстановить. Скачайте данные записных книжек до 13 2020 апреля.

В этой статье приводятся пошаговые инструкции по загрузке данных записных книжек (Предварительная версия).

1. Перейдите на вкладку **записные книжки** в машинное обучение Azure Studio (классическая модель).
    ![перейдите на вкладку записные книжки.](./media/download-notebooks/notebooks-list.png)

1. Выберите записную книжку, которую требуется скачать. Он будет открыт в Jupyter.
    ![выбрать записную книжку](./media/download-notebooks/select-notebook.png) одновременное скачивание нескольких записных книжек не поддерживается.

1. Перейдите в раздел **файлы** -> **скачать как**и выберите параметр формата.
    ![параметры загрузки записной книжки](./media/download-notebooks/download-options.PNG)

1. После выбора параметра файл записной книжки начнет загружаться. Игнорируйте ошибки 500 или 502, которые отображаются, они не будут влиять на загрузку.
