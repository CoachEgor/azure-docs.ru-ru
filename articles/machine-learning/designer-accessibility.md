# <a name="use-a-keyboard-to-use-azure-machine-learning-designer-preview"></a>Использование клавиатуры для использования Машинное обучение Azure конструктора (Предварительная версия)

Узнайте, как использовать средство чтения с клавиатуры и с экрана для использования конструктора Машинное обучение Azure. Список сочетаний клавиш, которые работают везде в портал Azure, см. [в разделе сочетания клавиш в портал Azure](../azure-portal/azure-portal-keyboard-shortcuts.md)

Этот рабочий процесс тестировался с помощью [экранного диктора](https://support.microsoft.com/help/22798/windows-10-complete-guide-to-narrator) и [JAWS](https://www.freedomscientific.com/products/software/jaws/), но он должен работать с другими стандартными программами чтения с экрана.

## <a name="navigate-the-pipeline-graph"></a>Навигация по графу конвейера

Граф конвейера организован как вложенный список. Внешний список — это список модулей, который описывает все модули в графе конвейера. Внутренний список представляет собой список соединений, в котором описаны все соединения определенного модуля.  

1. В списке модуль используйте клавишу со стрелкой для переключения модулей.
1. Используйте вкладку, чтобы открыть список соединений для целевого модуля.
1. Используйте клавишу со стрелкой для переключения между портами подключения для модуля.
1. Используйте "G" для перехода к целевому модулю.

## <a name="edit-the-pipeline-graph"></a>Изменение графа конвейера

### <a name="add-a-module-to-the-graph"></a>Добавление модуля в граф

1. Используйте сочетание клавиш Ctrl + F6, чтобы переключить фокус с холста на дерево модулей.
1. Найдите нужный модуль в дереве модулей, используя стандартный элемент управления TreeView.

### <a name="edit-a-module"></a>Изменение модуля

Подключение модуля к другому модулю:

1. Используйте сочетание клавиш CTRL + SHIFT + H при нацеливании на модуль в списке модулей, чтобы открыть вспомогательное приложение подключения.
1. Измените порты подключения для модуля.

Чтобы настроить свойства модуля, выполните следующие действия.

1. Используйте сочетание клавиш CTRL + SHIFT + E при нацеливании на модуль, чтобы открыть свойства модуля.
1. Измените свойства модуля.

## <a name="navigation-shortcuts"></a>Сочетания клавиш навигации

| Нажатие клавиш | Описание |
|-|-|
| CTRL + F6 | Переключение фокуса между холстом и деревом модулей |
| Ctrl + F1   | Открыть информационную карточку при фокусировании на узле в дереве модулей |
| Ctrl + Shift + H | Открыть вспомогательное приложение подключения, если фокус находится на узле |
| Ctrl + Shift + E | Открыть свойства модуля, если фокус находится на узле |
| Ctrl + G | Переместить фокус на первый сбойный узел в случае сбоя выполнения конвейера |

## <a name="action-shortcuts"></a>Ярлыки действий

Используйте следующие сочетания клавиш с ключом доступа. Дополнительные сведения о ключах доступа см. https://en.wikipedia.org/wiki/Access_keyв разделе.

| Нажатие клавиш | Действие |
|-|-|
| Ключ доступа + R | Выполнить |
| Ключ доступа + P | Публикация |
| Ключ доступа + C | Clone (Клонировать) |
| Ключ доступа + D | Развернуть |
| Ключ доступа + I | Создание или обновление конвейера вывода |
| Ключ доступа + B | Создание или обновление конвейера пакетного вывода |
| Ключ доступа + K | Открыть раскрывающийся список "создание конвейера вывода" |
| Ключ доступа + U | Открыть раскрывающийся список "конвейер вывода обновлений" |
| Ключ доступа + M | Открыть другие (...) раскрывающийся список |
