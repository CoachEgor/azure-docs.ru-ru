# <a name="use-a-keyboard-to-use-azure-machine-learning-designer-preview"></a>Используйте клавиатуру для использования дизайнера машинного обучения Azure (предварительный просмотр)

Узнайте, как использовать клавиатуру и считыватель экрана для использования дизайнера машинного обучения Azure. Список ярлыков для клавиатур, которые работают везде [Keyboard shortcuts in the Azure portal](../azure-portal/azure-portal-keyboard-shortcuts.md) на портале Azure, см.

Этот рабочий процесс был протестирован с [диктором](https://support.microsoft.com/help/22798/windows-10-complete-guide-to-narrator) и [JAWS](https://www.freedomscientific.com/products/software/jaws/), но он должен работать с другими стандартными читателями экрана.

## <a name="navigate-the-pipeline-graph"></a>Навигация по графику конвейера

График конвейера организован как вложенный список. Внешний список представляет собой список модулей, который описывает все модули в графике конвейера. Внутренний список представляет собой список соединений, который описывает все соединения конкретного модуля.  

1. В списке модулей используйте клавишу стрелки для переключения модулей.
1. Используйте вкладку, чтобы открыть список подключений для целевого модуля.
1. Используйте клавишу стрелки для переключения между портами соединения для модуля.
1. Используйте "G", чтобы перейти к целевому модулю.

## <a name="edit-the-pipeline-graph"></a>Оторите график конвейера

### <a name="add-a-module-to-the-graph"></a>Добавление модуля к графику

1. Для переключения фокусировки с холста на дерево модуля используйте Ctrl-F6.
1. Найдите нужный модуль в дереве модуля, используя стандартный элемент управления видом дерева.

### <a name="edit-a-module"></a>Отсечение модуля

Для подключения модуля к другому модулю:

1. Используйте Ctrl и Shift H при таргетинге на модуль в списке модулей, чтобы открыть помощник аппликатора соединения.
1. Отсваивай порты соединения для модуля.

Для регулировки свойств модуля:

1. Используйте Ctrl и Shift E при таргетинге на модуль, чтобы открыть свойства модуля.
1. Отсваивай свойства модуля.

## <a name="navigation-shortcuts"></a>Навигационные ярлыки

| Нажатие клавиш | Описание |
|-|-|
| CTRL + F6 | Фокус переключения между холстом и деревом модуля |
| Ctrl и F1   | Откройте информационную карту, сосредоточившись на узлах в дереве модуля |
| Ctrl и Сдвиг | Откройте помощник соединения, когда основное внимание сосредоточено на узлах |
| Ctrl и Сдвиг | Открытые свойства модуля, когда фокус делается на узлах |
| Ctrl и G | Перемещение фокусировки в первый неузлый, если запуск конвейера не удался |

## <a name="action-shortcuts"></a>Ярлыки действий

Используйте следующие ярлыки с ключом доступа. Для получения дополнительной информации https://en.wikipedia.org/wiki/Access_keyо ключах доступа см.

| Нажатие клавиш | Действие |
|-|-|
| Ключ доступа и R | Run (Запустить) |
| Ключ доступа и P | Публикация |
| Ключ доступа c | Clone (Клонировать) |
| Ключ доступа и D | Развертывание |
| Ключ доступа i I | Создание/обновление конвейера выводов |
| Ключ доступа - B | Создание/обновление конвейера выводов пакетов |
| Ключ доступа k K | Открытое "Создание трубопровода выводов" |
| Ключ доступа u u | Открыть "Обновление вывод трубопровода" выпадение |
| Ключ доступа и M | Открыть больше (...) выпадение |
