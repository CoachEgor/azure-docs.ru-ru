Показано, как можно выполнять основные классификации, включая операции обработки данных, разделить набор данных на обучающий и проверочный наборы, обучения модели, оценки проверочного набора данных и оценить прогнозы.

## <a name="data"></a>Данные

Мы используем набор данных немецкой кредитной карты из репозитория UC Irvine.
Набор данных содержит 1000 примеры с 20 функций и 1 метка. Каждый пример представляет человека. К функциям относится числовые и категориальные компоненты. См. в разделе [веб-сайте UCI](https://archive.ics.uci.edu/ml/datasets/Statlog+%28German+Credit+Data%29) для значения категориальных признаков. Последний столбец — это метка, которая обозначает кредитного риска и имеет только два возможных значения: высокий кредитный риск = 2 и низкого кредитного риска = 1.

## <a name="experiment-summary"></a>Сводка эксперимента


Мы выполните следующие действия, чтобы создать эксперимент.

1. Перетащите модуль набора данных German Credit Card UCI Data на холст эксперимента.
1. Добавить **изменить метаданные** модуля, чтобы мы могли добавлять значимые имена для каждого столбца.
1. Добавить **разбиение данных** модуль создания обучающий и проверочный наборы. Укажите дробное число строк в первый выходной набор данных на 0,7. Этот параметр указывает, что 70% данных будут выходные данные к левому порту модуля и rest, в нужный порт. Мы используем левого набора данных для обучения и подходящий для тестирования.
1. Добавить **Two-Class Boosted Decision Tree** модуль инициализировать классификатор увеличивающегося дерева решений.
1. Добавить **Обучение модели** модуля. Подключение классификатор из предыдущего шага к левому порту ввода **Обучение модели**. Добавить обучающий набор (порт вывода слева **разбиение данных**) в правый входной порт модуля **Обучение модели**. **Обучение модели** обучить классификатор.
1. Добавить **Score Model** модуля и подключите **Обучение модели** модуля к нему. Затем добавьте проверочного набора (правый порт из **разбиение данных**) для **оценка модели**. **Score Model** будет составлять прогнозы. Вы можете выбрать его порт вывода для просмотра прогнозов и вероятностей положительному классу.
1. Добавить **Evaluate Model** модуля и подключите к его левый порт ввода Оцененный набор данных. Чтобы просмотреть результаты оценки, щелкните порт вывода **Evaluate Model** модуля и выберите **визуализировать**.
