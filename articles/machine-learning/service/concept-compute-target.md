#  <a name="what-is-a-compute-target-in-azure-machine-learning-service"></a>Что такое целевого объекта вычислений в службе машинного обучения Azure? 

Целевой объект вычислений позволяет указать вычислительный ресурс, на котором выполняется сценарий обучения или узел развертывания службы. Это расположение может быть на локальном компьютере или облачных вычислительных ресурсов.

Целевые объекты вычисления позволяют легко изменить среду службы вычислений без изменения кода.  Жизненный цикл разработки обычной модели:

* Начните с разработки и экспериментов с небольшой объем данных. На этом этапе мы рекомендуем использовать локальную среду. Например, ваш локальный компьютер или облачную виртуальную машину.
* Масштаб обучения на более крупных наборов данных или распределенного обучения, с помощью одного из [целей обучения](#train).  
* Развернуть несколько сред размещения веб-сайтов или к устройствам Интернета вещей с помощью одного из [целевых объектов развертывания](#deploy).

Вычислительные ресурсы для целевыми объектами вычислений, присоединяются к [рабочей области](concept-workspace.md). Для вычислительных ресурсов, отличных от локальной системы являются общими для пользователей рабочей области.

## <a name="managed-compute"></a>Управляемые вычисления

Управляемый вычислительный ресурс создается и управляется службой машинного обучения Azure. Этот вычислений оптимизирован для рабочих нагрузок машины обучения. Azure вычислений для машинного обучения является единственным управляемой вычислительной по состоянию на 30 мая 2019 г. В будущем могут добавляться дополнительные управляемые вычислительные ресурсы.

### <a name="amlcompute"></a> Azure Machine Learning Compute

Вычислений машинного обучения Azure можно использовать для обучения, а также для определения пакета (Предварительная версия).  С этим ресурсом вычислений у вас есть:

* Одним или несколькими node кластера
* Автоматически масштабируется при каждой отправке запуска 
* Автоматического управления кластером и планирования заданий 
* поддерживает ресурсы ЦП и GPU;

Экземпляры вычислений машинного обучения Azure можно создать с любым из следующих:

* Портал Azure
* Azure Machine Learning SDK
* Интерфейс командной строки Azure

Все другие вычислительные ресурсы необходимо создавать за пределами рабочей области и затем подключенные к ней.

## <a name="unmanaged-compute"></a>Неуправляемые вычислений

Неуправляемые вычислительный ресурс *не* управляется службой машинного обучения Azure. Создание такого рода вычислений за пределами машинного обучения Azure, а затем присоединить его к рабочей области. Неуправляемые вычислительные ресурсы можно требуются дополнительные действия для поддержания или для повышения производительности для рабочих нагрузок машин обучения.
