# <a name="introduction-to-deep-learning-vs-machine-learning"></a>Введение в глубокое обучение и машинное обучение

Эта статья поможет вам понять отношения между глубоким обучением и машинным обучением. Вы узнаете, как сравниваются два понятия и как они помещаются в более обширную категорию искусственного интеллекта. Наконец, в статье описывается, как глубокое обучение можно применять к реальным сценариям, таким как обнаружение мошенничества, распознавание речи и лиц, тональности Analytics и прогнозирование временных рядов.

## <a name="how-do-deep-learning-machine-learning-and-ai-relate-to-one-another"></a>Как глубокое обучение, машинное обучение и AI связаны друг с другом?

![Схема отношений: AI и машинное обучение и глубокое обучение](./media/concept-deep-learning-vs-machine-learning/ai-vs-machine-learning-vs-deep-learning.png)

Чтобы понять разницу между глубоким обучением и машинным обучением и AI, сведения о каждом из этих полей исследования:

- **Глубокое обучение** — это подмножество машинного обучения, основанное на искусственных нейронных сетях, которые позволяют компьютеру обучать себя. В этом случае _процесс обучения_ называется глубоким,поскольку структура искусственных нейронных сетей состоит из нескольких входных, выходных и скрытых слоев. Каждый слой содержит единицы, преобразующие входные данные в информацию, которую следующий слой может использовать для выполнения определенной прогнозной задачи. Благодаря этой структуре компьютер может изучать собственную обработку данных.

- **Машинное обучение** — это подмножество искусственного интеллекта, которое включает в себя методы (например, глубокое обучение), которые позволяют компьютерам улучшать работу в задачах. В этом случае _процесс обучения_ основан на следующих шагах:

    1. Передать алгоритм с данными, предоставив ему дополнительные сведения (например, выполнив извлечение компонентов).
    2. Используйте эти данные для обучения модели.
    3. Тестирование и развертывание модели.
    4. Использование развернутой модели для выполнения определенной автоматизированной прогнозной задачи.

- **Искусственный интеллект (ии)** — это методика, которая позволяет компьютерам имитировать человеческий интеллект. Он включает в себя машинное обучение. Важно понимать связь между _искусственным и машинным обучением и глубоким обучением_: Машинное обучение — это способ достижения искусственного интеллекта, который означает использование машинного обучения и приемов глубокого обучения, которые позволяют создавать компьютерные системы и приложения, которые могут выполнять задачи, связанные с человеческим интеллектом, такие как визуальное восприятие, распознавание речи, принятие решений и перевод между языками.

## <a name="comparison-of-deep-learning-vs-machine-learning"></a>Сравнение глубокого обучения и машинного обучения

Теперь, когда у вас есть общие сведения о машинном обучении и глубоком обучении, Сравните два метода. В случае машинного обучения алгоритму необходимо сообщить, как выполнить точный прогноз, предоставив ему дополнительные сведения. в то же время, в случае глубокого обучения алгоритм может узнать, что благодаря собственной обработке данных благодаря структуре искусственной нейронной сети.

В следующей таблице приведено более подробное сравнение двух приемов:

| |Все машинное обучение |Только глубокое обучение|
|---|---|---|
|  **Количество точек данных** | Может использовать небольшие объемы данных, предоставленные пользователями | Для создания кратких выводов требуется большой объем обучающих данных |
|  **Зависимости оборудования** | Может работать на низкоуровневых компьютерах. Для этого не требуется большой объем вычислительной мощности. | Зависит от высокопроизводительных компьютеров. Он по сути выполняет большое количество операций умножения матриц. Эти операции можно эффективно оптимизировать с помощью GPU |
|  **Процесс Добавление признаков** | Требуется точная идентификация и создание компонентов пользователями | Он изучает высокоуровневые функции данных и создает новые функции самостоятельно. |
|  **Подход к обучению** | Делит задачи на небольшие части, а затем объединяет полученные результаты в одно заключение | Решение проблемы на сквозной основе |
|  **Время выполнения** | Это занимает меньше времени на обучение, от нескольких секунд до нескольких часов | Занимает слишком много времени для обучения, поскольку в алгоритме глубокого обучения имеется много параметров |
|  **Выходные данные** | Выходным значением обычно является числовое значение, например оценка или классификация. | Результат может быть любым из оценки, текста, элемента или звука. |

## <a name="deep-learning-use-cases-what-problems-does-it-solve"></a>Примеры использования для глубокого обучения: Какие проблемы он решает?

Из-за структуры искусственной нейронной сети глубокое обучение предназначен в определении закономерностей в неструктурированных данных, таких как изображения, звук, видео и текст. По этой причине можно быстро преобразовывать множество отраслей, включая здравоохранение, энергию, финансово, транспорт и др., чтобы пересчитать традиционные бизнес-процессы. Некоторые из наиболее распространенных приложений для глубокого обучения описаны в следующих абзацах.

### <a name="named-entity-recognition"></a>Распознавание именованных сущностей

Одним из способов использования сетей с глубоким обучением является распознавание именованных сущностей, которое представляет собой способ извлечения из неструктурированных, непомеченных данных определенных типов информации, таких как люди, места, компании или вещи. Затем эти сведения могут храниться в структурированной схеме для создания списка адресов или служить тестом производительности для обработчика проверки личности.

### <a name="object-detection"></a>Обнаружение объектов

Глубокое обучение было применено во многих вариантах использования для обнаружения объектов. Обнаружение объектов на самом деле состоит из двух частей: классификация изображений, а затем Локализация образа. Классификация изображений определяет, что такое объекты в изображении, как автомобиль или человек, а локализация изображений предоставляет определенное расположение этих объектов. Обнаружение объектов уже используется в игровых, розничных, туристическая и самостоятельных отраслях автомобилей.

### <a name="image-caption-generation"></a>Создание заголовка изображения

Как и в случае с задачей распознавания изображений, субтитры изображений — это задача, где для конкретного изображения система должна создать заголовок, описывающий содержимое изображения. После того как вы обнаружите объекты в фотографиях и создадите метки для этих объектов, можно увидеть, что следующий шаг заключается в том, чтобы преобразовать эти метки в описание предложения. Как правило, в системах используются очень крупные перегрузки нейронных сетей для обнаружения объектов в фотографиях, а затем перетекущая нейронная сеть (РНН) для преобразования меток в согласованное предложение.

### <a name="machine-translation"></a>Машинный перевод

Машинный перевод принимает слова, фразы или предложения на одном языке и автоматически преобразует их в другой язык. Автоматический машинный перевод занимает много времени, но глубокое обучение дает лучшие результаты в двух конкретных областях: Автоматический перевод текста (и преобразование речи в текст) и автоматический перевод изображений. Благодаря правильному преобразованию данных глубокая сеть может понимать текст, звук и визуальные сигналы. Машинный перевод можно использовать для распознавания фрагментов звука в больших звуковых файлах и транскрипция слово или изображение в виде текста.

### <a name="text-analytics"></a>Текстовая аналитика

Одной из важных задач, которые может выполнять глубокое обучение, является электронное обнаружение. Компании используют текстовую аналитику, основанную на глубоком обучении, для обнаружения торговых организаций и соблюдения нормативных требований. Хеджирование средства используют текстовую аналитику для детализации больших репозиториев документов, чтобы получить ценные сведения о будущих инвестициях и тональностиах рынка. Вариант использования для анализа текста на основе глубокого обучения основан на возможности анализа больших объемов текстовых данных для выполнения анализа или получения агрегатов.

## <a name="what-are-artificial-neural-networks"></a>Что такое искусственные нейронные сети?

Искусственные нейронные сети формируются с помощью слоев подключенных узлов. Модели глубокого обучения используют нейронные сети с очень большим количеством уровней. Наиболее популярные типологиес искусственной нейронной сети обсуждаются ниже.

### <a name="feedforward-neural-network"></a>Нейронная сеть фидфорвард

Нейронная сеть фидфорвард — это основной тип искусственной нейронной сети, в которой информация перемещается только в одном направлении от уровня входных данных к выходному слою. Нейронные сети фидфорвард преобразуют входные данные, поместив их через ряд скрытых слоев. Каждый слой состоит из набора нейронов, каждый из которых полностью подключен ко всем нейронам в слое. Наконец, имеется последний полностью подключенный слой — выходной слой, который представляет создаваемые прогнозы.

### <a name="recurrent-neural-network"></a>Перетекущая нейронная сеть

Реактуальные нейронные сети — это более широко используемый тип искусственной нейронной сетью, который работает с принципом сохранения выходных данных слоя и передачи его обратно на входной слой для помощи в прогнозировании результатов слоя. Эти нейронные сети обладают большими возможностями обучения и широко используются для более сложных задач, таких как обучение рукописного ввода или распознавание языка.

### <a name="convolutional-neural-networks"></a>Свертка нейронных сетей

Свертка нейронной сети — это особенно эффективный тип искусственной нейронной сети, который представляет собой уникальную архитектуру. Сначала уровни упорядочены по трем измерениям: ширина, высота и глубина. Кроме того, нейроны на одном уровне не подключаются ко всем нейронам на следующем уровне, но только в небольшой области. Наконец, окончательный результат будет уменьшен до одного вектора показателей вероятности, организованных вдоль измерения глубины. Эти нейронные сети использовались в таких областях, как распознавание изображений и классификация.
