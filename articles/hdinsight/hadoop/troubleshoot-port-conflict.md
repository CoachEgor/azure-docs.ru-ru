# <a name="scenario-port-conflict-when-starting-services-in-azure-hdinsight"></a>Сценарий: Конфликт в порте при запуске служб в Azure HDInsight

В этой статье описаны шаги устранения неполадок и возможные решения проблем при взаимодействии с кластерами Azure HDInsight.

## <a name="issue"></a>Проблема

Служба не заводит ся.

## <a name="cause"></a>Причина

Существует портовый конфликт.

## <a name="resolution"></a>Решение

### <a name="method-1"></a>Метод 1

Используйте ниже команды, чтобы получить / убить все запущенные процессы, которые зависят от проблемы порта.

```bash
netstat -lntp | grep <port>
ps -ef | grep <service>
kill -9 <service>
```

Затем начните службу.

### <a name="method-2"></a>Метод 2

Перезагрузите узла.
