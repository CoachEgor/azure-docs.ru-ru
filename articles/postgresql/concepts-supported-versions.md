# <a name="supported-postgresql-major-versions"></a>Поддерживаемые основные версии PostgreS'L
Корпорация Майкрософт намерена поддерживать n-2 версии движка PostgreS'L в базе данных Azure для PostgreS-L - Single Server. В Azure будут поддерживаться текущая основная версия (n) и две предыдущие основные версии (–2).

База данных Azure для PostgreS'L в настоящее время поддерживает следующие основные версии:

## <a name="postgresql-version-11"></a>ПостгреСЗЛ версия 11
Текущий незначительный релиз 11,5. Обратитесь к [документации PostgreS'L,](https://www.postgresql.org/docs/11/static/release-11-5.html) чтобы узнать больше об улучшениях и исправлениях в этом незначительном выпуске.

## <a name="postgresql-version-10"></a>ПостгреСЗЛ версия 10
Текущий незначительный релиз 10.10. Обратитесь к [документации PostgreS'L,](https://www.postgresql.org/docs/10/static/release-10-10.html) чтобы узнать больше об улучшениях и исправлениях в этом незначительном выпуске.

## <a name="postgresql-version-96"></a>ПостгреСЗЛ версия 9.6
Текущий незначительный релиз 9.6.15. Обратитесь к [документации PostgreS'L,](https://www.postgresql.org/docs/9.6/static/release-9-6-15.html) чтобы узнать больше об улучшениях и исправлениях в этом незначительном выпуске.

## <a name="postgresql-version-95"></a>ПостгреСЗЛ версия 9.5
Текущий незначительный релиз 9.5.19. Обратитесь к [документации PostgreS'L,](https://www.postgresql.org/docs/9.5/static/release-9-5-19.html) чтобы узнать об улучшениях и исправлениях в этом незначительном выпуске.

## <a name="managing-upgrades"></a>Управление обновлениями
Проект PostgreS'L регулярно выпускает незначительные релизы для исправления зарегистрированных ошибок. База данных Azure для PostgreS'L автоматически патчирует серверы с незначительными релизами во время ежемесячного развертывания службы. 

Автоматическое обновление основной версии не поддерживается. Например, автоматическое обновление с PostgreSQL 9.5 до PostgreSQL 9.6 не предусмотрено. Чтобы перейти на следующую основную версию, создайте [дамп базы данных и восстановите](./howto-migrate-using-dump-and-restore.md) сервер, созданный с новой версией двигателя.

### <a name="version-syntax"></a>Версия синтаксиса
До версии 10 PostgreS'L [политика версий PostgreS'L](https://www.postgresql.org/support/versioning/) рассматривала _повышение_ первой _или_ второй версии. Например, _основным_ обновлением версии считалось обновление от 9,5 до 9,6. По версии 10, только изменение первого номера считается основным обновлением версии. Например, 10.0 до 10.1 является _незначительным_ обновлением релиза. Версия 10 до 11 является _основным_ обновлением версии.
