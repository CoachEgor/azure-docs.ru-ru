# <a name="azure-vmware-solution-avs-upgrade-concepts"></a>Основные понятия обновления решения VMware для Azure (AVS)

Одним из основных преимуществ частных облаков решения Azure VMware (AVS) является то, что платформа поддерживается. Обслуживание платформы включает автоматическое обновление пакета с проверенным программным обеспечением VMware. Обновления выполняются регулярно, гарантируя, что у вас всегда есть последние проверенные версии программного обеспечения.

## <a name="avs-private-cloud-software-upgrades"></a>Обновления программного обеспечения частного облака AVS

Платформа частного облака AVS включает в себя определенные версии программного обеспечения VMware vSphere, ESXi, vSAN и НСКС-T. Пакет программного обеспечения частного облака проверяется на использование в новых установках частного облака, а также на обновление существующих частных облаков.

Процесс обновления управления жизненным циклом не требует простоя для частных облаков. Процесс обновления гарантирует, что вы будете автоматически использовать последнюю версию проверенного программного обеспечения частного облака AVS. Обновления применяются к регулярной ритмичности, чтобы частные облака никогда не были более одной версии за последним выпуском проверенного пакета программного обеспечения. Вы получите уведомления о запланированных обновлениях вашего частного облака. Вы можете отложить обновление, если ваше частное облако находится в пределах одной версии последнего выпуска.

Критические исправления и обновления применяются при проверке. Вы получите предварительные уведомления о необходимых критических обновлениях. Эта политика обеспечивает немедленное применение критических исправлений и обновлений для частного облака.

Версии программного обеспечения VMware предоставляются в [статье концепции частных облаков и кластеров](concepts-private-clouds-clusters.md) и [часто задаваемые вопросы](faq.md).
