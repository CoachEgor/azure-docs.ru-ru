# <a name="azure-firewall-remote-work-support"></a>Поддержка удаленной работы брандмауэра Azure

Брандмауэр Azure — это управляемая облачная служба безопасности сети, которая защищает ресурсы виртуальной сети Azure. Это высокодоступная служба с полным отслеживанием состояния и неограниченными возможностями облачного масштабирования. 

## <a name="firewall-rules"></a>Правила брандмауэра

Вы можете использовать брандмауэр Azure для защиты входящего доступа по протоколу удаленного рабочего стола (VDI) к виртуальной сети Azure с помощью правил брандмауэра Azure [ДНаТ](rule-processing.md). Для виртуальных рабочих столов Windows (ВВД) не требуется открывать входящий доступ к виртуальной сети. Однако необходимо разрешить набор исходящих сетевых подключений для виртуальных машин ВВД, работающих в виртуальной сети. Дополнительные сведения см. в статье [что такое виртуальный рабочий стол Windows?](../virtual-desktop/overview.md#requirements)

Вы можете настроить этот исходящий доступ с помощью правил приложения брандмауэра Azure. Дополнительные сведения см. [в разделе Учебник. Развертывание и настройка брандмауэра Azure с помощью портал Azure](tutorial-firewall-deploy-portal.md).
