# <a name="create-ip-groups-preview"></a>Создание групп IP-адресов (Предварительная версия)

> [!IMPORTANT]
> Эта общедоступная предварительная версия предоставляется без соглашения об уровне обслуживания и не должна использоваться для производственных рабочих нагрузок. Некоторые функции могут не поддерживаться, иметь ограничения и быть доступными не во всех расположениях Azure. См. [дополнительные условия использования для предварительных версий Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

Группы IP позволяют группировать IP-адреса и управлять ими для правил брандмауэра Azure. Они могут иметь один IP-адрес, несколько IP-адресов или один или несколько диапазонов IP-адресов.

## <a name="create-an-ip-group"></a>Создание группы IP-адресов

1. На домашней странице портала Azure выберите **Создать ресурс**.
2. Введите **IP-группы** в текстовом поле Поиск, а затем выберите **IP-группы**.
3. Нажмите кнопку **Создать**.
4. Выберите свою подписку.
5. Выберите группу ресурсов или создайте новую.
6. Введите уникальное имя для IP-группы, а затем выберите регион.

6. Выберите **Далее: IP-адреса**.
7. Введите IP-адрес, несколько IP-адресов или диапазоны IP-адресов.

   Существует два способа ввода IP-адресов.
   - Их можно ввести вручную
   - Их можно импортировать из файла.

   Чтобы выполнить импорт из файла, выберите **Импорт из файла**. Вы можете либо перетащить файл в поле, либо выбрать пункт **Обзор файлов**. При необходимости можно просматривать и изменять отправленные IP-адреса.

   При вводе IP-адреса портал проверяет его на наличие перекрывающихся, повторяющихся и форматированных проблем.

5. По завершении выберите **проверить и создать**.
6. Нажмите кнопку **Создать**.
