# <a name="create-ip-groups-preview"></a>Создание IP-групп (предварительный просмотр)

> Эта общедоступная предварительная версия предоставляется без соглашения об уровне обслуживания и не должна использоваться для производственных рабочих нагрузок. Некоторые функции могут не поддерживаться, иметь ограничения и быть доступными не во всех расположениях Azure. См. [дополнительные условия использования для предварительных версий Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

IP-группы позволяют группировать и управлять IP-адресами для правил Azure Firewall. Они могут иметь один IP-адрес, несколько IP-адресов или один или несколько диапазонов IP-адресов.

## <a name="create-an-ip-group"></a>Создание группы IP

1. На домашней странице портала Azure выберите **Создать ресурс**.
2. Введите **IP-группы** в текстовом поле поиска, а затем выберите **IP-группы.**
3. Выберите **Создать**.
4. Выберите свою подписку.
5. Выберите группу ресурсов или создайте новую.
6. Введите уникальное имя для IP Group, а затем выберите регион.

6. Выберите **Далее: IP-адреса**.
7. Введите IP-адрес, несколько IP-адресов или диапазоны IP-адресов.

   Есть два способа ввести IP-адреса:
   - Вы можете вручную ввести их
   - Вы можете импортировать их из файла

   Для импорта из файла выберите **Импорт из файла**. Вы можете перетащить файл в поле или выбрать **просмотр файлов.** При необходимости вы можете просматривать и отсваговать загруженные IP-адреса.

   При вводе IP-адреса портал проверяет его для проверки на наличие проблем с дублированием, дублированием и форматированием.

5. После завершения, выберите **Обзор и Создать**.
6. Выберите **Создать**.
