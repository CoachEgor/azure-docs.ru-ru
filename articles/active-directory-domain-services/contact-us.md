# <a name="azure-ad-domain-services---contact-us"></a>Доменные службы Azure AD — связь с нами

## <a name="feedback"></a>Отзывы
Если у вас есть отзывы по нашим службам или нетехническим вопросам, вы можете поделиться своим запросом с группой продуктов Azure AD Domain Service по электронной почте: [Azure AD Domain Services.](mailto:aaddsfb@microsoft.com) Электронные письма на этот адрес рассматриваются. Члены группы продуктов **могут** ответить на запрос дополнительной информации. Запросы на техническую поддержку на этот адрес электронной почты не будут отвечать.

## <a name="technical-assistance"></a>Техническая помощь
Если у вас возникла проблема с управляемым доменом, попробуйте выполнить действия, описанные в [руководстве по устранению неполадок](troubleshoot.md) . Если у вас все еще возникли проблемы, вопием на порталAz. Откройте домашнюю страницу **доменных служб Azure AD** и выберите **новый запрос поддержки** из панели действий под **поддержкой и устранением неполадок.**

![Новый запрос на поддержку](./media/contact-us/supportRequest.png) 

Заполните каждый раздел запроса поддержки как можно точнее. При выборе службы убедитесь, что вы выберите **службы домена Azure Active Directory Domain Services (VM - Domain Controllers),** чтобы направить ваш запрос к соответствующему специалисту по поддержке.

> [!IMPORTANT]
> Выберите **каталоги активных каталогов, доменов и объектов Из списков** служб для справки о проверке домена с помощью Active Directory Azure.
>
>

У вас есть представление о том, как мы можем улучшить службы домена Active Directory? Мы хотели бы услышать об этом. Перейдите на форумы отзывов Azure, перейдите в категорию [domain Services](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593) и поделитесь своей идеей. Члены группы продуктов рассматривают эти запросы.
