# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>Поддерживаемые версии языка и времени выполнения для Apache Spark и зависимых компонентов  

Служба Apache Spark в Azure Synapse Analytics поддерживает несколько различных времен выполнения и службы в этом документе перечисляют версии.

## <a name="spark-version"></a>Версия Spark

Spark 2.4 является в настоящее время поддерживается версия.

## <a name="operating-system-version"></a>Версия операционной системы

Apache Spark в Azure Synapse работает на версии Ubuntu 16.04.

## <a name="java-version"></a>Версия Java

Apache Spark в Azure Synapse включает java-версию 1.8.0-222

## <a name="scala-version"></a>версия Scala;

Искра Apache в Azure Synapse включает Scala 2.11.12

## <a name="net-core-version"></a>Версия .NET Core

Apache Spark в Azure Synapse включает в себя .NET Core 3.0

## <a name="net-support-for-apache-spark"></a>Поддержку .NET для Apache Spark

Azure Synapse включает в себя поддержку Apache Spark 0.5.0

## <a name="delta-lake"></a>Delta Lake

Synapse Analytics включает поддержку Delta Lake 0.5.0

## <a name="python-version"></a>Версия Python

Apache Spark в Azure Synapse включает в себя установку версии Python 3.6.1
