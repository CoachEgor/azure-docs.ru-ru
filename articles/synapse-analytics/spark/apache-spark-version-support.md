# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>Поддерживаемые язык и версии среды выполнения для Apache Spark и зависимых компонентов  

Apache Spark в службе Azure синапсе Analytics поддерживает несколько различных времени выполнения и служб в этом документе перечислены версии.

## <a name="spark-version"></a>Версия Spark

Spark 2,4 — это поддерживаемая сейчас версия.

## <a name="operating-system-version"></a>Версия операционной системы

Apache Spark в Azure синапсе работает в Ubuntu версии 16,04.

## <a name="java-version"></a>Версия Java

Apache Spark в Azure синапсе включает Java версии 1.8.0 _222

## <a name="scala-version"></a>версия Scala;

Apache Spark в Azure синапсе включает Scala 2.11.12

## <a name="net-core-version"></a>Версия .NET Core

Apache Spark в Azure синапсе включает .NET Core 3,0

## <a name="net-support-for-apache-spark"></a>Поддержка .NET для Apache Spark

Azure синапсе включает поддержку .NET для Apache Spark 0.5.0

## <a name="delta-lake"></a>Delta Lake

Синапсе Analytics включает поддержку Delta Lake 0.5.0

## <a name="python-version"></a>Версия Python

Apache Spark в Azure синапсе включает установку Python версии 3.6.1
