| Северный Китай 2 |31 марта 2021 г. |
| Республика Корея, южный регион |31 марта 2021 г. |
| Южная Индия |31 марта 2021 г. |
| Западная Индия |31 марта 2021 г. |
| Западная Швейцария |31 марта 2021 г. |
| Северная часть ОАЭ; |31 марта 2021 г. |
