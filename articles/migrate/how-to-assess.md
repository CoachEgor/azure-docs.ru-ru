# <a name="add-assessment-tools"></a>Добавление средства оценки

В этой статье описывается, как добавить средства оценки в ["Миграция Azure"](migrate-overview.md).

Служба "Миграция Azure" предоставляют концентратор средств для оценки и миграции в Azure. Она также включает собственного средства, средства, предоставляемые других служб Azure и сторонних независимых Независимым производителям предложения.

Если вы хотите добавить средство оценки и у вас еще нет проекта миграции Azure, выполните это [статье](how-to-add-tool-first-time.md).

## <a name="selecting-an-isv-tool"></a>Выбрав средство независимых поставщиков программного обеспечения

Если вы выберете [независимых поставщиков программного обеспечения средство](migrate-services-overview.md#isv-integration) для оценки, можно запустить, получении лицензии или регистрации для получения бесплатной пробной версии, в соответствии с политикой независимых поставщиков программного обеспечения. В каждое средство имеется возможность подключиться к "Миграция Azure". Следуйте инструкциям средства и документацию к сообществу средства рабочей области "Миграция Azure". 


## <a name="select-an-assessment-scenario"></a>Выберите сценарий оценки

1. В проекте "Миграция Azure", щелкните **Обзор**.
2. Выберите сценарий оценки, которую вы хотите использовать:

    - Чтобы обнаружить и оценить машин и рабочих нагрузок для миграции в Azure, выберите **оценки и миграции серверов**.
    - Чтобы оценить локальные компьютеры SQL, выберите **оценка и миграция баз данных**.
    - Чтобы оценить на локальном веб-приложений, выберите **оценка и перенос веб-приложений**.

    ![Сценарий оценки](./media/how-to-assess/assess-scenario.png)

## <a name="select-a-server-assessment-tool"></a>Выберите средство оценки server 

1. Нажмите кнопку **оценки и миграции серверов**.
2. В **"Миграция Azure" - серверы**, если вы еще не добавили средство для оценки, в разделе **инструменты оценки**выберите **щелкните здесь, чтобы добавить средство оценки**. Если вы уже добавили инструменты оценки, в **добавить дополнительные инструменты оценки**выберите **изменение**.

    > [!NOTE]
    > Если необходимо перейти к другому проекту, в **"Миграция Azure" - серверы**рядом с элементом **см. в разделе сведений о проекте другой миграции**, нажмите кнопку **здесь**.

3. В **"Миграция Azure"** , выберите средство оценки, вы хотите использовать.

    
    ![Средства оценки](./media/how-to-assess/assess-tool.png)

    - При использовании оценки сервера миграции Azure, можно настроить, запуска и Просмотр оценок непосредственно в проекте "Миграция Azure".
    - Если вы используете это средство оценки независимых производителей, перейдите по ссылке для своего сайта и выполнить оценку в соответствии с инструкциями, которые они предоставляют.


## <a name="select-a-database-assessment-tool"></a>Выберите средство оценки базы данных

1. Нажмите кнопку **оценка и миграция баз данных**
2. В **баз данных**, нажмите кнопку **добавить средства**.
3. В добавить средство > **средство оценки выберите**, выбрать инструмент, который вы хотите использовать для оценки базы данных.

## <a name="select-a-web-app-assessment-tool"></a>Выберите веб-инструмент оценки приложений

1. Нажмите кнопку **оценка и перенос веб-приложений**.
2. Перейдите по ссылке на средства миграции для службы приложений Azure. Используйте средство миграции, чтобы:

    - **Оценка приложения в Интернете**: Можно оценить приложения с помощью общедоступного URL-адрес в Интернете, с помощью Azure App Service Migration Assistant.
    - **.NET/PHP**: Внутренние приложения .NET и PHP можно загрузить и запустить помощник по миграции.
