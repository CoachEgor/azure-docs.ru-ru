# <a name="add-assessment-tools"></a>Добавление средств оценки

В этой статье описывается, как добавлять инструменты оценки в [Azure Migrate](migrate-overview.md).

Azure Migrate предоставляет центр инструментов для оценки и миграции в Azure. Она включает в себя инструменты Azure Migrate, а также другие инструменты и независимые предложения поставщиков программного обеспечения (ISV).

Если вы хотите добавить инструмент оценки и у вас еще нет проекта Azure Migrate, следуйте этой [статье](how-to-add-tool-first-time.md):

## <a name="select-a-tool"></a>Выбор средства

Если вы выбираете инструмент для оценки, не связанный с Azure Migrate, начните с получения лицензии или подписки на бесплатную пробную версию в соответствии с политикой инструмента. Инструменты имеют возможность подключения к Azure Migrate. Следуйте инструкциям и документации, чтобы подключить инструмент к Azure Migrate. [Узнайте больше](migrate-services-overview.md) об инструментах.


## <a name="select-an-assessment-scenario"></a>Выберите сценарий оценки

1. В проекте службы "Миграция Azure" щелкните **Обзор**.
2. Выберите сценарий оценки, который вы хотите использовать:

    - Чтобы обнаружить и оценить машины и рабочие нагрузки для миграции в Azure, выберите **Оценку и перенос серверов.**
    - Для оценки нахэляжных машин S'L выберите **Оценку и перенос баз данных.**
    - Для оценки предпосылок веб-приложений выберите **Оценку и перенос веб-приложений.**

    ![Сценарий оценки](./media/how-to-assess/assess-scenario.png)

## <a name="select-a-server-assessment-tool"></a>Выберите инструмент оценки сервера 

1. Нажмите **Оценка и миграция серверов**.
2. В **Azure Migrate - Серверы,** если вы еще не добавили инструмент оценки, под **инструменты оценки,** выберите **Нажмите здесь, чтобы добавить инструмент оценки.** Если вы уже добавили инструменты оценки, в **добавлении дополнительных инструментов оценки**выберите **Изменение**.

    > [!NOTE]
    > Если вам нужно перейти к другому проекту, в **Azure Migrate - Серверы**, рядом с **Просеяните детали для другого проекта мигрировать,** нажмите **Нажмите здесь**.

3. В **Azure Migrate**выберите инструмент оценки, который вы хотите использовать.

    - При использовании оценки Azure Migrate Server, можно настроить, запустить и просмотреть оценки непосредственно в проекте Azure Migrate.
    - Если вы используете другой инструмент оценки, перейдите по ссылке, предоставленной для их сайта, и запустите оценку в соответствии с инструкциями, которые они предоставляют.


## <a name="select-a-database-assessment-tool"></a>Выберите инструмент оценки базы данных

1. Нажмите **Оценка и миграция баз данных**
2. В **базах данных**нажмите **Добавить инструменты**.
3. В дополнение к инструменту > **Выберите инструмент оценки,** выберите инструмент, который вы хотите использовать для оценки базы данных.

## <a name="select-a-web-app-assessment-tool"></a>Выберите инструмент оценки веб-приложений

1. Нажмите **Оценка и мигрировать веб-приложений**.
2. Перейдите по ссылке на инструмент «Миграция» для службы приложений Azure. Используйте инструмент миграции для:

    - **Оцените приложения онлайн:** Вы можете оценить приложения с помощью общедоступного URL-адреса в Интернете, используя помощник по миграции службы приложений Azure.
    - **.NET/PHP**: Для внутренних приложений .NET и PHP вы можете скачать и запустить помощник по миграции.
