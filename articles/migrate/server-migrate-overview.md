# <a name="select-a-vmware-migration-option"></a>Выберите опцию миграции VMware

Вы можете перенести виртуальные машины VMware в Azure с помощью средства "Миграция сервера" службы "Миграция Azure". Это средство предлагает несколько вариантов миграции виртуальных машин VMware:

- Миграция с использованием безагентной репликации. Миграция виртуальных машин без необходимости установки каких-либо компонентов.
- Миграция с агентом для репликации. Установите агент на виртуальную машины для репликации.




## <a name="compare-migration-methods"></a>Сравнение методов миграции

Используйте эти выбранные сравнения, чтобы помочь вам решить, какой метод использовать. Вы также можете просмотреть полные требования к поддержке для миграции [без агента](migrate-support-matrix-vmware-migration.md#agentless-vmware-servers) и [агента.](migrate-support-matrix-vmware-migration.md#agent-based-vmware-servers)

**Параметр** | **Безагентное** | **Агентская**
--- | --- | ---
**Разрешения Azure** | Вам нужны разрешения для создания проекта Azure Migrate и регистрации приложений Azure AD, созданных при развертывании прибора Azure Migrate. | Вам нужны разрешения на авторизне в подписке Azure. 
**Одновременное репликация** | Максимум 100 вм - с 100 вм- могут быть одновременно реплицированы с сервера vCenter.<br/> Если у вас более 50 всм для миграции, создайте несколько партий VMs.<br/> Репликация больше в одно время будет влиять на производительность. | Н/Д
**Развертывание приборов** | [Прибор Azure Migrate](migrate-appliance.md) развернут на месте. | [Прибор репликации мигрирующие Azure](migrate-replication-appliance.md) размещается на месте.
**Совместимость с восстановлением сайта** | Совместимость. | Вы не можете воспроизвести с помощью миграции серверов Azure Migrate, если вы настроили репликацию для машины с помощью восстановления сайта.
**Целевой диск** | Управляемые диски | Управляемые диски
**Ограничения диска** | Диск ОС: 2 ТБ<br/><br/> Диск данных: 4 ТБ<br/><br/> Максимальный диск: 60 | Диск ОС: 2 ТБ<br/><br/> Диск данных: 8 ТБ<br/><br/> Максимальный диск: 63
**Проходные диски** | Не поддерживается | Поддерживается

## <a name="deployment-steps-comparison"></a>Сравнение шагов развертывания

После рассмотрения ограничений понимание шагов, связанных с развертыванием каждого решения, может помочь вам решить, какой вариант выбрать.

**Оценка** | Оценка серверов перед миграцией.  Оценка не является обязательной. Мы предлагаем вам оценить машины, прежде чем их мигрировать, но вам не нужно. <br/><br/> Для оценки Azure Migrate устанавливает легкий прибор для обнаружения и оценки Вымов. | Если после оценки вы запустите миграцию без агента, для нерабочей миграции используется тот же прибор Azure Migrate, созданный для оценки.  |  Если после оценки вы запустите миграцию на основе агента, прибор, настроенный для оценки, не используется во время миграции без агента. Вы можете оставить прибор на месте, или удалить его, если вы не хотите делать дальнейшие открытия и оценки.
