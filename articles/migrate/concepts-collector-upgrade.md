# <a name="collector-appliance-updates"></a>Обновления устройства сборщика

В этой статье приведены сведения об обновлении модуля сборщика в службе [Миграция Azure](migrate-overview.md).

Сборщик Миграции Azure — это простой модуль, который можно использовать для обнаружения локального окружения vCenter для оценки перед миграцией в Azure. [Узнайте больше](concepts-collector.md).

## <a name="how-to-upgrade-the-appliance"></a>Обновление устройства

Сборщик можно обновить до последней версии, не скачивая OVA-файл повторно.

1. Закройте все окна браузера и любые откройте файлы и папки на устройстве.
2. Скачайте последнюю версию пакета обновления из списка обновлений, упоминаемые ниже в этой статье.
3. Чтобы обеспечить безопасность загруженного пакета, откройте командную строку администратора и выполните следующую команду для создания хэша ZIP-файл. Созданный хэш должен совпадать с хэшем, предусмотренным для определенной версии:

    ```C:\>CertUtil -HashFile <file_location> [Hashing Algorithm]```

    Пример: **C:\>CertUtil -HashFile C:\AzureMigrate\CollectorUpdate_release_1.0.9.14.zip SHA256)**
4. Скопируйте ZIP-файл на виртуальную машину с модулем сборщика.
5. Щелкните ZIP-файл правой кнопкой мыши и выберите **Извлечь все**.
6. Щелкните правой кнопкой мыши **Setup.ps1** > **Выполнить с помощью PowerShell** и следуйте инструкциям по установке.
