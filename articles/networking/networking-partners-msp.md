# <a name="about-azure-networking-msp-partners"></a>О партнерах MSP сети Azure

Сетевые службы включает Партнерская программа Azure Networking управляемые службы поставщика (MSP) к технологии MSP, телекоммуникационными компаниями и системных интеграторов (SIs), предлагать облачных и гибридных сетевых служб по центру Azure портфель сетевых продуктах и службы.

Azure MSP сети представляют собой специализированный набор поставщиков управляемых служб, предназначенным для корпоративных бизнес-требований сети и возможных трудностей во всех аспектах облака и гибридные сети. Управляемые сетевые службы включают один или несколько из следующих служб: сетевой архитектуры, планированию, развертыванию, операции, обслуживания и оптимизации.

![MSP][0]

Клиенты могут приобрести предложения служб под управлением MSP в Azure Marketplace, или непосредственно из управляемых услуг. Поставщикам управляемых служб можно развернуть и управлять сетевыми ресурсами Azure в подписках и группах ресурсов клиентов с помощью свету Azure.
