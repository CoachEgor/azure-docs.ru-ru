# <a name="about-azure-networking-msp-partners"></a>О партнерах Azure Networking MSP

[Партнерская программа Azure Networking Managed Service Provider (MSP)](https://azure.microsoft.com/blog/enhancing-the-customer-experience-with-the-azure-networking-msp-partner-program/) позволяет ориентированным на сети компаниям MSP, Telcos и Systems Integrators (SI) предлагать облачные и гибридные сетевые услуги, ориентированные на портфель сетевых продуктов и услуг Azure.

MSP Azure Networking MSP – это специализированный набор управляемых поставщиков услуг, которые удовлетворяют потребности и проблемы корпоративных облачных сетей во всех аспектах облачных и гибридных сетей. Управляемые сетевые службы включают один или несколько следующих служб: архитектура сети, планирование, развертывание, операции, техническое обслуживание и оптимизация.

![Msp][0]

Клиенты могут приобрести предложения услуг, управляемых MSP, в Azure Marketplace или непосредственно в MSP. MSPs могут развертывать и управлять сетевыми ресурсами Azure в подписях клиентов и группах ресурсов с помощью [Azure Lighthouse.](https://azure.microsoft.com/services/azure-lighthouse/)
