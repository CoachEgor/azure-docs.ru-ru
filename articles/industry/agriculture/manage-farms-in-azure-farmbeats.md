# <a name="manage-farms"></a>Управление фермами

Управлять фермами можно в Azure FarmBeats. В этой статье содержится информация о том, как создавать фермы, устанавливать устройства, датчики и беспилотные летательные аппараты, которые помогают управлять фермами.

## <a name="create-farms"></a>Создание ферм

Выполните указанные ниже действия.

1. Войти в фарм-ускоритель, отображает страницу **Фермы.**
    Страница **Фермы** отображает список ферм в случае, если они уже созданы в подписке.

    Вот образец изображения:

    ![Проект FarmBeats](./media/create-farms-in-azure-farmbeats/create-farm-main-page-1.png)


2. Выберите **Создать ферму** и предоставить **имя,** **культур и** **адрес**.
3. В **define Farm Boundary**(обязательное поле) выберите либо **отметку на карте,** либо **код ГеоJSON.**

Вот два способа определения границы фермы:

1. **Отметьте на карте**: Используйте инструмент управления картой, чтобы нарисовать и отметить границу фермы. Чтобы обозначить ![границы,](./media/create-farms-in-azure-farmbeats/pencil-icon-1.png) Project Farm бьется и помечает точные границы.

    ![Проект FarmBeats](./media/create-farms-in-azure-farmbeats/create-farm-mark-on-map-1.png)

2. **Вставьте код GeoJson**: GeoJSON является форматом кодирования географических структур данных с использованием обозначения объектов JavaScript (JSON). Эта опция отображает текстовый ящик, в котором можно ввести строку GeoJSON для обозначения границ фермы. Вы также можете создать код GeoJSON из GeoJSON.io.
Используйте наборы инструментов, чтобы помочь заполнить информацию.

    ![Проект FarmBeats](./media/create-farms-in-azure-farmbeats/create-new-farm-1.png)

3.  Выберите **Отправить** для создания фермы. Новая ферма создается и отображается на странице **Фермы.**

## <a name="view-farm"></a>Посмотреть ферму

Страница списка фермы отображает список созданных ферм. Выберите ферму для просмотра списка:

 - **Количество устройств** — отображает число и статус устройств, развернутых в ферме.
 - **Карта** – карта фермы с устройствами, развернутыми на ферме.
 - **Телеметрия** - отображает телеметрию от датчиков, развернутых на ферме.
 - **Последние точные карты** - отображает последнюю карту спутниковых индексов (EVI, NDWI), Почвенную влагатепловую карту и карту размещения датчиков.

## <a name="edit-farm"></a>Эдрит фермы

Страница **Фермы** отображает список созданных ферм.

1.  Выберите ферму для просмотра и отобрагив его.
2.  Выберите **Edit Farm,** чтобы отсеять информацию о ферме. В окне сведения о ферме можно отобрадить **имена,** **культуры,** **адрес**и определить поля **границы** **фермы.**

    ![Проект FarmBeats](./media/create-farms-in-azure-farmbeats/edit-farm-1.png)

3. Выберите **Отправить,** чтобы сохранить детали отредактированы.

## <a name="delete-farm"></a>Удаление фермы

Страница **Фермы** отображает список созданных ферм. Используйте следующие действия, чтобы удалить ферму:

1.  Выберите ферму из списка, чтобы удалить сведения о ферме.
2.  Выберите **Ферму Удаления,** чтобы удалить ферму.

    ![Проект FarmBeats](./media/create-farms-in-azure-farmbeats/delete-farm-1.png)

    > [!NOTE]
    > При удалении фермы устройства и карты, связанные с фермой, не удаляются. Любая информация о ферме, связанная с устройством и картами, не будет иметь значения. Вы можете продолжить просмотр устройств, телеметрии и карт из службы FarmBeats.
