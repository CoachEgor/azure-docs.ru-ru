# <a name="integration-patterns"></a>Шаблоны интеграции

Azure FarmBeats — это бизнес-предложение, доступное в Azure Marketplace. FarmBeats позволяет агрегировать наборы данных о сельском хозяйстве между поставщиками и создавать действенные идеи путем создания моделей искусственного интеллекта (ИИ) или машинного обучения (ML) путем слива данных.

![Проект FarmBeats](./media/architecture-for-farmbeats/farmbeats-architecture-1.png)

В следующих разделах описывается шаблон интеграции для Azure FarmBeats.

## <a name="why-integrate-with-azure-farmbeats"></a>Зачем интегрироваться с Azure FarmBeats?

Этот раздел ориентирован на партнеров, которые хотят интегрировать свои системы данных (такие как датчики, беспилотные летательные аппараты, метеостанции) в Azure FarmBeats.

Azure FarmBeats является расширяемым предложением, которое позволяет сельскохозяйственным предприятиям добавлять свои различные исторические и рабочие наборы сельскохозяйственных данных в единую платформу. Azure FarmBeats помогает сельскохозяйственному бизнесу нормализовать, контекстуализировать и агрегировать свои данные в контексте фермы.

Став партнером по обработке данных в Azure FarmBeats, вы можете открыть свои системы для более широкого внедрения и охватить большее число клиентов с помощью ваших предложений данных. Azure FarmBeats обеспечивает расширяемый слой API под названием Datahub, который помогает вам познавать данные с ваших устройств систематически и в стандартизированную схему.

Как только данные будут доступны в экземпляре Azure FarmBeats ваших клиентов, ваши клиенты смогут создавать более богатую аналитику и инструменты поверх ваших данных.
