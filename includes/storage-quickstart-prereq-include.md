Для доступа к службе хранилища Azure требуется подписка Azure. Если у вас еще нет подписки, [создайте бесплатную учетную запись Azure](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), прежде чем начинать работу.

Доступ к хранилищу Azure осуществляется с помощью учетной записи хранения. Для работы с этим руководством создайте учетную запись хранения с помощью [портала Azure](https://portal.azure.com/), Azure PowerShell или Azure CLI. Инструкции по созданию учетной записи хранения см. в статье [Создайте учетную запись хранения](../articles/storage/common/storage-quickstart-create-account.md).
