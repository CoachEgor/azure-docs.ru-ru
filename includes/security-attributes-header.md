Управление безопасностью — это качество или функция службы Azure, которая предоставляет службе возможность предотвращать, обнаруживать и отвечать на уязвимости системы безопасности.

Для каждого элемента управления используется значение "Да" или "нет", чтобы указать, используется ли в данный момент для службы, "н/д" для элемента управления, неприменимого к службе. Также можно указать примечание или ссылки на дополнительные сведения об атрибуте.
