Общая папка данных Azure позволяет организациям просто и безопасно обмениваться данными с клиентами и партнерами.

| **Ресурс** | **Ограничение** |
| --- | --- |
| Максимальное число общих ресурсов данных на подписку Azure |50 |
| Максимальное число отправленных общих ресурсов на общий ресурс данных |100 |
| Максимальное число полученных общих ресурсов на общий ресурс данных |100 |
| Максимальное число приглашений на отправленный общий ресурс |100 |
| Максимальное число общих подписок на отправленный общий ресурс |100 |
| Максимальное количество наборов данных на общую папку |100 |
| Максимальное число расписаний моментальных снимков на общую папку |1 |
