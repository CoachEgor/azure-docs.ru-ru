Azure Data Share позволяет организациям просто и надежно обмениваться данными со своими клиентами и партнерами.

| **Ресурсов** | **Предел** |
| --- | --- |
| Максимальное количество ресурсов обмена данными в подписке Azure |50 |
| Максимальное количество отправленных акций на ресурс ехавого использования данных |100 |
| Максимальное количество полученных акций на ресурс ехающих данных |100 |
| Максимальное количество приглашений на одну отправленную акцию |100 |
| Максимальное количество подписок на отправку на отправленную акцию |100 |
| Максимальное количество наборов данных на акцию |100 |
| Максимальное количество графиков моментальных снимков на акцию |1 |
