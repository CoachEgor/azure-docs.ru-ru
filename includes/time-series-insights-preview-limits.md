### <a name="general-availability-and-preview-comparison"></a>Сравнение общей доступности и предварительной версии

В следующей таблице приведены некоторые основные различия между общедоступной версией службы "аналитика временных рядов Azure" и экземплярами предварительной версии.

| | Общая доступность | Preview (Предварительный просмотр) |
| --- | --- | ---|
| Ролевое граждан первого класса | Ориентированные на события | С использованием временных рядов |
| Семантическая причина | Нижний уровень (эталонные данные) | Высокий уровень (модели) |
| Контекстуальность данных | Не на уровне устройства | Устройство и уровень, не являющийся устройством |
| Хранилище логики вычислений | Нет | Хранимые в переменных типа часть модели |
| Управление хранилищем и доступом | Нет | Включено через модель |
| Агрегаты и выборка | Нет | Взвешенное и взвешенное время события |
| Реконструкция сигнала | Нет | Интерполяции |
| Рабочий ряд производных временных рядов | Нет | Да, объединения и объединения |
| Гибкость языка | Не допускающая композицию | Композици |
| Язык выражений | Строка предиката | Выражения временных рядов (строки предиката, значения, выражения и функции) |

### <a name="property-limits"></a>Ограничения свойств

Ограничения свойства "аналитика временных рядов" увеличились до 1 000 с максимальной закрепления 800 в общедоступной версии. Предоставленные свойства событий имеют соответствующие столбцы JSON, CSV и диаграммы, которые можно просмотреть в [обозревателе предварительного просмотра аналитики временных рядов](https://docs.microsoft.com/azure/time-series-insights/time-series-insights-update-quickstart).

| SKU | Максимальное число свойств |
| --- | --- |
| Предварительный просмотр PAYG | 1 000 свойства (столбцы) |
| ОБЩЕДОСТУПНАЯ ВЕРСИЯ S1 | 600 свойства (столбцы) |
| ОБЩЕДОСТУПНАЯ ВЕРСИЯ S2 | 800 свойства (столбцы) |
