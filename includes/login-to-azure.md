## <a name="sign-in-to-azure"></a>Вход в Azure

Используйте Azure CLIдля создания ресурсов, необходимых для размещения вашего приложения в Azure. Если команды Azure CLI выполняются в оболочке Cloud Shell, это значит, что вы уже выполнили вход. Для выполнения команд Azure CLI в локальной среде войдите в подписку Azure с помощью команды [az login](/cli/azure/) и следуйте инструкциям на экране.

```azurecli
az login
```
