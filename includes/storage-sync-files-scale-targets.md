| Число групп синхронизации на службу синхронизации хранилища | 100 групп синхронизации | ДА |

| Облачных конечных точек на группу синхронизации | 1 облачная конечная точка | ДА |
| Конечных точек сервера на группу синхронизации | 50 конечных точек сервера | Нет |

| Размер файла | 100 ГБ | Нет |

> [!Note]  
> Конечная точка Синхронизация файлов Azure может масштабироваться до размера файлового ресурса Azure. Если достигнут максимальный размер файлового ресурса Azure, синхронизация не сможет работать.
