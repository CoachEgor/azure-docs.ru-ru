1. Войдите на [портал Azure](https://portal.azure.com/) с помощью учетных данных для используемой подписки Azure. 

1. На портале Azure вверху слева щелкните **+ Создать ресурс**.

      ![Создать новый ресурс](media/aml-create-in-portal/portal-create-resource.png)

1. С помощью строки поиска найдите ресурс **Рабочая область Службы машинного обучения**.

1. Выберите ресурс **Рабочая область Службы машинного обучения**.

1. В области **Рабочая область Службы машинного обучения** выберите **Создать**.

  

1. Настройте новую рабочую область, указав ее имя, подписку, группу ресурсов и расположение.

   Поле|ОПИСАНИЕ 
   ---|---
   имя рабочей области. |Введите уникальное имя для идентификации рабочей области. В этом примере мы используем **docs-ws**. Имена должны быть уникальными в группе ресурсов. Используйте имя, которое позволит легко запомнить рабочую область и отличить ее от областей, созданных другими пользователями.  
   Subscription |Выберите подписку Azure, которую нужно использовать.
   Resource group | Используйте группу ресурсов, которая уже есть в подписке, или введите имя, чтобы создать группу ресурсов. Группа ресурсов содержит связанные ресурсы для решения Azure. В этом примере мы используем **docs-aml**. 
   Location | Выберите ближайшее к пользователям и ресурсам данных расположение, чтобы создать рабочую область.

1. После окончания настройки рабочей области нажмите **Создать**. 

   > [!Warning] 
   > Создание рабочей области в облаке может занять несколько минут.

   По завершении процесса появится сообщение об успешном развертывании. 
 
 1. Чтобы просмотреть новую рабочую область, выберите **Перейти к ресурсу**.
