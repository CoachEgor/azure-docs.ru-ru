1. Установите удаленное подключение к рабочему столу к машине, работая на сервере процесса. 
2. Запустите cspsconfigtool.exe, чтобы запустить инструмент конфигурации процесса восстановления сервера процесса восстановления сайта Azure.
    - Инструмент запускается автоматически при первом входе в процесссервер.
    - Если он не открывается автоматически, нажмите на его ярлык на рабочем столе.

3. В **configuration server F'DN или IP**укажите имя или IP-адрес сервера конфигурации, с помощью которого можно зарегистрировать сервер процесса.
4. В **порте сервера конфигурации**убедитесь, что указано 443. Это порт, на котором сервер конфигурации слушает запросы.
5. В **Connection Passphrase**укажите фразу, указанную при настройке сервера конфигурации. Чтобы найти фразу:
    -  На сервере конфигурации перейдите в папку установки установки сайта *:\**
    ```
    cd %ProgramData%\ASR\home\svsystems\bin
    ```
    - Выполнить ниже команды распечатать текущую фразу:
    ```
    genpassphrase.exe -n
    ```

6. В **порту передачи данных**оставьте значение по умолчанию, если вы не указали пользовательский порт.

7. Нажмите **Сохранить** настройки и зарегистрировать сервер процесса.
