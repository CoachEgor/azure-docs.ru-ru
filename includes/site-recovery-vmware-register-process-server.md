1. Установите подключение к удаленному рабочему столу на компьютер, на котором работает сервер обработки. 
2. Запустите cspsconfigtool.exe, чтобы запустить средство настройки сервера обработки Azure Site Recovery.
    - Средство запускается автоматически при первом входе на сервер обработки.
    - Если он не открывается автоматически, щелкните его ярлык на рабочем столе.

3. В поле **полное доменное имя сервера конфигурации или IP-** адрес укажите имя или IP-адреса сервера конфигурации, с которым будет зарегистрирован сервер обработки.
4. В окне **порт сервера конфигурации**убедитесь, что указан 443. Это порт, на котором сервер конфигурации прослушивает запросы.
5. В поле **парольная фраза для подключения**укажите парольную фразу, которая была указана при настройке сервера конфигурации. Чтобы найти парольную фразу, сделайте следующее:
    -  На сервере конфигурации перейдите в папку установки Site Recovery **\хоме\свссистемс\бин \* *:
    ```
    cd %ProgramData%\ASR\home\svsystems\bin
    ```
    - Выполните следующую команду, чтобы напечатать текущую парольную фразу:
    ```
    genpassphrase.exe -n
    ```

6. В поле **порт передача данных**оставьте значение по умолчанию, если не указан настраиваемый порт.

7. Нажмите кнопку **сохранить** сохранить параметры и зарегистрируйте сервер обработки.
