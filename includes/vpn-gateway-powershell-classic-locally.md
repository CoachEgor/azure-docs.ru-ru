При работе с классической моделью развертывания нельзя использовать Azure Cloud Shell. Вместо этого необходимо установить последнюю версию командлетов PowerShell для управления службами Azure (SM) локально на компьютере. Эти командлеты отличаются от командлетов AzureRM и AZ. Сведения об установке командлетов SM см. в разделе [Install Service Management командлеты](/powershell/azure/servicemanagement/install-azure-ps). Дополнительные сведения о Azure PowerShell в целом см. в [документации по Azure PowerShell](/powershell/azure/).
