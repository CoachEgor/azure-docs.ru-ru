Следующие ограничения относятся к системным разделам службы "Сетка событий Azure" и пользовательским темам, а *не* к доменам событий.

| Resource | Ограничение |
| --- | --- |
| Количество пользовательских разделов на подписку Azure | 100 |
| Количество подписок на события на раздел | 500 |
| Скорость публикации пользовательского раздела (входящий трафик) | 5000 событий на раздел в секунду |
| Публиковать запросы | 250 в секунду |
| Размер события | Поддержка 64 КБ в общедоступной версии. Поддержка 1 МБ сейчас доступна в предварительной версии. |

Следующие ограничения применяются только к доменам событий.

| Resource | Ограничение |
| --- | --- |
| Темы на домен событий | 1000 (во время действия общедоступной предварительной версии) |
| Подписки на события для каждого раздела в домене | 50 (во время действия общедоступной предварительной версии) |
| Подписки на события области домена | 50 (во время действия общедоступной предварительной версии) |
| Частота публикации для домена событий (входящий трафик) | 5000 событий в секунду (во время действия общедоступной предварительной версии) |
| Публиковать запросы | 250 в секунду |
