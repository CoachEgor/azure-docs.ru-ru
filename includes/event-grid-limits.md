Следующие ограничения применяются к темам системы Azure Event Grid и пользовательским темам, а *не* к доменам событий.

| Ресурс | Ограничение |
| --- | --- |
| Количество пользовательских разделов на подписку Azure | 100 |
| Количество подписок на события на раздел | 500 |
| Скорость публикации пользовательского раздела (входящий трафик) | 5000 событий на раздел в секунду |
| Публикация запросов | 250 в секунду |
| Размер события | 1 МБ (заряжено как несколько событий 64-KB) |

Следующие ограничения применяются только к доменам событий.

| Ресурс | Ограничение |
| --- | --- |
| Темы на домен события | 100 000 |
| Подписки на события по теме в домене | 500 |
| Подписка на события области домена | 50 |
| Публикация тарифа для домена события (вход) | 5000 событий в секунду |
| Публикация запросов | 250 в секунду |
| Домены событий по подписке Azure | 100 |
