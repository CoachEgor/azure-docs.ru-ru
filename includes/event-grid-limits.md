Следующие ограничения относятся к системным разделам службы "Сетка событий Azure" и пользовательским темам, а *не* к доменам событий.

| Ресурс | Ограничение |
| --- | --- |
| Количество пользовательских разделов на подписку Azure | 100 |
| Количество подписок на события на раздел | 500 |
| Скорость публикации пользовательского раздела (входящий трафик) | 5000 событий на раздел в секунду |
| Публиковать запросы | 250 в секунду |
| Размер события | 1 МБ. При выполнении операций за 64 КБ увеличивается. Таким образом, события свыше 64 КБ приводят к оплате операций, как будто они были несколькими событиями. Например, событие, 130 КБ, вызовет операции, как будто это было 3 отдельными событиями.  |

Следующие ограничения применяются только к доменам событий.

| Ресурс | Ограничение |
| --- | --- |
| Темы на домен событий | 100 000 |
| Подписки на события для каждого раздела в домене | 500 |
| Подписки на события области домена | 50 |
| Частота публикации для домена событий (входящий трафик) | 5 000 событий в секунду |
| Публиковать запросы | 250 в секунду |
| Доменов событий на подписку Azure | 100 |
