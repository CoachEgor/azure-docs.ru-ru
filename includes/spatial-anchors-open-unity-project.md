В Unity откройте проект в папке `Unity`. Unity может отправить запрос о другой версии Unity между проектной версией и той версией, которая установлена на вашем компьютере. Это предупреждение допустимо, пока версия Unity Editor новее, чем та, с помощью которой создан проект. В этом случае просто щелкните **Продолжить**. Если ваша версия Unity Editor старше, чем нужна проекту, щелкните **Quit** (Выйти) и обновите Unity Editor.
