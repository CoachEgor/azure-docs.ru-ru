>[!IMPORTANT]
>Созданные ресурсы можно использовать в качестве необходимых компонентов для других руководств и статей с практическими рекомендациями по Службе машинного обучения Azure.

### <a name="delete-everything"></a>Удаление всех ресурсов

Если вы не планируете использовать созданные ресурсы, удалите всю группу ресурсов, чтобы с вас не взималась плата.

1. На портале Azure слева выберите **Группы ресурсов**.
 
   ![Удаление группы ресурсов на портале Azure](./media/aml-ui-cleanup/delete-resources.png)

1. В списке выберите созданную группу ресурсов.

1. В правой части окна нажмите кнопку с многоточием ( **...** ).

1. Выберите **Удалить группу ресурсов**.

При удалении группы ресурсов будут также удалены все ресурсы, созданные в визуальном интерфейсе.  

### <a name="delete-only-the-compute-target"></a>Удаление только целевого объекта вычислений

Созданный вами целевой объект вычислений *автоматически масштабируется* до нуля узлов, когда он не используется. Это позволяет свести к минимуму расходы. Чтобы удалить целевой объект вычислений, сделайте следующее:

1. Откройте рабочую область на [портале Azure](https://portal.azure.com).

    ![Удаление целевого объекта вычислений](./media/aml-ui-cleanup/delete-compute-target.png)

1. В разделе **Вычисления** в рабочей области выберите ресурс.

1. Нажмите кнопку **Удалить**.

### <a name="delete-individual-assets"></a>Удаление отдельных ресурсов

В визуальном интерфейсе, в котором вы создали эксперимент, удалите отдельные ресурсы, выбрав их и нажав кнопку **Удалить**. Можно отменить регистрацию наборов данных в рабочей области. Для этого выберите каждый набор данных и щелкните **Отменить регистрацию**.

![Удаление ресурсов](./media/aml-ui-cleanup/delete-asset.png)
