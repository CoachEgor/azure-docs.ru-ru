- **Подробности заказа**. При создании заказа, почтовый адрес, адрес электронной почты и контактные данные пользователя хранятся на портале Azure. Сохраненная информация включает следующее.
  - имя контактного лица;
  - номер телефона.
  - Адрес электронной почты
  - почтовый адрес;
  - City
  - Почтовый код
  - Состояние
  - Провинция/страны
  - номер отслеживания доставки.

    Сведения о заказе шифруются и хранятся в службе. Информация хранится в службе, пока не будет явно удален ресурс или порядок. Удаление ресурса и соответствующий порядок блокируется с момента устройство поставляется в том случае, пока устройство вернется в Майкрософт.

- **Адрес доставки**. После размещения заказа, служба Data Box предоставляет адрес доставки для независимых производителей операторов как ИБП.

- **Совместное использование пользователей**. На устройстве можно также доступ к данных, расположенных на общих ресурсах. Можно просмотреть список пользователей, которым разрешен доступ к данным в общем ресурсе. При удалении общих ресурсов, этот список также удаляется.
