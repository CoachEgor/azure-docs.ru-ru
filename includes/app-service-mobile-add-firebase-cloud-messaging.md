1. [Добавить Firebase в свой android-проект](https://firebase.google.com/docs/android/setup)

2. В Android Studio выберите **структуру файлового** > **проекта.** Щелкните **Notifications** (Уведомления), выберите **Firebase Cloud Messaging** и нажмите кнопку **OK**.
