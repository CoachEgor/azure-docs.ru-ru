- Для доступа к службе Edge поле данных, вашей организации необходимо иметь подписку Enterprise Agreement (EA) или поставщика облачных решений (CSP). Дополнительные сведения см. в разделе [зарегистрироваться для получения подписки Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Так как эта служба управления размещается в Azure, она защищена функциями безопасности Azure. Дополнительные сведения о функциях безопасности, предоставляемые Azure, см. в статье [Центр управления безопасностью Microsoft Azure](https://azure.microsoft.com/support/trust-center/security/).
- Для операций управления пакета SDK, можно получить ключ шифрования для ресурса в **свойства устройства**. Ключ шифрования можно просмотреть только в том случае, если у вас есть разрешения для ресурса API Graph.
