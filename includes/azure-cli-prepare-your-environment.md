## <a name="prepare-your-environment"></a>Подготовка среды

1. Чтобы использовать команды Azure CLI, [установите Azure CLI локально](/cli/azure/install-azure-cli) или запустите [Azure Cloud Shell](/azure/cloud-shell/quickstart) с помощью среды Bash.

1. Если вы используете локальную установку, выполните вход с помощью команды Azure CLI [az login](/cli/azure/reference-index#az-login).  Выполните аутентификацию, следуя инструкциям в окне терминала.  Сведения о дополнительных возможностях входа см. в статье [Вход с помощью Azure CLI](/cli/azure/authenticate-azure-cli).
