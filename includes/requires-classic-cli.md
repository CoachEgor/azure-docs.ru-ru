> [!IMPORTANT]
> Для выполнения действий в этой статье необходим классический Azure CLI. Текущая версия Azure CLI не поддерживает функции, описанные в этой статье, поэтому требуется классический интерфейс командной строки. 
> 
> Классический интерфейс командной строки может устанавливаться параллельно с современный интерфейс командной строки Azure, но мы рекомендуем использовать Azure CLI для всех новых наборов символов и развертываний.
> Чтобы установить классический интерфейс командной строки, обратитесь к статье [Установка классического интерфейса командной строки Azure (Azure Classic CLI)](/cli/azure/install-classic-cli). 
> Чтобы установить последнюю версию интерфейса командной строки, см. в разделе [установите Azure CLI](/cli/azure/install-azure-cli).
