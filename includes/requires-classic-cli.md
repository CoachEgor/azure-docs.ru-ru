> [!IMPORTANT]
> Для выполнения действий в этой статье необходим классический Azure CLI. Текущая версия Azure CLI не поддерживает функции, описанные в этой статье, поэтому требуется классический интерфейс командной строки. 
> 
> Классический интерфейс командной строки можно установить параллельно с современным Azure CLI, но мы рекомендуем использовать Azure CLI для всех новых сценариев и развертываний с поддержкой данной функции. Чтобы установить классический интерфейс командной строки, обратитесь к статье [Установка классического интерфейса командной строки Azure (Azure Classic CLI)](https://docs.microsoft.com/cli/azure/install-classic-cli?view=azure-cli-latest). 
> Сведения об установке текущей версии интерфейса командной строки см. в статье [Установка Azure CLI](https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest).
