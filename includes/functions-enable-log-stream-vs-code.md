Чтобы включить потоковые журналы для приложения функции в Azure:

1. Выберите F1, чтобы открыть палитру команд, а затем выполнить и запустить функции команды **Azure: Запуск потоковых журналов.**

1. Выберите приложение функции в Azure, а затем выберите **«Да»,** чтобы включить журнал приложений для приложения функции.

1. Запуск функций в Azure. Обратите внимание, что данные журнала отображаются в окне вывода в Visual Studio Code.

1. Когда вы закончите, не забудьте запустить команду **Azure Functions: Прекратите потоковые журналы,** чтобы отключить журналы для приложения функции.
