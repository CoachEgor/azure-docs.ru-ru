### <a name="event-hubs-dedicated---quotas-and-limits"></a>Центры событий Выделены - квоты и лимиты
Предложение Event Hubs Dedicated выставляется по фиксированной месячной цене, с минимум 4 часами использования. Выделенный уровень предлагает все функции стандартного плана, но с емкостью корпоративного масштаба и ограничениями для клиентов с требовательными рабочими нагрузками. 

| Компонент | Ограничения |
| --- | ---|
| Пропускная способность |  20 CUS |
| Пространства имен | 50 за ТС |
| Центры событий |  1000 за пространство имен |
| Входящие события | Включено |
| Размер сообщения | 1 МБ |
| Секции | 2000 за Cu |
| Группы получателей | Нет предела на CU, 1000 на концентратор событий |
| Подключения через брокера | Включено 100 тыс. |
| Хранение сообщений | 90 дней, 10 ТБ включены в ТС |
| Сбор | Включено |
