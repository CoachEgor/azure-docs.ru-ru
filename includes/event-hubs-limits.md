### <a name="event-hubs-dedicated---quotas-and-limits"></a>Выделенный концентраторов событий — квоты и ограничения
Поддержка концентраторов событий уровня Dedicated оплачивается по фиксированному ежемесячному тарифу, с минимальным объемом 4 часа использования. Уровень Dedicated предоставляет все возможности плана Standard, но с емкостью уровня предприятия и ограничения для клиентов с ресурсоемкими рабочими нагрузками. 

| Компонент | Ограничения |
| --- | ---|
| Пропускная способность |  Более 20 единиц мощности |
| Пространства имен | 50 на единицу CU |
| Центры событий; |  1000 на пространство имен |
| Входящие события | Включено |
| Размер сообщения | 1 миллион байтов |
| Разделы | 2000 на единицу CU |
| Группы получателей | Отсутствие ограничений на единицу CU, 1000 на концентратор событий |
| Подключения через брокера | Включено 100 тыс. |
| Хранение сообщений | До 7 дней (90-дневный период хранения ожидается в ближайшее время), 10 ТБ включены на единицу CU |
| Регистрация | Включено |
