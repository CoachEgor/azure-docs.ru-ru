В следующей таблице описаны максимальные ограничения для кластеров Azure обозреватель данных.

| Ресурс | Ограничение |
| --- | --- |
| Кластеров на регион на подписку | 20 |
| Экземпляров на кластер | 1000 | 
| Число баз данных в кластере | 10 000 |

| Область действия | Операция | Ограничение |

| Кластер | чтение (например, получение кластера) | 500 за 5 минут |
