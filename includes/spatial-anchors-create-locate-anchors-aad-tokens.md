### <a name="azure-active-directory-authentication"></a>Аутентификация Azure Active Directory

Azure пространственных привязки также позволяет проверять подлинность с помощью маркеров Azure AD (Active Directory) для пользователей приложений. Например можно использовать токенов Azure AD для интеграции с Azure пространственных привязки. Если предприятия сохраняет пользователей в Azure AD, вы можете указать маркер пользователя Azure AD в пакете SDK Azure пространственных привязки. Это позволит вам осуществлять проверку подлинности непосредственно к службе Azure пространственных привязки для учетной записи, которая является частью того же клиента Azure AD.
