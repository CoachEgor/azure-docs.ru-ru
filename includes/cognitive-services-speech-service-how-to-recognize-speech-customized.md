## <a name="speech-recognition-with-a-customized-model"></a>Распознавание речи с помощью настраиваемой модели

[Пользовательская служба распознавания речи (CRIS)](https://www.cris.ai/) позволяет настроить подсистему Майкрософт для преобразования речи в текст для конкретного приложения.
В следующем фрагменте кода показано распознавание речи с микрофона с помощью модели CRIS. Перед выполнением кода укажите ключ подписки и свой идентификатор развертывания.
