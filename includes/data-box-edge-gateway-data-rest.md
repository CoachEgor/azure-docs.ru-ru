Для данных в состоянии покоя:

- Доступ к данным, хранящимся в акциях, ограничен.

    - Клиентам SMB, которые получают доступ к данным общего доступа, нужны учетные данные пользователей, связанные с долей. Эти учетные данные определяются при создании доли.
    - IP-адреса клиентов NFS, которые получают доступ к акции, должны быть добавлены при создании акции.
