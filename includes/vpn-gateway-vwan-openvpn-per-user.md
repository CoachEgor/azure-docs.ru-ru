### <a name="open-the-mfa-page"></a><a name="mfa"></a>Открыть страницу МИД

1. Войдите на портал Azure.
2. Перейдите в **Active Directory Azure - > все пользователи.**
3. Выберите **многофакторную аутентификацию,** чтобы открыть многофакторную страницу аутентификации.

   ![Вход](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="select-users"></a><a name="users"></a>Выберите пользователей

1. На **многофакторной странице аутентификации** выберите пользователя (ы), для которого вы хотите включить MFA.
2. Нажмите кнопку **Включить**.

   ![Выберите пункт](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)
