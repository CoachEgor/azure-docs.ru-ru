Файлы Premium используют уникальную учетную запись хранения с именем **филестораже**. Этот тип учетной записи предназначен для рабочих нагрузок с высокими операциями ввода-вывода в секунду, высокой пропускной способностью с постоянной задержкой. Хранилище файлов класса Premium масштабируется с учетом подготовленного размера общего ресурса.
