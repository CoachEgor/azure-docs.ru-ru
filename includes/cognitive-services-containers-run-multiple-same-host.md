### <a name="running-multiple-containers-on-the-same-host"></a>Запуск нескольких контейнеров на одном узле

Если планируется выполнить несколько контейнеров с помощью открытых портов, убедитесь, что для запуска каждого контейнера с разных предоставленным портом. Например, запускайте первый контейнер на порте 5000, а второй — на порте 5001.

Этот контейнер и в другой контейнер Cognitive Service, совместно работающих на узле или у вас есть несколько контейнеров же Cognitive Service контейнера под управлением.
