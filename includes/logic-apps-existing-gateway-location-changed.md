**Вопрос**. Почему изменяется расположение для существующего шлюза? <br/>
**Ответ**. Ресурсы шлюза, созданные до 3 мая 2017 г. были перемещены в исходные расположения клиента Azure AD для работы или учебная учетная запись, созданы эти шлюзы. Однако эти изменения не влияют на выполняющиеся сейчас приложения логики, которые должны работать в обычном режиме. После выпуска общедоступной версии шлюза в мае расположения ресурсов шлюза могут отличаться от расположений приложений логики.
