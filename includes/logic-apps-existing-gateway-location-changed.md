**Вопрос.** Почему расположение для существующего шлюза изменилось? <br/>
**Ответ.** Ресурсы шлюза, созданные до 3 мая 2017г., были перемещены в исходные расположения клиента Azure AD для рабочей или учебной учетной записи, с помощью которой были созданы эти шлюзы. Однако эти изменения не влияют на выполняющиеся сейчас приложения логики, которые должны работать в обычном режиме. После выпуска общедоступной версии шлюза в мае расположения ресурсов шлюза могут отличаться от расположений приложений логики.
