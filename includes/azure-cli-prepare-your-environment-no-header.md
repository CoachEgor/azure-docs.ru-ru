- Подготовьте среду к работе с Azure CLI:

  - Запустите [Azure Cloud Shell](/azure/cloud-shell/quickstart) с помощью среды Bash. Если вы хотите запустить Azure CLI локально, установите Azure CLI в [локальной среде](/cli/azure/install-azure-cli).

  - Если вы используете локальную установку, выполните вход с помощью команды Azure CLI [az login](/cli/azure/reference-index#az-login).  Выполните аутентификацию, следуя инструкциям в окне терминала.  Сведения о дополнительных возможностях входа см. в статье [Вход с помощью Azure CLI](/cli/azure/authenticate-azure-cli).
  - При появлении запроса установите расширения Azure CLI при первом использовании.  Дополнительные сведения см. в статье [Использование расширений с Azure CLI](/cli/azure/azure-cli-extensions-overview).
