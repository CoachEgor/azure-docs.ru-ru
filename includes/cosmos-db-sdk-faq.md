**1. Каким образом клиенты будут уведомлены о выводе пакета SDK из эксплуатации?**

Корпорация Майкрософт предоставит 12-месячное предварительное уведомление о завершении поддержки пакета SDK для снятия с учета, чтобы упростить переход на поддерживаемый пакет SDK. Кроме того, клиенты будут уведомлены по различным каналам связи — портал Azure, обновлениям Azure и прямой связи с назначенными администраторами служб.

**2. могут ли клиенты создавать приложения с помощью Azure Cosmos DB пакета SDK в течение 12 месяцев?** 

Да, клиенты будут иметь полный доступ к созданию, развертыванию и изменению приложений с помощью устаревшего Azure Cosmos DB пакета SDK в течение 12-месячного периода уведомления. В течение 12-месячного периода уведомления клиентам рекомендуется перейти на более новую поддерживаемую версию Azure Cosmos DB SDK. 

**3. что произойдет с приложениями, использующими неподдерживаемый пакет SDK Azure Cosmos DB?** 

После даты прекращения использования Azure Cosmos DB больше не будет вносить исправления в ошибки, добавлять новые функции и предоставлять поддержку для устаревших версий пакета SDK. Если вы предпочитаете не выполнять обновление, запросы, отправленные из отмененных версий пакета SDK, продолжат обслуживаться службой Azure Cosmos DB. 

**4. какие версии SDK будут иметь новейшие функции и обновления?**

Новые функции и обновления будут добавлены только к последней дополнительной версии последней поддерживаемой версии пакета SDK. Рекомендуется всегда использовать последнюю версию, чтобы воспользоваться новыми функциями, улучшениями производительности и исправлениями ошибок. Если вы используете старую, не отмененную версию пакета SDK, запросы к Azure Cosmos DB будут по-прежнему работать, но у вас не будет доступа к новым возможностям.  

**5. что делать, если я не могу обновить приложение до даты отсечения?**

Рекомендуется как можно раньше выполнить обновление до последней версии SDK. После того как пакет SDK помечается для выхода из эксплуатации, вы получите 12 месяцев на обновление приложения. Если не удается выполнить обновление по дате выхода из эксплуатации, запросы, отправленные из отмененных версий пакета SDK, будут по прежнему обслуживаться Azure Cosmos DB, поэтому работающие приложения будут продолжать работать. Однако Azure Cosmos DB больше не будет вносить исправления в ошибки, добавлять новые функции и предоставлять поддержку для устаревших версий пакета SDK. 

Если у вас есть план поддержки и требуется техническая поддержка, [свяжитесь с нами](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview) , указав запрос в службу поддержки.
