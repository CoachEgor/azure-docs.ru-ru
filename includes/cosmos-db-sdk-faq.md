**Как я буду уведомлен о выводе пакета SDK?**

Корпорация Майкрософт предоставит ежемесячное уведомление за 12 месяцев до окончания поддержки пакета SDK для снятия с учета, чтобы упростить переход на поддерживаемый пакет SDK. Мы будем уведомлять вас по различным каналам связи: портал Azure, обновлениям Azure и прямой связи с назначенными администраторами служб.

**Можно ли создавать приложения с помощью Azure Cosmos DB пакета SDK в течение 12-месячного периода?** 

Да, вы сможете создавать, развертывать и изменять приложения с помощью пакета SDK Azure Cosmos DB для использования в течение 12-месячного периода уведомления. Рекомендуется перейти на более новую поддерживаемую версию Azure Cosmos DB пакета SDK в течение 12-месячного периода уведомления, если это необходимо. 

**Что произойдет с приложениями, использующими неподдерживаемый пакет SDK Azure Cosmos DB, после даты выбытия?** 

После даты прекращения использования Azure Cosmos DB больше не будет вносить исправления в ошибки, добавлять новые функции или предоставлять поддержку для устаревших версий пакета SDK. Если вы предпочитаете не выполнять обновление, запросы, отправленные из отмененных версий пакета SDK, продолжат обслуживаться службой Azure Cosmos DB. 

**Какие версии SDK будут иметь новейшие функции и обновления?**

Новые функции и обновления будут добавлены только к последней дополнительной версии последней поддерживаемой версии пакета SDK. Рекомендуется всегда использовать последнюю версию, чтобы воспользоваться новыми функциями, улучшениями производительности и исправлениями ошибок. Если вы используете старую версию пакета SDK без использования, ваши запросы к Azure Cosmos DB будут по-прежнему работать, но у вас не будет доступа к новым возможностям.  

**Что делать, если я не могу обновить приложение до даты прекращения?**

Рекомендуется как можно раньше выполнить обновление до последней версии SDK. После того как пакет SDK помечается для выхода из эксплуатации, вы получите 12 месяцев на обновление приложения. Если вы не можете обновить дату выбытия, запросы, отправленные из отмененных версий пакета SDK, будут обслуживаться Azure Cosmos DB, поэтому работающие приложения будут продолжать работать. Но Azure Cosmos DB больше не будет вносить исправления в ошибки, добавлять новые функции или предоставлять поддержку для устаревших версий пакета SDK. 

Если у вас есть план поддержки и требуется техническая поддержка, [свяжитесь с нами](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview) , выполнив запрос в службу поддержки.
