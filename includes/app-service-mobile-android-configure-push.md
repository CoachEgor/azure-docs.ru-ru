1. На [портале Azure](https://portal.azure.com/) последовательно выберите **Browse All (Просмотреть все)**  > **Службы приложений**. Выберите серверную часть для функции "Мобильные приложения". 
2. В разделе **Параметры** выберите **Push**. Затем выберите **Настройка служб push-уведомлений**.
2. Откройте **Google (GCM)** . Введите старый ключ сервера FCM, который вы получили в консоли Firebase, и нажмите кнопку **Сохранить**.

Теперь ваша служба настроена для работы с Firebase Cloud Messaging.
