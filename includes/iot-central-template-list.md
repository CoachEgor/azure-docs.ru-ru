**Шаблоны приложений**

| Шаблон                 | Имя        | Описание |
| ------------------------ | ----------- | ----------- |
| iotc-pnp-preview@1.0.0   | Custom application (Пользовательское приложение) | Создает пустое приложение, в которое необходимо добавить собственные шаблоны устройств и сами устройства. |
| iotc-default@1.0.0       | Пользовательское приложение (устаревшее) | Создает пустое устаревшее приложение для заполнения с помощью собственных шаблонов устройств и устройств.
| iotc-condition@1.0.0     | Аналитика в магазине — мониторинг условий | Создает приложение для подключения и мониторинга среды хранилища. |
| iotc-consumption@1.0.0   | Мониторинг потребления воды | Создает приложение для мониторинга потока воды и управления им. |
| iotc-distribution@1.0.0  | Центр цифрового распространения | Создает приложение, повышающее эффективность вывода данных в хранилище с помощью средств хранения ключей и действий. |
| iotc-inventory@1.0.0     | Управление интеллектуальными складскими запасами | Создает приложение для автоматизации приема, перемещения продуктов, циклического подсчета и отслеживания. |
| iotc-logistics@1.0.0     | Логистика подключенных | Создает приложение для отслеживания поставок в реальном времени по воздуху, воду и наземный с мониторингом расположения и условий. |
| iotc-meter@1.0.0         | Интеллектуальная аналитика измерений | Создает приложение для отслеживания энергопотребления, состояния сети и выявления тенденций для улучшения поддержки клиентов и управления интеллектуальными измерениями.  |
| iotc-mfc@1.0.0           | Микроцентр исполнения заказов | Создает приложение для цифрового подключения и управления полностью автоматизированным центром выполнения. |
| iotc-patient@1.0.0       | Непрерывное наблюдение за пациентами | Создает приложение, которое позволяет продлить здравоохранение, снизить количество допусков и управлять заболеваниях. |
| iotc-power@1.0.0         | Солнечное наблюдение за питанием | Создает приложение для отслеживания состояния солнечной панели и тенденций создания энергии. |
| iotc-quality@1.0.0       | Мониторинг качества воды | Создает приложение для цифрового отслеживания качества воды. |
| iotc-store@1.0.0         | Аналитика в магазине — извлечение | Создает приложение для отслеживания потока извлечения в хранилище и управления им. |
| iotc-waste@1.0.0         | Подключенное управление отходами | Создает приложение для отслеживания расходных ячеек и операторов полей диспетчеризации. |
