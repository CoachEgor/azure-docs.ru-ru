## <a name="set-up-authentication"></a>Настройка проверки подлинности

Доступ к службе, необходимо предоставить ключ учетной записи, маркер доступа или маркер проверки подлинности Azure Active Directory.

### <a name="account-keys"></a>Ключи учетной записи

Ключи учетной записи — это учетные данные, которые позволяет вашему приложению для проверки подлинности в службе Azure пространственных привязки. Ключи учетной записи предназначен помочь быстро начать работу. Особенно на этапе разработки приложения интегрируются с Azure пространственных привязки. Таким образом можно использовать ключи учетной записи, внедряя их в клиентских приложениях во время разработки. По мере продвижения работы за пределами разработки, настоятельно рекомендуется для перемещения механизм проверки подлинности, уровне рабочей среды, поддерживаемые маркеры доступа, или проверка подлинности пользователей Azure Active Directory. Чтобы получить ключ учетной записи для разработки, войти в учетную запись Azure пространственных привязки и перейдите на вкладку «Ключи».
