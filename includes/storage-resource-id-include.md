Идентификатор ресурса Azure AD указывает аудиторию, для которой выдается токен, который может быть использован для обеспечения доступа к ресурсу Azure. В случае хранения azure идентификатор ресурса может быть специфичен для одной учетной записи хранилища или может применяться к любой учетной записи хранилища. В следующей таблице описаны значения, которые можно уведомить для идентификатора ресурса:

|Идентификатор ресурса  |Описание  |
|---------|---------|
|`https://<account>.blob.core.windows.net` <br /><br /> `https://<account>.queue.core.windows.net`    | Конечная точка службы для данной учетной записи хранилища. Используйте это значение для приобретения маркера для авторизации запросов только для этой учетной записи и службы хранения Azure. Замените значение в скобках с именем учетной записи хранилища.      |
|`https://storage.azure.com/`     | Используйте для приобретения токена для авторизации запросов в любую учетную запись Azure Storage.        |
