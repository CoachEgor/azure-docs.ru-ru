Ресурсами Azure, на которые вы подписаны, будет представлено семейство служб Azure Cognitive Services. Создайте ресурс для Детектора аномалий с помощью [портала Azure](../articles/cognitive-services/cognitive-services-apis-create-account.md) или [интерфейса командной строки (CLI) Azure](../articles/cognitive-services/cognitive-services-apis-create-account-cli.md) на локальном компьютере. Также можно:

* Получить бесплатный [ключ пробной версии](https://azure.microsoft.com/try/cognitive-services/#decision) на 7 дней. После регистрации он будет доступен на [веб-сайте Azure](https://azure.microsoft.com/try/cognitive-services/my-apis/).  
* Просмотреть этот ресурс на [портале Azure](https://portal.azure.com/).
