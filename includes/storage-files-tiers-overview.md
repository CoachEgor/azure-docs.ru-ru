Azure Files предлагает два различных уровня хранения, премиум и стандартный, чтобы позволить вам адаптировать свои акции к производительности и ценовых требований вашего сценария:

- **Премиум-файлы:** Акции файлов Премиум поддерживаются твердотельные диски (SSD) и развертываются в типе **учетной записи хранения FileStorage.** Премиум-файлы обеспечивают постоянную высокую производительность и низкую задержку в пределах однозначных миллисекунд для большинства операций с ентами io- интенсивных. Это делает их пригодными для широкого спектра рабочих нагрузок, таких как базы данных, веб-хостинг и среды разработки. Файловые ресурсы уровня "Премиум" доступны только в подготовленной модели выставления счетов. Для получения дополнительной информации о подготовленной модели выставления счетов для премиальных файлов, [см.](../articles/storage/files/storage-files-planning.md#understanding-provisioning-for-premium-file-shares)
- **Стандартные файлы:** Стандартные файлы акции поддерживаются жестких дисков (HDDs) и развернуты в **общей версии 2 (GPv2) типа учетной записи хранения.** Стандартные файлы обеспечивают надежную производительность для рабочих нагрузок исмо-водоотваемости, которые менее чувствительны к изменчивости производительности, таким как общие файлообменники и среды dev/test. Файловые ресурсы уровня "Стандартный" доступны только в модели выставления счетов с оплатой по мере использования.
