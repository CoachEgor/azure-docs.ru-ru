На данный момент, ультра диски имеют дополнительные ограничения, они являются следующими:

Единственными вариантами избыточности инфраструктуры, доступными в настоящее время для ультрадисков, являются зоны доступности. VMs, использующие любые другие варианты избыточности, не могут прикрепить ультрадиск.

В следующей таблице описаны области ультрадисков, а также соответствующие варианты доступности:

> [!NOTE]
> Некоторые зоны доступности в этих регионах не предлагают ультра диски.

|Регионы  |Отсутствие избыточности инфраструктуры  |Зоны доступности  |
|---------|---------|---------|
|западная часть США     |Да         |нет         |
|западная часть США 2    |нет         |Да         |
|Восточная часть США     |нет         |Да         |
|восточная часть США 2     |нет         |Да         |
|юго-восточная Азия     |нет         |Да         |
|Северная Европа     |нет         |Да         |
|Западная Европа     |нет         |Да         |
|южная часть Соединенного Королевства     |нет         |Да         |

- Поддерживаются только в следующих сериях VM:
    - [ESv3](https://azure.microsoft.com/blog/introducing-the-new-dv3-and-ev3-vm-sizes/)
    - [DSv3](https://azure.microsoft.com/blog/introducing-the-new-dv3-and-ev3-vm-sizes/)
    - FSv2
    - [M](../articles/virtual-machines/workloads/sap/hana-vm-operations-storage.md)
    - [Mv2](../articles/virtual-machines/workloads/sap/hana-vm-operations-storage.md)
- Не каждый размер VM доступен в каждом поддерживаемом регионе с ультра-дисками
- Доступны только в качестве дисков данных и поддерживают только размер физического сектора 4k. Из-за 4K родной размер сектора Ultra Disk, Есть некоторые приложения, которые не будут совместимы с ультра дисков. Одним из примеров может быть Oracle Database, которая требует выпуска 12.2 или позже для поддержки ультра дисков.  
- Могут создаваться только как пустые диски.  
- В настоящее время не поддерживает снимки диска, VM-изображения, наборы доступности, выделенные хосты Azure или шифрование дисков Azure
- В настоящее время не поддерживает интеграцию с резервной поддержкой Azure или восстановлением сайта Azure
- Текущий максимальный лимит для IOPS на ГВМ составляет 80 000.

Ультрадиски Azure предлагают до 16 TiB в регионе в регионе по умолчанию, но ультра диски поддерживают более высокую емкость по запросу. Чтобы запросить увеличение емкости, обратитесь в службу поддержки Azure.
