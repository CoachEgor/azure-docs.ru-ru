Проверьте тип соглашения, чтобы определить тип учетной записи выставления счетов
 
1. Войдите на [портале Azure]( http://portal.azure.com).
 
2. Выполните поиск по **Управление затратами + выставление счетов**.
 
   ![Снимок экрана, показывающий поиск на портале Azure](./media/billing-check-account-type/billing-search-cost-management-billing.png)    
 
3. Если у вас есть доступ только одну область выставления счетов, выберите **свойства** из левой части окна.
 
    ![Снимок экрана, показывающий соглашении клиента microsoft на странице свойств](./media/billing-check-account-type/billing-mca-property.png)
    
    **Типа учетной записи выставления счетов** свойств страницы определяет тип учетной записи. Он может быть программы Microsoft Online Service, соглашения Enterprise или соглашения для клиентов Microsoft. Дополнительные сведения о типе выставления счетов, см. в разделе [просмотреть учетные записи выставления счетов на портале Azure](../articles/billing/billing-view-all-accounts.md).  
 
4. Если у вас есть доступ для нескольких областей выставления счетов, выберите **платежный счет** или **все выставления счетов области** из левой части окна. Проверьте тип **типа учетной записи выставления счетов** столбца.
 
    ![Снимок экрана, показывающий соглашении клиента microsoft на странице списка выставления счетов учетной записи](./media/billing-check-account-type/billing-account-type-in-the-list.png)
