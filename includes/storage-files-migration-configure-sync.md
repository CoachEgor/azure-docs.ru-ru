Этот шаг объединяет все ресурсы и папки, настроенные на сервере Windows на предыдущих этапах.

* Войти на [портал Azure](https://portal.azure.com).
* Найдите ресурс службы синхронизации хранения.
* Создайте новую *группу синхронизации* в ресурсе службы синхронизации хранения для каждого раздела файлов Azure. В терминологии Azure File Sync доля файлов Azure станет конечной *точкой облака* в топологии синхронизации, описанной с созданием группы синхронизации. Создавая группу синхронизации, дайте ей знакомое имя, чтобы распознать, какой набор файлов синхронизируется здесь. Убедитесь, что вы ссылаетесь на общий файл Azure с соответствующим именем.
* После создания группы синхронизации в списке групп синхронизации появится строка. Нажмите на имя (ссылку), чтобы отобразить содержимое группы синхронизации. Вы увидите свою долю файла Azure в "Облачных конечных точках".
* Найдите кнопку команды, чтобы *добавить конечную точку сервера.* Папка на локальном сервере, который вы сделали, станет путем для этой *конечной точки сервера.*
