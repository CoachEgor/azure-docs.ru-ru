На этом шаге все ресурсы и папки, настроенные на сервере Windows Server, будут объединены в ходе выполнения предыдущих действий.

* Войдите на [портал Azure](https://portal.azure.com).
* Нахождение ресурса службы синхронизации хранилища.
* Создайте новую *группу синхронизации* в ресурсе службы синхронизации хранилища для каждой общей папки Azure. В Синхронизация файлов Azure терминологии файловый ресурс Azure станет *облачной конечной точкой* в топологии синхронизации, которую вы описываете при создании группы синхронизации. При создании группы синхронизации присвойте ей привычное имя, чтобы вы узнали, какой набор файлов будет синхронизироваться здесь. Убедитесь, что вы ссылаетесь на файловый ресурс Azure с совпадающим именем.
* После создания группы синхронизации в списке групп синхронизации появится строка для нее. Щелкните имя (ссылку), чтобы отобразить содержимое группы синхронизации. Общая папка Azure появится в разделе "облачные конечные точки".
* Нажмите кнопку "команда", чтобы *Добавить конечную точку сервера*. Папка на локальном сервере, которая была подготовлена, станет путем к этой *конечной точке сервера*.
