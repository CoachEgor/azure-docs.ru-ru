На этом шаге вам понадобятся учетные данные подписки Azure. Используемая подписка Azure может отличаться от той, которая используется для StorSimple.

Основной ресурс для настройки Синхронизация файлов Azure называется "Служба синхронизации хранилища".
Рекомендуется развертывать только один для всех серверов в компании, которые в настоящее время синхронизируют один и тот же набор файлов в будущем. При наличии нескольких устройств StorSimple можно создать ресурс службы синхронизации хранилища для каждого из них. Однако следует создавать несколько служб синхронизации хранилища, если у вас есть отдельные наборы серверов, которые не должны обмениваться данными. В противном случае рекомендуется использовать одну службу синхронизации хранилища.

Выберите регион Azure для службы синхронизации хранилища, расположенную ближе к вашему офису. Все остальные облачные ресурсы должны быть развернуты в одном регионе.
Рекомендуется создать новую группу ресурсов в подписке для размещения ресурсов синхронизации и хранения для упрощения управления.

В следующей статье описывается развертывание службы синхронизации хранилища. Используйте только эту часть документа. В последующих шагах будут приведены ссылки на другие подразделы этого документа.
