## <a name="prerequisites"></a>Предварительные требования 
Для работы с этим руководством требуется VHD-файл, переданный в учетную запись хранения, связанную с лабораторией, в которой создается пользовательский образ. Выберите любую из следующих ссылок, чтобы передать VHD-файл.

- [Upload VHD file to lab's storage account using AzCopy](../articles/devtest-labs/devtest-lab-upload-vhd-using-azcopy.md) (Отправка VHD-файла в учетную запись хранения лаборатории с помощью AzCopy)
- [Upload VHD file to lab's storage account using Microsoft Azure Storage Explorer](../articles/devtest-labs/devtest-lab-upload-vhd-using-storage-explorer.md) (Отправка VHD-файла в учетную запись хранения лаборатории с помощью обозревателя хранилищ Microsoft Azure)
- [Upload VHD file to lab's storage account using PowerShell](../articles/devtest-labs/devtest-lab-upload-vhd-using-powershell.md) (Отправка VHD-файла в учетную запись хранения лаборатории с помощью PowerShell)
