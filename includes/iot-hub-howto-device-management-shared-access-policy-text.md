В этой статье создается служба бэкэнда, которая вызывает прямой метод на устройстве. Чтобы вызвать прямой метод на устройстве через IoT Hub, службе требуется разрешение **на подключение службы.** По умолчанию каждый концентратор IoT создается с помощью службы общего доступа, названной **службой,** которая предоставляет это разрешение.
