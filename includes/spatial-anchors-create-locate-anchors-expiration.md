## <a name="set-expiration"></a>Срок действия

Можно также настроить для привязки автоматическое истечение срока действия в определенный момент времени в будущем. После истечения срока действия привязку больше нельзя будет найти или обновить. Срок действия можно задать только при создании привязки. После этого изменить срок действия нельзя. Таким образом, вы можете задать это значение перед сохранением в облаке.
