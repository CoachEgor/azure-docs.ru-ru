> [!NOTE]
> В Azure есть две различные модели развертывания, которые можно использовать для создания ресурсов и работы с ними: [Azure Resource Manager и Classic](../articles/azure-resource-manager/management/deployment-models.md). В этой статье рассматривается использование модели развертывания диспетчер ресурсов. Рекомендуется использовать модель развертывания диспетчер ресурсов для новых развертываний вместо классической модели развертывания.
