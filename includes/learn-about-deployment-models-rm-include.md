> [!NOTE]
> В Azure есть две различные модели развертывания, которые можно использовать для создания ресурсов и работы с ними: [развертывание с помощью Azure Resource Manager и классическая модель развертывания](../articles/azure-resource-manager/resource-manager-deployment-model.md). В этой статье рассматривается использование модели развертывания диспетчер ресурсов. Рекомендуется использовать модель развертывания диспетчер ресурсов для новых развертываний вместо классической модели развертывания.
