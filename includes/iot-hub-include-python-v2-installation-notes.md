* Активная учетная запись Azure. Если ее нет, можно создать [бесплатную учетную запись](https://azure.microsoft.com/pricing/free-trial/) всего за несколько минут.

* Рекомендуется использовать [Python версии 3,7 или более поздней](https://www.python.org/downloads/) . Обязательно используйте 32-разрядную или 64-разрядную версию установки согласно требованиям программы настройки. При появлении запроса во время установки обязательно добавьте Python в переменную среды соответствующей платформы. Сведения о других поддерживаемых версиях Python см. в документации по пакету SDK в разделе о [возможностях устройств Azure IoT](https://github.com/Azure/azure-iot-sdk-python/tree/master/azure-iot-device#azure-iot-device-features). Если вы решили использовать Python 2,7, возможно, потребуется [установить или обновить *pip*, система управления пакетами Python](https://pip.pypa.io/en/stable/installing/).
