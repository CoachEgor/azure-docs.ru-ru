> [!NOTE]
> Функция, [требующая безопасной передачи](../articles/storage/common/storage-require-secure-transfer.md), обеспечивает принудительную передачу всех запросов к вашей учетной записи через безопасное подключение. Только кластер HDInsight версии 3.6 или более новой поддерживает эту функцию. Дополнительные сведения см. в статье [Создание кластера Apache Hadoop с помощью учетных записей хранения с безопасной передачей в Azure HDInsight](../articles/hdinsight/hdinsight-hadoop-create-linux-clusters-with-secure-transfer-storage.md).
