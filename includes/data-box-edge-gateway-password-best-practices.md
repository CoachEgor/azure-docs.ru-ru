Имейте в виду эти рекомендации:

- Мы рекомендуем хранить все пароли в безопасном месте, чтобы вам не пришлось сбросить пароль, если он забыт. Служба управления не может получить существующие пароли. Он может сбросить их только через портал Azure. Если вы сбросили пароль, обязательно уведомите всех пользователей перед его сбросить.
- Вы можете получить доступ к интерфейсу Windows PowerShell вашего устройства удаленно через HTTP. В качестве наилучшей практики безопасности следует использовать HTTP только в надежных сетях.
- Убедитесь, что пароли устройств являются сильными и хорошо защищены. Следуйте [рекомендациям пароля](https://docs.microsoft.com/azure/security/fundamentals/identity-management-best-practices#enable-password-management).
