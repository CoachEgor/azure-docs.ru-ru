Учитывайте следующие рекомендации:

- Рекомендуется хранить все пароли в безопасном месте, чтобы не сбрасывать пароль, если он забыт. Службе управления не удается получить существующие пароли. Его можно сбросить только с помощью портал Azure. При сбросе пароля не забудьте уведомить всех пользователей, прежде чем сбросить его.
- Вы можете удаленно обращаться к интерфейсу Windows PowerShell устройства по протоколу HTTP. Для обеспечения безопасности рекомендуется использовать HTTP только в доверенных сетях.
- Убедитесь, что пароли устройств надежно и хорошо защищены. Следуйте рекомендациям по работе с [паролями](https://docs.microsoft.com/azure/security/fundamentals/identity-management-best-practices#enable-password-management).
