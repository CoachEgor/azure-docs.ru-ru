Эти рекомендации не забывайте:

- Рекомендуется хранить все пароли в безопасном месте, поэтому вам не нужно сбросить пароль, если вы его забудете. Служба управления не выполняет извлечение существующих паролей. А только сбрасывает их на портале Azure. При сбросе пароля, не забудьте уведомить всех пользователей перед его сбросить.
- Доступны в интерфейсе Windows PowerShell устройства удаленно по протоколу HTTP. По соображениям безопасности следует использовать HTTP только в доверенных сетях.
- Обеспечить надежные и хорошо защищенные пароли устройства. Выполните [рекомендации по паролям](https://docs.microsoft.com/azure/security/azure-security-identity-management-best-practices#enable-password-management).
