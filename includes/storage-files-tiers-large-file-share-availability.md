Для стандартных общих папок емкостью 100 ТиБ действуют определенные ограничения.

- Сейчас поддерживаются только учетные записи локально избыточного хранилища (LRS) и геоизбыточного хранилища (GRS).
- После включения больших общих папок нельзя преобразовать учетные записи хранения в учетные записи геоизбыточного хранилища (GRS) или хранилища, геоизбыточного между зонами (GZRS).
- После включения больших общих папок их нельзя отключить.
