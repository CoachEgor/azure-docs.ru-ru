## <a name="disable-email-verification"></a>Отключение проверки по электронной почте

По умолчанию Azure Active Directory B2C (Azure AD B2C) проверяет адрес электронной почты клиента для локальных учетных записей (учетных записей для пользователей, которые регистрируются с адресом электронной почты или именем пользователя). Azure AD B2C обеспечивает действительные адреса электронной почты, требуя от клиентов проверять их во время процесса регистрации. Он также предотвращает использование злоумышленниками автоматизированных процессов для генерации мошеннических учетных записей в приложениях.

Некоторые разработчики приложений предпочитают пропускать проверку электронной почты во время процесса регистрации и вместо этого заставить клиентов проверить свой адрес электронной почты позже. Для этого в Azure AD B2C можно отключить проверку адреса электронной почты. Это создает более плавный процесс регистрации и дает разработчикам гибкость для дифференциации клиентов, которые проверили свой адрес электронной почты от клиентов, которые этого не сделали.

> [!WARNING]
> Отключение проверки адреса электронной почты в процессе регистрации может привести к получению спама. Если вы отменяете проверку электронной почты Azure AD B2C, предоставленную Azure AD, мы рекомендуем внедрить систему проверки замены.
