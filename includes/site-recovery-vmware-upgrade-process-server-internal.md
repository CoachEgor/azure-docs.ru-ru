1. Войти на сервер процесса в качестве администратора.
2. Загрузите последнюю версию [Единой установки.](https://aka.ms/unifiedinstaller)
3. Дважды щелкните установщик для запуска процесса обновления.
4. Установка обнаруживает компоненты восстановления сайта, которые установлены, и обновляет их до последней версии.
