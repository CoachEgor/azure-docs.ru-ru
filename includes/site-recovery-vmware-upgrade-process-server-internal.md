1. Войдите на сервер обработки с правами администратора.
2. Скачайте последнюю версию [установщика](https://aka.ms/unifiedinstaller).
3. Дважды щелкните установщик для запуска процесса обновления.
4. Программа обнаружит установленные компоненты Site Recovery и обновит их до последней версии.
