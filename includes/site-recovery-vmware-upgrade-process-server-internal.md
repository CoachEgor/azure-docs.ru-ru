2. Скачайте последнюю версию [единой программы установки](https://aka.ms/unifiedinstaller).
3. Дважды щелкните установщик для запуска процесса обновления.
4. Установщик обнаруживает установленные компоненты Site Recovery и обновляет их до последней версии.
