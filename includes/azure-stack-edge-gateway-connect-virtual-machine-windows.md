Подключитесь к виртуальной машине Windows с помощью протокол удаленного рабочего стола (RDP) через IP-адрес, переданный во время создания виртуальной машины.

1. На клиенте запустите RDP. Перейдите в **Начало** и введите **mstsc**.
1. Введите IP-адрес виртуальной машины и учетные данные доступа, которые использовались в файле параметров шаблона виртуальной машины.

    ![Подключение через RDP к виртуальной машине Windows](media/azure-stack-edge-gateway-connect-vm-windows/connect-vm-rdp-1.png)
1. Может потребоваться утвердить подключение к ненадежному компьютеру. 
1. Теперь вы вошли в виртуальную машину, работающую на устройстве.
