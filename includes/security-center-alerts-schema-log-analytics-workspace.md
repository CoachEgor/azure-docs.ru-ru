### <a name="the-data-model-of-the-schema"></a>Модель данных схемы

|Поле|Описание|
|----|----|
|**AlertName**|Имя отображения оповещения|
|**AlertType**|уникальный идентификатор оповещения|
|**ДовериеУровень**|(Необязательно) Уровень достоверности этого предупреждения (High/Low)|
|**ConfidenceScore**|(Необязательно) Численный индикатор достоверности оповещения о безопасности|
|**Описание**|Текст описания для оповещения|
|**DisplayName**|Имя отображения оповещения|
|**Время в конце**|Время окончания удара оповещения (время последнего события, способствуя оповещению)|
|**Entities**|Список сущностей, связанных с предупреждением. Этот список может содержать смесь сущностей различных типов|
|**Расширенные ссылки**|(Необязательно) Сумка для всех ссылок, связанных с предупреждением. Этот мешок может содержать смесь ссылок для различных типов|
|**РасширенныйСвойства**|Мешок дополнительных полей, имеющих отношение к предупреждению|
|**IsIncident**|Определяет, является ли оповещение инцидентом или обычным предупреждением. Инцидент — это оповещение о безопасности, которое агрегирует несколько предупреждений в один инцидент безопасности|
|**ProcessingEndTime**|Метка времени UTC, в которой было создано оповещение|
|**Компонентноеимя**|(Необязательно) Название компонента внутри продукта, который генерировал оповещение.|
|**ProductName**|постоянная ('Центр безопасности Azure')|
|**ProviderName**|unused|
|**Восстановительные шаги**|Ручные элементы действий для устранения угрозы безопасности|
|**ResourceId**|Полный идентификатор пострадавшего ресурса|
|**Серьезность**|Тяжесть оповещения (High/Medium/Low/Informational)|
|**SourceComputerId**|уникальный GUID для пострадавшего сервера (если оповещение генерируется на сервере)|
|**SourceSystem**|unused|
|**StartTime**|Время начала удара оповещения (время первого события, способствуя оповещению)|
|**SystemAlertId**|Уникальный идентификатор этого экземпляра оповещения безопасности|
|**ТенантИд**|идентификатор родительского каталога Azure Active, нанимающего подписку, в соответствии с которой находится отсканированный ресурс|
|**TimeGenerated**|ETC timestamp, на котором проводилась оценка (время сканирования Центра безопасности) (идентично discoveredTimeUTC)|
|**Тип**|константа ('SecurityAlert')|
|**Имя поставщика**|Имя поставщика, предоставивего оповещение (например, "Microsoft")|
|**ПоставщикOriginalId**|unused|
|**Рабочее пространствоРесурсГруппа**|в случае, если оповещение генерируется на VM, Server, Virtual Machine Scale Set или экземпляре службы приложений, который сообщает в рабочее пространство, содержит имя группы ресурсов рабочего пространства, содержащее имя группы ресурсов рабочего пространства|
|**WorkspaceSubscriptionId**|в случае, если оповещение генерируется на VM, Server, Virtual Machine Scale Set или экземпляре службы приложений, который сообщает в рабочее пространство, содержит подписку на рабочее пространство|
