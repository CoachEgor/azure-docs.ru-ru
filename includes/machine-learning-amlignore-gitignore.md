Чтобы предотвратить включение ненужных файлов в моментальный снимок, создайте файл Ignore (`.gitignore` или `.amlignore`) в каталоге. Добавьте файлы и каталоги для исключения в этот файл. Дополнительные сведения о синтаксисе, который будет использоваться в этом файле, см. в разделе `.gitignore` [синтаксис и шаблоны](https://git-scm.com/docs/gitignore) для. Этот `.amlignore` файл использует тот же синтаксис. _Если оба файла существуют, приоритет `.amlignore` имеет файл._
