## <a name="create-an-azure-resource"></a>Создание ресурса Azure

Чтобы начать работу с API Визуального поиска Bing, создайте один из следующих ресурсов Azure.

[Пробный ресурс](https://azure.microsoft.com/try/cognitive-services/?api=bing-web-search-api)
   * Подписка Azure не требуется.
   * срок действия —7 дней, бесплатно. Когда вы зарегистрируетесь, ключ пробной версии и конечная точка будут доступны на [веб-сайте Azure](https://azure.microsoft.com/try/cognitive-services/my-apis/).

[Ресурс Поиска Bing версии 7](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingSearch-v7)
   * доступен на портале Azure до удаления.
   * Выберите ценовую категорию `S9`.

[Ресурс для нескольких служб](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne)
   * доступен на портале Azure до удаления.  
   * Используйте одни и те же ключ и конечную точку для приложений в нескольких экземплярах Cognitive Services.
