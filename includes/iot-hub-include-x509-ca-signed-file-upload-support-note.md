> [!IMPORTANT]
> Функция отправки файлов на устройствах, использующих проверку подлинности центра сертификации X. 509, доступна в общедоступной предварительной версии, а [режим предварительного просмотра должен быть включен](../articles/iot-hub/iot-hub-preview-mode.md). Он общедоступен на устройствах, использующих проверку подлинности X. 509. Дополнительные сведения о проверке подлинности X. 509 с помощью центра Интернета вещей см. в разделе [Поддерживаемые сертификаты x. 509](../articles/iot-hub/iot-hub-devguide-security.md#supported-x509-certificates).
