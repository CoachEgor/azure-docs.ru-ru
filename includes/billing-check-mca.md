Проверьте тип соглашения, чтобы узнать, есть ли у вас доступ к учетной записи выставления счетов для клиентского соглашения Майкрософт.
 
1. Перейдите в [портал Azure](https://portal.azure.com) , чтобы проверить доступ к учетной записи выставления счетов. Найдите и выберите **Управление затратами + выставление счетов**.

   ![Поиск по управлению затратами и выставлению счетов, портал Microsoft Azure](./media/billing-check-mca/billing-search-cost-management-billing.png)
 
2. Если у вас есть доступ только к одной области выставления счетов, выберите в меню пункт **Свойства** . Если тип учетной записи выставления счетов — **Клиентское соглашение Майкрософт**, то у вас есть доступ к учетной записи выставления счетов для клиентского соглашения Майкрософт.
 
    ![Соглашение клиента Майкрософт, тип счета для выставления счетов, свойства портал Microsoft Azure](./media/billing-check-mca/billing-mca-property.png)
 
3. При наличии доступа к нескольким областям выставления счетов проверьте тип в столбце учетной записи выставления счетов. Если для всех областей в качестве типа учетных записей выставления счетов будет **Клиентское соглашение Майкрософт**, то у вас есть доступ к учетной записи выставления счетов для клиентского соглашения Майкрософт.
 
    [![соглашение клиента Майкрософт, тип счета выставления счетов, список учетных записей выставления счетов, портал Microsoft Azure](./media/billing-check-mca/billing-mca-in-the-list.png)](./media/billing-check-mca/billing-mca-in-the-list-zoomed-in.png#lightbox)
