Проверьте тип соглашения, чтобы определить, имеется ли доступ к учетной записи выставления счетов для клиента соглашение Microsoft.
 
1. Войдите на [портале Azure]( http://portal.azure.com).
 
2. Выполните поиск по **Управление затратами + выставление счетов**.
 
   ![Снимок экрана, показывающий поиск на портале Azure](./media/billing-check-mca/billing-search-cost-management-billing.png)
 
3. Если у вас есть доступ только одну область выставления счетов, выберите **свойства** из левой части окна. У вас есть доступ к учетной записи выставления счетов для клиента соглашение Microsoft при тип учетной записи выставления счетов **соглашении клиента Microsoft**.
 
    ![Снимок экрана, показывающий соглашении клиента microsoft на странице свойств](./media/billing-check-mca/billing-mca-property.png)
 
4. При наличии доступа к области нескольких выставления счетов, проверьте тип в столбце учетной записи выставления счетов. У вас есть доступ к учетной записи выставления счетов для клиента соглашение Microsoft Если тип учетной записи выставления счетов для любой из областей **соглашении клиента Microsoft**.
 
    ![Снимок экрана, показывающий соглашении клиента microsoft на странице списка выставления счетов учетной записи](./media/billing-check-mca/billing-mca-in-the-list.png)
