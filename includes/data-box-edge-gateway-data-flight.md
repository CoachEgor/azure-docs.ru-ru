Для данных в процессе использования:

- Для данных, передающихся между устройством и Azure используется стандартный протокол TLS 1.2. Нет без отката для TLS 1.1 и более ранних версий. Связи с агентом будут заблокированы, если TLS 1.2 не поддерживается. Протокол TLS 1.2 также является обязательным для портала и пакет SDK для управления.
- При клиентов доступа к устройству через локальный пользовательский Интерфейс браузера, как защищенный протокол по умолчанию используется стандартный протокол TLS 1.2.

    - Рекомендуется настроить браузер для использования TLS 1.2.
    - Если браузер не поддерживает TLS 1.2, можно использовать TLS 1.1 или TLS 1.0.
- Мы рекомендуем использовать SMB 3.0 с шифрованием для защиты данных при копировании из серверы данных.
