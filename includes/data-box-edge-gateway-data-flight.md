Для данных в полете:

- Стандартный TLS 1.2 используется для данных, перемещаемых между устройством и Azure. Резервного возврата к TLS 1.1 и раньше нет. Связь с агентом будет заблокирована, если TLS 1.2 не будет поддержан. TLS 1.2 также требуется для управления порталом и SDK.
- Когда клиенты получают доступ к устройству через локальный веб-интерфейс браузера, стандартный TLS 1.2 используется в качестве безопасного протокола по умолчанию.

    - Наилучшей практикой является настройка браузера для использования TLS 1.2.
    - Если браузер не поддерживает TLS 1.2, вы можете использовать TLS 1.1 или TLS 1.0.
- Мы рекомендуем использовать SMB 3.0 с шифрованием для защиты данных при копировании данных с серверов данных.
