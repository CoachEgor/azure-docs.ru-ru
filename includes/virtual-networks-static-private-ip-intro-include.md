Виртуальная машина (VM) автоматически назначается частный IP-адрес из диапазона, который вы указываете, на основе подсети, в которой развертывается VM. VM сохраняет адрес до тех пор, пока VM не будет удален. Azure динамически назначает следующий доступный частный IP-адрес из подсети, в ней вы создаете VM. Если требуется определенный IP-адрес из подсети, назначенной VM, назначаем статический IP-адрес.
