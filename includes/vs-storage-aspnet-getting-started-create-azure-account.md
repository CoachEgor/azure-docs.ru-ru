## <a name="create-an-azure-storage-account"></a>Создание учетной записи хранения Azure
Самый простой способ создать первую учетную запись хранения Azure — воспользоваться [порталом Azure](https://portal.azure.com). Дополнительную информацию см. в статье [Об учетных записях хранения Azure](https://docs.microsoft.com/azure/storage/common/storage-quickstart-create-account).

Кроме того, создать учетную запись хранения Azure можно с помощью [Azure PowerShell](../articles/storage/common/storage-powershell-guide-full.md), [Azure CLI](../articles/storage/common/storage-azure-cli.md) или [клиентской библиотеки поставщика ресурсов хранилища для .NET](https://msdn.microsoft.com/library/azure/mt131037.aspx).
