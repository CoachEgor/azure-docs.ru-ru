## <a name="prerequisites"></a>Предварительные требования

* Учетная запись [MicrosoftTranslator](https://www.microsoft.com/translator).  

Перед использованием учетной записи MicrosoftTranslator в приложении логики необходимо авторизовать приложение логики для подключения к учетной записи MicrosoftTranslator. Это легко сделать в приложении логики на портале Azure.  

Ниже приведены указания по авторизации приложения логики для подключения к учетной записи MicrosoftTranslator.  

1. Чтобы создать подключение к MicrosoftTranslator, в конструкторе приложений логики в раскрывающемся списке выберите параметр **Показать API, управляемые Майкрософт**, а затем введите в поле поиска *MicrosoftTranslator*. Выберите триггер или действие, которые хотите использовать:   
   ![Этап создания подключения MicrosoftTranslator](./media/connectors-create-api-microsofttranslator/microsofttranslator-1.png)  
2. Если вы ранее не создавали подключения к MicrosoftTranslator, вам будет предложено ввести учетные данные MicrosoftTranslator. Эти учетные данные будут использоваться для авторизации приложения логики, чтобы оно могло подключиться и получить доступ к данным вашей учетной записи MicrosoftTranslator.  
   ![Этап создания подключения MicrosoftTranslator](./media/connectors-create-api-microsofttranslator/microsofttranslator-2.png)  
3. Подключение создано, и теперь вы можете перейти к другим действиям в приложении логики:   
   ![Этап создания подключения MicrosoftTranslator](./media/connectors-create-api-microsofttranslator/microsofttranslator-3.png)
