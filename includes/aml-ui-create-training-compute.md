Эксперимент выполняется на целевой объект вычислений, вычислительный ресурс, подключенный к рабочей области.  После создания целевого объекта вычислений, ее можно повторно при последующих запусках.

1. Выберите **запуска** внизу, чтобы запустить эксперимент.

     ![Выполнение эксперимента](./media/aml-ui-create-training-compute/run-experiment.png)

1. Когда **целевых объектов вычислений установки** откроется диалоговое окно, если вашей рабочей области уже вычислительный ресурс, можно выбрать его сейчас.  В противном случае выберите **создать**.

    > [!NOTE]
    > Визуальный интерфейс можно только запускать эксперименты для целевых объектов вычислений для машинного обучения. Другие целевые объекты вычисления не отображается.

1. Укажите имя для вычислительных ресурсов.

1. Выберите **Запуск**.

    ![Настройка целевого объекта вычислений](./media/aml-ui-create-training-compute/set-compute.png)

    Теперь будут создаваться вычислительный ресурс. Просмотр состояния в правом верхнем углу эксперимента. 

    > [!NOTE]
    > Занимает примерно 5 минут для создания вычислительных ресурсов. После создания ресурса, можно повторно использовать ее и пропустить это время ожидания при последующих запусках.
    >
    > Вычислительный ресурс будет выполнено Автомасштабирование до 0 узлов, во время простоя для экономии затрат.  При использовании попытку через некоторое время, повторно возникают примерно 5 минут, время ожидания при масштабировании назад.
