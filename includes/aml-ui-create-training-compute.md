Эксперимент выполняется с целевым объектом вычислений. Это вычислительный ресурс, подключенный к вашей рабочей области.  После создания целевого объекта вычислений его можно повторно использовать при последующих запусках.

1. Выберите **Выполнить** в нижней части окна, чтобы запустить эксперимент.

     ![Выполнение эксперимента](./media/aml-ui-create-training-compute/run-experiment.png)

1. Если в вашей рабочей области существует вычислительный ресурс, его можно выбрать в диалоговом окне **настройки целевых объектов вычислений**.  Если нет, выберите **Создать**.

    > [!NOTE]
    > Визуальный интерфейс может выполнять эксперименты только с целевыми объектами вычислений для машинного обучения. Другие целевые объекты вычислений не отображаются.

1. Укажите имя для вычислительного ресурса.

1. Выберите **Запуск**.

    ![Настройка целевого объекта вычислений](./media/aml-ui-create-training-compute/set-compute.png)

    Теперь будет создан вычислительный ресурс. Ход создания можно отслеживать в правом верхнем углу окна эксперимента. 

    > [!NOTE]
    > Создание вычислительного ресурса занимает порядка 5 минут. После создания ресурса его можно повторно использовать, экономя время при последующих запусках.
    >
    > Вычислительный ресурс автомасштабируется до 0 узлов во время простоя для экономии затрат.  При повторном его использовании после простоя придется снова ждать около 5 минут, пока выполняется масштабирование ресурса.
