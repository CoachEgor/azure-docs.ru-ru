Необходимо иметь [подписку API Cognitive Services](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) с доступом к этому API анализа текста. Если у вас нет подписки, создайте бесплатную [учетную запись](https://azure.microsoft.com/try/cognitive-services/?api=bing-web-search-api). Перед тем как продолжить, потребуется ключ подписки Анализа текста, предоставляемый после активации учетной записи.
