Ключ и конечная точка для ресурса Анализа текста. Ресурсами Azure, на которые вы подписаны, будет представлено семейство служб Azure Cognitive Services. Создайте ресурс для Аналитики текста с помощью [портала Azure](../articles/cognitive-services/cognitive-services-apis-create-account.md) или [интерфейса командной строки Azure (CLI)](../articles/cognitive-services/cognitive-services-apis-create-account-cli.md) на локальном компьютере. Ресурс также можно просмотреть на [портале Azure](https://portal.azure.com/).
