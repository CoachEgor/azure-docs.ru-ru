Azure Data Lake Analytics makes the complex task of managing distributed infrastructure and complex code easy. It dynamically provisions resources, and you can use it to do analytics on exabytes of data. When the job completes, it winds down resources automatically. You pay only for the processing power that was used. При увеличении или уменьшении объема хранимых данных или вычислительных ресурсов не нужно переписывать код. To raise the default limits for your subscription, contact support.

| --- | --- | --- |
| Максимальное количество параллельных заданий |20 | |
| Maximum number of analytics units (AUs) per account |250 | Используйте любую комбинацию единиц аналитики (не более 250 единиц) для 20 заданий. To increase this limit, contact Microsoft Support. |
| Максимальный размер скрипта для отправки заданий | 3 МБ | |
