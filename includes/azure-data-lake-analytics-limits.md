Azure Data Lake Analytics упрощает сложную задачу управления распределенной инфраструктурой и сложным кодом. Она динамически подготавливает ресурсы и его можно использовать, чтобы работать с эксабайтами данных. По завершении задания вычислительную ресурсов автоматически. Вы платите только за вычислительной мощности, который был использован. При увеличении или уменьшении объема хранимых данных или вычислительных ресурсов не нужно переписывать код. Чтобы увеличить значения ограничений по умолчанию для вашей подписки, обратитесь в службу поддержки.

| **Ресурс** | **Ограничение по умолчанию** | **Комментарии** |
| --- | --- | --- |
| Максимальное количество параллельных заданий |20 | |
| Максимальное количество единиц аналитики (кластера AUs) в учетной записи |250 | Используйте любую комбинацию единиц аналитики (не более 250 единиц) для 20 заданий. Чтобы увеличить этот предел, обратитесь в службу поддержки Майкрософт. |
| Максимальный размер скрипта для отправки заданий | 3 МБ | |
| Максимальное число учетных записей аналитики Озера данных на регион на подписку | 5 | Чтобы увеличить этот предел, обратитесь в службу поддержки Майкрософт. |
