Azure Data Lake Analytics упрощает задачу управления распределенной инфраструктурой и сложным кодом. Он динамически резервирует ресурсы, и вы можете использовать его для аналитики на эксабайтах данных. Когда задание завершается, оно автоматически сворачивает ресурсы. Вы платите только за используемую вычислительную мощность. При увеличении или уменьшении размера хранимых данных или количества используемых вычислений не нужно переписывать код. Чтобы повысить лимиты по умолчанию для подписки, обратитесь в службу поддержки.

| **Ресурсов** | **Предел** | **Комментарии** |
| --- | --- | --- |
| Максимальное количество параллельных заданий |20 | |
| Максимальное количество аналитических единиц (AUs) на счет |250 | Используйте любую комбинацию единиц аналитики (не более 250 единиц) для 20 заданий. Чтобы увеличить этот лимит, обратитесь в службу поддержки майкрософт. |
| Максимальный размер скрипта для отправки заданий | 3 МБ | |
| Максимальное количество учетных записей Data Lake Analytics в регионе за подписку | 5 | Чтобы увеличить этот лимит, обратитесь в службу поддержки майкрософт. |
