Azure Data Lake Analytics упрощает комплексную задачу управления распределенной инфраструктурой и сложным кодом. Он динамически подготавливает ресурсы, и вы можете использовать его для анализа эксабайтам данных. Когда задание будет завершено, оно подойдет к концу ресурсы автоматически. Вы платите только за используемую вычислительную мощность. При увеличении или уменьшении размера хранимых данных или объема используемых вычислений не нужно переписывать код. Чтобы увеличить ограничения по умолчанию для подписки, обратитесь в службу поддержки.

| **Ресурс** | **Ограничение** | **Комментарии** |
| --- | --- | --- |
| Максимальное количество параллельных заданий |20 | |
| Максимальное количество единиц аналитики (Австралия) на учетную запись |250 | Используйте любую комбинацию единиц аналитики (не более 250 единиц) для 20 заданий. Чтобы увеличить это ограничение, обратитесь в служба поддержки Майкрософт. |
| Максимальный размер скрипта для отправки заданий | 3 МБ | |
| Максимальное число учетных записей Data Lake Analytics для региона на подписку | 5 | Чтобы увеличить это ограничение, обратитесь в служба поддержки Майкрософт. |
