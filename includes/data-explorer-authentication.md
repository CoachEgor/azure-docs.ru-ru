## <a name="authentication"></a>Проверка подлинности

Для выполнения следующего примера необходимо приложение Azure AD и субъект-служба, которые могут получать доступ к ресурсам. См. статью [Создание приложения Azure AD](/azure/active-directory/develop/howto-create-service-principal-portal) для создания бесплатного приложения Azure AD и Добавление назначения ролей на уровне подписки. Также требуется `directory (tenant) ID`, `application ID` и `client secret`.
