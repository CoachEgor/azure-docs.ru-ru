| --- | --- | --- |
| Учетные записи Azure Batch в регионе за подписку | 1–3 |50 |
| Выделенные ядра на учетную запись Пакетной службы | 90-900 | Обращение в службу поддержки |
| Ядра с низким приоритетом на каждую учетную запись пакетной службы | 10-100 | Обращение в службу поддержки |
| **[Активные](https://docs.microsoft.com/rest/api/batchservice/job/get#jobstate)** задания и графики заданий на счет пакета **(завершенные** задания не имеют предела) | 100-300 | 1000<sup>1</sup> |
| Количество пулов на одну учетную запись пакетной службы | 20-100 | 500<sup>1</sup> |

<sup>1</sup> Чтобы запросить увеличение сверх этого лимита, обратитесь в службу поддержки Azure.

> [!NOTE]
> Ограничения по умолчанию зависят от типа подписки, используемой для создания учетной записи пакетной службы. Показанные квоты на ядра относятся к учетным записям пакетной службы в режиме пакетной службы. [Просмотр квот в учетной записи пакетной службы](../articles/batch/batch-quota-limit.md#view-batch-quotas).

> [!IMPORTANT]
> Чтобы помочь нам лучше управлять потенциалом во время глобальной пандемии здравоохранения, основные квоты по умолчанию для новых учетных записей пакетов в некоторых регионах и для некоторых типов подписки были сокращены с вышеуказанного диапазона значений, в некоторых случаях до нуля ядер. При создании новой учетной записи Batch [проверьте свою основную квоту](../articles/batch/batch-quota-limit.md#view-batch-quotas) и при необходимости [запросите увеличение основной квоты.](../articles/batch/batch-quota-limit.md#increase-a-quota)
