## <a name="go-to-your-api-management-instance"></a>Перейдите к экземпляру службы управления API.

1. На портале Azure найдите и выберите **Службы управления API**.

   ![Выбор "Службы управления API"](media/api-management-navigate-to-instance/view-apim1.png)

1. На экране **Управление API** выберите экземпляр службы "Управление API".

   ![Выбор экземпляра службы "Управление API"](media/api-management-navigate-to-instance/view-apim2.png)
