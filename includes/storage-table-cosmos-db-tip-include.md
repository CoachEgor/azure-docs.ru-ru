> [!TIP]
> Сведения в этой статье применимы к обычному хранилищу таблиц Azure. Но теперь есть расширенная версия хранилища таблиц в виде API таблиц Azure Cosmos DB. Для нее предусмотрены оптимизированные для пропускной способности таблицы, глобальное распределение и автоматическое вторичное индексирование. Существуют некоторые функциональные [различия между API таблиц в Azure Cosmos DB и Табличном хранилище Azure](../articles/cosmos-db/faq.md#where-is-table-api-not-identical-with-azure-table-storage-behavior). См. дополнительные сведения об [API таблиц Azure Cosmos DB](https://aka.ms/premiumtables).
