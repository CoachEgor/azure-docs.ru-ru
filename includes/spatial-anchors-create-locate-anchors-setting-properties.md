## <a name="set-properties"></a>Настройка свойств

Вы можете добавить несколько свойств, при сохранении вашего привязки облака. Как и тип объекта, сохраненного или основные свойства, например ли она должна быть включена для взаимодействия. Это может быть полезно после обнаружения: можно немедленно отображения объекта для пользователя, например рисунок с пустым содержимым. Затем разные скачивания в фоновом режиме получает сведения о дополнительных состояния, например, рисунок, отображаемый в рамке.
