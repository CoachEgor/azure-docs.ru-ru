Azure позволяет размещать _блокировки_ ресурсов, чтобы их нельзя было удалить или использовать только для чтения. __Блокировка ресурса может привести к непредвиденным результатам.__ Для некоторых операций, которые не должны изменять ресурс, фактически требуются действия, заблокированные блокировкой. 

Например, при применении блокировки удаления к группе ресурсов для рабочей области операции масштабирования для вычислений в кластерах машинного обучения Azure будут запрещены.

Дополнительные сведения о блокировке ресурсов см. [в разделе Блокировка ресурсов для предотвращения непредвиденных изменений](../articles/azure-resource-manager/management/lock-resources.md).
