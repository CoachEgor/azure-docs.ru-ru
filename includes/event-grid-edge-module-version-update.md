>[!NOTE]
> Всегда проверяйте версию образа, чтобы убедиться в том, что для него требуется феаутурес. При наличии предыдущей версии образа контейнера, уже извлеченного на компьютере, необходимо указать нужный тег версии или удалить существующий образ, прежде чем повторно извлечь его с помощью `:latest` тега. Дополнительные сведения о версиях и тегах образа см. в [заметках о выпуске](../articles/event-grid/edge/release-notes.md).
