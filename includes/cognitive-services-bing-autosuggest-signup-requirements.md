Необходимо иметь [учетную запись API Cognitive Services](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) с доступом к API автозаполнению Bing. Если у вас нет подписки Azure, создайте бесплатную [учетную запись](https://azure.microsoft.com/try/cognitive-services/?api=bing-news-search-api). Перед тем как продолжить, потребуется ключ доступа, предоставляемый после активации бесплатной пробной версии или ключ платной подписки, указанный на панели мониторинга Azure.
