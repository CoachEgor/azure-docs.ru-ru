Устройство связано с учетной записью хранения данных, которая используется в качестве пункта назначения данных в Azure. Доступ к учетной записи хранилища контролируется подпиской и двумя 512-битными ключами доступа к хранилищу, связанными с этой учетной записью хранения.

Один из ключей используется для проверки подлинности, когда устройство Data Box Edge получает доступ к учетной записи хранилища. Другой ключ держится в резерве, так что вы можете периодически вращать клавиши.

По соображениям безопасности многие центры обработки данных требуют вращения ключей. Рекомендации по смене ключей.

- Ключ учетной записи хранения похож на корневой пароль для вашей учетной записи хранения. Тщательно защитите ключ от учетной записи. Не распространяйте пароль среди других пользователей, жестко кодируйте его или не сохраняйте его в любом месте простого текста, доступного другим пользователям.
- Регенерировать ключ учетной записи через портал Azure, если вы считаете, что он может быть скомпрометирован. См. сведения о том, как [управлять ключами доступа к учетной записи хранения](../articles/storage/common/storage-account-keys-manage.md).
- Администраторazру должен периодически менять или регенерировать основной или вторичный ключ, используя раздел хранения портала Azure для прямого доступа к учетной записи хранилища.
