## <a name="configure-the-sensor-fingerprint-provider"></a>Настройка поставщика датчика отпечатков пальцев

Начнем с создания и настройки поставщика датчика отпечатков пальцев. Это позволит получить информацию с платформозависимых датчиков на вашем устройстве и абстрагировать ее в общее представление для сеанса привязки облака.
