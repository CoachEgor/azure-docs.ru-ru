**Общественные зоны DNS**

| Ресурс | Ограничение |
| --- | --- |
| Общественные DNS зоны за подписку |250 <sup>1</sup> |
| Рекордные наборы на одну публичную зону DNS |10 000 <sup>1</sup> |
| Рекорды за рекорд, установленный в публичной зоне DNS |20 |
| Количество записей Alias для одного ресурса Azure |20|
| Частные зоны DNS за подписку |1000|
| Рекордные наборы на частную зону DNS |25000|
| Рекорды в записи, установленные для частных зон DNS |20|
| Виртуальные сетевые ссылки на частную зону DNS |1000|
| Виртуальные сети Ссылки на частные зоны DNS с автоматической регистрацией включен |100|
| Количество частных зон DNS, к которые виртуальная сеть может быть связана с включенной автоматической регистрацией |1|
| Количество частных зон DNS, которые могут быть связаны виртуальной сетью |1000|
| Количество DNS-запросов, которые виртуальная машина может отправить в Azure DNS resolver, в секунду |500 <sup>2</sup> |
| Максимальное количество DNS-запросов в очереди (в ожидании ответа) на виртуальную машину |200 <sup>2</sup> г. |

<sup>1</sup> Если необходимо увеличить эти ограничения, обратитесь в службу поддержки Azure.

<sup>2</sup> Эти ограничения применяются к каждой отдельной виртуальной машине, а не на уровне виртуальной сети. DNS-запросы, превышающие эти пределы, отменяются.
