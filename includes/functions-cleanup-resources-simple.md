## <a name="clean-up-resources"></a>Очистка ресурсов

Когда завершите работу с ресурсами, созданными в рамках этой статьи, удалите их из своей подписки с помощью следующей команды:

```azurecli
az group delete --name myResourceGroup
```

При появлении запроса введите `y`. Эта команда удаляет группу ресурсов `myResourceGroup` и все ресурсы, которые принадлежат группе.
