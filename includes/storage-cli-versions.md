## <a name="versions-of-the-azure-cli"></a>Версии Azure CLI

Сейчас доступны две версии интерфейса командной строки Azure (CLI):

* [Azure CLI.](../articles/storage/common/storage-azure-cli.md) Текущая версия интерфейса командной строки Azure, для использования в модели развертывания Resource Manager.
* [Классическая модель: CLI Azure](../articles/storage/common/storage-azure-cli-nodejs.md): Более старую версию Azure CLI, предназначен для использования только с помощью классической модели развертывания.
