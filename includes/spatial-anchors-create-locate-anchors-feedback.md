## <a name="provide-feedback-to-the-user"></a>Пользователь должен предоставить отзыв

Можно написать код для обработки события сеанса обновлены. Это событие возникает каждый раз, чтобы сеанс помогает его понимать обстановке. Таким образом, дает возможность:

- Отзыв для пользователя, а это устройство окажется сеанса обновляет его основные сведения о среде.
- Определить, в какой момент существует достаточно отслеживались пространственных данных для создания или найдите пространственных привязки — мы узнаем, см. на более позднем этапе.
