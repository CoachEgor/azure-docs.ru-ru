Сейчас только твердотельные накопители уровня "Премиум" могут включать общие диски. Размеры дисков, поддерживающие эту функцию, — P15 и выше. Для разных размеров дисков может быть задан другой предел `maxShares`, который нельзя превысить при задании значения `maxShares`.

Для каждого диска можно определить `maxShares` значение, представляющее максимальное количество узлов, которые могут одновременно совместно использовать диск. Например, если планируется настроить отказоустойчивый кластер из двух узлов, то следует задать `maxShares=2`. Максимальное значение — это верхняя граница. Узлы могут присоединиться к кластеру или выйти из него (подключить или отключить диск), если число узлов ниже указанного значения `maxShares`.

> [!NOTE]
> Значение `maxShares` может быть задано или изменено только при отсоединении диска от всех узлов.

В следующей таблице показаны допустимые максимальные значения для `maxShares` по размеру диска:

|Размеры диска  |ограничение Максшарес  |
|---------|---------|
|P15, P20     |2         |
|P30, P40, P50     |5         |
|P60, P70, P80     |10         |

Значение `maxShares` не влияет на операции ввода-вывода и ограничения пропускной способности для диска. Например, максимальное число операций ввода-вывода для диска p15 — 1100, если Максшарес = 1 или Максшарес > 1.
