> [!NOTE]  
> Часовой пояс по умолчанию, используемый для запуска выражений CRON, является координированным универсальным временем (UTC). Чтобы выражение CRON было запущено на основе другого часового пояса, создайте настройку приложения для приложения функции под названием WEBSITE_TIME_ZONE. Чтобы узнать больше, смотрите [часовые пояса NCRONTAB](../articles/azure-functions/functions-bindings-timer.md#ncrontab-time-zones).
