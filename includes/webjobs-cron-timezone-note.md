> [!NOTE]  
> Часовой пояс по умолчанию, используемый для выполнения выражений CRON, — это время в формате UTC. Чтобы выполнение выражения CRON выполнялось на основе другого часового пояса, создайте параметр приложения для приложения функции с именем WEBSITE_TIME_ZONE. Дополнительные сведения см. в разделе [нкронтаб Time Zones](../articles/azure-functions/functions-bindings-timer.md#ncrontab-time-zones).
