## <a name="provide-frames-to-the-session"></a>Укажите кадров к сеансу

Сеанс пространственных привязки работает путем сопоставления пространство вокруг пользователя. Это способствует для определения, где находятся привязки. Мобильных платформ (iOS и Android) требуется собственный вызов к камере, веб-канала для получения кадров из библиотеки AR вашей платформы. Напротив HoloLens постоянно сканирует среде, поэтому нет необходимости для определенного вызова, как с помощью Mobile.
