# <a name="virtual-machines-documentation"></a>Документация по виртуальным машинам
Виртуальные машины Azure предоставляют по запросу безопасную высокомасштабируемую виртуализированную инфраструктуру. Вы можете создавать виртуальные машины Windows либо виртуальные машины Linux под управлением Red Hat, Ubuntu или дистрибутива Linux на ваш выбор. Чтобы узнать, как это сделать, ознакомьтесь со следующими материалами:

- [Создание виртуальной машины Windows](/azure/virtual-machines/windows/quick-create-portal/?WT.mc_id=UIHelpPilot).
- [Создание виртуальной машины Linux](/azure/virtual-machines/linux/quick-create-portal/?WT.mc_id=UIHelpPilot).
